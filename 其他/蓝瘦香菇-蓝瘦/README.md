

**蓝瘦香菇双手简谱** 和五线谱完全对应。

_蓝瘦香菇_
是“难受，想哭”的意思。广西的壮语里面的发音没有翘舌音，没有送气音。2016年10月，广西南宁一小哥失恋后录视频：“蓝瘦，香菇”，本来今颠高高兴兴，泥为什莫要说这种话？蓝瘦，香菇在这里。第一翅为一个女孩屎这么香菇，蓝瘦。泥为什莫要说射种话，丢我一个人在这里。

这里的蓝瘦香菇是微博蓝小瘦 蓝瘦版本。

歌词下方是 _蓝瘦香菇钢琴谱_ ，希望大家喜欢。
