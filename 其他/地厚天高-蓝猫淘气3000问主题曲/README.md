

**地厚天高**
是科普动画系列动画片蓝猫淘气3000问的主题曲。动画片将故事情节和少儿知识巧妙地融合在一起，是由湖南蓝猫卡通传媒有限公司制作的儿童科普动画系列，于1999年10月北京电视台青少频道首播。目前全集3000集，导演王宏（幽默、星战、恐龙、海洋、运动系列），扶梅陵、滕海波（太空历险记系列），贺梦凡等。

  
歌词下方是 _地厚天高钢琴谱_ ，希望大家喜欢。
