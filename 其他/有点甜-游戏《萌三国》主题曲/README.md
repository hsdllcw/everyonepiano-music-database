

**有点甜双手简谱** 和五线谱完全对应。

_有点甜_ 收录在汪苏泷的第二张实体专辑《万有引力》，发行于2012年6月26日，由汪苏泷作词作曲，这是汪苏泷与新加坡超人气双胞胎组合BY2组合首次合作。

有点甜是骏梦网络研发，腾讯游戏发行的网页游戏《萌三国》的主题曲,是2013年台湾偶像剧《借用一下你的爱》里的插曲,也是郑爽、杨洋主演2016年《微微一笑很倾城》电视剧的插曲。

歌词下方是 _有点甜钢琴谱_ ，喜欢的朋友可以免费下载学习。
