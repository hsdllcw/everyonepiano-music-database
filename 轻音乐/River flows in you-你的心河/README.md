_你的心河_ ** _-River flows in you_** 是韩国钢琴家李闰珉（Yiruma）2002年7月的专辑《 初恋》（First
Love）里的第4首钢琴曲。直译——流淌在你身体里的河。有两个版本，风格不同，较慢的弹奏版本柔情似水，较快的版本像欢快的溪流，而又不失水的柔美。
_你的心河-River flows in you_ 钢琴曲优美而略显伤感，温暖而柔情四溢。
