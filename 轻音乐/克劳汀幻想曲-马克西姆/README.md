_克劳汀幻想曲_ 是克罗地亚（The Republic of Croatia）钢琴演奏家马克西姆•姆尔维察（Maksim
Mrvica）作品，亦被译为克劳丝汀幻想曲。

且此曲收录在其纯音乐专辑The Piano
Player中。他的弹奏风格也是别具一格，与以往我们对钢琴演奏的理解完全不同，从他演奏的过程中我们可以感受到四射的激情，澎湃的热力和那种无处不在的节奏感，难怪有人会把他称作为"钢琴界的瑞奇•马丁"。如果说"钢琴王子"理查得•克莱德曼给我们的是一种古典中带有现代的钢琴曲风给了我们一个小小的惊喜的话，那现在的迈可森则是将所有现代的音乐元素都融入到了钢琴的演奏中，是对传统的一种彻底的颠覆，呈现出一种新鲜的魔幻色彩，这也就是人们把他称为钢琴玩家的原因。只有你亲身经历过之后才会发现他的音乐是如此的具有感染力，也会深深的为之所倾倒、所折服。
