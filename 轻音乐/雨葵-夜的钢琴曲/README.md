_雨葵_
出自石进《夜的钢琴曲》系列，这首作品融入了淡淡的哀愁和失落，同时又隐藏着一种期待和盼望。曲目的名字《雨葵》，雨天的向日葵，瞬间便读懂了其中的含义。汉语言文学，物喻的手法是很常见的，这种以物化来阐释心情的方式，温婉，曼妙。当然不可或缺的是，作为读者和聆听者必须有着细腻和灵敏的感知观。  
  
从2006年开始，三年多的时间，石进写了31首夜曲，这每一首夜曲，背后，都或多或少一个小故事，有些是他自己的，有些是朋友的，用音符把它一一记录，记录这些曾经走过的日子……
2011年出版夜的钢琴曲CD。首次出版即售空，随后《夜的钢琴曲—石进原创钢琴曲14首》一书发行，由安徽文艺出版社进行对书的出版工作，14首夜曲+14首曲谱+14个夜曲背后的故事，CD为赠送。
第一版内作品有：《花的微笑》《可惜不是双鱼座》《恍若如梦》《街道的寂寞》《惜别》《下雨时你会想起谁》《幸福时光》《一个人的时光》《忆》《雨葵》《隐形的思念》《夜曲五》十二首曲目。
2011年在其博客更新了最新的作品。包括《1981》《一封家书》《这一程，情深缘浅》《回忆的温度》 《被时光移动的城市》《白羊座的忧伤》《忘了我爱你》。
