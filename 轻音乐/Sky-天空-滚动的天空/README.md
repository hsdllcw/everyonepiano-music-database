_Sky_
，中文名为天空，是节奏休闲游戏《滚动的天空》的背景音乐。滚动的天空虽然是一款休闲游戏，但玩起来远比很多竞技类游戏都让人神经紧绷，游戏节奏非常的快，共有五个开放关卡，每个关卡都有不同的主题，温婉而带有韧性的第一章，充满神秘感与冒险气息的第二章，暗流涌动踏入丛林的第三章，紧张诡秘而机械的第四章，以及红黑碰撞的烈焰地狱终章。

滚动的天空中每个主题除了场景风格会有巨变以外，音乐也是风格也是完全不一样的，在这里不得不说这款游戏的背景音乐真的非常的棒。虽然画面与音乐不简单，但是游戏的玩法并不难，玩家只需要通过按住屏幕滑动，控制一个小球闯入一个充满障碍和奇异地形的天空之城，不过若稍有不慎碰上障碍物或是从平台坠落，小球就会完全粉碎，游戏就会结束。
