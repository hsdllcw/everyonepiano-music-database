_The Scale Song_ 是游戏寻找天堂中的插曲。

寻找天堂（Finding Paradise）是一款由Kan “Reives” Gao和他的独立游戏开发团Freebird
Games制作的游戏，已经于北京时间2017年12月15日发售。故事的主人公是此前Freebird Games制作的前传游戏《鸟的故事》中的小男孩Colin
Reeds。该作为《去月球》（To the Moon)的续集，前传为《鸟的故事》（A Bird
Story）。如同《去月球》一样，《寻找天堂》是一个独立完整的游戏，无需玩过任何本系列的游戏即可充分体验。
