

**秋月梧桐** 是由屠颖演唱的歌曲。屠颖，台湾地区乐坛编曲大师，华人演唱会顶级音乐总监与键盘手。
台湾乐坛重量级编曲大师—屠颖，音乐风格流行悦耳兼容古典严谨，
历年编写创作华语流行音乐作品超过两千余首，近二十年传唱佳作无数。屠颖长年参与流行专辑制作的历练及出神入化的琴艺，早为音乐界称道多时。近年华语乐坛演出活动兴盛，屠颖始终为两岸三地歌手个人演唱会顶级键盘手的不二人选。

_秋月梧桐_ 收录于其第二张钢琴演奏专辑《风·月·花·鸟》中。《风·月·花·鸟》精湛琴艺寄意深远，让知名布袋戏原曲超越时空，达春和景明之境。
