

**皮尔之街双手简谱** 和五线谱完全对应。

_皮尔之街_ 即是波尔维尔，也是第二号夜曲，由班得瑞团队演奏，收录在专辑《水之印象》中。

《Inmpression
Of_Water》(水之印象)是描写水的音乐合辑，其中包括了许多大家熟知的音乐，如：“雨后”、“莎莉花园”等，但也有一些以前听得不多的音乐。

乘着音乐的翅膀，展开一场卓绝的航程，一路听着海洋唱着歌，诉说着不同的波折与心情，发现每一朵浪花、每一簇浪群，在不规律中带点相同的节奏。和着吉他的轻柔、钢琴的纤细、长笛的悠扬，您聆听的不再只是音乐，而是人与自然最无暇的心灵呼唤。
