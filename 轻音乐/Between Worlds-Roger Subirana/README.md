_Between Worlds_ 是西班牙新世纪音乐作曲家Roger Subirana创作的曲子。

Roger Subirana
，自学成才的天才型音乐家。12岁时开始创作音乐，14开始录制音乐。他说：对我来说，做音乐就是一种生活方式，我需要用音乐来表达自己的感受。因而从最简单到最宏伟，无论悲伤还是高兴，成功或失败，专业与否都不能阻止我对音乐的热爱。Roger
Subirana的音乐不是为了吸引大众，而是非常个人化的音乐，他的音乐从不受风格所限，电影配乐和电视广告歌曲，新世纪，电子，现代爵士，流行，摇滚，摇滚交响乐，舞蹈，科技，古典音乐，凯尔特等都是他的创作空间。
