是模拟养成游戏《中国式家长》的背景音乐。这是一款现实主义的模拟游戏，游戏模拟从出生到高考的整段过程，探讨中国孩子与家长之间的关系。游戏中，玩家以扮演家长的角色，参与规划一个孩子从出生到高考的过程。除了小升初、中考、高考等中国学生熟悉的考试外，游戏中玩家还可以到“小卖部”购买零食、玩具，在班中和其他同学竞选班干部，甚至面对来访的亲戚朋友，玩家需要在合适的时机摆出孩子的种种“特长”，赢取“面子战争”的胜利。
