

**Seve双手简谱** 和五线谱完全对应。

_Seve_ 是法国歌手Tez
Cadey所创作并演唱的一首电子音乐。歌曲语言为斯瓦希里语。该歌曲改编于一首创作于80年代的基督教歌曲《OSifuniMungu》。

Tez Cadey，一位年仅23岁的在美国出生法国长大的音乐制作人，
2011年出道并活跃至今。他创作的魔性神曲《seve》搭配起源于蒙古的鬼步舞视频，上传youtube后便“病毒”式传播，风靡全球各大视频网站。据悉，Youtube相关视频播放量破200亿，在微博上更是掀起一股全民“鬼步舞”的狂潮，成为各大场合所之青睐的背景音乐。
