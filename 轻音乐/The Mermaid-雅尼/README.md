_The Mermaid_ 是希腊裔美国音乐家雅尼的曲子，收录在其1987年的专辑Out of Silence之中。

很多人把雅尼的音乐隶属于新世纪音乐，但雅尼并不这样认为。雅尼说：“我不喜欢对任何艺术贴上标签。对于我而言，我是在写音乐，而不归属于任何一种特殊的风格，包括人们说我的音乐涵盖古典到摇滚的风格。我实在想不出什么标签能够适用于音乐。标签在现实生活之中是有帮助的，比如在小卖店里的分类使用。但对于音乐或艺术，是不合适的。我的音乐是我的灵魂以及我生命中所有经历的真实反映，没有定义。”

所以雅尼之声本质上讲是没有固定风格分类的。
