_夜的钢琴曲 28_ 由国内作曲家石进创作，收录在其专辑《夜的钢琴曲》中。2011年5月，该专辑提名华语金曲奖，以及年度最佳独奏专辑。

石进的生活简单而平静，这座城市很少有人知道，冯小刚电影《非诚勿扰》中红极一时的插曲《夜的钢琴曲》（五），就出自这个年轻人之手。而他推荐给冯导即将上映的贺岁大片《一九四二》的另一首钢琴曲，虽然遗憾落选，但这并不影响他的创作热情。11月下旬，石进的第二张原创钢琴曲专辑将出版发行，10月30日，面对记者的采访，他说，每个人记录生活的方式都不一样，他只是用音符去记录，生活还在继续，《夜的钢琴曲》系列就不会停止。
