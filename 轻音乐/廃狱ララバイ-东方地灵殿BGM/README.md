_廃狱ララバイ_ 是游戏东方地灵殿 ～ Subterranean Animism的背景音乐。

东方地灵殿 ～ Subterranean Animism.（日语：东方地霊殿 ～ Subterranean
Animism.）是由同人组织上海爱丽丝幻乐团制作的弹幕射击游戏，是东方Project的第11作，也是发布在Windows平台上的（新作）第六个整数作，于2008年8月16日发售。

该作的系统大体上承继了前作《东方风神录》，但也有些是以之前的各作品为基础改变而成的。本作系列中重新开放了擦弹统计与决死Bomb。自机的选择和前作一样，都是博丽灵梦及雾雨魔理沙，不过武器选择则变为支援角色的形式，2名自机角色分别都有3名支援角色，总其有6种自机与支援角色的组合。游戏模式则是4种难度，共6关，达成“不续命通关Normal难度”的条件后可以开放Extra模式挑战。
