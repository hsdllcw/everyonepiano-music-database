

一首无数人循环的小提琴曲《 **告白之夜** 》，由日本小提琴手Ayasa（岛村绚纱）演奏。欢愉的乐章，如同面对心上人的狂喜悸动。

所谓告白，指的是《告别陈述》取材于《竹取物语》，辉夜姬在即将回到月球前，跟养育自己多年的爷爷奶奶告别陈述自己的甚是这一情节。

养育之恩，担得起这首曲子表现出来的离别之悲；而爱情，如果要强行联系，则显得过去单薄。
