_VALLIS-NERIA_ 初出于《动感弹珠》，英文为Reflec Beat，是出自BEMANI音乐游戏一首唤醒本能的曲目。

作为YOSHITAKA曲，不负众望的成为了推出时的高难度曲。不仅节奏复杂，谱面使用大量黄条锁手，中间还有高速纵连，是当时一般的中级玩家也难以过关的曲目。

《动感弹珠》官方简称“リフレク”或“Rb”。与pop'n
music和jubeat一样并非是对特定乐器或音乐器材的模拟游戏。而是引入了触摸屏以及对战的要素。这也是Bemani系列里首款使用触摸屏幕设备的游戏系列。改作融合了音乐游戏以及气垫球的游戏理念，配合音乐节奏，将画面上朝己方判定线飞来的节奏标靶以触控方式打回给对方，以获得的分数来与对手一较高低。
