

理查德·克莱德曼
1953年生于巴黎。五岁开始随其担任钢琴教师的父亲学钢琴；六岁时即被人们视为天才。十六岁时，他以优异成绩毕业于巴黎国立音乐学院，以擅长演奏肖邦、拉威尔、德彪西等人的作品著称。然而在偶然之间，他对通俗音乐产生了兴
趣，并且不顾亲友的反对，毅然转变方向。为了更好地表现克莱德曼的音乐气质，应注意下列几点：
1．弹奏旋律应触键明确，使音色充满光辉。尤其是高音区的单音和八度音程的旋律，由于钢琴本身的构造关系，音量变小。因此，在弹奏时要尽力克服，使音量增厚。
2．大多数的音都必需注入大量感情，悠然自得、轻松愉快地弹奏。 3．乐句不必过于精雕细琢，过分纤细反而有损整个音乐形象。
4．伴奏应有深厚的音质与明快的节奏。由于伴奏音型的跨度较宽，声音不易控制，因此要注意左手与右手旋律的音量比例。
