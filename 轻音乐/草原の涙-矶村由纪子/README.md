_草原の涙_ 是日本钢琴女作家矶村由纪子的作品。收录在其专辑《风居住的街道》中。

矶村由纪子的音乐被公认是“上等的治愈佳品”，而且其中包含活力。所以，会让人有百听不厌的感觉。同名曲《风居住的街道》自不用说，是2003年矶村由纪子与日本著名二胡演奏家坂下正夫合作的经典曲目，二胡与钢琴的搭配对话，令人耳目一新。整体上来说，感觉二胡的比重大于钢琴，或者说是感染力超越了钢琴，二胡的忧伤在这里胜过了钢琴的浪漫，专辑中其它的曲目都是值得期待的。

矶村由纪子的音乐可说是空洞清灵直入人心，听着她的作曲总会使人内心感到清凉和感动。反反复复的去欣赏她的作品总也使人一次又一次的感动。悲伤情怀也在你的眼泪中慢慢结束。
