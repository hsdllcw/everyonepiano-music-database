从DNF剧情上来看，列瑟芬本为诺斯玛尔的一个村子，在狄瑞吉来到之后，村子瘟疫横行，村民们也都被感染变成了怪物，故名为痛苦之村。居住此地的暗精灵们一直以为自己的族人大部分身患重病，是因为人类造成的，但其实是第六使徒的到来导致疾病散发的。副本中的地图配合BGM，给人一种对眼前景象深深的悲伤却又无可奈何的绝望感。

可以说《 _Leshphon_ 》讲述了一段存在即是错误的悲剧命运。音乐阴沉诡秘，非常符合狄瑞吉万疫之源的恐怖氛围。
