

**Enchantment双手简谱** 和五线谱完全对应。

_Enchantment_ 出自希腊裔美国音乐家雅尼的专辑In My
Time。雅尼是许多人最初接触的新世纪音乐家之一。大家一直都十分喜欢雅尼的音乐，除了因为流畅的旋律和自由随兴的编曲，还因为其中洋溢着乐观的精神、澎湃的激情和浓烈的感染力。诚然，雅尼的音乐因为富于鲜明的表演性质，一直受到评论家们的诟病，不过这也正是其令人如痴如醉的魅力来源，假如完全辟除这样的特质，那就不是雅尼了。可以说这就是雅尼音乐的个性吧，无关好或不好，只看是否对你的胃口罢了。更何况，雅尼其实还有为数不少的钢琴小品，内敛沉静、细腻动人的演绎迥异于其他作品大开大合的风格，值得细细品味，1993年
In My Time是他纯钢琴作品的集大成之作。

作为一个拥有独特创作方式的音乐人，雅尼不愿被纳入任何流派，他独创着一般魅力独具的音乐潮流。他擅长用电子合成技术将古典音乐以现代人的视角做重新诠释。在繁忙喧闹的快节奏生活之中，偷得半日闲来聆听雅尼的音乐，也许可以找寻到一份久违了的轻松和休闲。雅尼更是第一个人能将新世纪音乐音乐用如此平易近人的形式带到普通人面前，他对于新世纪音乐的推广普及起到了不可估量的积极作用。
