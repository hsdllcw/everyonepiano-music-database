

**Flower Dance** 是Dj Okawari的歌曲，收录在其专辑《A Cup Of Coffee》中。  
  
而 _Flower Dance_ 直译为花之舞。作者Dj Okawari来自于日本静冈县，曲风以jazz-
hiphop和chillout为主，有一种十分空灵自然的唯美感。
