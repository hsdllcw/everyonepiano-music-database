

**雨的印记** ，一首公认的好听的钢琴曲，出自韩国最擅长描会爱情的音乐家 李闰珉 YIRUMA之手，《夏日香气》的背景钢琴曲。  
  
小编特意为[
**EOP魔鬼训练营**](sale-10-【EOP魔鬼训练营】无敌键盘钢琴指法教程-30天成为钢琴大师.html)小伙伴们制作了带伴奏版文件及对应的伴奏，大家可以试听带伴奏版钢琴曲，或者跟随纯伴奏自己弹奏。该曲左手和弦部分没有任何难点，需要注意的是右手连续十六分音符的弹奏和三连音的节奏把握。  
  
此曲是4/4拍子，C大调，五线谱、双手简谱和EOP文件是对应的版本。  
  
写这首歌的时候，是在一个星星满天的夜晚，忽然间一场雨，让YIRUMA有感而发写下 _雨的印记_ 这首曲子。  
  
这首曲子，跳跃的优美，吟唱的动情，适合背诵时听，适合失恋时听，适合被表白时听。它让我们感受到生命的美丽，感受到人生的真谛。不同的人都会从这首曲子里找到最真实的自己。

### 小编听到的雨的印记：

一首好的曲子就是在任何场合，任何时候都能找到当时的感觉。无论是在自习室，办公室，卧室，甚至是在被表白的时候，它都能带给我们最真实的感觉。  
  
窗外雨滴轻柔地落在窗户上，带着耳机静静地听《 **雨的印记**
》这首曲子会使我们想起以前的点点滴滴，虽有些淡淡的伤感，但是它清新优美的旋律又会将伤感慢慢治愈，给你最美好的向往。  
  
窗外的雨，不知什么时候悄然停止。雨过天晴，闻着阳光的味道，心情豁然开朗。潮湿的感觉已然消失，只看到充满生机的世界，反复在诉说雨的印记。
