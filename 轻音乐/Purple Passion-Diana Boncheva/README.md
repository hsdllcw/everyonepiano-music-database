Virus》相信大家一定很熟悉了，改编自贝多芬：第八钢琴悲怆奏鸣曲(Grande Sonate Pathétique)第三乐章回旋曲Rondo
Allegro。Diana Boncheva在韩国WEG比赛上最先演奏这一曲目，后来该曲名声大振；而我们今天这首《Purple
Passion/紫色激情》的名气更是有过之而无不及。
