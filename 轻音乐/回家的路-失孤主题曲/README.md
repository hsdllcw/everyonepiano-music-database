

**回家的路**
是华谊兄弟、聚焦“打拐”题材的电影《失孤》的主题曲。该电影由著名影视剧编剧、制作人彭三源导演自编自导，刘德华和井柏然主演。刘德华饰演一位寻子十五载的农民父亲雷泽宽，井柏然则饰演的是外表阳光但内心忧伤的修车小伙曾帅，一次修车经历让两人结识，在了解到雷泽宽的寻子历程之后，曾帅不仅没有收取修车费用，更与雷泽宽一起踏上寻亲之旅。

  
《 _回家的路_
》这首歌的歌词由刘德华本人亲自操刀。与刘德华搭档的曲作者是陈思函。伴随着春运大幕的开启，羊年春晚剧组提前将此歌在网上公布，希望将此歌作为羊年春晚预热宣传曲，陪伴回家过年的游子踏上“回家的路”。刘德华这首新歌将成为回家征程上心灵的慰藉，拂去旅途的疲惫。

  
歌词下方是 _回家的路钢琴谱_ ，希望大家喜欢。
