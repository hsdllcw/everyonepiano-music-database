_街道的寂寞_ 是石进弹奏的一首钢琴曲，收录于《夜的钢琴曲 石进原创钢琴曲14首》中。

音符轻柔如水，细腻如诗，像一股清泉润泽心田；更像一颗石子，撩起千千思绪的涟漪，落在每个人的心田。

据悉，石进老师写这首音乐的时候，是在一个深夜下班回来。街道，在夜灯下，只有他一人，心中感触而创作出这首曲子。石进老师很多夜的钢琴曲系列，正如他说很多人用文字记录自己的点滴，他用音符记录自己的思绪，让生活变得更加诗意与美好。
