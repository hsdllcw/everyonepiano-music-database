帕萨卡利亚(Passagalia)是一种三拍子的慢板舞曲，大多用低沉的小调写成，以4～8小节的固定低音为基础写成的短曲，来围绕着主题而进行连续变奏。帕萨卡利亚有这样的特点，是因为它是一种在16世纪末起源于西班牙的民间舞蹈，由于最初吉他在音乐中的贡献，所以帕萨卡利亚保留了这种低音变奏的特点，从而发展成为在固定低音旋律之上进行变奏的分段结构的变奏曲。
