_Canon in D Minor_ ，即此首卡农是D小调的，比较简单。

卡农是一种音乐谱曲技法，复调音乐。卡农的所有声部虽然都模仿同一个声部，但不同高度的声部依一定间隔进入，造成一种此起彼伏的效果，连绵不断的效果，轮唱也是一种卡农。在卡农中，最先出现的旋律是导句，以后模仿的是答句。

根据各声部高度不同的音程差，可分为同度卡农，五度卡农，四度卡农等；根据间隔的时间长短，可分为一小节卡农，两小节卡农等；此外还有伴奏卡农，转位卡农，逆行卡农，反行卡农等各种手法。
