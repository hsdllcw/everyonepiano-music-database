最终幻想Ⅹ（ファイナルファンタジーX；FINAL FANTASY
X；简称：FFX；FF10），是日本SQUARE公司（史克威尔艾尼克斯）发行的角色扮演类电子游戏。该作于2001年7月19日在日本作为PS2的游戏软件进行首发。该作的一大特征是，作为最终幻想系列作品，少见的采用了很多东洋风的主题及世界观设计。该作是在PS2平台上发行的首部最终幻想作品，并且是SQUARE和ENIX合并前由SQUARE单独制作的最后一部最终幻想作品。
