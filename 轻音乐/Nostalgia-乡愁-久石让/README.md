

**Nostalgia-乡愁** 是久石让推出的钢琴故事第3辑中的曲子。

日本音乐大师久石让大家一定不会陌生，他的众多电影配乐原声专辑都是音乐爱好者的珍藏CD。《Piano Stories III: Nostalgia
钢琴故事3：乡愁》是他的“Piano Stories
钢琴故事”系列的第三张专辑，发行于1998年。在钢琴绵延的主旋律中，淡淡的忧伤情怀在空气中慢慢飘散开来，充盈了听者的内心。虽然我更喜欢充满阳光的音符，喜欢每天都开开心心，但是，偶尔的忧伤总能在我脑海中烙下深刻的印记。也许正因为平日的开心总是人们的追求，更显得忧伤的刻骨铭心。

专辑小标题是“ _Nostalgia-乡愁_
”，让身处异地的我不由得想起台湾著名诗人余光中的名作《乡愁》。这是脑海里永远不会抹去的一缕丝、一面纱。是呵，那远远的故乡啊，止不住的乡愁就是游子与你唯一的纽带，让在外奋斗漂泊的人们在辛劳之余得以回头望望，然后再鼓足勇气继续向前。将《Piano
Stories III: Nostalgia 钢琴故事3:乡愁》送给所有身在家乡以外的游子们，想说一句：无论多么忙，有空的时候一定回家看看。
