_流星哀歌_ 是游戏口袋妖怪 红宝石•蓝宝石中的音乐。该游戏是口袋妖怪系列的Game Boy Advance游戏，由GAME
FREAK开发，任天堂发行。游戏最早于2002年11月21日在日本上市，后于2003年推出美版。游戏在各地区发售的两年后发行了续作《口袋妖怪
绿宝石》。三款游戏均属口袋妖怪系列第三世代的主系列游戏。游戏的复刻版，任天堂3DS游戏《口袋妖怪终极红宝石•始源蓝宝石》于红蓝宝石版日本发售日12年后的2014年11月21日全球发售。

游戏玩法和操作方式与前作基本相同，玩家以鸟瞰的视角操作主角进行游戏。与前作一样，游戏的主要目的是打败四大天王并完成神奇宝贝图鉴，同时挫败敌对组织的计划。本作所新增的元素包括双打对战与神奇宝贝的特性和135只新神奇宝贝。由于GBA的功能较以前的主机有所增强，因此可提供4位玩家的联机对打。游戏可以连接任天堂E读卡机和其它第三世代游戏。
