

**情书** 出自Dj Okawari的《Mirror》专辑唯美的封面，用钢琴声，电子鼓和人声塑造出完美的Chillout。  
  
日本著名Chillout Dj—Okawari 09倾力打造Chillout专辑[Mirrior]。  
继08专辑[Diorama]后，Dj Okawari陆续发行单曲Sound Of Silence，You Gotta Be，并得到人们的认可。  
  
这张Mirror绝对可以让你感受到Chillout音乐的美妙，沉醉其中。

专辑的第二首曲子《 _Luv Letter_ 》，钢琴和电子鼓的结合，开始很缓和，接着节奏开始有跳跃感，中间那段的转折，慢慢的升华，有很美妙的感觉。
