_夜的钢琴曲 23_
是作曲家石进凭着自己业余爱好创作的《夜的钢琴曲》系列中的一首曲子。夜的钢琴曲是作者石进从2006年开始，历时3年半创作的31首乐曲。石进写的钢琴曲不是多么精深、高妙，演奏起来也不需要什么高难度的技巧，但却打动了包括冯导在内的众多听众，南宁钢琴家陈斯凡也很喜欢“夜五”，他觉得石进的作品非常真诚。对于“夜五”，石进坦言背后没有太多复杂的故事，“录音是即兴的，弹了三遍，"夜五"花了40分钟就完成了，只是记录爱与被爱的体会。”
