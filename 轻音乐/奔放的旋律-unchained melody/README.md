

**歌曲本名** ：unchained melody  
**作曲** ：Alex North  
**作词** ：Hy Zaret  
**首发** ：1955年作为监狱题材电影<Unchanged>  
**配乐流传** ：1965年由The Righteous
Brothers翻唱，1990电影《GHOST（人鬼情未了）》成功地引用了此版本作为插曲，使之风靡全球，成为20世纪一首经典流行音乐。
