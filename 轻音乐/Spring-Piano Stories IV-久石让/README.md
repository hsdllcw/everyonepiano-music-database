

**Spring双手简谱** 和五线谱完全对应，是EOP教学曲。

_Spring_ 是2005年久石让的第4张钢琴故事系列专辑《Piano Stories IV-Freedom》（奔放的音符）中的收录曲。

在久石让的专辑《Freedom Piano Stories
4》中，收录重新演奏之宫崎骏史上最高票房记录动画电影“哈尔的移动城堡”主题曲“人生的回转木马”、山多利绿茶“伊右卫门”广告曲、Brnesse“进研升学”广告曲《spring》、Tohato“焦糖玉米花”广告曲等广告曲，久石让与来自加拿大的11位弦乐手所组成的La
Pieta竞演，强烈的琴音、细腻的演奏技巧、丰沛的情感，将那熟悉的动人乐章，再一次让人萦回心中。电影、广告，还有流传在街头巷尾，那温暖人心的感动音乐，世界级电影配乐大师久石让的最新经典旋律就在这张专辑中。

久石让，日本著名作曲家、歌手、钢琴家，以担任电影配乐为主。特别是宫崎骏导演的作品，从《风之谷》至《悬崖上的金鱼公主》的二十多年间所有长篇动画电影的音乐制作，为宫崎骏作品中不可欠缺的配乐大师。

相信很多人都非常喜欢久石让的音乐吧。他的音乐里总是充满着幸福温暖的感觉，让人会觉得这个世界真是太美好啦。“ _spring_
”，原意有春天的意思。倾听这首曲子，会使人不经意的联想到春天里的气息，联想起很多美好的瞬间，就像春风拂面的感觉，很温暖很舒服。
