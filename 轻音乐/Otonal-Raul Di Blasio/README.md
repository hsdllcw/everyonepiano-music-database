

**Otonal** 是拉丁钢琴家Raul Di Blasio的音乐作品。其作品的影响跨越所有文化和宗教领域，Raul Di
Blasio的成功及其广受赞誉的作品已经超过了他最初的期望。Raul Di
Blasio从小就是一个音乐神童，他在个人乐章上的极富古典韵味的表达和技巧使其从阿根廷的农场生活中脱颖而出，最终登上了洛杉机国际剧院的舞台，并参加了全球巡演。

另外俄罗斯著名花样滑冰运动员大卡娅(Maria Butyrskaya) 98年长野冬奥会上自由滑音乐采用了这首《 _Otonal_
》，并在1999年以这套动作获得世界锦标赛冠军。
