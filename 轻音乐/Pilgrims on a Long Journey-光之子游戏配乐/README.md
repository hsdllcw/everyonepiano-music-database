_Pilgrims on a Long Journey_ 是光之子游戏热门配乐。由加拿大创作型歌手Coeur de Pirate创作。

《光之子》是育碧使用雷曼UbiArt引擎所开发制作的2D绘本风格JRPG游戏，曾绘制《最终幻想》的日本漫画家天野喜孝负责该作原画设定，游戏采用水墨风格绘画。该作于2015年3月20日正式发售简体中文PC版。

该游戏以同名童话故事为背景，玩家在游戏中经历一次冒险之旅。玩家将会面临众多谜题，游戏采用经典JRPG式的回合制战斗，加入了施法、咏唱、属性相克、控制魔法、打断、减速等元素。游戏画面充满着奇幻色彩，部分场景融合了欧洲古典风，还有一些场景是水墨风格，虽然不华丽但是清雅耐看。游戏音效出色，地图中的各色机关和玩家进行的各项操作都有对应音效。
