

**Routine** 是挪威音乐制作人Alan Walker和西班牙音乐制作人David Whistle制作的一首纯音乐流行舞曲。该歌曲延续了Alan
Walker以往作品独有的强劲节奏感，空灵的人声与电音的碰撞结合，恍若幻境。时隐时现的女声，像是在呼唤着每一个人，指引着听者在黑暗森林中寻觅。

We Rave You说：“《 _Routine_
》利用声乐失真对比他的以前的工作，旋律存在仍然可以清楚地感受到。带着比他早期曲子更重的贝斯线，所创造的是既能令人平静又激励人的氛围。游戏的合成器选择显而易见，是Alan
Walker和David Whistle带给我们最好的圣诞礼物。”Ranjiv
Asher表示：“这首曲子相对于轻度合成器来说旋律是相当温柔的，并且具有很好的节奏感。”

歌曲MV由Simon Diaz
Compagnet制作，于2016年12月23日YouTube上发行。该视频描绘了一个球体不断旋转与扭曲，并以某种网络之间进行流通。
