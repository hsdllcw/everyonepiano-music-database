_Angel's Serenade_ 或Legende valaque，中文译为天使小夜曲，由意大利作曲家、大提琴演奏家Gaetano
Braga创作。此曲讲述了一位担忧的母亲与听到天使呼唤的女儿的对话，最后女孩追随了天使的声音（"O mamma buona notte, io seguo
il suon, io seguo il suon!"）。它在契诃夫的短篇小说《黑修士》，以及俄罗斯一些其他19世纪末20世纪初的作品中有所提及。
