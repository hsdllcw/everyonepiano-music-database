

**天净沙双手简谱** 和五线谱完全对应。

_天净沙_ 出自《古剑奇谭》配乐，骆集益作曲。骆集益，生日是12月6日，作曲家、音乐制作人，台湾缪斯特(Musit)音乐工作室成员。

《古剑奇谭：琴心剑魄今何在》是由上海烛龙信息科技有限公司开发的一款角色扮演类单机游戏，于2010年7月10日在中国大陆发售。为《古剑奇谭》系列的初代作品，其单机续作为《古剑奇谭二：永夜初晗凝碧天》。游戏采用了架空历史的设定，在其中融入了众多真实的人文地理风貌，描绘出了一个风格清丽、古韵浓郁的游戏世界。游戏中有独特的战斗系统、星蕴系统、成就系统、特色的辅助系统等。
