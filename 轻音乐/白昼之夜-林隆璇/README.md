_白昼之夜_
系台湾音乐人林隆璇所创作的一首纯音乐，收录于2007年发行的专辑《幸福之初》中。这张专辑是由金革唱片与白金录音室联手打造的新世纪精致音乐典藏范本。

林隆璇的音乐承载了生活的点滴记忆，用充满温度的旋律打造出一张属于自己的音乐理想专辑，是给所有暌违已久的歌迷一个最温暖最知性的幸福聆听。电影配乐式的深情感动、细腻美丽的浪漫编曲，尽显林隆璇在创作和制作两方面驾轻就熟的功力，是值得反复聆听与推敲的一张发烧专辑。

林隆璇，1964年7月9日出生于中国台湾，毕业于文化大学音乐学系，中国台湾男歌手、作曲家及制作人，超级偶像的固定评审之一。
