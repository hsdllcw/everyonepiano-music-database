

**远方的寂静**
是中国新世纪音乐作曲家林海创作的钢琴曲。林海先后出了十数张个人专辑，并给大量的电视剧、电影做原声音乐，如《大明宫词》、《一个陌生女人的来信》等，获得乐迷的喜爱以及专业人士的赞扬。

  
《 _远方的寂静_
》是【心灵钢琴】系列第二张。同样旨在安抚受伤的心灵，为在这个纷繁杂乱的红尘中的芸芸众生带来一片清凉。多功能，治疗性；静思，冥想，吃饭，喝茶，聊天，睡觉，恋爱均适合。最大的特点是整张专辑没有重音，不突兀。你不会在上述的行为中被打断。慢慢地好像在倾诉。舒服，恬静，流淌，自然。很超脱的感觉。风格偏流行，少商业。与《月光边境》相比，这张更成熟，更具有“心灵钢琴”的功能。
