保罗·莫里哀（1925.3.4—2006.11.3），法国轻音乐大师。其创建的保罗·莫里哀乐团与德国詹姆斯·拉斯特乐团、英国曼托瓦尼乐团并称为三大轻音乐乐团。  
  
莫里哀出生于马赛，4岁开始学习钢琴，10岁进入马赛音乐学院学习。二战期间开始登台指挥。1965年创办保罗·莫里哀乐团，1968年以一曲蓝色的爱（Love
is
Blue）一举闻名世界。莫里哀十分注重旋律与和声，其节奏清晰活泼，风格清新明朗，不夸张。他成功地将古典音乐与流行音乐相结合，创造出崭新的音乐体裁，因此被誉为“情调音乐的使者”、轻音乐教父。1998年，莫里哀宣布告别舞台，2006年11月3日逝世于法国南部佩皮尼昂。
