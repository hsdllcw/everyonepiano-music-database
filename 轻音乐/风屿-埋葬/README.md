_风屿_
是由埋葬谱曲，Dream~of~sha制谱的一首原创音乐。歌曲开头，简单的钢琴音，本应单键处理，但因音乐似乎在为后面的叙事有所铺垫。心情难以平静，心有所想，所以在钢琴重音处，采用多键位处理。

歌曲第1副歌：简单的钢琴键位，看似疯狂，但是却非常容易，单键处理，后加入多键组合（内心想法），纠结的心情似乎让人基本接近疯狂。

歌曲第2副歌：整个歌曲的高潮部分，乱键处理，4分组合键，16分单键，强有力的节奏和精准的抓音，以及饱和的排列，此段难度大幅度提升。

结尾，短暂而仓促，可能是不希望继续堕落下去，还是需要赶快醒悟，所以很短想重获新生。然而一切都无法控制，只能听天由命。简单的音符至结尾完！

埋葬，又名YJ，中国90后知名原创乐师，独立创作并发布多部音乐作品，在音游玩家中享有极高的声誉。
