

**Memory** -记忆是久石让为《入殓师》的配乐。

《入殓师》里久石让配的 _ **memory**_
这段旋律分3个部分出现在剧中。)当时男主角的老婆知道他的职业后离他而去，希望他放弃他的职业（入殓师），但他开始逐渐爱上这份工作，一次次为各种各样的人进行纳棺仪式，虽然都是死亡，但死者们的亲友表现出来的感情却非常不同，德高望重的父亲死去，他体会到女儿们对父亲的爱，慈祥的奶奶去世，他体会到子孙们对奶奶的爱，年幼的男童死去，他体会到长辈对晚辈的爱。。。。。。

  
因为男主角童年特殊的经历（父亲和店里的女招待私奔，自己又离开母亲到外国求学），因此他没有机会对自己至亲、至爱的人做最后的道别，当时他母亲已经过世（他不在身边），父亲还下落不明。  
男主角在绿色的海边原野背景下，感受入殓师这份工作带给他情感上的冲击，也使他回忆起自己与家人的点点滴滴。尤其是他的父亲，小时候给过他“石头信”的父亲……
