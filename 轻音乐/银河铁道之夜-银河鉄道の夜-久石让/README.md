《 _银河铁道之夜_
》是日本著名诗人，儿童文学家宫泽贤治的代表作品。1934年被列入文图堂出版的《宫泽贤治作品集》中第三卷正式发表。此小说共4个版本，以最后一个为人所知。其实此小说很大部分仍未完成，全书仍在草稿阶段，但因充满唯美幻想色彩的独特描写，故仍受到许多人喜爱。该小说与1985年被改编成同名动画。同时，长久以来也有许多艺术家根据这部小说创作过很多作品。

1996年，为纪念宫泽贤治诞辰100年，著名作曲家久石让根据原著，创作了这张印象专辑。
