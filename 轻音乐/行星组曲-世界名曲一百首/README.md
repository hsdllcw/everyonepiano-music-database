

**行星组曲双手简谱** 和五线谱完全对应。

作为世界名曲一百首的 _行星组曲_ 是一部由英国作曲家霍尔斯特创作的管弦乐组曲。

完成于1914年到1916年之间，分为七个乐章，分别以九大行星中的七个星球（地球和当时不为人们所知的冥王星除外）命名，乐队编制也非常庞大，启用了一般很少登台的低音长笛、低音双簧管、低音单簧管、低音大管、次中音大号等管乐器，以及管风琴和众多的打击乐器，最后一个乐章中还有一段六声部的女声合唱（有时以两支独奏长笛取代）。如此众多乐器的组合产生了丰富的音响色彩，如在“火星”乐章的一段音乐中，乐队的全奏展示出了地动山摇的气势。但正是由于庞大，这部作品一般很少全曲演奏，通常仅演其中的三、五个乐章，有时则只是单独演奏一个乐章。
