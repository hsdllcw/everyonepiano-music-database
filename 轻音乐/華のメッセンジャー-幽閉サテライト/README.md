_華のメッセンジャー_ 是幽閉サテライト的曲子。

幽闭サテライト，中文幽闭星光，是一个东方Project同人音乐创作团体，乐团的主唱为senya（即森永真由美）。senya声线稳定且老成，毫无新人歌手的不适应。编曲也是非常的成熟，纵使还有其不够完善之处，但是幽闭サテライト这个音乐团体已经是东方project的新得力助手。

风格多为Classical电子迷幻乐&DJ舞曲&J-pop&电子金属核等（含有少量和风），以具有特色的东方系列的电子音乐塑造了一个全新的幽闭サテライト。
