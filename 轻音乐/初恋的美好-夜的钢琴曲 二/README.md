_初恋的美好_ 是石进2012钢琴曲集《夜的钢琴曲 2》中的一首曲子。

人的一生，最刻骨铭心的，就是初恋。不管它最后的结果是苦还是甜、但你在享受初恋的同时，也先享受到了你人生的第一次真正的甜蜜，这就是初恋。每个人的初恋都是不一样的。尽管大多都是辛酸而告终，但其中却在我们的不经意间得到了许多甜蜜。

《夜的钢琴曲》系列中的每一首曲子都静谧舒缓，让彼此心灵诗意地栖息。在这里，每个人总会找到一首属于自己情感的乐曲……
