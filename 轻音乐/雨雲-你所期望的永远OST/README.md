

《 **雨雲**
》由日本音乐人渡来亚人创作。同时，此曲也是日本游戏公司AGE于2001年8月3日推出的18禁电脑多重恋爱冒险游戏《你所期望的永远》的原声音乐。

《你所期望的永远》（君が望む永遠），简称君望（君のぞ）。在日本推出后大受欢迎，亦是此公司的成名作。期后于2002年推出Dreamcast版及于2003年推出PS2版本。2003年7月25日推出DVD-
ROM。2004年6月25日推出特别版《你所期望的永远~Special Fandisk~》。2008年3月28日推出重制版《你所期望的永远~Latest
Edition~》。2003年推出14集动画版《你所期望的永远》，删除了大部分18禁成份，动画播出后掀起另一论热潮。
