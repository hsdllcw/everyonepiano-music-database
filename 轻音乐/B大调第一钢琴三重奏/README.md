_B大调第一钢琴三重奏_ 是勃拉姆斯于1854年早期完成的第一钢琴三重奏。勃拉姆斯出生于汉堡，逝于维也纳，德国古典主义最后的作曲家，浪漫主义中期作曲家。

B大调三重奏原版总长为1，628小节，是勃拉姆斯篇幅最长的器乐作品。1889年修订作品时，作曲家把他最钟爱的自然音阶曲调之一的主要主题原封不动地搬了过来。钢琴引出第二主题，但很快就扩展到弦乐，形成一个高潮，而后音乐逐渐转入主要主题的再现部。勃拉姆斯选择钢琴上的三连音型使结尾更迷人，最后以作曲家钟爱的变格收束终止。

原作速度为流畅的快板，改编版为灿烂的快板。这一乐章的第一主题贝克形容为“彩虹飞架、彩蝶飞舞、夜莺伴奏”，原作在发展部中反复使用赋格手法，修改版改对位法替代赋格。
