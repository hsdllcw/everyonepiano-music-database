_Victory_ 是克罗地亚钢琴演奏者马克西姆的演奏曲目。马克西姆•姆尔维察（Maksim
Mrvica），1975年出生于克罗地亚希贝尼克。1984年，开始学习钢琴。1999年，赢得鲁宾斯坦钢琴大赛冠军。2000年，转到布达佩斯的李斯特音乐学院深造。2016年，为中国乐迷带来总共26场音乐会，刷新了在华巡演的年度场次记录。

他的音乐以特立独行的姿态颠覆了人们对音乐的所有想象。音乐上，打破传统严格的定律，古典曲目与现代元素结合，魔幻色彩四溢。衣着上，舍弃传统的燕尾服和西装领带，换上一身休闲的狂野劲装，十足一个流行偶像歌手，酷味十足。他从来不被框框条条束缚，他自立标杆，引领潮流。他不是在做音乐，而是在玩音乐，享受音乐。
