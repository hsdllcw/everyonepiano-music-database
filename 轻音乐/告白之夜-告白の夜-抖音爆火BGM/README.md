

**告白之夜** ，日文为告白の夜，是由日本小提琴手AYASA（岛村绚莎）演奏的歌曲，收录在专辑CHRONICLE V。

这首曲子取材于《竹取物语》。以辉夜姬向养育自己的双亲说明自己是月亮上的人，并且马上要回月亮的场景作为模型，写下了这首曲子。作曲是覆面组合“nowisee”。在日本，这首曲子作为TBS的赤坂サカス冬天的主题曲，在演奏会上也成为了人气歌曲。

《 _告白之夜_
》这段音乐一共有4分28秒。而抖音上大家耳熟能详的片段是截取自1分38秒到1分55秒左右。当《告白之夜》在抖音上走红后，网友们对这段音乐的理解也有了不同看法，从原曲的亲情转化为个人爱情：“1分40秒前是向他告白的欣喜，1分40秒是告白失败的倾诉，3分10秒已成为过往，4分15秒是对往事的释然。”
