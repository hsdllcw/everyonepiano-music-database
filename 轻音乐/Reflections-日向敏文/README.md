《 **Reflections** 》是一首纯音乐，由日本作曲家日向敏文（ひなた としふみ）创作。

听此曲有一种淡淡的忧伤，同时脑海中还会浮现类似于这样的画面：放一只玫瑰在墙角，一个人依靠在伦敦大街昏暗灯光下，望向彩色玻璃窗下的她。华尔兹响起，梦里是雨中轻舞的她，而我只有颓废和无尽的伤感。

日向敏文，日本作曲家，1973
高中毕业后到英国，翌年回国。1975年9月，进入美国Wisconsin州Ashland的Northland大学就读，1977年8月，转学到伯克利波士顿音乐学院钢琴科，1979年9月，
获得Minnesota州立大学外国奖学金，并转学到此校，以后四年之间专攻classic piano，修习理论、作曲和编曲，并且兼任讲师(part
time)。
