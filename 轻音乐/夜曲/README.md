

1995年，神秘园凭借一首仅有24个单词的单曲"夜曲"一举获得在欧洲拥有最多观众的欧洲电视歌曲大赛的金奖，创下了40年来这个具有极高声望的大赛从没有一部器乐作品获金奖的奇迹。词作者为Peter
Skavlan。演唱者Gunnhild Tvinnereim。2007年11月26日，神秘园最新专辑Inside I'm
Singing发行，nocturne被Anne Takle重新演绎。  
词作者为Peter Skavlan。
