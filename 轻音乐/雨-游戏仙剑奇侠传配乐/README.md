_雨_ 是一首轻音乐，由音乐创作及编曲奇才林坤信创作。同时作为角色扮演电脑游戏《仙剑奇侠传》的配乐，深受大家喜爱。

《仙剑奇侠传》是《仙剑奇侠传》系列的第一部作品，主题是“宿命”，于1995年7月发行。游戏讲述了平凡的客栈小伙计李逍遥因为机缘巧合结识了女娲族后裔赵灵儿。赵灵儿所居之仙灵岛意外遭人袭击，李逍遥遂担任起护送灵儿往苗疆寻母的重任。在旅程中，李逍遥先后结识了林月如、阿奴。在赵灵儿帮助苗人祈雨解旱、惩奸除恶的过程之中，赵灵儿的身世之谜以及她母亲的下落，也终究水落石出。此时，拜月教主的阴谋摆在众人的眼前，李逍遥、赵灵儿等人却不知这与拜月教主一战，将会铸成一段永恒的悲剧。
