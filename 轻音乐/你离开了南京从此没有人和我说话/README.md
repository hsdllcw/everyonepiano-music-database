偶然间，一段旋律闯入了我的耳朵。是的，虽无歌词，但整个人却深陷其中。前奏一响就已热泪盈眶，不是矫情，而是因为单曲循环了太多回忆。

_你离开了南京，从此没有人和我说话_ 。

短短一句话，是歌曲名，也像是道尽了曲作者的内心情感。偌大的南京，人来人往，熙熙攘攘，而我只是我，或者也不再是我。

李志的这首《你离开了南京，从此没有人和我说话》真正意义上并算不上一首完整的歌，全长一分多一点，没有一句歌词，评论却有上万。

其中有这样一句评论吸引了我的注意，“人海里的你我，分分合合，像一对摇去天涯的桨。”这让我想起了顾城的一句诗“我们告别了两年，告别的结果，总是相见。今夜，你真要走了，真的走了，不是再见”。这是多么大的孤独感？
