

《 **夜空的彼岸** 》（夜空ノムコウ）是日本钢琴家加羽沢美浓创作的钢琴曲。日本组合SMAP北京演唱会里唱过现场中文版的。

加羽泽美浓，生于日本横须贺市。3岁起习弹钢琴。在雅马哈原创音乐的组织里，初显才华，被誉为天才少女。她于1998年从东京艺术大学作曲系毕业，在校就读时就已经在哥伦比亚唱片公司旗下发表了个人首张专辑。目前硕果累累。

加羽泽美浓曾与铃木光司合作发行的《乐园》专辑使其人气急剧攀升，之后又与小提琴家高嶋ちさ子合作从事音乐活动，至今仍然十分活跃。

加羽泽美浓于1997年11月发行了自己的首张钢琴曲集《圣诞旋律》，这张专辑收录了古今圣诞名曲17首，加羽泽用自己的钢琴独奏赋予了它们新的生命。从那以后，她坚持将日本当年度最受欢迎的流行歌曲重新改变成钢琴独奏曲，
1997——2003年发行了系列专辑《年度回忆》。除此之外，1999年她还发行了非常富有空间和情境想象力的专辑《温煦的风~常见的风景》，2000年又发行了《钢琴·古典·狂想》，将《阿兰费斯协奏曲》、《第九交响曲》等古典名曲进行了重新改编，同时也收录了她的自创曲目《乐园》等。这一时期，加羽泽从循规蹈矩的学习向追求独特风格过渡的轨迹十分明显，并终于在2000年进入了映像音乐的领域，发行了NHK电视动画片《熊的儿子伍夫》的原声盘。此后，她一面与小提琴演奏家高岛千纱子结成组合，又与田部京子、加藤知子、须川展也、福田进一、古川展生、宫本文昭以室内乐组合的形式出版了可以称为其代表作的几张专辑，继续进行从古典音乐到自创曲目的内容广阔的音乐探索，一面作为日本顶级钢琴家深入到影视音乐领域之中，富士电视台电视剧《整形美人》、电影《七夕之夏》的音乐奠定了她作为影视音乐创作、演奏者的地位。2005年6月1日，她还将发行第四张影视音乐原声盘——电影《四天里的奇迹》原声音乐。
