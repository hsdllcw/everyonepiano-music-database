

《 **月光边境**
》是【心灵钢琴】系列第一张。旨在安抚受伤的心灵，为在这个纷繁杂乱的红尘中的芸芸众生带来一片清凉。多功能，治疗性；静思，冥想，吃饭，喝茶，聊天，睡觉，恋爱均适合。最大的特点是整张专辑没有重音，不突兀。你不会在上述的行为中被打断。好像在
慢慢地倾诉。舒服，恬静，流淌，自然。很超脱的感觉。风格偏流行，商业。
