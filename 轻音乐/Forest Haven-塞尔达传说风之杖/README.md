_Forest Haven_ 是动作冒险游戏塞尔达传说：风之杖的背景音乐。

塞尔达传说：风之杖（英语：The Legend of Zelda: The Wind Waker；日版被称为《塞尔达传说 風之杖 》 日语：ゼルダの伝説
風のタクト）是一款由任天堂情報開發本部开发并由任天堂发行在任天堂GameCube平台上的动作冒险游戏。本作是塞尔达传说系列的第十部作品；于2002年12月13日在日本率先发售，2003年3月24日在北美发售，2003年5月2日在欧洲发售。任天堂DS平台上的《薩爾達傳說：夢幻沙漏》被视为其正统续作。

音乐由永田権太、若井淑、峰岸透和近藤浩治担当制作。游戏的原聲大碟原声大碟于2003年3月19日以双碟的形式发售，包含了游戏中133首音乐。本作的音乐受到爱尔兰风格的影响，一些乐曲中采用了爱尔兰风笛（Uilleann
Pipes）来演奏。
