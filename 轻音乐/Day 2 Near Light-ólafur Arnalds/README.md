_Day 2 Near Light_ 是冰岛音乐制作人ólafur Arnalds的纯音乐曲目，收录在《Living Room Songs》专辑中。

Olafur Arnalds（奥拉佛 阿纳尔德斯）：一位才华洋溢的冰岛音乐创作人, 而且很年轻, 音乐以钢琴和弦乐为主, 与其说是post-rock,
不如乾脆说是modern classical。 专辑听来犹如电影配乐, 既优美又带些悲伤, 清澈细致, 牵动著听者的情绪, 让人沉醉。

2009年4月，Ólafur Arnalds曾每天作一首曲子，做好就以网站公布下载，连续七天，集结成「Found
Song」专辑。2011年10月，他再进行一次类似的方式，在他雷克雅维克家里的客厅录制新曲，一天一首，将现场影片及mp3放到livingroomsongs.olafurarnalds.com这个网页上，乐迷可以看到这些演奏录制实况，免费下载128kbs的mp3。并由英国实验电音厂牌Erased
Tapes Records发行这张唱片「Living Room Songs」。
