

《 **美景** 》是周杰伦自导自演的第二部电影作品《天台爱情》中的音乐。

影片是根据周杰伦自己编写的小说《天台》来拍摄的电影。早在拍《青蜂侠》时，周杰伦已经构思好《天台爱情》的剧本：“我连每一句对白都写好了。这6年里我一直都有在写剧本，《天台爱情》只是其中一个，对我来说，《天台爱情》只是正好适合在这个时机推出的一个。”

《天台爱情》的故事架构建立在一个虚拟的时空，其影片风格近似于20世纪70、80年代。《天台爱情》投资超过两亿新台币，电影题材丰富多彩，融合爱情、动感、歌舞、武打、青春、励志、热血、浪漫、喜剧、古典、华丽、特效等沙拉元素，才被宣传方称之为“一部前所未闻的电影类型”。

《天台爱情》的影迷都兴奋的表示，《天台爱情》很好看，而片中的音乐和歌舞更是大亮点，丝毫没有影响剧情的连贯性。当然，《天台爱情》也没有完全走纯歌舞的路线，影迷计算了下，片中大约11支歌舞，占总片长的五分之二。至于影片色彩艳丽，周杰伦坦言是受了张艺谋的影响。

截至21日《天台爱情》内地票房1.07亿元，超越《悲惨世界》（6213万元）成为中国市场首部票房破亿的歌舞片。
