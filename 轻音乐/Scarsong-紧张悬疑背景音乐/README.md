_Scarsong_
是一首紧张悬疑背景音乐。这首歌曲相信很多人都曾在无意间听过，但是却不容易找到这首歌，更多的是在看视频或者听电台中无意听到的，也许会觉得十分应景，但是却不大在意，然而也是久而久之，人越来觉得这首曲子不错，就会去特意去关注、在意，但是却没有太多头绪，所以也不是很容易找到，直到最后找到了，也许会发现曲子真的不错，的确值得收藏。

也许正是因为此曲的作者扑朔迷离，在网上关于这首曲子的争议也不间断地发生，此曲没有明确的创作时间、没有明确的作者信息，甚至于在各大浏览器上都是以模糊的答案作出解释，也许正因为这些，这首诡异的曲子从何而来，是否是由《月光心慌慌》改编而来，这些不确定的答案，都为它增添了一层神秘感。
