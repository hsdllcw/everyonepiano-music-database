_点星（Point The Star）_
是陆地钢琴师MT1990的曲子。看到陆地钢琴师MT1990这个名字，很容易让人联想到电影《海上钢琴师》，其中男主角的名字叫做1900，从小生活在邮轮上。他慢慢地长大，慢慢显示出了无师自通的非凡钢琴天赋，在船上的乐队演奏钢琴，每个聆听过他演奏的乘客，都被他的音乐深深打动。影中那位忧郁而迷人的即兴钢琴师将黑白键演绎得出神入化。我想MT1990也是钟情于1900的吧，他们的区别是一个在海上、一个在陆地，但却都极富才华，写给每一个孤单且勇敢的人的背景音乐，都像晚风拂过静谧的森林，治愈和温暖着晚睡的人们。
