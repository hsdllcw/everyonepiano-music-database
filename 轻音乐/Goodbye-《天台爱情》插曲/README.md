《天台爱情》的社会背景发生在繁华的加利利市。周杰伦在剧中饰演一名医馆穷小子，名叫浪子膏。电影描述着两种人生，一种是住在天台上的人，他们可以偏安一隅，天天载歌载舞，过着无忧无虑的日子；而住在天台下的人，却是比较有钱、有势、有权，影片将呈现这两种反差的生活。电影的整体感觉走比较古典的路线，其影片风格近似于20世纪70、80年代。这部电影主要以爱情为发展主轴，并且冲突的融合了歌舞、武打、动感、青春、励志、喜剧、热血、浪漫、华丽、特效等视听元素，不单是一种类型的电影所能框架，所以此片才被称之为「一部前所未见的电影类型」，周杰伦这次不仅自编、自导、自演，而且还为电影操刀配乐、甚至连指导武打动作设计周杰伦都参与想法、担任剪辑。
