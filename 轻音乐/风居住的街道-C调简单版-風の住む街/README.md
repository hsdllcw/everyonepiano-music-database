

**风居住的街道** ，日文風の住む街，是一首由矶村由纪子与日本著名二胡演奏家坂下正夫合作的经典曲目，是一首中国风的轻音乐。

由钢琴二胡对话演奏，曲调缠绵不尽，抑郁情深，令人听过后也觉思绪绵绵，悲人心怀。矶村由纪子将钢琴的深沉融入女性特有的细腻，在二胡的泛音延留中，或沉寂，或作轻微的回应，清澈，温暖，淡定。而坂下正夫的二胡，亦有着优雅而舒缓的气质，那是一缕相思，一腔柔情，一抹淡淡弥漫着的忧伤，更是一种直抵人心的美丽。

爱情之花是绚烂夺目的，然而在 _风居住的街道_
中里的这对恋人，还未正式开始就要结束，让听众扼腕叹息，在岁月的流逝中，年华老去。然而那炽热的爱情，却还依旧静静等待那人的归来，曲中的恋人在余生中却再没有得到对方的回音。整首曲子，动与静，温婉与低沉，交相错出，就像两个恋人，相互爱慕，相互惦记，却是一个江南，一个塞北。山水万重，烟雨千声，只能隔着岁月的风尘，遥遥地相望。几许情深，几许惆怅，不思量，自难忘。
