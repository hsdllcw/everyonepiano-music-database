

班得瑞（Bandari）音乐是瑞士音乐公司Audio Video Communications
AG旗下的一个新纪元音乐团体。其作品以环境音乐为主，亦有一些改编自欧美乡村音乐的乐曲，另外还有相当数量的 班得瑞是重新演奏一些成名曲目。  
  
1990年发迹瑞士的班得瑞，来自一群爱好生命的年轻作曲家、演奏家及音源采样工程师等等青年才俊。团长奥利弗·史瓦兹是一位多才多艺的音乐人，不但专长多种乐器，更熟稔先进的录音工程。他的特别率先采用先进的数字采样技术，将自然界的声响，运用后期母带24bit数字录制，巨细靡遗地记录下来音质无比清晰。奥利弗·史瓦兹更曾与为莎拉·布莱曼制作“Time
to Say Good-bye”的天王制作人亚历士克·里斯坦森合作，但挚爱新世纪音乐的他，最终选择将一身才华奉献给瑞士山林！  
  
班得瑞最独特之处莫过于每当执行音乐制作时，从头到尾都深居在阿尔卑斯山林中，坚持不掺杂一丝毫的人工混音，直到母带完成！置身在欧洲山野中，让班得瑞拥有源源不绝的创作灵感，也找寻到自然脱俗的音质。每一声虫声、鸟鸣、花落流水，都是深入山林、湖泊，走访瑞士的阿尔卑斯山，罗春湖畔、玫瑰峰山麓，少女峰等处实地纪录。  
  
安妮的仙境是班德瑞第一张新世纪专辑《仙境》（Wonderland）曲目之一 。

### 班得瑞《仙境》

  
{之一}
小小的礼物，足以预支孩子一年份的笑容；他的心，小得像芝麻一样，毫不费力就能将它填满；但是，若把你那已经长大的心送给他，也会变得连放都放不进去。孩子要的，除了你的呵护之外，有时是希望你能陪他一起当小孩。  
  
{之二}
枕边心事，是张仙境的蓝图；所有蛰伏、沉淀的狂想，化作柔软的草坡，任你在上头恣意翻滚。在那里，云是甜的，海是透明的，就连花都长齐了翅膀。你可以随兴邀请每一个访客，也可以独享这一片光鲜绿野。谁说天堂一定在远方？  
  
安妮的仙境总是给人莫名的感动,以清爽舒适的音调生动的诠释了自然的魅力与天地的风采。
