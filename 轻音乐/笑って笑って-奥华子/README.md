

**笑って笑って**
是日本歌手奥华子所演唱的歌曲。作为地下音乐人的她，用键盘自弹自唱在涩谷街头表演的奥华子有着1年内自主制作CD销售2万张的成绩。之后被PONY
CANYON相中，于2005年7月以单曲「やさしい花」进入主流音乐界，此曲因优美的旋律搭配东京电力「TEPCOひかり」广告歌曲而广为传播。

  
歌词下方是 _笑って笑って钢琴谱_ ，希望大家喜欢。
