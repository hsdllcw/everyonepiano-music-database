_藤崎诗织的主题_
是日本科乐美公司出品恋爱游戏《心跳回忆》中的音乐。该游戏首发于1994年5月27日，是全球第一套全年龄取向的恋爱养成游戏系列；在这一系列的恋爱游戏中，完全没有限制级的场景。内容主要是描述一位中学生在毕业前与诸位女同学交往的游戏。本作也在日本也曾被改编拍摄成同名电影。

游戏相当注重音乐与音效的表现，所有的配乐都是在专业的录音室中录制完成，而且邀集了许多日本当地著名的乐团。游戏中的每一个角色都有专人配音，在制作的过程中，配音人员并不只是捧着台词念稿，而是会配合游戏中人物表情的变化而有抑扬顿挫。玩者在进行游戏的同时，便能真切地感受到人物的情绪与语气的变化。
