_明治十七年の上海アリス_ ，中文《明治十七年的上海爱丽丝》，是《东方红魔乡》3面BOSS红美铃的主题曲。

《东方红魔乡 ～ the Embodiment of Scarlet
Devil.》是由同人社团上海爱丽丝幻乐团所制作的纵弹幕射击游戏，是东方Project的第6作，也是东方Project在Windows平台上的第一作。本作是在相隔PC-98推出该平台最后的作品“东方怪绮谈
～ Mystic Square.”的三年后开发的。在2002年6月10日发布体验版，并在2002年8月11日（Comic Market
62）正式发售，最后在同年9月27日开始在各同人商店开始了委托贩卖。当时在商店售卖的时候，漫画虎之穴店中有本游戏的示范。
