

**Limpid Luster双手简谱** 和五线谱完全对应。

_Limpid Luster_ 由小池雅人创作，是动作游戏《无双大蛇》（英文版名：Warriors
Orochi）的插曲。该游戏的特色将会是真•三国无双系列及战国无双系列角色首次正式在无双系列融合，此外，曾在《战国无双2》没有登场的角色亦会在该作品登场。本作可以允许在同一关卡中操控3个武将，只要在战斗中途按一个按钮就可以转换人物。故事讲述魔王远吕智尝试寻找世界真正的强者，于是将时空扭曲，将中国三国时代及日本战国时代的人物困在一个异世界共同登场。而各个势力有不同的难处，有些暂时加入远吕智军，希望寻找失踪的主君。有些则直接与远吕智军对抗，称为反乱军。
