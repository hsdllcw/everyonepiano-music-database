

**Raindrops双手简谱** 和五线谱完全对应，属于EOP教学曲。  
  
《Raindrops》（雨滴）是韩国钢琴家The Rain创作的曲子。收录于专辑《Misty Rain》（雨的秘密）中。  
  
Rain的独白，透露出对爱的渴望，也感动了有过相同渴望的乐迷。  
  
韩国新世纪音乐中，一个令人无法忽视的超龄天才，自从第二首Misty
rain被选为韩国SBS热门连续剧『雪花』的插曲之後，Rain终於推出个人首张钢琴演奏专辑《雨的秘密》。首张专辑作曲、编曲、钢琴演奏与制作全都他一人包办，19岁的Rain着实让人讶异他的超龄表现。  
  
《Misty
Rain》（雨的秘密），就像是一个较之同年人更具敏锐感受、更细腻的少年在静静吟唱的专辑。整张专辑表现的是Rain在日常生活中，无限透明的秘密日记。在Rain的音乐中，纯粹的喜悦、哀伤相生并存，写的是自己的情感，也道尽了都市人渴望被爱与爱人的内在渴求。  
  
其中收录曲《Raindrops》（雨滴），是一首节奏轻快，有着淡淡感觉的曲子。下面是 _Raindrops钢琴谱_ ，希望大家喜欢。
