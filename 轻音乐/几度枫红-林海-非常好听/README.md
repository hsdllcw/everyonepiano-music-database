

《 **几度枫红** 》选自林海的专集《爱情风华》。

钢琴才子林海在12月推出的最新钢琴大碟【爱情风华】，突破创作格局挑战连五年入围纪录，新专辑中以40人的管弦乐编制搭衬钢琴演奏，刻划11首动人的爱情故事，浩大的弦乐音色引导情绪澎湃，关于爱情的回忆就这么流泄而出，这次林海尝试以管弦乐搭配钢琴，期望黑键白键更加动人。

林海1969年出生于福建晋江，四岁半开始学钢琴，1988年进入大陆中央音乐学院，主修钢琴；隔年从全国竞赛脱颖而出，代表中国参加美国第八届VAN
CLIBURN大赛，只练了一个月即入围准决赛，成为参加此项大赛唯一曾入围准决赛的中国人。1992年毕业后，开始担任多出电影、电视配乐及两岸各地的现场表演，专心于爵士音乐及新音乐创作，是当前深受两岸瞩目的青年钢琴家。
