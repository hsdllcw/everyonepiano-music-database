“不知不觉，寒露已过，霜降将至，金黄的秋天即将过去，恰此刻起舞一曲，叶落满地，尘埃落定，盼来年花期。  
《秋逝》这首曲子节奏欢快，风格与大部分夜的钢琴曲作品不同，小提琴和钢琴完美配合，透露着享受当下的心情与未来的期许。UP把这首曲子改编为钢琴独奏，其中一段小提琴solo改编的轮指演奏为最大难点。”

《秋逝》出自国内钢琴音乐创作演奏爱好者石进钢琴曲集《夜的钢琴曲Ⅲ》。此曲欢快中却带着丝丝忧伤，你听出来了吗？

一切的等待，都是值得的。时隔三年，《夜的钢琴曲》系列终于在2015年8月迎来了第三张原创音乐作品。这张专辑，收录了三年来陆续创作的一些旋律，并融入了小提琴、大提琴、长笛、人声等不同的乐器来展现《夜的钢琴曲》系列的美丽与柔情
