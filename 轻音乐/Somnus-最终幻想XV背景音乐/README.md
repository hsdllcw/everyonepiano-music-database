_Somnus_ 是游戏最终幻想XV中的音乐。此曲由日本电子游戏作曲家下村阳子创作。下村阳子被誉为“世界上最著名的女电子游戏作曲家”。

最终幻想XV（Final Fantasy
XV，简称：FFXV、FF15）是由史克威尔艾尼克斯开发制作的动作角色扮演类游戏，于2015年3月17日发行了试玩版。该作品支持PS4和Xbox
One，是《最终幻想》系列、《FABULA NOVA CRYSTALLIS FINAL
FANTASY》系列之一。游戏以系列共通的新水晶神话为基础，同时展开独立的世界观、使用不同的视觉设计以及角色。游戏主人公诺克提斯•路西斯•凯拉姆是路西斯王国的王子，这个王族保护世界仅存的一颗水晶，并支配一个技术先进的城邦。
