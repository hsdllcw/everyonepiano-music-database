

**Let It Be** 是法国钢琴演奏家理查德克莱德曼的钢琴曲。理查德•克莱德曼（Richard Clayderman，原名：Philippe
Pagès ），被称为 “浪漫王子”，却不是一个专门录制专辑的艺术家。

  
理查德•克莱德曼是一个相当受欢迎的艺术家。克莱德曼的音乐的特殊的魅力，以及深入人心的特质，使得他成为了大众所喜爱的音乐家。

  
理查德•克莱德曼在第三世界国家有着很大的市场。这一点使得批评的人们指责他的作品主要围绕着一些国家的当地作品，或者流行作品；还有接近当地消费水平的定价。尽管如此，多数人还是认为他是一位很伟大的钢琴演奏家。
