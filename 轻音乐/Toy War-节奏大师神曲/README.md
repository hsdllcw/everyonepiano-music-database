

**Toy War** ，中文译为玩具战争，是游戏《节奏大师》里挺难的一首曲子。原由音乐节奏类游戏领域杰出的音乐创作人申东辉（M2U）创作。

《节奏大师》是由光速工作室开发的一款下落式触摸音乐节奏类益智游戏，于2012年5月25日发行。玩家在游戏中可以伴随音乐节奏敲击下落的音符，或通过滑动的方式连击连续的旋律音符以获得畅快的游戏体验，游戏过程中，不仅有音乐带来的听觉享受，更有“演奏”音乐带来的爽快感。
