

**夜的钢琴曲 22** 是作曲家石进所创作的夜的钢琴曲系列中的一首曲子。

石进来自柳州，在南宁工作，从2005年开始创作至今，他已经写了50余首钢琴曲和多首通俗歌曲，大学学的是计算机信息管理专业，现是区内一家国有网络企业的工程师。写钢琴曲对他来说只是业余爱好，“当初根本没想到会有这么多人喜欢”。

石进写的钢琴曲不是多么精深、高妙，演奏起来也不需要什么高难度的技巧，但却打动了包括冯导在内的众多听众，南宁钢琴家陈斯凡也很喜欢“夜五”，他觉得石进的作品非常真诚。对于“夜五”，石进坦言背后没有太多复杂的故事，“录音是即兴的，弹了三遍，"夜五"花了40分钟就完成了，只是记录爱与被爱的体会。”
