

**Man At Arms** ，别名《重骑兵》，由 Position Music 创作。

这首曲子跟其他曲子不同的地方：特别有一种亦步亦趋，一往无前的感觉。就像一个石像巨人，迈着沉重的步伐一步一步往前，不快，不慢，向着目标前进，什么也阻止不了，不管是雷电还是风雨，亦或是地裂天崩。也有一种马上要做什么伟大的振奋人心的事业一样，那种马上就要到来的感觉。

Position Music是家音乐公司，神曲众多，但是其独立的Trailer
Music系列质量上乘，而且多是录音室作品，由多位作曲家操刀的专辑风格也各有千秋，绝对是引人入胜的神砖。
