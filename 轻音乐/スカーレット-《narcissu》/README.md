

**スカーレット双手简谱** 和五线谱完全对应。  
  
《スカーレット》（Scarlet）出自游戏《narcissu》（ナルキッソス）中。所属专辑：《narcissu SIDE 2nd》。  
  
《narcissu》（ナルキッソス）是日本同人社团ステージなな（Stage
Nana）制作的一款免费同人游戏软件。日文版制作完成后不久，亦由各地的同好分别制作英文版、中文版、韩文版、法文版及越南文版游戏，同样是供网上下载的免费软件。于2009年4月27日发布了系列第三作《narcissu
3rd -Die Dritte Welt-》（ナルキッソス３rd ダイ ドリッテ
ヴェルト），首次以DVD形式发表，并同时收录narcissu和narcissu SIDE 2nd。与前两作不同，不是免费软件。  
  
《narcissu》是一款在网上公开发布的同人电子小说。由于软件是免费下载，并且在不改变内容及非商业用的条件下可任意转载，因此游戏在各下载网站迅速地广泛流传。  
  
值得一提的是，Narcissu中的音乐可以说在整部作品中撑起了半边天，是铸就这款经典之作的重要因素。音乐与游戏可以说是融合得完美，即更能投入到剧情当中。就算将音乐单独拿出来听，也是轻音乐中的精品。《スカーレット》就是其中一首很美的音乐。
