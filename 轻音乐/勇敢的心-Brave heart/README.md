

说起《 **Brave Heart** 》的原声音乐，就不得不说说曲作者James Horner这位大名鼎鼎的好莱坞电影配乐家。

国人熟悉Horner可能是从《Titanic》开始，那一首《My heart will go
on》和一支悠扬的爱尔兰风笛不知迷倒过多少人。其实在这之前，Horner就为多部优秀电影写了出色的配乐，其中有《Apollo
13》、《Glory》、《Legends of the Fall》、《The Mask of Zorro》。值得一提的是，当年《 **Brave
Heart** 》和《Apollo 13》同时获得奥斯卡最佳配乐的提名，虽然显示了Horner非凡的实力，但是造成了评委会选票的分流，留下了遗憾。
