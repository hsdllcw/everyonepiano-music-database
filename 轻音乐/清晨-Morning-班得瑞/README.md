《 _清晨_
》此曲开始从一阵鸟唱声中导引出钢琴混和竖琴的音色，舒缓地展现出一幅清晨宁静而充满生机的清馨画卷，粒粒水晶般的音符，让人似乎嗅到了那清新的空气，感受到了那缕缕的曙光，还有那涓涓的流水，清脆的鸟啼，缭绕的迷雾，习习的晨风.....
