

**古城之涟漪**
是由圣田康子创作的。圣田康子从事过现场演出，编曲，背景音乐制作等很多工作。2000年3月开始潜心于进行长年梦想的创作活动，直至今日。她的作品收录于和平之月系列,它是由日本一家名为Pacific
Moon公司录制发行，这是一家专门制作 New Age 和 World Music
的亚洲著名音乐公司，旗下精英荟萃[邵容、贾鹏芳、城之内美莎、菅井爱里等担当了大部分的作曲及演奏，融合民族和世界、东方乐器和西方配乐的典型效果创造属于
Pacific Moon 风格的作品，公司99%以上的专辑都用一个汉字来作为专辑的名称，显示了专辑本身的特点和音乐人的风格。

_涟漪_ ，一首唯美伤感的钢琴曲，曲调柔美，真如雨打湖面的 _涟漪_ 一样，让人有一种抛却烦恼,身临世外桃源之感。
