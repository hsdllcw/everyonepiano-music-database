_Outset Island_
是动作RPG游戏塞尔达传说风之杖中的音乐。本作延续着《时之笛》及其续作《穆修拉的假面》的游戏风格，也保留了这两个任天堂64游戏的基本玩法和操控系统。本作中的重要道具是可以使用并操控风来帮助帆船漂在空中飞行前进的风之杖。纵然本作公布之初宣布采用卡通渲染的画面风格及正太版林克的角色造型遭到了一些质疑，不过这并不能影响《塞尔达传说风之杖》成为任天堂GameCube上最受欢迎的游戏之一。
