_Butterfly_
是雷亚游戏开发的音乐游戏《DEEMO》的背景音乐。与其它落下式音乐游戏相同，Deemo的游戏方式也是从屏幕上方落下音符到达判定线时点击。游戏中只有点击与滑动两种音符。黑色的音符为点击音符，当落到判定线时点击即可。金色的音符为滑动音符，当落到判定线时点击不放，且依照音符位置及速度变更滑动手指，直到最后一个音符后放开。与其它落下式音乐不同的是，Deemo的音符落下时并没有固定的轨道，判定线的每个地方都可能有音符落下。此外，音符的宽度也不一致，大小从正常音符的一半到1/3屏幕大都有可能。
