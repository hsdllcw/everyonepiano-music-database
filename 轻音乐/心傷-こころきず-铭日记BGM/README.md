

《 **心傷** 》或《こころきず》是游戏《铭日记》的背景音乐。旋律平缓，但超级好听！

《铭日记》（みことにっき）是CHARON ねこふじかおる开发的一款游戏，于2013年5月28日发行。2021年11月26日跟YANASE GAME（NAO
YANASE）推出的手游《病娇恋爱冒险大全》里，发布铭日记的重制版《铭日记rewrite》（みことにっきリライト），玩法方面改成了《诗穗梨逃亡》那样的2D直立绘制风格探索，并追加了TE之后才能解锁的新补充剧情“铭memory”，关联到正统后续作揍好揍满幼稚园。2022年7月28日，扑家代理简中版登录ios与安卓双平台。
