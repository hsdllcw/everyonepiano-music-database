

**葬春双手简谱** 和五线谱完全对应。

_葬春_ 是网友西瓜JUN和Assen捷合作的一首古风歌曲，旋律蕴含淡淡的忧伤，请欣赏。

西瓜JUN，唱作人，独立音乐人。原创代表作：《狂野想乡》《忘记》《长生诀》。

歌词下方是 _葬春钢琴谱_ ，希望大家喜欢。
