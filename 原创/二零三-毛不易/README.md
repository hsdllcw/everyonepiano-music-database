

**二零三双手简谱** 和五线谱完全对应。

_二零三_
是明日之子总冠军毛不易为他的小屋创作的歌曲。如果说李宇春代表的是自下而上选择偶像的开始，那么毛不易就是为自己圈层选择代言人的开始。凭借自己的歌曲，让很多人记住了这个外表有些怂，有些衰，但是不甘于平凡的大男孩，他用八杯酒，道尽了人生所有。

歌词下方是 _二零三钢琴谱_ ，大家可以免费下载学习。
