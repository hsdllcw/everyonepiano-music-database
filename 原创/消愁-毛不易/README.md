

**消愁双手简谱** 和五线谱完全对应。

_消愁_ 是明日之子人气选手毛不易原创曲目，由毛不易演唱作词作曲。于2017年7月29日发行。

如果说李宇春代表的是自下而上选择偶像的开始，那么毛不易就是为自己圈层选择代言人的开始。一曲《消愁》让很多人记住了这个外表有些怂，有些衰，但是不甘于平凡的大男孩，他用八杯酒，道尽了人生所有。

歌词下方是 _消愁钢琴谱_ ，大家可以免费下载学习。
