

《 **守候** 》是游戏《新绝代双骄3》的主题曲，原唱是徐冥蕾。同时也是网络女歌手心然2005年7月翻唱的第一首作品。

《新绝代双骄3》是由著名游戏公司宇峻科技/BTpig制作，寰宇之星发行的一款角色扮演类型单机游戏。2002年8月发行上市。游戏以江小鱼及花无缺的后代为主角进行。歌词下面是
_ _守候钢琴谱__ ，喜欢的可以下载练习。
