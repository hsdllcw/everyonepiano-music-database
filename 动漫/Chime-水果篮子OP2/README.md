_Chime_
是电视动画《水果篮子》的片头曲，由大冢爱演唱。《水果篮子》是在2001年版电视动画后的第二次动画化作品，采用了全新的制作阵容以及配音阵容，在画面上也与前作有着巨大的差异。该作在剧情上不仅仅是部讲述恋爱的作品，角色们在经历挫折后得以成长的身姿会让观众们感同身受。总而言之，这是一部有着女主角与众多帅气男生们邂逅的剧情的同时也能够在其中认识到人类黑暗面之后可以感受到成长的感动的作品。
