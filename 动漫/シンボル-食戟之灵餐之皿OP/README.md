电视动画《食戟之灵：餐之皿》改编自附田祐斗原作、佐伯俊作画的漫画《食戟之灵》，由J.C.STAFF制作，是《食戟之灵》系列动画的第三期，于2017年10月3日起播出。在2017年12月16日12集前半季已完结。后半季远月列车篇于2018年4月8日在日本上映，中国大陆于4月9日每周一0.30放送。
