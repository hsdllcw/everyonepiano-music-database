_それは小さな光のような是_
出自日本动漫《只有我不存在的城市》里的片尾曲。电视动画《只有我不存在的城市》改编自三部敬原作的同名漫画，2015年6月19日宣布TV动画化。电视动画于2016年1月7日开始在富士电视台的“noitaminA”节目播出。该作品也是noitaminA继《乱步奇谭》、《全部成为F》之后的“推理三部曲”的最后一部。
