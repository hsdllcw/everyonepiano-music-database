_イコール_
是日本摇滚乐队Sumika为电视动画MIX演唱的片头曲。动画使用的柔和用色与整体构图很大程度的保留了原作漫画的风格，同时又有着自己独立的特色。运镜上的处理也用了不同漫画的分镜方式来引导剧本。让熟悉安达充作品的观众一方面有着怀念的感觉，一方面又颇感新鲜。总的来说，作品虽然并没有华丽的开场，但还是有着足以吸引人的元素。

歌词下方是 _イコール钢琴谱_ ，大家可以免费下载学习。
