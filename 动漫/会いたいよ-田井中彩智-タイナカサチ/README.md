

**会いたいよ** 是日本的女性创作歌手田井中彩智的歌曲。原名是田井中彩智，以艺名タイナカサチ进行歌手活动。所属经纪公司为“GENEON
ENTERTAINMENT”旗下的“SISTUS RECORDS”。2010年12月以“タイナカ彩智”名义下活动于演艺圈。

  
歌词下方是 _会いたいよ钢琴谱_ ，希望大家喜欢。
