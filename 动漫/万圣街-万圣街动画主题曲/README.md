

**万圣街** 是同名动画《万圣街》的主题曲，由洛天依、乐正绫、墨清弦以及言和共同演唱。

《万圣街》讲述一群来自不同地域的外国妖怪在中国贡献GDP的漫画故事。非人哉工作室制作。秉承非人哉的画风。“1031”是主角们在北京合租房的门牌号。
