

《 _この世の歌_ 》可以译为这个世界的歌，是岩崎琢作曲，丰口惠美演唱的《BLACK CAT》中的插曲。  
  
丰口惠美（豊口めぐみ）是一直非常活跃的声优，并演唱了《草莓100%》《爱丽斯SOS》《黑猫》等多部动画的插曲。于日本工学院在学同时参加文化放送的广播节目《広井王子のマルチ天国》火星女孩试镜夺冠出道后，透过同节目的千叶繁介绍加入了81Produce。  
  
歌词的下方还提供了 **この世の歌钢琴谱** ，欢迎感兴趣的朋友下载使用。
