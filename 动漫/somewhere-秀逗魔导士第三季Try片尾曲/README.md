**_Somewhere_ 是秀逗魔导士第三季Try的片尾曲，由桑岛法子演唱。  
  
秀逗魔导士，日本动画。由秀逗魔导士系列作品的改编的动画。是以原作本传小说第一部为主要内容的TV版动画。以及以原作外传小说为主的剧场版和OVA版组合而成，大致上分为两大系统。  
  
桑岛法子，日本著名的声优。代表作有《机动战舰》的尤里卡、《钢铁天使胡桃》的仲人、《银装骑攻》的哉生香织、《神风怪盗贞德》的贞德、《BLUE
GENDER》的玛琳、《无限之未知》的蓬仙葵、《魂之利刃》的成美娜等角色。除此之外，她还曾在《绝爱1989》、《快刀乱麻》、《VIRUS》、《特勤机甲队》、《BETTER
MAN》等作品中配过音。同其他声优一样，桑岛法子为多部动画演唱过主题曲，如《机动战舰》、《BLUE
GENDER》等，也出过多张CD如RADIO的ED曲、单曲“BELIEVE”等。
