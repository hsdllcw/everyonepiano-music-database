

**深い森双手简谱** 和五线谱完全对应。

_深い森_ 是根据高桥留美子著作的漫画《犬夜叉》作品改编的同名电视动画的片尾曲。此曲由Do As
Infinity演唱。该动画，由动画公司SUNRISE制作，全167话。后追加制作了26集的《犬夜叉完结篇》，至此全部完结，共193集，另外还有四部剧场版。

歌词下方是 _深い森钢琴谱_ ，希望大家喜欢。
