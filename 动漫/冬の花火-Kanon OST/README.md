

**冬の花火** 作为1999年发行的文字冒险恋爱类游戏《Kanon》的插曲，多为相泽佑一（游戏男主角，参见kanon词条）回忆时的BGM。

  
Kanon亦被称为Key社四季曲中的“冬”。起初是作为成人电脑游戏发售，之后Key推出了删除色情内容、适合一般玩家游玩的“全年龄向”的电脑游戏版本，而游戏也被陆续移植于Dreamcast和PlayStation
2等家用平台上。该视觉小说是一个描述因家庭因素而到北国小镇寄住在身为表妹的少女家中的少年，但几乎没有关于七年前在北国小镇的记忆。之后他遇到了几位少女，并慢慢恢复记忆。游戏提供了各式各样不同的默认故事路线，在玩家角色与游戏角色互动的过程中来决定进入不同的分支剧情。《Kanon》1999年版本首次销售随即成为排行榜第2名。到了今日《Kanon》已经推出许多游戏平台的版本，而销售量也已经超过300,000套。

_  
冬の花火_
此曲是由麻枝准创作的。麻枝准，Key所属脚本作家、作词家、作曲家、音乐制作人。生于三重县。海星高校、中京大学心理学科毕业。昵称为だーまえ。本名“前田纯”（与麻枝准同音）。
因其创作了众多神作级催泪弹作品，故经常被称作“大魔王”或“魔王大人”。一般被认为是作家，但同时也是一个拥有着纤细心灵的人。
