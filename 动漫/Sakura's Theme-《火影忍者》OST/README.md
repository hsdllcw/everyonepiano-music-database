《火影忍者》是日本漫画家岸本齐史的代表作，作品自1999年开始在周刊《少年JUMP》上连载后，读者反应非常热烈。随后，《火影忍者》的TV动画在日本东京电视台播放，成为最受欢迎的动漫作品之一。故事成功地将原本隐藏在黑暗中，用世界上最强大的毅力和最艰辛的努力去做最密不可宣和隐讳残酷的事情的忍者，描绘成了太阳下最值得骄傲最光明无限的的职业。
