

《 **桜ノ雨** 》（ _樱之雨_ ）是日本虚拟偶像歌手初音未来（初音ミク）演唱的歌曲。

这首桜ノ雨是由absorb(=halyosy)在日本中学毕业前期，也就是2008年三月份发布的。一经发布就夺走了即将毕业的日本中学生的眼泪，而且翻唱版本无数。其中还包括中国、韩国、新加坡等国家nico爱好者的参与。本专辑已于2008.11.26在日本发售，合人民币约90.00元。

歌词下方是 _桜ノ雨钢琴谱_ ，欢迎大家使用。
