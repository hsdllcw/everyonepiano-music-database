_ハナコトバ_
是由户松遥、白石晴香、三泽纱千香共同为电视动画《喜欢本大爷的竟然就你一个？》演唱的片尾曲。动画改编自日本小说家骆驼原作的同名轻小说作品，于2018年10月7日在电击文库25周年纪念秋之电击祭活动中宣布了动画化的决定。动画由CONNECT负责制作，于2019年10月2日起播出，全13话。
