

《 **ありがとう〜** 》是日本动画《今天开始做魔王》片尾曲。由BON'Z演唱。

《今天开始做魔王》其实是一部「今日からマのつく自由业！」的系列小说，原作者是乔林知。而这一套「今日からマ王！」，是偏向轻松、搞笑的幻想式故事，另一特点当然是美形男满车！

BON'Z：日本乐队组合，由北海道少年组成，词曲均有乐队成员亲自完成。

歌词下方是 _ありがとう〜钢琴谱_ ，欢迎大家使用。
