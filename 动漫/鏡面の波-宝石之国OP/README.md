_鏡面の波_
是日本歌手YURiKA演唱的歌曲，同时也是电视动画《宝石之国》的片头曲。《宝石之国》原是日本漫画家市川春子的漫画作品，讲谈社出版，于讲谈社杂志《月刊Afternoon》2012年12月号开始连载。2017年5月19日宣布推出电视动画，同年10月起在MBS电视台、东京都会电视台等电视台播出。台湾和香港则是在2018年由Animax进行电视首播。

歌词下方是 _鏡面の波钢琴谱_ ，大家可以免费下载学习。
