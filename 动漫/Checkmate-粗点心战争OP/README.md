_Checkmate_
是电视动画《粗点心战争》的片头曲，由MICHI演唱。动画改编自日本漫画家琴山（コトヤマ）原作的同名漫画。在漫画杂志《周刊少年Sunday》2015年43号上发表了其TV动画化的决定。电视动画于2016年1月7日~2016年3月31日播出，全12话。
