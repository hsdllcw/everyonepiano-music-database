

**さよならのこと** 是日本Leaf公司制作的一款十八禁恋爱游戏《白色相簿2》（英语：WHITE ALBUM2）的片尾曲。《白色相簿2》，分为《白色相簿2
序章》（WHITE ALBUM 2～Introductory Chapter～，简称：IC）和《白色相簿2 终章》（WHITE ALBUM
2～Closing
Chapter～，简称：CC）两部分发行。虽为1998年5月1日发售的《白色相簿》的续篇，但本作与前作的关联并不大。其极高的完成度和剧本素质使得本作成为难得的佳作，让人仿佛身临“剪不断，理还乱”的情感纠葛之中。

_  
さよならのこと_
此曲是由动画歌手上原れな演唱的。上原れな是近年来逐渐活跃起来的的一位新人，出生于1986年8月12日，音乐作品包括：《虹の架け桥》、《コスモスのように
》、《梦のつづき》
等等。2010、2011年间，以日本游戏公司Leaf的御用歌手身份，为PC游戏《白色相簿2》（简称WA2）演唱脍炙人口的主题曲《届かない恋》、插曲《After
All-缀る想い》、《closing》等游戏音乐，抒情的声线与游戏淡淡忧伤的主题完美融合，众多WA2玩家（患者？）赞不绝口。
