

**僕らの戦場** 是动画《超时空要塞Δ》第3、4、9、18话的插曲。

《超时空要塞Δ》或《MACROSS
DELTA》是《MACROSS》系列的日本电视动画，于2015年9月25日宣布制作决定。动画于2016年4月3日开始放送。全26话。故事发生在《Macross
Frontier》后的8年——公元2067年。
