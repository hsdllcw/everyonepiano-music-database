

《 **Sparkling Daydream** 》是日本动画《中二病でも恋がしたい!》的片头曲。  
  
《中二病也要谈恋爱！（中二病でも恋がしたい！）》是根据由日本小说家虎虎创作、逢坂望美负责插画的同名轻小说《中二病也要谈恋爱！》改编的动画作品。由动画公司京都动画制作，于2012年10月3日开始播放，全13话（含TV未播放的第13话）。  
  
歌词下方是 _Sparkling Daydream钢琴谱_ ，欢迎大家使用。
