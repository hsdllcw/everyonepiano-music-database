_ナツノハナ_
，中文译为夏花，是TV动画《怪化猫》的片尾曲。由日本创作女歌手JUJU演唱。《怪化猫》是由东映动画制作、富士电视台播出的怪谈动画作品，为动画《怪ayakashi》中的最后一个单元《化猫》的续作，制作人员也是原班人马。从2006年1月12日到3月23日播出，全11话。包含三个故事：《四谷怪谈》（1-4话）、《天守物语》（5-8话）、《化猫》（9-11话）。均以日本知名怪谈为基础、加入创作者们对原作的解读、以自己的观点来创作新的故事。这三篇故事之间并无相互关联，各个故事都有自己的风格，画风也各有差别。
