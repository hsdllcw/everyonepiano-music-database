

《 **ココロ** 》是日本虚拟歌手镜音リン（镜音铃）演唱的歌曲。由トラボルタP作词，作曲。

镜音铃，日本克里普顿未来传媒公司（CRYPTON FUTURE MEDIA）以Yamaha的VOCALOID
2语音合成引擎为基础开发贩售的虚拟歌手软件角色主唱系列的第二作，与镜音连一同于2007年12月27日发售。

歌词下方是 _ココロ钢琴谱_ ，欢迎大家使用。
