

**织梦行云双手简谱** 和五线谱完全对应，属于EOP教学曲。

《 _织梦行云_ 》这首乐曲是中文RPG游戏《仙剑奇侠传四》里面的柳梦璃主题音乐。

游戏中该乐曲第一次响起是在寿阳城柳府初遇柳梦璃之时。温婉舒缓的乐风带着一丝凄艳，古筝轻撩间透露出少女对世间感情憾事的无奈，混杂着莫名的伤感，心事难明、欲述不能。

2011年11月9日CCTV音乐频道播出了由鲁璐现场演绎的箜篌版《织梦行云》。鲁璐身着柳梦璃的大家闺秀装。同时CCTV在屏幕下方滚动显示《仙剑四》和该曲的简单介绍。

《仙剑奇侠传四》是国产单机角色扮演游戏《仙剑奇侠传》系列的第五部作品，由大宇资讯旗下的软星科技（上海）有限公司制作，于2007年8月1日发行。故事的起点始于黄山青鸾峰之巅，驰骋于神州大地、往来于天地之间，东临渤海、南抵巴蜀、北至传说中的“北极之地”、西达上古圣山昆仑仙境。既有繁华热闹、歌舞升平的中原富庶之地，又有大漠边关、丝绸之路上的小小驿站。每一个城镇之中的建筑、草木，乃至于店中出售的食物，都各有其特色。游戏的剧情受人称赞，迷宫难度较前几代有所降低，而且在国产单机游戏日渐式微的今天，四代更具特殊意义。
