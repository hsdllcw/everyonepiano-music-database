

**7 -seven-** 是A-1
Pictures改编的电视动画「七原罪」的片尾曲。漫画「七原罪」起初于2011年在「周刊少年magazine」52号上以短篇连载的形式登场，连载获得好评后于2012年在「周刊少年magazine」45号上开始正式连载，至今已经推出了6卷单行本，汉化内容至73话。本作世界观设定宏大，故事围绕离家出走的王国公主以及七原罪骑士的故事展开……
