

张韶涵一首《 **破茧**
》无缝衔接斗罗新篇章。该歌曲由曹德智，郭德紫毅作词，文颖秋作曲。腾讯音乐娱乐集团出品，制作家发行。于2020年5月23日上线腾讯音乐平台。

提到 _破茧_
，我的眼前浮现了这样一个画面：困在茧里的蝴蝶一点点挪动自己的小身板，忍受如撕裂两半一般的剧痛去靠近茧外的光明。割开茧的一瞬，沐浴到了黑暗过后的温暖。

Angela其实很适合这首歌，各种意义上来讲。宽广的音域、富有穿透力的声音和极具爆发的声线总能让大家深陷其中无法自拔，这声音传达了一种叫斗罗的力量：会挫伤会心痛，英雄依然奋勇去战斗，哪怕未来如洪流，绝处逢生见招拆招，也不曾退后！
