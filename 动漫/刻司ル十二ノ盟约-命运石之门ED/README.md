

《 **刻司ル十二ノ盟约** 》是日本动画《命运石之门》（STEINS;GATE）结尾曲。由ファンタズム演唱。

  

《命运石之门》（Steins;Gate，シュタインズ・ゲート）是根据同名游戏改编，由日本动画公司WHITE
FOX制作的动画作品，于2011年4月5日播放，全24话。游戏作品亦改编成同名漫画。本作剧场版于2013年4月20日在日本上映。
