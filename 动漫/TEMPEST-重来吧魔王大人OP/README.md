

**TEMPEST双手简谱** 和五线谱完全对应。

_TEMPEST_ 是电视动画《重来吧，魔王大人》的片头曲。且此曲由日本女性声优、偶像、歌手石原夏织演唱。

《重来吧，魔王大人！》改编自神埼黑音著作的同名轻小说，于2018年9月宣布了动画化的消息。动画由EKACHI
EPILKA负责制作。2019年7月3日起，每周三24点TOKYO MX首播。中国大陆于7月3日起在哔哩哔哩播出。
