

**Alive双手简谱** 和五线谱完全对应。

_Alive_
是电视动画片《达尔文游戏》的片尾曲。动画改编自FLIPFLOPs创作的同名漫画作品，于2018年11月宣布动画化的消息。动画由Nexus负责制作，于2020年1月3日起播出
，全11话。

另外，此曲由绫野真白演唱。绫野真白，出生于北海道洞爷湖町，日本女歌手，演唱流派J-POP、动漫插入曲、主题歌。

歌词下方是 _Alive钢琴谱_ ，大家可以免费下载学习。
