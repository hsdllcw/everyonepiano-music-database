

《花仙子》（花の子ルンルン）（はなのこるんるん）是日本1979年2月9日～1980年2月8日在日本的电视台系列中放映的动画节目。之后"花仙子"变为了表示欢快气氛的拟态语的一般用语、在1980年代初由林真理子的"ルンルンを买っておうちに帰ろう"（买了Runrun带回家）一句话让这个词成为了年轻女性间的流行语。
