_こわれかけのオルゴール_ 是动画《快要坏掉的八音盒》的主题曲。由佐藤ひろ美演唱。

《快要坏掉的八音盒》，原作为ElectromagneticWave，动画化以后于2009年夏天上映，监督为川口敬一郎。故事中的小女孩就是被人形化的八音盒。正如片名所表达的一样，一个快要坏掉的八音盒。
男主在一个夏天捡到了这个快要坏掉的八音盒，并陪伴她度过了这个夏天，这是她最后的夏天……
