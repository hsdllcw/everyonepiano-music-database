《 **光るなら** 》（若能绽放光芒）是由日本乐团Goose
house乐队演唱的歌曲，作为日本动漫《四月是你的谎言》中的第一首片头曲，歌曲爽快的曲风令人炫目，以至于很多观众就是因为这首歌才开始接触到作品。

《四月是你的谎言》原是日本漫画家新川直司的少年漫画作品，于讲谈社《月刊少年Magazine》2011年5月号开始连载至2015年3月号完结。2013年获得讲谈社漫画赏少年部门奖项。电视动画从2014年10月开始，在富士电视台的深夜动画时段《noitaminA》中播放。2016年荣获SUGOI
JAPAN Award动画部门首奖。
