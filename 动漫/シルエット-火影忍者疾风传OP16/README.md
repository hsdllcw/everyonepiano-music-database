

**シルエット**
是动画《火影忍者》的续篇《火影忍者疾风传》的片头曲。该动画是根据岸本齐史创作的漫画《火影忍者》的第二部所改编的，于2007年2月15日开始在东京电视台播出。

  
歌词下方是 _シルエット钢琴谱_ ，希望大家喜欢。
