_Elemental World_ 是TV动画《政宗君的复仇》片尾曲，由日本女歌手ChouCho演唱。

TV动画《政宗君的复仇》改编自竹冈叶月原作、Tiv作画的同名漫画，由SILVER LINK.负责制作。在2016年6月27日发售的《月刊Comic
REX》8月号上正式发布了TV动画化决定，10月27日官方公开了该作的宣传PV，于2017年1月5日开始播放。
