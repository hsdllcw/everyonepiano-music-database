

‘ **My Soul,Your Beats**!／ _Brave Song_ ’（マイ・ソウル
ユア・ビーツ／ブレイヴ・ソング）是日本女创作歌手Lia的第9张作品，多田葵的第3张作品。于2010年5月26日发行。

原曲为TV动画《Angel Beats!》片尾曲ED，由“大魔王”麻枝准为此动画量身打造，并由声优多田葵演唱。此曲另有Gldemo
ver为动画中的乐队Girl Dead Monster翻唱的摇滚版。
