

**龙骑伙伴双手简谱** 和五线谱完全对应。

_龙骑伙伴_ 是动画片《蓝猫龙骑团》的主题曲，歌曲风格可爱，由刘维演唱。

《蓝猫龙骑团》是由湖南蓝猫卡通传媒有限公司推出的一部动画片，其创作构想是将恐龙和蓝猫结合起来，这个节目的设计代表着中部地区卡通企业文化创意产业的一个成果，逐渐地从最初的发展走向经营理念的成熟，赢利模式逐渐清晰以后的一个举措。

歌词下方是 _龙骑伙伴钢琴谱_ ，希望大家喜欢。
