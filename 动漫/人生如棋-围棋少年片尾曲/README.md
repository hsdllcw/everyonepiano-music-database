_人生如棋_
是由马风清导演，孙晓松编剧，中国国际电视总公司制作出品的动画《围棋少年》的片尾曲。该片是以明末围棋神童江流儿的成长经历为主线，采用评书式的叙事风格，运用“侠”、“俗”、“情”等叙事元素，弘扬了中华围棋的精神和中国棋手的爱国情节。该片人物造型特点鲜明，制作精致，人物、背景具有立体感、层次感。

歌词下方是 _人生如棋钢琴谱_ ，大家可以免费下载学习。
