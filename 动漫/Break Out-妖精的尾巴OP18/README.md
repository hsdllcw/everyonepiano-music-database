

**Break Out**
是东京电视台出品的动画作品妖精的尾巴的片头曲。该剧作品讲述了魔导士公会“妖精的尾巴”的成员露西、纳兹、哈比、格雷、艾露莎和他们的众多伙伴们在这个魔法世界中的一系列热血故事。动画从2009年10月12日开始，在东京时间每周六的上午10时30分放送，于2013年3月30日暂时完结，共放送175集。重播版于2013年4月4日起每周四下午5:00在日本电视台播放。据《周刊少年Magazine》2014年4•5号合并刊消息，动画新系列已于2014年4月5日开播。

且 _Break Out_ 此曲是由日本的男子音乐组合V6演唱。组合所属日本杰尼斯事务所。V6之名取自于“Victory
six”，也据成员说V代表Vicycle及Veteran。而V6也是继光GENJI后，第二组以分组形成的组合。在1995年8月份成立，
并在1995年11月1日正式出道。

歌词下方是 _Break Out钢琴谱_ ，希望大家喜欢。
