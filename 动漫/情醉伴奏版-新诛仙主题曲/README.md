

**情醉**
是《新诛仙·六道轮回》主题曲。歌词有两个版本：其中一版为诛仙版，歌词讲述的是小说《诛仙》中为情而醉的两个女子——陆雪琪和碧瑶；另一版为陆雪琪版，是董贞出于对小说女主人公陆雪琪的喜爱，再次为陆雪琪填词，又名为缘碎，唱出了“人如桃花艳，心似冰霜剑”的痴情女子——陆雪琪。

_情醉_ 作词：董贞；作曲：董贞；演唱：董贞、盛威；制作：北京小旭音乐文化有限公司；所属专辑：返璞归真。
