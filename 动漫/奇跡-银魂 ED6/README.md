

**奇跡**
是改编自日本漫画家空知英秋同名漫画《银魂》的电视动画作品中的片尾曲。《银魂》的电视动画由2006年4月4日开始每周二19:00于东京电视台播放，2010年3月25日第一期完结，共计201话。2010年4月5日开始，每周一18:00播放“银魂精选集”。2011年4月4日开始播放动画第二期，集数承接第一期，从第202话开始

  
歌词下方是 _奇跡钢琴谱_ ，希望大家喜欢。
