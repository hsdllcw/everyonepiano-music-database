是大桥卓弥/常田真太郎作词、作曲，雨宫天演唱的歌曲，同时也是电视动画《一周的朋友》的片尾曲。动画《一周的朋友》改编自日本漫画家叶月抹茶的同名漫画。在漫画杂志《月刊GANGAN
JOKER》2013年12月号（11月22日发售）上，发表了《一周的朋友。》TV动画化的决定。电视动画于2014年4月6日首播，全12话。
