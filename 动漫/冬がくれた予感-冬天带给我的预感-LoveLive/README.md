

**冬がくれた予感**
，中文译为冬天带给我的预感，是为纪念《LoveLive!学园偶像祭》用户突破300万人制作的单曲，也是BiBi的第三首单曲，于2014年12月24日（平安夜）发售。

值得一提的是在这张单曲中两首RAP担当的空丸，给人完全就是一个女神一个逗比的巨大落差啊！

此曲于15年2月10日荣获日本金单曲奖，可见该曲制作的精良程度。

歌词下方是 _冬がくれた予感钢琴谱_ ，大家可以免费下载学习。
