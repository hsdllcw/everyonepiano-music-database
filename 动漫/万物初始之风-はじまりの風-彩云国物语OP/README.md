_万物初始之风_ ，はじまりの風，是平原绫香演唱的歌曲，由路川 ひまり填词，YANAGIMAN编曲，路川ひまり作曲，是日本动漫《彩云国物语》的片头曲。

电视动画《彩云国物语》改编自日本轻小说家雪乃纱衣原作的同名轻小说。2005年12月3日决定动画化。电视动画全2期，第1期于2006年4月8日首播，全39话，第2期于2007年4月7日首播，全39话。
