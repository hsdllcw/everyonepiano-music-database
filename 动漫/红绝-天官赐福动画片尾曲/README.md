

《天官赐福》动画片尾曲《 **红绝**
》由歌手胡夏演唱。胡夏用极具叙事性的声音诉尽“天下之大为你花满城，官名抛空执花仗剑救苍生”的情义，瞬间被带入到歌曲的情境中。

整首歌曲将「天官赐福，百无禁忌」「身在无间，心在桃源」的主题含蕴于歌词之中，构筑一个弘大壮丽浪漫而又至死不渝的世界，正如歌中所唱「源起便信缘不会终」。

待到花开满时，  
即使有无常有幻觉有业火，  
也知有温柔有信徒有余生。
