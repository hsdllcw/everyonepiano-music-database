《 _蓝染_
》是日本动画《地狱少女》第二季《地狱少女二笼》中的片尾曲，由阎魔爱的声优能登麻美子所唱。在OST中，《蓝染》的日文名是《あいぞめ》。蓝染意为“尽染悲伤”。  
  
能登麻美子，日本女性声优、演员、歌手。拥有丝丝细语的声线，多为神秘、弱气或冷静的少女角色配音。代表作品有《地狱少女》阎魔爱、《CLANNAD》一之濑琴美、《好想告诉你》黑沼爽子、《草莓100%》东城绫、《零度战姬》莎缇莱萨·L·布丽姬、《妖精的尾巴》梅比斯·维维亚米利欧等。
