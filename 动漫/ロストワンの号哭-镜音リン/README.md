

**ロストワンの号哭** 是日本虚拟偶像歌手镜音リン（镜音铃）演唱的歌曲。  
  
镜音铃，CRYPTON FUTURE MEDIA以Yamaha的VOCALOID
2语音合成引擎为基础开发贩售的虚拟歌手软件角色主唱系列的第二作。于2007年12月27日发售。  
  
歌词下面是ロストワンの号哭钢琴谱，欢迎大家使用。
