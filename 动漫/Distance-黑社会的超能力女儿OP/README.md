_Distance_ 是电视动画黑社会的超能力女儿的片头曲。由日本的女性声优、歌手村川梨衣演唱。

电视动画《黑社会的超能力女儿》改编自大武政夫原作的同名漫画，于2017年9月宣布动画化。动画由feel.负责动画制作，2018年4月6日起每周五在AT-
X首播。主要讲述了支持着芦川组的年轻智力型黑道新田义史，过着被喜欢的壶所包围的悠然自得单身生活。但某一天，随着装在神秘物体里的少女雏来到他家，他的生活为之一变。他被能使用念动力的雏所胁迫，迫不得已开始和她同居！容易暴走的雏，不论在组里还是在学校都为所欲为。新田为此头痛不已，但由于自己那老好人的性格而陷入总是要照顾她的境地。究竟这种生活会变得怎样呢？老好人不法之徒与任性超能力少女的危险而热闹的日常开始了！
