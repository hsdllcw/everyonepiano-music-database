

「天官赐福」动画插曲《 **一花一剑**
》，由《天官赐福》动画音乐监督、知名音乐人杨秉音作曲并担纲制作，由新锐作词人申名利、王晓倩联手打磨，邀高音清亮的实力偶像李鑫一献唱，歌曲宛如一幅琉璃画卷栩栩展开。

卷轴舒展，白衣少年与天煞孤星，风卷花落，一瞥惊鸿，轮回宿命开启。荒山野径中，有暗夜诡异的氛围，也有银蝶翩跹时暗藏的温柔重逢。月光之下，山似是为邀与君行而命名，而新嫁娘也似是不太遥远的缱绻美梦。

时间流幻，花树剑影一念桥边如浮光掠影。繁华覆灭黑云漫天，即使微弱也一如既往。是雨中笠，是无名花；是无数日夜，是百世沦陷。恍若回旧时，此去天庭，你为济苍生，我便为你。

一人一世与君行
