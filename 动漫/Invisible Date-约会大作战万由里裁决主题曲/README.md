_Invisible Date_ 是剧场版动画《约会大作战 万由里裁决》的主题曲，由sweet ARMS演唱。《约会大作战
万由里裁决》是由元永庆太郎执导，白根秀树编剧，Production
IMS制作的《约会大作战》剧场版动画。该作于电视动画第2期《约会大作战Ⅱ》放送完毕时公布剧场版制作决定。2014年9月19日在小说第11卷书腰中公布了剧场版将会采取有别于原作的完全原创剧情。2014年12月14日举办的现场活动“DATE
A Fes II”当中，剧场版的标题确定。影片于2015年8月22日在日本上映。
