是电视动画《异种族风俗娘评鉴指南》的片头曲。《异种族风俗娘评鉴指南》（日语：異種族レビュアーズ）原是由日本漫画家天原（原作）和masha（作画）创作的喜剧漫画，连载于Niconico静画的《月刊龙时代》。电视动画版由Passione负责制作，于2020年1月11日开播。
