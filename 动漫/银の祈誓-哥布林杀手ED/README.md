_银の祈誓_
是电视动画《哥布林杀手》的片尾曲，由歌手soraru演唱。《哥布林杀手》的动画表现虽然没能技惊四座，但也有着及格线以上的水准。尽管过激的演出明显地强调出了原作的黑暗风格，但动画也展现出了原作的其他重点部分，再加上监督尾崎隆晴在上部作品的表现，且作为剧本的漫画版也相当优秀，非常适合喜欢“硬派”奇幻战斗风格作品的观众。

歌词下方是 _银の祈誓钢琴谱_ ，大家可以免费下载学习。
