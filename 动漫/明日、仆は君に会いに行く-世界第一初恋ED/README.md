

**明日、仆は君に会いに行く双手简谱** 和五线谱完全对应。

  

《 _明日、仆は君に会いに行く_ 》（明天我要去见你）是日本动画《世界第一初恋》片尾曲。

  

《世界第一初恋》是根据中村春菊创作的同名漫画《世界第一初恋》改编的动画，共两季。第一季于2011年4月8日播出，第二季于2011年10月7日播出。

  

《世界一初恋》是一部内含“小野寺律的场合”“吉野千秋的场合”“木佐翔太的场合”三个故事的耽美向系列作品，原作者是著名漫画家中村春菊，而小说“吉野千秋的场合”则由藤崎都编写。作品中以非常接近真实的出版社为舞台，讲述了丸川出版社少女漫画编辑部内的各位编辑的故事，主编、副主编、新人编辑、又或者是娃娃脸编辑，各自交织出不同的恋爱人生的故事。

  

《明日、仆は君に会いに行く》是一首第一次听到就会喜欢上的，很美的歌曲。歌词下方是 _明日、仆は君に会いに行く钢琴谱_ ，希望大家喜欢。
