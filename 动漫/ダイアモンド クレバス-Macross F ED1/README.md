

《 **ダイアモンド クレバス** 》（Diamond Crevasse）是日本动画《Macross F》（マクロスF）ED1。

  

「Macross F」是日本动画「Macross Series」的25周年纪念企划作品。于2008年4月～9月在MBS,
TBS系放送，全25话。剧场版前篇「虚空歌姬」，于2009年11月21日公开。剧场版后篇「恋离飞翼」，2011年02月26日公开。
