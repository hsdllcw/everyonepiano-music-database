《熊猫京京》（Jin Jin and the Panda
Patrol）是一部由北京金熊猫动画公司和美国萨班国际娱乐有限公司于1994年合作制作的动画片，共二十六集。讲述了一只叫做京京的熊猫历尽艰辛，寻找熊猫乐园的经历。该片已在世界29个国家和地区放映。本动画于1996年在中国播放。
