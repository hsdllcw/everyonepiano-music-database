《偶像大师：走向光辉的未来》是由锦织敦史、高桥龙也创作的剧场版动漫，中村绘里子、高桥智秋参加演出。剧场版的故事将围绕合宿旅行来展开。偶像，是女孩子们一直以来的憧憬。但能站在顶点的，只有仅仅数人。13位少女，就此经她们所属的事务所“765
Prodution”，跨进了那个充满竞争的世界。出道约半年，事务所来了一位全新的制作人。他跟少女们都下定决心，向顶级偶像之路进发。剧场版讲述的是765Prodution宣布了将举办作为新的大目标的竞技场演唱会，集合了全部成员一起去合宿的故事。
