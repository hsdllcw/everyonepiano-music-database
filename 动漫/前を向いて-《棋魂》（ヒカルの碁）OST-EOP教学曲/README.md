《 _前を向いて_ 》是日本动画《棋魂》（ヒカルの碁）中的插曲。所属专辑：《ヒカルの碁 音楽撰 遥かなる高みへ》。由若草恵作曲。  
  
《棋魂》（日文原名ヒカルの碁），中文意义为“光的围棋”。台湾大然文化代理时代译为棋灵王，东立出版社译作棋魂；香港文化传信则译作棋魂；中国大陆天津人民美术出版社译名为棋魂·光之棋。是日本《少年JUMP周刊》（集英社）于1999年至2003年连载的以围棋为题材的少年漫画。后改编成动画搬上银幕。  
  
曾被改编为动画，由东京电视台、电通及Studio
Pierrot共同制作，并于东京电视台系列（TXN）在2001年至2003年播放，分为两辑，总共75集。另外，2004年1月3日日本播出特别篇《迈向北斗杯之路》。  
  
动画在北美在美国ImaginAsian电视台和Toonami Jetstream（网络）放映，英语配音演员几乎全部为加拿大人。  
  
动画在香港于2003年3月首播、2006年4月重播。2009在J2再度重播。此次为无删减版本。动画未涉及原著中北斗杯三国对战的剧情，仅点到北斗杯日本预选赛为止。
