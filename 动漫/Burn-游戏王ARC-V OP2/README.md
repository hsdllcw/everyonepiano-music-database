

**Burn!**
是动漫「游戏王」新系列「游戏王ARC-V」的OP2。由日本乐队组合「超特急」演唱。「游戏王ARC-V」是「游戏王」系列的第5部作品。2014年4月6日和11日分别在东京电视台和BS日本首播。在东京电视台接档「游戏王ZEXAL」的第二季「游戏王ZEXAL
Ⅱ」，于每周日17：30（北京时间16:30）放送。
