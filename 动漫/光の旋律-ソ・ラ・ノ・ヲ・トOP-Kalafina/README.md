

**光の旋律** 是Kalafina的第7张单曲。于2010年1月20日由SME Records公司发行。此单曲分别发行了初回生産限定盘及通常盘2个版本。

  
另外， _光の旋律_ 这首歌曲也是A-1
Pictures制作的日本电视动画，动画的力量的第1部作品空•之•音（日语：ソ・ラ・ノ・ヲ・ト）的主题曲，该动画2010年1月至3月在东京电视台播放。漫画于月刊Comic电击大王2010年1月号连载。

  
歌词下方是 _光の旋律钢琴谱_ ，希望大家喜欢。
