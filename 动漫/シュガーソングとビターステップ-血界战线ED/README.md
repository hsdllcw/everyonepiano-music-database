GARDEN（ユニゾン・スクエア・ガーデン）是日本的摇滚乐队，由铃木贵雄，斋藤宏介，田渊智也三人2004年结成乐队，是一支现场实力很强的乐队。

电视动画《血界战线》改编自由日本漫画家内藤泰弘原作的同名漫画。作品动画化决定的消息在2014年6月4日发售的漫画杂志《Jump
SQ》2014年7月号以及漫画单行本第9集上正式宣布。电视动画于2015年4月4日开始播出，在中国大陆地区的译名为《幻界战线》。
