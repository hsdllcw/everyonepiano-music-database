

**その声を覚えてる** 是改编自西尾维新原作的轻小说《物语系列》的电视动画《物语系列 2nd
season》的片尾曲。动画内容包括《物语系列》中的《猫物语（白）》、《倾物语》、《囮物语》、《鬼物语》和《恋物语》等五部作品。另外，《花物语》则纳入《物语系列
2nd seaso+α》名义之下，于2014年8月16日以SP的形式5话连续播放。

  
歌词下方是 _その声を覚えてる钢琴谱_ ，希望大家喜欢。
