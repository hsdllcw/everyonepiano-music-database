_  
Death Image_ 是日本动漫《死亡笔记》的插曲，由日本作曲家谷内秀樹创作。《死亡笔记》（Death
Note）为日本大场鸫原作/小畑健作画的漫画作品。2003年8月于集英社杂志《周刊少年Jump》36号以短篇形式刊载《死亡笔记》的原型故事，获得读者投票第一位，并于2003年12月至2006年5月刊登连载版，共108话，内容与短篇有所不同。漫画连载版亦改编成电视动画、电影、轻小说、电视剧等。
