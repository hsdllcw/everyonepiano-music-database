

**とあるいつもを双手简谱** 和五线谱完全对应。

_とあるいつもを_ 是动画作品《小森同学拒绝不了》的主题曲，由幽闭サテライト演唱。

《小森同学拒绝不了！》讲述了一名拒绝不了他人请求的女生（小森 しゅり）和她两位好友（西鸟 めぐみ、根岸 まさ子）之间，所发生的日常搞笑故事。

歌词下方是 _とあるいつもを钢琴谱_ ，希望大家喜欢。
