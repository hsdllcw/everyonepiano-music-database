Winter II_ 是日本动画《新恋爱白皮书》（ボーイズ ビー。BOYS BE）中的插曲。  
  
《新恋爱白书》（原名：BOYS BE…）是イタバシマサヒロ板桥雅宏原作，玉越博幸绘画的一部少年漫画作品，在台湾翻译成新恋爱白书。另被改编成动画BOYS
BE...  
  
玉越博幸作品从男孩的角度出发， 用男孩的观点来认识爱情，了解爱情。一部典型的少男思春期的作品。
和[恋爱季节],还有VC上的[初吻]很有些相似，都是在讲恋爱中男女的小故事。  
  
《Winter II》是一首很不错的曲子。下面是 _Winter II钢琴谱_ ，希望大家喜欢。
