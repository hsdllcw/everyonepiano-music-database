

**花ハ踊レヤいろはにほ**
是MADHOUSE改编的电视动画《花舞少女》的片头曲。《花舞少女》改编自日本漫画家浜弓场双原作的同名漫画。在一迅社旗下的漫画杂志《まんがタイムきららForward》2014年2月号（2013年12月24日发售）上正式发表了《花舞少女》TV动画化的决定
。电视动画官网于2014年4月13日开启。电视动画于2014年7月8日开始播放，全12集。

  
歌词下方是 _花ハ踊レヤいろはにほ钢琴谱_ ，希望大家喜欢。
