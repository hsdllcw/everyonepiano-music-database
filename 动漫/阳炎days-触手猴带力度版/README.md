_蜉蝣daze_
（カゲロウデイズ，又译为阳炎daze），是じん（自然の敌P）在2011年9月30日投稿的初音未来原创歌曲。也是他的第三首vocaloid原创作品。歌曲讲述了一个发生在“你”身上，再常见不过的令人目眩的夏天的故事。

  

阳炎days作为阳炎project（カゲロウプロジェクト）的其中一部收录在じん（自然の敌P）的商业专辑《メカクシティデイズ》中（重编曲再录版）。
