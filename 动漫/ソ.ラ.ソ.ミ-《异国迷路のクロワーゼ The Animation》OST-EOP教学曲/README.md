《ソ.ラ.ソ.ミ》是日本动画《异国迷路のクロワーゼ The Animation》（异国迷宫的十字路口）中的音乐。所属专辑：《異国迷路のクロワーゼ The
Animation オリジナルサウンドトラック》。  
  
《异国迷宫的十字路口》（日语：异国迷路のクロワーゼ）是根据日本漫画家武田日向著作的漫画《异国迷宫的十字路口》改编的动画作品。在2010年的12月号的《月刊Dragon
Age》上，在为下期作宣传的版面上，宣告了《异国迷宫的十字路口》及《魔具少女！》这两个正在连载的漫画将于下月号的《月刊Dragon
Age》上有重大发表。2011年7月，在AT-X、千叶电视台等全国独立UHF放送协议会中播放。动画版正式标题为《异国迷宫的十字路口 The
Animation》。  
  
19世纪后半叶，日本解除锁国政策，正热衷于西洋文化。而此时的欧洲，反而热情的迎接着拥有特殊价值观的日本文化。并且，这时代也是人事物激烈更迭的近代化过渡期。在半世纪前曾风光一时的商场“国王的长廊”，也因落后于时代潮流，正面临关门的危机。就在这种情况下，一名来自日本的萝莉来到了这个充满魅力的法国。
