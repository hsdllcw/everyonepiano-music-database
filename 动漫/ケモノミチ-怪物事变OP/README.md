_ケモノミチ_
是电视动画片《怪物事变》的片头曲，由日本歌手小野大辅演唱。动画《怪物事变》改编自蓝本松著同名漫画作品，于2019年12月宣布动画化，由亚细亚堂负责动画改编，于2021年1月10日起播出。

歌词下方是 _ケモノミチ钢琴谱_ ，大家可以免费下载学习。
