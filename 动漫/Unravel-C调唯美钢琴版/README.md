

东京食尸鬼动画片的主题曲非常的燃，很多朋友听过主题曲之后都表示无法自拔，其中有一首 **Unravel** 就深的大家喜爱。所以小编制作了 _Unravel
C调唯美钢琴版简谱_ 以及五线谱，感兴趣的朋友可以免费下载。

此曲是是TK from 凛として时雨的音乐作品，收录在《Signal》专辑中。于2014年11月获过“日本最佳动画主题曲奖”。
写这首歌时TK花了很多心思，特别是歌词，其内容贴合动画的主题。

歌曲采用了后硬核，独立摇滚，另类摇滚，前卫摇滚，情感核的手段。音乐瞬间同时响起，一种空明感油然而生，柔和而且平静。但这只是短暂的宁静，在“你却什么也看不见”唱完后，瞬间耳边传来了激烈的鼓声；再随后，充满快感的钢琴声奏起，快速的弹奏让人感到血液在沸腾，心情一并激动起来；然后所有的乐器全部上阵，点燃听众们的激情。随后，曲调却又柔和了下来，充满了节奏感，鼓声与钢琴的完美配合让听者的心情随之放松。突然的放松，让鼓声与钢琴声不再是主角，小提琴占据了双耳的全部，开启了音乐高潮的大门。歌词与曲调的完美搭配，让人不禁感受到这首歌曲的刚柔并济、感受到它完美的融合。在欣赏这美妙的节奏的同时，再来关注每一句歌词，原作一定会很自然地产生联想，歌词的每一句都与剧情有着密不可分的关系、与主角金木有着无法分割的关联。再加上主唱纤细的嗓音，很好的诠释了本作男主角的心态，挣扎、绝望、哭泣、嘶喊……
