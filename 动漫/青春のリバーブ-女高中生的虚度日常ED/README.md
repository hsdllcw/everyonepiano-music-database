_青春のリバーブ_ 是丰崎爱生、
赤崎千夏、户松遥于2019年7月24日发行的歌曲。同时也是动画《女高中生的虚度日常》的片尾曲。动画《女高中生的虚度日常》改编自ビーノ创作的同名漫画，由Passione负责制作，于2019年7月5日起播出。全12话。

歌词下方是 _青春のリバーブ钢琴谱_ ，大家可以免费下载学习。
