

《 _うたかた花火_ 》是由ryo作詞／作曲／編曲，歌手supercell演唱的，作为火影忍者疾风伝中的一首片尾曲使用。  
  
收录在专辑《 _うたかた花火_ ／星が瞬くこんな夜に》之中，于2010/08/25发售。  
  
歌词的下方还提供了 **うたかた花火钢琴谱** ，欢迎感兴趣的朋友下载使用。
