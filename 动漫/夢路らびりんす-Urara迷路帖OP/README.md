_夢路らびりんす_
是J.C.STAFF制作的电视动画《Urara迷路帖》的片头曲。由Labyrinth（千矢（CV.原田彩枫）、绀（CV.本渡枫）、小梅（CV.久保由利香）、乃乃（CV.佳村遥））演唱。动画改编自はりかも创作的同名漫画作品。在2016年5月号（3月16日发售）的《Manga
Time Kirara Miracle》上，发表了其TV动画化的决定。

歌词下方是 _夢路らびりんす钢琴谱_ ，大家可以免费下载学习。
