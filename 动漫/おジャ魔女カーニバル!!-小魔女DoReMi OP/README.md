

《 **おジャ魔女カーニバル!!》** 是日本动画《小魔女DoReMi》片头曲。由MAHO堂演唱。

  

《小魔女DoReMi》（日语：おジャ魔女どれみ，日语罗马字：英语：Ojamajo Doremi，官方英文：英语：Magical
DoReMi）是一魔法少女动画的系列，由东映动画制作。小魔女DoReMi动画在日本的播映时间，由1999年2月开始，直至2003年1月结束。是继美少女战士之后，最受日本学生欢迎的少女动画系列之一。
