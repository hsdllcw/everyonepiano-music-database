

**天使のゆびきり**
是1998年GAINAX制作电视动画《他和她的事情》的片头曲。电视动画《他和她的事情》改编自由日本津田雅美创作的同名漫画，于1998年10月2日开始在东京电视网首播，全26话。

  
歌词下方是 _天使のゆびきり钢琴谱_ ，希望大家喜欢。
