《夜樱四重奏》（日语：夜桜四重奏
～ヨザクラカルテット～）为ヤスダスズヒト的日本漫画作品，于《月刊少年天狼星》（讲谈社）2006年3月号连载中。2008年10月起，本作改编的动画开始在TBS系电视台放映。  
  
人类与妖怪共存的樱新町陆续发生超越人类力量的奇怪事件，拥有特殊能力4名少年、少女负责前往解决。
