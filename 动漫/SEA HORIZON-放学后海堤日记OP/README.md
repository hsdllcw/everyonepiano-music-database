_SEA HORIZON_
由海野高中海堤社演唱，同时也是电视动画片《放学后海堤日记》的片头曲。动画片《放学后海堤日记》改编自小坂泰之创作的同名漫画作品，由大隈孝晴担任导演，志茂文彦担任总编剧，熊谷胜弘担任角色设计，动画工房负责动画制作，于2020年4月7日开播，共12集。

歌词下方是 _SEA HORIZON钢琴谱_ ，大家可以免费下载学习。
