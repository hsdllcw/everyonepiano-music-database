

**爱爱爱に撃たれてバイバイバイ** 是manglobe制作的日本电视动画作品《武士弗拉明戈
》的片头曲。该动画由2013年10月10日深夜开始，在富士电视台的深夜动画节目中播放。

  
且 _爱爱爱に撃たれてバイバイバイ_
这首歌曲是由FLOW演唱的。FLOW是日本一组摇滚乐团，隶属于Amuse旗下，1998年结成，2003年发行首张单曲“ブラスター”后正式出道。

  
歌词下方是 _爱爱爱に撃たれてバイバイバイ钢琴谱_ ，希望大家喜欢。
