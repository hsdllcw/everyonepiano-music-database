

**願い**
是电视动画《仙乐传说》的片尾曲。由日本流行歌手引田香织演唱。仙乐传说的动画化风声是在2005年11月宣布OVA《幻想传说》再次跳票时传出，那时便已决定OVA化。由《我的女神》的作者藤岛康介担当人物原设。

  
《仙乐传说动画片（THE
ANIMATION）》OVA第一章《西尔瓦蓝特篇》在2007年发售，作品一共四卷，发行时间跨度半年。OVA的第二章《泰瑟亚兰篇》在2010年3月开始发售，作品同样包含四卷，发行时间跨度一年。OVA的第三章《世界统合篇》在2011年11月底开始发行。

  
歌词下方是 _願い钢琴谱_ ，希望大家喜欢。
