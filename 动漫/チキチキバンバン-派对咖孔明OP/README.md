チキチキバンバン是电视动画《派对咖孔明》的片头曲，由音乐组合QUEENDOM演唱。据悉，此曲改编自匈牙利蹦迪神曲《Bulikirály》（中文《派对之王》）。

电视动画《派对咖孔明》改编自小川亮著作的同名漫画作品，于2021年11月17日宣布了电视动画化决定的消息。该片由P.A.WORKS负责动画制作，于2022年4月5日起播出。
