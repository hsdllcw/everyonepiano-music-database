

**暁双手简谱** 和五线谱完全对应。

_暁_ 是Studio
Pierrot改编的电视动画《晨曦公主》的片尾曲。《晨曦公主》中充满虚伪与真情的故事，围绕着少女的柔弱之心拉开大幕。本作品由Kazuhiro
Yoneda担任导演、猪爪慎一和吉川真帆分别负责系列构成与人设，再加上斋藤千和、前野智昭、森田成一、冈本信彦等人气声优的加盟，令动画在开播前就备受瞩目。此次志方晶子的歌曲，也必将进一步提升作品的气质。
