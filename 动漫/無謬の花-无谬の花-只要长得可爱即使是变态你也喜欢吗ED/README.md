_無謬の花_
是电视动画《只要长得可爱，即使是变态你也喜欢吗》的片尾曲。《只要长得可爱，即使是变态你也喜欢吗？》（日语：可愛ければ変態でも好きになってくれますか？；官方略称：変好き）原是由花间灯创作、sune插画的一部轻小说，后有了漫画、动画等衍生作品。动画由Lerche负责制作，于2019年7月8日起播出。

歌词下方是 _無謬の花钢琴谱_ ，大家可以免费下载学习。
