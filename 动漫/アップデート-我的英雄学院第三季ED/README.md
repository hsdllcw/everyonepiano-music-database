_アップデート_ 是电视动画我的英雄学院第三季的片尾曲。由日本著名创作型女歌手Miwa演唱。

《我的英雄学院》改编自日本漫画家堀越耕平原作的同名漫画，在漫画杂志《周刊少年JUMP》2015年49期上，发表了其TV动画化的决定。动画由Bones负责制作。讲述了一个天生无能力的少年绿谷出久追随憧憬的偶像英雄欧尔麦特，在以培养未来英雄为目的的雄英学院中求学、成长的故事。
