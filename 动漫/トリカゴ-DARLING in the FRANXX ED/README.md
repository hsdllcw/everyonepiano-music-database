DARLING in the FRANXX是由TRIGGER和A-1
Pictures共同制作的原创科幻电视动画，舞台设定在遥远的未来。于2018年1月13日起每周六在TOKYO
MX、栃木电视台、群马电视台、BS11首播。中国大陆由爱奇艺独家正版发布。

歌词下方是 _トリカゴ钢琴谱_ ，大家可以免费下载学习。
