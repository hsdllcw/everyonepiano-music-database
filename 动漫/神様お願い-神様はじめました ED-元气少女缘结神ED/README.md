

**神様お願い** 是TMS
Entertainment制作动画《元气少女缘结神》（日语：神様はじめました）的片尾曲。《元气少女缘结神》改编自日本漫画家铃木JULIETTA创作的同名漫画作品，于2012年10月开始连载了13话的TV内容，并与2013年8月推出了两话的ova内容，其中一话为原创剧情。

  
歌词下方是 _神様お願い钢琴谱_ ，希望大家喜欢。
