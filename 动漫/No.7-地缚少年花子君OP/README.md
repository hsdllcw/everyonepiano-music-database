_No.7_
是电视动画片《地缚少年花子君》的片头曲。由地缚少年乐队（生田鹰司、大石昌良、ZiNG）演唱。动画改编自Aidalro著作的同名漫画，于2019年3月宣布动画化的决定。该片由Lerche负责制作，于2020年1月9日起播出，全12话。
