

**明日、仆は君に会いに行く**
这首歌是日漫《世界第一初恋》的主题曲，这部动漫可以说是几乎所有腐女入腐必看的作品，而这首歌也是给大家封做“腐国国歌”。不过只有了解背景的人才会在听歌的时候一脸蜜汁微笑，毕竟这首歌单独听起来就是一首非常甜的恋爱曲，听完让人想谈恋爱的那种。

《世界第一初恋》是根据中村春菊创作的同名漫画改编的动画作品，由Studio
DEEN制作，于2011年4月8日首播。续作《世界第一初恋2》于2011年10月7日起播出。
