

《 **あっちでこっちで**
》是日本动画《一起一起这里那里》片头曲。由あっち匡挰愰（摘希（大久保瑠美）、伊御（冈本信彦）、姬（福原香织）、真宵（生天目仁美）、榊（浅沼晋太郎））演唱。

  

电视动画《一起一起这里那里》是由异识漫画原作、AIC负责动画制作、一部讲述傲娇女主角御庭摘希与天然呆男主角音无伊御温馨恋爱日常的动画作品。电视动画在2012年4月5日开始放送，全13话。

  

歌词下方是 _あっちでこっちで钢琴谱_ ，欢迎大家使用。
