

**爱をとりもどせ**
是TV动画《北斗神拳》的片头曲。北斗神拳（日文：北斗の拳），日本格斗漫画，武论尊原著，原哲夫漫画，引起了巨大反响，成为《少年JUMP》杂志发行400万份的原动力之一。其单行本全球销量过亿，成为20世纪日本最有影响力的漫画之一。

  
歌词下方是 _爱をとりもどせ钢琴谱_ ，希望大家喜欢。
