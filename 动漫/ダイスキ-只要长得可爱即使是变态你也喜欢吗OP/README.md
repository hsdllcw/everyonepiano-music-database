_ダイスキ_
是电视动画《只要长得可爱，即使是变态你也喜欢吗》的片头曲。且此曲由日本的女性声优、歌手大桥彩香演唱。动画改编自花间灯著作的同名轻小说，于2019年2月20日宣布了动画化的消息。动画由Lerche负责制作，于2019年7月8日起播出。

歌词下方是 _ダイスキ钢琴谱_ ，大家可以免费下载学习。
