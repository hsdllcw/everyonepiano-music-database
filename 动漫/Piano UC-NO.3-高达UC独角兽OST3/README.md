

**Piano UC-NO.3双手简谱** 和五线谱完全对应，是EOP教学曲。

_Piano UC-NO.3_ 即高达UC独角兽OST原声集3。

高达独角兽原创音乐集OST包括Piano UC-
NO.3由泽野弘之先生担当配乐。曾经谱写过：《医龙》、《东京塔》、《向着阴天绽放》、《战国BASARA》等多部电视、电影、动画、游戏的OST。这次谱写高达独角兽(Gundam
Unicorn)的OST更是体现出深厚的音乐底力。并且坚持了泽野弘之一贯的真正的音乐，是发自内心、充满诚意的音乐创作路线。

《机动战士高达UC》，日文原名为《机动戦士ガンダムUC》是由福井晴敏先生执笔，安彦良和先生担当人设，カトキハジメ（KATOKI HAJIME，角木
肇这个名字他本人不承认)先生担当机体设定的于2007年所推出的以U.C.时代（宇宙世纪）为时代背景的小说。 本作时间轴设定在U.C.00
93年（夏亚叛乱，第二次新吉恩战争）与 U.C.0099年（月球危机）之间的U.C.96年。
本作以地球联邦军的绝密计划为主线，描述了为争夺“拉普拉斯之箱”而使宇宙间再掀战火的故事。
