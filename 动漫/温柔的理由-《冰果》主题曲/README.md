

《 **温柔的理由** 》（優しさの理由）是2012年4月新番电视动画《冰菓》的片头主题曲（OP），由日本知名女歌手ChouCho（ちょうちょ）演唱。

2012年5月2日，发行了同名专辑《 _温柔的理由_ 》中亦收录了这首歌，并作为主打歌推出。

《冰菓》，是由日本推理作家米泽穗信的古典部系列小说改编的电视动画，由京都动画制作，自2012年4月开始播出。冰菓也是该系列小说第一卷的标题。

歌词下方是 _温柔的理由钢琴谱_ ，希望大家喜欢。
