

**イノセント**
是由梅津泰臣和足立慎吾合作创作的原创动画伽利略少女的片尾曲。《伽利略少女》（ガリレイドンナ，Galileidonna），由2013年10月10日开始，在富士电视台的深夜动画节目《noitaminA》中播放。略称为《伽利女》（ガリドン）。

_  
イノセント_ 这首歌曲是由日本演唱团体earthmind演唱的。

  
歌词下方是 _イノセント钢琴谱_ ，希望大家喜欢。
