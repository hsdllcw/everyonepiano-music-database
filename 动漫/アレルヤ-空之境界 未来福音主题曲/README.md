

《 **アレルヤ** 》是剧场版动画《空之境界 未来福音》主题曲。由Kalafina演唱。

  

「空之境界
未来福音」是根据奈须蘑菇以及武内崇推出的传奇小说、漫画所改编的剧场版动画，主要内容是在98年8月，围绕黑桐干也与有着预测型未来视能力的少女濑尾静音的偶遇，以及两仪式与拥有测定型未来视能力的炸弹魔仓密目琉夏的对决而展开。动画于2013年9月28日在日本上映。

  

Kalafina（カラフィナ），剧场版动画《空之境界》主题曲的演绎者，所属唱片公司为SME Records，事务所为Space Craft
Group。知名音乐制作人梶浦由记在担任《空之境界》背景音乐过程中，同时担当起了发掘全七章主题曲演唱者的工作，这项音乐企划被命名为“Kalafina
Project”，那些被选中的歌姬们组成的团体就叫“Kalafina”。

  

歌词下方是 _アレルヤ钢琴谱_ ，欢迎大家使用。
