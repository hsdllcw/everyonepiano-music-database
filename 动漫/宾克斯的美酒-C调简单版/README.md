

**宾克斯的美酒** 源于日本动漫《海贼王》，是一首源于新世界的著名船歌，从水手到海贼王都将它再三传唱。

伦巴海贼团（布鲁克复活前所在的海贼团）全军覆没时，全船船员在倒下前合唱了这首“宾克斯的美酒”，布鲁克将其录在音贝里面，希望寄养在双子峡的拉布能听到。四重奏，三重奏，二重奏，独奏……成员相继倒地……布鲁克因是黄泉果实的能力者可以死而复活，但复活时因为魂魄迷路，等找到尸体时已仅剩白骨，只能以一具骷髅的形态复活。

这首“海贼之歌”，且保持了平常心态。整个作品并没有其他动漫作品的原声那么华丽和高调，但却完美的诠释了一种“海贼精神”。歌词内容让人心潮澎湃，热血沸腾，表达了男子汉不畏艰难险阻，乐观豁达的精神，是再为合适不过的海贼之歌了。同时也是伦巴海贼团船长最喜欢的一首歌。
