_One Day Sky_ 出自《游戏人生ZERO》剧场版。由日本歌手铃木このみ演唱。《NO GAME NO LIFE 游戏人生
ZERO》是一部改编自轻小说《NO GAME NO LIFE
游戏人生》、由MADHOUSE制作的剧场版动画，于2016年7月17日在MF文库J夏之学园祭上宣布了制作决定的消息。剧情改编自原作小说第六卷，讲述了六千多年前不为人知晓的故事。
