

**Here双手简谱** 和五线谱完全对应。

_Here_ 是电视动漫魔法使的新娘的片头曲。电视动画《魔法使的新娘》由WIT
STUDIO制作，于2017年10月8日起播出。作品改编自ヤマザキコレ创作的同名漫画。全24话。

另外，此曲由JUNNA演唱。JUNNA（境纯菜）是2000年11月2日出生于日本爱知县的女性歌手，隶属于キューブ。

歌词下方是 _Here钢琴谱_ ，大家可以免费下载学习。
