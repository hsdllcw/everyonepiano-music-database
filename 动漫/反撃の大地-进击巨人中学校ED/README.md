

**反撃の大地双手简谱** 和五线谱完全对应。

_反撃の大地_ 是Production
I.G改编的电视动画《进击！巨人中学校》的片尾曲。该动画改编自谏山创原作、中川沙树负责作画的《进击的巨人》的同名衍生漫画。2015年7月31日发表了其TV动画化的决定。配音阵容跟电视动画《进击的巨人》相同。电视动画于2015年10月3日首播。另外，此曲由梶裕贵、谷山纪章、石川由依from
attackers演唱。

歌词下方是 _反撃の大地钢琴谱_ ，希望大家喜欢。
