

**透明な翼双手简谱** 和五线谱完全对应。

_透明な翼_ 是TRIGGER制作的电视动画作品小魔女学园的片尾曲。由日本歌手大原ゆい子演唱。

动画区别于2015年10月9日上映的同名剧场版动画，于2017年1月8日在TOKYO
MX首播。主要讲述了小时候看过闪亮夏莉欧的魔法表演后，梦想成为魔女的敦子，进入了与莎莉温相同的传统魔女育成名门校“新月魔法学校”。新的城镇，新的学校生活，以及新遇到的朋友。魔女学校中的扫帚飞行课程以及充满魔法气息的不可思议的授课等等，在这些只有魔女学校中才有的课程当中，敦子与洛蒂、苏西等人引发了大骚动！将新月卷入其中，敦子成为魔女的第一步就此开始——
