### 哀迷们对《哀のテーマ》（哀的主题曲）的注释：

这是一首歌曲  
没有歌词的配乐  
虽然我不懂日文的意思  
但是光听音乐 我还是可以感觉的出来  
这是《名侦探柯南》中专属于【灰原哀】的主题曲  
喜欢【灰原哀】不是一时的心血来潮  
喜欢【灰原哀】由来已久  
只是以前没有像现在如此痴迷 喜爱一个漫画中的人物  
《名侦探柯南》这部动漫 我之所以会这么喜欢看  
剧情其实不是我特别喜欢的  
我追看的 其实是灰原哀  
我痴迷的其实是灰原哀的人  
痴迷她的天赋 痴迷她的特长  
痴迷她的美丽 痴迷她的魅力  
痴迷她的个性 痴迷她的倔强  
痴迷她坚强外表下的一颗脆弱受伤的心灵  
痴迷她明明喜欢新一却又不想破坏新&兰之间的感情  
而坚强的把这份喜欢暗藏在心里不愿让任何人知道  
小哀她可以为了心爱的新一  
明知道一旦新一恢复大人的样子时就会回到兰的身边  
却还是倔强的要为他研制出APTX4869的解药  
她宁愿一个人独自默默承受着孤独与寂寞  
来自黑暗组织的压力与恐惧  
以及失去世界上所有亲人的痛苦  
一直在顽强的支撑的就是不愿告诉她暗恋新一的事  
只是为了  
不想让新一因此背上心灵的包袱
