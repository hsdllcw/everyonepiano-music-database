

**冷たい部屋、一人**
是日本2012年原创动画《K》的片尾曲。《K》，是名为“GoRA”的七名作家团体原著、动画公司GoHands制作的原创电视动画。2012年10月4日开始播放。海外Animax
Asia与日本零时差同步播出，另外会剪掉裸露镜头，此举非但使剧情中某些片段消失，甚至造成大部份集数没有片尾，只留下下集预告。

_  
冷たい部屋、一人_ 这首歌曲由日本女演员、声优、歌手小松未可子演唱。小松未可子，所属事务所Hirata Office，偶像团体いもうと前成员之一。

  
歌词下方是 _冷たい部屋、一人钢琴谱_ ，希望大家喜欢。
