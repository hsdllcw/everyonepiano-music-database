

**我们的live与你的life双手简谱** 和五线谱完全对应。

_我们的live与你的life_ 是由畑亚贵填词，山田高弘谱曲，高田晓编曲，μ's演唱的一首歌曲，收录在2010年发行的出道专辑《仆らのLIVE
君とのLIFE》中，是该专辑的同名主打曲。在动漫《LoveLive!》第8话作为μ's集齐九人后的第一首演出歌曲出现。

歌词下方是 _我们的live与你的life钢琴谱_ ，希望大家喜欢。
