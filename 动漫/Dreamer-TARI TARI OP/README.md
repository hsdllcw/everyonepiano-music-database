

**Dreamer** 是以神奈川县的江之岛为舞台的原创动画TARI TARI的片头曲。TARI TARI，由P.A.
Works制作，2012年7月1日开始放送。该片是P.A.Works继“真实之泪”、“花开物语”之后的青春群像剧第三弹。

  
另外，日本女歌手AiRI 演唱了这首 _Dreamer_ 。AiRI，隶属于Lantis。曾经使用过“UR@N”的名义发表作品。

  
歌词下方是 _Dreamer钢琴谱_ ，希望大家喜欢。
