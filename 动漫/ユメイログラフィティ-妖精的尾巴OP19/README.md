

**ユメイログラフィティ**
是东京电视台出品的动画作品《妖精的尾巴》的第十九首片头曲。动画从2009年10月12日开始，在东京时间每周六的上午10时30分放送，于2013年3月30日暂时完结，共放送175集。重播版于2013年4月4日起每周四下午5:00在日本电视台播放。据《周刊少年Magazine》2014年4•5号合并刊消息，动画新系列已于2014年4月5日开播。

_ユメイログラフィティ_
该曲由タッキー&翼演唱。タッキー&翼，中文名为泷与翼，是一个日本双人组偶像团体，隶属于杰尼斯事务所。于2002年9月11日正式出道。泷与翼每年均有为对方互相庆祝生日的习惯，他们表示此为泷&翼与其他杰尼斯团体的不同之处。而每年的生日庆祝规模也逐年越变越奢华。

歌词下方是 _ユメイログラフィティ钢琴谱_ ，希望大家喜欢。
