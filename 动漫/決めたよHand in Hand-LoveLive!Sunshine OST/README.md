_決めたよHand in Hand_
是日升动画制作的电视动画《LoveLive!Sunshine!!》的插曲，由高海千歌（伊波杏树）、樱内梨子（逢田梨香子）、渡边曜（齐藤朱夏）演唱。动画是二次元偶像企划中的一环，消息发表在2016年1月11日的“Love
Live!Sunshine!! AqoursNICO直播课外活动～初次DOKI-
DOKI直播！～”上。主要讲述了以高海千歌为首的九位少女以校园偶像Aqours的身份，以光芒为目标勇往直前的故事。
