

**Territory双手简谱** 和五线谱完全对应。

_Territory_ 是电视动画《百合熊风暴》的片尾曲，由百合城银子（CV：荒川美穗）、百合咲露露（CV：生田善子）、椿辉红羽（CV：山根希美）演唱。

《百合熊风暴》是2015年1月开始播出的原创电视动画，由执导过《少女革命》、《回转企鹅罐》等作品的几原邦彦担当监督。

歌词下方是 _Territory钢琴谱_ ，希望大家喜欢。
