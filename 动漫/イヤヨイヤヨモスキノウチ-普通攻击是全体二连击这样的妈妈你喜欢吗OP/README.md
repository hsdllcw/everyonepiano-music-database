_イヤヨイヤヨモスキノウチ_ 是Spira
Spica为电视动画《普通攻击是全体二连击，这样的妈妈你喜欢吗？》演唱的片头曲。动画改编自井中だちま著作，饭田ぽち。负责插画的同名轻小说作品，于2018年10月21日在“Fantasia文库大感谢祭2018”上宣布动画化的决定。动画由J.C
STAFF负责制作，于2019年7月12日起播出。

歌词下方是 _イヤヨイヤヨモスキノウチ钢琴谱_ ，大家可以免费下载学习。
