

《 **孤勇者** 》作为《英雄联盟：双城之战》中文主题曲，是去年国内唯一破圈的二次元歌曲，由陈奕迅演唱。

战吗？战啊！  
以最卑微的梦，  
致那黑夜中的呜咽与怒吼，  
谁说站在光里的才算英雄。

这是一首令人热血沸腾、充满力量的励志歌曲。然而就是这样一首歌，却成为了广为流传的儿歌。可能陈奕迅本人也想不到吧。

为什么《孤勇者》变成了儿歌？这首歌无论从旋律还是歌词上，都跟儿歌搭不上半点关系，但为什么孩子却喜欢唱，而且是全国小孩子的统一喜好呢？

首先，是词曲以及演唱者的功劳。陈奕迅独特的嗓音，再配合上唐恬和钱雷等一些外界因素的助力，成就了《 _孤勇者_ 》的争相传唱，成就了它的辉煌。

再者，疫情当下，许多人感受到了祖国的庞大力量。《孤勇者》不论是歌词还是曲调，都能够激发人们的爱国情怀，很有感染力。

还有，孩子们喜欢这首歌，可能与歌曲是动画剧集《英雄联盟•双城之战》的主题曲也有关系。同时，每个孩子尤其是男孩子都有英雄梦，这首歌曲就塑造孤孤胆英雄的形象，深入人心。
