

《 **世界第一的公主殿下** 》（ _World is Mine_
。ワールドイズマイン。）是日本虚拟偶像初音未来（初音ミク，中文界部分人简称为“初音”）的其中一首知名曲。

该曲诞生于2008年5月20日，收录专辑：《supercell》 番号： MHCL-1493。

相信初音迷应该会喜欢这首曲子的。歌词下方提供了 _世界第一的公主殿下钢琴谱_ ，欢迎大家使用。
