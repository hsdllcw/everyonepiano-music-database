

相信很多人都看过宫崎骏的《 **千与千寻** 》，甚至，有的人听见这歌曲旋律，可能说不出歌曲名字，但一定会知道是《 _千与千寻_ 》的主题曲。

原曲是女声版本，由木村弓演唱，是木村弓个人第一首音乐单曲作品，于2001年7月18日发行。她的声音高亢、明亮，又蕴藏着温柔和美，这种力量甚至超越了语言的隔阂。每次听，都能从中获得在别处所不能有的慰藉，每次听，能从那优美旋律获得力量。
