《幽灵公主》是吉卜力工作室于1997年推出的宫崎骏动画电影 _幽灵公主主题曲_
。宫崎骏在《幽灵公主》中探讨对环境的破坏，和人是否能够与自然真正和平共处等问题。《幽灵公主》的背景设定在日本的室町时代，这是中古时代进入近代的一段转变期，征夷大将军的权势大幅的衰弱。《幽灵公主》的配乐与片尾曲《もののけ姫》都是由久石让所创作，其中《もののけ姫》的CD在日本销售量超过110万张。《幽灵公主》在日本的票房高达193亿日圆，共有1,420万人次观看，在当时是日本史上票房最高的电影。

影片没有通常的大团圆结局，阿希达卡认识到战争还远远没有结束，他对珊说的那句：“一起活下去吧”，可以看出宫崎骏在宣扬的理想没有答案的情况下所表现出的无奈和期待。宫崎骏没有指出一个光明的未来，他清醒地认识到人与自然难以和谐共处。

当然，这部影片中，每一个角色都蕴含着对“自由”和“生存”的向往。如果不怀有这样的思想，就没有了《幽灵公主》这部20世纪伟大的动画史诗电影。可以说的确是一部回味无穷的动画。
