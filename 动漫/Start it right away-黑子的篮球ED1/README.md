

《 **Start it right away** 》是日本动画《黑子的篮球》ED。由ヒャダイン演唱。

  

黑子的篮球（黒子のバスケ）改编自藤卷忠俊创作的同名漫画，于2012年4月开播第一季，由Production
I.G担纲动画制作。动画第一季共25集现已完结。第二季已经于2013年10月开播。动画获得2013年东京动画奖所颁发的电视动画部门优秀奖。动漫讲述“梦幻的第六人”黑子哲也和从美国归来的天才篮球选手火神大两人一起打败“奇迹的世代”成为日本第一的故事。

  

前山田 健一（1980年7月4日－），即ヒャダイン（HYADAIN），大阪出身的作词家、作曲家、编曲家，音乐制作人，创作歌手，MC，艺人，隶属Five
Eighth（株式会社ファイブエイス），ヒャダイン名义则为Stardust
Promotion所属。作品以动画音乐和偶像歌曲为主，因以“ヒャダイン”名义在NICONICO动画投稿大量作词编曲并演唱的作品而相当有名气，同时也为大量偶像和偶像团体提供乐曲。写歌时使用本名，演唱和主持等使用ヒャダイン这个名字。

  

歌词下方是 _Start it right away钢琴谱_ ，欢迎大家使用。
