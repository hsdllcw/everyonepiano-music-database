

**永遠のAria** 是《七大罪 愤怒的审判》TV动画片头曲，由日本歌手雨宫天演唱。

DEEN负责制作，Marvy Jack协助制作，于2021年1月6日起在东京电视台播出，中国大陆由bilibili独家正版发布。
