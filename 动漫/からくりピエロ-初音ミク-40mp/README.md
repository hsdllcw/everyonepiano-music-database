

「 _からくりピエロ_ 」是40㍍P作曲的初音ミク演唱的歌曲。收录在专辑《小さな自分と大きな世界》之中。  
  
2012年6月15日销量达到100万张。是VOCALOID原创歌曲销量破百万的众多歌曲之一。  
  
歌词的下方还提供了 **からくりピエロ钢琴谱** ，欢迎感兴趣的朋友下载使用。
