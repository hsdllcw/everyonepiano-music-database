FISH”这个词。之后在1985年，只手支配纽约著名的暗黑街道上的不良少年的，是一位具有IQ200的少年亚修。他和一个来自日本的男孩奥村英二相遇，就在这时，与BANANA
FISH有关的黑势力也悄然而至……
