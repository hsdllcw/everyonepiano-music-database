

2021年11月8日，陈奕迅携手英雄联盟，献唱其首部动画剧集《英雄联盟：双城之战》中文主题曲《 **孤勇者**
》上线。歌曲搭配陈奕迅独特的气声与深厚的声线，讲述出动画中双城英雄那些真实、厚重的故事。

以平凡英雄、背后英雄的角度开展创作。陈奕迅在演唱中注入了深厚情感，感同身受一般唱出了“小人物”的呐喊——每一个孤独的英雄，不管身处何种境地，都能奋力为自己而战，为梦想而战。当今乐坛普遍飙高音，中低音的歌手能够拥有如此演绎能力的着实不多。而《双城之战》围绕英雄成长之间的磨炼和蜕变，让陈奕迅成了演唱主题曲的最佳人选。
