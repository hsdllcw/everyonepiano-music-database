

**TERMINATED**
是根据川上稔创作的同名轻小说改编的动画第一季《境界线上的地平线》的片头曲。《境界线上的地平线》由SUNRISE制作，小野学担任监督，浦畑达彦担任系列构成，于2011年2月宣布动画化，2011年10月1日开始放送，至12月24日放送完毕，共13集。

_  
TERMINATED_
此曲由茅原实里演唱。茅原实里，日本女性声优、歌手。枥木县宇都宫市出生，埼玉县成长。所属事务所为“M-Peace”，唱片公司为Lantis。
