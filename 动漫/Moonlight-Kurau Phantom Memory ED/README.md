

**Moonlight** 是日本动画公司BONES于2004年制作的动画Kurau Phantom
Memory的片尾曲。此片没有华丽的画面，没有美丽的人设，有的只是优秀的剧情和浓浓的人情味•通过主角和"对"所经历的事情，体现了亲情的魅力。

  
歌词下方是 _Moonlight钢琴谱_ ，希望大家喜欢。
