《风之甬道》是日本著名配乐大师久石让为电影《龙猫》写的一段配乐，被改编成了许多不同的版本，有合唱版，有钢琴版，也有乐队版。

《风之甬道》有着日本乡间祈福歌谣风格的作品。音乐以鼓声与木琴的丁冬声为基点，佐以童声那情绪饱满的哼唱，令传说故事中那可爱又神奇的精灵邻居，一时间栩栩如生。那感觉既纯真、又神圣，象是一群孩子很庄严地呼唤对自我梦想世界的尊重。
