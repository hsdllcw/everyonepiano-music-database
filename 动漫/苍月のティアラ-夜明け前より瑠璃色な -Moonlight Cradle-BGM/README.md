《苍月のティアラ》是冒险游戏《夜明け前より瑠璃色な -Moonlight Cradle-》中的背景音乐。  
  
《夜明前的琉璃色》（夜明け前より瑠璃色な）（直译应该是“更胜夜明前的琉璃色”，但使用率较低）是一个由August创作的十八禁恋爱冒险游戏，其后推出小说、漫画、动画系列，情节主要围绕一个来自月球的公主在地球家庭寄宿的故事。  
  
其中游戏系列之PS2版《夜明け前より瑠璃色な -Moonlight Cradle-》中的音乐《苍月のティアラ》是一首很好听的曲子。
