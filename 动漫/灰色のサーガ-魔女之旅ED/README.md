_灰色のサーガ_
是电视动画片《魔女之旅》的片尾曲，由日本女性歌手ChouCho演唱。动画《魔女之旅》改编自白石定规著作的同名轻小说。2019年10月19日，在“GA FES
2019”活动上宣布了动画化的决定。动画由C2C负责制作，于2020年10月2日至12月18日播出，全12集。

歌词下方是 _灰色のサーガ钢琴谱_ ，大家可以免费下载学习。
