

《 **裏切りの夕焼け** 》是日本动画《无头骑士异闻录》片头曲。由日本乐队THEATRE BROOK演唱。

《无头骑士异闻录》：改编自成田良悟创作的同名轻小说，2009年宣布动画化，2010年1月开始播放，由Brain's
Base担任制作，现已完结。2014年宣布第二季制作决定。

歌词下方是 _裏切りの夕焼け钢琴谱_ ，欢迎大家使用。
