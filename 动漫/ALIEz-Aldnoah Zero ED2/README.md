

**ALIEz** 是继动漫《罪恶王冠》的拔剑神曲βίος之后，泽野弘之次祭出混合语言+战争音乐艺术的组合，在2014年新番《ALDNOAH
ZERO》中的ED2。受到漫迷的广泛好评与追捧，其火爆程度甚至超过了同期的拔剑神曲！aLIEz尚未发售，就如雨后春笋般地出现了大量爱好者们的各种乐器演奏版本。由于配乐最初出现在第一话火星骑士伴随着爆炸开始接近地球，因此通常赞其名为“降落神曲”或“核爆神曲”。
