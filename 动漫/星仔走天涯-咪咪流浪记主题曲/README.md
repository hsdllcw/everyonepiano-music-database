**_星仔走天涯_ 此曲是日本动画《咪咪流浪记》的主题曲。此曲表达了对爸爸的爱，无畏艰难，一定要找到爸爸。《咪咪流浪记》
原名：《家なき子レミ》（意为：无家可归的孩子Remi ） 。又名《星仔走天涯》。改编自法国作家耶克特·马洛（Hector Malot）于1878年出版的小说
《苦儿流浪记》Sans famille ，于1977年在日本被改编成动画。
讲诉了八岁的小男孩——咪咪接受命运的安排一夕之间告别童年，带着身世之谜走上流浪艺人的酸甜苦辣之路。
