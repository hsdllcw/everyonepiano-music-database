是DracoVirgo为电视动画《平凡职业造就世界最强》演唱的片尾曲。动画改编自白米良著作的同名轻小说，于2017年12月2日宣布了动画化的消息，由asread和WHITE
FOX负责制作，于2019年7月8日起每周一在AT-X首播。
