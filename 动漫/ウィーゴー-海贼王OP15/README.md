

**ウィーゴー** 是东映改编的电视动画海贼王的片头曲。《ONE
PIECE》（海贼王、航海王）动画，于1999年10月20日在日本富士电视台开始播放，至今仍在播放中。中国搜狐视频网站拥有该片中国大陆地区独家版权，每周日10：00更新一集，截止至2014，点阅率已超过40亿次。
