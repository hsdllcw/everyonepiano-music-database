

**橙** （"Daidai", "Bitter Orange"）是以漫画《BLEACH》为原作，Studio
Pierrot制作的同名电视动画死神的片尾曲。《BLEACH》（死神）2004年10月5日至2012年3月27日于东京电视台系列21个电视台播放。系列构成当初为一系列约20话的架构，由于DVD出版同时进行之故，导致剧情快速发展，超越原作单行本的发行速率，因此播放部分动画原创剧情，而展开长达1年的故事。同时DVD的构成重新规划，原先一系列20话的构想也被改变。除此之外，日本地区在每集动画的片尾曲结束后，还会附加长达30秒的“死神黄金图鉴”。  
_  
橙_ 这首歌曲是由日本二人组女子摇滚乐队 Chatmonchy演唱的。Chatmonchy,据说中文译名是恰萌奇，2004年起活耀于地下乐团活动，推
出过多张独立创作专辑如‘我想成为恰萌奇’（已绝版）等等，累积固定人气。
