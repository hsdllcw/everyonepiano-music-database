_Q-vism_
是科幻动画名作《心理测量者》第3季的片头曲。《心理测量者》是由虚渊玄编剧的经典动画，自从第三季消息公开以来，就引起了不少漫迷的关注。“防患于未然”，这个恒久不变的话题就是《心理测量者》的主题思想，在人类的心理状态和性格倾向都能被数值化的未来。所有的感情、欲望、社会病态心理倾向等全部被记录并管理，大众以“好的人生”作为目标，竭力于数值性地实现它。
