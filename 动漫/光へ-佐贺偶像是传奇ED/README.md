_光へ_
是电视动画《佐贺偶像是传奇》的片尾曲。动画水准精湛、音乐引人入胜、再加上独特的话题性，让《佐贺偶像是传奇》成为了该作播出季度动画中的黑马。另外，此曲由フランシュシュ演唱。フランシュシュ包含源咲良（本渡枫）、二阶堂咲（田野麻美）、水野爱（种田梨沙）、绀野纯子（河濑茉希）、夕雾（衣川里佳）、星川莉莉（田中美海）。
