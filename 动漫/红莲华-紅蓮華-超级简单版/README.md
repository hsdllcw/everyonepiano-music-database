《 **红莲华**
》（日语：紅蓮華）是日本女歌手LiSA演唱的一首歌曲。同时，此曲也是电视动画《鬼灭之刃》的片头曲。此曲深受大家喜爱，所以小编特意制作了超简单版。

红莲华是一朵盛开在水面上的红色莲花，在寒冷的皮肤上绽放的红莲地狱。是LiSA读过原作漫画后，将作品世界观融入歌曲中创作歌词，歌词包含了对炭治郎面对着应该保护东西而变强的思念，而电视动画版的歌词与原版之间有些许的不同。

《 _红莲华_
》的下载数在2019年9月突破30万次，获得日本唱片协会认证为“白金”。该歌曲的下载数其后分别在2020年2月和同年6月突破75万次和100万次，分别获得日本唱片协会认证为“三白金”和“百万”。而LiSA也凭借《红莲华》一曲于第70回NHK红白歌合战初次登上红白舞台。
