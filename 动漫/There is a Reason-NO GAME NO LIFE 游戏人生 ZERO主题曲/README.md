

**There is a Reason** 是2017年7月15日剧场版《NO GAME NO LIFE 游戏人生
ZERO》的主题曲，由鈴木このみ演唱，岩里祐穗作词，ヒゲドライバー作曲。

这首曲子是利库（松冈）的视点，表达了对朱碧（爱衣）思念。象征利库从遇到朱碧以前苟且偷生的绝望，到两人相遇向世界发起挑战再到最后把游戏交付给未来的人类，可说是第六卷的缩影。

《NO GAME NO LIFE 游戏人生 ZERO》是一部改编自轻小说《NO GAME NO LIFE
游戏人生》、由MADHOUSE制作的剧场版动画，于2016年7月17日在MF文库J夏之学园祭上宣布了制作决定的消息。剧情改编自原作小说第六卷，讲述了六千多年前不为人知晓的故事。
