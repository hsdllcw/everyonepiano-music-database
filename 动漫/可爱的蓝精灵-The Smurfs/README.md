

《蓝精灵》在1986年的暑假就已经开始在北京电视台播出了，全国各地方电视台播出本片的时间应该也在此前后。动画片的片头就是这首主题曲。它节奏明快，富于动感；旋律轻盈、优美、动人，深得儿童们的喜爱。
