

**恋西游**
是TFBOYS于2015年4月14日发行的歌曲，是一首由方文山作词、毕晓迪作曲的一首中国风歌曲，是《梦幻西游之化境飞升》的主题曲，也是TFBOYS首唱的中国风歌曲。

据悉，此次TFBOYS还与国际知名摄影大师陈漫联手，为《梦幻西游2》拍摄了一组大片，首次尝试古装，COS游戏中三大人气角色——剑侠客、龙太子和逍遥生。三大角色性格各不相同，却同样坚毅、勇敢，在降服蚩尤的坎坷征途上相互砥砺，相互帮助，结下了深厚的友谊。而TFBOYS的三位成员王俊凯、王源和易烊千玺也情同兄弟，再加上他们英气逼人、青春健康的外型，自然将剑侠客、龙太子和逍遥生三大角色扮演的惟妙惟肖。
