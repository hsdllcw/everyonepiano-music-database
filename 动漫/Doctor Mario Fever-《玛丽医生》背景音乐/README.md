

《 **Doctor Mario Fever** 》是游戏《玛丽医生》（Doctor Mario）中的音乐。

玛丽医生小游戏(英文名：Dr.Mario Vitamin Toss)
，玛丽医生是一款非常有趣的Teteris概念的战略游戏。这种健脑益智将使你做充分的思想工作，充分开发你的大脑，但同时也会给你带来很大的快感和乐趣。
