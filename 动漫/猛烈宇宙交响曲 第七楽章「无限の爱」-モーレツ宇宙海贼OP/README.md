《猛烈宇宙交响曲
第七楽章「无限の爱」》（猛烈宇宙交響曲・第七楽章「無限の愛」）是桃色幸运草Z的第7张单曲。于2012年3月7日由STARCHILD发售。当初的名称是“猛烈宇宙组曲・第七番“无限の爱””。2011年12月25日的“ももいろクリスマス2011
埼玉超级竞技场大会”中初次披露。该曲同时是日本动画《モーレツ宇宙海贼》片头曲。
