

**Fairytale双手简谱** 和五线谱完全对应。

_Fairytale_ 是剧场版动画《空之境界》主题曲，由Kalafina演唱。

《空之境界：未来福音》自改编同名轻小说，于2013年9月28日公映，DVD于2014年2月19日发售。

歌词下方是 _Fairytale钢琴谱_ ，希望大家喜欢。
