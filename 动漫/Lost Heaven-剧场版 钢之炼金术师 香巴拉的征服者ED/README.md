

**Lost Heaven** 是剧场版 钢之炼金术师 香巴拉的征服者的片尾曲。《剧场版 钢之炼金术师 香巴拉的征服者》，为接续钢之炼金术师
(动画)故事发展的剧场版作品。背景从炼金术世界搬到现实世界，时为1923年德国慕尼黑。

  
歌词下方是 _Lost Heaven钢琴谱_ ，希望大家喜欢。
