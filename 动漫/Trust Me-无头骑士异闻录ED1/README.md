

**Trust Me** 是Brain's
Base制作动画《无头骑士异闻录》的片尾曲。《无头骑士异闻录》根据由成田良悟创作、安田典生负责插画的同名轻小说改编而成的电视动画。2009年7月宣布动画化
，动画由Brain's
Base担任制作，于2010年1月开始在日本每日放送播放，共播出24话，另有第12.5话和第25话两话电视未播放，收录于DVD中，动画总全26话。

  
歌词下方是 _Trust Me钢琴谱_ ，希望大家喜欢。
