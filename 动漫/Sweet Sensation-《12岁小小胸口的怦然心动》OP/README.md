_Sweet Sensation_
是电视动画《12岁小小胸口的怦然心动》片头曲，由村川梨衣演唱。电视动画《12岁。～小小胸口的怦然心动～》改编自日本漫画家米田菜穗创作的少女同名漫画《12岁。》，动画化于《CIAO》2016年3月号发表。

动画描写了12岁的少女们的纯洁的烦恼和初恋。

歌词下方是 _Sweet Sensation钢琴谱_ ，希望大家喜欢。
