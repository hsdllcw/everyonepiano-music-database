

**Skyreach** 是WHITE
FOX制作的电视动画《斩•赤红之瞳！》的片头曲。《斩•赤红之瞳！》改编自タカヒロ原作、田代哲也作画的同名漫画。电视动画由WHITE
FOX制作，于2014年7月6日开始播放，全24话。

  
歌词下方是 _Skyreach钢琴谱_ ，希望大家喜欢。
