

**月光**
是胡彦斌作曲、编曲并演唱的歌曲，收录于胡彦斌2008年7月4日发行的专辑《音乐斌潮》中。该曲也是中国3D武侠动画系列《秦时明月》的片头曲和主题曲。

一首肝肠寸断的《 _月光_
》将听者拉回到战马嘶鸣，侠客纵横的年代。此曲属于R&B曲风，而且曲子琅琅上口，歌词韵律十足。轮回爱恨纠葛，又同有千军万马气势，加上胡彦斌百转千回的唱法，更给这首歌曲锦上添花。胡彦斌其出色的唱功和迷人的嗓音，完美演绎了该曲所表达的爱恨情仇，也征服了无数的喜爱《秦时明月》的听众。
