_骄傲的少年_
是由南征北战创作并演唱的歌曲，2015年10月，该作品成为爱国主义网络动画《那年那兔那些事儿》第二季片尾曲。2016年1月，该作品登上《中国好歌曲》第三季，同时作为2016年中国好歌曲总决赛开场曲展现给大家。

2017年4月，该作品被王俊凯、刘昊然、董子健、张一山、王大陆共同演绎，成为浙江卫视推出的真人秀节目《高能少年团》主题曲。
