

**変幻自在のマジカルスター**
是ProductionI.G制作动画黑子的篮球的片头曲。黑子的篮球（黒子のバスケ）改编自藤卷忠俊创作的同名漫画，于2012年4月开播第一季。动画第一季共25集现已完结。第二季已经于2013年10月开播。动画获得2013年东京动画奖所颁发的电视动画部门优秀奖。

  
歌词下方是 _変幻自在のマジカルスター钢琴谱_ ，希望大家喜欢。
