

**Fantasy** 是堀田由美的同名漫画改编的电视动画《棋魂》的片头曲。该由东京电视台、电通及Studio
Pierrot共同制作，2001年10月10日－2003年3月26日在东京电视台系列播放，分为两辑，总共75集。另外，2004年1月3日日本播出特别篇OVA《棋魂Special：迈向北斗杯之路》，香港于2005年3月首播、2006年4月重播。动画未涉及原著中北斗杯三国对战的剧情，仅点到北斗杯日本预选赛为止。

  
歌曲 _Fantasy_
是片瀬那奈演唱的。片瀬那奈片濑那奈（1981年11月7日－），是一位日本女演员及歌手，出身于东京都，经纪公司为研音。天蝎座、A型、身高172厘米，喜欢运动，读书及弹琴。专长：书道六段。

  
歌词下方是 _Fantasy钢琴谱_ ，希望大家喜欢。
