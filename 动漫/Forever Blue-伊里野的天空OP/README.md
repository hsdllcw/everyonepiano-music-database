

《 **Forever Blue** 》是OVA动画《伊里野的天空》片头主题曲。

是2005年02月发行，由小说全4卷改篇的OVA动画，原作作者秋山瑞人也是凭借着这部作品一举成名。讲述
暑假最后的那天夜晚，悄悄潜入学校泳池的浅羽直之与一位手腕上嵌有银色球体不可思议的少女伊里野加奈相遇。

歌词下方是 _Forever Blue钢琴谱_ ，欢迎大家使用。
