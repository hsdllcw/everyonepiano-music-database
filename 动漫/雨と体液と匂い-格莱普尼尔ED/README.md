_雨と体液と匂い_
是Mili为电视动画片《格莱普尼尔》演唱的片尾曲。《格莱普尼尔》改编自武田寸创作的同名漫画，于2019年3月6日宣布了动画化的消息。动画由PINE
JAM负责制作 ，于2020年4月5日起每周日在TOKYO MX、SUN电视台、KBS京都、BS11首播，中国由bilibili独家正版发布。

歌词下方是 _雨と体液と匂い钢琴谱_ ，大家可以免费下载学习。
