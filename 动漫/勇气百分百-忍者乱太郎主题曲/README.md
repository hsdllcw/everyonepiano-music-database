_勇气100%_ ，日语：勇気100%，是日本男子偶像团体光GENJI的第21张单曲。1993年5月13日发行，由松井五郎作词，马饲野康二作曲。  
  
同时，此曲也是1993年NHK开始播放的电视动画《忍者乱太郎》的主题曲，此后这首歌被多个杰尼斯后辈团体翻唱，并且长期作为《忍者乱太郎》的主题曲。  
  
另外，个人看来，此曲可是一首励志歌曲，鼓励大家勇于向前，面对挫折，困难要无所畏惧。
