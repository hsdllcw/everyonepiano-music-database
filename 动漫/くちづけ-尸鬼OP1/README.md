

**くちづけ**
是根据日本小说家小野不由美的恐怖小说，以及同名改编的动画作品《尸鬼》（しき）的片头曲。《尸鬼》的动画制作由动画公司“童梦”负责，预定全22话，从2010年7月8日起的每周四23：15分播出。

  
动画讲述了在一个封闭的小村庄外场村的村民在山上发现死尸后，村子的居民就一个个离奇的死去和搬走。敏夫医生和少年夏野发现了这些事情都是新搬来村子的尸鬼所为，经过一番斗争，村民最终冲到尸鬼的巢穴，赶走尸鬼。

  
另外，BUCK-TICK演唱了这首 _くちづけ_ 。Buck-Tick--
日本老牌视觉系乐队之一，这支乐队在1985年创建，成员包括主唱樱井敦司，吉他手今井寿、星野英彦，贝司手樋口豊和鼓手YAGAMIDORU。当时在日本是红极一时，现在也是视觉系乐队迷的心中之重，全体成员的年龄都已经四十岁以上，出道至2012年已经25周年。乐队曲风风格多变，不拘泥于一般视觉系的风格，作词、作曲、表演形式都自成一系，是乐队中的实力派。

  
歌词下方是 _くちづけ钢琴谱_ ，希望大家喜欢。
