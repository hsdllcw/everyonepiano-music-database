

《 **幸せ願う彼方から** 》日本动画电影《幸运☆星》（らき☆すた）泉彼方角色歌曲，由泉かなた （岛本须美）演唱。

  

《幸运☆星》是由美水镜的同名四格漫画改编的动画作品，由京都动画制作，并于2007年开始播出，共24集，而OVA于2008年9月26日发售。

  

歌词下方是 _幸せ願う彼方から钢琴谱_ ，欢迎大家使用。
