_竈門炭治郎のうた_
是椎名豪、中川奈美于2019年8月30日发行的歌曲。同时此曲也是电视动画《鬼灭之刃》的片尾曲。动画改编自吾峠呼世晴创作的同名漫画，于2018年6月4日在《周刊少年JUMP》2018年6月第27号上发表了动画化的消息。动画由ufotable负责制作，于2019年4月6日起播出，全26话。
