

《 **ココ** 》是日本动画《京骚戏画》片头主题曲。由日本女歌手たむらぱん演唱。

  

动画《京骚戏画》（京騒戯画（きょうそうぎが）原是Banpresto和东映动画首次合作的OVA作品，在推出第2弹后人气很高因此决定推出TV动画，动画内容是2话总集篇、1话特别篇，以及全新的10话TV内容和计13话，于2013年10月2日播出。

  

歌词下方是 _ココ钢琴谱_ ，欢迎大家使用。
