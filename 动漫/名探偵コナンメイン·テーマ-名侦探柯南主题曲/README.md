_名探偵コナンメイン•テーマ_
是动画名侦探柯南的主题曲。《名侦探柯南》是根据日本漫画家青山刚昌创作的侦探漫画《名侦探柯南》改编的同名推理动画作品系列。其中电视动画由V1
Studio制作，于1996年1月8日开始在日本读卖电视台播放，至今仍在播出。

另外，此曲由大野克夫创作。大野克夫（おおの かつお），1939年出生于京都市，日本音乐家。1962年，成为“the
spiders”（蜘蛛乐队）的键盘手。1977年，他的歌曲《随心所欲》获得日本唱片大奖。1981年，成立大野克夫乐队。
