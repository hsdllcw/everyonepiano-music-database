《 _見えないからね_
》，中文译为《我才看不见呢》，由雨宫天演唱，是电视动画片《阴阳眼见子》的片头曲。《阴阳眼见子》（日语：見える子ちゃん）原是日本漫画家泉朝树创作的漫画，最初于2018年在作者的个人Twitter发表，同年11月2日开始在漫画网站“ComicWalker”连载。2019年8月，获得下一部漫画大奖网络漫画第10名。改编电视动画于2021年10月3日起播映。2021年10月为止单行本发行量累计超过60万册。
