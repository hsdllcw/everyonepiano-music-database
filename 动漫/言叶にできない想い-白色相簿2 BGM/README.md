_言叶にできない想い_
是日本LEAF会社制作的十八禁恋爱冒险游戏《白色相簿2》的背景音乐。《白色相簿2》的故事发生在前作大约十年后，但人物关联性不大，即使对于完全没有接触过《白色相簿1》的观众也不会产生太大影响。
