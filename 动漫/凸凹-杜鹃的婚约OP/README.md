

《 **凸凹** 》是日本女歌手吉冈圣恵演唱的歌曲。同时，此曲也是电视动画片《杜鹃的婚约》的片头曲。

动画《杜鹃的婚约》改编自吉河美希著作的同名漫画作品，于2021年4月6日宣布动画决定的消息。该片由SHIN-
EI动画、SynergySP负责制作，于2022年4月23日起播出，全24集。

歌词下方是 _凸凹钢琴谱_ ，大家可以免费下载学习。
