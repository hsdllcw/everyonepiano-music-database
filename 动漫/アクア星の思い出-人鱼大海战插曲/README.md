**_アクア星の思い出_ 中文可译为《水波星的回忆》出自沢田完之手。《アクア星の思い出》
是哆啦A梦《人鱼大海战》配乐，这个背景音乐旋律非常优美，常被网友们用着手机铃声。  
  
沢田完（1968年4月9日－），作曲家一名，于东京都出身。东京音乐大学中途退学。现在为广泛种类的音乐作曲而出名，很多音乐，如连续剧、电影和舞台剧也尝试过。2005年开始，更担任哆啦A梦的音乐担当者。
