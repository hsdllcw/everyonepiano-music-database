

**ふゆびより双手简谱** 和五线谱完全对应。

_ふゆびより_ 是电视动画摇曳露营△的片尾曲。由佐佐木惠梨演唱。

动画《摇曳露营△》改编自日本漫画家あfろ所创作的同名漫画，在2017年3月号《Manga Time Kirara
Forward》上宣布了动画化的消息。动画由C-Station负责制作，于2018年1月4日起 播出。

歌词下方是 _ふゆびより钢琴谱_ ，大家可以免费下载学习。
