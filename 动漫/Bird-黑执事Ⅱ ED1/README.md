

**Bird** 是电视动画作品《黑执事Ⅱ》（Black ButlerⅡ）的片尾曲。同时也是松下优也的音乐作品，收录在《U ~Best Of
Best~》专辑中。

《黑执事Ⅱ》（Black ButlerⅡ）是由日本动画公司A-1
Pictures制作的电视动画作品，于2010年7月1日播放，全12话，外加6话OVA。此作改编自漫画《黑执事》，剧情方面基本上全部为原创。中国大陆由bilibili独家正版。
