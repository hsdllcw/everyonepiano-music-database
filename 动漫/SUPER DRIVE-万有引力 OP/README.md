

**SUPER DRIVE**
是耽美动漫《万有引力》的片头曲。《万有引力》（日语：グラビテーション）是村上真纪的漫画作品。在《你与我》连载。另外还有电视动画、OVA、广播剧CD等作品。

  
第1部漫画全12卷。第2部漫画《万有引力EX》现在《Spica》Web连载中。

  
歌词下方是 _SUPER DRIVE钢琴谱_ ，希望大家喜欢。
