

《 **笑颜に会いたい** 》是日本动画《橘子酱男孩》（ママレード・ボーイ）主题曲。由滨田理恵演唱。

  

橘子酱男孩（ママレードボーイ，英文名Marmalade Boy）是由吉住涉绘制的日本少女漫画连载作品，后改编成动画。

  

歌词下方是 _笑颜に会いたい钢琴谱_ ，欢迎大家使用。
