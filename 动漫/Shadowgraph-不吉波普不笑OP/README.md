_Shadowgraph_ 是日本音乐组合Myth &
Roid为电视动画《不吉波普不笑》演唱的片头曲。动画改编自上远野浩平著作，绪方刚志负责插画的同名轻小说，于2018年3月宣布动画化，由Madhouse负责制作，于2019年1月4日起每周五在AT-
X首播。而Myth & Roid是由制作人Tom-H@ck为中心组成的一支年轻的现代音乐创作型组合。以首张单曲《L.L.L》于2015年8月26日正式出道。
