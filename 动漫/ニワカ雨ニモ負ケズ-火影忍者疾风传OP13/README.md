

**ニワカ雨ニモ負ケズ**
是电视动画火影忍者疾风传的片头曲。《火影忍者疾风传》是根据岸本齐史创作的漫画《火影忍者》的第二部所改编的动画，是动画《火影忍者》的续篇，于2007年2月15日开始在东京电视台播出。  
  
且 _ニワカ雨ニモ負ケズ_ 此曲由NICO Touches the Walls演唱。NICO Touches the Walls是2004年结成的J-
Rock Band，2007年进入Ki/oon后正式出道，2010年将经纪约签入A-Sketch。由光村龙哉 (Vo.&Gt.)、古村大介
(Gt.)、坂仓心悟 (B.)、対马祥太郎 (Dr.)4人组成。
