

这是 **金庸群侠传主题曲**
钢琴版。金庸群侠传是于1996年台湾发行的电脑游戏，由河洛工作室开发，智冠科技发行的DOS中文角色扮演游戏，是一款相当受欢迎的经典游戏。此作的成功亦引伸出很多不同的作品，影响深远。游戏是以金庸笔下的十四部武侠小说所改编，玩家可以与部份金庸小说人物共同冒险。游戏内自由度非常高，绝大部分的人物、武功、物品及剧情发展都十分忠于金庸原著。在游戏内，需要寻找“飞雪连天射白鹿，笑书神侠倚碧鸳”十四部天书，玩家可以任意结交各种金庸小说人物，要求他们加入队伍，一起冒险，前提就是主角需要符合一定的条件。当到了结局要打十大高手。

《金庸群侠传》的游戏出品商是河洛工作室，河洛工作室是原台湾智冠科技游戏开发组，后脱离智冠科技，加入东方演算，不久后东方演算解散。

河洛工作室在智冠科技开发制作《金庸群侠传》《武林群侠传》《三国群侠传》等游戏。其作品极具创意，提倡“高自由度”的特色。每一款都堪称华人单机RPG游戏史上的经典之作，其中以《金庸群侠传》、《武林群侠传》这两款游戏最深入人心。
