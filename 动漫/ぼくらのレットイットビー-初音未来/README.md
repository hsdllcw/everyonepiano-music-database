

《 _ぼくらのレットイットビー_ 》是はりー作词、作曲、编曲，初音未来演唱的歌曲。收录在专辑《Vocamelloid 7》之中。  
  
歌词的下方还提供了 **ぼくらのレットイットビー钢琴谱** ，欢迎感兴趣的朋友下载使用。
