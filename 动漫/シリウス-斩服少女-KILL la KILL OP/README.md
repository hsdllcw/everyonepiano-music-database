《斩服少女（KILL la
KILL（キルラキル））》是由中岛一基创作，今石洋之执导的热血奇幻原创电视动画。官方标语是“穿着、斩杀、饥渴、鬼化，以及生存，最后斩碎世界！”。动画以武力和恐怖支配的本能字学园为舞台，讲述了少女缠流子为寻找父亲仇敌，作为转校生来到本能字学园，与主要支配着该学园重要势力方的学生会会长鬼龙院皐月和学生会四大天王们战斗的故事。
