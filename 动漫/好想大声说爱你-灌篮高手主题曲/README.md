

**好想大声说喜欢你** （又名《无力去爱谁》） 为日本动漫《灌篮高手》的主题曲。由山田恭二作词，BAAD组合演唱。国人重新填词版《 _好想大声说喜欢你_
》这个版本曾经由老爹迪克牛仔演唱过。  
  
歌词下方是 _好想大声说喜欢你钢琴谱_ ，大家可以免费下载学习。
