YURI!!! on
ICE，中文译为冰上的尤里，是MAPPA以花样滑冰为题材制作的日本原创电视动画作品，于2016年10月6日开始播出。全12话。标题上的“YURI”指的是两个角色的名字，勇利和尤里（日语读音相同）。该作品曾获TAAF
2017（2017年东京动画奖盛典）动画迷奖。
