该曲由久石让创作的，久石让以优雅而美丽琴音表现小女孩千寻的内心世界，曲调细腻动人，又带着些许哀伤，让人心为之触动，似乎真的能感受到有什么总是伴随着我，心里会有种莫名的感动。  
  
《千与千寻》是一部由吉卜力工作室制作的日本动画电影，导演和编剧为宫崎骏。《千与千寻》于2001年7月20日在日本首映，创下约2350万观影人次和304亿日元的票房收入。《千与千寻》被誉为宫崎骏的《爱丽丝梦游仙境》。故事讲述的是一个小女孩在神灵异世界里发生的故事。《千与千寻》获得柏林国际电影节最高荣誉的金熊奖，截止2013年唯一一部获得金熊奖的动画电影。同时亦获得第75届奥斯卡最佳动画长片奖。主题曲《
_Always With Me_ 》也深受大家喜欢的。是一首旋律温暖，美好的治愈音乐。
