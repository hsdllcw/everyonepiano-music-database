_アザレア_
是电视动画Citrus的片头曲。动画改编自三郎太原作的同名漫画，于2017年3月25日宣布动画化。由Passione负责制作，于2018年1月6日起每周六22:30在AT-
X首播。中国大陆由bilibili独家正版。

另外，此曲由nano.RIPE演唱。nano.RIPE是日本千叶县出身的一支新生代4人POP、ROCK乐队。所属事务所为I
WILL，唱片公司为Lantis。

歌词下方是 _アザレア钢琴谱_ ，大家可以免费下载学习。
