

**Break through** 是东京电视台出品的动画作品妖精的尾巴的第十三首片头曲。 _妖精的尾巴_ 由A-1
Pictures和SATELIGHT负责动画制作，改编自真岛浩创作的同名魔法题材少年漫画。作品讲述了魔导士公会“妖精的尾巴”的成员露西、纳兹、哈比、格雷、艾露莎和他们的众多伙伴们在这个魔法世界中的一系列热血故事。动画从2009年10月12日开始，在东京时间每周六的上午10时30分放送，于2013年3月30日暂时完结，共放送175集。重播版于2013年4月4日起每周四下午5:00在日本电视台播放。据《周刊少年Magazine》2014年4•5号合并刊消息，动画新系列将于2014年4月5日开播！
