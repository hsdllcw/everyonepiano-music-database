

《 **カタストロフィ** 》是日本虚拟偶像歌手初音ミク和巡音ルカ演唱的歌曲。由いたちまP作詞，作曲。

初音未来（初音ミク，华人世界通常简称为“初音”）是CRYPTON FUTURE MEDIA以Yamaha的VOCALOID
2语音合成引擎为基础，开发贩售的虚拟女性歌手软件，角色主唱系列的第一作、VSTi（日语：Virtual Studio
Technology）规格的电子乐器；或此软件的印象角色（这只是软件的象征，不会在实际使用时出现）。2007年8月31日发售。原只可用于Microsoft
Windows，2008年3月19日随CrossOver Mac 6.1发表而可用于Mac OS
X，在GNU/Linux系统上通过修改过的wine也能够正确运行初音未来。发售后大受欢迎，并出现大量用户制作的翻唱歌曲、原创曲等。官方表示初音未来擅长由1980年代至最新的流行歌曲。

巡音ルカ（巡音露卡、巡音流歌、巡音流香）是CRYPTON FUTURE MEDIA以Yamaha的VOCALOID
2语音合成引擎为基础开发贩售的虚拟女性歌手软件角色主唱系列的第三作；或此软件的印象角色（这只是软件的象征，不会在实际使用时出现）。和前两作不同的是，CV03同时包括日语和英语的歌声数据库。

歌词下方是 _カタストロフィ钢琴谱_ ，欢迎大家使用。
