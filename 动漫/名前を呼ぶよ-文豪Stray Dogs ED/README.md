

**名前を呼ぶよ双手简谱** 和五线谱完全对应。

_名前を呼ぶよ_ 是电视动画文豪Stray
Dogs的片尾曲。电视动画《文豪野犬》改编自朝雾卡夫卡原作，春河35负责作画的同名漫画。于2016年3月27日，官方举办了动画第1话先行上映会。

名前を呼ぶよ由日本乐队Luck Life演唱。

歌词下方是 _名前を呼ぶよ钢琴谱_ ，希望大家喜欢。
