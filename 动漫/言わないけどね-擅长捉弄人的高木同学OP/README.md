

**言わないけどね。双手简谱** 和五线谱完全对应。

_言わないけどね。_ 是电视动画擅长捉弄人的高木同学的片头曲。由大原ゆい子演唱。

动画改编自山本崇一郎创作的漫画《擅长捉弄人的高木同学》、《明天是星期六》，由SHIN-
EI动画制作，于2018年1月8日首播。主要讲述了“今天一定要捉弄高木同学，让她害羞！”某所初中里，邻座女生高木同学总是捉弄男生西片。西片为了反过来戏弄高木同学，每天都在奋斗着……高木同学和西片的全力“捉弄”青春之战就此开始。

歌词下方是 _言わないけどね。钢琴谱_ ，大家可以免费下载学习。
