

**スタートライン** 是Studio
Comet制作的电视动画《凉风》的片头曲。2005年7月开播的电视动画《凉风》改编自日本漫画家濑尾公治创作的同名漫画《凉风》，由Studio
Comet担当动画制作，共26集。

  
歌词下方是 _スタートライン钢琴谱_ ，希望大家喜欢。
