是电视动画《我们无法一起学习》的片头曲。动画改编自筒井大志创作的同名漫画，于2018年8月27日在《周刊少年JUMP》2018年3月9号上宣布了动画化的消息，由st
Silver和ARVO ANIMATION共同负责制作，于2019年4月6日起每周六在TOKYO
MX、群马电视台、栃木电视台、BS11首播，中国大陆由bilibili独家正版发布，全13话。
