

**罪の名前双手简谱** 和五线谱完全对应。

_罪の名前_ 是来自日本歌手Ryo和初音未来的一首慢摇歌曲，很多人非常喜欢，请慢慢欣赏。

初音未来（初音ミク/Hatsune Miku），是2007年8月31日由CRYPTON FUTURE
MEDIA以Yamaha的VOCALOID系列语音合成程序为基础开发的音源库，音源数据资料采样于日本声优藤田咲。

歌词下方是 _罪の名前钢琴谱_ ，希望大家喜欢。
