_永远ラビリンス_
是动画我的女友是个过度认真的处女bitch的片头曲。动画改编自松本ナミル创作的漫画《我的女友是个过度认真的处女bitch》，在2017年6月9日发表了将制作电视动画的消息，由diomedéa
× Studio Blanc共同制作。

且此曲由悠木碧演唱。悠木碧，本名八武崎碧，1992年3月27日出生于日本千叶县。日本女性声优、演员、歌手。4岁时以童星身份进入演艺圈，2007年改成现在的艺名，现属Pro
Fit事务所。毕业于早稻田大学。
