_Muse_
是原创电视动画片《LISTENERS》的片尾曲。动画由MAPPA负责制作，于2020年4月4日起在MBS/TBS的“Animeism”栏目首播，全12集。该作以不存在被称为“音乐”的世界为舞台，讲述了一名少年与一名神秘的少女缪相遇后踏上旅途，满是交织摇滚音乐的青春故事。

歌词下方是 _Muse钢琴谱_ ，大家可以免费下载学习。
