

**Daydream Syndrome**
是J.C.STAFF制作电视动画《食梦者玛利》的片头曲。《食梦者玛莉》（日语：夢喰いメリー）是由牛木义隆所著作的日本漫画作品。漫画版由《Manga Time
Kirara Forward》（芳文社出版）杂志连载中。“REM○”为连载单位（其中○是话数的阿拉伯数字）。同名改编的电视动画于2011年1月开始播放。

  
歌词下方是 _Daydream Syndrome钢琴谱_ ，希望大家喜欢。
