

**海色**
是Diomedéa改编的电视动画《舰队收藏》的片头曲。《舰队收藏》改编自角川游戏开发、DMM.com营运的同名网页游戏。动画以全部3D效果的战斗场景为特色，于2015年1月7日开始播出。

  
歌词下方是 _海色钢琴谱_ ，希望大家喜欢。
