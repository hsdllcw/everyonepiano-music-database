

该曲是动画电影 **龙猫主题曲**
。《龙猫》（日语：となりのトトロ）是吉卜力工作室与德间书店于1988年推出的一部动画电影，由宫崎骏所执导。电影描写的是日本在经济高度发展前存在的美丽自然，那个只有孩子才能看见的不可思议世界和丰富的想像，因为唤起观众的乡愁而广受大众欢迎。《
_龙猫_ 》可以说是宫崎骏带给社会大众的一份恩泽，这部作品让人看到森林时，愿意打从心里相信トトロ确实就藏在里头，可说是世上少有的美妙体验。

迄今为止很少有影片能像《 _龙猫_
》一样重现无拘无束的童年画面，并准确捕捉到旧时的快乐情感。真的是一部单纯，平静而温馨的电影。而且适合很多年龄阶段的朋友欣赏。影片中说只有听话，善良的好孩子才能看得到可爱的龙猫噢。所以大家要从小做个好孩子哈~

_龙猫主题曲_ 是一首很可爱，很温暖的一首曲子。相信大家都会喜欢这首歌，也会从这首曲子中感受到很多童年的快乐和美好。

歌曲《豆豆龙》就是范晓萱翻唱的 _龙猫主题曲_ ，收录于1996年4月《小魔女的魔法书》。如果大家喜欢 _龙猫主题曲_ ，而且是钢琴爱好者的话，不妨下载
