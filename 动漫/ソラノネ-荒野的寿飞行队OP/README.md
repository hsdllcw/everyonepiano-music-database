_ソラノネ_
是ZAQ为动画《荒野的寿飞行队》演唱的片头曲。《荒野的寿飞行队》是由水岛努执导、横手美智子编剧、GEMBA负责制作的原创电视动画作品，于2018年9月10日宣布制作，于2019年1月13日至3月31日每周日晚在TOKYO
MX播出，全12集。讲述的是西部剧风格的世界背景下由6名少女组成的保镖集团“寿飞行队”驾驶活塞式飞机的空战日常故事。

歌词下方是 _ソラノネ钢琴谱_ ，大家可以免费下载学习。
