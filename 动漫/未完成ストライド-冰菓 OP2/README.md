

**未完成ストライド** 是京都动画制作的动画作品《冰菓》的片头曲。

  
〈冰菓〉系列（日语：〈古典部〉シリーズ），是日本作家米泽穗信所著、以“古籍研究社”（日语：古典部）为中心的系列推理小说，于2001年11月开始由角川书店出版至今。正体中文版由独步文化代理发行；简体中文版由天闻角川代理发行。2011年11月，京都动画宣布制作电视动画《冰菓》，并于2012年4月播出。

  
歌词下方是 _未完成ストライド钢琴谱_ ，希望大家喜欢。
