Lost Paintings简介：

_Lost Paintings（消失的油画）_ 是恶魔城月下夜想曲逆城的主题曲。

《恶魔城X：月下夜想曲》是日本KONAMI（科乐美）公司旗下游戏品牌《恶魔城》系列作品之一。1997年在SONY的PS主机上发行，翌年移植SEGA的SS平台，2007年再次登陆SONY手掌机PSP。《恶魔城X：月下夜想曲》是《恶魔城》系列公认的最成功的作品。《月下夜想曲》一改以往作品纯动作的风格加入大量RPG要素与地图探索元素，辅以精彩的剧情、精美的图像与动听的乐曲，因此广受好评。
