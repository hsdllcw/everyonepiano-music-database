《 _Shiny Tale_ 》是日本动画《男子高中生的日常》（男子高校生の日常）片头曲。由日本视觉摇滚乐队Mix Speaker's,Inc.演唱。

男子高中生的日常（男高日常）是由同名漫画改编的动画，是一部讲述男子高校的学生的日常故事的作品。于2011年的11月3日在niconico动画上先行放出第一集，2012年1月开始正式播放。

歌词下方是 _Shiny Tale钢琴谱_ ，欢迎大家使用。
