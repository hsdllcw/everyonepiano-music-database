

**Agape**
是电视动画《圆盘皇女》中的歌曲。《圆盘皇女》（日语：円盤皇女ワるきゅーレ）是日本漫画家介錯的一部漫画，《月刊少年GANGAN》2002年3月号至2007年9月号连载。单行本共11卷。4度改编动画。

  
歌词下方是 _Agape钢琴谱_ ，希望大家喜欢。
