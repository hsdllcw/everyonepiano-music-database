动画《守护甜心！》改编自日本漫画家组合PEACH-
PIT原作的同名漫画。在漫画杂志《好朋友》2007年7月号（6月1日发售）上，正式发表了《守护甜心！》TV动画化的决定。全系列共分为三期播放，第1期《守护甜心！》于2007年10月6日起播放，全51集。而第2期《守护甜心！！心跳》和第3期《守护甜心！派对！》则分别2008年10月4日和2009年10月3日起播放。
