

《 **Snow Fairy** 》是日本动漫《Fairy Tail》（妖精的尾巴）的片头曲。由FUNKIST演唱。

  

由东京电视台出品，A-1
Pictures和SATELIGHT负责动画制作的长篇电视动画《妖精的尾巴》改编自真岛浩创作的同名魔法题材少年漫画。作品讲述了魔导士公会“妖精的尾巴”的成员露西、纳兹、哈比、格雷、艾露莎和他们的众多伙伴们在这个魔法世界中的一系列热血故事。动画从2009年10月12日开始，在东京时间每周六的上午10时30分放送，于2013年3月30日暂时完结，共放送175集。重播版于2013年4月4日起每周四下午5:00在日本电视台播放。
