

《 **止マレ!** 》是日本动画《凉宫春日的忧郁》（涼宮ハルヒの憂鬱）ED，由剧中角色扮演者平野绫、茅原实里、后藤邑子演唱。

  

电视动画《凉宫春日的忧郁（2006年版）》是根据日本轻小说家谷川流原作、插画家伊东杂音负责插画的轻小说《凉宫春日系列》改编的动画作品。2006年版的电视动画全14集。
