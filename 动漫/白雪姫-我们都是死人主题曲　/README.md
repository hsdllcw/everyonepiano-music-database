

《 **白雪姫** 》是日本女子组合Flower演唱的，日剧《我们都是死人》（僕らはみんな死んでいる♪）主题曲。

  

Flower是日本女子8人舞蹈及歌唱组合。LDH所属。唱片公司是Sony Music Associated
Records。Flower成员：鹫尾伶菜、武藤千春、市来杏香、藤井萩花、中岛美央、重留真波、佐藤晴美、坂东希 已退出成员：水野绘梨奈（前队长）。

  

歌词下方是 _白雪姫钢琴谱_ ，欢迎大家使用。
