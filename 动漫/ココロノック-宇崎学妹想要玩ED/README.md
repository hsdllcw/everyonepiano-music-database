_ココロノック_
是日本虚拟歌手YuNi演唱的歌曲，同时也是电视动画《宇崎学妹想要玩！》的片尾曲。动画改编自日本漫画家丈绘制的同名恋爱漫画《宇崎学妹想要玩！》，故事情节以单元剧为主，描述大学生樱井真一和她的学妹宇崎花的日常生活。2020年2月，官方宣布该作品将改编为电视动画，动画由ENGI制作，自同年7月播出。

歌词下方是 _ココロノック钢琴谱_ ，大家可以免费下载学习。
