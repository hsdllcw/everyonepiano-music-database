_Philosophyz_
是日本女音乐人水谷瑠奈为电视动画《Rewrite》演唱的片头曲。动画是根据Key同名游戏改编，动画由8bit负责制作。2015年9月27日，在电视动画《Charlotte》的第13话终回放送后发表TV动画化的消息。游戏故事以一个绿意盎然、文明与翠绿并存的地方都市“风祭市”作为舞台，玩家们将扮演过着平凡生活的主角天王寺瑚太朗，并跟好友神户小鸟、吉野晴彦等一起迎接整个都市一年一度的大收获祭活动。
