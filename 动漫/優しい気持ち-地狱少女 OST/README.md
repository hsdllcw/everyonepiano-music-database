

**優しい気持ち**
是《地狱少女》的原声。《地狱少女》是日本动画以及由此改编的漫画及电视剧。动画于2005年10月4日开始播出，全26集，具有神秘恐怖风格。其漫画则同时在讲谈社月刊《Nakayoshi（なかよし）》上连载，作者为永远幸。2006年10月7日起，在每日放送、Animax以及东京MX电视台等播映第二季动画，名为《地狱少女
二笼》，全26集，2007年4月7日完结。第三季动画《地狱少女
三鼎》于2008年10月4日在东京MX电视台播出，全26集。于此外在2006年11月4日，在日本电视台播映真人出演的电视剧版本，全12集。
