

**CRAWL** 是电视动画《隠の王》的片头曲。《隐王》（日语：隠の王）是镰谷悠希的漫画作品。2004年开始在《月刊GFantasy》（Square
Enix）上连载，已于2010年8月完结，单行本最终卷（第14卷）于2010年12月在日本发行。2008年4月6日至9月28日播放电视动画，2008年7月网络广播配信。

  
歌词下方是 _CRAWL钢琴谱_ ，希望大家喜欢。
