

**Let Me Hear** 是MADHOUSE制作的电视动画《寄生兽 生命的准则》的片头曲。《寄生兽
生命的准则》是根据日本漫画家岩明均创作的漫画《寄生兽》改编而成的电视动画，于2013年11月宣布电视动画化，作品主要描写了平凡的高中生泉新一和寄生在他体内、与其右手同化的生物“右”共存的坎坷命运。
