

《 **Glitter** 》是TV动画《杜鹃婚约》的OP2，由日本四人摇滚乐队sumika演唱。收录于同名单曲，发售于2022年7月27日。

EI动画、SynergySP负责制作，于2022年4月24日起播出 ，全24集。
