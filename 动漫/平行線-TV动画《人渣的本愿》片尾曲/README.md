

**平行線双手简谱** 和五线谱完全对应。

_平行線_ 是TV动画《人渣的本愿》的片尾曲，由酸欠少女演唱。

电视动画《人渣的本愿》改编自横枪メンゴ原作的同名漫画，于2016年3月17日宣布动画化，此消息于同日在东京都内开办的“noitaminA
Project发表会2016”上发表；将于2017年1月12日开播。

歌词下方是 _平行線钢琴谱_ ，喜欢的朋友可以免费下载学习。
