_アノーイング！さんさんウィーク！_
是电视动画片《关于前辈很烦人的事》的片头曲，由楠木灯、早见沙织、青山玲菜与古贺葵共同演唱。动画《关于前辈很烦人的事》改编自しろまんた创作的同名漫画作品，于2020年7月1日宣布动画化的消息。该片由动画工房负责制作，于2021年10月10日起播出。

歌词下方是 _アノーイング！さんさんウィーク！钢琴谱_ ，大家可以免费下载学习。
