

**爱 think so** 是虚拟女性歌手软件，角色主唱系列的第一作初音ミク演唱的歌曲。

  
初音未来（初音ミク，华人世界通常简称为“初音”），2007年8月31日发售，开放价格，官方估计系列软件实际价格约15,750日圆。原只可用于Microsoft
Windows，2008年3月19日随CrossOver Mac 6.1发表而可用于Mac OS
X[2]，2014发布的VOCALOID3版本的软件已经原生支持OS
X。在GNU/Linux系统上通过修改过的wine也能够正确运行初音未来。发售后大受欢迎，并出现大量用户制作的翻唱歌曲、原创曲等。官方表示初音未来擅长由1980年代至最新的流行歌曲。

  
歌词下方是 _爱 think so钢琴谱_ ，希望大家喜欢。
