

**Dear双手简谱** 和五线谱完全对应。

_Dear_ 是游戏《初音ミク-Project DIVA-F2nd》的插曲，由初音未来和19's Sound Factory演唱，敬请欣赏。

初音未来（初音ミク/Hatsune Miku），是2007年8月31日由CRYPTON FUTURE
MEDIA以Yamaha的VOCALOID系列语音合成程序为基础开发的音源库，音源数据资料采样于日本声优藤田咲。

歌词下方是 _Dear钢琴谱_ ，希望大家喜欢。
