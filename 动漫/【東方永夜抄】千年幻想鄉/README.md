《 _千年幻想郷_ 》是游戏《東方永夜抄》的第五面BOSS——八意永琳的bgm。

《东方永夜抄 ～ Imperishable Night.》（日语：东方永夜抄～ Imperishable
Night.）是由同人组织上海爱丽丝幻乐团制作的弹幕射击游戏，是东方Project的第8作。

八意 永琳（Yagokoro Eirin）是系列作品《东方Project》中的角色，于《东方永夜抄》首次登场。
