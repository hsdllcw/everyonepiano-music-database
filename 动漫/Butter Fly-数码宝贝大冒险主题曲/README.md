

**Butter Fly** 是1999年日本动漫《数码宝贝大冒险》主题曲。作词：千绵伟功，作曲：千绵伟功，编曲：渡部チェル，演唱：和田光司。

数码暴龙大冒险（第一季）（台译： _数码宝贝_ 、港译：数码暴龙、英文：Digimon
Adventure，网络简称为“无印”、“DA”）。1999年，由于数码世界资料异常增生，导致数码世界和现实世界的连接发生紊乱。地球上相应地出现奇怪的自然现象：东南亚没有下雨，池塘全部干涸；中东却因为连场大雨发生洪水，美国则遇到有史以来最寒冷的冬天，日本则突降大雪。少年八神太一和他的伙伴们一起参加了夏令营，却神奇地被神秘地送到了一个多彩的数码世界。而且无数的谜团出现在被召选的少年面前。现在冒险已经开始了！！！
