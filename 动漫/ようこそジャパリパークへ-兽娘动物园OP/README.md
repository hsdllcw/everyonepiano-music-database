

**ようこそジャパリパークへ双手简谱** 和五线谱完全对应。

_ようこそジャパリパークへ_ 是电视动画《兽娘动物园》的片头曲，由动物Biscuits×PPP主唱。

《兽娘动物园》改编自韩国游戏公司Nexon于2015年推出的RPG游戏（2016年12月14日结束运营）。

歌词下方是 _ようこそジャパリパークへ钢琴谱_ ，喜欢的朋友可以免费下载学习。
