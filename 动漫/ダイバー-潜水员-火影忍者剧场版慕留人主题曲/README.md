

**ダイバー双手简谱** 和五线谱完全对应。

_ダイバー_ 是岸本齐史创作的漫画《火影忍者》的第11部剧场版动画《火影忍者剧场版：慕留人》的主题曲。且该曲由KANA-BOON演唱。

作为《火影忍者》在2015年的“新时代开幕”企划的一环，于2015年8月7日上映。该作品是首部“主角不是鸣人”的《火影忍者》剧场版，同时也是首部由岸本齐史亲自执笔全部剧本的《火影忍者》剧场版。

歌词下方是 _ダイバー钢琴谱_ ，希望大家喜欢。
