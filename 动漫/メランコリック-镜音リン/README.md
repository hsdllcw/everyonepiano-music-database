

《 **忧郁的心情** 》（日： _メランコリック_
，英：melancholy）是日本虚拟歌手镜音铃主唱，日本知名虚拟歌声制作人Junky于2010年4月19日上载的原创歌曲，是镜音铃的知名曲目之一。

《忧郁的心情》是2010年04月19日由Junky上载镜音铃主唱的原创歌曲（sm10444862），自2011年东京演唱会开始，经常在初音未来的演唱会中出现。其以活泼轻快的旋律和镜音铃萌萌的“电音”而大受喜爱，截至2014年1月24日有超过240.5万人观看，为《初音ミク-
ProjectDIVA-f》的收录曲。

镜音铃是CRYPTON FUTURE MEDIA（克里普顿未来传媒公司）以Yamaha的VOCALOID
2语音合成引擎为基础开发贩售的虚拟歌手软件角色主唱系列的第二作。于2007年12月27日发售。

歌词下方是 _メランコリック钢琴谱_ ，欢迎大家使用。
