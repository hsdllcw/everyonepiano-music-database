

《 **有谁共鸣**
》获第4届无线【十大劲歌金曲】，第4届无线十大劲歌金曲【年度金曲金奖】。张国荣：籍贯广东，1956年出生在香港，是一位在亚洲地区有影响力的著名演员、歌手和音乐人，演艺圈影视歌多面发展最成功的代表之一，亦是大中华地区的乐坛和影坛巨星。2003年4月1日逝世于香港文华酒店，终年46岁。

2013年3月31日，张国荣逝世十周年纪念演唱会《继续宠爱。十年Miss You Much
Leslie》在香港红馆举行，其经纪人陈淑芬所说埋藏了十年的秘密终于揭晓：我终于可以好好地舒服睡一觉。

经过略微改动，《 _有谁共鸣_ 》的国语版作为电影《80后》主题曲，由陈楚生演唱，歌名为《天长地久》。《有谁共鸣》很好听，感兴趣的可以下载
