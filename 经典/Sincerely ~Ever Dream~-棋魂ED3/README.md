_Sincerely ~Ever Dream~_
是电视动画棋魂的片尾曲（第三十一局～第四十六局）。《棋魂》改编自堀田由美的同名漫画，由东京电视台、电通及Studio
Pierrot共同制作，2001年10月10日－2003年3月26日在东京电视台系列播放，分为两辑，总共75集。

另外，Sincerely ~Ever
Dream~是由松室麻衣作词，菊池一仁作曲，dream演唱。歌的曲调部分是在2001年02月28日发行的首张专辑“Dear...”中的一首名为“Sincerely”歌曲的基础上进行编曲，
并由已毕业的成员松室麻衣进行重新填词而成，于她本人生日当天2002年6月10日发行。该歌曲是为即将告别组合的成员松室麻衣而发行的纪念单曲，PV被收录于dream
live 2002 “Process” 之中。
