

康康舞曲（Cancan），也称康康进行曲，是指19世纪末在法国流行的一种舞蹈音乐，为康康舞伴奏。舞曲为2/4拍，速度快而情绪热烈。最著名的为雅克·奥芬巴赫（Jacques
Offenbach）所作轻歌剧《地狱中的奥菲欧》中的康康舞曲。

电影《河东狮吼》中《我是一个芒果》的曲子就来自此曲。
