

《 **坐在宝座上圣洁羔羊** 》是赞美之泉演唱的一首基督歌曲，来自专辑《全新的你》。

赞美之泉音乐事工成立于1993年，1995年在美国加州正式成立非营利机构。1997年在台湾正式登记为赞美之泉文化事业基金会。赞美之泉自1995年出版第一张敬拜赞美专辑《让赞美飞扬》，得到许多的鼓励，继有共十多张敬拜赞美专辑、一张闽南语敬拜赞美专辑、三张演奏曲精选、一张钢琴演奏，2002年香港实况录音专辑以及一张GLOW青年敬拜专辑。我们依靠着主的能力，巡回脚踪从美国、加拿大，到巴拿马、新西兰、澳大利亚、台湾、香港、新加坡、至马来西亚作主的见证。赞美之泉服事至今，坚持要以神所赐给我们的一切所有，尽心、尽性、尽意、尽力的来服事他。我们歌，因为这是神给我们的恩典；我们唱，因为被神的爱摸着；我们见证，因为要高举主的名，传扬他奇妙的作为！因此，赞美之泉的目标是借着我们的事奉，训练出一批全方位的属灵军队；建立属神荣耀的祭坛；看见他的作为被彰显，藉此复兴的火能临到世界各地华人。
在赞美之泉，神兴起许多爱神又愿意用音乐服事神的年轻一代，看到新的一代我们知道神要大大的使用他们，因此除了沿袭赞美之泉过去一贯作风的敬拜赞美系列之外，我们正朝向制作一系列属于新世代的敬拜赞美，对主耶稣更多的委身是我们的目标。
