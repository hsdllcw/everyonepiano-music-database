_小夜曲_
是奥地利作曲家舒伯特（1797-1828）创作的歌曲。当时的民间传说认为，天鹅将死的时候，会唱出最动人的歌。维也纳的音乐出版家哈斯林格在舒伯特逝世后不久发现了他未曾问世的作品，认为他去世前半年所写的十四首歌，当属动人的绝笔，于是将它们汇编成集，以“天鹅之歌”命名出版，其中的第四首就是这首《舒伯特小夜曲》，是1828年用雷布斯塔布的诗谱成的。《小夜曲》这是舒伯特短促的一生中最后完成的独唱艺术歌曲之一，也是舒伯特最为著名的作品之一。此曲采用德国诗人莱尔斯塔勃的诗篇谱写成。
