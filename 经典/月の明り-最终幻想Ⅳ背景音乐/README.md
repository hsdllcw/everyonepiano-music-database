

**月の明り双手简谱** 和五线谱完全对应。

_月の明り_ 是游戏最终幻想Ⅳ的背景音乐。由伊田恵美演唱。

最终幻想IV，日文名：ファイナルファンタジー IV，是SQUARE SOFT的著名RPG游戏“最终幻想”（Final
Fantasy)系列的第四部作品（原定为第五部）。FF系列在SFC游戏平台上的第一作，由FC平台晋升为SFC平台时，伴随着的是画面质量与音效质量的大幅上升，游戏中的怪物形象真正具备了插画的水准，在一定程度上展现了天野喜孝的飘逸画风。本作同时奠定了之后最终幻想系列的主路线。

歌词下方是 _月の明り钢琴谱_ ，大家可以免费下载学习。
