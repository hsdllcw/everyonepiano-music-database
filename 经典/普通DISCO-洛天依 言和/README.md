

**普通DISCO**
是2015年3月21日由ilem上载至哔哩哔哩的Vocaloid原创歌曲，由洛天依与言和合唱。传说曲，超过6.7万人收藏。是ilem的第一首传说曲。同时也是洛天依的第一首传说曲，言和的第一首传说曲，国内第一首传说曲，中文V家第一首传说曲。

于2015年6月14日达成播放量200万。截止2015年8月7日播放量为237.8万。《暴走撸啊撸第一季》的结尾曲。

编曲作词与歌唱皆为ilem一人所做，除了三无marblue演唱作品外其余皆为翻唱。

作者4月9日微博：“正式强调一下。 三无marblue所演唱的《普通DISCO》与洛天依，言和演唱的《 _普通DISCO_
》制作上属于双本家。分类上均为原创音乐作品，两名作者享有同等的权利和义务。
故三无版本并非翻唱版，并非翻唱版。此前若造成误会我们感到抱歉，希望出现相关说明有误的地方予以更正。”
