

**インフェルノ双手简谱** 和五线谱完全对应。

_インフェルノ_ 是电视动画《炎炎消防队》的片头曲。由日本乐团Mrs. GREEN
APPLE演唱。《炎炎消防队》改编自大久保笃创作的同名漫画，于2018年11月14日宣布了动画化的消息。动画由david
production负责制作，于2019年7月5日起播出。

歌词下方是 _インフェルノ钢琴谱_ ，大家可以免费下载学习。
