

**再回首**
由台湾女歌手苏芮演唱，是80年代末期，90年代初期脍炙人口的一首歌曲，曾一时风靡大街小巷，传唱率颇高，被李翊君，姜育恒等众多歌手接连翻唱，其中以姜育恒的版本最红。

_  
再回首_ 的词曲作者分别是陈乐融和卢冠廷。《一切为明天》是1988年10月推出，没有收录
再回首。大陆1989年引进这张专辑时，抽掉了3首当时认为有问题的歌曲，换上了3首歌曲，《再回首》便是加入的一首。

  
歌词下方是 _再回首钢琴谱_ ，希望大家喜欢。
