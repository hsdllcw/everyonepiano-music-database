

《 **火的战车**
》（又名《烈火战车》或《火战车》）是休·赫德森1985年执导的一部电影。影片讲述剑桥大学犹太学生为了对抗种族偏见，为同胞争光，一心想成为赢得奥运会百米赛金牌的第一个犹太人。最后，自信而有拼搏精神的他终于夺得百米决赛冠军。  
  
北京时间7月28号，伦敦奥运会开幕式文艺表演又出现了新的高潮，伦敦交响乐团在现场演奏奥斯卡获奖电影《火的战车》中的经典旋律，指挥是英国最成功的指挥家拉特尔。  
  
《 _火的战车_
》演奏队伍中出现了一位大家非常熟悉的人物，他就是大名鼎鼎的憨豆先生。憨豆先生在现场“滥竽充数”，好比南郭先生。只见他左手不停地在电子琴上佯装弹琴，表情一如既往地搞怪可笑，但他的右手却拿出相机不停拍照。也许是天气太凉，憨豆先生禁不住打了个喷嚏，但他并未觉得尴尬连忙找出纸巾擦拭鼻涕，随后顺手将纸巾扔到旁边的演奏乐器上。憨豆先生起身拿出雨伞生怕影响到自己，在远处用雨伞触及琴弦。

憨豆先生困了慢慢地进入了梦乡，《 _火的战车_
》画面出现了，憨豆先生穿越到电影中，他加入到影片中一起奔跑，并且自己想方设法率先撞线，憨豆先生自己都陶醉了。当他梦醒后发现指挥家拉特尔在训斥他，他手忙脚乱，有口难辨，不知所错的他弹奏乐器走音。更出糗的，憨豆先生在众目睽睽下放了一个很响的屁......当然，这只是演出设计的一个桥段，一个发生在奥运会开幕式这种重大场合上、最可爱的一个“屁”，典型的英国式或者说憨豆式幽默，让整个开幕式的气氛变的其乐融融。
