

“你说你爱了不该爱的人，你的心中满是伤痕，你说你犯了不该犯的错，心中满是悔恨。”

对于70、80和90后，也许你记不得 **陈淑桦** 这个人，但你一定还记得她唱过的这首《 **梦醒时分**
》。这首1989年发行的歌曲，如今30多年已经过去，歌声依旧，但这位曾经的华语歌坛天后，已经永远从人们的视线中消失了。

这是一个悲伤的故事，一位因为母爱成就的巨星，也因为母爱而毁掉了整个人生的巨星。![](/Public/editor/plugins/emoticons/images/36.gif)

1958年，陈淑桦生于台湾一个普通家庭。她在5岁多的时候，就表现出了惊人的音乐天赋。在文化公司上班的母亲，对陈淑桦的音乐天赋有着敏锐的观察力，于是母亲有针对性的对小淑桦开始了培养。

事实证明，母亲的眼光是正确的。

8岁的时候，陈淑桦就在当时的台湾最大型的歌唱比赛中，凭借一首《个个满足》夺得冠军。

第二年，在母亲的建议帮助下加入了五虎唱片公司。

1988年，加入滚石唱片公司，在李宗盛的大力举荐下，公司开始包装陈淑桦。

在遇到李宗盛10年后，陈淑桦成了家喻户晓的当红歌手，也成了当之无愧的华语歌坛天后。

1997年，是陈淑桦命运的转折点。母亲的意外离世，彻底摧毁了陈淑桦的整个人生。

母亲是她的经纪人兼保姆，她承包了女儿的饮食起居，在工作、生活以及人际交往方面，也是尽心尽力替她搞定一切。

这样事事亲力亲为的母亲，自然也会在女儿的感情生活方面细心把关，据说陈淑桦的两次感情都是因母亲“横加干涉”而半途夭折。

母亲为女儿遮风挡雨，令人动容，但是她却忘了她并不能陪伴女儿一辈子。

在母亲的庇护下，陈淑桦连基本的生活能力都没有，母亲就是她的全世界，当她40岁那年母亲突然因病离开后，她的世界坍塌了——

“ **她不会谈工作，不会签合同，不会买机票，更不会处理和媒体的关系，一时之间，寸步难行。** ”

巨大的精神压力下，陈淑桦得了抑郁症，宣布退出歌坛。自此，她再没能走出母亲离去的阴影。

“你说你尝尽了生活的苦，找不到可以相信的人，你说你感到万分沮丧，甚至开始怀疑人生。”

谁能想到，18年前的歌曲，竟然印证了她的人生。

2006年时，李宗盛举办了“ **理性与感性** ”全球巡回音乐会，并且将第一场演唱会的主题定为：“ **献给陈淑桦**
”，希望此次能让她走出家门，重新回到原来的世界。
