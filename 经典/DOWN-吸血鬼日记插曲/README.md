

_DOWN_
这首歌被作为美剧《吸血鬼日记》第1季第6集的片尾曲。是一部集青春、魔幻、恐怖、剧情于一身的美国电视连续剧，由美国女作家L.J.史密斯的同名畅销系列小说改编而成。

jason Walker他来自宾夕法尼亚州卡农斯堡，唱美国本土音乐、流行音乐，住在纽约布鲁克林区。在舞曲上有很大成就。
