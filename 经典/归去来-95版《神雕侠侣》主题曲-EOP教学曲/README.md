

**归去来双手简谱** 和五线谱完全对应，属于EOP教学曲。  
  
《归去来》是95版《神雕侠侣》内地版主题曲。由小柯作词作曲，胡兵，希莉娜依演唱。所属专辑：《真本色》。  
  
1995年TVB版《神雕侠侣》电视剧，由导演李添胜执导，该剧汇集了古天乐、李若彤、傅明宪、李绮红、张可颐、苏玉华等国内众多知名实力演员，属中国武侠连续剧经典之作。1995年7月31日首播，影片上映时曾掀起一阵热潮。  
  
相信很多朋友再次听到这首歌曲，一定非常怀念吧。歌词下方提供了 _归去来钢琴谱_ ，欢迎大家使用。
