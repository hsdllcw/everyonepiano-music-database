

### 简介：

这首歌曲选自英国作曲家安得鲁.洛伊德.韦伯（Andrew Lloyd
Webber）根据Ｔ.Ｓ.艾略特（T.S.Eliot)的诗集谱曲的音乐歌舞剧．自从1981年在伦敦首场演出之后，已经成为迄今以来最著名的歌舞剧。《猫》曾一度成为歌舞剧的代名词．其中最经典的唱段就是朋友们现在听到的这首《回忆》。

### 《回忆》中文歌词：

午夜，  
路上寂静无声，  
月儿是否也失去记忆？  
他笑得多孤寂，  
街灯下，  
枯叶在我的脚下堆积，  
风儿也开始叹息。  
  
回忆，  
当我独自在月光里，  
我笑看着往昔，  
当时多么美丽，  
回忆当时我知快乐是什么含义，  
让回忆重新降临。  
  
每盏街灯似乎都  
在发出宿命的警讯，  
喃喃低语，  
街上灯光又熄，  
清晨就快要来临。  
  
天明，  
我要等待太阳升气，  
我要为未来思虑，  
而我不能放弃。  
黎明到来，  
今夜就将要成为回忆，  
新的一天就要开启。  
  
每日都像烟雾燃烧，  
霉味充满清晨空气。  
街灯已熄，  
又是一夜过去，  
又将是一个黎明。  
  
靠近我，  
离开我多么容易，  
让我独自回忆，  
回忆昨日美丽。  
如果靠近我，  
你将明了快乐的意义，  
看，新的一天已来临。
