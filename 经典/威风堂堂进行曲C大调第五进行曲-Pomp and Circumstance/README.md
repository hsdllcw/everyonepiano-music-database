

由Edward Elgar（爱德华•埃尔加）作曲的《 **威风堂堂进行曲** 》（Pomp and Circumstance
Marches），是风靡全球的著名管弦乐。总共包括Elgar在1901至1934年间创作的六首短小的进行曲。今天我们提供的是《C大调第五进行曲》。

《C大调第五进行曲》，又可称作《第五号进行曲》，完成于1930年，远晚于前四号作品。题献辞是“致我在赫里福德的朋友珀西•C•赫尔博士（Dr. Percy
C.
Hull）”。其舍弃了引子部分，直接由一段充满自信的旋律展开到了中段部分。中段部分的起始类似《埃尔加第一交响曲》，只有一段安静的旋律。随后这段旋律变强并发展。随后开头的旋律又柔缓地再现了一次，之后中段主题经过发展又重现了一次。结尾部分的处理则类似于爱德华•格里格的《在山魔王宫中》。
