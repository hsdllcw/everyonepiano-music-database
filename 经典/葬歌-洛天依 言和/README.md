_葬歌_
是2015年7月22日11时31分由UP主ilem上载至中文弹幕网bilibili的Vocaloid原创歌曲，由洛天依与言和合唱。该歌曲在B站上已有超过6.6万人收藏，弹幕超过6.4万条，硬币2.5万余枚。为其黑暗曲系列第五部，依然继承了前四部洗脑却诡异的曲风和ppt式的PV。同时本曲达成殿堂用时21小时19分，是ilem投稿的最速殿堂曲。

该曲于2015年10月5日16:34达成一百万播放，获得“VOCALOID中文传说曲”称号，历时75日5小时1分，中文V家第六首传说曲，ilem第三首传说曲，洛天依第六首传说曲，言和第三首传说曲，中文V家第三首合唱传说曲。

歌词下方是 _葬歌钢琴谱_ ，大家可以免费下载学习。
