

**Gracefully双手简谱** 和五线谱完全对应。

_Gracefully_ 是钢琴诗人乔瓦尼·马拉蒂 (Giovanni Marradi)的一首经典钢琴曲，收录在同名专辑里。

乔瓦尼·马拉蒂 (Giovanni Marradi) 出生在意大利，5岁开始步其父亲后尘练习音乐演奏，8岁开始学作曲。现在已成为世界著名的NEW
AGE音乐作曲家和钢琴家。Giovanni 的钢琴音乐犹如清晨荷花上的露珠，晶莹、透明；有时又象森林深处夜莺的私语，饱含不食人间烟火的凄美。
