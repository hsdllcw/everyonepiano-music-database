### 鸽子汉语歌词：

  
当我离开可爱的故乡哈瓦那，你想不到我是多么悲伤。  
天上飘着明亮的七色的彩霞，心爱的姑娘靠在我身旁；  
亲爱的我愿同你一起去远洋，像一只鸽子在海上自由飞翔；  
跟你的船帆在海上乘风破浪，你爱着我啊像一只小鸽子一样。

亲爱的我愿随你同去远航，像一只鸽子在海上自由地飞翔。
