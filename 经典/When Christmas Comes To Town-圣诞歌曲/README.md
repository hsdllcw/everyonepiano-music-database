

_When Christmas Comes to Town_
此歌曲是2004年北美上映的圣诞电影《极地特快》（又名北极特快车）的插曲。当寻找圣诞老人的火车驶入北极圈，相信圣诞老人存在的小男孩和多才多艺的小女孩站在极地特快车厢的尽头，合唱了这首忧伤的《当圣诞来到小镇上》。歌曲旋律充满童真，给人以温馨感，纯净的童音，就像孩子的心灵一尘不染；简单的歌词，就是孩子的内心独语，带着梦想，充满真诚，期盼圣诞老人的到来。美妙的爱尔兰天籁弦乐由美国的两名少年Matthew
Hall 和 Meagan Moore倾情演唱，作为最动听的圣诞金曲代表作之一，给人尤其儿童的印象极为深刻。
