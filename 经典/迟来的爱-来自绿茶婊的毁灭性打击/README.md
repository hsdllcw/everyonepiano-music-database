

“这是一封迟来的告白，许多年以来我一直都在逃避，不敢面对你多情的关怀，今天我终于鼓起勇气，向你表达我的爱。”

这首由 **李茂山** 演唱的经典歌曲《 **迟来的爱** 》发行于1987年1月1日，一晃30多年过去了，仍然唱出了很多人的心声。

曲子唯美真切，哀婉动人，令人不禁感叹，这爱情的告白来的太迟了。

小编不禁想起了一个故事：女孩追问男孩：‘你都追了我99天了，为什么最后1天不追了？’。男孩：‘前面的99天用来表达我对的真情，最后的1天是为了保留我的自尊。’

是呀，爱情中如果让对方爱的卑微，毫不在意对方的付出，那么你早晚会失去这个曾经爱你的人。

正如歌曲中的旁白：

“我知道，一切都太迟了，明天，你就要带着她，走进结婚礼堂，我羡慕她，同时也给你，我最深的祝福！”

_（你说人家都要结婚了，你还来一个告白，这不是给人家添堵的吗？![](/Public/editor/plugins/emoticons/images/27.gif)
就是一个绿茶！）_

这不禁让小编又想起了一个故事：男孩和女孩闹矛盾约定背对背各走100步，回头时如果还能看见彼此就不分手。走到99步的时候女孩终于忍不住蹲在地上哭了，一双手臂从背后圈住了她。男孩说，我一直都在等你回头，一辈子我都不会放你一个人走！

是呀，在爱情中占尽优势的一方，你只要稍微顾及一下对方的感受，给对方那最后一步的机会，也许你就不会失去眼前这个最爱你的人。

小编借此奉劝一下某些人：不要把真心爱你的人当作备胎！
