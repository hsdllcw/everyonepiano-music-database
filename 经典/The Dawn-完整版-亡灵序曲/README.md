

**The Dawn** ，即亡灵序曲，出自网上广为流传的一部经典的关于魔兽争霸3的亡灵族的MV背景音乐，此音乐由芬兰Dreamtale乐队演奏。

值得一提的是，本曲在节奏处理上，保留了金属音乐最基本的连续猛烈的伴奏，却淡化了狂乱、低沉和无序的节奏，这使得该曲少了一些狂躁、恐惧和迷惘，多了一份神秘和向往。而那段迷惘的男声独白，恰如破晓前黑暗中初现的第一缕晨光，给人以直接和真实的感受。此曲也因此被金属与非金属迷所喜爱，并被奉为金属音乐中的经典之作。
