_  
单身情歌_
是由陈耀川作曲，易家扬作词，林志炫演唱的轻摇滚歌曲。为了配合曲风的改变，林志炫在MV中和以往的形象有极大的反差，一改一贯的温文儒雅，穿皮装，玩摇滚，亦酷亦炫的样子，变得狂野奔放了起来。在MV中，林志炫和他的音乐伙伴们组成一个乐队在酒吧演唱，无意之中成为了一段爱情悲剧的旁观者，男女主角约在酒吧见面，女主角等候多时留下对方送的口红失望而去，男主角因故迟到带着准备求婚的鲜花迟到而来，
然而已经人走座空，徒剩凄凉。

  
歌词下方是 _单身情歌钢琴谱_ ，希望大家喜欢。
