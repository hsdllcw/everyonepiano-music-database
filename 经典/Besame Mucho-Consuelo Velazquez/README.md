

**Besame Mucho** 是一首非常怀旧的老歌。这首歌曲的作者墨西哥女作曲家(也是著名的钢琴演奏家)Consuelo
Velazquez，1941年在一次探视了病重的亲友后，忽然感触到人生短暂，要珍惜现在美好的生活和爱人，于是创作了《Beseme Mucho 》 这首歌曲。

  
很难用中文来准确表达" _Besame Mucho_
"的意思，在拉丁文中"Besame"的意思是"吻"，而"Mucho"则是"多"的意思，有的人将它翻译为《热情相吻》或《吻吧，热烈的吻吧》，也有翻成《深深的吻》的，还可以找到一些其它中文名字，但似乎都不确切而都有道理。

  
这首经典的歌曲被很多著名歌唱家演绎成不同风情的感觉和版本。有擅爵士风的lisaono，也有Lena
valaitis等，后者的拉丁风情更为浓郁，当然，各有所见。

  
歌词下方是 _Besame Mucho钢琴谱_ ，希望大家喜欢。
