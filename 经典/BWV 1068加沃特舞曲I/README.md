出自巴洛克时期德国作曲家巴赫的《管弦乐组曲》。巴赫的这套《管弦乐组曲》一共有四首，编号为BMV1066-BMV1069。作品创作于1717年至1723年之间，从《音乐圣经》一书上查询到前三首创作于1723年巴赫迁居莱比锡前夕，第四首创作于作曲家定居莱比锡之后不久。由此可以看出，《管弦乐组曲》和《勃兰登堡协奏曲》基本同属于巴赫科藤时期的重要作品。根据相关资料考证，巴赫所作的《管弦乐组曲》应该不止这四部，可惜其他作品由于各种各样的原因而失散。遗留下的这四首曲目则显示了当时流行于欧洲的法国式庄重、典雅的舞曲乐风，其中不少乐章广为流传。
