我等候你_
选自原声大碟《人间四月天》(同名电视剧主题曲)，是一曲如诗般的音乐，用音乐将徐志摩的诗意语言“最是那一低头的温柔”给解读出来了，也传神表达了《再别康桥》的意境“轻轻地我走了,正如我轻轻地来”。更刻划了集才气、集美质、集傲岸于一体的林徽因--“一身诗意千寻瀑，万古人间四月天”。飘逸的钢琴也仿佛是林徽因在诉说“你是一树一树的花开,
你是人间四月天”！  
  
此情只应天上有，今闻竟在人世间！假如徐志摩和林徽音结合，他就写不出那种朦朦胧胧的、有一点距离感的诗。美就是一种距离，爱就应该远离，正如金岳霖远离林徽因那样，《
_我等候你_ 》这曲调也诠释了金岳霖对林徽因的距离。  
  
这首干净的曲子，象涓涓的小溪，流淌着幸福和欢乐，叮叮咚咚的琴声，一下下撞击着年轻时美好的情感；又象一滴一滴掉下的泪，是对遥远的回忆，象月光一般无遮无掩地洒在心上。  
  
《人间四月天》是一部由台湾纵横国际影视股份有限公司、中央电影事业股份有限公司以及中国北京荣信达影视艺术中心所合拍的电视剧，主要描述徐志摩与三位女性（原配张幼仪，心仪对象林徽因，以及最后的伴侣陆小曼）的爱情故事。《人间四月天》于1999年先后在台湾公共电视、台湾超视、台湾台视、中国中央电视台、香港亚视播出。
