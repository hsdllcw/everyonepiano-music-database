

《 _金婚式_ 》（ _金婚曲_ , La Cinquantaine, "The Golden Wedding"）是著名作曲家吉恩·加布瑞尔·马利（Jean
Gabriel Marie）在1887年为大提琴与钢琴所写的一首著名乐曲。  
  
作为一首流传至今、经久不衰的经典曲目，《 _金婚式_ 》曾被改编为小提琴，钢琴、口琴、萨克斯等多种版本。“金婚”（The Golden
Wedding）原指结婚50年的忠贞不渝的婚姻。乐曲的旋律轻快诙谐，优美动听，安逸中透出灵动的感觉。  
  
吉恩·加布瑞尔·马利（Jean Gabriel
Marie）(1852–1928)是一位来自法国的浪漫主义音乐作曲家，1852年1月8日生于巴黎，1928年8月29日死于西班牙加泰罗尼亚赫罗纳省的普伊格塞尔达。马利的作品包括了为大提琴与钢琴，或弦乐八重奏，或其他乐器组合所写的《金婚式》，以及其他的许多舞曲。其中最具代表性是圆舞曲《在梣树下》（Sous
les firnes）和波尔卡《新鲜的面孔》（Frais minois）。
