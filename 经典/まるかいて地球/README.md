

**まるかいて地球双手简谱** 和五线谱完全对应。

_まるかいて地球_ 是浪川大辅创作的作品。浪川大辅，日本男性声优、歌手，东京都出身，所属事务所为Stay
Luck。1985年由一龙斋贞友介绍进入剧团知更鸟，当时年仅9岁。从童年开始就作为
声优活跃，声优经验丰富。浪川大辅素来以多才多艺闻名，在演唱、电台节目发明均有涉足。在电影吹替方面，曾为美国电影《星球大战》中的阿纳金和《魔戒》中的弗洛多献声，大获好评的同时也为其赢得了越来越多的拥护者。
