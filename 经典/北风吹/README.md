

《白毛女》舞剧插曲。《白毛女》故事梗概佃农杨白劳与女儿喜儿相依为命，喜儿与同村青年农民大春相爱。杨白劳因生活所迫向恶霸地主黄世仁借了高利贷，之后外出逃债。在除夕之夜杨白劳偷偷回家。黄世仁闻讯后强迫杨白劳卖女顶债，杨白劳喝做豆腐用的卤水自杀（在芭蕾舞中被黄世仁用手枪打死）。喜儿被抢进黄家，遭黄世仁奸污，逃入深山，头发全白。两年后，大春随八路军回乡，在山洞里找到喜儿，替她申冤雪恨。结尾处，村民们和喜儿一起开会批斗黄家的罪行，庆贺穷苦人的重见天日。
