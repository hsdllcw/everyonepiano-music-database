曲子开始是一段细碎叮咚的快速音符，仿佛是少年无忧无虑的嬉戏笑声；中间主体部分以变奏的形式反复出现，似乎在回忆甜蜜青涩的爱情；接着再次响起开始部分的旋律，然此时已没有最初的无忧无虑，有的只是不安，而这份不安与躁动最终在响亮的音符中达到顶点；最后，左手低音响起，在一段低沉琴声中结束全曲。

对于弹奏，大家练习时要多注意升降，避免有错音，另外，声音要清晰，踏板要换得彻底，特别是在转调的时候。左右手的三对四要配合严谨，左手要轻而有节奏，不能混过去，否则会使整体的效果显得喧闹。右手要注意声音渐变，以及需要突出的音符。
