

**热情奏鸣曲第3乐章** ，也就是F小调钢琴奏鸣曲, Op.
57（贝多芬第23首钢琴奏鸣曲）第3乐章，此曲的情感达到了非常高的境界，是音乐教学选用的重要素材。

《热情》并不是贝多芬起的名字，而是出版商，其寓意是面对困难时不要轻易低头，相信自己能够战胜困难，在生活中一定要充满对光明的向往，对社会现实的不满。我们要不断地抗争，但最重要的还是对自己的提升。

《 _热情奏鸣曲第3乐章_
》为奏鸣曲式，但不寻常的是只有第二部分需要重复。它与第一乐章有不少相似之处，如经常运用拿坡里六和弦、记于谱上的华彩乐段等。第二部分后为急板的尾声，完结时调性保持F小调，Donald
Tovey指这是少数悲伤地完结的奏鸣曲(其余为C小调钢琴三重奏、月光奏鸣曲、第7小提琴奏鸣曲、第14弦乐四重奏)
