

**冬风练习曲** ，作品25-11（Op.25
No.11）是肖邦练习曲中技巧最艰深篇幅最大的音乐会练习曲之一。它的训练课题是手指触键的灵敏，快速和准确，与此同时，由于篇幅大，因此对手指快速跑动中的耐力也是一大考验。
