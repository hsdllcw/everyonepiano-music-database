

**天国的孩子** 是马克西姆的作品。马克西姆·姆尔维察（Maksim
Mrvica），克罗地亚钢琴演奏家。他的作品多将古典钢琴作品与现代流行音乐元素结合，并受到广泛好评。代表作有《野蜂飞舞》《克罗地亚狂想曲》等。

_  
天国的孩子_
收录在其专辑《Electrik》中。《Electrik》和以往一样，改编了多首古典音乐中的名曲，马克西姆从葛利格的「皮尔金」组曲中挑选出「山魔王的宫殿」，轻快的触技曲，加上俏皮的电音，让人重新品味、领会这首古典名曲的趣味所在；比才歌剧「卡门」的间奏曲，则把原本由长笛主奏的名曲，化身为有如肖邦的夜曲；佛瑞著名的「安魂曲」中，飘邈的「天国」，更是被出乎意外地改写成电子混音和钢琴合奏的驰放（chill-
out）舞曲，那强大的电子声浪，片片袭卷而来，让人陶然神往。

  
天国的孩子是一首很好听的曲子。这里我们提供 _天国的孩子钢琴谱_ ，希望大家喜欢。
