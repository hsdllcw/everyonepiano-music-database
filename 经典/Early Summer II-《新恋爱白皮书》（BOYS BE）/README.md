**_Early Summer II_ 是日本动画《新恋爱白皮书》（ボーイズ ビー。BOYS BE）中的一首插曲。  
  
《新恋爱白皮书》（原名：BOYS BE）是イタバシマサヒロ板橋雅宏原作，玉越博幸繪畫的一部少年漫畫作品，后改编成动画。  
  
这部片子的清新风格还是挺不错的。它描写的是一群17岁少年的懵懂爱情。观看此剧，让人不由的想到十七八岁时的初恋，轻涩而又甜蜜，短暂却又绽放着光芒。虽然此剧的画风稍嫌老套，但樱花飘落时的画面处理确是非常唯美。  
  
这部不算很长的片子里汇集了很多的著名声优，如白鸟由里、长泽美树、石田彰、今井由香等，而且三木真一郎、子安武人等人在其中的几集中有客串表演，最后一集中出现的少女的声优竟然是林原惠美，让人大跌眼镜。如果你是个铁杆声优迷的话，这部片子倒是值得一看。
