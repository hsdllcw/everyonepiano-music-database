_康康舞曲_
，也称康康进行曲，是指19世纪末在法国流行的一种舞蹈音乐，为康康舞伴奏。舞曲为2/4拍，速度快而情绪热烈。最著名的为雅克•奥芬巴赫（Jacques
Offenbach）所作轻歌剧《地狱中的奥菲欧》中的《天堂与地狱序曲》，作于1858年，同年在巴黎首演。

序曲自始至终贯穿着欢乐的喜剧气氛，节奏明快、流畅，在一定程度上体现了作者对整个歌剧作品的调侃色彩。
