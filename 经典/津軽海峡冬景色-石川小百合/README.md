《津軽海峡冬景色》是日本歌手石川さゆり的经典歌曲。  
  
石川小百合（石川さゆり），熊本县出身的著名演歌女歌手，堀越高等学校毕业。本名石川绢代。1977年以名曲《津轻海峡・冬景色》成名。之后[天城越え].[能登半岛][风の盆恋歌]等发行，成为最为人所知的演歌歌手之一。代表作天城越え，津轻海峡・冬景色，越前竹舞い
等。
