

**亲密爱人双手简谱** 和五线谱完全对应，属于EOP教学曲。

《亲密爱人》选自梅艳芳唱片《亲密爱人》。

《 _亲密爱人_
》应该是梅艳芳最经典的国语唱片之一了。那几年，到台湾发行国语唱片的香港女歌手都大获成功，阿梅也不列外。从这个时期开始，阿梅的唱片不再像以前那样过于注重造型的百变和包装，而是把更多的心思花在音乐本身上。听"亲密爱人"，我们在她那低沉的嗓音中、听到情感的细腻和甜蜜，给人的感觉是"温馨"。也许，感情的路往往不是憧憬顺遂，局外人看着她挫折、也看着她坚强。《珍爱真爱》、《很难再说》……一首首真性真情、刻骨铭心的歌曲，就这样深深地植入我们的心底。梅艳芳，来传音，去留声——永恒不灭的传奇女王!

梅艳芳（Anita
Mui，1963.10.10—2003.12.30）生于香港，祖籍广西合浦，是中国流行乐坛已故著名女歌手，二十世纪末期大中华地区乐坛和影坛巨星；香港乐坛殿堂级艺人，香港演艺人协会的创办人之一及首位女会长。梅艳芳以醇厚低沉的嗓音和华丽多变的形象著称，被称为““百变天后”和“东方麦当娜”，她是大中华地区首位在同一个演唱会换多套歌衫、在不同专辑封面和音乐录像带有不同造型的歌手，曾引领粤港地区的时代潮流，是香港乐坛最高荣誉“金针奖”和“中国金唱片奖艺术成就奖”的最年轻得主，至今保持着华语女歌手全球演唱会场次的最高纪录。梅艳芳同样蜚声影坛，荣获过两岸三地的影后，入选“中国电影百年百位优秀演员”。她一生致力公益慈善，推进大中华演艺事业的交流合作，被誉为“香港的女儿”。

《亲密爱人》给人的感觉很温馨，很甜蜜。歌词下方是 _亲密爱人钢琴谱_ ，希望可以帮得到需要的朋友。
