

**斯瓦尼河**
，又叫做故乡的亲人，是美国作曲家斯蒂芬•柯林斯•福斯特创作的。福斯特在远离自己家乡和亲人的辛辛那提州工作时，感到远离亲人的孤寂和惆怅，于是创作了这首歌，借以寄托思乡之情，本曲的歌词也是作者自己填写的。

  
当时美国的一家音乐杂志叙述了本曲风靡美国的情景：“《故乡的亲人》是一首无与伦比的、具有黑人音乐旋律的歌曲。所有的人都在哼唱着它。钢琴、吉它不分昼夜地弹奏着它；伤感的女士在唱它；浪漫的绅士在唱它；潇洒的青年在唱它；歌星们唱着它；街头的手风琴艺人也在边拉边唱着它……”据说她被评为美国在世界流传最广的民歌，同时也是佛罗里达州的州歌。

  
歌词下方是 _斯瓦尼河钢琴谱_ ，希望大家喜欢。

### 斯瓦尼河歌词：

沿着那亲爱的斯瓦尼河畔，千里迢迢，  
在那里有我故乡的亲人，我终日在想念。  
走遍天涯，到处流浪，历尽辛酸。  
离开了我那故乡的亲人，使我永远怀念。  
世界上无论天涯海角，我都走遍，  
但我仍怀念故乡的亲人，和那古老的果园。  
走遍天涯，到处流浪，历尽辛酸。  
离开了我那故乡的亲人，使我永远怀念。
