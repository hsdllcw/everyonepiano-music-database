

**下沙** 是歌手游鸿明于2000年9月1日推出的个人创作专辑《下沙》主打歌曲。此曲是一首相当诗意的情歌，以管弦乐和合成器共同交织出壮阔的情境。

  
而鸿明在内地最红的一首歌非《 _下沙_
》莫属。同时游鸿明的假声也是很多人交相称赞的。其实在本专辑中除了《下沙》，还有许多非常不错的音乐小品，比如《寻你》、《过往的新歌》等。

  
而本专辑的文案中有这样一段话：沙漠，隐藏无尽的美丽与危险，爱情也是。追寻，一种金黄色的幸福背负的回忆太重。足迹深陷风呼啸，四面八方，回荡，回荡，思念震耳欲震聋。下沙了。瞬间覆盖所有真相，毁灭降临，绝望迎面袭击。清醒的血液，是灵感的唯一，狂窜肆流光影边缘的最终，在诗里看见永恒。

  
歌词下方是 _下沙钢琴谱_ ，希望大家喜欢。
