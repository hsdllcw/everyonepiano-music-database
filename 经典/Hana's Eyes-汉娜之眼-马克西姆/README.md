

Maksim被冠名为钢琴玩家，在“钢琴家”里，加上“玩”字，代表他的演奏风格，不是传统钢琴家的严谨，也不是理查德克莱德曼般的绅士，他惯于一身酷黑，甚至袖子上缀满长长的流苏演奏，一副很MAN的型男模样，把他的痴狂注入指尖下的琴键——于是我们得以享受一种与众不同的，听觉与视觉盛宴。

在Maksim演奏的钢琴曲中，《 **汉娜之眼** 》（hana's
eyes）是我最喜欢的曲子之一。因为在这首曲子中，娓娓的钢琴旋律与如泣如诉的女声完美结合，我听到了汉娜眼中无法熄灭的幽怨。

据说，《 _汉娜之眼_ 》是影片《英国病人》的插曲。记录了战争阴影下，一名叫汉娜的护士，一段谨慎而克制的爱情。

闭上眼睛听《 _汉娜之眼_ 》，就像在听一个耐人寻味的故事。

一段谨慎而克制的爱情，就是这首曲子幽怨的全部吧！
