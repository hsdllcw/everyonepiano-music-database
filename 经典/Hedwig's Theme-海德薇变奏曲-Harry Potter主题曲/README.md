

《 **Hedwig's Theme** 》（ _海德薇变奏曲_ ）是电影《哈利·波特》（Harry Potter）主题曲。

《Hedwig's
Theme》是约翰·威廉士为哈利波特电影系列作的。它经常被人标志为电影系列的主题曲，在哈利波特—神秘的魔法石电影中首次使用。有一场同名的音乐会亦以此曲为最尾的音乐。“Hedwig's
Theme”已经被加插至到现时为止的每一套哈利波特电影中，包括由那些由帕特里克·道尔和尼古拉斯·霍珀配乐的电影。它更被加入至凤凰会的密令游戏中。“Hedwig's
Theme”取得了显著流行文化的地位，出现在铃声、预告片音乐和不同形式的多媒体中。
