

**渴望**
是毛阿敏为电视剧《渴望》录制的同名主题曲，也称作“悠悠岁月”，是她的代表作。在毛阿敏的演唱生涯中和成名曲《思念》和国外获奖歌曲《绿叶对根的情谊》有着同等重要意义的第三首歌。改革开放30年电视剧歌曲盛典，全程以渴望为背景音乐，可见这首歌的意义。

  
另外， _渴望_
此曲是易茗，雷蕾，夫妻二人创作的，歌曲深刻揭示了，人生路漫漫的生活渴望，对真情的眷恋，对现实的执着，对未来的期盼，而这首歌正是毛阿敏当时的生活写照，所以唱起来共鸣更多，情绪情感都很到位，成就了这首经典中的经典。

  
歌词下方是 _渴望钢琴谱_ ，希望大家喜欢。
