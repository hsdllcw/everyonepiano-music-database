

**爱江山更爱美人**
收录在李丽芬的《就这样约定》专辑，作为专辑中的主打歌之一，由滚石唱片制作发行于1994年1月22日，是由著名音乐人小虫根据她独特的声线，特意量身创作的中国古典风格歌曲。同时也是1994年马景涛、叶童、周海媚主演台湾武侠剧《倚天屠龙记》（第1集至第22集）的片尾曲。

  
其实，《 _爱江山更爱美人_
》起初并非专门为《倚天屠龙记》而作，而是在机缘巧合下成为它的片尾曲，却意外地与剧中情境高度契合。古典而悠婉的辞藻，为歌曲增添了些许文气和古韵，不但写出了隐士的超脱，仁者的豁达，也道出了智者的欲求，一经推出，便传唱至各个街头巷尾，成为不朽的金曲。

  
歌词下方是 _爱江山更爱美人钢琴谱_ ，希望大家喜欢。
