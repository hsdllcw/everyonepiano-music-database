《 **La Cumparsita** 》,被称为《假面舞会》或《假面游行》，是一首非常有名的探戈舞曲，由乌拉圭音乐家Gerardo Matos
Rodríguez作于1916年。

在传入欧洲之时，这首乐曲曾被命名为《Si Supieras》，是由词作者Pascual
Contursi命名的，在当时可以说是风靡所有的管弦乐队，而音乐家Gerardo本人却为了这首乐曲打了将近二十年的官司，才将作品的标题改回了《 _La
Cumparsita_ 》，也就是大家所熟知的化装舞会。

这首歌深受南美的音乐家们喜爱，像是之前介绍过的一步之遥的作者Carlos Gardel，以及新探戈之父Astor
Piazzolla，他们所带领的乐队都曾多次演出过这首乐曲。在1997年乌拉圭的法律条例里将这首歌视为乌拉圭的文化代表之一。在众多影视作品以及艺术体操花样滑冰等体育赛事中都有使用过这首歌，是世界上最为出名的探戈音乐之一。
