

**在水一方** 由琼瑶填词，林家庆谱曲，江蕾原唱。1980年时，邓丽君翻唱了这首歌并发行了同名专辑，从此，《 _在水一方_
》红透了大江南北，也因此被内地听众误认为是邓丽君的原唱。  
  
歌曲曲调婉转悠扬，邓丽君的歌声更是柔情似水。邓丽君的音色与词曲完美地结合，体现了歌词中塑造的流水般的意境。而且该版本乃小编为[

另外，这首歌是琼瑶1975年时为她的电影《在水一方》所作的同名主题歌，是电影《在水一方》的主题曲，收录在邓丽君2006年发行的专辑《花样年华》中。1988年电视剧版《在水一方》播出，这首歌改由李碧华演唱，作为电视剧的插曲。
