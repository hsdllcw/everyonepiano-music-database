

**红梅花儿开**
是一首俄国的歌曲，原名《红莓花儿开》，由伊萨科夫斯基作词，杜那耶夫斯基谱曲，表达了少女对心上人的思念之情。这首歌也是电影《幸福生活》的插曲，电影和歌曲均获1951年斯大林文艺奖金。

  
歌词下方是 _红梅花儿开钢琴谱_ ，希望大家喜欢。
