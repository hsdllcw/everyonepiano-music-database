《 **Flohwalzer** 》（Flea
Waltz），中文被称为《跳蚤圆舞曲》或《踩到猫儿》，是一首简单的钢琴曲，因其指法很简单，弹起来也相当和谐轻快，故成为常见的初学者曲目。

虽然名称有“圆舞曲”，但这首歌并没有像一般圆舞曲一样是三拍子的。作曲者不详。有一本音乐家艾瑞克•包曼所著的书中提到这首歌的作者为Ferdinand
Loh，但这显然是个玩笑（“F.
Loh”=Floh，在德语中意为跳蚤）。为了让弹奏时的指法更为容易，这首歌通常是降G大调，这样一来大部分的琴键都会是钢琴上的黑键。
