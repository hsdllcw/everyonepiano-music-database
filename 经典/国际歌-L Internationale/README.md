

**国际歌** 是由欧仁•鲍狄埃在1871年作词，皮埃尔•狄盖特于1888年谱曲而成的歌曲。

《 _国际歌_ 》曾是第一国际和第二国际的会歌
；20世纪20年代，苏联以《国际歌》为国歌，并在1944年把《国际歌》作为联共（布）党（1952年改名苏联共产党）党歌。1920年中国首次出现由瞿秋白译成的中文版《国际歌》。1923年由萧三在莫斯科根据俄文转译、由陈乔年配唱的《国际歌》开始在中国传唱。1962年译文重新加以修订。
