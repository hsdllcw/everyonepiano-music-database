_Only You (And You Alone)_ ，常简称为Only You，是著名黑人乐队The platters于1955年唱红的一首经典流行歌曲。  
  
The
Platters首次演唱该曲并不成功，但是在1955年7月3日再次推出此曲后，获得了极大的成功。该曲在全美R&B榜单中连续7个星期位居榜首。包括约翰·列侬和林格·斯塔在内的多位歌手曾经翻唱过此曲。该曲也出现在许多好莱坞电影中，作为背景音乐；也有一些广告、网络背景音乐喜欢用该首歌。
