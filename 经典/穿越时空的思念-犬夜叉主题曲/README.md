_穿越时空的思念_ 是日本动漫《犬夜叉》第一部剧场版的主题曲。《穿越时代的思念》，又名跨越时代的思念。

日本著名动漫《犬夜叉》第一部剧场版之名，该片于2001年12月15日上映。《跨越时代的思念》当年连续四周跻身全国票房前十位，是动画史上首部原创故事电影，说的是被犬夜叉爸爸封住的玛瑙丸苏醒了，父债子偿，要向"犬子"复仇。动画同名主题曲《穿越时代的思念》，由和田薫谱曲，原版动画中共有两个版本。后该音乐衍生出许多翻唱作品。

穿越时空的思念一曲，随着《犬夜叉》的播放，往后的年月里，在中国网络上，尤其是网络音乐圈以及二次元圈中，享有著名声望，很多MV以及同人视频中，频繁出现这首曲子的身影，即使是在向来盛产动漫名曲的日本动漫中，也是十分罕见的作品。
