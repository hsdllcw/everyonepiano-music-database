

**FourFiveSeconds双手简谱** 和五线谱完全对应。

_FourFiveSeconds_ 是Rihanna，Kanye West和Paul
McCartney合作的新单，于2015年1月25日发行，是Rihanna将于2015年发行的新专辑“R8”的首支单曲。歌曲在包括美国在内的47个国家的iTunes下载榜夺得冠军(仅登顶半天就被火星哥拉下),单曲成绩在全球还是十分理想。

Rihanna为Def
Jam唱片公司旗下艺人。来自加勒比岛国巴巴多斯，是欧美乐坛当红歌手、演员和时尚女星，在全球多个国家都享誉盛名。她是21世纪以来全球销量最高的歌手之一，在Billboard
Hot
100榜单上有13首冠军单曲，是Billboard榜单52年历史上最快达到该成绩的个人歌手。同时她还是多座格莱美奖、全英音乐奖、MTV音乐大奖获得者。她在2008年荣获第50届格莱美奖，是首位获格莱美奖的巴巴多斯女歌手。刚出道7年的Rihanna被各大媒体杂志纷纷封上“最受期待的明日之星”的称号，被视为未来世界流行乐坛天后宝座的强有力竞争者。2009年更被奢侈品品牌GUCCI看中，成为GUCCI
“Tattoo Heart Collection” 少女系列包包广告代言人。Rihanna也是家乡巴巴多斯的流行文化代表和旅游大使。
