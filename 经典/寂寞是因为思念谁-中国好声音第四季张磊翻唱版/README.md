

今天我们推荐一首经典老歌《 **寂寞是因为思念谁** 》，1994年由 **景岗山** 首唱，这里的版本是《中国好声音第四季》 **张磊** 翻唱的版本。

“你知不知道思念一个人的滋味，就像喝了一杯冰冷的水，然后用很长很长的时间，一颗一颗流成热泪。”

优秀的歌曲并非一定要有华丽的语言和难读的汉字，简单朴素的歌词更容易直击心底，泛起阵阵涟漪，这首《 **寂寞是因为思念谁** 》就做到了。

整首歌曲都透出一种孤单寂寞的“冷”，而 **张磊** 带有故事的嗓音，听着别有感触，思念的滋味最为痛苦。

歌词太扎心，令人难忘怀，有经历的人，痛彻心底。

我们无法改变现实，只能“用很小很小的声音，告诉自己——要坚强面对”。
