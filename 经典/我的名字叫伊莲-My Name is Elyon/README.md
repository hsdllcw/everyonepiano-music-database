

伊莲娜·霍莱（Hélène
Rollès)，法国著名女歌手，1966年12月20日出生在法国的MANS（SARTHE），具有阳光一般的金发和一丝哀愁的褐色眼睛。
1993年她以一曲《我的名字叫伊莲》（Je m'appelle Hélène）连续25周夺得法国单曲排行榜冠军。蔡淳佳将此歌翻译成《依恋》。
