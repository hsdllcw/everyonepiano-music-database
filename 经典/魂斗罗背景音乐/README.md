

**魂斗罗**
（Contra）是Konami于1987年至今在大型游戏机，游戏主机上推出的一系列卷轴射击游戏。魂斗罗的故事背景是根据著名恐怖片《异形（Alien）》改编，人物原型来源于著名影星施瓦辛格和史泰龙。1987年第一款魂斗罗诞生在名为Jamma的街机上。游戏名称的含义是“具有优秀战斗能力和素质的人”，它是赋予最强战士的称呼。  
  
还记得红白机时代KONAMI公司的“魂斗罗”系列带给我们的快乐日子吗？在FC游戏机上《魂斗罗》和《超级魂斗罗》（《魂斗罗》的续作，又叫《魂斗罗二代》或《魂斗罗2》）是许多卡带上（尤其是合卡）不可或缺的游戏，七、八十年代出生的玩家一定记得穿梭在魂斗罗世界枪林弹雨中的爽快感觉。
