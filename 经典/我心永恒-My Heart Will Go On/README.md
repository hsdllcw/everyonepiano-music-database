

**我心永恒双手简谱** 和五线谱完全一致，是EOP教学曲。

_我心永恒_ ，又名《心与你相依》，泰坦尼克号主题曲，由好莱坞主流电影著名作曲家詹姆斯·霍纳（James
Horner）一手制作，具有浓烈民族韵味的爱尔兰锡哨在他的精巧编排下，尽显悠扬婉转而又凄美动人。我心永恒的旋律从最初的平缓到激昂，再到缠绵诽恻的高潮，一直到最后荡气回肠的悲剧尾声，短短四分钟的歌曲实际上是整部影片的浓缩版本。
