

**真的好想你** 是歌手周冰倩所演唱的歌曲，收录于1996年发行的《真的好想你-
周冰倩珍藏精选》专辑中。她充满感情的声音使她成为近年被公认为上海最具代表性的歌手。

  
歌词下方是 _真的好想你钢琴谱_ ，希望大家喜欢。
