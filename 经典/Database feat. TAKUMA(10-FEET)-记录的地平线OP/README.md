

**Database feat. TAKUMA(10-FEET)**
是SATELIGHT制作电视动画《记录的地平线》的片头曲。《记录的地平线》（ログ・ホライズン，LOG
HORIZON）是由日本网络小说家橙乃真希创作，由原和弘负责插画的轻小说作品改编而成的同名电视动画。动画由日本放送协会（NHK）出品。于2013年10月5日起在NHK教育频道播放，全25话。

  
动画第二季《记录的地平线第二季》在2014年10月4日起在NHK教育频道正式播放，动画改由STUDIO DEEN负责制作。
