As Time Goes By_ 是电影《卡萨布兰卡》的主题曲，由Herman Hupfeld（赫尔曼·赫普费尔德）词曲。  
  
北非谍影（英语：Casablanca），也译为卡萨布兰卡，是1942年上映的一部美国电影，世界史上最成功的经典电影之一。本片荣获1944年奥斯卡的最佳影片、最佳导演，和最佳改编剧本奖。这部电影的角色、剧情、以及主题曲都成为了一种文化标志。2007年美国编剧协会公布影史“最伟大的101部电影剧本”名单，由“卡萨布兰卡”夺冠。
