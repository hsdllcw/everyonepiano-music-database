_乘着歌声的翅膀_ 原为德国诗人海涅（Christian Johann Heinrich Heine）所作的一首诗，因门德尔松为其谱了曲（Op. 34,
No. 2）而广为传播。其旋律舒缓、温柔、甜蜜，通常由女高音演唱。

全曲以清畅的旋律和由分解和弦构成的柔美的伴奏，描绘了一幅温馨而富有浪漫主义色彩的图景--
乘着歌声的翅膀，跟亲爱的人一起前往恒河岸旁，在开满红花、玉莲、玫瑰、紫罗兰的宁静月夜，听着远处圣河发出的潺潺涛声，在椰林中饱享爱的欢悦、憧憬幸福的梦……
曲中不时出现的下行大跳音程，生动地渲染了这美丽动人的情景。
