_卡农_
，这个版本非常简单，是C大调初学者版。右手没有了十六分音符，大多是以四分音符和八分音符为主，左手全部都是非常规律的和弦样式。如果是学习过我们魔鬼训练营的朋友，弹奏键盘钢琴的话，基本都可以秒过。所以喜欢卡农的朋友可以试一试这个版本。

卡农，一种音乐谱曲技法，复调音乐。卡农的所有声部虽然都模仿一个声部，但不同高度的声部依一定间隔进入，造成一种此起彼伏，连绵不断的效果，轮唱也是一种卡农。在卡农中，最先出现的旋律是导句，以后模仿的是答句。根据各声部高度不同的音程差，可分为同度卡农，五度卡农，四度卡农等；根据间隔的时间长短，可分为一小节卡农，两小节卡农等；此外还有伴奏卡农，转位卡农，逆行卡农，反行卡农等各种手法。
