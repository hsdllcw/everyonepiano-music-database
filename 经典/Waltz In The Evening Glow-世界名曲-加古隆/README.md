_Waltz In The Evening Glow_
是由日本当代音乐前辈级大师加古隆演奏。加古隆1971年前往法国留学，在欧美游学多年，出版有多张作品，风格涉及广泛，古典、jazz和当代newage音乐都有触及。加古隆经常以钢琴独奏的身份在欧美举行演出，所以他的钢琴技艺有相当的水准。

加古隆是目前在世界能够听到的最好的钢琴家之一，他那令人惊叹的和声与音色配置的感觉，将钢琴提升到了可与管弦乐团相媲美的地位。他那无人可及的赋予旋律线以清晰和均衡的能力，正使他成为完全彻底的独奏者。
