_巴赫前奏曲_ 是巴洛克时期的德国作曲家巴赫所创作的钢琴曲。

前奏曲是音乐体裁的一种。前奏曲以前只是组曲之前的器乐引子，在演奏之前一般演奏家要即兴演奏一小段，逐渐演变成一种艺术形式。前奏曲的起源最早可追溯到15、16世纪，当时是曲琉特琴或管风琴作为一种“开场白”式的即兴演奏而来，常用和弦与走句交织而成，起到引入“正题”的作用。音乐巨匠巴赫等继承和发展了这种体裁，以前奏曲与赋格、组曲、合唱等连结成套的出现，从而使其取得了“合法”的地位。被举为“音乐圣经”的《十二平均律钢琴曲集》就是一套绝佳范例。这一时期前奏曲总的来说，仍具有引子的特点与即兴的手法。
