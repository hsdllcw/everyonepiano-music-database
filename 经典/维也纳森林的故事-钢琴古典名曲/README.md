

《 **维也纳森林的故事** 》圆舞曲是小约翰•施特劳斯代表作之一，这首圆舞曲用音符为我们描绘了维也纳茂密森林的美好景色、乡村生活的轻松氛围。

施特劳斯是土生土长的维也纳人，在维也纳郊区有一片森林，那里景色优美，吸引了大批人来游玩，于是施特劳斯由此而突发灵感，创作了这部继《蓝色多瑙河圆舞曲》后的又一部圆舞曲杰作。而这也是他献给故乡的赞歌。

这部《 _维也纳森林的故事_ 》圆舞曲的特色在于曲子加入了奥地利民族特色的民族乐器--
齐特尔琴。到了曲子的中段，我们会听到一个不一样的拨弦乐器的清脆的声音，那就是齐特尔琴，据说是奥地利最古老的拨弦乐器，而且造价昂贵。这首圆舞曲以一段很长的序奏开始，之后圆号和木管乐器吹奏出来一段优美的牧歌。这时候大提琴奏出了乐曲的第一个主题，好似森林美妙清晨的情景。乐曲行进着，齐特尔琴的独奏插入，使得乐曲音色更加丰富。全曲在快慢节奏的变化中，展现了维也纳森林的优美景色，让人心旷神怡。
