

“晚祷”这首古典音乐选自童话歌剧《亨泽尔和格蕾泰尔》，又译《汉泽尔与格蕾太尔》。由德国后浪漫主义作曲家-
洪佩尔丁克作曲。作曲家原本只是根据格林童话的同名故事写了《跳舞二重唱》，反响不错，后来竟由此写成了歌剧。他一生著有7部歌剧，其中最优秀的就是根据格林童话写成的童话歌剧《亨泽尔和格蕾泰尔》，这是他的第1部歌剧，1893年12月23日由R.施特劳斯指挥在魏玛首演,大获成功,一举蜚声世界。这部歌剧主题清新明快,曲调妙趣横生,管弦乐手法变化多端。剧中似乎出现许多民歌，然而许多听似民歌的曲调其实是作曲家本人创作的，却没有破坏这部童话剧的民族风格，最著名的例子之一就是这首《晚祷》。同时许多剧中的旋律也成了德语民歌。让我们一起来听一下这首清新安静的音乐吧~
