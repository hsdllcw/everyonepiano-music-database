_Just Blue_
是中央电视台栏目《动物世界》的片头曲。当时被央视音乐编辑看中，结果风靡全中国，令亿万观众如醉如痴！许多年过去了，《动物世界》的收视率与影响力已雄风不在，而片头曲却还是这首SPACE乐队的《Just
Blue》，此曲也成为了国人心中的经典与美好回忆。

《动物世界》栏目从1981年12月31日开播。主旨在于向电视观众介绍大自然中的种种动植物，使观众足不出户就可以了解和认识地球上生存的各种生命，认识自然对人类的影响。1994年，中央电视台的编委会提议，在《动物世界》的基础上，衍生出一个更符合世界环境与发展理念的杂志性专题的栏目——《人与自然》。
