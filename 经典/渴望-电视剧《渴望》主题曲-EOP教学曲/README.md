

**渴望双手简谱** 和五线谱完全对应，是EOP教学曲。

《 _渴望_ 》是毛阿敏演唱的当年不知感动了多少人的同名电视剧《渴望》的主题曲。后来收录于毛阿敏的同名专辑之中。

《渴望》这部电视剧是1990年出品的。该剧成为轰动全国感动千万人的电视剧，被称为中国电视剧发展的历史性转折的里程碑，它创下的巅峰效应成为一个时代的神话。

歌词下方就是 _渴望钢琴谱_ ，感兴趣的朋友可以下载使用。
