

**知音**
是歌唱艺术家李谷一的歌曲，同时此曲也是电影《知音》的插曲。曾经登上1983年首届春节联欢晚会的舞台，歌词内容十分切合“铁血与浪漫相映，豪情与柔美相融”的主题，既赞颂了蔡锷拯救祖国的高尚情怀和坚忍不拔、勇敢机智的英雄品质，又表现了小凤仙对蔡锷忠贞不渝的爱情。

  
歌词下方是 _知音钢琴谱_ ，希望大家喜欢。
