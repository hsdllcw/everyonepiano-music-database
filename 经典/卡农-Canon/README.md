_卡农_
的所有声部虽然都模仿同一个声部，但不同高度的声部依一定间隔进入，造成一种此起彼伏的效果，连绵不断的效果，轮唱也是一种卡农。在卡农中，最先出现的旋律是导句，以后模仿的是答句。

简单的讲，卡农（canon）是一种作曲手法，意思是：“同样的旋律间隔两拍或一小节、两小节不等先后演奏”。“卡农”就是根据严格模仿的原则，用一个或更多的声部相距一定的拍子模仿原有旋律的曲式。至于用什么乐器，几个声部，就要随作曲家的便了，一般来说，2-6个声部就差不多了，要是非有人写个几十个声部也未尝不可，倒是现代音乐里常有的事。

我们经常听到的轮唱曲就是卡农曲的一种。卡农同时也指以此种技法创作出来的音乐作品，比如巴赫的《五首卡农变奏曲》。在历史上所有的卡农曲当中，帕海贝尔《卡农Canon》最为著名，一般人说起《卡农Canon》都是指这首。
