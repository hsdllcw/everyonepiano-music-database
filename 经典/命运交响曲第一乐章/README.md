命运交响曲（Fate Symphony）即贝多芬c小调第五交响曲，作品67号（Symphony No. 5 in C minor, Op.
67），德国作曲家路德维希·凡·贝多芬最为著名的作品之一，完成于1805年末至1808年初。此曲声望之高，演出次数之多，可谓"交响曲之冠"。

该乐曲是 _命运交响曲第一乐章_ 。全曲贯穿着命运的敲门声“PomPomPom-
Pom~~”，在宏伟的结构、流畅的音乐线条里，充满惊人的张力与力度，气势磅礴，使每个段落都充满生气。乐曲中所激发的强大生命力与命运抗争的执着，实在让人从心底感动。

贝多芬在交响曲第一乐章的开头，写下一句引人深思的警语：“命运在敲门”，从而被引用为本交响曲具有吸引力的标题。作品的这一主题贯穿全曲，使人感受到一种无可言喻的感动与震撼。下面是
