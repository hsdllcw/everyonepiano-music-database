

《 **笑傲江湖**
》，作家金庸1967年开始创作的一部作品，1969年完成，属于其后期作品，最初连载于《明报》。“笑傲江湖”源自吴承恩著《西游记》中的一句词。该书没有时代背景，“类似的情景可以发生在任何朝代”。笑傲江湖折射政治斗争，同时也表露对斗争的哀叹，具有一定的政治寓意。

拍摄和的电影电视剧版本有很多，相信1996年香港TVB版吕颂贤，梁艺玲主演的版本是其中比较经典的一部。无论哪个版本，其中的经典插曲《笑傲江湖》都是必不可少的。

此版《 _笑傲江湖_ 》是香港无线电视出品的古装武侠电视剧，改编自金庸同名小说。描述武林派别为追求技术，不惜争夺武学密笈，引致江湖中腥风血雨的故事。

刘正风是金庸武侠小说《笑傲江湖》中的人物。衡山派，酷爱音律，擅长吹箫，与曲洋合奏《笑傲江湖》之曲。本欲“金盆洗手”，投身仕途，不再过问江湖世事，但却遭到左冷禅的极力反对，最后丁勉、陆柏掌力震伤，与曲洋自绝经脉而死。
