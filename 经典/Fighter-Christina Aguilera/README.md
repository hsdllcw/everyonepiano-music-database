

**Fighter** 是美国女歌手、演员Christina Aguilera的歌曲。Christina
Aguilera（克里斯蒂娜·阿奎莱拉）是一位坐拥五座格莱美奖和八首冠军单曲的超级巨星。克里斯蒂娜在主唱迪士尼卡通电影花木兰的主题曲《Reflection》后和唱片公司RCA
Records签约，凭借首张同名专辑《克里斯蒂娜·阿奎莱拉》（1999年）的浑厚嗓音和亮丽外表走红。亦因能歌善舞及创作才华凭首张专辑便赢得格莱美最佳新人奖头衔。

  
歌词下方是 _Fighter钢琴谱_ ，希望大家喜欢。
