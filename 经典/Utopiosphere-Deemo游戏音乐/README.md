

**Utopiosphere双手简谱** 和五线谱完全对应。

_Utopiosphere_
，中文：乌托邦星球，是Mili创作的音乐作品，2013年1月发布于youtube。由momocashew填词及演唱。HAMO作曲及编曲。歌曲PV的背景图片由レンツァイ绘画。

为2013年11月发售的音乐游戏Deemo的演奏曲目。

Utopiosphere为作者自造词，Utopioa（乌托邦的）+sphere（星球）。Melodia,Photosynthesia,Fantasia等词，是为了押韵而把结尾的y改为ia。

歌词下方是 _Utopiosphere钢琴谱_ ，大家可以免费下载学习。
