**上海滩**
是叶丽仪演唱的一首歌曲。由顾嘉辉作曲，黄沾填词，收录于叶丽仪1980年发行的同名专辑《上海滩》中。这首歌曲是1980年周润发、赵雅芝主演的电视剧《上海滩》的主题曲。1980年，这首歌曲获得了第三届十大中文金曲奖。  
  
这首歌曲婉转动听，有强烈的高低起伏，叶丽仪浑厚奔放的嗓音唱出了上海滩纷纷扰攘的争斗和刻骨柔情。《 _上海滩_
》里暗藏了很多的情怀，从这首歌里可以听到家国、故园等声音，也可以在歌曲中品尝到特有的人生百味。
