

**甜蜜蜜**
出自1979年邓丽君的国语专辑《难忘的一天》。此曲大受欢迎，风头一度盖过原本作为此张专辑主打的专辑同名歌曲《难忘的一天》，此后邓丽君的《甜蜜蜜》作为华语乐坛的优秀音乐作品传遍海内外，成为一首在华人世界经久不衰的流行音乐，《
_甜蜜蜜_ 》也因此被人们认定为邓丽君的经典代表作之一。

  
歌词下方是甜蜜蜜钢琴谱，希望大家喜欢。
