

**秘密的庭院** ，又名Jardin
Secret，是著名钢琴家理查德▪克莱德曼发布的专辑《水边的阿狄丽娜》中的一首钢琴曲。这首曲子中透出的不仅仅是凄美、忧郁，更多的是一种隐隐的神秘。

理查德•克莱德曼所演奏的曲目，大部分是Paulde Senneville与Olivier
Toussaint两位作曲家的曲子。此二人曾为红歌星作曲，也为电影配乐，后因欣赏理查德的才华转而专门替他写曲。他的乐念十分容易了解，音和规则没有复杂之处，节奏是缓慢的四拍圆舞曲系统，有时根据曲子使用电子琴表现出轻快感并有幻想的印象，把理查•克莱德曼的魅力完全表露无遗。
