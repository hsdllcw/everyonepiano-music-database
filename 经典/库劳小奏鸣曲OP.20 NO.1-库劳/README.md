_库劳小奏鸣曲OP.20 NO.1_
是德国著名的钢琴演奏家库劳的曲目。他是一位有很高造诣的职业长笛演奏家，为长笛创作的乐曲流传至今。库劳一生的创作包括歌剧、室内乐、钢琴曲、长笛曲、小提琴曲和许多声乐乐曲，但大多已散失，其中以钢琴小奏鸣曲最为出色，具有很高的练习价值。

库劳的钢琴小奏鸣曲虽然简易、短小，不具有深刻的艺术内涵，但是它们同样是建立在维也纳古典主义音乐结构的基础之上，体现了典型的古典主义风格的作品。通过这些小奏鸣曲的学习，不但可以从中了解古典乐派钢琴奏鸣曲的曲式结构，而且可以学习和训练敏锐、轻快地触键，准确严密的节奏，均匀、透明的音色，稳定统一的速度，典雅、纯净的情趣。同时，也为进一步学习海顿、莫扎特、贝多芬乃至舒伯特的钢琴奏鸣曲作好准备。库劳创作的钢琴小奏鸣曲并非仅12首，但是大师逝世前两年，即1830年住宅的一场大火烧毁不少创作手稿，留下了难以弥补的历史遗憾。
