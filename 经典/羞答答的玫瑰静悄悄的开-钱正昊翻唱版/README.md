

2019年3月14日， **钱正昊** 个人首支翻唱单曲《 **羞答答的玫瑰静悄悄地开** 》暖心上线，在 **白色情人节** _（3月14日）_
当天，这位2001年出生，刚满18岁的新生代音乐人，用全新演绎的方式（加入rap元素），把这首经典的情歌，作为情人节礼物送给了所有歌迷。

这首曲子来自一位18岁小男孩对爱情的理解，在经典中注入了少年的青春活力，让人一听就印象深刻。小编听过很多版本，其他人的版本，都多少有一些悲伤的味道，让人感觉这朵玫瑰花要凋谢了。而这个大男孩用他辨识度极高的嗓音，把这朵玫瑰花羞答答的样子表现的淋淋尽致，有兴趣的朋友可以到[
**网易云**](https://music.163.com/#/song?id=1346104347&autoplay=true
"羞答答的玫瑰静悄悄地开")听听这首曲子。
