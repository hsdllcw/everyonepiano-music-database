

**绒花**
是电影《芳华》的片尾曲，此曲创作于1979年。这首歌曲由刘国富、田农作词，王酩作曲，最初由李谷一首唱，曲调新颖流畅。曾在1980“听众喜爱的广播歌曲”评选活动中获奖。

《芳华》是由浙江东阳美拉传媒有限公司出品的剧情片，根据严歌苓同名小说改编，以1970至1980年代为背景，讲述了在充满理想和激情的军队文工团，一群正值芳华的青春少年，经历着成长中的爱情萌发与充斥着变数的人生命运故事。
