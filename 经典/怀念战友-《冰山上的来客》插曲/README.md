

### 简介：

《怀念战友》 - 创作背景电影

《冰山上的来客》中的几首插曲是雷振邦的代表作，具有浓郁的新疆民歌风味，旋律婉转动听，深受广大群众欢迎、历经近半个世纪传唱不衰的歌曲。

《怀念战友》是李俊广“单飞”加盟雨林后的首张个人“民歌”大碟。当中收录包括“花儿为什么这样红”、“怀念战友”、“送别”、“一剪梅”等多首经典民歌。
