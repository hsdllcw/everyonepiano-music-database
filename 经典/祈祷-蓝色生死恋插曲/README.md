

**祈祷** 是2000年韩剧系列电视剧《蓝色生死恋》插曲，由郑日荣演唱。《蓝色生死恋》（朝鲜语：가을동화，英语：Autumn in My
Heart）是韩国KBS自2000年9月18日起播放的月火迷你连续剧，由尹锡湖执导的四季系列（蓝色生死恋系列）的第一部曲，宋承宪、宋慧乔、元斌、韩彩英及韩娜娜等主演。

  
歌词下方是 _祈祷钢琴谱_ ，希望大家喜欢。
