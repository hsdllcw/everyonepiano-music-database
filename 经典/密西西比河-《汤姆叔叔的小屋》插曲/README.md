

### 简介：

《密西西比河》是美国电影《汤姆叔叔的小屋》插曲。这部电影是根据斯托夫人的小说“汤姆叔叔的小屋”拍摄的。描述了美国南北战争前，黑奴的悲惨遭遇，以及以林肯为代表的北方白人基督徒为了废黜美国黑奴制与黑人奴隶共同抗争的悲壮历史。

而林肯总统被南方白人奴隶主枪手刺杀身亡时，手里拿着的一本书，就是“汤姆叔叔的小屋”。美国历史上第一位黑人总统奥巴马的出现，向世界宣告了美国自由民主制度的伟大胜利！
