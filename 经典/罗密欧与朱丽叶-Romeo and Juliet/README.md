

由理查德·克莱德曼演奏普罗科菲耶夫作曲的钢琴曲《 **罗密欧与朱丽叶** 》堪称经典之作，其影响力不亚于《梦中的婚礼》、《秋日私语》等。

### 罗密欧与朱丽叶歌词：

由八月秋寻依据该钢琴曲填词，又名为《秋之思念》；  
作曲：普罗科菲耶夫； 作词：八月秋寻；  
我看着你，  
走过街头，  
离开以后，  
我的心头,  
忧愁谁能休；  
西风东流  
落叶满秋,  
时光,  
匆匆,  
寂寞有谁能看得清透,  
不懂；  
这一生,  
你的身影  
在我心中,  
永远守侯，  
我最美的爱，  
痴守；

走过街头,
