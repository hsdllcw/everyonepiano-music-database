这首曲子原作于1914年。是墨西哥作曲家庞塞的作品。是非常著名的一首小曲。由于曲调极为迷人，所以被改编成小提琴和其他乐器的独奏曲，以及改编成管弦乐曲。之后，被法国钢琴演奏家理查德克莱德曼改编。  
  
理查德·克莱德曼（Richard
Clayderman，1953年12月28日－），法国钢琴演奏家。本名菲利浦·罗贝路易·帕杰斯。理查德·克莱德曼不是以古典钢琴曲而见长，而是主要弹奏一些现代爱情曲目。
