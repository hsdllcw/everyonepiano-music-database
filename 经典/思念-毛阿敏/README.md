

**思念**
是歌手毛阿敏的专辑《歌坛大姐大》中的一首歌曲。这首歌曲由乔羽作词，谷建芬作曲。毛阿敏（1963年3月1日－）壮族，生于中国上海，是1980年代和1990年代中国大陆流行乐坛天后级女歌唱家。

  
歌词下方是 _思念钢琴谱_ ，希望大家喜欢。
