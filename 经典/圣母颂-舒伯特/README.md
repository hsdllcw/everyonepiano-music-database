

**圣母颂双手简谱** 和五线谱完全对应。

_圣母颂_ 是舒伯特在1825年根据英国诗人瓦尔特•司各特的叙事长诗《湖边夫人》中的《爱伦之歌》谱写而成。

歌曲抒发了叙事诗主人少女爱伦祈求圣母饶恕其父罪行的纯真感情。歌曲采用分节歌形式，曲调柔美委婉、纯净朴实，音乐表情细腻丰满，表现了作者对真善美的向往。由于这首歌曲的艺术魅力，后人将其改编成器乐曲演奏，又以小提琴独奏及弦乐演奏主旋律、竖琴伴奏的谱本流传较广。
