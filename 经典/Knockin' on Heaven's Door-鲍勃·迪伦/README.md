

**Knockin' on Heaven's Door** （敲响天堂之门）是民谣诗人Bob
Dylan的传世之作，同时也是1973年上映电影《比利小子》的插曲。Bon jovi、枪炮与玫瑰乐队（Guns N' Roses）、周华健、Eric
Clapton、艾薇儿、Roger Waters、罗琦、等都有不同的翻唱版本。

  
这首歌在片中是对人性中暴力面的一个探索,到底杀戮对不对,有人为了正义和法律而杀戮,有人被生活逼着去杀戮,有人天生就喜欢杀戮。而杀戮本不应存在。

  
由于20世纪末美国对外的战争不断,众多摇滚歌手都反对政府的这一行为,鲍勃·迪伦也创作了一些反战歌曲,后来人们把这首经典歌曲也归到了鲍勃·迪伦的反战歌曲当中,广为传诵。

  
歌名 _Knockin' on Heaven's Door_
是出自旧约的圣经的列王篇："以色列众人听见耶罗波安回来了，就打发人去请他到会众面前，立他作以色列众人的王。除了犹大支派以外，没有顺从大卫家的。
