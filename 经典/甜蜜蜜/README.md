

《甜蜜蜜》是由庄奴作词，邓丽君演唱。1979年11月，宝丽金公司发行的专辑《甜蜜蜜》的第一主打歌曲，当年就售出超过百万张,是1979年邓丽君获得的三张白金唱片之一！从那个时候开始，很多人的家里开始出现邓丽君的大幅海报。流行音乐开始在中国普及开来！到现在甜蜜蜜仍然是最经典的永恒金曲，众多新辈歌手拿来翻唱。
