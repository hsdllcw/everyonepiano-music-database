

**Cinema Paradiso** 是意大利导演朱赛贝•托纳多雷所执导的一部电影天堂电影院中的曲子。《天堂电影院》（意大利语：Nuovo Cinema
Paradiso）1988年首度在意大利上映。雅克•佩兰（Jacques Perrin）、菲利普•诺瓦雷（Philippe
Noiret）、莱奥波尔•多里雅斯特（Leopoldo Trieste）、马哥•莱昂纳迪（Marco
Leonardi）和萨尔瓦多•卡西奥（Salvatore Cascio）皆在电影中演出。电影配乐则由恩尼奥•莫里科内与他的儿子安德烈•莫里科内（Andrea
Morricone）所创作。

  
另外， _Cinema Paradiso_ 此曲由意大利作曲家Ennio Morricone创作。Ennio
Morricone曾为超过500部的电影电视写过配乐。2007年他获得奥斯卡终身成就奖，成为第二位获此殊荣的作曲家。他获得两次格莱美奖，两次金球奖，五次英国电影和电视艺术学院奖等多项音乐奖项。
