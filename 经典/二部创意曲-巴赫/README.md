《 **二部创意曲**
》是巴赫在1720-1722年间为童年的儿子威廉•弗利德曼•巴赫写作的一套练习（即《为威廉•弗里德曼•巴赫而作的键盘小曲》）中的一部分，曾命名为“前奏曲”，后改用“创意曲”。

《 _二部创意曲_
》是以一个短小的主题动机为材料，运用模仿、卡农、对位等手法进行变化发展、创造新意的小曲，多数以三个部分构成，少数为两个部分。正如巴赫在《1723年手稿》的序文中所指出，这些用于教学的作品是为了让“古钢琴的爱好者、学习者不但有能力把二声部演奏得美妙，而且在熟练之后，亦有能力正确、巧妙地演奏三声部。同时，还为获得作品创作的展开技巧，领会如歌的演奏法，产生作曲的强烈愿望提示了明确的方法。”
