

### 简介：

圆舞曲，又称华尔兹。它的来源有不同的看法：有德国起源论，有法国起源论还有就是意大利起源。它是一种三拍子的旋转的舞蹈，情绪欢快，活泼。华尔兹一词在1780年左右出现，但是旋转性的舞蹈早得多就在民间出现，但曾还遭到过禁止。

中国人一般都知道"蹦嚓嚓"三拍的周期的反复，这也是圆舞曲其中的特点。在德奥地区和在意大利的起源的民间的舞曲就是圆舞曲的源，十六世纪就有了，到了十八世纪时圆舞曲就成了主导的舞蹈体材。在整个十九世纪它成就了压倒一切的优势。二十世纪直到今天仍然方兴未艾。

杜鸣心的《青年圆舞曲》，这是粉碎"四人帮"后，于1976年产生的第一首圆舞曲。它以欢唱的旋律，明快的节奏表达了人民庆贺打倒"四人帮"胜利的欢乐心情。乐曲充满活力，具有青年人的性格特色。全曲为A,B,A三段体的曲式结构.
在强有力的前奏后, 出现了第一段的旋律, 随后又出现了它的派生旋律.

乐曲经过短暂的过渡, 进入中段, 分为三小段, 第一段是抒情的旋律，富于歌唱性，描绘了青年们纯洁质朴的心灵，B的小中段与A的第二段在节奏上有共同之处,
休止符号的运用使旋律显得更为欢跳. B的第三小段又再现了第一小段, 但更为热情奔放, 为再现全曲的A段, 起到承前启后的巧妙作用.
A段的再现是用小号来演奏的. 除了具有欢乐的因素外. 更具有召唤性, 逐步达到全曲的高潮后结束, 象征着青年们意气风发, 投身祖国建设的奔放热情.
