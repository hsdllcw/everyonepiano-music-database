

**千古**
是古装玄幻大型仙侠电视连续剧《花千骨》第25~56集的片头曲，由创作才子许嵩作词、作曲，纯净歌者阿兰献声演唱。这也是迄今为止最被歌迷期待的主题曲。经工作人员透露，许嵩是首次为仙侠玄幻题材电视剧创作歌曲，为此许嵩认真研究剧本，做足功课，将所擅长的中国风发挥到极致。一曲“夏蝉冬雪，不过轮回一瞥；悟道修炼，不问一生缘劫；白纸画卷，寥寥几笔绘江湖深浅；难绘你；不染纤尘的容颜”道出人生在世，无尽执念之情。

阿兰用直抵人心纯净的歌声激荡着无数人的心灵，始终追寻着心中从未改变的承诺，让我们在不知不觉间，感到这个外表看似柔弱，实则坚韧无比的女孩，竟带着一颗不甘的心灵走了这么远，让我们不由自主联想到剧中种种固执而纯粹的爱情。《
_千古_ 》这首歌曲温婉动听，荡气回肠中又饱含着剧中人物的恋恋情深，配上阿兰不染纤尘的演绎，将剧中白子画和小骨的动人故事演绎的淋漓尽致、感人至深。

歌词下方是 _千古钢琴谱_ ，希望大家喜欢。
