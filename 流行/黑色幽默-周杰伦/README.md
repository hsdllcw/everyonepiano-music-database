

**黑色幽默双手简谱** 和五线谱完全对应，是EOP教学曲。双手简谱中间有变调，大家练的时候要注意下。

  

《黑色幽默》，是周杰伦第一张音乐专辑《JAY》中的第六首歌曲，由周杰伦谱曲、填词，悲伤情歌，2000年11月6日阿尔法音乐有限公司发行。

  

《 _黑色幽默_
》深刻的描述着周杰伦内心感情世界的一环，剖白的记忆着女友离去时，那一刻的心情，全新大陆十二人弦乐团加上传统旧式Band的表现，呈现出一种英式的复古风格。

  

彭羚于2002年翻唱了《 _黑色幽默_ 》，并且收录于《给我唱过的男孩们》专辑。
