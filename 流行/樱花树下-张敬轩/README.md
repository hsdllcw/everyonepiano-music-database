

**樱花树下** 是一首张敬轩演唱的歌曲，该首歌曲发行于2013年04月23日，出自于其专辑《Imperfect Live 2013
Collection》。张敬轩（Hins
Cheung），创作歌手，主要在香港发展。祖籍北京，出生于广州，中学毕业于广州市五羊金融财经职业学校。2002年10月张敬轩签约环球唱片。2013年7月，与英皇娱乐签约。

  
歌词下方是 _樱花树下钢琴谱_ ，希望大家喜欢。
