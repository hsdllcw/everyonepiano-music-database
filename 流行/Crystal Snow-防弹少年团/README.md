防弹少年团（BTS），是韩国BigHit
Entertainment旗下2013年出道的男子演唱组合，由金南俊（RM）、金硕珍（JIN）、闵玧其（SUGA）、郑号锡（J-HOPE）、朴智旻（JIMIN）、金泰亨（V）、田柾国（JUNG
KOOK）7名成员组成。他们抓住了与自己年龄相符合的主题，制作出了属于防弹的特色音乐；此后成员们通过自作曲与制作，准确定位了防弹少年团的音乐风格，连载系列制作专辑也成为他们的特色之一。
