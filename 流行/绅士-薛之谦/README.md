_绅士_
是薛之谦创作并演唱的歌曲，收录在薛之谦2015年EP《绅士》中。歌曲讲述了男主角喜欢着女主角却始终无法表达出心意，只能以“好朋友”身份来关心因男友车祸去世伤心不已的女主角。

歌曲MV：清早出发宜兰抵达阴阳海边的山顶民宿，天开始下起蒙蒙雨，为导演设计的“雨戏”加了一把力。从首尔空降的女演员裴涩琪，这一次颠覆《小苹果》的“复古女神”形象，不跳可爱舞蹈，挑战感情戏。雨越下越大，《绅士》
的纠结之情越演越烈，导演加足马力出动洒水车，双管齐下。

歌词下方是 _绅士钢琴谱_ ，希望大家喜欢。
