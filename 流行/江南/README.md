

《 **江南**
》是新加坡歌手林俊杰演唱的一首歌曲，由林俊杰作曲，李瑞洵，陈少琪填词，收录于林俊杰2004年发行的专辑《第二天堂》中。这首歌曲另有重新填词的粤语版本，收录于林俊杰2007年发行的专辑《西界》中。  
  
2005年， _江南_ 获得CCTV-MTV音乐盛典年度最佳单曲、百事音乐风云榜港台十大金曲、第十二届中国歌曲排行榜港台最受欢迎歌曲等奖项。
