

《 **白桦林**
》，女歌手叶蓓、男歌手朴树都曾演唱过。1999年，当时的麦田音乐推出"红白蓝"3张概念专辑，蓝色为叶蓓的《纯真年代》，白色为朴树的《我去2000年》，两人演唱的《
_白桦林_ 》就分别收录于这2张专辑中，均于当年1月发行。
