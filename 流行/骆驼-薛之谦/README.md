_骆驼_
是由甘世佳作词，郭顶作曲，薛之谦演唱的一首歌曲，发行于2017年11月16日。这首歌曲通过沙漠和骆驼这两个神秘的意象，影射出现实生活中的城市和人，用全新的视角深刻的解读了孤独和自由的意义。

歌曲随着笛子和吉他温柔的和弦搭配，徐徐开启了《骆驼》对自由的诉说和对故事的阐述。这种柔软持续包裹着歌曲的第一段主歌部分，薛之谦用轻缓的咬字拨开这沙漠故事的帷幕，仿佛那烈日烘烤下的无垠沙漠映入了眼帘。而曲目进入副歌前近似留白的简约处理，与副歌编曲突然饱满起来的层次形成强烈的反差。薛之谦在此处使用坚定而充满力量的声音，带领听众进入故事的下一篇章。充满异域风情的间奏将沙漠景象刻画的更加细致立体，引导出后半阙不断在繁复和简约中更迭的编曲风格，好像日间酷热、夜晚冰冷的沙漠无情的交替无常，而我们就似在这广袤沙漠里渺小的骆驼。迷失在无限的壮阔中而形成的孤独感不断充斥着歌曲的每个细节。后半段情节逐渐进入尾声，将满溢的浓情一一收敛，用极简的笔墨为歌曲画上了句号。
