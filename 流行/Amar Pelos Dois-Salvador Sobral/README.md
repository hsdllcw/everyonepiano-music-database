_Amar Pelos Dois_ 是葡萄牙歌手Salvador Sobral所演唱的一首歌。Salvador
Sobral，曾参加欧洲歌唱大赛，并成功夺得大赛冠军，也创造了葡萄牙参与欧唱的最佳战绩。Salvador

Sobral的出色发挥夺冠，也囊获了下一届欧唱的主办权。
