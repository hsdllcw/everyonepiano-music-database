

**When You're Gone** 这首歌是艾薇儿第三张专辑《The Best Damn Thing》里的一首歌曲。《The Best Damn
Thing》是她人生中一个新的台阶，她已经从年少成名的焦虑中走出，时刻准备狂欢，摇滚这个世界，哪怕是跳舞。正如她所指的，这是她有史以来做得最棒的作品The
Best Dame Thing。
