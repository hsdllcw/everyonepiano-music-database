《空の向こうに》（天空的彼端 / 天空的彼方）是游戏《交响乐之雨》（Symphonic Rain）片头曲。  
  
《 _交响乐之雨_ 》（日语：シンフォニック=レイン，英语：Symphonic
Rain）是日本工画堂工作室所发行的恋爱冒险音乐游戏，于2004年3月26日在日本地区贩售。繁体与简体中文版发售日期分别是2004年11月11日及2005年5月25日贩售，皆由光谱资讯代理。在电脑键盘上演奏虚构的乐器“符德鲁琴”的音乐冒险游戏，游戏最大的魅力之处，就在于音乐所营造出的雨城恋爱故事气氛。  
  
工画堂的音乐恋爱游戏，一直是玩家心中最爱的选择，承继了融合角色扮演与音乐游戏的《天使演唱会》、《AS～天使小夜曲》系列之后，工画堂更重金打造了「交响乐之雨」，除了承袭之前的系统功能之外，这一次策划了全新的游戏舞台、人物角色、故事剧情，再邀请业界著名的唱作人来担任游戏音乐制作，更有新颖的粉水彩插画家来一同营造气氛。本作为冈崎律子女士的遗作，是冈崎律子女士在病榻上完成的最后一部作品。  
  
游戏最大的魅力之处，就在于音乐所营造出的雨城恋爱故事气氛，工画堂邀请到的音乐制作人-
冈崎律子，曾经担任多部脍炙人口的卡通音乐创作者，像是在日本荣获众多奖项的《水果篮子》，她就一手包办了片头片尾曲，其它还有幻影天使、彩梦芭蕾等。不仅作词作曲功力一流，甚至还能担任演唱者，夸她是日本游戏业界的音乐才女一点也不为过。而“交响乐之雨”里也重用了她的能力，除了一手包办游戏中共十首歌曲的所有作词作曲之外，还担纲演唱片头、片尾曲，而且游戏的结局歌曲也会随着不同的女孩而有各种的片尾曲产生，这更是玩家的另一种期待了。一个由雨声以及背景流泻出的优美隽永音乐所创造出的梦幻寂寞『音乐空间』，让冈崎律子所演奏的凄美旋律，更能完美地诠释出业界拥有傲人成绩的Q'tron所谱出的故事剧本。  
  
片头曲《 _空の向こうに_ 》就是一首很美的音乐。歌词下方提供了 _空の向こうに钢琴谱_ ，希望大家喜欢。
