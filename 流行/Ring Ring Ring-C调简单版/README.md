

《 **Ring Ring Ring** 》是由张家玮作词，张博彦作曲，S.H.E演唱的歌曲，发行于2006年7月21日。收录于专辑《Forever
新歌+精选》中。

歌曲描写时下年轻人用手机谈恋爱的写照。S.H.E表示，其实自己的也不是那种情窦初开的年纪，但因为这歌相当可爱，因此在配唱这首RING RING
RING时，就把想象自己是个刚情窦初开期待又怕受伤害、用手机谈恋爱的女孩，还有加上一些口白，三个人在录音室玩的很开心，整首歌也因此听来相当可爱俏皮。
