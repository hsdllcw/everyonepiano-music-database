

《 **第三十八年夏至**
》出自河图首张个人专辑《风起天阑》，歌曲开始即传出沙哑的京剧声，颇有岁月的气息，相继伴有河图独特的古风演唱，其中与京腔混搭相和，韵味十足。

这首歌在古风圈当中应当是非常出名的，其翻唱次数非常多。《 _第三十八年夏至_
》是以民国为创作背景的，写的是一个悲剧，爱而不得，从一开始就注定不会等来的人。而其背景故事注定这首歌要以悲伤为基调，但是这首歌是以戏子的角度在对这段感情进行回忆，这时候时间早已如流水般不再复返了，再大的伤痛也早已失去了歇斯底里的力气了，一切都已慢慢沉淀、堆积在心底成为一道无法愈合的伤疤，所以《第三十八年夏至》这首歌虽伤，但却并不是撕心裂肺的，而是和缓的、轻柔的，犹如绵绵细针刺入心底一般，恍如局外人在说着他人的故事……
