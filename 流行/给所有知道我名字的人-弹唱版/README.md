

**给所有知道我名字的人**
是《我是歌手》第二季罗琦在决定退赛之后表演的最后一首歌曲。这首歌的原唱是著名摇滚音乐人赵传，收录在专辑《我是一只小小鸟》中，和我是一只小小鸟一样，拥有着的头顶苍天脚踩凌云的豪情壮志奋力朝着梦想的远方飞去。

《 _给所有知道我名字的人_
》是一首有情怀的歌，是每一位“疯狂摇滚”过歌手的光荣奖牌，这是一首带有王者风范的歌曲，李宗盛先生创作的歌词铿锵有力，简单又有分量，用质朴的歌词表达内心心情激动的波涛汹涌，以小见大足以显示歌词的魅力，振奋激动地旋律让人热血沸腾，这首歌下面的我们将会毫不犹豫的站起来挥动着双手合唱这首歌，告诉这些我们深爱着的歌手，你们将永远不会被我们遗忘。
