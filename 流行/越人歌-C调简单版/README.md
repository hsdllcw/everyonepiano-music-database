

**越人歌** 是由音乐人河图作曲编曲混编演奏演唱、Finale作词的古风歌曲，收录于河图第四张个人专辑《NL不分》中。

河图，湖南怀化人，原创音乐制作人、歌手，墨明棋妙原创音乐团队曲部成员。作曲编曲混编演唱，擅长演奏吉他、二胡等乐器， 曲风唱风独树一帜。
