

**选择双手简谱** 和五线谱完全对应。

_选择_ 是由叶倩文、林子祥演唱的经典歌曲，至今仍然有很高的传唱度，是华语音乐中最具影响力的对唱情歌歌曲之一。此曲由陈大力谱曲，陈大力和陈秀男填词。

叶倩文是一位实力派歌手，早年歌路有很强的欧美风格，但她走红的歌曲则充满台湾流行曲味道。而林子祥是香港乐坛高音方面很厉害的男歌手之一，他天生有一副极具冲击力和爆破力的嗓音，换句话说就是歌喉特别“呛”，几乎无法模仿。

歌词下方是 _选择钢琴谱_ ，大家可以免费下载学习。
