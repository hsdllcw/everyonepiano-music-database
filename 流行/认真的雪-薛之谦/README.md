

**认真的雪**
是薛之谦演唱的一首歌曲，由薛之谦填词、谱曲。收录在薛之谦2006年发行的同名专辑《薛之谦》中。2007年，该歌曲获得第十四届东方风云榜十大金曲奖。

  
在这首歌的创作初期，薛之谦多次提交给制作人过目，希望能得到对方的建议和意见，但是老师总是用一句“你放心大胆地去做”来打发薛之谦，为此闷闷不乐的谦谦直到同名专辑《薛之谦》制作结束时才完成了这首歌曲。

  
为了充分展现出《 _认真的雪_
》这首歌当中的伤感意境和悠长韵味，唱片公司特别请到了环球唱片御用的香港MV导演JACKY到上海帮薛之谦拍摄MV，随着歌曲中男女主角在雪中分离的凄美意境，基于导演JACKY“忠于情感”的要求，美女贾青献出了“荧幕初吻”。

  
歌词下方是 _认真的雪钢琴谱_ ，希望大家喜欢。
