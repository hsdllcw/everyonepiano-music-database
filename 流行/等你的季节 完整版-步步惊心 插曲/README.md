

**等你的季节双手简谱** 和五线谱完全对应。

_  
等你的季节_ 是电视剧《步步惊心》插曲，也是演员刘诗诗的第一首单曲。在2012年湖南卫视的跨年演唱会上吴奇隆和刘诗诗改编演绎了男女合唱的版本。

  
歌词下方是 _等你的季节钢琴谱_ ，希望大家喜欢。
