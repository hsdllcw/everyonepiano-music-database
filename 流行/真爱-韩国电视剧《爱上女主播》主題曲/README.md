

《 _真爱_
》这让首歌曲是韩国电视剧“爱上女主播”主題曲。娃的诱惑，原名《爱上女主播》，是韩国MBC电视台于2000年4月至7月期间播放的水木连续剧，由蔡琳、张东健、韩在石、金素妍、金廷恩等人领衔演出。下面是真爱钢琴谱，感兴趣的朋友可以使用。  
  

歌曲《 _真爱_
》是由权熙镇作词,韩国第一女子团体Fin.K.L演唱的。Fin.K.L在韩国乐坛的号召力和影响力依然无可比拟，众多新生代女子团体都以Fin.K.L为奋斗目标，但毫无疑问，FIN.K.L是仅有的也是无法超越的。同时也是韩国最受期待复合的女子合唱团体。  
  

《真爱》是一首很好听的歌曲。下方是 **真爱钢琴谱** ，希望大家喜欢。
