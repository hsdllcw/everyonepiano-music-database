

一个人可以没有天分，但只要肯坚持就会成功，这就是《 **笨小孩** 》告诉我们的道理。

《 _笨小孩_ 》是刘德华作词，高枫作曲，江建民编曲的一首歌曲，分刘德华独唱版和刘德华、柯受良、吴宗宪合唱版，是一首脍炙人口、颇具社会意义的经典曲目。

《笨小孩》是刘德华演唱的一首歌曲，由高枫等作曲，刘德华作词，江建民编曲。《笨小孩》的歌词中可以看出刘德华的成长、历程和踏实本性。歌词很简单，没有迂回没有说教，但很感人。描写的是困苦环境中长大的孩子，不畏艰险、力争上游的故事。
