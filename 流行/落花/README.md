_落花_ 是热播电视剧《美人心计》中的一首歌曲，由台湾著名艺人林心如演唱。

  
《落花》这首歌曲旋律朗朗上口，曲调朗朗上口，很容易学会，曲子中悠扬的笛子与心如柔美的声音结合，相得益彰。花形容女人的青春，女人一生中青春是最宝贵的，所以看花莫待花枝老，要珍惜年轻的日子，尽情地去爱，去感受刻骨铭心的爱恨，同时，相爱也是容易受伤的，所以受了伤，要相互安慰。  
  
这首歌由主演林心如来唱，似乎讲述了窦漪房的一生，流了许多泪，才明白真正的爱与恨，冷冷的夜里北风吹，试想一个瘦弱的女子，在这样寒冷的夜里，背负着整个宫廷和家族的爱恨，以及亲人的背叛，却找不到一个人来安慰，当初的誓言曾经是多么完美，此刻物是人非事事休，相思已经化成了灰，这样的意境难道不让人落泪，不让人心疼么？漪房本不想卷入宫廷斗争，可是一切身不由己，太多的牵绊，到死也没有解脱，一代佳人就这样在深宫中如落花一般凋零了，这就是汉室后宫中每个女子无法摆脱的命运。
