

**透明** 是邓紫棋演唱的一首歌曲，收录于专辑《摩天动物园》中。

此曲的歌词也是邓紫棋自己撰写的，讲述了，当一个女生爱上了一个男生之后，两个人相处的日子越来越久，但是自己活的好像透明人一样，渐渐的不被对方记在心里了，失去了自我这样的感慨，还有一系列伤感的感情，在mv里面，邓紫棋也亲自的出演女主角，一会儿和男生进行交流，一会儿变成
_透明_ 的样子，看起来令人感到异常的伤感。

邓紫棋的歌曲之所以能够引起共鸣，恐怕就是其中的歌词，打动了大众的一些伤感的心情吧。大家在恋爱或者是失恋的时候，总会有一些共同的感受。所以在听这种伤感的歌就会忍不住的联想到自己，紧接着再次的泪流满面，这就是歌曲的魅力，也是音乐给大家带来的共鸣，而邓紫棋正式坚守着，在音乐的战线上不停的创作，才会给大家带来更多更好听的音乐作品。
