

“雪落聆听二三世，梦回空庭烟雨间”不知道大家有没有被这样的意境所打动，这是吴亦凡新歌《 **贰叁** 》里的一句歌词。

作为一首情歌，吴亦凡在音乐里面加入了古风的元素，可谓满满的都是中国风啊！歌中的说唱部分，也加入了我们平时听到过的“吴亦凡风格”，由此可见，吴亦凡的用心之处。

而在微电影中，吴亦凡携手日本人气女星木光村希共同演绎了一段从古到今历经两世的奇幻恋情。MV讲的是将军与公主的故事，前世讲述了将军与公主爱而不得，公主最后为保护爱的人而死，而将军和蛮族的国王也因此受到了诅咒，他们以不死之身重生，每天午夜饱受锥心的折磨，为了解除诅咒，他们只有找到轮回转世的公主，并且和公主相爱才能解除诅咒。故事中，两人在一起的过程虽然几经周折，但好在最后的结局能够给人一点安慰，故事里转世的公主，失去了光明，不过还有将军的陪伴，哪怕在最后将军消失了，我想他一定会在公主看不到的地方，默默的守护，上一世是公主去救了将军，而这一世换将军来守护公主。
