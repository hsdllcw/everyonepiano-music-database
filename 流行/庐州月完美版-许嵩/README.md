

《 **庐州月**
》是著名原创音乐人许嵩（Vae）的一首歌曲，是许嵩第二张个人原创专辑《寻雾启示》中的曲目，这首中国风的歌曲作为新专辑的第二波主打曲目，于2009年12月10日在其官方网站vaecn上面首发。
