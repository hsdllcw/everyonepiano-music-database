

**走着走着花就开了双手简谱** 和五线谱完全对应。

_走着走着花就开了_ 是卓舒晨演唱的歌曲，于2018年12月15日发行。歌曲很阳光很正能量，听着让人觉得很舒服，仿佛让人行走在天上白云，地上花开。

歌词下方是 _走着走着花就开了钢琴谱_ ，大家可以免费下载学习。
