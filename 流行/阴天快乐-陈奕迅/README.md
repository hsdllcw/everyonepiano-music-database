_阴天快乐_
是香港流行男歌手陈奕迅的一首国语歌曲，由易家扬作词，林俊杰作曲，C.Y.Kong编曲，陈奕迅演唱。收录于专辑《米•闪》中，发行于2014年05月15日。

爱情神出鬼没，长久思念一个人，就像抬头即使看到乌云密布，内心寂寞笼罩，心里也充满那份秘密独占的快乐。Eason富有感情的声线加上JJ出色的旋律，让《阴天快乐》成为动人的抒情小品。
