

《 **陪你长大** 》是2018年 **大攀** 写给四岁儿子生日的一首歌，曲子虽然不算流行，但小编依然要推荐给大家。

这是一首没有悲伤的曲调和歌词，但细细品味后，却能让人双眼充满泪水的歌曲。曲子从父母的角度，表达了对孩子的无限关爱的情感。每个父母都把自己的孩子当作上天给予的最好的礼物，一生都在细心的呵护着他们长大。  

“希望你快一点点长大...希望你不要太快长大，慢一点点长大...”

看似矛盾，实则不然，这矛盾的背后，是父母大如山、深似海的爱。

有的人说，“喜欢这首歌的，一定都是宝爸宝妈，只有为人父母才知歌词中的含义”，其实不然，单身的EOP小编在听过几遍歌曲后，也坚定的推荐给了大家。

不管你现在有没有孩子，想想你小的时候，父母一字字的教你说话，一次次的教你穿衣，一遍遍的扶起跌倒的你...

等他们老的时候，想不起事情、接不上话、动作缓慢的时候，请不要责备、催促他们，因为你在慢慢长大，而他们在慢慢变老。你只需要默默的陪着他们，就像他们当年陪你长大一样，陪着他们慢慢变老。
