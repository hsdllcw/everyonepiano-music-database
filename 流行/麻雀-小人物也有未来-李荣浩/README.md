_麻雀_
是由李荣浩一人完成的音乐作品于2019年12月5日正式发行。李荣浩的歌曲大部分是用最朴素的语言为最普通的人写歌，所以歌曲很容易打动人心，让人们感同身受。《麻雀》这首歌的发行可谓困难重重。这张专辑本在2019年5月份已经差不多接近尾声，可是发行时间从九月等到12月都没有定论，所以在11月28日李荣浩发文称：如果歌曲无法在12月5日发出，就自己找个电动车维修论坛去发表新歌。可见，音乐人为了将好作品发表出来也需要付出一定的努力。

再渺小的存在也有他的未来，这句歌词仿佛是李荣浩对自己经历的真实写照。熟悉李荣浩的朋友应该知道，初期的李荣浩是十分艰难的，所以他用麻雀去指代自己，好像只要一个轻微的动荡都会让他伤痕累累。后来，它如同麻雀，无论天气如何都在向自己的目标飞翔，才有了今天我们所熟知的李荣浩。
