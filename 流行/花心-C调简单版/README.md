

《 **花心** 》是周华健演唱的一首歌曲，由厉曼婷填词，喜纳昌吉作曲，收录于周华健1993年发行的同名专辑《花心》中。这首歌曲是周华健的代表作品之一。

作为华语乐坛的经典歌曲，周华健凭借亲切朴实的嗓音成功演绎了这首歌曲。歌词中的那句“花的心藏在蕊中”预示了花期易逝，应该珍惜每一寸光阴。这首歌表达了一种态度，萌动的青春对于人们来说都是一种难忘的回忆。
