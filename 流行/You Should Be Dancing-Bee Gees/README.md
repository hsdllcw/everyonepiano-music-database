

**You Should Be Dancing** 是澳洲三人兄弟乐队组合Bee Gees的单曲。组合成员包括大哥巴里•吉布（Barry
Gibb）、及双胞胎罗宾•吉布（Robin Gibb）和莫里斯•吉布（Maurice
Gibb），他们跨越多个音乐流派，把摇滚乐、迪斯科、节奏布鲁斯融为一体，风靡了澳洲、英国，甚至世界各地的民众。他们对现代音乐与大众文化产生极其深远的影响。
