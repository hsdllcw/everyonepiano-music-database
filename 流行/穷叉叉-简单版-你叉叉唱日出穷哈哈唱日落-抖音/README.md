

“你叉叉唱日出，穷哈哈唱日落，哈利波特骑着扫帚飞”最近一段时间，相信不少人都在短视频平台上刷到过《 **穷叉叉**
》这首“神曲”，这首歌其实并不是一首真正意义上的歌，而是有网友将几个明星的“高能片段”剪辑到了一起，结果形成了爆笑的效果。

第一个“你叉叉”来自关晓彤主演的电影《浪漫天降》当中的《空姐之歌》；第二个“穷哈”则是已经“风靡”了两年，让古力娜扎被嘲讽了两年之久的梗，在2019年MAMA颁奖典礼上，因为不是事先安排的，所以古力娜扎暴露了自己的塑料英语，结果被笑到了现在，还做成了梗。

第三个“唱日出唱日落唱星星闪烁”，则是来自2009级北京舞蹈学院《爱的季节》刘昊然的演绎，因为这首歌，让不少人表示“唱得挺好的，下次不要唱了，好好拍戏”。第四个“骚瑞”的梗则流传更为广泛，因为2019年COSMO时尚美丽盛典主持人朱丹屡次口误，叫错明星的名字，各种“骚瑞”道歉，让她饱受争议，质疑她主持功底不行。

第五个“哈利波特骑着扫帚飞”则是来自陈学冬在《跨界歌王》中演唱的一首《童话镇PLUS》，硬核说唱，说唱就唱！最后一个“什么是快乐星球”，在《穷叉叉》之前就已经风靡过一阵，这次在《穷叉叉》中加入马嘉祺的这段演唱也完全没有违和感，只能说网友的脑洞很大。
