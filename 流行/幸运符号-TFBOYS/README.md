_幸运符号_
是由TFBOYS演唱的歌曲，由朱鸽作词谱曲、颜小健编曲，该首歌曲于2014年9月12日发行，收录于专辑《青春修炼手册》。11月3日，TFBOYS作为形象大使出席“星钻圆梦计划”在北京举行启动发布会，现场表演歌舞《幸运符号》。

在创作此曲之初就秉着“赠人玫瑰，手有余香”的理念，把正能量化作幸运的符号，传递正能量。而在实际生活中，TFBOYS作为“星钻芒果基金”的“圆梦大使”以自己微薄的力量帮助贫困儿童实现梦想，传递爱和希望。

歌曲MV于2014年9月24日首发，导演为叶少琥。MV中，三位成员这次进入积木世界寻找幸运符号。和TFBOYS以往MV不同的是，本次全部场景由积木搭设，从汽车，家具到电话，茶杯，相机等，充满着甜蜜的童话色彩。本次MV讲述的也是积木世界不快乐的积木人是如何在TFBOYS的感染下，拿到幸运符号而重新开始微笑的故事。而MV中出现的积木版四叶草似乎也在向所有粉丝说明，四叶草就是TFBOYS的幸运符号
