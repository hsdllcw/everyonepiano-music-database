_Awake_
是韩国男子演唱组合防弹少年团的成员金硕珍演唱的歌曲。作为防弹少年团的队内大哥，金硕珍在团里担任声乐的角色，并不断在歌声上下工夫练习；对于舞蹈，因为总觉得自己跳得不好，所以就不断地练习自我加强，在防弹的各种难度高的舞蹈表演里面几乎没有出错过。金硕珍外表有着花美男的长相，私下却是喜欢开玩笑、很会照顾人的讨喜性格，和团员弟弟们相处就像朋友一般，丝毫没有哥哥的架子。
