

**素颜**
是由许嵩作词并作曲，许嵩与何曼婷演唱的一首歌曲。上口的旋律，所有简单的词汇，一点一点去拼凑，最后聚在一起。一首歌，把你带回去，那些永远也回不去的回去。

《 _素颜_
》并不是真的教唆人人都要素装，有些人，即使扔掉所有的化妆品，也回不到素颜。它只是在用易懂的文字，去表达一种思想。“轰轰烈烈那几年，我怀念，别怀念，怀念也回不到从前。”那些感情的纯洁，不仅仅是容颜，包括这个时代，所有的所有，都可以，需要素颜。这就是许嵩的《素颜》，他能在他的风格里做出新的东西，给《素颜》以俊俏的小模样，亮眼的小性格。
