

抖音“像风吹过八千里，流云和月都曾爱过你”出自歌曲《 **风吹过八千里** 》，由苏星婕演唱，发行于2021年9月17日。

两个人的游戏要两个人才能完成，当一个人说出，爱情仅此而已时，会发现这一切都是自作多情。决绝是对过往最好的应答，纠结过去种种，只会让本该带有些许美好回忆的蒲公英越飘越远，或许保持距离，才是产生美的契机。  
  
感情世界里的人，就像顽皮的小孩，好奇心驱使着他执意要触碰带刺的花，但剎时的欢愉抵消不掉刺带来的伤痛，哭泣就成了最好的证明。  
  
不过，请相信，你总会在经历过这些后，于纷纷扰扰之中，与世界上的另一份美好相遇。
