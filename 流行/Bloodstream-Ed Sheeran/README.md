

**Bloodstream双手简谱** 和五线谱完全对应。

_Bloodstream_ 是英国著名歌手Ed Sheeran收录于专辑X 的一首歌曲。Ed
Sheeran，1991年2月17日出生于英国英格兰西约克郡，是大西洋唱片（Atlantic Records）旗下的一位歌手同时也是一位创作人。

其专辑《X》在英国销售达170万张。2015年在全英音乐奖颁奖盛典荣获“年度最佳男歌手”和“年度最佳专辑”两项大奖。

歌词下方是 _Bloodstream钢琴谱_ ，希望大家喜欢。
