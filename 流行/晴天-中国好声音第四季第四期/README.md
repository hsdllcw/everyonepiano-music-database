_晴天_
是关诗敏在《中国好声音》第4季翻唱的一首歌曲，原唱为周杰伦。这是首浪漫的少年情歌，有风花雪月，更多的是怅然若失，很散文诗式的情怀，用周杰伦的方式演绎，就更带点伤感。这首歌将回忆的思绪充分放大，校园时代的青涩，甜美纯真的爱情，桀骜不驯的叛逆，在这首歌里全都有体现。

歌词下方是 _晴天钢琴谱_ ，大家可以免费下载学习。
