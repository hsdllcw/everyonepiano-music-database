》，由深海鱼子酱原唱，在2021年01月06日上线。

鱼子酱打破“甜美嗓音就该唱小甜歌”的桎梏，用歌声诉说无尽思念。岁月如梭，春去秋来，经历分分合合过后的我依旧相信落日会回归山川，星河会沉睡于夜幕，真心也终会得到回应。
