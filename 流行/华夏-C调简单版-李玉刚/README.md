

李玉刚2022年开年新作《 **华夏** 》于2022年1月15日上线。李玉刚以悠扬国风，奏响又一曲真挚的爱国之声。

在歌曲中，李玉刚把浓厚的家国情怀和悠久的文化底蕴注入到了国风音乐之中，因小见大，以真动人，并将箜篌、古筝、箫、笛子等古老的传统国乐贯穿其中，营造出温情古典的意境与气质；李玉刚沉稳大气的演绎以及童声的加入使其极具感染力，炽热的爱国情感浓烈而激昂，如百川奔东海，如千峰朝泰山，直达心扉。歌曲实现了其音乐作品的艺术价值，与新时代的爱国精神相契合，相信《
_华夏_ 》也将成为又一个引发跨代际情感共鸣的新篇章。
