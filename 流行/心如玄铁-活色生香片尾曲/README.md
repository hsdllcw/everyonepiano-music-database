

**心如玄铁** 是由李易峰献唱的民国传奇电视剧《活色生香》片尾曲。该曲由白金明作词，曾缔、谭旋作曲 。

  
李易峰深情而富有张力的演唱充斥着的淡淡忧伤， _心如玄铁_
歌词中“苦难沿途结界，心如玄铁，绝不会向苍天哽咽，把泪水铸成一颗勇敢的心”将李易峰在剧中饰演的宁致远的命运性格一语道尽，令人为之动容。

  
歌词下方是 _心如玄铁钢琴谱_ ，希望大家喜欢。
