_You Are the Only One_ 是俄罗斯偶像歌手Sergey Lazarev的一首歌。

谢尔盖·拉扎列夫(Sergey
Lazarev)是俄罗斯当红偶像实力歌手，他集作曲和演唱于一身，是前《SMASH》组合成员之一，也是俄罗斯流行乐坛最炙手可热的男歌手之一
。2006年俄罗斯MTV年度颁奖典礼上勇夺最佳男歌手。代表作《Shattered Dreams》、《Don't be fake 》等。
