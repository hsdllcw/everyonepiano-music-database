_  
如果的事_
是王蓝茵作词作曲，范玮琪、张韶涵合唱的歌曲，这首歌的歌词讲述的是为爱奋不顾身的勇敢，得到了很多女同性恋者的认同，这是一份描述远距离的爱情，一段还没开始的恋情，女孩对被动的男孩说，如果你下定决心跟其在一起，她也愿意为了爱而奋不顾身。  
  
2006年这首歌曲获得MusicRadio中国TOP排行榜港台年度金曲奖和马来西亚最佳原创金曲奖和雪碧榜最优秀合唱歌曲奖。  
  
因歌曲《如果的事》被染上女同志色彩，因此，决定抽起这首歌，中国大陆的歌迷将无缘听见她的创作只能交给台湾音乐人来唱了，这首歌曲作于2003，当时王兰茵就是在听着范玮琪《真善美》的专辑，觉得如果能写歌给范玮琪唱也不错，毕竟她的歌声一直陪着她。当蓝茵把《如果的事》通过经纪人，交到范玮琪手中时，据说当时的范玮琪并没有对蓝茵的创作感兴趣。
两年之后，蓝茵所属版权公司的经纪人建议，让蓝茵和另一个女生合唱《如果的事》，再把Demo交给范玮琪时，范玮琪意外地接受了这个女生对唱的版本。  
  
歌词下方是 _如果的事钢琴谱_ ，希望大家喜欢。
