

**无所谓** 是杨坤发行的第一张个人同名专辑中的代表歌曲。凭借这张专辑杨坤一炮走红，也是杨坤人生命运转变的开始。

  
《 _无所谓_
》可以说是杨坤专辑里最用心最深情的作品了，难怪它一上榜就是黑马一匹，冲至季军位置!这样的境况在成名已久的歌坛老将里都是少见，更何况是杨坤一个名不经传的新人!

  
歌词下方是 _无所谓钢琴谱_ ，希望大家喜欢。
