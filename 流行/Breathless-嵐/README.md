

**Breathless** 是日本杰尼斯事务所旗下的男性偶像团体嵐（Arashi的歌曲。嵐，于1999年11月3日发行首张单曲正式出道，所属唱片公司为J
Storm。自2010年起，岚已两次获得日本金唱片大奖，并连续4年当选为日本最受欢迎艺人，被誉为日本的新一代“国民偶像”。
