

**月亮河** 是一首由亨利•曼奇尼创作的歌曲 ，该曲第一次亮相是在电影《蒂凡尼的早餐》由奥黛丽•赫本演唱。并一举拿下当年奥斯卡最佳电影音乐奖
。之后被众多明星翻唱。其中 安迪•威廉姆斯 的版本最广为流传。

  
由中国当代童话作家王雨然先生翻译的中文版《 _月亮河_ 》，在国内广为流传。
