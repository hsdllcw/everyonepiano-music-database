

《 **天天看到你** 》是由林秋离作词，许环良作曲，Billy
Koh编曲、华语著名歌手、演员阿杜演唱的流行歌曲。并且收录于阿杜2002年4月12日发行的专辑《天黑》中。

听阿杜的歌，会有一种莫名的伤感，也许是怀念曾经的懵懂，也许是对现状的不满，又或许是对自己的失望。
