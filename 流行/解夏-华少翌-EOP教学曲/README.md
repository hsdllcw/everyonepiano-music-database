

**解夏双手简谱** 和五线谱完全对应，属于EOP教学曲。

《 _解夏_
》是华少翌于2005年2月发行的首张个人同名专辑《华少翌》中的主打歌曲之一。歌曲曲风飘逸，跟解夏的意思一样，意味著逝者已矣来日可追，一切都将宛如新生，重新开始。被用作富润爽口片广告插曲。

华少翌，出生于文艺世家，自小便进入戏校，并且专攻文武小生。之后在北京和香港进修影视表演，拍过电视剧，当过PUB驻唱歌手和广告模特。后来进入唱片公司，担任唱片策划和监制。

《解夏》是一首伤感的歌曲。歌词下方是 _解夏钢琴谱_ ，感兴趣的朋友可以使用。
