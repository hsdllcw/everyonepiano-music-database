

**九百九十九朵玫瑰**
是台湾歌手邰正宵的代表作，由林利南作词，邰正宵作曲，涂惠源编曲，收录于邰正宵1993年11月发行的专辑《找一个字代替》中。1994年，该歌曲荣获年度金心最佳国语歌曲奖金奖。2007年，荣获第四届劲歌王年度总选最爱华语经典金曲。  
  
此曲唱了18年，只要有邰正宵的舞台，就永远有这些经典老歌。邰正宵的歌声温柔而阳光，给浮华生活带来些许清静之感，让他堪称一个时代的“情歌代表”，足见这些情歌影响力之广泛。
