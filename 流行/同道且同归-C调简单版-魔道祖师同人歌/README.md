_同道且同归_
是《魔道祖师》同人歌。魔道中都是可怜的，被灭道观的宋岚，瞎双眼的晓星尘，断小指的薛洋，为道长死去的阿箐，等了十三年的蓝忘机，剥金丹的魏无羡，孤身一人的江澄，机关算尽金光瑶，被分尸的聂明玦，挫骨扬灰的温情，成死尸的温宁，口硬心软虞夫人，杀了金光瑶的蓝曦臣，都让人心疼。魔道成就了忘羡，可也只成就了忘羡。
