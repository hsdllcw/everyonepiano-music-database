

周杰伦是中国流行音乐的开创者和践行者，他的歌曲往往朗朗上口，传唱度也非常高。而歌曲《 **稻香**
》就是其中的一首，周杰伦用简单的文字，创造出极富想象的禅意境界，非常好听！

这首歌告诉大家一个很简单也很容易被遗忘的道理，那就是知足。周式嘻哈民谣传递的是一种暖暖的，清新的风味。在这温馨和甜蜜中，周杰伦鼓励大家勇敢地向前走，珍惜这一切哪怕就算没有拥有。

《 _稻香_
》的MV结合了田园风光与嘻哈元素，周杰伦穿着民俗风的服装在牛车上、乡间小路与稻田上对嘴演出。MV之所以特地到中坜新屋乡取景，除了当地有绿油油的稻田，独立的四合院也是吸引导演目光的重要原因。MV拍摄当天动用了30多位演员扮演老、中、青三代在四合院里演出其乐融融的情景，被人群围绕的周杰伦与杨瑞代打着木箱鼓、弹着吉他，创造出无忧无虑的气氛，也营造出家的温暖。
