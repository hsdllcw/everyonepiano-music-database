Ed Sheeran，1991年2月17日出生于英国英格兰西约克郡，是大西洋唱片（Atlantic
Records）旗下的一位歌手同时也是一位创作人。被称之为宇宙蓝颜好闺蜜。作为一个优秀的唱作人，他不只为自己创作，还为许多歌手写过歌。
