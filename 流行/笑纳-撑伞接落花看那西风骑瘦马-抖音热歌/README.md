谁能为我一眼望穿流霞  
公子是你吗

最近，抖音上好多人用这首歌做背景音乐。据悉，此曲叫做《 **笑纳** 》,
由花僮演唱，发行于2020年8月26日。词意古风悠然，旋律温婉清新朗朗上口，超级赞！

花僮的声音极其清澈，一字一句听到的是一种活泼俏皮，而且拥有生命且有活力的感觉，词意古风悠然，曲境温婉清新，旋律优美动听，国粤流转自然，用心了！此曲只应天上有，闻得此曲似神仙！

歌词下方是 _笑纳钢琴谱_ ，大家可以免费下载学习。
