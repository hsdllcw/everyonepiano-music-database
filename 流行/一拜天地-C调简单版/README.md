

**一拜天地** 是由火火作词，黄师傅HBY作曲，不是Av的Ay君/畅畅酱共同演唱的歌曲。

切实的幸福，从来不是生生世世，而是一生一世。他们原不奢望生生世世的幸福，只想相偕今生。可是，他们并肩走过荒芜、涉过炮火，终究躲不过那个时代，和险恶的人心。他们不求来生，却最终只能以待来生。他们在“一拜天地”的呓语中，俯首扣入尘埃，走完了这一生。
