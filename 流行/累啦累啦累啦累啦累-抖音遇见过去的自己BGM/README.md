这是一首俄语歌曲，名字叫《 **это ли счастье?** 》， _（小编大声的朗读了一遍，然而并没有人能听懂）_
，中文名“这就是幸福吗?”。来自阿塞拜疆&俄罗斯的孪生兄弟组合 **Rauf** & **Faik** 于2019年推出的热门单曲。это ли
счастье? 这就是幸福吗？歌词中的自问自答透着爱而不能得的些许无奈，搭配着抒情悠缓的曲调，悲伤无尽蔓延开来~。歌曲听起来像“
**累了累了累了累了累...** ”也已预示了曲子的悲伤旋律。

抖音上的视频，两个不同时期的自己相遇，巨大的现实反差，令人陷入沉思。如果上天真的给你一个机会，可以对十年前的自己说一句话，你会说什么呢？

**“快去买房！”** 十年前的自己有钱吗？

**“快去买比特币！”** 十年前的自己相信吗？

**“不要放手她！”** 十年前的自己能做到吗？

**“一定要参加EOP的造神计划！”** 十年前EOP有造神计划吗？
