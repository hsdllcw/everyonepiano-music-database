

周杰伦的音乐无疑是前卫的，即便这么多年过去了再听还是那么好听，比如这首《 **米兰的小铁匠** 》，无论是歌曲结构、旋律还是和声都非常出彩。

而此曲也被称作“另类的励志歌曲”！

木炭一箩筐，木炭一直放，木炭剩一半，火炉烫，小铁匠存钱买希望，在流汗  
米兰的天空刚破晓，那马蹄铁还要敲多少，吉他才能买得到

简简单单的歌词，却给我们描述了小铁匠想要买一把吉他，但是钱不够，于是拼命打铁来赚钱的场景。此曲的励志就在于没有直接向我们灌输大道理，只让大家受到故事当中角色的感染，激发出追求自己愿望的热情。
