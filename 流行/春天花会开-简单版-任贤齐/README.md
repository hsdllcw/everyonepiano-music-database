

《 **春天花会开** 》出自任贤齐2000年1月27日发行的专辑《为爱走天涯》。欢快的曲风，朗朗上口的旋律，给人一种惬意盎然的感觉，是一首极为经典的情歌。

说实话，任贤齐是我小时候的偶像，那时候特别喜欢听任贤齐的歌曲，比如《[ **对面的女孩看过来**](Music-2451-对面的女孩看过来-任贤齐.html
"对面的女孩看过来")》，而我的第一张CD就是任贤齐的。再次听到任贤齐的歌曲真的是满满的回忆，即使这么多年过去了，他的歌声依然充满阳光且温暖。

此曲除了旋律轻快外，歌词中也充满希望，内容虽然不是歌颂春天，但也是借助春天作为背景，抒发了渴望爱的情怀，这与春天给人的舒服感受有一种不谋而合的效果。
