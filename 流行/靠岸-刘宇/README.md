

过尽千帆，等你靠岸。国风美少年归来，新起点，新征途。潜心音乐，打磨作品。国潮少年刘宇诚意奉上首支个人单曲《 **靠岸** 》。

歌曲一经上线，便收获众多好评，歌曲以鲜明的国风因素，极具画面感的编曲与刘宇治愈系的声线巧妙结合，悦耳流畅的旋律，源远缠绵的曲调，整首歌古韵悠然又充满浩然之气，给人一种美的享受。

正如《 _靠岸_
》歌词所描绘的那样“过尽千帆，未来为我们而绽”，这位国风美少年仿佛此前积蓄已久的能量开始爆发。他并没有止步前进，不仅在音乐领域发光，更是自我突破，从音乐领域到表演领域，这样一位认真少年，相信他可以完美塑造出角色形象，让我们一起期待他出色的演绎吧。
