

**月半小夜曲** 是香港男歌手李克勤的经典歌曲，改编自日本女歌手河合奈保子作曲并主唱的Halfmoon
Serenade(ハーフムーン•セレナーデ)。填词为向雪怀，编曲谢德豪。此歌曲于1987年推出，并收录于专辑《命运符号》中，是李克勤的首本名曲之一。  
  
此曲有多个翻唱版本。《 _月半小夜曲_ 》的日文原版亦是相同意思，歌曲及歌词均表现出幽怨和伤感，描绘出故事的主人翁更伴侣分手的伤痛经历。
