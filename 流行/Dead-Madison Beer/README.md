Bieber在YouTube上发现，随后被贾斯汀和其经纪人斯科特培养进入歌坛。2013年2月，Cody
Simpson发行与Madison合作的首单《Valentine》。2013年9月12日，Madison发行首支个人单曲《Melodies》。
