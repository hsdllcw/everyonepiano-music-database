

出自梁静茹国语专辑《静茹&情歌—别再为他流泪》， **情歌**
，就该是一首纯净动人的情歌。在沈淀的弦音之中，听静茹娓娓唱着时光的流转、唱着爱情的变换，唱着经过岁月之后所留下来的记忆与情感。这是一首能听见静茹嗓音起伏迭宕的爱情之歌。
