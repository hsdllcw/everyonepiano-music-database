_只对你有感觉_
是飞轮海和she中的Hebe（田馥甄）合唱的歌曲，由张家玮作词、林俊杰作曲，是偶像剧《东方茱丽叶》的插曲，2010年林俊杰重新编曲演唱该歌曲，收录在其个人概念自选集《她说》中。  
  
这首歌是飞轮海首张专辑内的歌曲，是一首甜美的Pop
Dance舞曲。这首歌曲由林俊杰谱曲，抒发对于青春的情感，年轻的我们应该充满活力，且动静皆宜。此首歌曲使用Hip-Pop舞曲，让听众有气氛轻松和跳跃的感觉。
