回归最纯稚的音乐，用倒叙形式将过往情感谱写成歌，以独特又赋有磁性的音色，唤醒记忆深处的爱恋。《 _迟来的情话_
》在编曲上加入了Trap、和声等元素，搭配歌手精准的音域把控，使整首歌节奏更加轻快明了，却又带着些许伤感，仿佛那青春年少错过的彼此就在眼前，只是被一阵风吹过，便消失在指缝里。
