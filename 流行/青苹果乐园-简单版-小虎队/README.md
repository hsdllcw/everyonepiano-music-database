

“周末午夜别徘徊，快到苹果乐园来”歌曲《 **青苹果乐园** 》由中国台湾演唱团体小虎队演唱，到目前为止依然是值得许多歌迷回忆的动人旋律。

歌曲发行后，小虎队立刻在亚洲乐坛迅速走红，奠定了他们偶像鼻祖的地位。男生们都喜欢模仿他们的舞蹈和造型，女生们都喜欢收集他们的杂志和海报。而这首歌曲也理所当然成为了他们最招牌的歌曲。

作为一首恣意青春、挥洒汗水的歌曲，《 _青苹果乐园_
