_红玫瑰_ 此曲是2015年12月4日，江苏卫视明星音乐真人秀《唱游天下》伦敦站终极比赛，张碧晨致敬陈奕迅，翻唱其歌曲。

红玫瑰表达的就是：怀中拥着一个女人，心中期待另一个。对怀中女人有所愧疚，对另个女人无限想念。尝试用替代的方式转移感情，不料却是徒劳。原来得不到的才是心中最渴望的。
