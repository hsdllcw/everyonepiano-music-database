

《 **约定** 》收录于台湾歌手周蕙1999年出道的首张个人专辑《周蕙精选》，当年选歌的时候，制作老师在KTV听到了王菲的粤语版《 **约定**
》，就推荐给了公司，公司非常大胆的决定让初出茅庐的周蕙来翻唱这首歌，于是邀请姚若龙陈小霞重新填词编曲，果不其然，这张专辑凭借《 _约定_
》的国语版，迅速红遍大街小巷，创下100多万张的惊人销售量。
