

“我还是不是让你捧在手心的小朋友，我还能不能撒娇耍赖提无理的要求”出自歌曲《 **小朋友** 》，由任然演唱，发行于2021年7月3日。

或许曾经有一个人对你的爱，让你像个孩子一样沉浸在安全感中，随着时间的推移，被推开之后，残忍的留下一个人独自面对复杂的世界，而这就是爱情成长的过程。任然的这首新歌《
_小朋友_ 》描述的这段情感当中正是这样一段情感阶段。爱是平等的，是需要双方经营，互相尊重的。只有双向的付出才能有美好的结局。
