

**天上人间会相逢**
是根据琼瑶小说《天上人间》改编的一部古装言情剧《还珠格格Ⅲ》的插曲，由中国大陆女歌手刘盼演唱。该剧是《还珠格格》系列的完结之作，讲述了小燕子与永琪、紫薇与尔康完婚之后的故事。全剧于2002年4月初在北京开机、同年9月初杀青，2003年7月在台湾中视首播。

  
歌词下方是 _天上人间会相逢钢琴谱_ ，希望大家喜欢。
