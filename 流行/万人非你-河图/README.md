_万人非你_
由河图演唱，收录于河图的第四张专辑《NL不分》中。歌曲写了秋景残阳，也写了冬雪飘摇，描绘出远游他乡而风雅孤单的离人形象。离人尚思远，不管是“烟火辉煌，万人空巷”的繁华，还是“除却风声，万物喑哑”的孤单，都难忘了远在天角的她。风霜之后，难免又忆起她的温柔。
