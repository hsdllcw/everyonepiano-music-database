

**Across the Universe** 是摇滚乐队The
Beatles所演唱的一首歌曲。乐队成立于1960年，其音乐风格源自20世纪50年代的摇滚乐，之后开拓了迷幻摇滚、流行摇滚等曲风。

同时，此曲也是美国革命电影工作室 Revolution Studios制作的电影 _Across the Universe_
中的插曲。影片是一部非常有创新意义的音乐电影，不仅重述了一遍发生在60年代的所有大纪事，同时还为以披头士音乐为象征的时代加进了更浪漫、更理想化的艺术情结。
此片挑出了大多数音乐电影先有故事以及剧本，然后再挑选合适的音乐进行关键点缀的限制，通过音乐来创造故事。
