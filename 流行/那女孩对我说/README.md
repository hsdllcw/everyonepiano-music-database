

**那女孩对我说** 是由易家扬作词，李偲菘作曲，Adam
Lee编曲，新加坡歌手黄义达演唱的一首歌曲。收录在黄义达2005年6月3日发行的第二张创作专辑《专属密码》中。

这首歌曲是他为自己暗恋的一位女孩而创作的，有一次黄义达回新加坡过春节时认识了一位女孩，女孩会发亮的眼睛和独特的高雅气质深深吸引了义达，但女孩对他却没有感觉。虽然难免有一点落寞，不过黄义达也感谢那女孩，如果没有她就没有这首歌了。黄义达将音乐想法全面延伸，不论是词、曲内涵或编曲、混音等表现形式，都将他的理想实际执行出来。

这是一首偏重私人情绪的情歌，是黄义达一段感情的真实写照，体现了黄义达柔情的一面。这首歌曲悲伤中有大气，音乐风格在曲目中自由变换，歌曲的段落打动人心，展现了黄义达的音乐才华。
