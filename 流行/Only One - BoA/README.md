Hop体裁歌曲结合甜美钢琴旋律、雄壮管弦乐形成哀婉梦幻的氛围。《Only One》的舞蹈是为麦当娜、Christina Aguilera、Jennifer
Lopez等明星编舞的NappyTabs的作品。
