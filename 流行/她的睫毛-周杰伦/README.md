

《 **她的睫毛** 》是周杰伦专辑《叶惠美》中的歌曲。《叶惠美》是2003年最被期待的专辑－周杰伦第四张全新国语大碟。

《 _她的睫毛_ 》由杰伦作曲编曲、方文山作词。于2003年7月29日由阿尔法唱片公司发行。
