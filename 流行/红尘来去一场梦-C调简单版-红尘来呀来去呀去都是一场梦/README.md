

《 **红尘来去一场梦** 》是巫启贤作词，作曲并演唱的一首经典代表作。最早收录于1993年发行的同名专辑《红尘来去一场梦》中
。1994年巫启贤推出这首歌的广东话版，这一版由刘卓辉填词，收录于1994年专辑《心酸的情歌》中。

有些歌手很容易被模仿，也许是演绎者抓住了歌手自身的某种特质，但巫启贤曾说，极少有人模仿我。究其原因，他说，是因为我唱歌主要是以情感作为主轴。

或许不少人第一次听到这首歌，都是来自杨宗纬翻唱的《 _红尘来去一场梦_
》。但是对比巫启贤的原唱，好像少了一些厚重的岁月感，还有红尘里的缥缈感。这应该就是他所说的“情感”和复古情怀。

因为红尘中并不只有情爱，更包含了整个人生。

巫启贤和杨宗纬两个人用不同的声音和不同的经历，诠释出了对于《红尘来去一场梦》不同的领悟。
