

近日，抖音上有一首新歌非常的好听，其中有句歌词是“少年呀，太张扬，轻裘白马踏金榜”。据悉，这首歌叫做《 **长街万象**
》，是热单女王程响深情演绎的一首原唱作品，发行于2021年2月23日。

歌曲以“万象”的“长街”为场景，生动讲述了长街的风土人情和万物复苏的景象。老夫子的胡须、顽童们的白马蹄、小女儿的醉红妆、书生笑靥中的黄金榜，千姿百态，红尘跌宕，熙熙攘攘，尽显众生之相。曲调婉转，琴声悠扬，唱腔清雅，锣鼓欢腾，是一首让人听了就心生悦然、喜出望外的欢快歌曲。
