

“曲中人不觉秋色浅，问一句何处寻清欢，若寂寞只当消遣，又怎会卷入回忆不成眠”，又一首古风歌曲火爆抖音。据悉，此曲叫做《 **不觉秋色浅**
》由上弦月作词，彭扬作曲，以冬演唱，发布于2021年1月21日。

歌曲旋律带有种爱情的诗韵，也有种快意江湖的感觉。同时，边听给人一种身临其境的感觉，秋日漫漫烟雨，油纸伞下岁月微凉，一曲愁肠入骨相思君可知，奈何曲中人心不静无风起波澜，灯火阑珊处，应是良辰好景虚设。太美了！
