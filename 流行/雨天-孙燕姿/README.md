

**雨天** ，这首歌收录在孙燕姿精选集《My Story Your
Song经典全纪录》里，是孙燕姿最感人代表作之一，杨宗纬、萧敬腾、A-LIN等人都翻唱过这首歌。

  
这首歌主要表达一种很纯粹的感觉，虽然现实有很多的不允许，但还是打从心底希望人生不要过得太复杂，学会体谅自己、爱自己，珍惜平凡简单的快乐。就好像污浊的空气里骤然下一场倾盆大雨，雨过天晴后泥土的芬芳，树叶上的水珠，阳光的五彩斑斓，只有懂的人才会去欣赏。

  
歌词下方是 _雨天钢琴谱_ ，希望大家喜欢。
