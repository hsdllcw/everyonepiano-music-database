《 _小寒_ 》是音阙诗听以节气打造的音乐专辑《二十四节气》系列歌曲，由音阙诗听与赵方婧演唱。于2020年1月6日正式发行。

小寒，是二十四节气中的第23个节气，冬季的第5个节气，民谚：“小寒时处二三九，天寒地冻冷到抖”，这说明了小寒节气的寒冷程度。但《小寒》这首歌曲旋律轻松，听后并没有冰冷的感觉，反而给人一种愉悦舒心之感。
