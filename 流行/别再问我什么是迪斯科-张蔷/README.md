

**别再问我什么是迪斯科** 是张蔷演唱的歌曲，由彭磊词曲创作。收录于张蔷2013年12月27日发行的专辑《别再问我什么是迪斯科》。

整张专辑中，新裤子乐队的彭磊和庞宽几乎包办了所有的词曲创作和编配，庞宽则负责所有的装帧设计。为了还原八十年代的音色审美，新裤子在录音过程中不仅在硬件上采用了80年代的录音设备，还做了多种复古的声音模拟。张蔷的声音跟30年前出道时也别无二致，完美的诠释了八十年代的时代心声。
