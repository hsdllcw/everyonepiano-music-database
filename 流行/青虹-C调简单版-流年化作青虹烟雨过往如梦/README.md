

**青虹**
是由黑子作词作曲，等什么君演唱的歌曲，发布于2021年1月30日。青虹寓意着彩虹的意思，红橙黄绿青蓝紫每一种颜色也都有不同的意思。彩虹，象征着美好与和平。

等什么君，本名邓寓君，中国内地女歌手，嗓音颇有特色。她在诠释国风歌曲里种种悲欢离合的故事时，带有戏剧化的张力，既有儿女情长的浪漫，也有笑看红尘的霸气。她的天赋，加上持续汲取养分，让她成为了当下最受年轻人喜爱的国风歌手。
