_青春的约定_
是由上海久尚演艺经纪有限公司制作发行，SNH48的S、N队各8名成员共同演唱的一首歌曲，收录在snh48两周年纪念EP《青春的约定》中。参与成员有陈观慧、龚诗淇、鞠婧祎、李宇琪、林思意、陆婷、莫寒、邱欣怡、万丽娜、吴哲晗、许佳琪、易嘉爱、曾艳芬、张语格、赵嘉敏、赵粤。

精心打造的青春励志MV《青春的约定》，由S队和N队各8位成员组成超强阵容共同参演，周杰伦御用导演“珍妮花”执导，首次身着男装以“乐队组合”形式亮相。我们不知道未来会有什么困难，不知道未来会遇到什么人和事，但我们在此约定，不管未来是什么模样，也要像现在一样对生活充满热情和希望，来吧，我们一起击掌，这是我们青春的约定。
