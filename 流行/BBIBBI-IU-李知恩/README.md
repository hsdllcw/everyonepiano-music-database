

**BBIBBI双手简谱** 和五线谱完全对应。

_BBIBBI_ 是韩国女歌手李知恩（IU）演唱的一首歌曲，歌曲已于2018年10月10日公开音源。

据韩联社报道，此曲一经公开就强势登顶韩国Melon、Genie、Bugs
Music、Soribada等7大音源排行榜。另外，这也是IU首次挑战另类节奏布鲁斯（Alternative
R&B）曲风，IU亲自作词，词中蕴含了对在人际关系中无礼越线者的警告信息。
