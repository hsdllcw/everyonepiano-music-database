

《 **宇宙之大** 》是许嵩的第五张专辑《不如吃茶去》中的歌曲。

歌词非常梦幻，营造了一个非常浪漫的气氛。编曲以印度传统乐器西塔尔琴作点缀，营造出了虚空幻境。无论从词从曲都不失为一首好歌。宇宙之大每个生命都在孤寂，爱情永远都是描述不清的，只有亲身经历，才能体会，品味。

作品表现出东方的清音雅乐，西方的律动节奏，交织并进；披着古色古香的外套，内里却是一颗极为现代的精神内核；既书写怡然洒脱的态度，也不掩内心的感伤离愁，生活五味熔于一炉。
