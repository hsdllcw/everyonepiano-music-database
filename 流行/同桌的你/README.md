

歌曲《同桌的你》脍炙人口，由老狼演唱，高晓松作词、作曲。曾风靡大街小巷，成为无数80后少男少女的最爱，是校园民谣的代表作，具有里程碑的意义。乃至今天，仍让人津津乐道。是一首温婉动听、勾人回忆的国语歌曲。

所属专辑：《校园民谣第一辑》  
专辑语种：国语  
发行日期：1994年5月6日  
原唱：老狼  
作词作曲：高晓松
