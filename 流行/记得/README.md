

林俊杰根据自己真实经历写的一首单曲，被张惠妹翻唱而走红，《 **记得**
》的曲子是将JJ原有专辑《乐行者》中的《不懂》进行了再度填词后的作品，该歌曲的曲风与曲调与《不懂》几乎一模一样。JJ也非常有信心，到时候大家到KTV女生就点张惠妹的《记得》，男生一定要点这首他的版本，一定能够因此感动女生，希望大家都能记得最初的那些感动。
