

《 **大地** 》是beyond的专辑《 _大地_ 》的收录曲。

这是乐队成立以来发行的第一张国语大碟，但其影响力并不亚乐队的粤语专辑，因为有太多的好歌收录其中！全是以前唱红的粤语歌曲的国语化，但不是简单的“翻译”而是一种升华，一种“超越”。这里面的每一首歌都是经典之作，而且有几首还反映了BEYOND对社会问题的关心与看法，这是其它歌手不能带给我们的！
