

**最初的梦想** 是范玮琪于2004年5月7日发行的国语唱片，是受到偶像中岛美雪的歌曲《骑在银龙的背上》的启发创作而成，由知名作词者姚若龙、古力奇填词。  
  
这首歌描诉人们在遭遇到困难、挫折的时候，只要坚持自己“ _最初的梦想_ ”，回想当初立志实现梦想的那一份努力，就决不会轻言放弃。  
  
歌词下方是 _最初的梦想钢琴谱_ ，希望大家喜欢。
