

**我离孤单几公里**
由华晨宇演唱。歌曲主要以抒情为主，华晨宇委婉的气声与柔和的唱腔延续了他慢歌的风格，悠扬的歌声贴合完美的配器，营造一种放松自然的感觉，这也是华晨宇最自然的本色。

歌曲中华晨宇使用了略带深厚的音色，但仍然很清亮，旋律有淡淡的忧伤。过满则亏，这首歌曲刚刚好。空灵的声音，几个小调旋律：A段，声音像拂在空中的软芦苇，能看见长出来的杂草，映照在冬天非常蓝的蓝天中。B段，声音像蒲公英那样掉落在心里的小角落，拨动吉他上的弦，拨出你嘴角笑的弧度，手心细密的纹路。歌曲没有夸张的抒情，却能让人蛰伏在沿途的风景。你的想象走到了哪里，你的孤独就跟随到哪里。华晨宇干净纯洁却不腻的声音，传达着稚气的怯懦与沧桑，自然而清新。
