

《 **唱给你一支歌** 》是由石顺义作词，王永梅作曲，中国女高音歌唱家贾堂霞演唱的一首歌曲，于2007年发行。

贾堂霞，在民族歌曲演唱上音色明亮纯正，感情真挚，风格大气，表现力较强。特别是北方风格民歌演唱颇得王音旋、韦有琴等山东籍著名歌唱家的真传。擅长于演唱各类创作歌曲（民族唱法）、山东民歌、豫剧、吕剧等。
