

《让世界充满爱》是一首录制于1986年的中国公益歌曲，由郭峰作曲，陈哲等作词，100多位中国流行音乐歌手演唱。

1986年，100名歌手在首体同声高唱《让世界充满爱》，这在中国流行音乐史上留下了令人难忘的一笔。事件的发起者郭峰兴奋地回忆起当年的许多动人故事。
