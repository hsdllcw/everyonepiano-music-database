

薛之谦的情歌，听着总是让人不由的纠结和悲伤。《 **演员**
》以平白的文字续写深情伤感，歌词以男生的视角对女生倾诉：既然不再爱了，就要坦荡的表达。延续以往悲伤情歌路线，薛之谦再一次用歌曲诠释出爱恨纠葛的爱情寓言。

《 _演员_
》MV由黄中平导演，以薛之谦、裴涩琪为男女主角呈现，“跨国雨恋”故事，再现悲伤爱情前曲。MV讲述了薛之谦扮演的男主角和裴涩琪扮演的女主角曾是一对热恋中的情侣，但是彼此之间的摩擦日益增大，早已经无法回到过去的甜蜜。男主角知道女主角已经厌倦继续，现在也只是女主角在扮演过去的甜蜜。于是为了让女主角幸福，男主角扮演起狠心的角色。最终女主角拿起刀子发泄自己的情绪，男主角看着女主角离去，一滴眼泪顺着脸颊滴落。MV中的画面呈现出“室内下雨”营造歌曲氛围，以“倒带”的形式来叙述两人的关系，掀开另一个迷局序幕。
