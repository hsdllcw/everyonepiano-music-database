_何日君再来_
，此曲首唱为周璇作为1937年电影《三星伴月》的插曲。1939年香港一部抗战电影《孤岛天堂》也将《何日君再来》选为插曲(黎莉莉演唱)。后来为李香兰翻唱，遍红大陆。  
  
《何日君再来》在上世纪80年代初因邓丽君的翻唱进入大陆而再度走红，却被错误认为是“精神污染”、“汉奸歌曲”因此被禁，同样是一桩历史冤案。如今刘雪庵被平反，《何日君再来》已解禁，其中体现出的开放与包容，值得人民欢欣鼓舞，改变使人进步。
