

**闲时灯照月** 是Winky诗与知性的小方块一同合作的单曲。

Winky诗，本名赵景旭。90后原创音乐人，所作中国风元素音乐广受好评，唱作俱佳。名字中的“Winky”来源于歌手最喜欢的小说《哈利波特》中家养小精灵闪闪的名字。至今创作歌曲百余首，撰写音乐类型丰富。爵士，摇滚，民谣皆有良作。为多部著名影视、动漫、游戏献唱主题曲并负责其音乐制作。
