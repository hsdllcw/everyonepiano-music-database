

抖音最近又开始有老歌翻红了，比如这首“一开始我只顾着看你装作不经意”的歌。此曲叫做 **说爱你** ，由刘至佳翻唱，原唱为华语流行女歌手蔡依林。

不管长相还是曲风，刘至佳都给人一种“混血儿”的感觉，她演唱的“爱情甜歌”，少了害羞， 多了一份“敢于表白”的勇气。如果说Jolin版《 _说爱你_
》是一个活泼可爱的少女在吐露心事，刘至佳版则像一个酷帅的御姐在霸气告白。

重新编曲的《说爱你》融入了更多电子元素，搭配刘至佳独特的欧美范儿十足的嗓音和对歌曲情绪的完美驾驭，为经典歌曲《说爱你》又增添了一层全新的听觉感受。

歌词下方是 _说爱你钢琴谱_ ，大家可以免费下载学习。
