

阿桑的《寂寞在唱歌》在旋律、曲风、编曲及阿桑本身的唱腔上，较上一张《受了点伤》专辑再有令人激赏的演出，尤其是在阿桑本身技巧及VOCAL情绪的表达及感情收放上都比上一张作品更趋成熟与内敛，有令人惊艳的表现，曲风上也呈现出更。多样貌的阿桑。
