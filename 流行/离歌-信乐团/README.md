

《 **离歌** 》是信乐团主唱的歌曲名称，信乐团也以该曲成名。同名的还有饶雪漫青春疼痛系列小说《离歌》、诗词《离歌》和游戏中的诗歌《离歌》等。

《 _离歌_
》是一首很伤感的歌。翻唱自台湾女歌手辛晓琪2000年专辑《谈情看爱》中的“心里有树”，翻唱自韩国金建模1999《恋歌》。收录于信乐团第二张专辑《天高地厚》。

歌词下方提供了 _离歌五线谱_ ，欢迎感兴趣的朋友下载使用这首歌的钢琴谱。
