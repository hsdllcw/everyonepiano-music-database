

**其实我不快乐** 是钟嘉欣首张粤语专辑《一人晚餐，二人世界》中的歌曲。本歌曲由张美贤作词、张家诚作曲、JOHNNY
YIM编曲，2008年08月20日发行。

  
歌词下方是 _其实我不快乐钢琴谱_ ，希望大家喜欢。
