林一卡（欸-61）根据巫哲同名小说《撒野》中的歌曲《 _撒野_
》进行编曲，改编成的同名歌曲《撒野》。后被翻唱无数次，其中比较受欢迎的是奇然和沈谧仁的版本，给人的感觉是希望，是顾飞和蒋丞在一起后的甜蜜与希望。即使一脚踏空也能彼此依靠，二淼（顾飞妹妹）的病情会慢慢好起来，会和顾飞蒋丞一起走很远很远，会和他们一直在一起。

歌词下方是 _撒野钢琴谱_ ，大家可以免费下载学习。
