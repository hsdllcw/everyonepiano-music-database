抖音此曲叫做《 **1022-比尔的歌**
》，是由Bomb比尔作词作曲并演唱的一首歌曲，歌曲发行于2020年11月6日。这首一个充满了甜蜜恋爱的味道，让你在这个冬日温暖愉悦。

歌曲延续了比尔特有的风格，歌曲中慵懒的声线搭配上Hip Hop，使整首曲子充满流动性，仿佛冬日温泉般包裹着听众的灵魂，又因Hip
Hop而变得动感起来。就如同歌曲中那个温暖可爱的大男孩，在月色如水的夜晚，期望和心怡的女孩拥有一段浪漫爱情。

《 _1022-比尔的歌_ 》曲谱由 **EOP** 大佬[
**M小冯哒**](https://www.everyonepiano.cn/user-174829.html)给大家编配，曲子有些难度，刚学《[
**魔鬼训练营**](/Sale.html)》的小伙伴请绕行！
