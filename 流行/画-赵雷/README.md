_画_
是民谣歌手赵雷原创歌曲，词曲均由赵雷一人完成，收录于2011发行的个人首张专辑《赵小雷》中。在2013年举办的《中国好歌曲》全国30进20比赛中，赵雷现场演唱了这首歌，被刘欢称赞为“神来之笔”，并收录于刘欢的原创大碟《新九拍》中。

歌曲描述了四合院的生活图景，一把木吉他配上悠扬的笛声，旋律简单却分外动人，而歌词更是令人叫绝，“我没有擦去争吵的橡皮，只有一支画着孤独的笔”！歌词有姑娘、远方之类的词汇，但却并不僵化与那些标准的中国民谣词汇里。更趣味、更生动的歌词，显得特别有画面感，在看似胡思乱想的臆语中，不乏真诚与诗意。除了能够让人共鸣之外，还能够提供一种十足的代入感。
