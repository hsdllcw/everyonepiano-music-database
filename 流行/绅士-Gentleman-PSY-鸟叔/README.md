

“鸟叔”PSY所属的经纪公司YG娱乐2013年4月1日表示，继风靡全球的《江南Style》后，PSY下一首歌的歌名为《 **绅士（Gentleman）**
》。PSY于2013年4月12日零时在全球119个国家同时发布了最 新 单
曲《Gentleman》(《绅士》)，PSY除了与《江南Style》共同作曲者刘根衡一起为这首歌谱曲外，还亲自作词。《绅士》新歌发布以来，两周点击量已经超过2.48亿次。  
  
遵守诺言的鸟叔说到做到，除了在全球范围内同步发行新歌《 _绅士（Gentleman）_
》，他还发布了这首新歌的宣传海报，海报上有13位鸟叔正在抱着胳膊纵情大笑，而标题上则赫然是一句粗口：“我是一个肏他妈的绅士（I'm a mother
fucking gentleman）!”与新歌标题《绅士》对照，海报的反讽意味非常强烈！  
  
歌词的下方还提供了 **Gentleman五线谱** ，需要钢琴谱的朋友可以下载使用。
