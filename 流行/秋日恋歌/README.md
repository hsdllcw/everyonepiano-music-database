

《秋日恋歌》悲伤的木吉他满浸对恋情已逝的惆怅，感情爆发的季节迎来天赐的创作灵感。

  
泛黄的记忆在木吉他的撩拨中显得更加纯净与遥远。秋日做为水木年华最喜欢的季节，给他的创作带来了无以伦比的才思与感悟。卢庚戌曾说过他的每一段恋情都开始在秋天，所以这首歌曲不经意间一气呵成，歌词中更充满着留恋与无奈。吉他是歌曲中最主要的伴奏乐器，在尼龙弦与钢弦之间做了精心的选择。
