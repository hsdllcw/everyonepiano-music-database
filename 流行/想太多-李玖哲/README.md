《 _想太多_
》为李玖哲2007年发行的同名专辑中的主打歌曲，属节奏舒缓的动人情歌。歌曲展现的为情人牺牲的主题，表现出爱情的凄美。李玖哲用深刻浓厚的情感诠释着都市的恋爱故事。旋律沉静的钢琴为全曲营造了平和的情绪。李玖哲对副歌旋律的处理也显得得心应手，浪漫的思绪在略带沙哑的嗓音之中，升华到了极致。
