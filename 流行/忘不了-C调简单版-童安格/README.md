

《 **忘不了** 》是台湾歌手童安格作词、作曲并演唱的一首歌曲，收录于1989年发行的专辑《其实你不懂我的心》中，是童安格的代表作品之一。

童安格的歌，最有怀旧的味道，他能够把一首情歌唱得这么大气而又一点不缺少温柔细腻。他有着儒雅的气质和风度翩翩的演唱风格，而这也正是他的过人之处。《 _忘不了_
》已成为经典，有人说《忘不了》似如青橄榄，慢慢咀嚼，越品越有回味，需要很长时间才能体会到它的好处，刻在心痕，忆在心扉。
