

抖音“多想留在你的身边，让爱渗透了，整个世界”出自歌曲《 **雀跃** 》，小来哥不仅仅邀请顶尖的音乐人创作词曲，更是力邀温暖治愈系女声任然跟自己一起合唱。

任然独具魅力的声音，与音乐的情绪配合得天衣无缝，而小来哥也在任然的帮助下越发精进，将自己对爱情的理解和感悟，在音乐中尽情释放。

长厢厮守是每个深陷爱情之中男女的共同愿望，可世事无常造物弄人，多少痴情男女无法执手相守，只能各安天涯。既然深爱的人不在身边，那么睹物思人简直就是确定中的确定。

一首只为TA写的歌，恐怕最能拨动心弦，将思念的情绪掀起，让爱意穿越时间和空间。
