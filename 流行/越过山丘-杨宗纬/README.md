_越过山丘_ 此曲由杨宗纬演唱，高晓松作词，2017年7月18发行。

此曲歌词意在打破时间的平衡，越过山丘分别“遇见”十九岁和六十岁的自己，得意有时失落有时，彼时受过的磨砺，也会逐渐淡然，或许这就是对人生最好的领悟。每个人在年少的时候，都有独特的经历和回忆，此番《越过山丘》新歌发布，还将开启#老夫撩发少年狂#的话题，所有歌迷都可以在话题下分享属于自己的青春故事，让那些或美好或疯狂的瞬间，成为生命中最美好的回忆。

人生最好的表达，或许只是一首歌，轻轻地唱，淡淡地印在心里，慢慢地遗忘。无论是十九岁的忧愁和迷茫，或是六十岁的等待和微笑，一点一滴都是人生的宝藏。7月18日，高晓松与杨宗纬带你一起《越过山丘》，共同追忆青春年华。
