夏至后，气温逐渐升高，所以给推荐大家一首比冰镇西瓜更清爽的歌《夏至》。

《 _夏至_ 》由音阙诗听成员小魂演唱，收录在专辑《二十四节气》中。

想到夏至，想到的大概有蝉鸣，有冰镇饮料，有西瓜，还有悠闲的假期时光。古语有云“鹿角解，蝉始鸣，半夏生”，描述的是夏至这个节气，不少地方的人还会祭祖。

不过对于少年们来说，他们对夏至的印象，就像这首歌一样，有着很多的消暑小吃，荔枝、凉面、绿豆汤、竹筒饭、槐花糕、白桃乌龙、毛尖、苦瓜炒饭，还有腊肉饭、青枣等盛夏美食。
