

**Married Life**
是2009年皮克斯动画工作室电影《飞屋环游记》（Up）的原声。《飞屋环游记》由《怪兽公司》导演皮特•多克特负责执导，爱德华•阿斯纳、克里斯托弗•普卢默和约翰•拉岑贝格尔配音演出。

  
讲述了一个老人曾经与老伴约定去一座坐落在遥远南美洲的瀑布旅行，却因为生活奔波一直未能成行，直到政府要强拆自己的老屋时才决定带着屋子一起飞向瀑布，路上与结识的小胖子罗素一起冒险的经历。

_
