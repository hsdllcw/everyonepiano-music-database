不同于其他那些商业歌曲，这首真的好走心，能够完完全全打动人。要趁早享受年轻，趁早去爱，趁早去经历一切冒险，然后找一个爱的人一不小心就变老。强大而沉默的鲸鱼注定是孤独的，孤独的生物最怕老去。这是一首男人的歌，唱出了强者的弱。有没有爱看的天，爱踏的地，我爱着的你，陪我做执迷的鲸鱼，等到步履蹒跚的那天互相喊着对方老伴。或许李荣浩的心里也是希望有一个人能陪他到老吧，就像我们每个人所期望的一样，这是一个美好的愿望。
