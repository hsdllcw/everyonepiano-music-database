_蝶恋花_
是邓紫棋主唱的一首粤语歌，词维谷，曲谢宇毅，2012年9月2日发行。此曲为广东电台音乐之声重拳推出的金庸经典武侠小说系列四部曲之《天龙八部》的主题曲。

邓紫棋不仅唱得好，在乐器方面也有很好的天分，演唱会上她弹吉他、打鼓像一个女摇滚战士，她一袭白裙安静地弹钢琴，顿时又变成女神。
