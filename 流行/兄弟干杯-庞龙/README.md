_兄弟干杯_
是庞龙携手好友王岩与中国羽毛球队总教练李永波，在2008年1月21日北京奥运会倒计时200天这个极有意义的日子，推出的最新奥运单曲。以用此曲来支持和宣传2008年北京奥运会。

歌词下方是 _兄弟干杯钢琴谱_ ，希望大家喜欢。
