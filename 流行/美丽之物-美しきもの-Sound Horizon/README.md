

美しきもの又译作《 _美丽之物_ 》是由Revo作词作曲，Sound Horizon (幻想乐团)演唱的。

**美丽之物** 收录于Sound Horizon (幻想乐团)第五张故事音乐CD《Roman》中，是专辑中的第八首歌曲。

_美丽之物_
，这是一首姐姐写给弟弟Laurant的歌。短暂的生命中却一直饱受病痛折磨的弟弟，望着姐姐描绘的风景画赞叹着“啊……真是漂亮呢”直到微笑着病逝。春之风花、夏之流云、秋之夜月、冬之初雪……姐姐用口琴吹着弟弟生前最爱的旋律，在四季的更迭中怀念他的面容。众生降临的意义，就是为了采集这个世界中美丽的事物啊。
