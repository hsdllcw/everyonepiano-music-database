

方文山、林俊杰协力打造“2008赤壁On Line Game”主题曲：《 **醉赤壁**
》。收录于2008年10月18日发行的专辑《JJ陆》。JJ与方文山透过“赤壁online
game”的虚拟世界题材元素，将三国时代的女子“小乔”透过文字及音符，共同创作出“赤壁online game”的主题曲《 _醉赤壁_
》。林俊杰和方文山，online game互相较劲，将喜好化为创作，共谱三国时代的小乔故事，让“赤壁online game”更添阳刚气之外的铁汉柔情。

R&B和古典，这两个原本格格不入、毫无交集的名词和音乐形式，在21世纪居然被完美统一地结合起来，而且，并不像很多流行的元素一样转瞬即逝，它的生命力时至今日还如此旺盛，且大有无休无止、持续蔓延的趋势。

静心，清晰回忆，定心，有了彼此的守护和牵挂，此生无憾。只是一切已逝，遗憾留在时光隧道里，背影恍惚，红颜消散，伸手牵住的只是一把淡淡的哀伤。对你已用情太深！
