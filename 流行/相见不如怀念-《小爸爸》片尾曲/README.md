

《 **相见不如怀念** 》是文章导演，文章、马伊琍等主演的电视剧《小爸爸》的片尾曲。

  

《 _相见不如怀念_
》由著名创作型歌手樊凡演唱。樊凡，作为一个创作型歌手，他不但为国内多部知名电视剧量身打造主题曲，更积极参与各种公益活动奉献爱心；作为一个车手，他使亚太拉力赛上第一次升起了五星红旗。许多人气电视剧演唱主题曲。

  

电视剧《小爸爸》讲述了一个生长在美国的男孩回到国内来找爸爸的故事。2012年12月5日，该电视剧在北京举行发布会。2013年9月2日，该剧在深圳卫视
、东方卫视、浙江卫视、陕西卫视晚间黄金时段首播。本剧由上海尚世影业有限公司、北京君竹影视文化有限公司、北京华美时空文化传播有限公司、完美世界（北京）影视文化有限公司联合出品。
