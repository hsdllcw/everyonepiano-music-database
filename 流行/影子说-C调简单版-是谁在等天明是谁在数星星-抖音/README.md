

抖音“是谁在等天明，是谁在数星星，这首歌谁在听，谁又哭红了曾经”出自歌曲《 **影子说** 》。《影子说》由洛先生演唱，于2021年11月发行。

不得不承认，这是一首唱给痴情人的歌，歌曲旋律平缓而渐进，基调伤情而动人。就像过来人诉说一段爱情往事，平淡而认真。

虽然故事中的人看似对待感情执拗以致于有点消极，但折射到现实中来，也许并不足为奇。

对已经走出来的人来说，听此歌也会有一番感慨，而后一笑而过；对还未走出的人来说，却是一声声慰藉。
