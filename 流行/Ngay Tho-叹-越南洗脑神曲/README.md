抖音魔性神曲《 **Ngay Tho** 》是来自于越南的一首歌曲，由歌手Tăng Duy Tân和Phong
Max演唱。而后，中国内地流行乐女歌手黄龄又与 Tăng Duy Tân合作了中文版，名为《叹》。

《 _Ngay Tho_ 》这首歌中有一个非常受大家争议的声音，有人觉得是一种无奈时候的叹气声，但是也有人觉得是喘气声。而且不同的人听《Ngay
Tho》这首歌，感觉都是不一样的。有网友觉得这首歌给人的感觉是妩媚性感。但也有网友觉得这首歌表达的是孤独，伤感。可能正是因为是一首外文歌，大家听不懂歌词，才会对于这首歌有很多的遐想空间。
