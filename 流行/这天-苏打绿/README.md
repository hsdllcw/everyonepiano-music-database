

**这天双手简谱** 和五线谱完全对应。

_这天_ 是由吴青峰作词作曲，苏打绿演唱的一首歌曲。该歌曲于收录在苏打绿于2007年11月2日发行的《无与伦比的美丽》专辑中。

歌曲试图描写的是，越亲密的情感越容易引发冲突，我们常常伤害别人，也时常被伤害，但有时候我们应该练习在面对那些试图伤害自己的人时，以陪伴的温柔心情相还。
《这天》的旋律带来的情绪是不容中断的，第一句歌词出现时，心的回声就很巨大，以“眼泪带来的歌声”形容青峰在这首歌中的表现一点也不为过，覆盖上稳定的贝斯、弦乐的线条与钢琴的和声时，音乐澎湃了也平静了。

歌词下方是 _这天钢琴谱_ ，大家可以免费下载学习。
