

**艺术家双手简谱** 和五线谱完全对应。

音乐人徐秉龙的单曲《 _艺术家_ 》以音乐表达疼痛与深思，道出少年成长之烦恼。

敏感，细腻体验生活况味，感知世界，感性中寻求真理，这是“艺术家”感知世界的方式。徐秉龙以独特的少年视角解构少年烦恼，剖析内心世界。

一首耐人寻味的作品，歌词细致深刻，曲调清澈悠扬，但经由徐秉龙的吟唱却道出了天真少年的迷茫与无助。歌曲的目的是共情，迷茫的也不止少年徐秉龙，烦恼与哀愁是每个人在成长过程中都会遇到问题，而听者或少都能在歌中找到自己的共鸣。
