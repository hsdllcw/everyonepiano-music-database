

**第一次双手简谱** 和五线谱完全对应，属于EOP教学曲。

_第一次_ 是张天成作词，光良作曲，光良演唱的一首歌曲，收录在专辑《第一次个人创作专辑》中。这是一首流行风格的歌曲
，歌曲展现了光良温柔的嗓音和创作功力。同时，这首歌也是偶像剧“白色恋"曲的片尾曲。

光良，马来西亚籍华裔，台湾流行歌手，出生于马来西亚怡保。1995年签约台湾滚石唱片，曾和品冠组成歌唱组合无印良品。其代表作有《童话》、《第一次》、《约定》。现为星娱公司艺人，在多部影视剧中出演角色。

歌曲《 _第一次_
》是一首甜美幸福的歌。相信听着这首歌，大家也许会想起和自己喜欢的人的许多“第一次”吧。第一次邂逅，第一次牵手，第一次一起冒雨回家......如果大家需要第一次曲谱的话，可以免费下载
**第一次五线谱** 和简谱，希望可以帮的到大家。
