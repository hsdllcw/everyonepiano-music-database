_歌在飞_
是苏勒亚其其格演唱的歌曲。苏勒亚其其格，现任内蒙古东方艺术团独唱演员。2008年1月荣获第三届全国网络音乐节既网络歌手大奖赛冠军。嘹亮高亢的歌喉，极富个性的舞台表现力，让这个爽朗的内蒙古女孩赢得了大多数网友的喜爱。
