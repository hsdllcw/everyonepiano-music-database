

**另一个天堂** 是王力宏、张靓颖合唱的一首歌曲。收录在王力宏2008年12月26日发行的专辑《心•跳》中。

这首歌曲非常浪漫，有意境，表达的是恋人间的深情。在王力宏和张靓颖深情嗓音的诠释之下，那份割舍不掉的爱让听到的人为之感动。在这首二人合唱的歌曲里，张靓颖不再一味炫技巧，王力宏和张靓颖互相迁就但不失特色，就像歌词那般默契。

可以说《 _另一个天堂_
》是专辑中的一个意外之喜，张靓颖在间奏部分的海豚音和失真吉他一唱一和，很是精彩，而王力宏的飙音则展现了另一种张力，仿佛古筝和电吉他的那种混响，有错位感和新鲜感。
