

**Survival**
是英国Muse缪斯乐队的全新单曲。缪斯（Muse）是一支英国摇滚乐乐队，这支乐队于1994年在英国德文郡的Teignmouth镇创立。乐风融合了独立摇滚、前卫摇滚、重金属音乐、古典音乐与电音。在Muse的这三位成员刚刚聚到一起开始正式组乐队的时候，他们都只有13岁。在不同时期，他们共用过Gothic
Plague，Fixed Penalty和Rocket Baby Dolls做过乐队名。
