

**山楂树**
是中国内地男歌手钟立风演唱的一首歌曲，由彼利平科作词，叶.诺代金作曲。钟立风，祖籍浙江，现居北京。内地最具文艺气质民谣歌手。Borges(博尔赫斯)乐队主唱。老狼《弄错的车站》、水木年华《再见了最爱的人》词曲作者。被誉为内地最具文艺气息的民谣歌手。

  
歌词下方是 _山楂树钢琴谱_ ，希望大家喜欢。
