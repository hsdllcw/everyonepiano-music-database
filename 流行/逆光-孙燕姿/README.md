_逆光_
是孙燕姿演唱的一首歌曲，由廖莹如作词、李伟菘作曲。收录于专辑《逆光》中，于2007年7月22日发行。2007年12月，该歌曲获得第六届中国金唱片奖音乐录影带奖。

此曲的创作灵感来源于逆着光前行，歌曲所要传达的是当决定了爱上了一个人的时候，就要有一种永不放弃意念。在录制过程中，李伟菘对作品的要求很严格，孙燕姿投入了极大的精力和时间来录制。

歌曲中板轻快的节奏配合上孙燕姿有力量却又肯定的声线，以钢琴为主底，加上大量合声及弦乐的配合，给听众一种看到了希望的感觉。
