

《 **刀马旦**
》是方文山、周杰伦、李玟共同合作完成的一首歌曲，于2001年11月17日正式发行，“刀马旦身段，演出风靡全场，耍花枪，一个后空翻”，一幅幅画面仿佛就在眼前。

李玟是首位进军美国流行乐坛的华人歌手，也是首位全球发行英文专辑的华人歌手。她在第73届奥斯卡颁奖礼上的演出，让全世界领略到了中国的流行音乐。李玟凭借扣人心弦的歌声、天赋的节奏感和自然的表演才华，在国际舞台上留下了浓墨重彩的一笔。她具备才华和取得成功的信心，真正令她与众不同的是她的音乐融合了东西文化的特色。
