

**离人** 是林志炫翻唱张学友1998年专辑《不后悔》中的歌曲，收录在他的2001年专辑《擦声而过》中。

_  
离人_ 这首歌曲由厉曼婷作词，何家文作曲，林志炫优美的歌声，获得一致好评，并成为其经典曲目之一。

  
2014年，这首歌再度进行改编，采用流行与京剧对话的方式，谱写出了一曲经典而又别样的《离人如梦》。

  
歌词下方是 _离人钢琴谱_ ，希望大家喜欢。
