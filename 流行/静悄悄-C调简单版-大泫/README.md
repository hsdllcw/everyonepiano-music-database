这段抖音上很火的歌词出自歌曲《 **静悄悄** 》，由伍华填词作曲，陈泫孝（大泫）演唱，发行于2018年3月27日。

这是一首甜甜的歌，温柔细腻的嗓音，治愈动听的歌声，唱出了爱情的甜蜜与美好，静悄悄，就是我爱你的声音~将暗恋者的心理展现得淋漓尽致。动人的旋律，唯美的歌词，使此曲一经推出就大受欢迎。
