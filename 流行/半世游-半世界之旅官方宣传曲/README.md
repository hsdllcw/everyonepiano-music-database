_半世游_
是游戏《半世界之旅》的官方宣传曲。由伦桑、奇然、小魂,以及小曲儿共同演唱。古风乙女恋爱手游《半世界之旅》为你呈现最梦幻完美的东方异世界心动物语。在这里你将以唯一女性的身份受到所有男生的极致宠爱，与性格各异，种族横跨人、妖、魔、兽的他们展开一段段刻骨铭心、荡气回肠的动人爱情故事。
