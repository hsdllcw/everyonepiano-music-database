

**不要责备我吧 妈妈** 是一首俄罗斯民歌，关牧村翻唱了此曲，所属专辑《中外歌曲精品 想念你
妈妈》。关牧村，女，辽宁沈阳人，生于河南新乡，满族，中共党员，硕士学历。第七、八届全国青联副主席。历任中国音乐家协会会员、天津市音乐家协会理事，香港佛教文化产业
--佛教音乐委员会委员，全国青联常委，全国青年联合会副主席，天津市青联副主席，第七至九届全国政协委员。

  
歌词下方是 _不要责备我吧 妈妈钢琴谱_ ，希望大家喜欢。
