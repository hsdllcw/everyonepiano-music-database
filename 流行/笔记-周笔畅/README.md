

**笔记**
是周笔畅演唱歌曲。由黄友桢作曲，唐恬恬和黄友桢作词。该歌曲收录于2005年超级女声《终极PK》合集，后被收录于由人民音乐出版社编辑的第一部《流行音乐》教材中。  
  
2006年获第12届全球华语榜中榜“搜索最热门年度歌曲奖”、第13届东方风云榜“十大金曲奖” 、香港新城国语力“热爆K歌奖”。  
  
简单的歌词与旋律，却被周笔畅温暖的声线升华的格外动听。歌曲中诉说的青春迷惘、梦想张扬的故事，充溢的自然温暖、轻快明丽的旋律，使这首歌不仅成为歌颂青春、铭记成长的歌曲，也是一首能唱给妈妈，抒发对母爱赞美的作品。
