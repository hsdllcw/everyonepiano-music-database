

《 **黑色毛衣**
》这首歌曲是由周杰伦作词作曲的。杰伦写了这么多抒情歌曲味道都各不相同，但旋律都如此的棒，由此看出周杰伦在旋律上的思维能力的确是超乎常人。  
  

这首《 _黑色毛衣_ 》，歌词很简短，但是其中很多地方都有诗意之体现（例如“看着那白色的蜻蜓/在空中忘了前进”一句），可以看出周董的作词能力也是非同寻常的。  
  

另外，优美上口的情歌，旋律走向在是周氏情歌中并不常见，在感情融入上很到位，曲中悔恨之意甚重，也是这首低沉乐曲《 _黑色毛衣_ 》与发如雪的不同之处。
