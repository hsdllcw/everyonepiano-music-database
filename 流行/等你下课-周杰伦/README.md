

**等你下课双手简谱** 和五线谱完全对应。

_等你下课_ 是由周杰伦作词、作曲，周杰伦与杨瑞代演唱的一首歌曲。该曲在2018年1月18日零点在网易云音乐、虾米音乐、QQ音乐等平台发布。

周杰伦在七天内完成配唱、音乐制作、歌词版MV，来给歌迷一个大惊喜。2018年周杰伦的生日这天，他选择用该曲跟歌迷一起度过。

该曲的灵感来自周杰伦发想的一个故事，想象一个年轻人暗恋一个女孩，懊悔自己没有好好念书，跟她考上同一个大学；为了等她下课，在女孩住处附近租房、默默守候着她；弹着琴写情书，只想把自己的心意想办法传达给她，却一直找不到时机与方式，只能每天等着，等你下课。

此曲除了有宛如《不能说的秘密》的校园浪漫，更多了些文青情怀、暖心陪伴。在2018年的开始，回归最纯稚的音乐，周杰伦以发生在生活里的小故事，谱写成歌，像暖暖包一样，温暖着每颗等待的心。
