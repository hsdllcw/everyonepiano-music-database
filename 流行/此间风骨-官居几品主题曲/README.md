

**此间风骨** 由古风音乐人落樱风演唱，同时也是橙光游戏《官居几品》的主题曲。

温柔的嗓音，缓缓道来的是云佑的一生。此时的他，没有云家天下中的野心，唯有对君主，对妻子的真心，正凸显了这份深情。或许，他曾在权利的彼端迟疑过，却从未迷失。云家兴复，天下归定。此间少年，风骨依旧。
