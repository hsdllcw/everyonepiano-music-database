_还是觉得你最好_ 收录于张学友1992年发行的《爱火花》专辑，已成为张学友粤语歌曲代表作之一。由刘卓辉填词，米米CLUB作曲。  
  
依然是一首经典的老歌，很优美的旋律，歌词也是不错，个人比较喜欢歌曲的开场段，那个熟悉的旋律，曾经去K歌，我们都抢着每个人要唱一次的歌曲，到现在还是那么的熟悉，每次听到这个歌，就想到曾经一起玩耍的小伙伴们了。  
  
歌词下方是 _还是觉得你最好钢琴谱_ ，希望大家喜欢。
