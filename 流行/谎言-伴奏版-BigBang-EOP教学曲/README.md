来自于BIGBANG韩国迷你一辑《ALWAYS》的一首非常好听的歌曲--《 _谎言_
》。《谎言》是当今韩国最红的组合之一BIGBANG的一首歌曲，该作品是成名之作，以其令人中毒的旋律，在剧情性和观赏性都结合的很好的MV，红遍了全韩国，被奉为“神曲”。无论是年轻人或者是年长者，口中都会哼《谎言》的调子。本歌还出了英文版和钢琴版。此外，韩国最百变女团T-
ara也有一个专辑，叫谎言。

bigbang是当今韩国最红的组合之一，该作品是成名之作，以其令人中毒的旋律，在剧情性和观赏性都结合的很好的MV，红遍了全韩国，被奉为“神曲”。无论是年轻人或者是年长者，口中都会哼《谎言》的调子。本歌还出了英文版和钢琴版。但是当时有一段谎言是抄袭的风波，谎言言是抄袭日本dashi
dance的p.i.a.n.o这首歌，但是听过了就会发现，两首歌唯一相似的地方就是都有那一段钢琴的旋律在后面，当然旋律也是差很多的，更何况《谎言》作曲作词的GD（BIGBANG队长）澄清自己绝对没有抄袭，而采访了daishi
dance后，他个人也否定了抄袭之说。BIGBANG是现在韩国实力偶像才华个性都具备的超人气组合。
