**_白色恋人_ （white lover）这首曲子法文名叫《jours en
france》，选自法国电影《白色的恋人》也叫《法国十三天》。作为韩剧〈蓝色生死恋2〉--《冬季恋歌》里的插曲。  
_  
_ 白色恋人是法国配乐大师Francis Lai在1968年为同名电影作的主题曲。 其中描述他们的爱情、亲情洋溢在这首热情的华尔兹旋律中。
这首歌作为法国国宝级配乐而闻名全球。 其配乐的巅峰已经成为法国香颂。 无数电影电视引用，比如我们熟知的《冬季恋歌》和《艾伦的无冬之夜》。
