

**真的双手简谱** 和五线谱完全对应。

_真的_ 收录在孙燕姿第三张国语专辑《风筝》中，由徐世珍作词，孙燕姿作曲，包小松编曲，包小柏制作。

孙燕姿（Stefanie Sun），新加坡人，华语著名女歌手、亚洲歌坛天后。

歌词下方是 _真的钢琴谱_ ，喜欢的朋友可以免费下载学习。
