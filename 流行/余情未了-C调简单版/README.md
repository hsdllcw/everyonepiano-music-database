

抖音热门歌曲《 **余情未了** 》是一首小清新的歌曲，讲述了多少情痴醉。气质美女魏新雨唱出了恋恋不舍的爱情，发行后便在各大播放器爆火。

余情未了是指两个人因某种原因而表面上分开了，可单方或双方心中却依然念念不忘，始终没有放下，三情中多用于比喻男女之间爱情的情丝难断，这首歌曲恰恰表达了这个意思。情死于猜忌，爱生于无惧。
爱而不得，念而不见，是你曾经沧海，而今天涯，是你。爱情里有两种遗憾，一种是你曾经那么用力地爱过，最后却发现那个人根本不值得，另一种是你没有好好去爱，失去了，才发现那是一个真正值得的人。但对你，不管值不值得，我都甘愿喜欢，只是不甘心最后的结局。
