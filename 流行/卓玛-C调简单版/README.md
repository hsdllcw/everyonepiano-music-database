

《 **卓玛** 》是尼玛泽仁•亚东演唱的一首歌曲，由真知作词，秋加措作曲，收录于2007年专辑《故乡•卓玛》中。

亚东的歌声有的清澄明净，让人心中升起一个蓝色的高天，心灵得到净化，精神得到安宁；那些深沉悠扬的歌，带着你走向美丽的草原雪山和遥远神秘的西藏。亚东的歌挺独特，像盘旋辽阔天地间的巨鸟，沉漫于人生和命运的思考，表达藏传佛教思想，令人浮想联翩，胸襟开阔。
