《 **旅立ちの日に**
》（中文译名《启程的日子》）是最流行的日本中学生毕业歌，在日本广为流传，许多日本歌星都唱过这首歌。作词是当时的校长小嶋登；作曲为时任音乐教谕高桥浩美（原称坂本浩美）。同名歌曲有多首，其他较为著名的是川岛爱（川嶋あい）的版本。该歌曲的国语版为《分别还有重逢时》。

歌曲最初只是打算在“三年生送别会”时，由教职员向毕业生们献唱，作为对他们的一份惊喜，不过从第二年开始，就变成由学生演唱了。另一方面，歌曲发表当年，正值是小嶋校长结束自己41年的教师生涯而退休，这首曲子也成为了小嶋校长最初、也是最后的发表作。

过往，《 _旅立ちの日に_
》只用于影森中学校内，后来市内其他的中、小学校也开始唱着这歌来。后来在东京都中学当教谕的松井孝夫把乐曲改为混声三部合唱，并在音乐之友社旗下的杂志《教育音乐》内刊登后，受到了不少的欣赏。由1998年起，隆续在全日本的学校内兴起演唱本曲。2004年3月于富士电视台的节目“情报现场
EZ!TV”作介绍；2007年由SMAP为东日本电信电话所拍摄的广告亦采用了本曲，现时，中小学毕业礼上演唱《旅立ちの日に》，已超越代了传统的毕业歌曲。
