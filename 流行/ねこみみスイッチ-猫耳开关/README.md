

《 **ねこみみスイッチ** 》，中文译为《猫耳开关》，是daniwellP作曲，初音未来（初音ミク）演唱的一首歌曲。

初音未来（ 初音ミク，Hatsune Miku ），是2007年8月31日由 Crypton Future Media 以雅马哈的 Vocaloid
系列语音合成程序为基础开发的音源库，音源数据资料采样于日本声优藤田咲。
2010年4月30日，发布初音未来6种不同声调的版本“初音未来Append”。2013年8月31日，初音未来英文版本同VOCALOID3一并发行。此外，初音未来还担任日本音乐团体Sound
Horizon的演唱与合唱。

同时，网站还为大家提供了《[ **千本樱-初音未来**](Music-1801.html "千本樱-初音未来")》曲谱下载

歌词下方是 _ねこみみスイッチ钢琴谱_ ，大家可以免费下载学习。
