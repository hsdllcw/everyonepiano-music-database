

**爱殇双手简谱** 和五线谱完全对应。

_爱殇_
是董贞翻唱的一首歌曲，原唱为小时姑娘。董贞，出自人才辈出的中国传媒大学，是目前中国大陆地区唱功实力派且比较擅长创作的女歌手。其嗓音温柔而甜美，曾一度被誉为大陆的邓丽君与杨钰莹的综合体，又因其大部分歌曲都是自己创作，又有才女之称。早期主要从事幕后工作，为许多韩国及国内的歌手创作歌曲，例如韩国巨星张娜拉，郑在娟，国内新人曹璐等等。

歌词下方是 _爱殇钢琴谱_ ，希望大家喜欢。
