

抖音“以后的日子你要好好照顾她，我不在她身旁你不能欺负她”是歌曲《 **阿拉斯加海湾** 》当中的一句歌词，演唱者是蓝心羽，原唱是菲道尔。

在2020年11月25日，这首歌在被蓝心羽之后，在网上火了起来，深受大家的欢迎。并登上了各大排行榜！

简单的旋律，却把句句歌词带入了内心，好像在诉说一些遗憾，遗憾虽只能是遗憾，但那些回忆历历在目，那些情感仍弥足珍贵，深藏于心。
