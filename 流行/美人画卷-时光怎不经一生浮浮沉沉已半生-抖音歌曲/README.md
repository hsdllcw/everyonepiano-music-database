一壶浊酒欲随风

最近抖音上这首歌特别的火。据悉，此曲叫做《 **美人画卷** 》，由魏楚沅编曲，斐琳作曲，十满作词，闻人听書演唱。发行于2020年10月13日。

有些人遇到了就是命，该走的始终留不住，留下的确永远是最伤的。正所谓：时光荏苒，曾是惊鸿照影来，而如今情字难懂，依稀间可见那美丽容颜，青山常伴绿水，燕雀已是南飞，美人画卷残留一丝青灰叹余美。
