_五月雨恋歌_
是由虚拟歌姬初音ミク演唱的一首歌曲，曲风轻柔细腻。此外，在虚拟歌姬巡音ルカ演唱的《秋风恋歌》中，穿插有与这首歌曲相同的旋律，两者互为联歌（也称对歌）。

初音未来（初音ミク/Hatsune Miku），是2007年8月31日由CRYPTON FUTURE
MEDIA以Yamaha的VOCALOID系列语音合成程序为基础开发的音源库，音源数据资料采样于日本声优藤田咲。

歌词下方是 _五月雨恋歌钢琴谱_ ，喜欢的朋友可以免费下载学习。
