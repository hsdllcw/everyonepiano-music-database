《 _暖暖_
》是马来西亚流行女歌手梁静茹演唱的一首歌，李焯雄作词，人工卫星谱曲，收录于2006年10月6日梁静茹发行的第8张专辑《亲亲》中，是梁静茹甜蜜情歌的代表作之一。2007年该歌曲获得2006年Hit
FM 年度百大单曲第52名。2011年梁静茹发行的精选专辑《现在开始我爱你》中亦收录该歌。  
  
《 _暖暖_
》是一首非常甜蜜温暖的情歌，歌词和旋律恬淡宜人，十分贴心，能代表一个爱情观，即平凡的爱情，情侣间在可以互相分享共同的快乐、分担彼此的困难，同时这首歌曲也是很适合表白的一首歌曲，在梁静茹感性的诠释下，甜蜜却甜腻，给人幸福的感觉。
