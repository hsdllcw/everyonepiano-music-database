

**说谎** 是林宥嘉专辑《感官/世界》中的收录曲。同时是电影《针尖上的天使》主题曲、电视剧《杜拉拉升职记》中的插曲。

_《说谎》_
这首歌其实是林宥嘉在专辑中最后配唱完成的歌曲，因为编曲的一再求好而不断重编，也因为他一直坚持希望可以找到一个对的方式来诠释这个虚构的故事中的真实心情，而一再地配唱。

林宥嘉，2007年7月以大学生的身份，勇夺台湾中视主办的歌唱选秀节目《超级星光大道》第一季冠军，并获华研音乐签约成为专属歌手。林宥嘉拥有独特的声线和极佳的唱功，人称“迷幻唱腔”，“迷幻王子”。
