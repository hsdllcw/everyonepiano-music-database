这是一首受到以抒情为题材的流行音乐歌曲《Lost》影响的、防弹少年团主唱成员：金硕珍、朴智旻、金泰亨、田柾国声线及钢琴伴奏的小分队歌曲。该歌曲由制作《MIC
Drop》混音单曲的史蒂夫•青木参与合作，干练的旋律、节制的感性与富有魅力的声音相结合，给人一种舒适的感觉。
