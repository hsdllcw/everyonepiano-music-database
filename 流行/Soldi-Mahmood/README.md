

**Soldi** 是意大利歌手Mahmood演唱的歌曲，发行于2019年2月6日，收录在其首张录音室专辑Gioventù bruciata中。

Mahmood原名Alessandro Mahmood，为埃及－撒丁岛裔1992年出生米兰。“ _Soldi_
”以歌手的童年为背景，讲述了一个单亲中东移民家庭的挣扎，是一个关于在充满背弃的世界中生存的故事。
