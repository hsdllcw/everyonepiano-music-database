本首歌曲与上一首《少年与海》一问一答，上下呼应，G.E.M.邓紫棋以她独到的音乐人视角解答出关于“我为何醒来？我该如何存在？”这一亘古的究极哲学命题。同时，在MV中，G.E.M.邓紫棋也将我们带到了奇迹一刻。当海浪拔地而起，Gloria看到的是爱人的深情守望，而Evan看到的是爱情的失而复得。对他们彼此而言，这都好像Gloria于废土中虔诚祈祷的神迹降临。

《 _老人与海_
》灵感来自于G.E.M.邓紫棋在阅读海明威《老人与海》中产生的共鸣，人生正如同汹涌的海浪，每个人都孤独的前行在风浪之中，正如G.E.M.邓紫棋在《老人与海》歌词中写道“桑田沧海，爱是依旧无改，我是因爱而在，为爱醒来。”也许爱，才是这亘古的人生命题的唯一解法。
