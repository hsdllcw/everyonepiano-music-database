

**Worth It双手简谱** 和五线谱完全对应。

_Worth It_ 是美国女子组合Fifth Harmony和Kid Ink共同演唱的一首歌曲。组合成员包括：Dinah Jane
Hansen，Lauren Jauregui，Ally Brooke，Normani Kordei Hamilton和Camila
Cabello。五位成员的平均年龄在16岁左右，她们中最大19岁，最小15岁。这五位有才华的女孩子从一档著名选秀节目The X Factor USA
脱颖而出。之后签约了公司Syco Music+Epic Records。

歌词下方是 _Worth It钢琴谱_ ，希望大家喜欢。
