

**香水有毒**
是由陈超填词、谱曲，江建民编曲，胡杨林演唱的一首宣传单曲。该歌于2007年获得“最佳无线单曲大奖”和“内地2006Jamster最受欢迎手机铃音奖”两大奖项。

  
MV中胡杨林一改清纯甜美的形象，玩起了湿身诱惑，这是胡扬林首次挑战大尺度，然而这个镜头本来不在计划之内，是在拍摄当天，胡杨林突然想到了玛丽莲•梦露那句名言：“当我什么也不穿时，我只穿香奈儿N5号香水。”而歌曲又叫做《
_香水有毒_ 》，于是胡杨林将这个想法告诉了导演，得到了导演的赞许，于是在拍摄中加上出浴、裸背等镜头。

  
歌词下方是 _香水有毒钢琴谱_ ，希望大家喜欢。
