

**Lucky**
是韩国SM娱乐有限公司于2012年4月8日正式推出的12人男子组合Exo的歌曲。该组合，由金珉硕、鹿晗、吴亦凡、金俊勉、张艺兴、边伯贤、金钟大、朴灿烈、都暻秀、黄子韬、金钟仁和吴世勋（按年龄降序排列）组成，其中8名为韩国成员，4名为中国成员。  
  
EXO这个名字从意为太阳系以外的外部行星EXOPLANET一词中得来，蕴含了他们是从未知的世界来的新星之意。在EXO后添加代表Korean（韩语）的K与代表Mandarin（中文）的M，组成EXO-
K和EXO-M两个小组，分别在韩国和中国展开活动。  
_  
Lucky_ 这首歌曲收录在其专辑《XOXO Repackage Hug Ver.(中文版)》中。
