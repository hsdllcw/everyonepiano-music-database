

“天都黑了你都没来过没来坐，戏都完了眼都没眨过没哭过”歌手品冠2004年的歌曲《 **门没锁** 》，如今在自己的儿子翻唱几句后火爆大江南北。

朗朗上口的歌词，加上8岁孩童颇具磁性的声线随意哼唱，虽略有瑕疵，但正是这样亲切的矛盾，将歌曲最朴素的情感展露无遗，打动了听众。

在抖音平台，话题“门没锁”的视频，有7.6亿次播放，品冠的儿子翻唱版本的《 _门没锁_
》已被使用过44.7万次。也就是说，有几十万个带着这段声源的作品在抖音传播、刷屏，孵化出了当下现象级的文娱事件。

网友戏称，沉寂多年的品冠终于靠着“父凭子贵”火出圈了。
