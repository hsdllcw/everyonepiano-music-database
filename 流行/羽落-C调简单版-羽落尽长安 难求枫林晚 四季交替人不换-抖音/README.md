

羽落尽长安 难求枫林晚  
四季交替人不换  
我情拆几段 你泪眼潸然  
儿女情长戏子唱不完

大家在刷抖音的时候，是不是也常听到这段旋律。据悉，此曲叫做《 **羽落**
》由烟十八作词，古月作曲，孟维来演唱，发行于2021年4月6日。歌词中满满都是对感情的回忆以及如今的孤独。

_羽落双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
