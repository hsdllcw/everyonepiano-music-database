

**耿耿于怀双手简谱** 和五线谱完全对应。

_耿耿于怀_ 是刘昊然、谭松韵主演的网剧《最好的我们》的主题曲，由王笑文演唱。我们都感伤与青春里遇见的人，耿耿于怀是因为不能忘却。

《最好的我们》是北京爱奇艺科技有限公司出品、小糖人文化传媒有限公司承制的校园青春网络剧。该剧根据八月长安的同名小说改编，讲述了意外进入振华高中的普通学生耿耿和“学霸”余淮成为了同桌，他们共同成长，经历分离又重逢的一系列故事。2016年11月，该剧荣获2016中国泛娱乐指数盛典“中国网生内容榜-
网络剧榜top10”。

歌词下方是 _耿耿于怀钢琴谱_ ，希望大家喜欢。
