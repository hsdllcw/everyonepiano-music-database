_爱的奉献_
是韦唯演唱的一首歌曲，由黄奇石填词，刘诗召作曲。收录在韦唯1989年发行的专辑《走不完的爱》中。1989年，韦唯在中央电视台春节联欢晚会上演唱了这首歌曲。

这首歌曲凭借其浅显易懂的歌词、舒缓温暖的旋律以及高尚大气的主题而广为流传。此曲洋溢着人间最美的温情，不仅脍炙人口，更成为公益歌曲的标签；当国家和人民需要温暖和感动的时候，这首歌的旋律总是会响起。
