

**哭砂** 是黄莺莺演唱的歌曲，由林秋离作词、熊美玲作曲，收录于黄莺莺1990年个人专辑让爱自由中，并被众多歌手重新翻唱了20余次。

  
歌词下方是 _哭砂钢琴谱_ ，希望大家喜欢。
