

**金玉良缘双手简谱** 和五线谱完全对应。

《 _金玉良缘_ 》是2014年古装言情轻喜剧《金玉良缘》同名片尾曲，由桐华作词，林海、冯硕作曲，李琦、贾青演唱。

《金玉良缘》是由浙江梦幻星生园影视文化有限公司与华杰工作室（霍建华工作室）联合制作，黄祖权导演，霍建华、唐嫣、邬君梅、王阳、贡米、黄明等联袂主演的大型古装言情轻喜剧。本剧讲述了明朝初年永乐年间，掌管兵器制造局的朝廷世家金家独子金元宝偶遇江湖女子玉麒麟，于是展开了一段轻松幽默又真挚感人的故事。该剧于2014年04月21日在江苏卫视和深圳卫视首轮上映播出。
