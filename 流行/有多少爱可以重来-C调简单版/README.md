

《 **有多少爱可以重来**
》作为一首经典的老歌，在人们心中也有着极高的地位，想必当年听过这首歌曲的人早已经青春不再，而这首歌曲也是给人们的心中留下了极为深刻的印象，因为歌曲里面包含的是他们的青春！

此曲原由黄仲昆演唱。后迪克牛仔翻唱。迪克牛仔的嗓音沙哑，具有磁性，给人分外沧桑的感觉。顶着摇滚头衔的他凭借真情的自然流露打动了众多歌迷，在他的身上，是岁月的沉淀，是沧桑而不失情。这首《
_有多少爱可以重来_
》配上他的声音是完美的演绎，歌曲描述的是两人分手多年之后，男人回想起那些往事，回忆起那些争吵的缘由，不由得有些感叹，谁对谁错又如何，只是那些曾经的爱再也回不来了，最后，仿佛是在告诉听众好好珍惜身边的爱人。
