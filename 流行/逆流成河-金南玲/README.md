

**逆流成河双手简谱** 和五线谱完全对应。

_逆流成河_ 由廖伟志作词，廖伟志作曲，金南玲演唱的歌曲。

“每当我深夜辗转反侧，悲伤就逆流成河”，从咿呀学语，到途径青春，生命就像一朵花儿，在“青春”之城绽放。这一段旅程，我们有过怦然心动的初恋时刻，也有男男女女情感纠葛，“剪不断，理还乱”的虐心纷争。青春如诗如歌，像一本泛着酸涩的小说，静默时，悲伤就逆流成河……

这是一首很悲伤的歌曲，它会勾起我们心中的的一些悲伤，但是这种悲伤很难记起，只会默默伤感，可惜，惘然若失。
