

**Moonlight** （ _月光_ ）是韩国超人气组合EXO演唱的歌曲。  
  
EXO是韩国SM娱乐有限公司于2012年4月8日正式推出的12人男子组合。现以11名成员的形式展开活动，分别为金珉硕、鹿晗、金俊勉、张艺兴、边伯贤、金钟大、朴灿烈、都暻秀、黄子韬、金钟仁和吴世勋（按年龄降序排列），现活动成员中8名为韩国成员，3名为中国成员。EXO这个名字从意为太阳系以外的外部行星EXOPLANET一词中得来，蕴含了他们是从未知的世界来的新星之意。在EXO后添加代表Korean（韩语）的K与代表Mandarin（中文）的M，组成EXO-
K和EXO-M两个小组，分别在韩国和中国展开活动。
2012年4月，EXO正式出道，发行首张迷你专辑《MAMA》，并于同年11月获香港亚洲音乐盛典最佳亚洲新人团体奖。2013年12月，EXO首张正规专辑主打歌曲《Growl》被美国Billboard榜单选为2013最佳K-
Pop音乐，EXO正规一辑《XOXO（Kiss&Hug）》专辑销量突破100万张。
