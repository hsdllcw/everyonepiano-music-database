

**彼此的未来双手简谱** 和五线谱完全对应。

_彼此的未来_ 是SNH48演唱的歌曲，由甘世佳作词，OKboys作曲、编曲，收录于2017年发行的春季新单同名EP中。

《彼此的未来》曲风清新温暖，旋律和歌词朗朗上口，伴着成员温暖动人的演绎，引发无限青春遐想。“许多年以后，当我们再回首，会感激这些年与你相伴，共同成就了彼此的来。”SNH48春季新单用音乐的形式，将这一份最真挚的礼物感恩回馈给一路陪伴和温暖自己的粉丝们——致现在与未来的你们，愿时光不老，我们不散。

歌词下方是 _彼此的未来钢琴谱_ ，喜欢的朋友可以免费下载学习。
