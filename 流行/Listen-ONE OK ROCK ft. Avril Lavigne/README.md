

**Listen双手简谱** 和五线谱完全对应。

_Listen_ 是Avril Lavigne助阵日本摇滚乐队One Ok Rock的新单。

ONE OK ROCK（ワンオクロック），日本四人摇滚乐队，简称「ワンオク」、「OOR」。所属AMUSE事务所，唱片公司为A -
Sketch。多支热门单曲进入Oricon公信榜，被视为日本摇滚界最有潜力的团体备受瞩目中。

歌词下方是 _Listen钢琴谱_ ，希望大家喜欢。
