_怎样遇见你_ 是中国内地女歌手孙露的一首歌曲，发行于2017年03月29日。

孙露歌曲风格低声浅唱治愈系。她翻唱了很多歌曲，互联网歌手排名一度超过众多天王天后。以低声浅唱的歌声吸引人，她的歌声一直温婉如水，她的独一无二的嗓音，穿透灵魂的磁性女声
。
