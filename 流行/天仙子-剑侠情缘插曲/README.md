_天仙子_
，这首歌的歌词在金山公司游戏《剑侠情缘》中即已产生，在2000年金山公司推出的游戏《剑侠情缘2》时对其谱曲，并作为暴机结束曲。同年，这首歌被由金庸作品《侠客行》改编的同名电视剧采用当片尾曲。歌词是按照词牌“天仙子”的格律填写的，分上下两阙，每一阙的最后一句是7个字，不能断行。
