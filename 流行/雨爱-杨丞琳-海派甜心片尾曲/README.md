《 _雨爱_ 》由Wonderful
作词，金大洲作曲，杨丞琳演唱的歌曲，收录在杨丞琳2010年发布的专辑《Rainie&Love...?雨爱》中，这首歌曲还是偶像剧《海派甜心》的片尾曲。

台湾偶像剧《海派甜心》片尾曲，《雨爱》MV重金邀请日本天王天后摄影大师川口贤典亲自掌镜，跨海拍摄专辑照，浪漫绝美新造型，珍珠闪耀。MV男主角：王柏杰，MV拍摄地点：高雄。《雨爱》
融合英式抒情摇滚 绝美忧郁的 Rainie Love 杨丞琳唱将激情感渲染。
