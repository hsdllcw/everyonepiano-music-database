_Gorgeous_ 是Taylor Swift第六张录室专辑《reputation》的第三支单曲（专辑中是第八首歌）。该歌曲由Taylor
Swift、Max Martin、Shellback作词作曲，发行于2017年10月20日。

Taylor
Swift的歌为她赢得了最大的歌迷群：年轻人。她写歌的习惯也成为捕获年轻人的杀手锏，她以写自己的成长经历闻名，歌曲里都是青春的味道，这一点正好切合了年轻歌迷的心，他们能听懂歌曲的内容，自然就会产生共鸣，从青春期开始，这一路下来，会有一批歌迷跟着她一起成长。她很善于利用她年轻这个优势。
