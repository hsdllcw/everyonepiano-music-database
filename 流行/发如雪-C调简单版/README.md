

**发如雪** 出自周杰伦2005年11月1日发行的专辑《十一月的萧邦》。周杰伦的音乐作品绝对是一座最富饶的矿脉，当你徜徉于此，各种惊叹美不胜收。

《 _发如雪_
》这首中国风的音乐作品，具有浓厚的音乐元素，其中的很多语句道出哲理，从一首音乐作品放大到前世今生，有人曾经如此评价：所谓生生世世，缘起缘落，一切只因心尘未脱的世间痴男怨女的心境。的确，对于感情的倾诉是亘古不变的话题。

这首歌的MV是周董第一次执导，充分的准备，场景的精心挑选，角色的把关，可以看到周杰伦非常用心，尽管稍稍有些紧张，但绝对属于精品之作。

“繁华如三千东流水，我只取一瓢爱了解”，喜欢这样的词，应该是出自于“弱水三千，只取一瓢饮”这段佳话吧！唱出了一个人对爱情如磐石般的坚定！
