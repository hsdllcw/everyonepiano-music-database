

“是否冥冥之中注定你我没有结果”是不是很多小伙伴对这句歌词都比较熟悉，其实下一句就是“为何还要飞蛾扑火。而这首歌的名字正是《 **飞蛾扑火**
》由王键演唱。

这是一首非常老的歌曲，听起来能够让人想起一些往事，物是人非，想起来能够让人伤心不已，歌曲舒缓缠绵，夹杂着丝丝温柔，让人久久不能恢复心情。

_飞蛾扑火_ 有两层意思，其一是指飞蛾扑到火上，比喻自取灭亡。其二是指赞扬人们为了自己的理想和追求而付出时的义无反顾、不畏牺牲的精神。

明明看不到希望，却还是爱上了一个人，就像飞蛾扑火，明知道是死路一条，却还是扑上去。
