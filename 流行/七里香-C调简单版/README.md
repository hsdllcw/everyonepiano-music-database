

《 **七里香**
》由中国台湾男歌手周杰伦演唱。这是一首中国风味的诗意情歌，夏天听有一种的凉爽感觉，因为只要前奏一响起，你的眼前就会不自觉地浮现出一幅爱情画卷，美好、羞涩，如海边的风轻抚脸庞，更如爱恋的她吻在脸上！

从音乐上看，《 _七里香_
》已从欧洲风情中走出回到了自由清新的曲调中。和《东风破》比同样是在弦乐铺垫下的浅唱，同样是被中国的古典风情冠了名，但《七里香》少了迟暮，多的是时过境迁后的怀念，有一份收获的心情。

歌曲MV走的是唯美浪漫路线，整个场景在日本的群马县高崎市拍摄，找了当地的神社、稻田、花海拍摄。周杰伦在日本拍摄电影《头文字D》时，特地向剧组请假一天拍摄新歌MV。和女主角田中千绘在美丽的场景里谈恋爱，手牵手在花园里奔跑、荡秋千。MV封面照在海参崴拍摄，由于海参崴的治安不太好，有一名警察及两名高大保镖贴身保护。
