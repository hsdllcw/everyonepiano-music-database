

**采茶纪双手简谱** 和五线谱完全对应。

_采茶纪是_ 来自重庆开县16岁少女双笙的动听古风歌曲。

采茶纪词曲轻快明朗，趣意十足。不同于以往古风歌曲常表达的离愁、伤逝、求之不得、爱恨交织等浓烈或凄清的基调。它以主人公“四宝”作为第一人称讲述了一段采茶时节的生活片段。

歌词下方是 _采茶纪钢琴谱_ ，喜欢的朋友可以免费下载学习。
