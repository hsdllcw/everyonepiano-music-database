

周杰伦最新专辑《最伟大的作品》再添一首中国风必传经典《 **红颜如霜** 》！此曲由方文山填词，以梨园名剧《凤求凰》为主题，于 2022年7月15日发行。

不同于名剧《凤求凰》中卓文君和司马相如那样琴瑟相合，周杰伦的这首《 _红颜如霜_
》有点伤感。听歌词应该是分隔两地的恋人，无法见面只能通过信件传达情意。盼着的信好不容易到来，娟秀的笔迹却只有寥寥几行，一句“甚安勿念”让人彷徨，让人惆怅。热烈的心此时冷如冰霜。最后不得不无奈的感叹一句“红颜如霜”。

此曲在保留以往风格的基础上，又有了一些创新，在歌曲里加入了戏剧元素，让人耳目一新。整首歌曲的曲调非常抓耳，让人不知不觉地就单曲循环起来。
