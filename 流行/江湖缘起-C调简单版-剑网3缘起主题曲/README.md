

《 **江湖缘起** 》是中国大陆男歌手周深为游戏《剑网3缘起》演唱的主题曲。歌曲中，周深的声音，纯净悠扬，空灵婉转，并加入了戏腔，超赞。

而这首歌也是周深首次独立担任制作人，此前只和别人一起担任共同制作人。

此曲获得QQ音乐巅峰人气榜人气黄金单曲认证、人气榜周榜冠军、人气榜日榜冠军、游戏音乐榜周榜冠军、国风热歌榜周榜冠军，酷狗音乐国风新歌榜周榜冠军、ACG新歌榜周榜冠军、飙升榜日榜冠军，及酷我音乐飙升榜日榜冠军、华语榜日榜冠军。

由你音乐榜乐评人点评：“歌曲开头由婉转的弦乐引进，随后笛萧、琵琶、二胡等乐器演奏逐渐融入，营造出了浓郁的古风氛围，搭配周深缥缈空灵的质感嗓音，让听众仿佛置身侠气十足的江湖场”。
