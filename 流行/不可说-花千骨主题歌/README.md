

**不可说**
，该歌曲是2015年电视剧《花千骨》的主题曲，于2015年5月25日发布。《花千骨》是由慈文传媒集团制作并发行，林玉芬、高林豹、梁胜权联合执导，霍建华
、赵丽颖领衔主演，蒋欣、杨烁、张丹峰、李纯、马可、鲍天琦、安悦溪、蒋一铭、杨安琪等主演的古装玄幻仙侠剧。  
  
另外， _不可说_ 是由麦振鸿作曲，钱丽婧、fresh果果、饶俊作词、麦振鸿、吴清华编曲，霍建华、赵丽颖演唱的歌曲。
