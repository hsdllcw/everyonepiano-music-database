

**阳光总在风雨后双手简谱** 和五线谱完全对应。

_  
阳光总在风雨后_ 是陈佳明作词作曲，吴庆隆编曲，许美静演唱的一首歌曲，收录在专辑《都是夜归人》中。
这首歌成为中国女排的队歌，也因为女排队员的演唱而风靡一时。

  
许美静，原名许美凤，是新加坡女歌手。在1990年代后期于台湾、中国大陆、新加坡等国语流行音乐市场上受欢迎。2000年后隐出流行音乐界，开始从事公益活动。曾于2003年独自在意大利生活一年。2006年6月因“酒店事件”重新出现在媒体视线内，之后被诊断为轻度精神分裂和抑郁症。2006年11月在朋友的帮助下开始录歌。

  
歌词下方是 _阳光总在风雨后钢琴谱_ ，希望大家喜欢。
