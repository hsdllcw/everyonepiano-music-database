

**那个男人双手简谱** 和五线谱完全对应。

_那个男人_
是杨宗纬自己填词的新歌，翻唱自韩国歌手玄彬演唱的韩剧《秘密花园》的OST。歌名耐人寻味，是否写自己的爱情故事？杨宗纬说：是看完韩剧所写。根据男女主角故事所写。

杨宗纬（Aska
Yang），1978年4月4日生于中国台湾桃园市，中国台湾歌手。其歌声阳刚而细腻，富含感情，辨识度高，感染力强，以演唱抒情歌曲见长，被誉为“催泪歌神”，又昵称“鸽王”；他的特点在于个性特质鲜明，声线发挥也很稳定。

歌词下方是 _那个男人钢琴谱_ ，希望大家喜欢。
