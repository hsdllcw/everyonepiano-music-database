

**钟无艳双手简谱** 和五线谱完全对应。

_钟无艳_ 是林夕作词，Christopher
Chak作曲，陈珀编曲，谢安琪演唱的歌曲。收录于2007年12月11日发行的新曲+精选专辑《3/8》中。2007年，该曲荣获SINA
Music乐坛民意指数颁奖礼2007最高收听率十大歌曲。

此曲所描画的是那种一厢情愿地去爱一个人的故事。从编曲上“钟无艳”更能轻易地将聆听中带入到歌词的意境里面，此曲的那个带着几分玩味气质的名字，应该是可以满足到时下乐迷的新鲜感，加上贴近歌词所描画的情绪和剧情，很能让人想到那位因为样貌而有过崎岖情路的历史人物—钟无艳。
