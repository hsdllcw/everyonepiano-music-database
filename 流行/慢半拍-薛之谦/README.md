

**慢半拍** 是由薛之谦作词并演唱，许嵩作曲，许嵩编曲，许嵩混音，许嵩担任制作人和母带工程师的歌曲，收录于2019年6月26日发行的专辑《尘》中。

Guitar音色都让整首歌回归根源性音乐方向，附加上了超越时代记忆的人生感知力。鲜活的SOLO演唱，推动着歌曲之于内心世界返璞归真的情感输出。
