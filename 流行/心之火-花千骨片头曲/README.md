

**心之火双手简谱** 和五线谱完全对应。  
  
_心之火_ 是2015年古装仙侠电视剧《花千骨》片头曲，由飞儿乐团、彭佳慧演唱，阿沁作曲，陈建宁和谢宥慧作词，已于2015年6月2日正式推出。  
  
《花千骨》该剧改编自fresh果果同名小说，讲述少女花千骨与长留上仙白子画之间关于责任、成长、取舍的纯爱虐恋。该剧于2014年5月6日开机，9月15日杀青，8月12日发布中文及英文版的预告片。  
  
歌词下方是 _心之火钢琴谱_ ，希望大家喜欢。
