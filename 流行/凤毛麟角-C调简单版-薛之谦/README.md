薛之谦歌曲《 **凤毛麟角** 》由郭冠廷作词作曲，发行于2021年12月24日零点。

整曲透露着薛之谦长久以来的人格特质——“个性”“不服输”。每个人刚出世时都是一块充满棱角的石头，进入社会后，石头便开始滚动，磕磕碰碰，磨去了自己的“凤毛麟角”，最终成为一颗圆滑的石头。一个人追求目标时“当初的真我”和“现在的我”自我挣扎、最后被社会规则同化的过程，而这个过程你是选择坚持自己还是真的为了得到自己想要的而不择手段，从而抛弃初心去放弃独特的自己，这正是薛之谦在这首歌里所质问自己也是质问大家的一个问题。
