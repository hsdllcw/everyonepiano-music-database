

**等我回家双手简谱** 和五线谱完全对应。

_等我回家_
是由薛之谦创作并演唱的一首歌曲，收录在薛之谦2013年发行的个人专辑《意外》中。深情款款的“薛氏情歌”是他的不变招牌，他的歌曲既迎合了市场的喜好，又是他自己擅长的情歌类型，这些歌帮助他积累了人气，创作才子的名号也由此沿袭下来。

像《等我回家》这样的歌，要是放在以前的唱片当中，肯定会被至少升高两个调来唱。用了偏低的调，正是为了契合歌词中放低自己，诚恳认错，与过去那个年少贪玩的自己划清界限，回归平淡真实生活的意义。

歌词下方是 _等我回家钢琴谱_ ，大家可以免费下载学习。
