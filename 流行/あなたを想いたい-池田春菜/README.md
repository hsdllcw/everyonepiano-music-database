

**あなたを想いたい双手简谱** 和五线谱完全对应，属于EOP教学曲。

_  
あなたを想いたい_ 是池田春菜的歌曲，是由須谷尚子作词，石川真也作曲的。是一首很好听的曲子。同时这首曲子也是Routes的片头曲。

  
歌词下方是 _あなたを想いたい钢琴谱_ ，希望大家喜欢。
