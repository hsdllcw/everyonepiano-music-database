

**给未来的自己** 是梁静茹专辑《崇拜》中的收录曲。同时也收录在专辑《美丽人生 （台湾换季庆功版）》中。

崇拜专辑中，静茹尝试更多元的曲风，制作人小虎、小芬姐、石头合力在音乐上创造丰富的听觉感受，配器从极简到极繁，曲风从清唱、小品抒情、流行抒情、节奏民谣、流行摇滚、甚至到Cold
Play式节奏摇滚等，都打定主意要让静茹“不好过”。三个制作人比赛似的对静茹一磨再磨，只为激发唱歌像喝白开水一样容易的静茹，充分发挥凌驾所有新世代女歌手的神样唱功。

梁静茹，马来西亚女歌手。1999年出道至今已发行了11张大碟和2张精选专辑，深受两岸三地和新马歌迷的爱戴。代表作品有《勇气》、《无条件为你》、《分手快乐》、《宁夏》、《暖暖》等，被誉为“情歌天后”。

_歌曲给未来的自_
己的歌词写得很真实，有生活中的伤害和各种无奈，但是也有对未来的期待和为了未来的自己会更好而努力抓住梦想的勇气。同时，该曲的旋律虽然会有些淡淡的忧伤，但却是首很美的曲子。如果大家有喜欢
_给未来的自己钢琴曲_ 的话，可以免费下载 _给未来的自己曲谱_ 。当然，对于钢琴学习者来说给 **未来的自己五线谱** 应该也有需要的吧～
