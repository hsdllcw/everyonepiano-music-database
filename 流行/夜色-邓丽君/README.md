

**夜色**
是中国台湾歌唱家邓丽君演唱的歌曲，情深意切，真挚感人，值得一听。邓丽君的歌很有中国韵味，这种韵味是汉语的特征，也是区别于其它文化的音乐特色。对于中国文化的意蕴来讲，她已经不是一个歌手，其歌艺成就已超越流行音乐的范畴与层次。她以柔美动人的歌声和令人心醉的情歌广泛吸引了众多地区的青年人和老年人，如中国、马来西亚、新加坡、印尼、日本、朝鲜半岛等地。

  
歌词下方是 _夜色钢琴谱_ ，希望大家喜欢。
