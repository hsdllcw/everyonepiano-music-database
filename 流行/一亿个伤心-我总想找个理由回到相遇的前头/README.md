

**一亿个伤心** 是蒙面哥和李超演唱歌曲，由蒙面哥作词、作曲，琴之编曲，收录在其2012年5月发行专辑《恋恋红尘梦》之中。 该歌曲与《To Love's
End》存在很多相似之处。

该歌曲歌声伤感 、歌词也伤感，唱出了80、90后无奈和悲伤亦唱出了这代人的纯真与小浪漫，让大家的听觉耳目一新，深受网友们的喜爱。

歌词下方是 _一亿个伤心钢琴谱_ ，大家可以免费下载学习。
