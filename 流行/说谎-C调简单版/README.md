

**说谎**
是林宥嘉演唱的一首歌曲，同时也是电视剧《杜拉拉升职记》插曲和电影《针尖上的天使》的主题曲。该歌收录在林宥嘉2009年10月30日发行的专辑《感官/世界》中。

作为专辑内地首波主打曲的《 _说谎_
》，延续了林宥嘉最为擅长的抒情曲路线，由华研网络词曲大赛作曲组冠军的内地新秀李双飞作曲，作词人则由为S.H.E写过无数动人情歌的施人诚老师担当。整首歌曲启用了类似小说的结构，几乎都用对白式的句子来交代故事与主角的心情，歌中反复唱着“我没有说谎”，但几乎句句都是谎话，可说是布局结构相当大胆的一首作品。
