

《 **是风动** 》是银临和河图的首次合作，收录于银临第二张原创专辑《蚍蜉渡海》。歌词意境超美，旋律也很洗脑。可谓是余音袅袅，句句走心！

全曲以“是风动”为题，自佛典而言，则暗喻“心动”之意，以云与山之关系作比，自相对运动而言，云动亦可言是山动，而云之动，则亦是风动之结果。而心动则将归于心定，所谓“不动于心见你如是才无憾”！
