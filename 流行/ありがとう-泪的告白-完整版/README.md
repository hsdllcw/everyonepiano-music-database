《 _ありがとう_ 》（泪的告白）由日本女歌手KOKIA（吉田亚纪子）演唱。这首歌并不是情歌，而是她为了纪念逝去的陪伴自己多年的宠物狗而写的回忆曲。

2007年，由于富士电视台节目《カワズ君の検索生活》使用了乐曲《ありがとう•••》并得到了强烈的反响，所以该单曲再次贩售。

1999年，《ありがとう•••》在香港国际流行音乐大奖中获得了第三名的成绩，并被郑秀文用粤语翻唱为《Arigatou》并获得第一名。
