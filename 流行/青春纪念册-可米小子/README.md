

**青春纪念册双手简谱** 和五线谱完全对应。

_青春纪念册_ 是台湾组合可米小子演唱的歌曲，原曲是日本组合I
WiSH演唱的《サマーブリーズにのって》。可米小子是可米家族与新力唱片2002年联手推荐新一代的青春活力组合。成员包括王传一、曾少宗、申东靖、张庭伟以及安钧璨和许君豪。可米小子是F4的成功所引起的男子团体风潮下的产品之一，2002年11月19日推出第一张专辑，许君豪伤重退出。可米小子后来由于唱片销售不佳和风潮退烧等缘故在2005年宣布解散。2012年10月19日，成员申东靖因病去世，可米小子成员再次引起关注。

歌词下方是 _青春纪念册钢琴谱_ ，希望大家喜欢。
