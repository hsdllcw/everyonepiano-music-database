

《 **其实不想走** 》是周华健演唱的一首歌曲。作为华语歌坛的巨星，周华健发布了不少令大家耳熟能详的经典歌曲，比如《[
**刀剑如梦**](Music-2285-刀剑如梦-周华健-EOP教学曲.html "刀剑如梦")》、《[ **朋友**](Music-6467-朋友-
周华健.html "朋友")》等。

在爱情里我们都有口是心非的时候，往往因为这样，人生才会生出那些遗憾，正如歌词所唱“其实不想走，其实我想留”。

虽然这首歌曲看似略平凡且平淡，但正因为其朗朗上口的词曲顺应了当年的下海打工潮，所以很适合大众去传播。编曲用晶莹剔透的吉他弹拨来搭配歌手的中音，靠着大量节奏性打击乐器和体鸣乐器以及华健有力的喉腔来合奏，天时地利加上人和，成为特别容易大合唱的KTV聚会神曲之一。
