**_一个人睡_
是魏晨首张EP中的第一主打歌曲，于2007年12月6日在中歌榜独家首播。EP的第一主打歌曲《一个人睡》是一首易于传唱的歌曲，表达了一个人孤独之时，面对回忆和逝去情感的细腻独白和小小无助。  
  
一个人睡MV，将魏晨塑造成一位重病女孩的守护天使，编号为“5201314”的隐形天使，因为女主角反复默念前男友留下分手纪念项链上“5201314”的数字而被启动现身，故事在天使与女主角互见彼此的小世界里展开，“天使”魏晨为了安慰失恋心伤的女主角，每日变换心思令她开心，变魔术、到天台聊天，甚至变身成女主角前男友的样子...可是，他们都忘记了“天使不可以恋爱，爱上即消失无踪”的咒语。魏晨忽然消失在女主角的视线中，令重病的她再次陷入一个人的孤独中，魏晨在一旁默默看着她，无能为力的心痛......
