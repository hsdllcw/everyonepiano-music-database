

最近抖音上有一首传唱度非常高的歌，因为被一群学生翻唱而走红，其中有句歌词叫做“还记得你答应过我不会让我把你找不见”，此曲叫做 **西海情歌**
，原唱为刀郎，后降央卓玛翻唱。

歌曲创作背景：瑛与勇儿都是大学学生一同报名参加了环保志愿者，一同来到零下40摄氏度的可可西里因高寒缺氧而被称为“生命禁区”。勇儿被安排在条件艰苦的陀陀河观察站，勇儿却牺牲在收集资料的路上，这是在可可西里第一个牺牲的志愿者。刀郎在一次采风中，听到了流传在西海的这个故事，他感动之余为瑛和勇儿的爱情故事写下了《
_西海情歌_ 》。
