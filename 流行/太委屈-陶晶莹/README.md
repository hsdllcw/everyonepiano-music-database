

《 **太委屈** 》是台湾艺人陶晶莹演唱的一首歌曲，收录在2000年发行的专辑《爱太缺》中，是陶晶莹经典代表作。

《 _太委屈_
》是最懂女人心的郑华娟为陶子量身创作的抒情歌曲，说的是情人出轨时的委屈心情，由都会新女性代言人陶子来唱分外适切。陶子诠释的方式较为内敛，情感投入颇深，将个中细微的情绪掌握得极佳，很让人感动。这首抒情强打，让你聪见最犀利的陶子，最女人的心情，唱出痛心无奈的情殇觉醒，是
KTV 必练指定曲。
