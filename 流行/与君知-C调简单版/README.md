

**与君知** 是《天官赐福》花怜原创同人曲，由了睡安晚作曲，李蚊香演唱。此曲唱出了无数痴情男女的心痛，唱哭了了无数听众。

原创小说《天官赐福》，作者墨香铜臭。2017年6月16日于晋江文学城开始连载，正文于2018年2月25日完结。同名网络动画片《天官赐福》就改编自此小说。该片由上海绘界文化传播有限公司负责制作，于2020年10月31日起每周六在bilibili独家网络发布，全11集。
