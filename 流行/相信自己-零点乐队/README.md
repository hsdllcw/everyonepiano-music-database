_相信自己_
是由零点乐队作词、作曲且演唱的歌曲，收录于在其2002年发行的专辑《没有什么不可以》中。前奏，骤起骤落、大开大合的旋律，如天风海雨迎面而来；如滔滔江水一泻千里；如火山喷发气贯长虹。自信的精神、非凡的勇气和昂扬的激情，在歌声里展现的淋漓尽致，宏大奇伟的阳刚之美，给人以无穷的力量！
