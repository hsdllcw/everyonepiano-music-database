

蔡徐坤一首《 **情人** 》，在5个国家同时获得超高热度，并稳居“音乐疯传榜”上。如此盛况，不仅仅是对《情人》这首歌给予了认可，也是对蔡徐坤本人的肯定。

《 _情人_ 》，慵懒深情的唱法配合痴情热烈的歌词，可谓是相得益彰，听来有种想谈恋爱的冲动。
歌曲以“眼色”开曲并在演唱是予以短暂停顿，方便拉动听者投入情绪。 歌词更是惹人注听，“眼色 是幻觉
泳池边你的身影构成线”中你让我爱你爱到幻觉，看到什么都是爱你，“肆无忌惮 醉梦酣欢”是你的“热度 包围了我”使我“无意追逐 无法止步”，让我“小鹿乱撞”
“你轻轻一个吻”“我疯狂体会” “be my lover”让我们夜夜起舞，这样你就不再畏惧。
