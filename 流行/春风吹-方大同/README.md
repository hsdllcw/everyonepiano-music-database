_春风吹_
是方大同演唱的一首歌曲，由周耀辉填词，方大同谱曲。这是一首蓝调与中国风结合的上品之作。歌曲开头便已简洁明了的叙述了这是春天里的故事，“桃花”、“柳树”、“青蛙”、“燕子”，些许的田园风格。在歌曲的副歌部分仔细听会听到京剧在混音，感觉很怪异，但是结合的很不错。“春风一吹想起谁”、“春风一吹忘了谁”，这是有所谓也好、无所谓也好，洒脱的做人，这也是大同写这首歌的另一个初衷吧。

歌词下方是 _春风吹钢琴谱_ ，大家可以免费下载学习。
