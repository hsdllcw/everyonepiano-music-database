

**Penny Lane** 是The Beatles（披头士乐队）的一首歌，作者是Paul McCartney，署名“列侬麦卡特尼”。

  
录制于Sgt. Pepper时期，歌曲于1967年2月发行作为双A面单曲，另一首歌曲是Strawberry Fields
Forever，这两首歌最终都被收录到Magical Mystery TourLP 中。这是唱片公司因为乐队有较长时间没有发片才决定发行的。

  
在2004年，Rolling Stone将 _penny lane_ 评为史上最伟大五百首英文歌曲#449.

  
歌词下方是 _Penny Lane钢琴谱_ ，希望大家喜欢。
