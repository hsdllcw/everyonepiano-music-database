

**作曲家**
是李荣浩创作并演唱的歌曲。歌曲里包含他对于快乐最简单的理解。他希望借该曲去提醒自己和每个人：其实抛弃一些矫情的身份修饰，不在刻意制造人生伪高潮，去直面真实自我，人人都是自己乐章的《作曲家》。

《 _作曲家_ 》流行摇滚风格浓烈，在编曲上，通过钢琴与鼓点的交错，令听众感受到自己久违的心跳，并获得挣脱桎梏后，自由天性的释放。

从该曲略带自传性质的歌词中，听众能窥见李荣浩内心关于音乐创作的态度：“音符不会写/写歌是感觉/理论书上的观点有缺陷”，真正的创作高手向来都是无招剩有招，有时候就是那个感觉；“怕歌词写的肉麻/怕浮夸/不谈情说爱/也尽量精彩”，在俗烂情歌满天飞的华语乐坛，歌词题材少写两句台湾偶像剧式肉麻爱情，就如同喝到一杯清爽解腻的绿茶；只是古往今来，才华横溢之人往往内心都是孤独的，才会唱到“而你还不太明白/孤独是个节拍”。
