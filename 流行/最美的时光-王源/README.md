

**最美的时光双手简谱** 和五线谱完全对应。

_最美的时光_ 是王源演唱歌曲，由王源作曲，是王源为母校南开中学80周年生日创作。最美的时光，里面有着一种恬淡的美，一种对过往的感激，一种对未来的期许。

最美的时光表达了对母校诚挚的祝福，温柔质朴打动人心。为了感谢学校的培养，在已有南开学子某次校园征集的歌词基础上，王源把几首歌词中好的部分融合在一起，然后根据歌词去谱曲，歌名也是由王源所取。

歌词下方是 _最美的时光钢琴谱_ ，希望大家喜欢。
