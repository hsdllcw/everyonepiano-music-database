

**你是另一个我** 由陈曦填词、董冬冬谱曲、闫天午编曲、黄征演唱，是2014年7月14日电视剧《神犬奇兵》片尾曲,被收录在专辑《神犬奇兵 电视原声带》。

  
该主题曲MV《 _你是另一个我_ 》也于2014年7月14日同步上线。

  
歌词下方是 _你是另一个我钢琴谱_ ，希望大家喜欢。
