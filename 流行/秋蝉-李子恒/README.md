_秋蝉_
是台湾民歌时代非常著名的一首歌曲，由著名音乐人李子恒作词作曲。1980年凭借此首歌曲获金鼎奖作曲奖。首唱为徐晓菁杨芳仪二重唱，此后有大量的歌手翻唱过此曲，著名的有刘文正、费玉清、小虎队、江蕙、杨钰莹、张清芳、南方二重唱等等。
