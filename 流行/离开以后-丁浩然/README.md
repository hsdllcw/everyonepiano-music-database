

**离开以后双手简谱** 和五线谱完全对应。

_离开以后_
是丁浩然演唱的一首歌曲，收录在专辑《你的故事我的歌》中，2011年03月发行。丁浩然，男，回族，1987年11月出生于天津，他在2008年荣获天津市校园歌手大赛一等奖，他是内地80后原创音乐人。词曲创作、制作及发行，无论是人气还是网络的点击量都取得了前所未有的成功，火爆程度更是锐不可挡。

歌词下方是 _离开以后钢琴谱_ ，希望大家喜欢。
