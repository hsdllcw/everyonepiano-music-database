

《 **小小的太阳** 》是由 **张宇** 演唱的歌曲，由 **十一郎** 填词， **张宇** 谱曲 _（这是典型的秀恩爱）_
。发行于1998年12月01日，收录于专辑《 **月亮 太阳** 》。

这是一首经典老歌，是当年KTV的热门歌曲，唱的人声嘶力竭，听的人泪流满面。遥想当年，小编一曲多情的《 **小小的太阳**
》，当场感动哭了5位小姐姐，非小编不嫁，如果不是KTV保安和经理来了，非出人命不可...
