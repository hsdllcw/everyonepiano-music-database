

**Bad Romance** 是美国流行女歌手Lady Gaga演唱的一首歌曲，歌词和曲谱由Lady
GaGa、RedOne共同负责制作。该歌曲作为推广专辑的首支单曲，被新视镜唱片公司、KonLive等发行公司发布于2009年10月26日，收录在Lady
Gaga的第二张迷你专辑《The Fame Monster》。

作为一个风格多变的创作女歌手，Gaga通常会改变她本人的形象来反映各种类型的女歌手。而在这首歌曲中，我们都可以从中听到布兰妮•斯皮尔斯、麦当娜•西科尼和艾米•怀恩豪斯的音乐风格。歌曲结构由重音鼓点节拍、迪斯科副歌部分和温和的歌声桥段组成。在音乐氛围上，Gaga试图运用忧郁的歌声来打动听众，开头的几段歌声带有浓厚的“仿欧语”口音。这首歌唯一的缺点就是缺乏凝聚力，歌词的中心思想也不明确。
