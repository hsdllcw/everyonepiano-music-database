_动物世界_ 是中国内地男歌手薛之谦的一首歌曲，由郭顶作曲 ，薛之谦作词并演唱。发行于2017年03月29日。

深情款款的“薛氏情歌”是他的不变招牌。大男孩的顽皮、成熟男人的优雅绅士，光影中的薛之谦亦是如此，歌手、演员、主持人，拥有多重身份的他，就像他的文字一样，简单而又纯粹。
