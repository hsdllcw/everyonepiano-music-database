

《 **认错** 》出自许嵩的专辑《自定义》。这首歌曲真的是非常好听，歌词也比较易懂，可以给我们带来很多的感动。

可以说歌中有情，有爱，又仿佛在诉说一个故事。通过讲述他个人的感情经历，让我们听完了之后，内心会与歌手本身产生一种特殊的情感。这种感觉应该就是觉得歌曲像是在写自己，因为这首歌的曲词都是比较悲伤的，因为他是写了男主人公在分手之后的内心活动。
