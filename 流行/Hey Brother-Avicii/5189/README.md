

**Hey Brother** 是瑞典王牌DJ Avicii联手美国蓝草音乐Dan
Tyminski超赞新单。动感电音与温情民谣完美融合，带给你耳目一新的心灵感动，难得的佳作！

  
Avicii 原名Tim Bergling，中文译名：艾维奇。1989年9月8日出生于瑞典的斯德哥尔摩省。自小迷恋音乐的他对电音舞曲情有独钟，受到了Daft
Punk的熏陶，加上瑞典地区发达的电音产业让他萌生了成为DJ的念头。
