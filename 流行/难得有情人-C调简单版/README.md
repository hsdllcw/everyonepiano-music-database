

《 **难得有情人** 》是一首轻快温暖的浪漫情歌，由向雪怀作词，卢东尼作曲，关淑怡演唱，收录于同名专辑《难得有情人》中。

虽然此曲商业味道颇重，但在关淑怡甜蜜的歌声互相晖映下成为了关淑怡的经典回响金曲之一，接着被TVB电视剧《新婚蜜语》采用成为剧集主题曲。

1989年作为乐坛新秀的关淑怡更是凭借这首《 _难得有情人_ 》入围当年“十大劲歌金曲”，一炮而红，这首歌也传唱至今。
