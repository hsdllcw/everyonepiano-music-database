

**Sofia双手简谱** 和五线谱完全对应。

_Sofia_ 是一首有魔力的歌，来自西班牙民谣唱作歌手Alvaro
Soler。欢快的旋律，动感的节奏，让人不由地翩然起舞，心情立刻嗨起来！现亦有英文翻唱版。

Alvaro Soler全名是Álvaro Tauchert Soler，是一位新晋西班牙歌手，流行音乐作曲家。出生于1991年，西班牙巴塞罗那。
