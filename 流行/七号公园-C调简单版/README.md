

《 **七号公园**
》是由许嵩作词作曲并演唱的一首歌。此曲通过主题“七号公园”，描写深沉萧瑟的景物烘托出作者的忧伤以及对前途或者对爱情的迷惘。歌词充满诗意，给人一种唯美的感觉，是一首很值得欣赏的歌曲。

歌词开头以缓慢的旋律，勾画出一种失落而冷清的环境和心情。为后文直接抒情支撑了稳固的舞台。其中通过“街灯”这个意境描述出自己对生活或爱情之间距离，以及对飘渺不定的梦想怀有的一丝遐想。最后用“终于明白，或许有些事情没有必要再等”表现出作者的醒悟，及时改变自己的人生观，从新燃起对新的生活或爱情的希望。
