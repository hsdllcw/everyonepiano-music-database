

_One More Night_ 是来自美国加利福尼亚州洛杉矶的流行摇滚乐队Maroon 5的歌曲。Maroon
5，中文译名魔力红乐队。此乐队融合了红色的狂野魅力，神秘热情，奔放烈爱再掺入摇滚力道，灵魂旋律及放克节奏，这样的红流泻着新灵魂乐性感风情的放克摇滚这样的红，在流行歌坛注入了一股新的颜色。

  
One More Night这首曲子收录在其专辑Overexposed中。于2012年6月底于全球同步发售的专辑，本专辑共有12首歌。

  
歌词下方是 _One More Night钢琴谱_ ，希望大家喜欢。
