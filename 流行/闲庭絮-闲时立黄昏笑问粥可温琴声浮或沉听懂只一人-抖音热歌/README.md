抖音此曲出自国风歌者刘珂矣发布的全新国风唱作单曲《 **闲庭絮** 》。这首歌是在珂矣读了清代文学家沈复《浮生六记》后的有感而发。

对于此书，前人曾有“读之心醉”的评价，曾翻译过英文版《浮生六记》的著名作家、学者林语堂也曾称芸娘是中国文学史上最可爱的女人。

而在珂矣看来，《浮生六记》最动人之处就在于沈复与芸娘的彼此成就与相互滋养。此次的歌曲制作团队质诚精良，吉他编配及演奏邀请到著名歌手李健的御用吉他手向巍老师，箫则由刚刚发布全新EP的青年演奏家“埙公子”吴题录制，老搭档百慕三石参与了作曲并担任制作人，该单曲由慕音乐全网发行，倾情推荐。”闲“是一种生活态度，但它却时常体现在沈复与芸娘略显拮据的生活当中。
