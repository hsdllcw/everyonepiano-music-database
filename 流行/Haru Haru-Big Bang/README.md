

**Haru Haru**
是韩国人气组合BIGBANG继〈谎言〉后最经典的歌曲之一。此歌曲横扫韩国大街小巷，以其动听的旋律、凄美的歌词、伤感的MV剧情赢得了众多人的喜爱，更有被奉为韩国“神曲”之一的美誉。
