是周杰伦演唱并作曲，方文山填词的一首歌曲，收录在周杰伦2016年发行专辑《周杰伦的床边故事》。告白气球是在巴黎取景。在MV里，周杰伦帅气走在塞纳-
马恩省河畔对嘴型、在巴黎铁塔前留下身影，营造了一幅浪漫的景致。
