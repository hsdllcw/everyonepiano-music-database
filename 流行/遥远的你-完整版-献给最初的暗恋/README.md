

**遥远的你**
这首歌是由221小伙伴成员王明明、李亚然作词，王明明作曲，并由六个小伙伴演唱。此曲打破了以往的情歌形式，不是单一的讲故事，更不是单一的惋惜，抒发情感，更像是一个男孩对自己青春的自叙。在歌词中我们能够感受到男孩对女孩坚贞不渝的爱情，男孩对青春的那种珍惜回忆。

“七月的风，八月的雨”，这两句歌词显得都是那么凄凉，可以体会到男孩心里的那种孤独、无助感，从伴奏上用吉他单一的伴奏，吉他不仅代表着民谣，也代表着情歌，虽然没有钢琴那么直入人心，但是吉他更巧妙的表现出情歌的那种氛围。

《 _遥远的你_
》虽然是献给最初的那个初恋，但是更多的是对爱情的追逐，有爱就要说出口，无论多么艰难，天气多么恶劣，只要你需要，我拼死也要守护你。爱情本来就是这样，无关于一切，全凭我爱你，这首歌虽然带给我们很多爱情回忆，但是这首歌也让我们认清了藏在内心的那个人，跌跌撞撞一路走在不容易，彼此珍惜才是对彼此的负责。
