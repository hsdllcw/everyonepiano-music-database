

**白色风车双手简谱** 和五线谱完全对应。

_白色风车_ 由周杰伦作词作曲，并演唱的一首歌曲，收录于周杰伦2006年9月5日推出的专辑《依然范特西》。

歌曲的MV使用了现实与回忆相交错的构思，一方面是美好的回忆，另一方面是独看晚霞的寂寞现实。而同样的海滩、浪花，以及埋下的戒指，是作为现实与回忆的线索和桥梁，也就是用这些勾起回忆。
