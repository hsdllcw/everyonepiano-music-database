

**陪你左右双手简谱** 和五线谱完全对应。

_陪你左右_ 是由祖峰演唱的一首歌曲，由刘婕作词，捞仔作曲， _陪你左右_ 同时是电视剧《欢乐颂》插曲，收录在《欢乐颂电视原声带》中。  
作为一部都市女性剧，《欢乐颂》并未包裹在女性成长爱情的套路之下，它从女性视角出发，在格局上进行一定创新。从播出的剧情来看，故事虽侧重于辅线的铺设，但剧情却不落俗套，加之不傻白甜不玛丽苏的人设，在开播首日便俘获了观众的好口碑。

歌词下方是陪你左右钢琴谱，希望大家喜欢。
