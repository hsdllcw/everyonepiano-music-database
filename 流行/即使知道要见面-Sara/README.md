

**即使知道要见面双手简谱** 和五线谱完全对应。

_即使知道要见面_ 出自泰国歌手Sara 2001年发行的专辑《Girl Friend》，乐曲轻松舒缓且赋予轻柔缠绵的少女情感。

乐曲前部带有淡淡的忧伤。总体来说是一首非常不错的泰国流行歌曲，虽然不够朗朗上口，但是对于网站编辑来说，《即使知道要见面》是一首非常合适的网络背景音乐。
