

《 **因为是你** 》由少女时代成员Tiffany 演唱的《爱情雨》OST
歌曲，并在剧中放送，伴随着允熙与仁河拥抱分别的场景，夹杂一丝感伤，也同时迎来了40 年后主角初遇的场景，引发观众的关注。

《 _因为是你_
》以钢琴伴奏的旋律为主，Tiffany凄婉低缓的嗓音展现了爱情结束后的惋惜，为歌曲增添不少风味，为剧中张根硕和允儿的爱情添上一抹悲情，让观众的心也随之哀伤。

_因为是你钢琴曲_ 也别有一番风味，很好听，希望大家会喜欢！
