抖音上这首歌曲很火。据悉这段歌词出自歌曲《 **还是会想你** 》由h3R3与林达浪共同演唱。

此曲最初由林达浪原创，在抖音上有发布一个歌曲片段，在线征集歌名。后正式命名为《还是会想你》。歌曲不仅受到了很多网友的喜爱，还被翻唱了很多版本。

_还是会想你双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！

同时，网站还为大家提供了《[ **他只是经过**](Music-12930-他只是经过-你的世界并没有停留-抖音歌曲.html "他只是经过")》曲谱下载
