

2012年4月24日纯爱影片《第一次》公布的第一首主题曲《 **都要微笑好吗**
》，由乐坛天王周杰伦作曲、方文山作词，由饰演女主角宋诗乔的Angelababy演唱。

  

青春纯爱影片《第一次》于2012年4月24日在京举行情歌告白会，导演韩延携女主角Angelababy、Cindy袁咏琳一同出席为影片造势，同时宣布电影将有双主题曲：由Angelababy演唱、周杰伦和方文山这对乐坛黄金搭档倾力打造的《
_都要微笑好吗_ 》，和周杰伦小师妹袁咏琳为赵又廷创作的另一首。

  

《都要微笑好吗》的演唱者Angelababy（1989年2月28日－），中文名杨颖，出生于上海，13岁时移居香港，有四分之一的德国血统，现职为电影演员，歌手及模特。在香港有「第一代嫩模」、「宅男女神」、「潮流教主」的称号，当她为Tokyo
Fashion
Week揭幕时，更是被日本传媒获封为「香港女神」。2009年开始进军影坛接拍多部大片，2010年正式杀入日本市场，签约日本艾回唱片，并发行日文音乐EP，事业已全面开花。

  

歌词的下方还提供了 _都要微笑好吗钢琴谱_ ，欢迎感兴趣的朋友下载使用。
