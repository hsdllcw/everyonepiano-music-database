

**田纳西华尔兹** ，又译《 _田纳西圆舞曲_
》，是创作于二十世纪四十年代的一首美国流行歌曲，乡村音乐的代表作，1965年被选为田纳西州的第四首州歌。作词Redd Steward，作曲Pee Wee
King，原唱Patti Page。
