《 _依然爱你_
》是一首温暖甜蜜的王式情歌，歌曲由王力宏创作词曲。力宏更是感性告白：要表达给我的fans、family、friends！我依然爱他们、我依然爱这份工作、我依然爱音乐！而导演设计的“翻字钟”的概念，来表达爱情在时间中的改变与不变。
