

《 **第一次爱的人** 》由王心凌演唱，改编自挪威组合M2M的《The Day You Went
Away》。歌曲透露着初恋的青涩、无奈和伤感之情，传唱度极高。

人非草木，孰能无情。而初恋是刻骨铭心的，是单纯的爱，纯洁的爱。在该曲中，听者甚至都感觉不出这首歌的心碎，忘记了每个人的世界都有眼泪和悲伤，忽视了安静的那一面。歌词“第一次爱的人，他的坏他的好。却像胸口刺青，是永远的记号”反映出了第一次爱的人成了生命中永恒的符号，智者是痛苦的，经历太多次爱情的人似乎是清楚的、也是麻木的，就是因为年少的单纯与无知才成就了刻骨铭心的爱。白纸上的画永远胜利黑纸上的画，第一次爱的人永远占据心灵庞大的空间。
