

**舍不得双手简谱** 和五线谱完全对应。

_舍不得_
是电视剧《漂亮的李慧珍》的插曲，由陈曦作词，董冬冬作曲，迪丽热巴演唱。迪丽热巴温情走心的嗓音诠释出整首歌词的意境，在钢琴独奏的旋律中，浅唱不一样的爱情独白。是的没错，一切都会过去的；放下曾经的无奈，拥抱美好的未来。

歌词下方是 _舍不得钢琴谱_ ，大家可以免费下载学习。
