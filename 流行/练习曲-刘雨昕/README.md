

**练习曲**
是刘雨昕演唱的歌曲，由八三夭阿璞（李贤璞）和刘雨昕共同作词，李贤璞作曲，罗龙杰RevoJay编曲并制作。收录于刘雨昕2021年4月20日发行的专辑EPSILON中。

《 _练习曲_
》也是刘雨昕送给自己成长之路的。我们都是追梦路上长途跋涉的旅人，将沿途的景色写成诗篇，把行囊里珍藏的记忆串成音符，跨过山海，穿过森林，迎来花信。仍踏实，仍刻苦，仍有爱，初心不忘。更勇敢，更热情，更强大，能量不息。

_练习曲双手简谱_ 是网友提供的，由网友 **rubyloobee** 制作，感谢rubyloobee对EOP的支持！
