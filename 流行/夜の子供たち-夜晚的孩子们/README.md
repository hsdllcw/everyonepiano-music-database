

《 **夜の子供たち** 》，中文译为《夜晚的孩子们》，是日本女性同人歌手叶月由罗（葉月ゆら）演唱的一首歌曲，收录在其专辑《[Roses
Epicurean]闇童話工房》中。

叶月由罗，早期叶月被称作“治愈系萌音歌手”，演唱过一些VOCALOID及东方曲目，也为某些PC游戏（典型的如光与影的十字架）的OP演唱。经历了岁月的沉淀之后，目前叶月大部分专辑以黑童话为主题，一大批风格独特的作曲人为其作曲，曲风更为多样，古典新唱、摇滚、爵士、金属、电子等风格均有涉及。另外叶月的歌曲中很多歌词都是她自己写的。叶月有着独特的暗黑萝莉高音，加上优秀的作编曲和文采斐然的作词，使得她的每一张专辑都有着强烈的吸引力。
