

最近小编在刷抖音的时候听到一首歌曲特别的好听，其中歌词是：“我们只配惋惜，从即刻起说对不起谎话没恶意”。据悉，这首歌的歌名叫做《 **即刻**
》，由周仁作词，郑聪慧编曲， joysaaaa演唱，发行于2021年4月1日。

目前此曲被很多网络歌手翻唱，一首平淡的但也是非常悲伤的歌曲。虽然没有很大的起伏，但是会让人感觉非常的受触动。大概这就是大家喜爱此曲的原因吧。

歌词下方是 _即刻钢琴谱_ ，大家可以免费下载学习。
