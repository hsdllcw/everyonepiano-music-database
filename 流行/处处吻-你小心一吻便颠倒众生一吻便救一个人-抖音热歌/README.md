此曲叫做《 **处处吻** 》，出自杨千嬅2004年6月25日发行的专辑《电光幻影》。

虽然这首歌曲并不是杨千嬅专辑中最出名的单曲，可是在当今这个时代，却因为歌词性感，曲调节奏感强烈而顺利出圈，甚至因为容易卡点成为各圈层改编混剪的“御用”神曲。

还记得2019年末，《 _处处吻_
》突然爆红，成为各个短视频平台的常驻BGM，一度占据网易云热搜榜第一的位置，在不到一个月时间，就成为家喻户晓的歌曲，强制扒开网友们的耳朵。
