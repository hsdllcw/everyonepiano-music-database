

《夜夜夜夜》是齐秦原唱的一首歌曲，熊天平词曲。有很多人翻唱过此歌。

### 基本信息

歌曲：夜夜夜夜 所属专辑封面  
作词：熊天平（出道前给齐秦写歌时叫熊天逸）  
作曲：熊天平  
制作：熊天平  
原唱：齐秦  
所属专辑：《丝路》  
发行时间： 1996-12-04
