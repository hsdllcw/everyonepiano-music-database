

延续《泪海》的好口碑，季忠平再次为许茹芸量身创作《如果云知道》，并成功写下佳绩；整张专辑由刘天健、Peter
Wong、袁惟仁、徐德昌、黄国伦为制作班底空灵的气音唱腔成为许茹芸独树一格的招牌，这是其它歌手难以取代的声音识别，却不是许茹芸诠释歌曲的唯一方式；轻快的《执着》、民谣的《半首歌》、神经质的《你讨厌》…淋漓尽致的声音表现，带着这张专辑入围当届最佳女歌手、最佳国语流行专辑等金曲奖项。齐秦和快乐女声郁可唯也翻唱过。
