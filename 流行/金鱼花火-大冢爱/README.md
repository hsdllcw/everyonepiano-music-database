

《 _金鱼花火_ 》是大冢爱的一首歌曲，于2004年8月18日发行为中国地区大冢爱毫无争议最受欢迎的歌曲。  
  
《 _金鱼花火_ 》是大冢爱的第5张单曲。2004年8月18日由艾回发行其之。初回版五万张CD附本人创作之限定特典绘本。DVD版本也同时发售。  
  
此曲以夏夜风情为主调。同时，大冢爱表示金鱼花火是指夏日祭典捞金鱼所用的袋子装着烟花的意思，但其实是以金鱼花火实物存在，但与大冢爱所想表达的不同，金鱼花火是指能在水面上移动的烟花。与「星象仪」相同，是大冢爱著名的抒情歌曲之一。  
  
歌词的下方还提供了 **金鱼花火钢琴谱** ，欢迎感兴趣的朋友下载使用。
