

**第三人称**
是台湾独立乐团Hush!的歌曲，Hush!看似云淡风轻的凄美演绎，却生生触动了大家的内心。另外，戚薇在电视剧《北京女子图鉴》中以独特的魅力烟嗓翻唱了此曲。

《 _第三人称_
》的意境就跟歌名一样，歌曲描写了在人生或爱情的困境里，有时会跳开，采用第三人称角度，与自己保持距离，随着时间游走在自我逃避与自我和解的拉锯中。歌曲通过这个神秘角色的第三人称视角，让人体会走过爱情或人生困境后的希望与可能性。
