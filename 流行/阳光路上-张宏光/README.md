

**阳光路上双手简谱** 和五线谱完全对应。

_阳光路上_ 由甲丁作词，张宏光作曲的一首歌曲，雷佳演唱的歌曲。

张宏光，中国著名作曲家，毕业于中央音乐学院作曲系，现任中国歌剧舞剧院的专职作曲。主要从事影视音乐、流行音乐及民族音乐的创作。多年来一直被音乐界尊称为“幕后音乐高手”。
他说：“人不能躺在自己成功的路上，那样是走不远的。我自己的作品自己很少听，即使这个作品已经在中国有很大的反响的时候，我也基本不去听。我怕自己会觉得有了成就感，对未来失去了追求。”尽管他已经以众多的优秀作品证明了自己的价值，在圈里享有的名誉也越来越大，但是他依旧将自己生活的大部分时间都用在创作上，并且提携和培养了一批年轻的音乐创作者。
