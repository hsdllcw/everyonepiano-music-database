

瓦蓝蓝的天上飞雄鹰  
我在高岗晀望北京  
侧耳倾听母亲的声音  
放眼欲穿崇山峻岭

大气磅礴的歌词配上既悠扬又铿锵有力的曲调，民族音乐与流行元素结合在一起，再由草原歌手乌兰图雅深情演唱，《 **站在草原望北京**
》这首歌曲成为富有少数民族风情的中国梦主题歌曲。

这首歌曲以草原儿女真实的内心体验为情感抒发点，以充满诗情、宽广舒展又富于激情的旋律描绘了内蒙古大草原的自然风貌；用朴实而不失热烈的歌声传达了草原儿女热爱家乡、心系祖国的爱国情怀。
