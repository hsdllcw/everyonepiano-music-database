

由陈小霞作曲，林夕填词的这首《 **十年**
》，经由陈奕迅的独特演绎，受到无数歌迷的喜爱，它代表着心绪的零乱和无止境的苦涩，有着很高的传唱度，也是KTV点唱次数最多的歌曲之一，是香港伤感情歌中的经典曲目之一。

《 _十年_
》和陈奕迅就像是双生儿，只要是陈奕迅演唱会几乎必然会上演《十年》，只要《十年》的旋律缓缓奏起，听众们心底的感情防线便会在顷刻间瓦解，无数的情感故事开始随着音乐轻轻流淌，也让听众无所遁形。
