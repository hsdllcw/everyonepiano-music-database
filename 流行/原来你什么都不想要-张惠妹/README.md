

**原来你什么都不想要双手简谱** 和五线谱完全对应。

_原来你什么都不想要_ 是张惠妹演唱的歌曲，1995年9月以单曲形式发行，此后收录在张惠妹1996年发行的个人专辑《姐妹》。歌曲由郭子作曲、邬裕康填词。

张惠妹（A Mei），1972年8月9日出生于中国台湾台东县卑南乡泰安部落，中国台湾原住民歌手。

歌词下方是 _原来你什么都不想要钢琴谱_ ，喜欢的朋友可以免费下载学习。
