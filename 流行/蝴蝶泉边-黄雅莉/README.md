

**蝴蝶泉边** 是中国女歌手黄雅莉演唱的歌曲，由彭青谱曲、作词，收录于黄雅莉在2006年发行的专辑《崽崽》中。

《 _蝴蝶泉边_
》的MV由中国台湾导演金卓亲自执导，并且请来雅莉比赛时的评委老师夏青参与MV的拍摄，而黄雅莉3岁的妹妹黄雅卓也首次入镜为MV助阵。金卓表示，初见黄雅莉，第一次听到《蝴蝶泉边》后，他的脑海呈现出这样的画面：一个小女孩在满是烟雾的胡同里跟着一只洒着金粉飞舞的蝴蝶走着，在一间有教室的地方蝴蝶从窗户飞了进去，小女孩慢慢的走近那扇窗，垫起脚尖只露出两个眼睛，手扳着窗框往内看，女老师正在弹着钢琴，清纯干净的雅莉唱了起来。

歌词下方是 _蝴蝶泉边钢琴谱_ ，希望大家喜欢。
