

**独角戏**
是台湾女歌手许茹芸的代表歌曲，收录在其专辑如果云知道中。其“芸式唱腔”为台湾乐坛的独特风格。在1990年代末期与李玟、张惠妹并称“台湾歌坛三大天后”。

  
歌词下方是 _独角戏钢琴谱_ ，希望大家喜欢。
