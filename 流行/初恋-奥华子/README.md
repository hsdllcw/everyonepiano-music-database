《 _初恋_ 》是日本女歌手奥华子演唱的歌曲。

  

奥华子（おく はなこ），日本歌手。作为地下音乐人的她，用键盘自弹自唱在涩谷街头表演的奥华子有着1年内自主制作CD销售2万张的成绩。之后被PONY
CANYON相中，于2005年7月以单曲「やさしい花」进入主流音乐界，此曲因优美的旋律搭配东京电力「TEPCOひかり」广告歌曲而广为传播。

  

《初恋》是一首很美的歌曲。歌词下方是 _初恋钢琴谱_ ，欢迎大家使用。
