_Mylo Xyloto_ 是英国流行摇滚乐队Coldplay于2011年10月24日发行的第五张录音室专辑Mylo
Xyloto中的一首歌曲。专辑，中文意译为《彩绘人生》，由知名制作人Markus Dravs, Daniel Green, Rik Simpson,
Brian Eno等连手打造。专辑主题是一个关于爱的并且有着唯美意境和完美结局的故事。

Coldplay是英国摇滚乐坛进入新世纪之后崛起的最受欢迎的摇滚乐队之一，他们秉承了英式摇滚乐队一贯的Britpop/Alternative
Rock英伦风格，成为了英国新一代乐队中的杰出代表。
