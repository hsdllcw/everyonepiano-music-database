_不必在乎我是谁_
是张浩泽在梦想的声音第二季演唱的歌曲。此曲原唱为林忆莲，由李宗盛作词、作曲，收录在同名专辑《不必在乎我是谁》中，于1993年5月19日发行。《不必在乎我是谁》的粤语版本是《假如让你吻下去》。在湖南卫视于2017年1月21日播出的音乐节目《歌手》中，林忆莲再度演唱该曲。

歌词下方是 _不必在乎我是谁钢琴谱_ ，大家可以免费下载学习。
