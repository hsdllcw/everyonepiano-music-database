Bullet（韩语：체리블렛；日语：チェリー・ブレット），最初由10名成员组成，现有7名成员，包括五名韩国成员海允、裕姝、紫萝、智媛、彩麟，两名日本成员莉美、MAY。2019年1月21日以首张单曲专辑《Let's
Play Cherry Bullet》正式出道。
