

《 **老鼠爱大米** 》，是一首网络歌曲。于2004年走红网络，红极一时。由 杨臣刚 作曲， 王启文 主唱，后被 香香
翻唱，在网络上迅速走红的一首歌曲。这首歌旋律简单，歌词琅琅上口，尤其是以「 老鼠 喜爱 大米
」(老鼠与大米是吃与被吃的利益关系，并不适合用来比喻爱情）来比喻对于情人的喜爱，生动活泼的表现出网络族群的想象力，是其迅速走红的原因。
