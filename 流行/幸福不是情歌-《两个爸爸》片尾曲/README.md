《 _幸福不是情歌_
》是台湾偶像剧《两个爸爸》中的片尾曲，由《我们没有在一起》词曲作者陈韦伶和黄婷再次量身订做、刘若英演唱。其所属专辑为刘若英2013年专辑《亲爱的路人》。  
  
暨2010年在相信音乐发行专辑之后，奶茶刘若英找到终身伴侣「在一起」，进入人生新阶段。尘埃落定的沉淀心情，使她对于爱情有更多体悟，忍不住又想借由自己最喜欢的唱歌，跟歌迷分享。专辑制作一启动，刘若英又开始了她的认真，总是认真到把单纯的动机，越搞越大……以为只是唱情歌抒发心情，没想到跟制作人聊着聊着，越玩越开心，到后来民谣、摇滚、电音等等各种歌型都来了。  
  
这张暌违三年的全新专辑《亲爱的路人》，制作期长达一年多，过程里陆续有奶茶的好友创作人、制作人们进来参与，在笑闹与深刻地讨论间，完成了一张丰富的刘若英音乐。双子座的奶茶，跨入人生新阶段，也有双重体悟。专辑做好后，大家在主打歌讨论中僵持不下，已婚与未婚者各有立场，互不相让，都觉得自己比较幸福，最后决定以“首播双主打”形式，一次让大家听到两首主打，交由歌迷去“认领”心情，于4/15～4/17在HitFM同步进行《亲爱的路人》与《幸福不是情歌》两首主打歌的全球大首播。专辑同名主打《亲爱的路人》是《那些年》作曲者木村充利的最新作品，词神林夕献出号称他近年来最深刻的歌词，一首歌道尽千帆过尽的爱情。《幸福不是情歌》由《我们没有在一起》词曲作者陈韦伶和黄婷再次订做原汁原味奶式情歌，歌一做好立刻被三立华剧《两个爸爸》选为片尾曲，90秒曝光后询问度居高不下。  
  
《 _幸福不是情歌_ 》是一首会让人忍不住哭的歌。但歌里却有一股温暖的力量，让人怀有幸福到来的期望。
