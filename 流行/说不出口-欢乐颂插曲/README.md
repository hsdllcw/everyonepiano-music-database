

**说不出口双手简谱** 和五线谱完全对应。

_说不出口_ 是著名演员刘涛演唱的一首歌曲，由韩冰填词，张江谱曲，是热播剧《欢乐颂》的插曲。这首 _说不出口_
，不仅切合刘涛在剧中扮演角色的人设，更邀来韩红友情献唱和声。自该剧开播仪式刘涛现场演唱后，这首歌在歌迷和观众中引发超高讨论度。

歌词下方是说不出口钢琴谱，希望大家喜欢。
