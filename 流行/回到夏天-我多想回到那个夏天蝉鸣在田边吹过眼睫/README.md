那么这首歌是什么歌呢？下面就一起跟着EOP小编来看看吧。

这首歌的歌名叫做《 **回到夏天** 》，是由罗洋（卡其漠）编曲，爱写歌的小田、傲七爷演唱，发行于2020年5月21日。

夏天又是一个毕业季。一岁岁的年华流逝，曾经捧在手中不肯放下的童话书，早已不知踪影。不想长大，时光又匆匆，于是怀念过去，过去却只存在回忆中，来不了，回不去。曾经隔壁班的穿白色衬衣的少年，都化作美好的回忆，只能永留在心房！

不行了，小编又要泪目了。。。

《 **回到夏天** 》曲谱由 **[EOP造神计划](/Article-210-EOP造神计划——免费学琴从今天开始.html)** 大佬[
**M小冯哒**](https://www.everyonepiano.cn/user-174829.html)给大家编配，虽然有一些难度，但小编相信学过《[
**魔鬼训练营**](/Sale.html)》的小伙伴一定可以掌握。同时大家也可以留言多久学会了这首曲子。
