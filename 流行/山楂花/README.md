

由张艺谋导演执导的《山楂树之恋》的宣传主题曲《山楂花》，陈楚生用他干净的声线、自身的纯良气质和对影片故事及歌曲情感的自然把握，将歌曲表现的清新纯粹，与影片纯净的风格非常符合。
