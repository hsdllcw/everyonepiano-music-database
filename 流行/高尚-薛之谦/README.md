

**高尚双手简谱** 和五线谱完全对应。

作为薛之谦2017年的开山之作，最新单曲《 _高尚_ 》延续了特有的薛氏风格，既闪烁着沉静思考的光芒，同时又有新鲜元素融入。

据“薛之谦2017全国巡回演唱会”主办方太合音乐集团透露，4月22日，薛之谦将在巡演首场大连站盛大演绎《高尚》，打响“头炮”。接下来的巡演期间，薛之谦还将陆续推出同系列的另外三首新歌，全部献给所有粉丝免费下载、聆听，持续为粉丝们制造惊喜。

歌词下方是 _高尚钢琴谱_ ，喜欢的朋友可以免费下载学习。
