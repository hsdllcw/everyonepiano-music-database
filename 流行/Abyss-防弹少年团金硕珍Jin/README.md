

BTS防弹少年团金硕珍Jin生日发新歌《 **Abyss** 》。

《 _Abyss_ 》意为“深渊”，从歌词我们就可以感受到JIN的心境变化：“最近陷入了Burnout状态”。

Jin表示：“虽然是有点忧郁的歌曲，似乎不太适合生日，但如果不是生日这一天或许更不合适，所以公开了《Abyss》。”

“屏住呼吸潜入我的海洋，看到美丽却在悲伤哭泣的自己，那黑暗中的我。”伤感的旋律真的让人感受到Jin的情绪。
