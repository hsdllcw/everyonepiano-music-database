《 **美人鱼**
》是由林秋离作词，林俊杰作曲，林俊杰演唱的一首歌曲，收录在专辑《江南》中。这首歌是一首流行风格的歌曲。歌曲展现了林俊杰的高超唱功和创作才能。

林俊杰（JJ
Lin），新加坡人，流行歌手、词曲创作者、音乐制作人。2003年以专辑《乐行者》出道。2004年凭《江南》成名。2007年成立音乐制作公司JFJ
Productions，负责单曲及专辑制作。

歌词下方是 _美人鱼钢琴谱_ ，欢迎大家使用。
