

《 **这一生关于你的风景** 》原唱是枯木逢春。给人的感觉是躺在摇椅上晃晃悠悠的向心爱的人讲述这一生。

而抖音上走红的是隔壁老樊翻唱的版本。虽然只有短短的十秒，也仅仅只唱了一句歌词“我多想，能多陪你一场把前半生的风景对你讲”，但是却赢得了许多人的喜欢，纷纷去寻找这首歌的完整版。这个版本像是经历了世间沧桑之后，依然自己心中的人说一句：陪你一起成长。

这首歌我最喜欢的一句是：在你的生命中留下阳光。这一生在你的风景里，也恰恰因为那一段你投入的时光，留在心底的“风景”才如此美好。
