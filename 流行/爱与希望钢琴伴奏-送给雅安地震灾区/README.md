_爱与希望_
是JJ林俊杰创作的5.12赈灾歌曲，北京时间2013年4月20日8时02分四川省雅安市芦山县（北纬30.3,东经103.0）发生7.0级地震。震源深度13公里。震中距成都约100公里。EOP特将爱与希望钢琴伴奏谱以及EOP奉上，以此鼓励灾区人民：
_爱与希望_ 永在。用我们的方式为雅安祈福。

如果你也想以自己的方式来为雅安祈福，那就下载 _爱与希望双手简谱_ 或五线谱，为灾区人民弹唱一曲吧。
