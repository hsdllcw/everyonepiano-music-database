

《 **下山**
》这首歌由朱斌语作词作曲，是要不要买菜演唱的一首歌曲，于11月22日发布。要不要买菜绝对是一个宝藏男孩。他的歌都是特别特别温柔的，用温柔细腻的声线大方又肆意的勾勒出了一个恣意快活的江湖。

歌曲一经发布，就登顶各大视频、音乐网站，成为各大视频音乐App的宠儿。而它本身是一首偏古风歌曲，但是曲调又非常活泼，轻快的旋律让人一听就感到很舒服。被很多视频引用作为背景音乐，让人过耳不忘。脑海中不禁浮现出一幅翩翩少年努力练功，想要下山除暴安良的画面。
