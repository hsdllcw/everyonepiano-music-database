

**万圣节之夜双手简谱** 和五线谱完全对应。

_万圣节之夜_ 是AKB48第41枚单曲唱片，单曲为迪斯科乐，曲风欢快。

作为AKB48的第41枚单曲，她们首次采用黑胶唱片限量于9月16日发售。包含有初回限定盘、通常盘共计8种风格的唱片。A到D盘的CD根据不同的风格配有不同的搭配曲目，DVD里所收录的内容也有所不同。

歌词下方是 _万圣节之夜钢琴谱_ ，希望大家喜欢。
