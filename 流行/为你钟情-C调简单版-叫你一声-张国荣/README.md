

《 **为你钟情**
》由张国荣演唱，收录在其1985年5月14日发行的《为你钟情》同名专辑中。其国语版本《叫你一声》亦由张国荣演唱，收录于1986年11月26日发行的《英雄本色当年情》专辑。

《 _为你钟情_
》与上一张专辑中的《侬本多情》有异曲同工之妙，不过这一次，奥金宝的编曲更多的加入了古典的弦乐原素，也让这首歌曲更多了一种古典美，而张国荣有些拖拽的演唱无疑是对这首音乐最好的诠释，那一句“I
Do！I Do！”更像是一种情之所至的即兴，听闻时总让人呼吸停止。《为你钟情》等歌表现了张国荣对伤感深情慢歌的演绎功力。
