

《暗号》描述了两个恋人之间的无线电通信。歌词使用了无线电的术语，如“杂讯”“干扰”“信号弱”等来强化这一主题。《暗号》的歌词写作有唱片公司内部人员参与，在周杰伦的创作史上是第一次。使用的主要乐器是吉他。没有说唱部分。
