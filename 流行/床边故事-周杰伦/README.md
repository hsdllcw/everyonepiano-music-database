

《 **床边故事**
》是周杰伦演唱的一首歌曲，收录在其2016年6月24日发行的专辑《周杰伦的床边故事》中。此曲不同于一般印象中的温馨甜美，而是充满了暗黑奇想、幻术魔咒、神秘幽暗的种种。
