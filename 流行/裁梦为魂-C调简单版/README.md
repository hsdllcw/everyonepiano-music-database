

《 **裁梦为魂** 》收录于银临第二张原创专辑《蚍蜉渡海》。由锦衣小盆友填词，灰原穷编曲。这是一个关于梦的故事，一首写梦之歌！

歌曲以魂画心，或许这就是这首歌讲的那个故事。看着歌曲的MV，讲的是董小妹和薛梦梅的故事，在古代，女子相恋更是不容，二人再不舍，最后终究输给了世俗，最后到底是薛梦梅为爱赴死还是嫁与他人没有点明，或许这才是现实。
