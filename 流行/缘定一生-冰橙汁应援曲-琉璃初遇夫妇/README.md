

**缘定一生** 是一首专门为冰橙汁制作的歌曲，即成毅与袁冰妍主演的东方玄幻仙侠剧《琉璃》的CP粉。

《琉璃》该剧根据十四郎小说《琉璃美人煞》改编，讲述了天生“六识”残缺的少女褚璇玑和离泽宫弟子禹司凤在面临爱情与前世阴谋的双重压力下，携手共对百般腥风血雨，缔造了一段极具仙侠传奇色彩的故事。
