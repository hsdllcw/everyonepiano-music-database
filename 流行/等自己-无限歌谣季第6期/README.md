

**等自己双手简谱** 和五线谱完全对应。

_等自己_ 是无限歌谣季第6期今生第一次演唱会上毛不易与张绍刚带来的打动人心的原创音乐作品。

在演唱会上，毛不易和张绍刚这对“父子档”用这首《等自己》感动全场，让人潸然泪下。前奏是毛不易的招牌“口哨杀”，中间加入了黑管悠扬唯美的旋律，还将《大头儿子小头爸爸》的主题曲融入歌中首尾呼应。这些丰富而有创意的编曲元素，让“老不容易组合”的这场“父子对话”充满了画面感，父子之间最想表达给对方的话，通过这首歌深深的流淌进心间。观众听后也感同身受：“张老师用暖心的辞藻，毛毛用淡淡的音符叙述着父子之间的浓重情义，让我想起了自己的父亲，真的很感动。”

令观众意想不到的是，两人在谈到《等自己》这首歌的创作初衷时，一向沉稳睿智的“场控”张绍刚竟然忍不住泪洒当场。原来，这首歌的歌词灵感来源于绍刚老师对父亲的想念，在聊到自己和父亲的故事时说道：“我最大的遗憾，是到了我四十岁的时候，自己没能好好了解父亲。”张老师一袭发自肺腑的话，令现场观众也为之动容：“希望我们都能静下心来，试着走近我们的父母，珍惜与他们相处的每一天。”
