

**解脱双手简谱** 和五线谱完全对应。

_  
解脱_ 收录于张惠妹1996年的首张个人专辑《姐妹》中，由许华强作曲、姚若龙填词，是阿妹经典代表作之一，后被众多歌手翻唱。

  
歌词下方是 _解脱钢琴谱_ ，希望大家喜欢。
