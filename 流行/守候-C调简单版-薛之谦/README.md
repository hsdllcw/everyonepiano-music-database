

歌曲《 **守候** 》，由薛之谦演唱，赵英俊作词，赵英俊谱曲，于2020年8月21日发行。薛之谦独特的嗓音翻唱这首《 _守候_
》，演绎出了人生长河里有人来了、有人走了，来去都是抉择与故事。

薛之谦浑厚又充满故事的嗓音展现的淋漓尽致，吉他声缓慢地响起，颤抖的琴弦声就像悸动的心脏，每一次拨动琴弦都与心脏同频，使人陷入回忆的漩涡。心中的不快与不甘尽释放，然后回归平静，归于寂寥，孤独是一个人的事，守候也是。
