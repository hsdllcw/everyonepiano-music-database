_是我在做多情种_
是曾经演唱过爆红歌曲《香水有毒》的大陆著名歌手胡杨林演唱的一首歌曲，收录在其2013年发行的专辑《爱神来了》之中，亦是由钟汉良、李小冉主演的热播电视剧《来不及说爱你》的插曲。  
  
悲情天后匪我思存新剧《来不及说我爱你》中，胡杨林大展身手，主题曲
《是我在做多情种》又一次展现了她天籁般的歌声。还有插曲《左眼微笑右眼泪》，《一生只能爱一个》也观众大受喜爱。
