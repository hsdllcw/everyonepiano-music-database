《 _爱在记忆中找你_ 》是林峯演唱的一首歌曲，由张美贤填词，邓智伟谱曲，Danny
Leung编曲，该歌曲收录于林峯2007年11月23日发行首张专辑《爱在记忆中找你》中，并为2007年电视剧《岁月风云》插曲。2008年1月12日，林峯凭借该歌曲获得2007年度十大劲歌金曲颁奖典礼最受欢迎男新人金奖。  
  
《爱在记忆中找你》是林峯的首张粤语唱片，于2007年11月23日由英皇娱乐集团有限公司发行，全碟共收录十首歌曲，同名单曲《爱在记忆中找你》和《赤地转机》为该碟的主打歌。  
  
该专辑发行三周即获IFPI金唱片销量，林峯获封“销量新人王”，夺得IFPI香港唱片销量大奖“十大销量广东唱片”和“最畅销本地男新人奖”，林峯凭借该专辑夺得2007年度十大劲歌金曲“最受欢迎男新人”金奖。  
  
林峰，当红歌手、演员，香港乐坛最具号召力歌手之一，实力与偶像兼具的全能艺人。自幼学唱歌，1999年出道，主演《寻秦记》《大唐双龙传》等热门剧集，2008年凭《太极》成为首位荣膺「亚洲视帝」的香港明星。2007年进入乐坛，2009-2013年4年4度在香港红馆举行个人演唱会打破新世纪红馆连开个唱记录，并在美加、澳洲等地举行世界巡回个唱，在海外和两岸三地有极高人气，个人售票演唱会24场。连续2年荣获IFPI香港唱片销量大奖「全年最高销量男歌手」，香港十大劲歌金曲「最受欢迎男歌星」。2010年进军影坛，接拍《白蛇传说》等上亿投资大片。2012年成立个人设计工作室「L-For」。2013年动态：扛鼎过亿投资3D武侠巨制《陆小凤与花满楼》出演陆小凤，9月27-28日A
Time 4 You林峯巡回演唱会广州站。  
  
TVB首席小生林峰刚在万千星辉颁奖典礼2007中凭重头制作的回归剧集《岁月风云》而勇夺“内地观众最喜爱TVB男艺人”，演技有目共睹，而他为剧集深情演绎插曲“
_爱在记忆中找你_ ”更是经典，此曲乃他加盟英皇的第一首歌，随即登上TVB劲歌金曲每周冠军歌。
