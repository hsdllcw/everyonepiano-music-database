

**Crazy** 是美国乡村流行音乐歌手Patsy Cline的歌曲。Patsy
Cline被誉为上世纪50-60年代第一天后的佩茜•克莱恩原名Virginia Patterson
Hensley；1932年9月8日出生于弗吉尼亚州的Winchester。

  
歌词下方是 _Crazy钢琴谱_ ，希望大家喜欢。
