

**春庭雪** 原是原创歌手橙翼于2016年7月29日发布的一首古风歌曲。后于2020年国风人气女歌手等什么君重新演绎了此曲。

前尘如梦，却有着如鱼饮水，冷暖自知的隐忍。殷殷的盼，痴痴地等，也不见良人归来。这一生虽还漫长，却已觉如同沉水香，不知不觉化为灰烬。

锦绣华年，独自空守在庭院深处，孤芳自赏，冷冷清清。无人拭我相思泪，无人与我赏星辰。这一世，终究是风雪多晴天少，梨花满地空寂寥。
