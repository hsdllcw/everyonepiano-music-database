

**吉祥三宝**
是中国国际广播电台蒙语节目主持人布仁巴雅尔创作的一首蒙古语歌曲。收录在《天边》专辑中。“吉祥三宝”是一个蒙古族图腾：一匹马上驮着的三件宝物，代表吉祥如意。据布仁巴雅尔所述，这首歌是他在1994年为庆祝女儿诺尔曼3岁生日所创作。在1998年出版的布仁巴雅尔专辑中收录此曲。2006年中国中央电视台春节联欢晚会中，布仁巴雅尔和妻子乌日娜、侄女英格玛共同演唱了这首歌，从而引起广泛关注。

  
由于歌词的内容非常简单纯洁，而女孩子的声音亦可爱嘹亮，所以歌曲迅即在网上大受欢迎，并流传至中国大陆以外的地区。随后，不少以《吉祥三宝》为题材的相关产品更适时推出。最热闹之时（2005年农历新年），在香港的地铁车厢亦随时可听得到《
_吉祥三宝_ 》的手机铃声。

  
歌词下方是 _吉祥三宝钢琴谱_ ，希望大家喜欢。
