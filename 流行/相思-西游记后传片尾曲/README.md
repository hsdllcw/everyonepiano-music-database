_相思_
是2000年版古装神话电视剧《西游记后传》的片尾曲。且此曲由毛阿敏演唱。《西游记后传》是陕西省电视节目交流中心2000年出品的一部神话剧，改编自吴承恩所著小说《西游记》，由李源执导，钱雁秋改编，曹荣、黄海冰、吴健、黑子、马雅舒等人主演。

毛阿敏是1963年3月1日出生于上海的天后级女歌唱家，是首位在国际流行歌曲大赛中获奖的中国流行歌手。她以深情、婉转、大气的唱法，高贵、端庄的形象成为中国流行音乐的代表人物，她开创了中国流行歌曲的大歌时代，被公认为内地歌坛大姐大。
