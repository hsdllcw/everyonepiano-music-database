

《 **笑忘歌**
》是阿信作词，怪兽作曲，五月天演唱的歌曲，收录在五月天2008年发行的专辑《后青春期的诗》中，是2008年度饮料“黑松沙士”的广告主题曲。

看尽悲欢离合，再大苦痛也只化成嘴边一抹微笑，以及手牵手一起唱过这首琅琅上口的歌。团长怪兽压卷创作。永远孩子头，把其他四个小孩，还有我们心里那个小孩，统统叫到天台，这些年的辛苦不快乐，好像真的“伤心的
都忘记了”“在这个数字化的时代，人们一买到唱片可能立刻就用计算机转成MP3，专辑的概念变弱了，反而被切割成一首首独立的歌曲，一切都变成单曲导向。但那是使用者端的个人选择，作为一名创作者，我相信还有许多概念可以挖掘。”怪兽解释着。“好比漫画《将太的寿司》里便有提到，师傅在捏寿司时，每一道的滋味浓淡都会影响到下一道寿司在嘴里尝起来的味道；听歌时的顺序就跟上菜的先后次序一样重要。”“如果没有仔细咀嚼歌词涵义的话，单独去听〈
_笑忘歌_
〉可能就会觉得它太过正面、太过光明了。然而光明接在黑暗之后出现，才显得出它的可贵，听众也才会从中被治疗。”这一生到终点怎样是好？也许平凡中的快乐是最终的解答。

《笑忘歌》是一首会让人忘记过去的伤心，在历经很多苦痛，失败之后依然可以微笑着面对将要继续走下去的路，会意识到“平凡快乐也是伟大的”。下面是 _笑忘歌钢琴谱_
，希望大家会喜欢。
