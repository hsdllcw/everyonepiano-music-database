

**相信双手简谱** 和五线谱完全对应。

_相信_ 此曲收录在苏打绿的《无与伦比的美丽》专辑之中。该专辑获得2008年第八届华语传媒音乐大奖年度国语专辑、最佳乐队、最佳制作人、最佳编曲人四项大奖。

苏打绿的音乐，没有刻意的落笔，没有虚伪的诉求，但却极其真诚与善良。多元的音乐喜好和风格的撞击之下，造就出苏打绿无法归类的音乐风格。苏打绿以自己独特清新的音乐风格和扎实的音乐功底最终获得编辑青睐。绿色，年轻，是他们最鲜活生动的招牌。
