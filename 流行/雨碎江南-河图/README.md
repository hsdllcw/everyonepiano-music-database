

**雨碎江南**
是由墨明棋妙原创音乐团队创作的曲子，由河图作曲，后被多个音乐团队引用来作词。《雨碎江南》原为二胡独奏，后经改编为《雨碎江南•印象》，收录于《唱给你的歌》专辑中。

  
歌词下方是 _雨碎江南钢琴谱_ ，希望大家喜欢。
