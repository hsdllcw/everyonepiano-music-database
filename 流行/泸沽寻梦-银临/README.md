_泸沽寻梦_
是银临专辑《腐草为萤》中收录的一首原创古风歌曲。此曲很有意境：梦里我是你的阿夏，一切仿佛真实存在过，循着零碎的记忆来到这里。等到决定离开的最后一夜，你出现了，湖光山色都已经来不及。

银临的嗓音清澈干净、质朴自然，彷佛盛夏一汪碧绿的泉清可见底，又像是初冬深不见底的湖泊，充满了流动性和深邃感。时而高亢，时而轻柔，时而宽阔，时而幽远。但无论哪种，都能穿过都市的喧嚣和生活的庸碌，直达我们每个人的心底。
