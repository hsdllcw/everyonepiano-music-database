最近抖音上这首歌很火，非常的有年代感。这种老年迪斯科舞风格的音乐让人梦回90年代。

据悉，歌曲叫做《 **大风吹**
》，2021年3月11日发行。由刘涛作曲，刘涛/圆圆/李坚涛/李浩瑞作词，音乐人王赫野演唱。王赫野，1998年2月2日出生于黑龙江哈尔滨市。“好乐无荒”签约歌手，凭借个人单曲《大风吹》于2021年3月正式出道。
