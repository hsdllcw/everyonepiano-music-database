_水墨江南_
是少司命演唱的古风歌曲，由落雁知秋作词，少司命作曲，灰原穷编曲，收录于专辑《试剑江湖》中。歌曲前奏是青花瓷的一小部分，间奏也是。让人回到江南小镇，心情会愉快些。如幽篁清香般清幽的曲调，再配上那莺歌燕语似的歌喉，以及这令人陶醉其中的歌词，让听者欲罢不能，深深陷入其中，恍然如梦。
