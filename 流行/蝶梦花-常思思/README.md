

2019年10月23日，歌手常思思推出歌曲《 **蝶梦花**
》，这是常思思转型流行乐以来的又一力作，历时五年，常思思一直在刻苦钻研如何将美声民族与流行以及时尚相结合，这首《 _蝶梦花_
》便是她献给歌迷朋友的一份完美答卷。

整首歌以“对忠贞爱情的向往”为主题，作曲家邓垚老师用了很多环绕式的旋律音型来表现花瓣随风翩翩起舞的美景，来展现爱情的缠绵悱恻，辗转优柔，传递了对爱人深情思念的情感；在编曲上一股古风悠然袭来，优美而大气，表达在当下怀念过去，被回忆拉扯的无奈感受；在演唱技巧方面，常思思彻底抛开了一贯使用的花腔美声，更多的使用了“接地气”的流行演唱的方式，用柔美的声线将一场以相对天涯，却因对爱的执着而存于心底的深情故事演绎得淋漓尽致，直击听者灵魂；歌词三个字为一句，如诗般的表现出心中一幅寂静的画卷，展开了梦境与现实交错、回忆与当下交织的世界，深情与释怀，遗憾藏于心中，使整首歌笼罩在忧伤的余韵之中，构造出一段荡气回肠的爱情故事。
