

**陪你度过漫长岁月** 是歌手陈奕迅演唱的歌曲。由葛大为填词、
黎晓阳与谢国维作曲，是电影《陪安东尼度过漫长岁月》的主题曲。此曲延续Eason一贯的“E式情歌”风格，透过感人的旋律传达心声，贯彻了他多年以来的音乐固定风格。

而 _陪你度过漫长岁月_
MV于2015年11月13日发布。久违了的磁性嗓音配上唯美电影画面，让长达4分11秒的MV完美传达了影片的暖心内核。MV首次把《陪安东尼度过漫长岁月》的情感戏份完整曝光，刘畅、白百何、唐艺昕、金世佳四大主演之间的情感纠葛终于揭开神秘面纱，露出了冰山一角。故事在暖心动人的音律之间缓缓展开，刘畅默默喜欢同学多年的白百何，他们一起旅行、压马路、赏樱花，两人关系高于朋友却恋人未满。在洞悉白百何心有所属后，暖男刘畅默默做起了守护者的角色，“陪她把想念的酸，拥抱成温暖”，“陪她把独自孤单，变成了勇敢”，而MV中的独白“我原以为亲眼目睹自己喜欢的人，爱上另一个人，会很难受；想不到，当看到自己喜欢的人，失去一个人的时候，会更难过”则把刘畅、白百何、金世佳三者之间的情感纠葛直接呈现在一个景深镜头里。
