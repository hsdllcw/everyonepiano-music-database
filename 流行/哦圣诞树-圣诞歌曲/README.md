

《 _哦，圣诞树_ 》（ _O Christmas Tree_ ）是一首源自德国的圣诞颂歌，曲调是德国一首古老的民间名曲。

圣诞树象征着生命的复苏，圣诞树那常青的特质一直是德国音乐家灵感的源泉，围绕圣诞树德国音乐人士写了很多歌曲。《 **哦，圣诞树**
》就是世界上广为流传的一首。
