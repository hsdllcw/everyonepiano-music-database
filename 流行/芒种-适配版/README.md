

《 **芒种** 》是音阙诗听和赵方婧演唱的歌曲，发行于2019年6月6日。

整首歌唱出青涩女子对爱隐藏的那点羞涩，却又对爱情的期待，不由的让人触动心弦，这也许就对爱情的感觉，音阙诗听和赵方婧两人的演唱，把轻柔女子对爱情的渴望展现的淋漓尽致惟妙惟肖，歌曲中夹杂着古筝的美感，为歌曲添加一种古色的韵味，使歌曲更加完美动听，真是百听不厌。
