《 _惊蛰_ 》出自音阙诗听以节气打造的音乐专辑《二十四节气》，由音阙诗听/王梓钰演唱，2018年6月16日发行。

但在这首歌中“惊蛰”却包含更深层的涵义，当“蛰”则伏，当“惊”便起，学会顺时而动、应势而为。时机未到，学会蛰伏，时机一到，惊而奋起。明者因时而变，知者随事而制。如果你现在一腔抱负却得不到施展，就尽管蛰伏，那正是为日后的腾跃做准备。

歌曲既融入了中国古风，又掺和了现代电音，若即若离。既有古代的感觉，又有现代的味道，真的好听。
