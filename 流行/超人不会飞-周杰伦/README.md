

**超人不会飞双手简谱** 和五线谱完全对应。

_超人不会飞_ 是周杰伦演唱的歌曲，由周杰伦填词、谱曲并担任歌曲的制作人，收录在周杰伦2010发行的专辑《跨时代》中。

这是一首旋律温暖的嘻哈歌曲，听来有一种难以描述的莫名感动。简单的旋律和编曲具有回归的意义，叙事性的歌词谈及拍电影、上《时代周刊》封面等工作成绩，在淡然中透露出周杰伦的自信。《超人不会飞》这一自叙性作品的意义在于周杰伦真正地做到了言之有物。这首歌不仅在音乐性上出众，它的难能可贵在于周杰伦通过音乐非常坦诚地表现了他的精神世界。
