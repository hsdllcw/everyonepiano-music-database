_通天大道宽又阔_ 是崔京浩演唱歌曲，由姚明作曲、编曲，该歌是央视电视连续剧《西游记续集》中的片头曲。

在音乐节奏上，歌曲采用流行歌常用的手法，让它更趋向于流行化。姚明还把孙悟空的一句口头禅“吃俺老孙一棒”加到歌里，更加生动有趣，而这句道白姚明特意找孙悟空的配音演员李扬来说，让歌里的形象更加鲜明生动。
