

**心痛2009双手简谱** 和五线谱完全对应。

_心痛2009_ 是欢子演唱的一首歌曲，由欢子作词作曲，收录在欢子2009年发行的专辑《其实很寂寞》中。

起初，此曲并未收录欢子的新专辑，只作为替补歌曲收编。由于欢子对这首歌曲的忠爱程度超乎一般，最终还是由孔雀唱片决定收录。虽收录但并未成为专辑主打单曲中的“一员”。对于这首歌曲，用欢子的话说，是投入自己真情实感最多的歌曲。并不在意是否成为公司所宣传的主打曲目。
