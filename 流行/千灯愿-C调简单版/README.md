

唯美古风歌曲《 **千灯愿**
》由古风大神弦森倾情演唱，同时也是《天官赐福》花怜的同人曲，无论是歌词还是曲调都完美的诠释了花城和谢怜之间的情感，弦森的声音配上有些伤感的歌词，听着让人陶醉！

歌词中，最令人着迷的一句便是：若能护你无恙，身处无间地狱又何妨。花城也确确实实做到了这一点，一首歌曲完美诠释花怜CP，太赞了。
