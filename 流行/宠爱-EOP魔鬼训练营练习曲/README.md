宠爱_ 是TFBOYS组合演唱歌曲，由刘佳作曲，王韵韵填词，发行于2015年3月20日。  
  
2015年6月19日，《宠爱》MV正式上线发布，《宠爱》MV启用韩国新锐导演朴承勋掌镜拍摄，MV在拍摄地的挑选上颇费心思，画面中有充满青春气息的中学校园、干净唯美的街道、糖果色系的内景屋等场景，将青少年的阳光积极向上、青春活力充分凸显。MV画面清新亮丽，歌曲轻松欢快呈现出暖心的意境。  
  
《宠爱》由《青春修炼手册》原班金牌词曲组合刘佳与王韵韵联合打造，
歌词的内容准确的表达了青少年内心的感受，直白简单的文字单纯的凸显了甜蜜温暖的感觉，正如歌词所说“第一次为一个人紧张”、“这算不算爱，我还搞不明白”，此首歌曲节奏欢快，旋律律动容易感染周围的人，歌词用青涩懵懂的表达方式与TFBOYS同龄段的少年相匹配，让听众不禁会心一笑，给人甜甜温暖的感觉。让已不再是少年的人颇为怀念曾经的时光，让正值少年的人满心欢喜。希望每观看MV的人，能从饱满的剧情、直白的歌词、活泼唯美的画面，感受到这淳朴的情感。
