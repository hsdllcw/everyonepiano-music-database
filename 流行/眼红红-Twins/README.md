

**眼红红双手简谱** 和五线谱完全对应。

_眼红红_ 是Twins演唱的歌曲，收录于2002年5月1日发行的专辑《我们的纪念册》中，作曲伍乐城，填词林夕。

Twins的歌曲特别应节，《眼红红》所在的专辑《我们的纪念册》就赶上了学期即将结束、同学即将分离的季节，而她们的歌也不讲生生死死的故事，只是校园里恋爱中的一些小插曲，所以讲初尝恋爱苦涩的《眼红红》就应此而生了。

歌词下方是 _眼红红钢琴谱_ ，喜欢的朋友可以免费下载学习。
