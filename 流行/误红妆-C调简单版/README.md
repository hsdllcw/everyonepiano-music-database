

**误红妆**
是一首古风歌曲，由中国内地流行女歌手等什么君演唱。歌词唯美，歌声清澈动人，仿佛置身于古代的人流车马，曾欲仗剑走天涯，寻觅心中的远方，太惊艳了。

等什么君清澈之中又带有一丝沙哑的嗓音，犹如从大漠中吹来的风，将其中苦涩的思念传递，也将这种情意共鸣吹进了许多听众的心里。一直以来，她在荧幕前都保持着不露脸的神秘感，但是她磁性又具颗粒质感的嗓音独具特色，能将歌曲演绎的韵味十足，直击听者的灵魂。
