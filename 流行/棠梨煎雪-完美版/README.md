

《 **棠梨煎雪**
》这首歌是银临演唱的原创古风歌曲，收录于专辑《腐草为萤》。开头淡淡的钢琴声，缓缓的流水声引入了甜甜的歌声，结尾流水声又变成了雨声。前后呼应，中间欢快结尾哀伤，无可挑剔的唱功配上优美的词曲实在让人身临其境赏心悦目！

歌曲描述的是女主人公昔日与朋友用采来的新鲜棠梨果融入去年的枝头雪，一起酿酒，陶醉于田园风光的美好生活与现在天各一方，重聚之时的感慨。我们每个人都是彼此的过客，而银临的这首歌更是勾起了大家内心深处的因友情而引起的伤感之情。
