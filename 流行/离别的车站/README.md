

《离别的车站》是电视剧《情深深雨濛濛》的插曲，由赵薇演唱，歌曲情深意长，寄托了歌唱者无尽的思念。当时插曲与电视剧剧情巧妙结合，让观众们感同身受，当时电视剧一播即火，其中的插曲《离别的车站》也随之走红，成为大街小巷传唱度极高的曲目之一。
