

《 **女人是老虎** 》是李娜演唱的一首经典歌曲。此曲歌词取材于清代袁枚的《子不语•沙弥思老虎》，内容幽默，音乐节奏明朗、欢快。

小和尚下山去化斋  
老和尚有交待  
山下的女人是老虎  
遇见了千万要躲开
