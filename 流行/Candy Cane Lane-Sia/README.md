_Candy Cane Lane_
是一首非常欢快的歌曲，由Sia演唱。聆听此曲的旋律，细品此曲的歌词，瞬间脑补出很多人在圣诞晚会上穿着五颜六色的衣服，拿着拐杖糖在跳欢快的舞。

Sia的嗓音有种扭曲的质感，沙哑时像刀锉，嗞嗞作响，高音嘹亮引人入胜，即便有几个小破音，也成了歌中真挚情感的佐料，再加上澳洲人独特的英语发音，让她的歌声别具一格。而Sia随意慵懒的唱腔，像把急速的城市节奏重新调整，也恰好反映了她不想做巨星的真实心情。
