

**隅田川夏恋歌** 是KONAMI的音乐游戏「jubeat」系列收录的乐曲。之後的pop’n music系列，Dance Dance Revolution
X3中也有移植收录。

_  
隅田川夏恋歌_ 第一次出现在jubeat ripples，Gold
隠藏曲。描述的是隅田川焰火大会主题的恋爱感觉的曲子。主唱的「ALT」是电脑的合成语音（村井圣夜先生手制），并无真人演唱。EXTREME副歌有从左面上向右下象流动一样的初见杀乐谱，被玩家称之为洪水段「浊流」。由于实际难易度比标示的要高，jubeat
knit上的EXTREME从Lv8调整到了Lv9。

  
歌词下方是 _隅田川夏恋歌钢琴谱_ ，希望大家喜欢。
