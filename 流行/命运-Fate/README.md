

《命运》是韩国电视剧《浪漫满屋》的主题曲。《浪漫满屋》是韩国KBS电视台于2004年7月14日首播的水木连续剧，由宋慧乔、郑智薰（Rain）、韩恩廷、金圣洙等演出。
