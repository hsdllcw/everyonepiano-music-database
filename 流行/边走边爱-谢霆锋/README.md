

**边走边爱双手简谱** 和五线谱完全对应。

_边走边爱_ 收录于谢霆锋2003年专辑《REBORN》，分别有粤语、国语两种版本。

谢霆锋自己说这是专辑里他最喜欢的歌。到底是国语还是粤语版他可没说清楚。一样的曲子，听起来的感觉可又不尽相同。（粤语版）的词写的是路上的爱情，这一生这麼长赶上了你，偏不知终点时你在何地，那种边走边爱的无奈感觉，谢霆锋唱起来也表现了那种短暂爱情的急促感。（国语版）的感觉则是趋向凄美，作为某产品的广告歌，国语版倒是更为适合，曾经大街小巷都一个人在听在唱。

歌词下方是 _边走边爱钢琴谱_ ，大家可以免费下载学习。
