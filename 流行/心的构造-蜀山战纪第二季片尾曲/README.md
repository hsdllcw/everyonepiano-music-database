

**心的构造双手简谱** 和五线谱完全对应。

_心的构造_
由曾咏欣演唱的歌曲，作为《蜀山战纪》第二季的片尾曲。此曲由香港金牌搭档杨镇邦、甄健强担纲作词作曲，根据剧中情节、主角之间的各种爱恨纠缠为“蜀山”量身打造。曾咏欣更是在进录音棚前把故事大纲通通看了个遍，把自己的感情融入到歌曲中，不负众望超水平发挥的完成了录制，唯美的画面加上Nicola版本的主题曲为整部电视剧增色不少。

歌词下方是 _心的构造钢琴谱_ ，希望大家喜欢。
