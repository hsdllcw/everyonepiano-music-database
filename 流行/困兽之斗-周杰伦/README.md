

《 **困兽之斗** 》是周杰伦演唱的一首歌曲，由周杰伦作曲，刘畊宏作词，蔡科俊编曲，收录在周杰伦2004年8月3日发行的专辑《七里香》中。

《 _困兽之斗_
》的风格是新派的抒情重金属，歌曲的配器很地道。采用了摇滚配器，这首歌曲无论是乐队伴奏还是歌曲风格都是周杰伦力求突破的一种尝试。这首歌曲描述了爱情中的困兽，整首歌曲却听来有些柔情。这种所谓的英伦风格的摇滚歌曲被周杰伦演绎得有点离谱了。
