_让一切随风_
是由大野克夫作曲，黄沾重新作词，钟镇涛演唱的粤语歌曲，收录于钟镇涛1987年10月1日发行的专辑《听涛》。新版收录于钟镇涛2017年7月17日发行的专辑《A
Better Bee》。

钟镇涛的音乐风格受欧西流行曲影响较深，所以在他80年代的唱片里，除了个别歌曲之外，都比较少加入当时深深影响香港乐坛的日本流行音乐元素，与谭咏麟、张国荣的唱片的大量加入日本曲风的做法相异，也间接使钟镇涛的音乐稍微受普罗大众忽视。不过钟镇涛的创作才华，使他与陈百强、林子祥等比较注重音乐创作与铺排的歌手一样，受到比较资深的乐迷欢迎。
