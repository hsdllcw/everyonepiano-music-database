**_爱丫爱丫_ 这首歌曲是新加坡双胞胎人气组合“By2”2008年发行的首张专辑《16未成年》中的一首。同时也是电视剧《爱情是从告白开始的》插曲。  
  
《爱丫爱丫》也是一首混搭风的歌曲，融合和电子音乐、抒情摇滚、电声吉他摇滚等曲风为一体。  
  
这首歌曲风明快、节奏鲜明、旋律讨喜，加上姐们俩毫不转弯、直接明了的情感表达和演唱方式，整首歌表现出了年轻人对爱的渴望和执着的信念，讲述了一个少女苦恋一个男孩、等待男孩给予爱的回应的经历。
