

任然的歌曲《 **凉城** 》描写了都市感情的山盟海誓。

寒冬袭来，城市渐凉，任他车水马龙，心事却无人能懂。城市的每个角落都是回忆，每个回忆都跟你有关，每个回忆都让人心痛。谁不曾爱过？只是海誓山盟变成笑话，春夏秋冬倏忽而过，只留下我将红尘看破。

尽管是任然擅长的疗伤情歌，但王柏鸿在Xun的曲调基础上，根据耕耕极具中国风的歌词，在歌曲前后增加了声音的做旧，混入的广播调频声更是在一瞬间将人拉回金色斑驳的大上海。任然担纲和声的编写和演唱，在淡淡忧伤中将旧时代的余香缓缓唱来，令人回味悠长。
