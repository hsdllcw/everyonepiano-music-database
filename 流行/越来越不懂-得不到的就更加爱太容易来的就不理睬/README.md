

最近抖音上有很多音乐博主都翻唱了一首歌，其中有句歌词为：得不到的就更加爱太容易来的就不理睬，据悉，这是蔡健雅演唱的歌曲《 **越来越不懂** 》。

此曲改编自美国歌曲《Gotta Have You》，原唱是由The Weepies演唱（奏）的一首歌曲，《 _越来越不懂_
》于2008年7月29日由亚神音乐公司发行，歌手蔡健雅在第19届台湾金曲歌后兼最佳制作人两项大奖。

不同人生阶段对于感情的不同经历、但对于爱还是抓不住，琢磨不透。因为感情是人最难割舍的，对于怎么去爱也没有定义。确实很多事真是弄不明白，就如歌曲所说一般。爱一个人需要努力，爱的坦白，爱的绝对，要彼此相信对方，要嫁给爱情！
