KI所组成。其名字ENHYPEN，象征着在不同环境下过着不同人生的七名少年彼此连接（Connection）在一起，互相发现（Discovery）并共同成长（Growth），还包含了要用音乐将人与人、世界与世界联系在一起的抱负。另外，HYPEN亦有“连字号（Hyphen）”的意思。
