

**行き着く先**
是佐坂めぐみ为《英雄传说：闪之轨迹Ⅲ》演唱的片头曲。《英雄传说：闪之轨迹Ⅲ》是一款由FALCOM于2017年发行的角色扮演游戏，该作是《英雄传说：闪之轨迹》的续作。Switch版《英雄传说
闪之轨迹3》日版于2020年3月19日发售，6980日元，日本一发行。游戏中玩将扮演主角黎恩，带领队员们为了解开各种迷团在大陆中开展冒险。

_行き着く先双手简谱_ 是网友提供的，由网友 **帕泊里客** 制作，感谢帕泊里客对EOP的支持！
