_空谷的回音_
是一首赞美诗。据作者一变倾城所述：虽然不是基督教徒，但是音乐无国界，不分宗教信仰，心情低落的时候听还是挺合适的，所以就顺便扒一扒，给需要的人。这个是网易云里Dongye编的版本。

整曲共分两大段。头一大段，作者开门见山以自己为例子（第一人称的我），老老实实地描述我们人生的真实写照；第二大段，作者便诚诚恳恳地，以听众为对象（第二人称的你），喜喜乐乐地见证了耶稣基督的宝贵救恩。
