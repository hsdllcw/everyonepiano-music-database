

**QQ爱** 是由S翼乐团 （S.WING）演唱的飞乐唱片公司出品的同名唱片中的一首歌曲。成立于2006年初的S翼乐团以一曲《 _QQ爱_
》迅速走红网络，成为内地新崛起的流行乐团。其平民化清新爽朗的唱作、鲜明的节奏被广大乐迷接受和喜爱，业内人士也非常看好S翼乐团，认为他们是中国内地又一潜力巨大且风格独特的乐团。

  
歌词下方是 _QQ爱钢琴谱_ ，希望大家喜欢。
