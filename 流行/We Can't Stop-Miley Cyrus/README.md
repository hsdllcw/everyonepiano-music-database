**_We Can't Stop_ 是一支2013年的欧美流行音乐单曲，收录自Miley
Cyrus的第四张录音室专辑《Bangerz》中，被RCA唱片公司发布于2013年6月3日。  
  
《We Can't Stop》这首歌集成了R&B、舞曲、抒情和House
Party的音乐风格。单曲占据了新西兰、英国单曲排行榜的冠军榜首，另外这张单曲超过了一百万份的销售。2013年7月26日，MV以最快的速度大破VEVO预览记录，预览次数超过1000万次。  
麦莉•赛勒斯（1992年11月23日—），美国新生代女演员兼女流行乐歌手，乡村歌手比利雷•赛勒斯的女儿。凭借迪士尼频道电视剧《汉娜•蒙塔娜》中饰演14岁女孩——麦莉•斯图尔特而成为家喻户晓的明星。
