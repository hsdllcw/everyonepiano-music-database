

**你怎么舍得我难过双手简谱** 和五线谱完全对应。

_  
你怎么舍得我难过_
是黄品源演唱的一首歌曲，收录在同名专辑《男配角心声》当中。电影《蓝宇》片尾曲。黄品源，台湾歌手，二十岁时与哥哥报名参加“第一届青春之星校际音乐大赛”，初试啼声即获得第二名。之后，他向制作人沈光远自荐自己的创作，而得其赏识，正式与他签约。在台湾，黄品源的专辑总销售量超越一佰万张以上，但其出碟频率颇低。

  
歌词下方是 _你怎么舍得我难过钢琴谱_ ，希望大家喜欢。
