

《私奔到月球》是陈绮贞与五月天对唱的一首甜蜜、温馨的浪漫情歌。这是陈绮贞首度演唱他人创作的歌曲。对于这次合作，阿信直言因为害怕一个人离开地球表面太寂寞，因而谱了一首新曲。当这首新情歌被确定为男女对唱后，便第一时间想起了陈绮贞。

歌曲名：私奔到月球  
年份：2007年  
专辑：《离开地球表面》  
作词：阿信  
作曲：阿信  
演唱：五月天+陈绮贞
