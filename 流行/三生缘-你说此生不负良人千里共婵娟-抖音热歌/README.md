

抖音一首“你说此生不负良人千里共婵娟”的歌曲非常的好听，同时也深受众多网友的喜欢和关注。此曲叫做《 **三生缘** 》，由程响演唱。

整首歌大概的意思是：前世因、今生果，前世心心念念的情根深种，造就了今生的情缘。前世的执念，造成了今生的缘。不管是善缘还是孽缘，都因一个情字。
