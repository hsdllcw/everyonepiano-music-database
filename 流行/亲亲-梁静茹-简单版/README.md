《 _亲亲_
》是身为静茹这一次新专辑的制作人之一，五月天的怪兽谱曲，并且担纲制作的2006静茹新专辑之中第一首单曲，它扣人心弦的旋律加上充满青春甜蜜的歌词，像是为静茹量身订做的新衣裳一般让人注目。  
  
华语乐坛情歌天后
梁静茹自2005年9月发行轰动全亚洲、夺奖无数的“丝路”专辑后，已整整一年没有推出全新专辑，让全亚洲歌迷苦苦等待。新专辑今年3月就开始筹备，超过半年的努力，静茹2006年全新专辑，终于在大家的期待下推出首支单曲”
_亲亲_ ”。这不但是五月天怪兽首度为静茹量身谱曲并且亲自制作的歌曲。并马上成为le
tea乐活苹果茶的广告主题曲。这对于热爱唱歌的静茹来说，无疑是一件非常开心的事情。“我第一次拿到怪兽的demo就觉得这歌的旋律非常特别，本来还担心自己不知道可不可以把他唱好，之后怪兽又答应帮我制作，这实在是太开心的一件事…再加上这个精彩的歌词，他把这个旋律想表达的情绪完全贴切的写出来了…再加上亲亲不但被选中广告的主题曲，我还亲自拍摄了这支广告，新专辑第一首新歌能以这种方式跟大家见面对我来说真是快乐极了！”
