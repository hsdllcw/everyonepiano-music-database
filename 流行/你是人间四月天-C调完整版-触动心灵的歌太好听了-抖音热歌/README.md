

四月，绿柳吐烟，陌上花艳，正是好时光。而解忧邵帅的这首《 **你是人间四月天** 》正是对爱人满满的祝福与爱意。触动心灵的歌，简直太好听了。

愿你在我看不到的地方安然无恙，  
愿你的冬天永远不缺暖阳。  
愿你的明天不再经历雨打风霜，  
愿你的未来永远热泪盈眶。

一声问候一片心，一世承诺一生情。一点一滴存余韵，一枝一叶总关情！一生之中最难得有一个知心的爱人，不管是现在还是遥远的未来，我们彼此都保护好今天的爱。
