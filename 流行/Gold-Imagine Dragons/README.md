

**Gold双手简谱** 和五线谱完全对应。

_Gold_ 是美国另类摇滚乐队Imagine Dragons第二张录音室专辑《Smoke +
Mirrors》中的歌曲。此专辑在上榜第一周空降榜首，这是他们首次在公告牌专辑榜中夺冠。乐队被国内乐迷称为“梦龙”。2008年诞生于犹他州普洛佛，之后驻扎于拉斯维加斯。2012年凭借主流出道的录音室专辑《Night
Visions》获得巨大成功。

歌词下方是 _Gold钢琴谱_ ，希望大家喜欢。
