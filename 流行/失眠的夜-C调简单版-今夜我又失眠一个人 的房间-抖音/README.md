

最近，在抖音上听到一首好听的歌曲，其中有句歌词是“今夜我又失眠，一个人的房间，我把灯光都熄灭，熬着最难熬的夜”。据悉，此曲叫做《 **失眠的夜**
》由宋孟君演唱，发行于2021年6月16日。

好听的歌曲总能引起大家的共鸣，而《 _失眠的夜_
》就是这样一首作品，“你对我说着抱歉，试着慢慢闭上眼，脑海中浮现画面，那是一个雨天，你对我说着抱歉”，宋孟君将那种伤感的气息变换成DJ炫动的节奏，形成了强烈的反差，从而带动大家的思绪。

也正因如此，在短时间内，已有刘维、朱梓骁、张庭等多位明星跟着宋孟君这首歌舞动起来。更有大批网友跳起自编自导的《失眠的夜》舞蹈。一样的舞蹈，不一样的味道，或许这就是一首音乐作品的魅力所在。
