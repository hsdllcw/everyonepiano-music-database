_我的骄傲_
是香港女歌手容祖儿演唱的一首歌曲。由黄伟文填词、陈光荣编曲。收录在容祖儿2003年6月发行的同名专辑《我的骄傲》中，是专辑的主打歌曲。其国语版歌曲为《挥着翅膀的女孩》。

2003年该歌曲获得劲歌王榜十大金曲、第26届香港十大中文金曲奖等多个奖项；2004年在2003年度劲歌金曲颁奖礼上获得十大劲歌金曲奖、金曲金奖、最佳作曲大奖3个奖项。
