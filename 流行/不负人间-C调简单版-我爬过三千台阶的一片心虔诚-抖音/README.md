

抖音“我爬过三千台阶的一片心虔诚，拜过了菩萨焚香独善其身”还挺火的，是一首很有韵味的古风歌曲，歌词写得很美。据悉，此曲叫做《 **不负人间**
》，由阿YueYue演唱。

不用刻意的讨好谁，也不会纠缠于某个人，不能委屈了自己，也不愿意去将就，把生活过成自己想要的样子，就不负人间了。阿YueYue的这首《 _不负人间_
》表达的就是这样的情感。人生来既是夏花，就应轰轰烈烈地绚丽一方，任你密雨斜倾，我只华丽绽放，不负此生，不负人间，处处是芬芳。
