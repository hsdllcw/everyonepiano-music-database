没有了你

抖音最近这首歌曲突然就火了，此曲叫做《 **只要有你** 》，出自《还珠格格》的第三部，由还珠格格的主演们演唱，原本“腐朽” 的曲调，为何突然火了？

不怕歌好听，就怕词入心！纵然情深，奈何缘浅！

加上抖音上各路大神的神剪辑，使这首原本琼瑶式“酸溜溜”的歌曲，充满了伤感。每次听完都感动的流泪。歌曲唱出了生死相许的凄美爱情，同时又勾起大家的青春往事。不禁使人感叹，如果世上没有无奈的分离该有多少。。。。
