

**等待双手简谱** 和五线谱完全对应。

_等待_ 是黄绮珊演唱的一首歌曲，由汪峰作曲。这首歌在《我是歌手》第一期演唱过。

著名乐评人宋柯所言，“听到《等待》，情绪感蔓延便扑面而来，黄绮珊确实是被埋没的好声音。”而《等待》的原创者汪峰也在微博中感慨：“生命中有些人，有些事需要等待，十几年前我写下的这首歌，十几年后的昨晚黄绮珊早已用属于她的梦幻般的歌声，以接近完美的表达，让所有聆听者为之动容。”

歌词下方是 _等待钢琴谱_ ，希望大家喜欢。
