

**杀手EOP** 由网友yuhuang制作，感谢yuhuang对EOP的支持！

《 _杀手_ 》是由林秋离作词，林俊杰作曲，林俊杰演唱的一首歌曲，收录在专辑《西界》中。这首歌是一首流行风格的歌曲。歌曲展现了林俊杰的高超唱功和创作才能。

歌词下面是 _杀手双手简谱_ ，喜欢的朋友可以下载。
