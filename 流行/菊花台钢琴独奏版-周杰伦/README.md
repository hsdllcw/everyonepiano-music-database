周杰伦与周润发、巩俐合演的电影《满城尽带黄金甲》的主题曲。干净的吉他配上沧桑的弦乐，描绘出电影中皇后死前的一幅唯美的画面，方文山中国风的歌词再创高峰！如诗如歌的词，北风、菊花、雨，写尽情的伤、命运的多舛，而周杰伦亦以一种和以往不同的细腻嗓音与咬字的演唱方式，搭配简单却唯美的编曲，唱出凄美绝伦的菊花台。
