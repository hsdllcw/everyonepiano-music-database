

《 **俏郎君** 》是香港歌手张敬轩的单曲，于2021年1月7日由飞图唱片派台发行，并收录于张敬轩同日推出的专辑《The Brightest
Darkness》。

歌曲主题的创作灵感源自芭芭拉•史翠珊与罗拔•烈福于1973年主演的同名电影。歌曲推出后获得极大成功，成为903专业推介、中文歌曲龙虎榜、新城劲爆流行榜、劲歌金榜跟Chill
Club 推介五台冠军歌。
