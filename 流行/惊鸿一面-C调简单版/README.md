

《 **惊鸿一面**
》是由许嵩亲自创作的一首歌，编配工作由曾为《卷珠帘》编曲的田汨先生参与完成。为了确保完全实现许嵩自己对于乐器和人声层次的编排构思，该曲他并未邀约以往曾合作过的所有知名混音师，而是埋首工作室亲自完成混音工作，体现十足诚意。

另外，此曲由许嵩和黄龄演唱。黄龄的歌声实在媚到骨子里，却也不令人反感，而且这样独特的声音甚至难以被模仿。而许嵩给人一种翩翩书生的形象，黄龄则是杨柳下起舞的温婉少女。十足的画面感与故事感。两人在这首歌曲里表现得完美无瑕，惊艳亮相，把风花雪月唱的得酥到骨子里。可以说合作很成功，这首歌曲很适合看古言时听。
