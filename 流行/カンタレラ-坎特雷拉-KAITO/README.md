_カンタレラ_
，中文为坎特雷拉，是黒うさP所作歌曲，描写KAITO与初音之间的禁忌之恋。人物原型是文艺复兴时期意大利的罗马尼亚公爵西泽尔•波吉亚，拉丁文是凯撒•博尔吉亚（有毒药公爵之称，据说常年带着面具）及之后的卢克雷奇娅•波吉亚，费拉拉、摩德纳和雷吉奥公爵夫人（Lucrezia
Borgia）。
