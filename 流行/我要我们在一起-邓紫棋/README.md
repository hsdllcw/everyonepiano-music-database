

**我要我们在一起双手简谱** 和五线谱完全对应。

_我要我们在一起_ 是由邓紫棋于我是歌手第二季第四期进行翻唱，收录在专辑《X.X.X.Live》，2013年11月发行。并获得一致好评。

此曲是由李泉填词并作曲，范晓萱原唱(收录于1999年11月发行专辑《我要我们在一起》）中的主打歌曲。李泉本人也曾经唱过。

歌词下方是 _我要我们在一起钢琴谱_ ，大家可以免费下载学习。
