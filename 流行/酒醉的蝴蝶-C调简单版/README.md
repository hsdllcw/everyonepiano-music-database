

抖音“怎么也飞不出，花花的世界”是歌曲《 **酒醉的蝴蝶** 》中的一句歌词。

《 _酒醉的蝴蝶_ 》是由刘海东填词、谱曲，冯丹编曲，崔伟立演唱的一首歌曲。该歌曲是在2019年8月7日发行的，歌曲一经发行，便获得了很多歌迷朋友的喜欢。

歌曲表达了作者迷茫的心情。把自己打扮得花枝招展，像蝴蝶在五彩缤纷的花丛中飞来飞去，寻找着心中的目标。心中被那种原始的欲望驱使，怎么也飞不出去。自己就像一只喝醉酒的蝴蝶，跌跌撞撞迷失在花丛中，失去归途。直到有一天突然明白过来，已经伤痕累累。

春来春去，花开花落；月升月落，月圆月缺，这一切只不过是自然规律而然。爱情是漫长而短暂人生中一晃而过的众多体验中的一种，而痴情的人却总是念念不忘那最刻骨铭心、销魂蚀骨的那一刻而不能自拔，就像一只醉酒的蝴蝶，迷失了。为了一句轻描淡写的誓言，留下永远打不开的心结。
