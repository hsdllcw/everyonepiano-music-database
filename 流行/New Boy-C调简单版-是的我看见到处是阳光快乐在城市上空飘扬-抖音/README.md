

“是的我看见到处是阳光，快乐在城市上空飘扬”出自歌曲《 **New Boy**
》。抖音上常听到的版本是房东的猫翻唱的。此曲原唱是朴树，是朴树摇滚音乐代表作。

房东的猫版《 _New Boy_
》注入了青春烂漫。歌曲融合了电子舞曲的元素，用轻松欢快的曲调代替沉重的鼓点，展现了房东的猫独有的色彩和感性，仿佛回到了少年少女18
岁的时候，对未来充满向往，心生悸动。

而原唱朴树，是在世纪之末的1999年发行的，即将迎接千禧年的到来，唱出了人们对于新时代的向往，对未来的向往和期待，但也或多或少有着对未来的恐惧与迷茫。
