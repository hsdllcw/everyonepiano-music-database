

**《第一次爱的人》** 是由林怡芬作词，Marit Elizabeth/Marion作曲，王心凌演唱的。这首歌曲收录于王心凌的专辑：《爱你Cyndi
Loves You》，并与2004年03月26日发行，一经发行就受到广大乐迷的支持。

另外，《 _第一次爱的人_ 》是中国女歌手王心凌翻唱M2M的《 **The day you went away** 》
王心凌凭着电视剧《西街少年》，令她的人气飙升至全亚洲！她的第二张专辑《爱你》亦于3月26日上市！从这张专辑内，各位歌迷可看到
王心凌在加拿大安大略大湖泊所拍的写真照，由摄影师黄中平亲自操刀。美景好歌，王心凌最新唱片必定令乐迷们更加《爱你》。
