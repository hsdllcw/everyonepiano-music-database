

**你瞒我瞒** 出自陈柏宇2009年发行的专辑《CLOSE UP》。歌曲道出了不少失意中情侣的心声，每次听到都让人倍觉忧伤，难过。

《 _你瞒我瞒_
》的副歌几乎全段假音，歌词讲拍拖拍到徒具形式貌合神离的地步，编曲大部分时间都只有一个钢琴伴奏，反而营造出很冷的气氛，直到最后那种郁闷才随着音乐的推进爆发出来。
