_辞めてやるよ歌い手なんか_
是なまはげP创作，重音テト演唱的歌曲。重音テト，中文重音teto，在2008年4月1日的愚人节前夕诞生，原型是模仿CRYPTON（CRYPTON
FUTURE
MEDIA）公司的VOCALOID产品而诞生。但受初音未来的影响，在NICONICO动画网上属于比较高影响力的虚拟偶像。现在是UTAU歌声合成软件的虚拟歌手，而官设画像及声音资料所有权已集中归TWINDRILL（ツインドリル）保护。

歌词下方是 _辞めてやるよ歌い手なんか钢琴谱_ ，大家可以免费下载学习。
