_  
真爱_
是台湾偶像剧《王子变青蛙》的电视原声，由183club演唱。《王子变青蛙》是一部台湾偶像剧，由陈铭章和刘俊杰执导，明道、陈乔恩、赵虹乔、王绍伟主演。2005年6月5日在台湾上映。香港于2006年02月12日在翡翠台播出。根据尼尔森收视统计，《王子变青蛙》首播获得3.92的收视率，随后以平均7.09、最高8.05的收视率，打破台湾偶像剧《流星花园》所保持的最高收视率纪录6.43。

  
歌词下方是 _真爱钢琴谱_ ，希望大家喜欢。
