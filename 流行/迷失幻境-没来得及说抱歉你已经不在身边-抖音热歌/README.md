每一个人都曾经迷失在幻境之中，来不及的说出的分手就这样结束，就像歌中唱的一样。抖音这首歌曲最近很火，名字叫做《 **迷失幻境**
》，是一首伤感情歌，由IN-K/王忻辰演唱，超级温柔治愈！

音乐有意思的地方在于，它总是会偷偷为你储存很多不同的回忆。并且每一首都像是一间带锁的小房间，每间小房间都装着不一样的故事。繁忙的现实生活常常压的你喘不过气，你以为自己忘记了很多人很多事，却会在某个不经意间因为某首歌的旋律而让回忆变得无比锋利。听着此曲，你又想到了什么呢？
