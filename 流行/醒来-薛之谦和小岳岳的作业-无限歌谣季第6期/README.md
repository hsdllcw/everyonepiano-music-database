

**醒来双手简谱** 和五线谱完全对应。

_醒来_
，又名薛之谦和小岳岳的作业，是薛之谦与岳云鹏在无限歌谣季今生第一次主题演唱会首唱的主题原创歌曲。歌曲以全新曲风，直击小人物内心的挣扎，不仅现场收获高票，更在音乐平台获得傲人成绩。

而最大的惊喜莫过于，藏在《薛之谦和岳云鹏的作业》（歌曲原名）幽默逗趣歌名背后，走心而动人的全新音乐。“我疲倦的灵魂重演着，我沧桑的肉体缝补了”
、“醒来后，继续活着”等一句句直指人心的歌词，道尽了都市小人物逐梦过程中的沧桑与迷失，同时又不甘于屈服命运安排的复杂情感。两人阐述，这首歌所写的“今生第一次”，落点在第一次想要勇敢面对自己。薛之谦解读：“在五花八门的社会上，我希望哪怕每个人，坚持一下，再找一下自己的梦想，不要被生活所左右。”节目一经推出，就引起了网友们的热烈讨论，纷纷留言：“没想到薛之谦和小岳岳的组合，泪点比笑点还多，我在你们的歌里，又找到了自己，谢谢你们。”

在大家建议下，“签约组合”歌曲正式更名为《醒来》，不仅记录了“签约组合”两人的友情和惺惺相惜，也是他们突破自己音乐态度最好的见证。薛之谦在微博感性发言：“如果我不再写情歌了，小岳岳也不再搞笑了，新歌《醒来》，愿每个人心中的炽热在妥协后还没有死去。”薛之谦没有选择自己擅长的“薛式情歌”，转而抒写质朴普世的小人物内心世界，而不变的是始终上乘的词曲创作。不仅如此，在曲风上，薛之谦大胆尝试了摇滚风格，相得益彰的歌词和曲风，让观众们直呼：“今晚的这首《醒来》，燃炸了。”作为主持人的张绍刚也感慨：“作为一个已经成熟的唱作人，要突破自己实属不易。”

歌词下方是 _醒来钢琴谱_ ，大家可以免费下载学习。
