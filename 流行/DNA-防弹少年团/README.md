《 **DNA** 》是韩国男子组合防弹少年团（BTS）韩、日语歌曲。歌曲是他们第五张韩语细碟 《Love Yourself 承
'Her'》的主打歌曲，之后也被收录在第四张特别专辑《Love Yourself 结
'Answer'》。韩语版在2017年9月18日正式推出，日语版则在同年12月6日由环球音乐发行。

歌曲主题为命运和一见倾心的爱。《 _DNA_
》是表现青春时期充满青涩与冲劲的爱情歌曲，表达了青年们衷心的爱，它包含了一个信息，即“我们两个从最初就是命中注定要连结的，我们的DNA可以视作同一个东西”。该曲风以流行电音为基础，加上具有中毒性的口哨声和纯吉他的旋律，虽然是像防弹少年团的风格，但又与原本组合的色彩做出差异化，带来更高一层的音乐。
