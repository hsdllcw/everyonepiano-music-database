

“我的背脊如荒丘，而你却微笑摆首，把它当成整个宇宙。”7月25日也是周深出道6周年的纪念日，他用这首歌，感谢歌迷们一路的陪伴，希望有机会用自己的方式，温暖每一只“
**化身孤岛的鲸** ”。

其实《化身孤岛的鲸》最早并不是周深的歌，再说得明白一点，就是《 _化身孤岛的鲸_
》就不是“正规军”——它源于网络上对香港歌手谢安琪原创作品《我们都被忘了》的改编。后经沃特艾文儿重新填词，只佛演唱才有了《化身孤岛的鲸》的雏形。在周深等众多歌手的演唱后翻红。

歌曲采用全新的编曲，巧妙地将悠远寂寥的鲸鸣编入曲中。当周深的歌声与鲸鸣融为一体时，极尽空灵与美好，诉尽心中所想，讲述着一个在孤独与爱中成长的故事。全新的编曲下，我们看到了那只频率52赫兹的，孤独的灰鲸，它藏于深海，正独自深情地告白。这首歌不仅见证了周深六年来的蜕变，也象征着他作为成熟歌手又一个全新的起点。
