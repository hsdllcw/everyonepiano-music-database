《 _冬至_ 》是音阙诗听以节气打造的音乐专辑《二十四节气》系列歌曲，由音阙诗听与王梓钰演唱。于2019年12月22日正式发行。

俗话说：“冬至到，吃水饺。”此曲通过描写“吃饺子”这个冬至习俗，让大家感受到了淡淡相思之情。

其实，冬至吃饺子是因纪念“医圣”张仲景冬至舍药留下的，感谢他在冬天用羊肉、辣椒和驱寒的药材熬制“祛寒娇耳汤”治疗人们的耳朵冻疮，所以，现在一些地区，还有“冬至不端饺子碗，冻掉耳朵没人管”的民谣。

那么问题来了，冬至你吃饺子了吗？
