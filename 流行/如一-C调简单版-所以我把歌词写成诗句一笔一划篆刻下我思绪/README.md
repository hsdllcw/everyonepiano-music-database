这首歌曲在抖音上受到了很多网友的喜爱，还被翻唱了很多版本。据悉，歌曲的名字是《 **如一**
》，由不是花火呀/小田音乐社原唱，在2020年11月18日上线。

这是一首暖暖的情歌，历经晴空烟雨 ，见过四季朝夕，这些我想讲给你听， 旧时爱恨别离，今夕花好月明，画岁月依旧，待你如一。
