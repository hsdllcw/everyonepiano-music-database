

沙宝亮，杂技演员出身，“顶坛子”技巧得过国际大奖，之后于1986年毕业于北京艺校，1993年步入歌坛和现代人乐队合作开始创作、演唱。沙宝亮写歌已经很多年，创作一般比较随意的，96年时与其好友段目然成立沙漠工作室，沙宝亮负责作曲，他负责作词，期间为戴娆写了《期待》和叶蓓的《我是谁》。沙宝亮现签约点时唱片有限公司，推出单曲《我的秋天谁来过》，位居中央人民广播电台《中国流行歌曲榜》第二名，并在全国各地电台几十家排行榜上榜。
