

《 **认真的雪**
》是由薛之谦自己创作的歌曲，该曲记录了他第一次深刻的感情经历，歌中的种种伤心也都是他的真实心情写照。如歌词中“爱得那么认真”、“雪下得那么深”、“倒映出我淌在雪中的伤痕”等都写出了薛之谦对爱情的执着和伤怀。

在这首歌的创作初期，薛之谦多次提交给制作人过目，希望能得到对方的建议和意见，但是制作人总是用一句“你放心大胆地去做”来打发薛之谦，起初为此闷闷不乐的薛之谦直到专辑《薛之谦》制作完成，才明白了制作人的苦心，因为制作该曲时制作人不加以干涉，就能给薛之谦最大的自由创作出原汁原味的歌曲。
