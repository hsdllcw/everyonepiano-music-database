

《 **离别** 》是阿杜演唱的一首悲伤情歌，在阿杜的深情演唱下，给人一种很悲伤的感觉，正是题目所表达的感情。收录于阿杜2002年发行的专辑《天黑》之中。

阿杜（1973.03.11—），原名杜成义，新加坡人，祖籍福建闽南，华语著名歌手，演员。乐华娱乐签约艺人。2002推出首张专辑：《天黑》，沧桑沙哑的嗓音唱出许多动人情歌、缔造出许多“阿杜风潮”，成为华语音乐新一代的情歌天王。出道至今发行九张专辑：《天黑》、《坚持到底》、《哈啰》、《醇情歌》、《I
Do》、《小城故事》、《差一点》、《没什么好怕》、《第9次初恋》。参演电视剧：《爱情烘焙屋》、《都市恋人的追逐》。出演舞台剧：《爱上卡门》。
