

**莫斯科没有眼泪**
是TWINS演唱的歌曲，由许常德作词，伍乐城作曲，收录在首张国语专辑《见习爱神》中。该专辑集合了台湾音乐人共同制作，词曲有周杰伦御用词人方文山，台湾金曲奖常客李焯雄、林子良、许常德和五月天阿信，并请到三大台湾金曲奖制作人陈子鸿、陈伟和周世晖共同操刀，发片当日即售出20万张，10天内销量达到80万张。

  
歌词下方是 _莫斯科没有眼泪钢琴谱_ ，希望大家喜欢。
