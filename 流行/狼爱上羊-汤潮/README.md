

**狼爱上羊** 是汤潮演唱歌曲，由汤潮作词，作词，2006年发行。汤潮说这首歌创作的时候运用了逆思维，歌唱的是美好的爱情，同时也歌唱和谐。

  
歌词下方是 _狼爱上羊钢琴谱_ ，希望大家喜欢。
