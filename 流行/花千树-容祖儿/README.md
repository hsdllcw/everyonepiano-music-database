

**花千树** 是由黄伟文作词，Matthew Tishler与Andrew Ang 作曲，由容祖儿演唱的一首粤语歌曲。

  
该曲讲及要珍惜感情，别轻易为了一片森林或艳丽的花而放弃最适合自己的爱人，错过了最适合自己的伴侣，因此比尔贾导演于MV中要求祖儿做出于树林寻觅的感觉。对于首次合作的比尔贾导演，Joey欣赏过其作品，已对他的独特又新鲜的创作感赞不绝口。祖儿穿着的飘逸长裙是与导演沟通过后，Joey精心挑选了好几件衣服，从香港带过台北，拍摄当天比尔贾导演看到所有衣服都十分钟爱，很合心意又配合意境，在场工作人员都笑言Joey与导演心灵相通，令大家都非常期待今次的合作。《
_花千树_ 》MV中Joey在树林绰约生姿，飘缈漫步，场面充满美感，更满是诗情画意。

  
歌词下方是 _花千树钢琴谱_ ，希望大家喜欢。
