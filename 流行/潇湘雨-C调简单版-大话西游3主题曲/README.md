

《 **潇湘雨** 》是胡彦斌为网络游戏《大话西游3》创作的主题曲。歌曲中所唱到的潇湘是游戏中的一位主角。由林文炫作词，胡彦斌作曲并演唱。

胡彦斌的歌确实有魅力，独立的嗓音让人无法忘怀，听他的歌好像回到古代看着意境里的风土人情，好像他本该属于那个时代一样。
