

**我们的合唱歌**
是2R演唱的。2R，香港二人组合，由新加坡籍姊妹黄婉君和黄婉伶组成。她们年龄相差三年，能歌擅舞，有浓厚的表演欲，当时身在新加坡的2R，被环球音乐集团看中签约，来港发展演艺事业，两人表现兴奋雀跃。她们于2003年推出的首张唱片《Two
of Us》曾一口气连改三版，声势绝不亚于当时的任何新人。2R出了音乐，还演出过不少影视作品，现已解散。

_我们的合唱歌_ 收录于2R的专辑Greatest Hits。另外， _我们的合唱歌_ 是由吴国恩作曲的。歌曲很是好听。
