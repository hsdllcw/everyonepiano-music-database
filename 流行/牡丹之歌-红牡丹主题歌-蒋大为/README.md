

**牡丹之歌** ，创作于1980年，是电影 《红牡丹》 的主题歌。词作者乔羽，曲作者唐诃、吕远，原唱蒋大为。

  
蒋大为是我国最著名的男高音歌唱家，是一位最具实力的男歌手，是中国大陆歌坛上的一棵长青树。他具有深厚的声乐造诣，音清质淳。八十年代他演唱 “ _牡丹之歌_
”一举成名，近30年来他演唱过上千首歌曲和上百部电影、电视剧的主题歌，深受歌迷追捧。他的歌声传遍祖国的大江南北，响彻海内外,他的歌声将永远留在人们记忆的脑海里。
