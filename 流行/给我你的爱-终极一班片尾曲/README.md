

《给我你的爱》为偶像剧《终极一班》的片尾曲，歌迷询问度最高歌曲，最具渗透力的温暖情歌。

2006年最受瞩目的音乐发电机-
TANK！创作、演唱、舞台魅力兼备的华研唱片超猛重装新人TANK，来自“GAME世代”，堪称目前流行乐坛最年轻的创作型歌手！从电玩、爱情、生活的小细节激发出创作灵感，具有丰沛音乐天赋的TANK
21岁发表第一首创作曲，今年24岁，短短2年多便创作出200多首歌曲，创作爆发力令人惊艳！出道前在民歌西餐厅6年的驻唱经验，加上曾经自组舞团、乐团，扎实的磨练更是让TANK练就出极富魅力的舞台肢体表现。
