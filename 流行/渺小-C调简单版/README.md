

《 **渺小** 》是周耀辉填词，周子朴谱曲，JerryC编曲，田馥甄演唱的歌曲。

是一首华语流行主流乐坛少见但欧美独立乐界最爱的电气民谣佳作，更是一首贴近田馥甄心灵状态的歌曲，这体现在音乐上除了继续“田氏音乐”的独特风格，制作团队还从田馥甄的个人的音乐爱好里撷取元素来拓宽新的视觉，而歌词内容就锁定在剖析田馥甄个性上的冲突矛盾与对生命生活情感的自我反思上。
