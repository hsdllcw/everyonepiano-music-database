_滚滚红尘_
是一首由罗大佑作词、作曲，陈淑桦演唱的歌曲，收录在专辑《一生守候》中。同时此曲也是同名电影《滚滚红尘》的主题曲。电影《滚滚红尘》由严浩导演，三毛编剧，林青霞、秦汉、张曼玉和吴耀汉主演，1990年公映。影片内容影射民国时期女作家张爱玲与胡兰成的感情纠葛以及张爱玲与炎樱的姐妹情感。

歌词下方是 _滚滚红尘钢琴谱_ ，希望大家喜欢。
