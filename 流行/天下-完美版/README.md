

《 **天下**
》是一首大气磅礴又不失侠骨柔情的歌曲，由中国流行乐男歌手张杰演唱。张杰希望通过这首歌能传递一种自信，每个人经过打拼能拥有自己一片天地的自信。

《 _天下_
》传统的中国风曲调配上创新的流行曲势，佐以如诗如画的古典诗词，再加上画龙点睛般的古式弦乐管乐，华丽大气、气概磅礴。张杰用激昂而不失柔情的演绎将一段不爱江山爱美人的侠骨柔情表达得淋漓尽致，从壮志凌云到儿女情长，歌声中有压抑、柔情、暴烈和坚毅。歌曲的过渡段采用了一整段中国式的rap，体现了中国诗词般特有的节奏感和韵律感，也让这首歌在许多类似的作品中显得有些与众不同。
