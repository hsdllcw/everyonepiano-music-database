

**弯弯的月亮**
是刘欢演唱的，曾经风靡了整个神州大地，歌曲优美的曲调感动着人们的内心，它也被认为是中国流行音乐界从港台翻唱、西北风等等热度流行过后走向沉静和内敛时期的代表作品。

  
刘欢，生于1963年8月26日，流行音乐家，对外经济贸易大学音乐教授。是集作词作曲、编曲、制作、演唱和音乐教育于一身的音乐人，也是中国内地头一批能用电脑作曲、编曲和制作的音乐人之一。

  
歌词下方是 _弯弯的月亮钢琴谱_ ，希望大家喜欢。
