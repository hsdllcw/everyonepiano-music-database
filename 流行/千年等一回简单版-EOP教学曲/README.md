_  
千年等一回_
是由陈自为填词，左宏元谱曲，尤景仰编曲，台湾著名女歌手高胜美演唱的，是首播于1992年的中国经典歌舞片《新白娘子传奇》的主题曲。歌曲旋律优美、唱词经典、朗朗上口、回味悠长，配合着经典《新白娘子传奇》的热播而被推向巅峰，成为华语乐坛传唱率最好的歌曲之一，亦成为白蛇传故事和浪漫爱情的代表旋律。该曲作为20世纪经典经典不衰的好歌，至今仍是卡拉OK点唱率极高的歌曲。
