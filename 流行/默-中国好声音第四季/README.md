

**默** 是电影版《何以笙箫默》的主题曲, 由尹约作词，钱雷作曲，那英原唱。后周杰伦在2015中国好声音第四季中演唱了这首 _默_ 。

“我被爱判处终身孤寂，不还手，不放手。笔下画不完的圆，心间填不满的缘，是你”，
简单干净的旋律，却有一种无可比拟的坚定，没有让你听到一丝颤抖，却让你的心随着她的呼吸、她的语句、不住地绞痛与颤抖，直至难以自制。

歌词下方是 _默钢琴谱_ ，希望大家喜欢。
