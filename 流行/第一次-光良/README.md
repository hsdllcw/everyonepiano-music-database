

**第一次**
是张天成作词，马来西亚籍华裔歌手光良作曲、演唱光良演唱的一首歌曲，收录在其首张个人专辑《第一次个人创作专辑》中。这是一首流行风格的歌曲，歌曲展现了光良温柔的嗓音和创作功力。

  
歌词下方是 _第一次钢琴谱_ ，希望大家喜欢。
