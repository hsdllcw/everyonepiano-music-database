

**花满楼**
是由许嵩（Vae）演唱的一首歌曲，发行时间为2008年3月。此曲表达了一段带有复古诗意、刻骨铭心的爱情。清新的曲调，优雅的曲风。令人听后感到一缕忧伤。

《 _花满楼_
》至今成为网络询问度最高的歌曲，因为这是第一首内地创作团队（好友音乐）敢跟周杰伦挑战的歌曲，其歌词方面让很多人眼前一亮，尤其是间奏部分的诗词唱段让人觉得仿佛进入诗情画意之中。

“江山如画心纷飞，只等伊人生死陪，强说愁容春不回，单膝跪地月已碎”，这首诗词唱段出自海雷之手，表达了一段带有复古诗意刻骨铭心的爱情，用现代手法铺垫复古情节，绝对是中国风式歌词中的靓词佳品。
