_纯白_ 是中国台湾配乐作曲家、流行钢琴家V.K克的一首钢琴曲。同时也是音乐手机游戏《Deemo》中的一首音乐。

V.K克的音乐作品有种魔力，让人一听就彷佛有画面出现在眼前。听完他的专辑，好像谈了一场刻骨铭心的恋爱。

V.K克出生于台湾省基隆市，4岁开始学琴，幼儿园时就展现出他的音乐天赋，后受日系歌曲影响打开钢琴即兴之路。12岁多次拿下基隆市少年组小提琴比赛优胜及代表参加台湾区赛入围决选，13岁开始尝试作曲。在14岁时诊断出罹患僵直性脊椎炎，但仍不放弃音乐，开启自己的音乐人生。

他的音乐有故事、有情绪，还有行云流水般的想像力，可以媲美日韩流行音乐演奏文化。
