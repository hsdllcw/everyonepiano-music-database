

**继续 - 给15岁的自己双手简谱** 和五线谱完全对应。

_继续 - 给15岁的自己_ 的最原始版本为日本歌手Angela Aki2008年的一支单曲。原名为《信，写给15岁的自己》「手纸 ～拝启
十五の君へ～」（别名《信，敬启者给十五岁的你》），后被翻唱收录在刘若英2010年发布的专辑《在一起》里，这首歌也是整张专辑的第一首歌曲。

这首歌曲里表现出了也许很多东西早已在时光的打磨下物是人非，但我们还是要头也不回地去抓住那些生命中最初也最终的美好的情感。

歌词下方是 _继续 - 给15岁的自己钢琴谱_ ，希望大家喜欢。
