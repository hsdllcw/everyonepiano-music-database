

《 **退后** 》是收录于周杰伦专辑《依然范特西》里的一首歌曲。

《 _退后_
》是南拳妈妈的弹头宋健彰的词，单纯而深刻的描述了一段纯爱，这是每个爱过的人都曾有过的伤感与遗憾，听完了气势磅礴的爱情大戏，周杰伦的的番石榴情歌，朗朗上口旋律优美，是品种绝佳的番石榴，也很适合失恋时在KTV狂K一下。

MV剧情：周董街头当混混，最爱喝Hebe卖的果汁。
为了绯闻女友Hebe，“周董”周杰伦和小混混争风吃醋，大打出手，但Hebe最后仍琵琶别抱，嫁给别人，徒留伤心的周董……

这个精采的MV故事——“退后”，今天将在本报自由广场2楼的国际演艺厅独家首播，周董本人更将专程从日本赶回台，现身现场与本报幸运读者共赏MV、共度中秋。

周董当混混 Hebe卖果汁
周董今天下午4点将出现在本报举办的“依然范特西MV首映中秋自由见面会”，因座位有限，许多无法参加的读者向隅，本报特别和周董合作，预先提供“退后”MV的纸上首映，让读者抢先看。
周杰伦MV中，周董饰演街头混混，暗恋摆摊的“水果妹”Hebe，有一回看见小痞三搭讪Hebe、摸Hebe的手，他就挥拳相向，导致冲突；后来又有少年家来搭讪，这回Hebe以眼神示意自己会解决，然后送对方水果，轻松打发走，Hebe对周董说：“你看，这不就解决了吗？”

浪子回头摆摊 Hebe已嫁人
但周董不能体会“水果妹”的心意，不改好勇斗狠习性，有时被打得很惨，Hebe会心疼他的伤口，两人甜蜜交往；周董有一次和Hebe去打台球，摸Hebe手.然后被周董打的那个小痞三带人来找他们的麻烦.周董和他们大打出手,那个小痞三的老大打不赢周董就走了,然后带人去把Hebe的水果店砸了,周董带人去酒店找小痞三的老大报仇.后来被警察抓了,Hebe只能眼睁睁地望着情郎被警察抓走。

终于，周董洗心革面摆摊卖金桔柠檬，对街一个小孩跑来买饮料，他才发现小孩的妈妈竟是Hebe，感慨地说：“我已经变了，但也来不及了！”歌词下方提供了
