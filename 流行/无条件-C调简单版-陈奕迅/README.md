

《 **无条件** 》是香港流行男歌手陈奕迅的一首粤语歌曲，其初版是《异梦》。收录于专辑《准备中》中，发行于2015年07月13日。

《 _无条件_
》贯切了陈奕迅一贯的情歌风格，透过感人的旋律传达了陈奕迅内心的心声，传递无条件感知世界，无条件去创造自己的力量的情感。歌曲词曲编唱俱佳，陈奕迅的演绎可圈可点，把这首歌诠释得非常到位。

另外，《无条件》作为专辑《准备中》的主打，其在副歌部分的旋律悦耳程度，以及那种氛围感营造，至少没能让这一次的陈奕迅寒酸得连一首门面歌都拿不出来。
