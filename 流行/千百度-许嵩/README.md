《 _千百度_ 》是许嵩2011年最新专辑《苏格拉没有底》中的一首中国风歌曲，民族清乐歌曲。于2011年04月01日推出。  
  
《千百度》歌名出自辛弃疾的
《青玉案•元夕》中的“众里寻他千百度”。当中国风的歌曲泛滥，人们都在奇怪2010年之后为什么除了方文山，能把中国风写红唱红的就只剩许嵩了？《清明雨上》、《半城烟沙》、《庐州月》等等口碑之作，全部出自许嵩笔下。典故的化用，创造新词汇的想象力，借古喻今的手法，传统戏曲技巧的融入——他自有他自己的一套。在《千百度》里，温婉内敛与开阔大格局兼容并蓄，这是许嵩独有的音乐气质，换个人唱就没有这个味了。其实纵观专辑里的每一首作品，都具备这种“非他莫属”的专属质感。

》中自成一派的制作风格与词曲质感，随性慵懒的演绎方式，标志性的假音与转音，舒服惬意的中慢板R&B节奏……令人一听便可品出强烈的个人风味。在悠然自得的节奏摆荡中，浪漫与批判并存、务实与务虚交织。
