最近在刷抖音的时候，总是能听到这段旋律。据悉，此曲叫做《 **想你这件事** 》，由晨晨晨作词、作曲，烟(许佳豪)演唱，发行于2021年5月27日。

伤感的情感总是能引起很多人的共鸣，共同经历的过往岂能说忘就忘，只是用了一种比较隐蔽的方式来表达罢了。也正因为这样，这首歌曲在抖音上传唱度很高，很受大家的喜爱。

歌词下方是 _想你这件事钢琴谱_ ，大家可以免费下载学习。
