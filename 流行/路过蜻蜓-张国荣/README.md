

**路过蜻蜓双手简谱** 和五线谱完全对应。

_路过蜻蜓_ 是歌手张国荣演唱的一首粤语歌曲，由林夕作词、陈晓娟作曲、Adrian
Chan编曲。这首歌收录在张国荣于2000年3月1日发行的专辑《Untitled》中。

作为歌手，张国荣是粤语流行音乐的指标性人物之一。他是香港歌坛第一位全方位能快能慢的男歌手；在舞台上声色艺俱全。除了对粤语流行音乐贡献良多外，他亦为华语流行音乐在韩国开拓市场做出了贡献。他在亚洲很受欢迎，作为第一个打开韩国唱片市场的粤语歌手，他保持着华语唱片在韩国的销量纪录；打破了只有欧美流行歌手垄断韩国外国乐坛的传统。
