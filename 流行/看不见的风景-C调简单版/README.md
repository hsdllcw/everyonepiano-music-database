

《 **看不见的风景**
》是许嵩演唱的歌曲，由许嵩作词作曲。此曲给人一种压抑感，而这种压抑感是来自于许嵩那慵懒的嗓音，来自这首歌的旋律，来自这首歌的歌词。听这首歌曲感觉像是从地狱中慢慢爬出来的一般，有绝望忧伤之感。

许嵩（Vae），中国内地流行乐男歌手、词曲创作人、音乐制作人。他罕有绯闻或博眼球版面的言论，以作品本身稳固乐坛地位。他在个人生活方面的这种低调，亦一直为人称道。
