

《粉末》为贺岁电影《十月围城》主题曲，由李宇春演唱。李宇春的声音就好像是歌曲的催化剂，整首歌曲大气悲情，但李宇春的演唱又不失细腻。要在这首MV中传达出电影的气质，对家国的情怀和刺激的观影体验。这就注定了这首歌的拍摄和剪辑应该是三者之间的平衡——李宇春的演唱；她在电影里的表现；以及电影的大时代氛围，和那群义士的情怀。”
都值得人回味。并且歌曲提名第二十九届香港电影金像奖最佳原创电影歌曲大奖。
