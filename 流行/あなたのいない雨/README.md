_あなたのいない雨_
是日本艺人、歌手若槻千夏演唱的歌曲。若槻千夏，1984年5月28日生于日本埼玉县。参演了多部电视剧、电影，如《车灵》、《极道鲜师Ⅱ》、《新三人时代》等。
