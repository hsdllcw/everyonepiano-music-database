

东方神起携第六张专辑《Catch Me》回归乐坛。东方神起的《Catch
Me》，包揽泰国、台湾各大音乐榜单第一的位置，他们的高人气又一次得到认可。东方神起的这首歌在泰国Channel
[V]的亚洲榜单中连续6周（10月29号~12月9号）占据第一。
