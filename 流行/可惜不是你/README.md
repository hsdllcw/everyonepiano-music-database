

《 **可惜不是你**
》是亚洲情歌天后梁静茹专辑《丝路通往爱的路途》中收录的一首歌。在静茹的演绎下，生动感人，催人泪下。有着很高的知名度和传唱度，是一首很有代表性的爱情歌曲。整首歌旋律节奏优美，快慢相辅，轻重有致又有层次感，最主要的是梁静茹在这首歌当中真假音的完美结合。
