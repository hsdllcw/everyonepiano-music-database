这是一首纤细的旋律和钢琴声尤为突出的Pop
R&B歌曲。歌曲以朴素简单的钢琴roofing为基础，渐渐展开，到副歌部分则呈现出冒险一般的变化，让人印象深刻。歌词描写和相爱之人两个人，追溯曾经忘却的记忆，走在暮夜之中的情形，将梦幻般的氛围推向高潮。
