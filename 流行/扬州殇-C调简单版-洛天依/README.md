

**扬州殇**
是杏花包子作词，铅笔作曲并于2013年9月12日投稿至bilibili的VOCALOID中文原创歌曲，由洛天依演唱，为妄想历史系列的第一作。殿堂曲，截至现在已有29.89万次观看，1.03万人收藏。

歌曲以南宋词人、音乐人姜夔(kui)为第一人称创作的歌曲，讲述国破山河之际，主人公望已笔伐敌的豪迈情绪。歌词中更是化用了白石道人姜夔的《扬州慢》、《凄凉犯》的几句诗词，更添色彩。微凄凉又热血的曲调中，带着义无反顾的坚定。

### 扬州殇歌词：

绿杨巷陌 秋风起瑟瑟凄凉  
追念镜湖上 晚花行月小舫  
春风 翠叶 青花  
疏篱 红砖 新瓦  
当年杜郎俊赏 遣怀歌尤唱  
吟一曲 扬州慢 鬓已老  
桥边红药 也难抵这岁月刀  
年少轻狂 潇洒墨扬  
空得虚名一场  
十年匆忙 国破只留残墙  
念这兴亡 叹这过往  
若年华容得我嚣张  
愿将这竹笛换做血马缨枪  
衰草寒烟 淡雾勾惘惘思量  
遥想边域狼 戍楼吹角城长  
肃风 翠凋 红落  
腐木 黑云 黄沙  
如今二十四桥 仍在人惆怅  
唱一首 凄凉犯 恨不孝  
胡马 窥江 身不在 心如焚烧  
寒雪冻疮 飞石迷瘴  
纵惹珠泪盈眶  
十年茫茫 长恨对月独怆  
念这兴亡 叹这过往  
若年华容得我嚣张  
愿将这笔墨换做铁甲酒囊  
昨夜犹惊梦 今朝尚难醒  
竹笛斩敌首 笔墨收广陵  
棹移慢桨 新月中当  
行舟倒影幢幢  
一眼荒凉 再叹亦是神伤  
感这兴亡 放这过往  
如东去的江水流淌  
纵白发苍苍 报国心尚痴狂
