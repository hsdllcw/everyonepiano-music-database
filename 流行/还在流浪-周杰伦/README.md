

周杰伦暌违六年的全新专辑《最伟大的作品》第二主打单曲《 **还在流浪** 》于2022年7月15日上线。其MV带给了所有听众一份足够的意境深远与玩味！

从方文山的歌词，到周杰伦的曲调与演唱，整首歌包裹在一个核心词里“回忆”，回忆过去的时光，回忆曾经一起的美好，回忆我们共同拥有的记忆，和那些没有来得及拥有的未来。

周杰伦试图用创作与歌声，把非常私有化的“回忆”，唱成每位听众都会深有感触的共鸣。一如歌词所记录，“当你收到信，我还在流浪；那年的约定，都放在心上；随身的照片里，你微笑的模样；亲爱的，你现在怎么样？”
