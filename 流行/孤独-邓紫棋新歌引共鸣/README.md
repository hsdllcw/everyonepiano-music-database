在新歌《孤独》中，邓紫棋这样写道：“原来孤独，是灯光下，所有人都对我佩服。但月光下，却一直害怕自己退步。”从这两句话中我们可以感受到作为艺人、明星、音乐人，他们承受的那种压力感是与日俱增的。
