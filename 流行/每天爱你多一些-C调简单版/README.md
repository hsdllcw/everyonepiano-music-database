

《 **每天爱你多一些** 》是张学友演唱的一首经典歌曲，原曲缘自桑田佳佑的《真夏の果実》。后灵魂战警在蒙面歌王第一期翻唱了此曲，赢得了不少人的好评。

这首歌曲首次在张学友1991年发行的专辑《情不禁》中被收录。张学友于1991年举办演唱会《张学友每天爱你多一些演唱会1991》，受到广泛欢迎。1991年，《
_每天爱你多一些_ 》成为当年香港至尊金曲。
