

**故梦** 是墨明棋妙原创音乐团队橙翼主唱的一首歌，收录于《天命风流》专辑发行。后被无数人翻唱，如双笙。

个人感觉，橙翼唱的更有故事感，有层次，含蓄又情深，歌声缥缈，似能将人带入歌里那遥远的故事中，身临其境地踏入一幅幅如歌词描述般美妙却悲伤的画面中。似无情却又字字饱含深情，婉转的颤音为这首歌增添无限韵味。曾经许诺好要相守此生，如今却恍如隔世，是情深缘浅，抑或只是一场空梦终醒。
