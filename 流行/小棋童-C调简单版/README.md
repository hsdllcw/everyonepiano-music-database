

**小棋童** 是墨明棋妙团队原创歌曲，最初由不纯君原唱，后被无数人翻唱。其中最著名的是双笙的版本。悠长的声线
将古风和流行音乐组合在一起，不得不说这首歌是极好的。

歌曲可分为六段，主要给大家呈现了一个悲伤的故事。  
第一段，技不如人称风寒，小小棋童犯了难。  
第二段，异人异象饭量高，望穿星河悟天道。  
第三段，扳拆断关如天神，白龙斩蛟赢美人。  
第四段，流连难断算机关，棋童搬家惴不安。  
第五段，天花乱坠骗爹娘，自欺掩耳弄离殇。  
第六段，溯流直上寻伊人，梦中落英缤如纷。
