

**祝你平安**
是电视剧《都市平安夜》的主题曲，由刘青作词作曲，孙悦演唱，该歌曲于1994年发行。1994年，该歌曲获上海东方风云榜年度十大金曲奖和中央电视台年度MTV金奖。

  
《 _祝你平安_
》，旋律优美动听，歌词涵义给人以平安、安定、轻松和温馨的感觉。加上演唱者孙悦的演唱，饱满而富有张力，更为跟着歌声轻轻和的你我增添了几许厚重的回忆。

  
歌词下方是 _祝你平安钢琴谱_ ，希望大家喜欢。
