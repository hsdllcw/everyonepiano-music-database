_故乡游_
是毛不易演唱歌曲，包珍妮作词，钟易轩作曲。毛不易，1994年出生于黑龙江省齐齐哈尔市泰来县，毕业于杭州师范大学，中国内地男歌手。2017年9月获得音乐偶像养成节目《明日之子》“最强厂牌”（总冠军）。
