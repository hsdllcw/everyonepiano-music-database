

《 **A.I.N.Y.** 》是由邓紫棋作词，并与奥地利音乐人Lupo
Groinig共同作曲完成。整首歌想要表达的意思是“就算我需要你，可是我还是要离开你”。

作为专辑《18》主打歌之一，邓紫棋用实力高音及流畅的唱功，诠释情感的伤痛，上线三天点击率就突破350万。可以说，此曲的难度度极高，邓紫棋以娴熟的真假声转换征服听众的同时，这首歌也以充满话题性的MV在网络上迅速传播，成为她的代表作之一。
