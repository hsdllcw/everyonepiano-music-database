

**雪千寻双手简谱** 和五线谱是完全对应的，是EOP教学曲，很适合新手练习。

雪千寻也是一首凄美的乐曲，是1993年影片《东方不败风云再起》的经典配乐，这首曲子与影片中的女主角——雪千寻同名。

关于雪千寻这首曲子的作者，是中国香港的武侠配乐大师胡伟立，后收录在：1993年《东方不败风云再起电影原声带》。

_雪千寻钢琴曲_ 能让你体会到：孤寂的哀伤，凄美的爱情，短暂的友情。《东方不败》
东方不败以前是朝廷中人，也是个千户，是汉人，为人敢爱敢恨，敢做敢当，而闻名于世。

他在巡查的时候遇上了青楼女子雪千寻 ，上天注定他们相爱，于是娶了雪千寻， **雪千寻** 也是汉人 ，在东方不败没有变成女人之前
，和她有过一段轰轰烈烈的感情故事 。
