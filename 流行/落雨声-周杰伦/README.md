_落雨声_
是一首感谢亲恩的台湾闽南语歌曲，由方文山作词，周杰伦作曲，收录在1999年台语天后江蕙（二姐）发行的《半醉半清醒》专辑里，这首歌也是周杰伦与方文山在出道前卖出的第一首歌。

江蕙华语女歌手。出道20多年的台语歌后江蕙，堪称当今台湾乐坛身价最高的女歌手，从以前唱到现在，张张专辑都大卖。因所唱歌曲大部分都是悲伤情歌，与妹妹江淑娜合称“苦情姊妹花”，近年来苦情不再，应称“幸福姊妹”。江蕙歌曲共426首，江蕙专辑63张。

歌词下方是 _落雨声钢琴谱_ ，喜欢的朋友可以免费下载学习。
