

《将爱》邀请到了天王天后级人物王菲与陈奕迅为主题曲《 **因为爱情**
》献唱，王菲被粉丝称为“女神”，陈奕迅的微博ID则是“吹神”，“二神合体”奉献的这首《 _因为爱情_
》也因此获得了“神曲2”的美名。对于与王菲首次合作的作品，陈奕迅称自己当天也是第一次看到成品，“我单看MV就已经很感动了，而且能跟王菲合作，这实在是太给力了，其它你懂的。”
