_我好吗_
是梁静茹演唱的歌曲，由黄婷填词、阿超谱曲，该曲于2019年4月26日以单曲的形式发行。后收录在梁静茹于2019年5月16日发行的专辑《我好吗？－太阳如常升起》中。

在七年未发专辑的期间，梁静茹在人生里细致地用音乐体验人生。人们总在问候别人好不好，却忘记停下来问问自己“我好吗？”，《我好吗》是唱给听众自己的歌。
