

《画心》，是电影《画皮》的主题曲，2009年4月19日，《画心》荣获第28届香港金像奖最佳电影歌曲，由张靓颖演唱。《画皮》是2008年陈嘉上导演中国首部东方新魔幻巨献影片。

双手简谱和五线谱是完全对应的，属于EOP教学曲。
