

《 **画** 》是由邓紫棋谱曲、填词、演唱的音乐单曲，《Live Piano SessionⅡ》系列的第二首全新创作单曲，等待秋去春来，等待下一次花开。

《Live Piano SessionⅡ》是邓紫棋2016年发布的全新现场钢琴录音系列，已发布作品《给你的歌》。《 _画_
》是现场钢琴录音系列II中发布的第二首G.E.M.邓紫棋全新创作歌曲。它是一首似水彩画一般简单直接的一首情歌，温柔和眷恋是它的画笔，思念被画下，期许被画下。对着这张色彩斑斓的画，请相信你对爱的灵感。
