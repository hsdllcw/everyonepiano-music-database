_走在冷风中_
是刘思涵个人首支单曲，收录在合辑《花开的声音》中及其个人专辑《拥抱你》中。2011年刘思涵参加青海卫视《花儿朵朵》选秀节目后签约种子音乐。唱片公司特邀创作人为其量身打造了《走在冷风中》。

这首歌曲是刘思涵首次进棚录歌，其录制过程也由陈信荣全程在旁亲自监督指导，为达到理想的效果，陈信荣抓住刘思涵的声音特质，因材施教并帮助她寻找感觉。

很多歌迷都是通过《走在冷风中》这首歌爱上了刘思涵独特的声音。
