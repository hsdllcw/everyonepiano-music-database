

**浮生辞** 是银临演唱的一首原创古风歌曲，由嵯嵨作词，银临作曲，灰原穷编曲。初次听这首古风歌曲时，便被它优美的曲调惊艳了！

爱江湖，爱的不是它的快意豪情、恣意潇洒，而是在江湖中的人。他们各有各的故事，各有各的喜怒悲欢、意气选择，在茫茫人海中相遇，又不留牵挂的离开。因为在等，所以愿意，喜欢这个曲调，就好似人的喜欢经过曲折，后的欢喜。
