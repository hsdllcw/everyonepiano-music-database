

《 **简单的幸福** 》是罗聪演唱的一首歌曲，发行于2018年4月26日。

然，近期却陷入了抄袭风波。被大家指出与许嵩2010年的歌曲《[ **有何不可**](Music-2598-有何不可-许嵩.html
"有何不可")》几近相似。抛开抄袭，此曲轻松悦动，俏皮可爱，听后确给人满满的幸福感。

_简单的幸福双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
