

抖音“所有美好全部come for ya，我的爱也全部come for ya”出自歌曲《 **给你呀** 》，又名：For Ya，由蒋小呢演唱。

其实这首歌讲的大概是暗恋吧，就如歌词里所说的“喜欢你是我的秘密深度就像在海底十万米”，而你，也是我的全部美好。

蒋小呢，中国内地女歌手，00后女音乐人，曾是篮球队员，现居住在南京。果然不会唱歌的法学生不是个好篮球运动员。

_给你呀双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
