

**当你孤单你会想起谁双手简谱** 和五线谱完全对应。

_  
当你孤单你会想起谁_ 是飞图唱片公司旗下组合新少年俱乐部(New Boy's
Club)首唱，张栋梁翻唱的歌曲。这首歌在华人地区的传唱程度，让各地的歌迷及传媒，为栋梁取了“华语K歌小天王”的封号。

  
张栋梁，马来西亚歌手、演员。1981年11月29日出生于马来西亚沙捞越州古晋市，祖籍福建。

  
歌词下方是 _当你孤单你会想起谁钢琴谱_ ，希望大家喜欢。
