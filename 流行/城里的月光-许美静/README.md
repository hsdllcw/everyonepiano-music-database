

**城里的月光** ，由陈佳明作词、作曲，原唱为新加坡女歌手许美静，收录许美静第二张个人专辑《遗憾》中。

  
另外，如王菲、Tank等歌手，也曾翻唱过这首歌。《 _城里的月光_ 》粤语版名为《月光光》，由林夕填词，依旧由许美静演唱。

  
歌词下方是 _城里的月光钢琴谱_ ，希望大家喜欢。
