

**许愿双手简谱** 和五线谱完全对应。

_许愿_ 是梁咏琪、古巨基演唱的歌曲，由凌伟文作曲，何启弘填词。

梁咏琪（Gigi Leung）1976年3月25日生于香港，歌手、演员，毕业于玛利诺修院学校，获得香港理工大学设计学院文凭课程。

古巨基，1972年8月18日出生于香港，中国香港男歌手、演员、主持人，毕业于沙田官立中学。

歌词下方是 _许愿钢琴谱_ ，喜欢的朋友可以免费下载学习。
