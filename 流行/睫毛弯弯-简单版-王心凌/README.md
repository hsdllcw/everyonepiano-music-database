

《 **睫毛弯弯** 》是中国台湾歌手“甜心教主”王心凌的代表作之一，收录在专辑《Cyndi With
U》中。歌曲继续了活泼可爱的形象，节奏欢快，歌词甜蜜，真实，超甜！

这是一首充满东方风味的歌曲。风格上延续了王心凌的可爱俏皮风，配乐上采用了三个东方音乐元素（印度西塔琴、中国笛子、冲绳三味线）乐器的演奏。其中最特别的还当数印度的西塔琴与中国笛子的融合，而在编曲上又加入了西方的雷鬼嘻哈节奏和古典和弦。编曲和词意紧密融合，以东西方音乐的融合互动，来呼应歌词中两个不同背景的人的相遇与一见钟情。整首作品的音乐元素丰富多彩。
