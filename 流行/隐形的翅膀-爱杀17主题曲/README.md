隐形的翅膀_ 为张韶涵主演的电视剧《爱杀17》主题曲，收录在专辑《潘朵拉》中。因为其通俗易懂，传唱大江南北。歌词健康，积极向上，又被当做励志歌曲。  
  
拍摄《隐形的翅膀》MV时，原本导演设计要让张韶涵吊钢丝，飞上天空，但因下起小雨无法拍摄，于是导演想出了另一招，让她站在摄影吊臂上，加装了一个框架，把张韶涵送上了天，虽然拍起来效果不错，但镜头下的张韶涵却冷得发抖。另外MV剧情也安排张韶涵是一名具有魔力的天使，可以用眼神就让杯子移动，甚至只要一举手，连地震都会停止，不过这些神奇的法力都是透过导演事先的巧妙安排与特效。  
  
从歌词中我们也可以清楚的看到，这首歌曲所要表达的情感：面对困难，无助时，我们所要做的是勇敢的面对，用于克服，而不是退缩。
