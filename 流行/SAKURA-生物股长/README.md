

**SAKURA**
是日本著名音乐组合生物股长的成名曲，也是除之前三张地下专辑外第一张正式的唱片，收录了SAKURA、ホットミルク以及翻唱松谷任由实的名曲卒业写真。  
  
主打歌 _Sakura_ 为DENPO115东日本地区CM歌曲，并成为2007年生物股长首次登上红白歌对战的选用歌曲。  
  
歌词下方是 _Sakura钢琴谱_ ，希望大家喜欢。
