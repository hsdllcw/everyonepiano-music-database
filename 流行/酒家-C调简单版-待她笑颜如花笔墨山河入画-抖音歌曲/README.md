抖音这段歌词出自网络歌手小阿七发行于2020年5月22日的歌曲《 **酒家** 》。

这是一首很有江湖侠气的歌曲，一壶温酒入喉，随你浪迹天涯！如果你有一颗仗剑纵马走天涯的心，不妨听完这首歌，便浪迹天涯，感受这尘世的繁华。
