

**蒲公英的约定** 是周杰伦演唱的一首歌曲，收录在其专辑《我很忙》中。  
  
《 _蒲公英的约定_
》是一首传统的周氏情歌，延续了周杰伦以往的风格。此曲写的是周杰伦的中学和他的初恋。他初中所在的淡江中学，在学校后面有个蛮有趣的篱笆园，周杰伦特别为了这个写的这首歌曲。创作灵感源自班得瑞乐团的《在辉煌的季节中》，并且最开始作为电影《不能说的秘密》中的插曲出现，后发展为的这首情歌。
