亥俄定居过。MGK出生后的4年都在埃及度过，所以他首先说的语言是阿拉伯语。MGK从6年级开始听说唱歌曲，14岁时搬到了现 在的家乡克里夫兰。

，她出生于1997年。Camila出生在古巴，她五岁时随父母搬到了美国，在这之前，她曾住在哈瓦那和墨西哥。2016年12月19日退出Fifth
