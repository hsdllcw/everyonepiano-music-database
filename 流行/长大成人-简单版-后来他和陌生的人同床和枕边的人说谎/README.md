《 _长大成人_ 》由范茹演唱，发行于2022年7月30日。

此曲的词作者在歌曲下评论说：“写了瓶瓶罐罐很多歌，讲了汤汤水水很多故事，这一次我们来讲一个硬一点儿的故事，它横竖藏着点儿真实，又高低带点儿想象，也多少沾点儿矫情，但是我们希望这个故事会让你心动这么一下。”

这首歌之所以能引来如此大的共鸣，是它故事感的歌词，面对长大，就是要面对失去，面对突如其来的分别，面对那些说不清道不明的犹豫。也就是说，正是切合实际且真实的歌词，打动了人心。
