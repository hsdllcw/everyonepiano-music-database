_  
醉清风_
此曲是中国流行女歌手、演员弦子的代表曲目。弦子（Killer），全名张弦子。1986年4月22日生于广西百色，毕业于广西艺术学院附属艺术学校。高中时与同学创作网络歌曲，在网络界小有名气。曾考入星海音乐学院，但因种种原因放弃就读。2005年被台湾制作人相中后签约成为歌手。  
  
第一次听这首歌的时候，觉得很普通。后来越发觉得这不是一首歌，而是歌手用音乐来诉说了一个故事。至于故事的主角是谁，或许不是歌手本人，我觉得更可能是那些陶醉于歌声的听者。那如天籁的声音，一直陪伴着我们，度过了多少个无眠的夜晚。
