

**求佛**
是誓言演唱的歌曲，此曲使他红遍祖国的大江南北，也让他独特的嗓音像曾经的刀郎一样铭刻在了无数歌迷心中。曾在2006年无线音乐咪咕汇上获得年度十大畅销金曲奖。

  
歌词下方是 _求佛钢琴谱_ ，希望大家喜欢。
