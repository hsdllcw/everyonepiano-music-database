

《 **等你等到我心痛** 》是张学友演唱的粤语歌曲。此曲煽情的很透彻，把爱情的痛演绎的淋漓尽致。

伴随着经典悠扬的口哨声，歌曲旋律缓缓起伏，张学友的声音逐步升高，直到“等你等你等你，一世一世等你，我真的真的不愿舍弃”，让人感慨万千。

接着，歌曲逐步升华，渐入高潮，张学友的声音几乎是喊出来的，这种等真的让人伤感与无助。想一下，一个失恋的人，在雨夜中等一个不会来的人，会是怎样的绝望？幻想着那一个人的背影在人群和雨中慢慢的消失，又是多么得无情。只有深爱的人，才会这么傻傻地去等一个人，即使是一世一世地等，可是现实总是残酷的。

今天再次重温这首歌，当真是初听不识曲中意，再听已是曲中人，歌中的情谁没有经历过呢？
