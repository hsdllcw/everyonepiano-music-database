

天ノ弱是虚拟歌手GUMI演唱的一首歌曲，由164作词、作曲，于2011年05月28日投稿至日本视频网站niconico动画，拥有超过500万次的播放数。2011年12月4日播放百万达成，也是164的第一个百万曲。

歌曲贯彻了164一直擅长的神调教与曲风，内容是叙述一个人无法率直的表达自己的心情，只能靠唱著「反义词」的歌曲来抒发他的情感，是一首非常特别的情歌。
