

**探窗** 是由东方千月作词，郭丰琳作曲，国风新语、浮生梦、汐音社演唱的歌曲，发行于2021年6月12日。戏腔一出，道尽了伤感之情！超级好听！

本是青梅竹马，两心相许，后来男子赴京赶考，女子在故乡等候。日子一天天过去，男子仍未归来，女子便决定去找男子。可在古代女子没什么地位，女子只能投靠进京的戏班一边卖艺赚钱一边盼着早日寻到男子。而男子却已金榜题名，得到高官赏识，赐了好姻缘，终是负了故乡等候的女子。男子成婚那天请了戏班来庆贺，恰巧是女子所在的戏班，女子在台上看着自己心爱之人正和别人拜堂成亲，心里难受万分，浓妆粉墨穿着戏服在台上一步一句诉说相思之意，男子已沉浸在众人的祝贺之中，又怎会在意到台上的戏子。
