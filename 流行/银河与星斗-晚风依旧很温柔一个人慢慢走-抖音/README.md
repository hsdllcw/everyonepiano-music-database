

晚风依旧很温柔  
一个人慢慢走  
在街道的岔路口  
眺望银河与星斗

抖音此段旋律出自歌曲《 **银河与星斗** 》。《银河与星斗》是由李昀格、刘思情作词，轮子作曲，yihuik苡慧演唱的歌曲，发行于2021年5月30日。

这是一首伤感情歌。世界这么大，两个人能遇到已经是缘分了，能够在一起更是难得，每个人都希望感情能够天长地老，可惜很多事情根本不是光想就可以，还有很多未知的原因。时间就像是一个小偷，让原本相爱的两个人，走着走着就散了，所以，在一起就要好好珍惜。
