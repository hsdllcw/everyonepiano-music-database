

**如果我们不曾相遇双手简谱** 和五线谱完全对应。  
_  
如果我们不曾相遇_
是由台湾摇滚乐团五月天演唱的一首歌曲。五月天曾预告仅出10张专辑，这回“作品9号”保密到家，首波推出狂欢电音歌曲《派对动物》，21日曝光如果我们不曾相遇歌词版MV，旋律、歌词逼出粉丝泪水。有人形容：“现代版的《萍聚》！”被看好有望风靡各大校园，成为学子最爱的毕业歌。  
  
歌词下方是 _如果我们不曾相遇钢琴谱_ ，希望大家喜欢。
