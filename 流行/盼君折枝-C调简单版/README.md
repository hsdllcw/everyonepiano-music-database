

《 **盼君折枝** 》是《[ **怎叹**](Music-13287-怎叹-C调简单版-怎叹呐山有木兮那木有枝心悦君兮啊君不知.html
"怎叹")》的兄弟篇，是郑鱼回馈支持《怎叹》的歌迷和山人粉丝们的作品。青丝白首，山河，折扇，千山暮雪，唯此心如旧……诸多元素与意境，汇集成了我们心中的盼君折枝。

歌曲发行于2021年8月6日。歌名中“折枝”的灵感来源于汉乐府《折杨柳歌辞》中的“折柳”一词，借以表达惜别怀远之意。

相较于《怎叹》，《 _盼君折枝_ 》中更多了一份江湖恩怨的风格，会更具有力量感，更多元化的诠释国风文化魅力，光是短短的副歌部分就已经值得循环播放了。
