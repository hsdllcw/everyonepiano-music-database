

**稻香** ，是周杰伦08年新专辑《魔杰座》中的主打歌。
此曲于9月22日全球同步首播，10月9日专辑全球同步发售。在这首歌里，周杰伦展露自己的音乐创作以外的出色填词功力，《 _稻香_
》是继《蜗牛》、《听妈妈的话》后，周杰伦再次用简单的文字，创造出极富想象的童年世界的意境。曲的部分他也大胆尝试，以嘻哈与民谣两种音乐混合，甚至在前奏里加入蟋蟀声。而歌曲要传达的概念就是“知足”，周杰伦表示歌曲的创作来源是有感于近来天灾不断，加上身边不少朋友常陷在困难的负面情绪里，他更决定选择“稻香”做为第一主打。  
  
歌词下方我们提供了 _稻香钢琴谱_ ，希望大家喜欢。
