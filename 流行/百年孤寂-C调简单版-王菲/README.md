

《 **百年孤寂** 》是由林夕作词、王菲演唱的歌曲。该曲的粤语版为《守望麦田》，均收录于王菲1999年发行的专辑《只爱陌生人》中。

《 _百年孤寂_
》在编配上有着浓郁的中东色彩，歌词中的后一段词写得尤为气势磅礴，非林夕不能作。林夕用佛法式的比喻写下了歌词“悲哀是真的、泪是假的，本来没因果。一百年后，没有你、也没有我”。从这首《百年孤寂》到其粤语版《守望麦田》，林夕用歌词记录了曾经的刻骨铭心，而王菲把这首歌唱得跌宕起伏、十分大气，显示出了她超强的歌曲驾驽能力。
