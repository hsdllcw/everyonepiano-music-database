

《 **爱乘以无限大** 》（爱X无限大）是元若蓝第一张同名专辑的主打歌。曲调引用：拜尔的钢琴练习曲 8(踩到猫儿)。

元若蓝(1986年5月28日生)，为台湾的一个歌手，在2006年10月发行第一张个人专辑《爱X无限大》。2008年，与吴忠明合唱偶像剧《命中注定我爱你》主题曲而一炮走红，并发行电视原声带。  
歌词的下方还提供了 _爱乘以无限大钢琴谱_ ，欢迎感兴趣的朋友下载使用。
