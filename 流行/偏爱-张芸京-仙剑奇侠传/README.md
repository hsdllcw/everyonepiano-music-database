

偏爱收录于张芸京首张个人专辑《破天荒》,作为电视剧《仙剑奇侠传》主题曲。张芸京，本名张芸菁，台湾台北人，歌手。于2007年参加台湾台视主办的歌唱选秀节目《超级偶像》第一季，并获得冠军头衔。于2010年12月4日于香港STAR
HALL首度举办个人售票音乐会－“张芸京正不正初回音乐会”。爆满的三千多名来自香港、中国台湾、中国大陆、美国、澳门、新加坡、马来西亚等地，献唱25首歌曲。
