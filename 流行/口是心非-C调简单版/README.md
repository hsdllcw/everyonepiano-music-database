

**口是心非** 是由张雨生作词作曲并演唱的一首歌，收录于同名专辑《口是心非》中
，为著名歌手张雨生去世前的最后绝唱。1998年，获得第9届台湾金曲奖最佳流行音乐演唱唱片奖。

一首如诗的歌，一首如歌的诗，在激烈高亢的节奏中，在哀怨悲愤的歌词中，都深深表达出创作者对爱情的自我感知和深深的无奈，而这种滋味无可消除，都只能随雨生的歌声一一向听者道来。再过百年，这首音乐作品将依然是经典！
