

**青春赋双手简谱** 和五线谱完全对应。

_青春赋_ 是2015年上映的日本电影《大幕拉开》，日文幕が上がる的主题曲。且此曲由桃色幸运草Z演唱。桃色幸运草Z（ももいろクローバーZ，Momoiro
Clover Z），隶属日本著名艺人事务所Stardust
Promotion艺能3部，于2008年结成。“Clover”是指三叶草，但由于官方英文团名“Momoiro
Clover”中间有一四叶草标志，因此团名中的“Clover”特指四叶草，通常一万株三叶草才有一株四叶草，故在日本又名幸运草。

歌词下方是 _青春赋钢琴谱_ ，希望大家喜欢。
