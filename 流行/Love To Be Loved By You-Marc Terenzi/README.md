Love To Be Loved By You_ 是一首旋律优美的英文歌曲，演唱者是美国流行歌手马克·特伦茨（Marc
Terenzi）。2005年，Marc与德国歌手莎拉·寇娜（Sarah
Connor）结婚时，将这首歌作为爱的礼物献给妻子。Marc凭借此曲迅速在德国以及欧美走红，单曲强势占领2005德国排行榜亚军，抢攻欧洲流行单曲榜Top
10。
