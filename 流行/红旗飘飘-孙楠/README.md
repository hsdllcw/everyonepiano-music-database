

**红旗飘飘** 是由乔方作词，李杰作曲，孙楠演唱的歌曲。这首歌的原唱是李杰，出自1996年电视剧《警苑神掌》主题曲，又名《五星红旗》 ，是孙楠的成名曲。

  
歌词下方是 _红旗飘飘钢琴谱_ ，希望大家喜欢。
