

**相依为命双手简谱** 和五线谱完全对应。

_相依为命_ 是陈小春2004年精选集《夜生活》中的歌曲，在2015年的古惑仔之岁月友情演唱会中演唱而被内地观众所熟悉。

陈小春，本名陈小臻，客家人，1967年7月8日出生于广东省惠州市惠阳区淡水镇沙田村，中国香港男歌手、演员。

歌词下方是 _相依为命钢琴谱_ ，希望大家喜欢。
