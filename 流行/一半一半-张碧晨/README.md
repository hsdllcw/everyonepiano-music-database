_一半一半_ 是华语流行乐女歌手张碧晨演唱的歌曲，歌曲发布于2017年4月26日。

这首最新推出的单曲由金牌制作人荒井一郎、携手葛大为作词、徐佳莹作曲共同打造，朗朗上口清新浪漫的旋律成为张碧晨“晨式情歌”的全新路线，这首《一半一半》作为新专辑《开往早晨的午夜》绝对福利彩蛋回馈给众多粉丝。

歌词下方是 _一半一半钢琴谱_ ，大家可以免费下载学习。
