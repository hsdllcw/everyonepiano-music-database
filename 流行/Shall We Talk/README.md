解析  
在这里喝汤其实指的就是沟通与交流的方式。在香港，喝汤也是一种重要的家庭文化。  
第一个"宁愿在公园躲藏 不想喝汤"说的是小时候宁愿躲在公园里也不愿意回家面对父母。  
第二个"情人在分手边缘只敢喝汤" 意思是两个人差不多到分手的地步了互相面对也不愿意多说话，只沉默的喝汤。  
第三个"孩儿在公司很忙不需喝汤"表示的是自己已经长大了，不再需要父母的照顾。
