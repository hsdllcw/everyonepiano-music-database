

**大梦想家双手简谱** 和五线谱完全对应。

_大梦想家_
是由TFBOYS演唱的歌曲，是由青春修炼手册的原班人马打造。在2015年8月15日TFBOYS两周年粉丝见面会上进行首唱，并于2015年8月17日正式发布。

暖暖的音乐律动、怀旧与梦想交融的歌词，再一次碰撞出TFBOYS成长路上的“追梦”宣言。

歌词叙述了TFBOYS出道两年来对于成长与梦想的感悟与追求，也表达了即使平凡，只要敢想，人人都是“大梦想家”。

歌词下方是 _大梦想家钢琴谱_ ，希望大家喜欢。
