

**霸王别姬** 是中国内地歌手屠洪刚1996年发行的单曲，歌曲由陈涛作词，冯晓泉作曲，薛瑞光编曲。最早收录在屠洪刚1996年发行的EP《霸王别姬》中。

屠洪刚的“中国风”有一种自己的生命力，屠洪刚的音乐具有“独特性”，从听觉上非常独特。而这首《 _霸王别姬_ 》日后也成为屠洪刚的代表作品。
