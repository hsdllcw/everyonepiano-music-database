

《发如雪》，周杰伦演唱的歌曲，也有相关MV。是周杰伦推荐的作品。蕴含三种唱腔，被誉为中国风流行乐开创始祖，继“东风破”后再登巅峰靓作！歌中演绎中国传统色彩是短诗，继续走五音宫调路线，延续了《东风破》中式古典的哀怨婉约，RAP儿话音的加入使音乐层次更丰富。另有同名小说。
