

《 **少年中国说** 》由张杰演唱。2018年3月24日在央视一套节目《经典咏流传》演唱，2018年在央视一套节目《开学第一课》演唱。

近期在2021上海大学开学典礼现场上，张杰又再次演绎了此曲。

虽然已经38岁了，但是张杰丝毫没有步入中年的模样，反而如同20岁的年轻少年般！而这首歌也鼓舞了在场的所有新生，正如歌词中的“不负年少”一样，张杰也希望新生们能够发扬自己的朝气精神，追逐自己的理想！
