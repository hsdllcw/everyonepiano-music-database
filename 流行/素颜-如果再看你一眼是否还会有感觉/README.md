

如果再看你一眼，  
是否还会有感觉

这段歌词来自《 **素颜** 》，由许嵩与何曼婷共同演唱。轻松愉悦的曲调，唱出了一代人的青春，好听治愈！

看似简单的歌词，其实唱出了很多含义。歌曲讲述了一对男女同学从校园毕业，分手之后，他的不变和她的改变。两人的隔空对唱充满对青春岁月的回忆和怀念。

每个人都有青春，都会有回忆！经历过青春的酸，甜，苦，辣，才会成长为一个真正有意义的人。所以要珍惜青春，不要让它白白流走！
