

《 **给我一首歌的时间** 》是周杰伦每张专辑都必须要有的周式经典R&B情歌，而此曲也是继《稻香》后另一首周杰伦自己作词的歌曲，歌词是描写时下浪漫的爱情。

MV中，红砖墙的建筑古屋、酷酷的跑车“幽灵之子”，在现场袅袅炊烟气氛烘托下，一身嘻哈装扮的周杰伦跳着嘻哈舞步，还不时展露足下的滑步技巧。和这首R&B情歌非常相衬。而原本就感冒的周杰伦，为了要更有意境效果，不惜让洒水车把自己淋成落汤鸡。这支MV拍了两天，辛苦的拍摄工作甚至直到隔天的凌晨三点多才收工，真心不易啊。
