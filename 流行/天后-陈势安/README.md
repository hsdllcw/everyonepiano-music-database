

**天后双手简谱** 和五线谱完全对应。

_天后_ 是歌手陈势安演唱的歌曲，由彭学斌填词和谱曲，收录于陈势安2009年同名专辑《天后》。2010年11月22日这首歌获马来西亚娱协奖“娱协十大金曲”。

《天后》是疗伤失恋时最舒服的声音。歌曲荣登过各大歌曲排行榜，这首作品直到现在依旧成为了很多人在K房里点唱的心头好。

歌词下方是 _天后钢琴谱_ ，大家可以免费下载学习。
