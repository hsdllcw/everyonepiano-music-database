

**脆弱星球** 是由王薇演唱的一首歌，这首歌杨胖雨也翻唱过，并且在网络上有一定的传唱度，尤其是在抖音上。

这是一首由评论写成的歌，我也许会撞上流星，我的飞船可能失控在宇宙边际，但我冒着所有风险去见你，只想和你一起沉没，在这脆弱星球里。有人骂你野心勃勃，有人爱你灵魂有火，这个星球偶尔脆弱，而我也偶尔想与你沉没。简单的歌词却唱出了多少人内心的无助与落寞。

歌词下方是 _脆弱星球钢琴谱_ ，大家可以免费下载学习。
