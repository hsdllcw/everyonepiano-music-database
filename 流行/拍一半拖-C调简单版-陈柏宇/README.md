

**拍一半拖** 是中国香港歌手陈柏宇演唱的歌曲。

有着白马王子外表的陈柏宇在内里有的却是黑马王子的心，不想红，不想成巨星甚至只想过普通的生活，这才会让他一直在偏离白马的路线骑着黑马越走越远。陈柏宇的音乐故事并没有结束，只是不再如想象中的动听、美妙，一如《Tales》中的这般，很残酷，很写实却有很自我，这才是最应该被了解的陈柏宇式音乐。
