

**好好过** 是电视剧《风中奇缘》的插曲，由胡歌演唱。在胡歌生日（20日）前一天，片方很有心地发布了《好好过》的精制MV。

  
胡歌完成这首歌录制只用了很少的时间，将莫循的情感真切地注入其中：“现在有好多的歌曲都被过多地修饰，那这一次胡歌录《 _好好过_
》是以最真实的声音，我们没有经过过多的修饰，只为了保持最自然的一份真诚。

  
歌词下方是 _好好过钢琴谱_ ，希望大家喜欢。
