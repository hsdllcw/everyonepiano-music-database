

**雨幕** 是音乐人许嵩为《新天龙八部》端游创作并演唱的游戏主题曲。许氏中国风，词为魂曲为骨，整首歌丝丝入扣地将听众带入他一手构建的故事。

潇潇的雨幕里，三日无言的福船上，饮酒、听曲、赏雨、忆人，然而故事却在有人卧于绯泊之中时戛然而止，故事表象下的诸多隐喻也浮出水面。独此一家别无分号的叙事笔法，布局谋篇像本逻辑缜密的小小说，起承转合间充满戏剧张力。文趣盎然的遣词造句，既有新颖的意象与丰富动人的细节，也有言而未尽的留白与隐喻。作为《新天龙八部》的主题曲，全篇只字未描写武打争斗，却在看似绘景的潇潇雨幕里暗藏着汹涌伏笔，为作品平添了悬疑色彩。
