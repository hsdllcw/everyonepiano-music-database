

又是一首抖音爆火歌曲，名字叫做 **如果寂寞了**
，由人气歌手郑晓填演唱。其中有句歌词是：离开你心会很痛心会很痛一痛再痛。感情并不是只有甜甜蜜蜜，很多时候会有忧伤失望与伤感。

“如果因为寂寞，请别再来找我”，歌词朴实无华的描述了现实生活中绝大部分人对感情的态度。相爱是两个人真心在一起，而不是因为其他因素的将就。歌曲写实，引人共鸣。

歌词下方是 _如果寂寞了钢琴谱_ ，大家可以免费下载学习。
