

抖音“我坠入蛮荒荆棘的陷阱，曾一度坚信这就是爱情”出自歌曲《 **笑柄**
》，演唱者是陈小满。这首歌比较独特，利用自嘲的方式来抒发自己内心的不满，很能激起大家的共鸣。

淡淡的忧伤交杂着心酸铸成了前奏，随着时间推移，深深的无奈与懊悔融入歌声谱写出高潮，“我坠入”、“曾一度”、“当目睹”、“到最后”、“我退出”、“多羡慕”。短短的几个字却说出了世间最复杂事情的过程。没有完全合适的两个人，只有互相迁就的两颗心，初听不知曲中意
再听已成曲中人。
