

《 _越长大越孤单_
》这是艺人组合牛奶@咖啡在2008年发行的第二张中文专辑，共包含有11首歌曲。以真挚感性的一面重新面对听众，将成长过程中内心世界的细腻感受完整呈现在新专辑的全部11首曲目中，与逝去的懵懂青春挥手作别。同名主打歌曲也为《越长大越孤单》，另出版了与专辑同名的相关书籍《越长大越孤单》。  
  
《 _越长大越孤单_ 》获得的成就：  
  
在2008年度的牛奶@咖啡发展极为迅猛，推出备受各方好评的第二张专辑《越长大越孤单》；主打歌曲《越长大越孤单》迅速成为最受关注的流行金曲，并在中歌榜上榜达12周之久；远赴意大利威尼斯拍摄主打歌曲MV；进行了为期三周的上海、厦门、广州、长沙、武汉、成都、西安、郑州8个城市的巡回歌友会，各地媒体反响热烈，在每场演出过程中各地歌迷更是全场合唱，牛奶@咖啡已经成为目前国内最具人气及品质的音乐组合。
