_路过我耳朵的忧伤_
是中国内地女歌手苏运莹的一首翻唱单曲。苏运莹是个喜欢微笑，拥有创作天赋的可爱女孩。她在生活中是一个high点很低的女神经，她爽朗的笑声和大幅度的肢体语言则是其最真实的个性体现。苏运莹的自由勇敢让她的创作充满了无限的正能量，她是我们的小太阳。

此曲原唱是已故歌手覃桢，由潇彬作词、覃桢作曲。覃桢是中国内地流行乐男歌手、音乐制作人，2017年2月11日因过劳导致心肌梗塞离世，享年34岁。
