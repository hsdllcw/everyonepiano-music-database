

**朱砂泪** 全名为《朱砂泪•慕容紫英•韩菱纱》。该曲为仙剑同人原创歌曲，主题为“紫纱”，
改编自卓依婷2006年个人专辑《我的眼泪不为你说谎》同名主打歌曲。曲调古典婉转，又不乏一丝淡淡的忧伤，受到众多仙剑迷的喜爱。

_朱砂泪_
，原唱是夏一可。后经过好多人翻唱，其中翻唱比较出名的就是董贞的版本。董贞相信还是有很多人喜欢的一个歌手，声音甜美，唱的歌曲也真心不错。而董贞也是古风圈比较早的歌手，这首《朱砂泪》也是比较老了，很好听，属于一首改编填词的歌，可惜现在鲜有耳闻了，超级可惜啊。
