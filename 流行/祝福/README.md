

**祝福**
出自同名专辑《祝福》，是香港歌手张学友发行的第19张唱片，亦是他的第5张国语专辑，距离其上次发行的国语唱片《吻别》仅九个月。张学友曾数次在公开场合演唱过单曲《
_祝福_
》，包括其数个个人演唱会，赈灾义演的公益场合，在2000年的台北市新年演出中更是同现任中华民国总统，时任台北市市长的马英九合唱，而马英九当时亦表示《
_祝福_ 》是他最喜欢的中文流行音乐。
