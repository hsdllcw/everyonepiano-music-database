

该版本的《简单爱》很简单，双手简谱和五线谱是完全对应的，EOP教学曲，很适合新手练习。

简单爱是周杰伦第二张专辑《范特西》第三支歌曲，由周杰伦作曲、徐若瑄填词，歌词简单易懂、旋律朗朗上口，2001年9月14日发行。
