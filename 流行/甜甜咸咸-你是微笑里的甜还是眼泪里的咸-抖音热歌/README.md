谁不希望是可甜可咸的？牵念的东西越多，日子就越是需要调味剂来更改一下味道的。

无论你是微笑里的甜，还是眼泪里的咸，是夏天里的雪，还是白昼里的月，都让我无比想念。恋爱的味道在这个适合恋爱的季节，我们什么事都不要做，听它！爱它！在副歌优美的旋律中，快去找你的另一半吧，一起品味幸福。
