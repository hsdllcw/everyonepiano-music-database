

《 **配角** 》是音乐人周思涵的原创单曲。再次独自包揽词曲的她，给大家讲述了她/他在你的世界是如何做你的配角。

阿涵不想成为你的过客， 也不愿傻傻做你的备爱。这一次她只想心甘情愿做你的配角。而有时候配角并不卑微，她比主角还更有锋芒。

_配角双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！

同时，网站还为大家提供了《[ **失眠**](Music-10719-失眠-周思涵.html "失眠")》曲谱下载
