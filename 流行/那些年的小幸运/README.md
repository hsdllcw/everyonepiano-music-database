_那些年的小幸运_
是MICappella演唱的一首歌曲，该曲将《那些年》和《小幸运》结合了起来。MV上线不过5天，点击率就破百万，更有网友表示这是会让耳朵怀孕的声音。MICappella是来自新加坡国宝级人声乐团，同时也是深圳卫视第一届The
Sing-off清唱团的亚军。

值得一提的是麦克疯不仅创作了众多新作品，而且在曲风融入了多一些rap、重金属，再加上他们纯粹的、纯人声的、没有乐器遮掩瑕疵的独特演唱方式，相信会给内地的观众带来不一样的视听体验。
