

《经典咏流传》第二季中，经典传唱人谭维维以一首全新风格的《 **敕勒歌** 》震撼全场咩，成为当期节目所展示的多首咏唱歌曲中，收获观众投票点赞数最高的作品。

在经典原作基础上，被重新填词的新咏唱版《 _敕勒歌_
》随旋律缓缓流淌入耳，一曲结束，满场陶醉。一同随歌曲扑面而来的，是大草原辽阔壮美的气息。著名文化学者康震对新咏唱版《敕勒歌》赞美有加，他认为，这首歌唱出了《敕勒歌》苍茫雄浑的气质，为观众创造出无边无际的巨大想象空间。歌曲焠取了草原情怀和基因的精髓，有浓浓，辽远、忧伤、悲天闵人而又雄健豪迈的气质。歌者出神入化的演绎与器乐的点睛之笔，已达至臻之境。
