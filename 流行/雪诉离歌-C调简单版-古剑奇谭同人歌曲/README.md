

**雪诉离歌** 是单机游戏古剑奇谭同人歌曲。这是一首非常优美的古风歌曲，曲子节奏悠扬，感情丰富真挚，具有极强的感染力。原唱为梦岚与凌之轩。

《古剑奇谭》是由上海烛龙信息科技有限公司研发、北京网元圣唐娱乐科技有限公司发行的国产RPG游戏系列。游戏设定源自于《山海经》，以中国神话时代为背景。

_雪诉离歌双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
