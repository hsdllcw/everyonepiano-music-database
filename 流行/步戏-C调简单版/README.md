

**步戏** 原由国风原创音乐人五音Jw演唱，后在“春晖纪 • 2020国风音乐盛典”上，五音Jw与Assen捷重新演绎了此曲。

《 _步戏_
》也是一首非常惊艳的古风戏腔。《步戏》是解雨臣角色歌，就是《盗墓笔记》的花儿爷。“这番好戏一开腔”到“这番好戏已散场”，完全就是一段传奇。歌曲的戏腔在高潮部分，属于瞬间点燃全场的一段，超级好评。
