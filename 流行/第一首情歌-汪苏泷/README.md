

**第一首情歌** 是汪苏泷的一首歌，由汪苏泷自己作词作曲，收录在其2012年的专辑《万有引力》中，为该专辑的主打曲目之一。

而为了感谢歌迷一路以来的支持，美妙音乐斥巨资为汪苏泷量身打造同名主打歌《 _第一首情歌_
》的音乐微电影。首次“触电”的汪苏泷大胆挑战“姐弟恋”，”野蛮女友“的小野蛮小任性和最后的小感动，从帮助哥哥照顾他的野蛮女友到自己有一点心动，剧情结尾的处理更是引发歌迷无数遐想。

歌词下方是 _第一首情歌钢琴谱_ ，希望大家喜欢。
