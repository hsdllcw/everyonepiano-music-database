

**我的果汁分你一半** 是花儿乐队演唱的歌曲，由大张伟创作词曲。此曲看似歌词简单口水歌似的，其实饱含深意，都是哲理。

《 _我的果汁分你一半_
》是一首描写友情的歌曲。该曲延续了花儿乐队欢快的曲风，简洁直接地表达了分享、分担不同情感的快乐。该曲充满了积极乐观的内涵，和一样执着快乐的情绪。大张伟的词作简洁直接，没有多余的华丽修辞，短短几句道出清透无暇的简单心情，和朋友间分担失意的爱的宣言。

MV中，花儿乐队四个成员，乔装成“店小二”在客栈里面与各路英雄豪杰一试身手。在拍摄遇见“芙蓉妹妹”的一场戏时，导演要求他们做出夸张的吃惊表情，在反复拍摄这个镜头后，花儿乐队成员四人的脸部都已失去了知觉。
