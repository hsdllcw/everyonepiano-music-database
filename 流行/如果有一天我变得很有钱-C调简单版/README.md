

因为有一天查银行账户的时候发现钱挺少的，但是想花钱的地方又有很多，所以毛不易写了这首歌《 **如果有一天我变得很有钱** 》。

这首歌旋律简单，通俗，调式简单，符合广大听众的审美。但却写出了千千万万老百姓的愿望和心声。歌词大胆的假设了如果有一天我变得很有钱会发生什么。躺在世界上最大最软的沙发上表达了所有人美好的愿望，不想工作就能有花不完的钱。不用担心明天和离别真实地反映了世态炎凉，金钱是衡量友情的一切。买下所有笑脸，让所有可怜的孩子不再胆怯，让所有邪恶的人不在掌握话语权，反映了作者善良童真的一面，也侧面体现了在这个社会有了金钱能得到一切的现实。这首歌唱出了大部分人的心声，朴实但很真实，简单但不廉价。
