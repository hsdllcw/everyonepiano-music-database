

《 **雨落长安** 》是一部2014年开篇巨作神话剧《封神英雄榜》片尾曲。

  

该片《封神英雄榜》于2014年1月31日在各大卫视首播，以及《山东卫视》、《贵州卫视》、《湖北卫视》，2月2日在CCTV-1综合频道登映。

  

歌词下方是 _雨落长安钢琴谱_ ，欢迎大家使用。
