

**玫瑰三愿**
是中国著名歌唱艺术家李谷一演唱的歌曲，该曲由龙七填词，黄自谱曲。李谷一是建国后华语乐坛的开创性人物，在国内首倡通俗唱法，贡献之大甚至可与华语乐坛歌后邓丽君比肩。1988年，她被美国传记协会列入《世界杰出人物录》。

  
歌词下方是 _玫瑰三愿钢琴谱_ ，希望大家喜欢。
