

《我是歌手》2013年第2期，黄绮珊《离不开你》惊天逆袭。这首《离不开你》，原唱是刘欢，黄绮珊的演绎，把这首歌曲，以及自身的委屈、孤傲，不甘心，虔诚，激情和向往，都很好地包含在歌声中。

  
她就是一个天生的灵魂歌手，能将歌声向钉子一样扎在你心中最柔软的地方，挺疼，挺激动，更多的是无以伦比的震撼。她的嗓音的音域非常宽广，气息又出奇的浑厚绵长，低音部分清晰细腻，深远宁静，彷佛天外传来的歌声，又彷佛是从心里流出来的音乐，中音部分稳定，持久，精准，如同平滑的丝绸般令人舒畅，而高音的H到了C的高度，还能具有超常规的、出人意料的爆发力，能在这个高音区进行排山倒海般，激浪拍岸般的一波一波的冲击。

人到中年，1985年出道的黄绮珊，才赢来自己真正的春天，祝福她！
