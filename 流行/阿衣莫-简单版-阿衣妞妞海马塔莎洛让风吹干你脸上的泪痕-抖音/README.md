

“长夜未央的陪在我的身边，阿衣妞妞，海玛塔莎洛，让风吹干你脸上的泪痕，逝去的爱不必再留恋”，最近这首歌在抖音上特别的火。据悉，此曲叫做《 **阿衣莫**
》由阿吉太组合演唱。

“阿衣妞妞”可以理解为对一个女性的称呼。

“海玛塔莎”在彝语里的意思是不要伤心、不要难过。这首歌曲奔放自然，听起来有一种豪放不羁的感觉，在当下快节奏的时代里是一首不可多得多民族歌曲。

在国内，或者民族音乐世界，阿吉太组合是家喻户晓的歌手组合。在圈内阿吉太组合翻唱的音乐最具代表性，而这首歌和之前音乐风格截然不同，是成员吉子古且的处女座，从某种意义上讲，可能是自己青春所经历的感情，然而让吉子古且的气质形象唱功等使他成为大凉山彝族新生代音乐的代言人。
