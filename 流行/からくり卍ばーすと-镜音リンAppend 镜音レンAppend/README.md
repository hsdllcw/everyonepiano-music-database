

**からくり卍ばーすと双手简谱** 和五线谱完全对应。

  

《 _からくり卍ばーすと_ 》是日本虚拟偶像歌手镜音双子演唱的歌曲。

  

镜音铃、镜音连是CRYPTON FUTURE MEDIA以Yamaha的VOCALOID
2语音合成引擎为基础开发贩售的虚拟歌手软件角色主唱系列的第二作。于2007年12月27日发售，于2008年7月上旬发售更新版“act2”。

  

歌词下方是 _からくり卍ばーすと钢琴谱_ ，欢迎大家使用。
