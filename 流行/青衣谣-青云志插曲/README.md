_青衣谣_
是电视剧《青云志》的插曲，由周洁颖作词，谭旋作曲，郁可唯演唱。梦未央，怎能独自去云遥。谁今后再与我谈风笑。青衣谣是电视剧青云志中女主角碧瑶（赵丽颖饰）的人物主题曲，描写了凡瑶之间的生死痴恋。
