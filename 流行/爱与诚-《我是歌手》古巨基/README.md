

**爱与诚双手简谱** 和五线谱完全对应。

_爱与诚_ 是香港歌手古巨基发行的专辑《大雄》中的一首歌曲，由林夕作词，曹雪芬谱曲。该曲也是古巨基在《我是歌手》第三季的竞演歌曲。

歌词下方是 _爱与诚钢琴谱_ ，希望大家喜欢。
