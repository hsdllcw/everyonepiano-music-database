

**蝶** 是由日本女歌手天野月演唱的歌曲。天野月，旧名天野月子。有外号“日本爆音女歌手”之称。她所属公司为音仓唱片公司、Pony
Canyon。曾为著名PS2游戏《零～紅蝶～》及《零～刺青之聲～》唱作《蝶》及《声》。

  
歌词下方是 _蝶钢琴谱_ ，希望大家喜欢。
