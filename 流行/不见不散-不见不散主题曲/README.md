_不见不散_
是孙楠演唱的一首歌曲，由张和平作词，三宝作曲，收录到专辑《南极光》中，是电影《不见不散》主题曲。同时，此曲也获得了很多奖项，如西安音乐台98十大金曲奖，抚顺电台十金曲奖，“上海东方风云榜”十大金曲，“北京晚报”民意票选99年最喜欢歌曲第一名等等。

孙楠，1969年2月18日生于辽宁省大连市甘井子区，中国大陆男歌手、演员、配音、导演，任梦舟明星足球队队长。

歌词下方是 _不见不散钢琴谱_ ，希望大家喜欢。
