

五月天的《 **我不愿让你一个人** 》，相信大家都很熟悉了吧。

据悉，这首歌曲是以世界末日为主题而创作，向我们传递了如果没有明天和如果还有明天的两种感触。

歌曲前半部分深情温柔，副歌部分撕心裂肺，表露出无奈与些许感伤。在歌中，不仅有对爱情誓言的宣告，更有对活在当下珍惜眼前的人生态度。

2013年，《 _我不愿让你一个人_ 》获得台湾Hit-Fm年度百大单曲冠军和第八届KKBOX数位音乐风云榜 “年度百大单曲”冠军。
