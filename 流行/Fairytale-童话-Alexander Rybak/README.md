《 **Fairytale** 》，中文《童话》，是白罗斯裔挪威歌手Alexander
Rybak于2009年1月12日发行的单曲。歌曲的词曲皆由Alexander Rybak创作，是首流行曲风的民间音乐。

2009年，Rybak参加2009年挪威一级音乐大赛，以747,888的高分打破大赛纪录成为当届冠军代表挪威参加在俄罗斯莫斯科举办的2009年欧洲歌唱大赛。最后《
_Fairytale_ 》以387分获得比赛冠军。

《Fairytale》获得了极大的成功，除了获得挪威一级音乐大赛与欧洲歌唱大赛冠军，还获得2009年马塞尔•贝桑松奖“报界奖”以及“欧视爱好者总会（OGAE）”评选最喜欢的作品。随着比赛的成功，歌曲打进多国单曲榜前十名，获得比利时、丹麦、芬兰、希腊、挪威、俄罗斯、瑞典等国音乐排行榜冠军位置。在挪威，歌曲停留在世界之路榜单长达22周，蝉联8周冠军。
