

**给你们双手简谱** 和五线谱完全对应。

_  
给你们_ 是张宇《雨一直下》专辑中的曲子，由于歌词中表达的情感，该歌曲在网络及婚礼现场亮相率极高。

  
张宇，本名张博翔，台湾男歌手兼音乐创作人。出生于台北市，毕业于逢甲大学银行保险系，擅长钢琴及吉他。于学生时代在民歌餐厅驻唱。1992年，经过凡人二重唱（莫凡、袁惟仁）的引荐，被大滚唱片的制作人蔡宗政发掘而正式出道。并于1993年4月1日发行了第一张专辑《走路有风》。令他真正走红的是同年11月发行的第二张专辑《用心良苦》，同名主打歌《用心良苦》一时在传唱，可谓张宇最重要的作品。

  
歌词下方是 _给你们钢琴谱_ ，希望大家喜欢。
