

抖音“我只想要拉住流年，好好地说声再见”是来自《 **纪念** 》的一句歌词，此曲由RAiNBOW计划与雷雨心共同演唱。

每到一年一度的毕业季，也正是初夏闷热的时节，夹杂着毕业生分离的感伤与对未来的憧憬。透过点滴词句，我们可以看到她们在忙碌学习，中考在即，离别在即，铭记和纪念的心很真诚。也正是为此，《
_记念_ 》的原创歌曲MV，在网络上引起众多刚刚参加完中高考学子的共鸣，从而走红。
