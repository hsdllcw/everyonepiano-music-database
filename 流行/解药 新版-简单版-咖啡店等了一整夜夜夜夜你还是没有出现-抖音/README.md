

咖啡店等了一整夜夜夜夜  
你还是没有出现  

最近，小编在刷抖音的时候总是能听到这段旋律，看歌词不难得出——这是一首伤感情歌。

歌曲名字叫做《解药 (新版)》，由来一碗老于演唱，发行于2021年7月23日。

回想一下，你有没有做过这样的啥事：  
在某个场所，苦苦等待一个不会出现的人  
独自一个人在城市中游荡  
闭上双眼，不停回想过往的点点滴滴

而这正是歌曲中表述的场景！虽然感情受挫，但小编想说的是：人生苦短，何必这么难为自己呢？天涯何处无芳草，何必单恋一枝花。更何况，这年代，有人恋爱，有人出轨，有人从头寡到尾（大麦客就是寡的那一位），擦，突然发现大老师最近总是锅从天上来，无端端受连累！

其实，在爱情中，时间不能作为解药，但是解药却在时间的长河当中。此曲用歌词传达失落的情感，极易触动听者内心深处的情感，也正如此，才大受欢迎！所以小编特意制作了
