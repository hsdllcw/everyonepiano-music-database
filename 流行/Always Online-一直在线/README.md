

期盼已久的音乐才子林俊杰第六张大碟即将发行，其公司发起“快乐上碟”主题活动，活动主题歌曲“《 **always online** 》将收录在专辑里。《
_always online_ 》讲得是对爱情的坚持。让我们微笑着面对人生、面对世界，让爱心与感动通过微笑脉脉相传。
