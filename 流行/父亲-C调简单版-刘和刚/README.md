

《 **父亲** 》是车行填词，戚建波作曲，刘和刚深情演唱给父亲的歌曲。父母在，家就在，这首《父亲》唱到每个儿女心里。

父爱深沉，不易察觉；父爱不语，不会表达。刘和刚用歌声倾述对父亲的爱，对父亲的感恩，他的情感真挚，很有穿透力，听他的歌很有共鸣和震撼，让人不知不觉就到了歌的意境，叩响自己心灵深处思想亲人的情结。
