

抖音最近有首歌很好听，歌词是“你是我触碰不到的风，醒不来的梦，寻不到的天堂，医不好的痛”。据悉此曲叫做《醒不来的梦》，是由伍威编曲，秦飞琼作曲，曹陌迁作词，回小仙演唱的一首歌曲，歌曲发行于2020年9月29日。

歌曲表达的是如果时光可以回到从前，回到我们相遇的那一天，该多好。可所思所想竟已成遥不可及的梦，找不到那只撞破梦到蝴蝶，我是永远困在回忆与梦境中的受刑者。那些爱那些恨，感受真切却又恍如隔世般，令人难过。

我相信，喜欢这首歌的你，心中也有一个遗憾，一个放不下，忘不了，爱而不得的人。但是，生活总要继续，没有谁能够停留在昨天。

《 _醒不来的梦_ 》曲谱由 **EOP** 大佬[
**M小冯哒**](https://www.everyonepiano.cn/user-174829.html)给大家编配，曲子有一定难度，学过《[
**魔鬼训练营**](/Sale.html)》的小伙伴，一定要勤加练习哦！
