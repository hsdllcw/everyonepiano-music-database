

《 **有点甜** 》是汪苏泷的首次与超萌双胞胎团体By2合作。是游戏《萌三国》的主题曲，也是电视剧《微微一笑很倾城》的插曲。

歌曲延续对唱情歌的甜蜜与暧昧情愫，展现更多层次的声线变化。姐姐Miko的声音有power，妹妹Yumi的声音感觉细腻，更尝试从没过的一男两女的对唱方式。这也是汪苏泷希望在音乐上能带给大家新的听觉感受，KTV情歌对唱新选择，我们的小小爱情才要开始。
