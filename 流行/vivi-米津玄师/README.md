

《 **vivi** 》是日本音乐人米津玄师创作的作品。

ハチ【米津玄师】（mylist/12682175）早期niconico上活跃的音乐人，现已转战自作自唱歌曲创作从2009年至2011年投稿VOCALOID作品
此后从2012年02月20日起开始以「米津玄师」的名字投稿自作自唱作品。

歌词下方是 _vivi钢琴谱_ ，欢迎大家使用。
