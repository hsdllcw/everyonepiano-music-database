

《 **背对背拥抱**
》是由林怡凤作词，林俊杰作曲，林俊杰演唱的一首歌曲，收录在专辑《JJ林俊杰100天》中。这首歌是一首流行风格的歌曲。歌曲展现了林俊杰的高超唱功和创作才能。
