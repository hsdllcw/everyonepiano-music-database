

**那片海双手简谱** 和五线谱完全对应。

_那片海_ 是一首选自韩红专辑《醒了》的国语歌曲，由韩红作词、作曲并演唱。在2001年5月1日由麒麟童唱片出版并发行，同时《那片海》有几个版本。

《那片海》是《风雨中的美丽》之后又一首精心炮制的流行佳作，在同样执着的情感和动人的旋律下是韩红日渐成熟和深邃的内心世界，如果有机会一定要听一下韩红的钢琴自弹自唱版本，技巧虽不完美但别具韵味。
