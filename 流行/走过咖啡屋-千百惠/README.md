

总有一些经典的歌曲，无论是以前还是现在一直被传唱。而《 **走过咖啡屋** 》就是这样一首经典之作。对于70、80年代的人都很熟悉，随口就能唱上两句。

这首歌被称为经典，一直流传到现在，去KTV里面依然有很多献唱。

而当年演唱这首歌的人叫做千百惠。这首歌发行之后，很快获得了观众的喜欢。而且这首歌在当时流传在整个大街小巷。一时间千百惠凭借着这首歌打开了名气，让自己彻底爆红。歌词描述了情人在之前相爱在咖啡屋，现在分开，作者通过歌曲表达了对情人的不舍和思念。

歌词下方是 _走过咖啡屋钢琴谱_ ，大家可以免费下载学习。
