_身骑白马_
是华语女歌手徐佳莹演唱的歌曲，由徐佳莹及其老师苏通达共同创作。并收录在徐佳莹首张专辑《LALA创作专辑》，其最大的特点是改编自台湾的民间剧种歌仔戏。

这首歌曲呈现出徐佳莹丰沛的创作才能与想象力，充满勇敢与坚持力量的《身骑白马》实为佳作。此曲有种倔强，敢爱敢恨的情绪在里面。所以在编曲上的变化对于情绪的表达可谓一拍即合。可谓是颠覆与延续碰撞出的华美乐章。
