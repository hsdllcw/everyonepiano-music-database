

**陌上花开** 是2016年天涯明月刀OL不删档测试一周年纪念曲, 由焰31填词，陈鹏杰谱曲并且编曲，小爱的妈演唱。

《天涯明月刀》是腾讯北极光工作室研发的一款武侠题材的3D大型多人在线角色扮演电脑客户端游戏，由《天涯明月刀》小说改编而成。该作有丰富的社交系统，在游戏中玩家可以拜师和收徒，可以参加帮会，和帮会成员参加任务活动，还可以加入联盟、盟会，体验不同阵营之间的战斗。
