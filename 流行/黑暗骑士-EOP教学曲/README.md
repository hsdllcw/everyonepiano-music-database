

**黑暗骑士双手简谱** 和五线谱完全对应，属于EOP教学曲。  
  
「黑暗骑士」收录于林俊杰2013年发行的专辑《因你而在 Stories
Untold》中，由林俊杰作曲，阿信作词并跨刀拍摄MV，由新锐导演比尔贾执导。林俊杰扮演心魔，而阿信扮演把他救回来的力量。一身大红色装扮，让两位音乐才子以英雄之姿，在歌曲内激励不一样的火花。JJ林俊杰说，「黑暗骑士」这首歌，舞曲混搭摇滚，歌词点出了社会现象，是一首激励人心的歌曲。  
  
『新四大天王』林俊杰在3月发行出道10年的第10张专辑【因你而在】，JJ说：「人生就这么一个10周年纪念日，想为自己留下值得纪念的事！」所以找来期盼合作已久的『亚洲天团』五月天来跨刀合作，一起创作出这首新歌『黑暗骑士』！  
  
主打歌『黑暗骑士』不仅有阿信跟JJ合唱还拍摄MV，五月天其他团员更是为这首歌弹奏乐器，JJ说：「我喜欢的不单单是阿信写的词，更希望有五月天每个团员的声音在这首歌里，真的很谢谢『亚洲天团』五月天帮我圆了这个梦，也帮我一起庆祝这10周年！」今天五月天派出阿信与JJ合拍MV，阿信并饰演最重要的『黑暗骑士』角色，JJ在MV中饰演忧郁症的男主角，数度走不出人生的关卡，最后在『黑暗骑士』阿信的拯救下看见人生曙光，JJ甚至因此拯救了更多和他一样曾经不快乐的人！象征着音乐就是黑暗骑士，用音乐拯救世界！
