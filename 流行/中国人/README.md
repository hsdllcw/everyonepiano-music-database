_  
中国人_ 是由李安修作词，陈耀川作曲，涂惠元、江港生编曲，刘德华演唱，收录于1997年刘德华专辑《爱如此神奇》，也是刘德华歌曲的代表作之一。  
  
流行乐坛有很多排行榜，但主旋律歌曲的评选榜单则很少见。在2009年中宣部曾公布了一份“100首爱国歌曲”排行榜，入选的歌曲内容涵盖了革命、建设、改革的各个时期，都是历年来在国内相当有影响力的主旋律作品。而刘德华的《中国人》也入选其中。  
  
此曲也是小编特别喜爱的歌曲之一，高亢的旋律，每每听后都牵动着无数中国儿女的心。  
  
歌词下方是 _中国人钢琴谱_ ，希望大家喜欢。
