

**彩虹节拍双手简谱** 和五线谱完全对应。

_彩虹节拍_ 是一首由李懋扬(T2o)作词作曲，肥皂菌、易言、音阙诗听/赵方婧、西瓜Kune演唱的一首歌曲。同时也是2020年拜年祭插曲。

歌词下方是 _彩虹节拍钢琴谱_ ，大家可以免费下载学习。
