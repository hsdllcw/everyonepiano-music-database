

向来以情歌见长的李宗盛，在知天命之年用一首《 **山丘**
》打动了所有听者的内心。心若在，歌就在。歌里不只有对往昔的哀叹，它更多的，是李宗盛对待感情和人生的崭新态度。

一曲《 _山丘_
》李宗盛写了整整十年，旋律易成，填词却难。从2003年到2013年，十年的时间跨度，也是李宗盛从“不惑”到“天命”的人生感悟。然而，当须发花白的老李一开口，唱的却是“想说却还没说的，还很多”。这一段人生的旅程，他本想喋喋不休，把岁月过往都说得饱满，话到嘴边，却只剩“越过山丘，才发现无人等候”的感伤和惆怅。
