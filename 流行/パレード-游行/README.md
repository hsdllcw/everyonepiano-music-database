《 **パレード** 》（游行）是Yorushika（ヨルシカ）演唱的一首歌曲，由n-buna作词、作曲。

Yorushika是由作曲家n-buna与歌手suis组成的日本摇滚乐团。于2017年组成，出道时所属的音乐厂牌为Dwango旗下的U&R
records，目前所属的音乐厂牌为日本环球音乐。2019年6月14日以单曲《心上破了洞》（心に穴が空いた）自日本环球音乐主流出道。
