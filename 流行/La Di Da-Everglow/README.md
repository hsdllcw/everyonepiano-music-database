

**La Di Da**
是女子演唱组合EVERGLOW演唱的歌曲，收录在其第二张迷你专辑《-77.82X-78.29》中。组合，由朴智媛（E:U）、金施贤（SIHYEON）、
韩恩智（MIA）、王怡人（YIREN）、许庾琳（AISHA）、曹世林（ONDA）6人组成，她们凭借独特的舞台魅力掀起了一股海内外翻跳风潮。
