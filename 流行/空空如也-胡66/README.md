

**空空如也双手简谱** 和五线谱完全对应。

_空空如也_ 是胡66演唱的歌曲，由雪无影作词、作曲，于2017年12月14日发行。

此曲不复杂，却有抓耳的旋律。而胡66的演唱没有小女生的柔弱，反而带一点浑厚的成熟韵味。

两个人能否走在一起，时机很重要。你出现在他想要安定的时候，那么你的胜算很大，你出现在他对这个世界充满了好奇的时候，那么就算你多美多优秀都是徒劳无用。女孩从素颜坦诚变得精致漂亮，正因为那些情感宣泄的出口其实并不只有流眼泪，音乐是一剂慰藉心灵的良药，不妨尝试下用音乐来舒缓下自己的情绪，汲取养分，然后大步往前走。
