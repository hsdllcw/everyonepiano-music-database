

2020年6月29日，由周深演唱，方文山作词，陈致逸作曲的歌曲《 **花西子**
》上线QQ音乐，华丽升级东方美学之路。这首歌惟妙惟肖地描绘了一场梦境，以自然的手笔出现在观众的眼前，仿佛画卷，虽说是一首歌，却胜却画卷。

加上周深唯美的声音，他开口的瞬间才是这歌词赋予灵魂，成为有生命的音乐的要素，每一句，仿佛琴声传入，让人不禁沉醉。

可以说《 _花西子_
》这首歌，从一定程度上来说，还原了音乐的古典之美。其风格十分独特，静中有动，让人安静聆听的同时，仿佛灵魂被牵引而感觉不到单调，这才是真正的古典美。
