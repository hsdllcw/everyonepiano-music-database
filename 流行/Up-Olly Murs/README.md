

**Up双手简谱** 和五线谱完全对应。

Murs和美国青年歌手Demi Lovato合作演唱。该歌曲是《Never Been
Better》的第二支单曲，2014年12月1日正式打单，歌曲MV已于2014年12月11日放出。

歌词下方是 _Up钢琴谱_ ，希望大家喜欢。
