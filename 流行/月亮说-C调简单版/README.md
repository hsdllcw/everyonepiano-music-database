

2014年Back To Priscilla三十周年演唱会上，陈慧娴翻唱了这首《 **月亮说**
》，歌曲中的内容让人联想，舞台闪耀星光下，等待爱情的陈慧娴唱出了别样的味道。

陈慧娴是一位实力兼偶像派歌手，以清晰而充满感情的唱腔著称。她有着纯净舒服的音色与声线，其歌声有着慰藉心灵的作用，因此她又被人称为“疗伤歌手”。陈慧娴独特的磁性音色、字正腔圆的粤语咬字发音以及极具天赋的唱功都深深地印在听众的脑海里。
