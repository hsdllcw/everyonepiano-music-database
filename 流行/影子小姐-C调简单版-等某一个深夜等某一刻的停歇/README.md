

“等某一个深夜，等某一刻的停歇”出自歌曲《 **影子小姐** 》是由封茗囧菌演唱的一首歌曲，发行于2018年7月29日。

歌曲通过“影子”来描述爱情中的一种状态。爱情像是影子，当你忽略它的存在时，它总是形影不离！有时候，爱情总会以一种你不清楚，甚至也不了解的方式出现在你的身边，总是扮演着一个默默无闻的角色。往往这样的爱情是谦卑的，是很容易被忽略的。但这样的爱情一定是真实的，是值得你一生去守护和珍惜的爱情，希望你可以早点发现。
