

**别惹哭我双手简谱** 和五线谱完全对应。

_别惹哭我_ 是台湾歌手郭静演唱的歌曲，由林建良填词、阿超谱曲，是电视剧《青丘狐传说》的插曲。

《青丘狐传说》（又名《仙狐传奇》）是取自于《聊斋志异》中的《封三娘》、《阿绣》、《婴宁》、《胡四》、《长亭》、《恒娘》六个故事进行改编的古装单元电视剧。该剧以狐仙喻人，讲述了在“存天理，灭人欲”的炎凉世态下，至情至性的“狐们”在爱欲悲欢中的浮沉纠葛。

歌词下方是 _别惹哭我钢琴谱_ ，希望大家喜欢。
