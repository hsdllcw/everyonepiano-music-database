_アイロニ_
，中文译为反语，是由すこっぷ使用初音ミクAppend（Soft）创作的歌曲，于2012年5月24日上传于NICONICO，后被许多唱见翻唱，如majiko。majiko(まじ娘)，日本创作型女歌手，她用能唤起听众情感共鸣的嗓音，作为一个能鼓舞人心的女唱见在网络上日益活跃。

歌词下方是 _アイロニ钢琴谱_ ，大家可以免费下载学习。
