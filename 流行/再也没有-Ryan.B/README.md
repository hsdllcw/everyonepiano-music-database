

**再也没有双手简谱** 和五线谱完全对应。

_再也没有_ 是Ryan.B创作的一首慢板情歌，整首歌充满了悲伤的色彩，Ryan.B的声音如泣如诉、引人泪下。

歌曲更是力邀好友（AY杨佬叁）加入，两人声音的配合天衣无缝，这首歌也是ONO音乐集团推出的第一首慢板歌曲，也是国内乐坛不可多得的虐心情歌佳作。

歌词下方是 _再也没有钢琴谱_ ，大家可以免费下载学习。
