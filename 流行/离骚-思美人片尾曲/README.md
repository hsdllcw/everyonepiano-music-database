_离骚_ 是易烊千玺于2017年4月21日发行的单曲。该曲目为电视剧《思美人》片尾曲，由金玟岐作词作曲，颜小健操刀制作。

《离骚》浓浓的中国风曲调搭配《思美人》剧中饰演小灵均的易烊千玺充满磁性的嗓音，细腻动人百转千回。易烊千玺用深沉婉转的千式苏音娓娓道来，委婉动听的旋律展现出屈原孤芳自赏心系天下的忧思。
