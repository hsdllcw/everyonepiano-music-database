

“总有人觉得你过得体面，却不曾问问你都经历什么”出自歌曲《 **人生如歌**
》，由海来阿木演唱，发行于2021年4月10日。歌曲动听，情真意切，富有人生沧桑，让人感慨万千！

可以说，海来阿木把前半生的所有成长经历全放在了这首《 _人生如歌_
》里，过去的岁月里，他凭借自己的音乐实力，一次次地在歌坛创造着奇迹，海来阿木就如同一匹黑马，仅仅用半年的时间，横扫歌坛，成为了公认的主流明星，其火箭般的攀升加速度，令业界惊叹！

海来阿木说，想为那些只是为了过的好些而透支年轻身体的人们唱首歌，谁不想要每天都感受家的温暖，可他们永远只能在清晨醒来望着自己七零八落的样子，收拾一下继续出发，海来阿木希望用自己的歌声，唤起大家的共鸣，多给拼搏的人一点爱和关心！
