

**想唱就唱双手简谱** 和五线谱完全对应。

_  
想唱就唱_
是张含韵于2005年推出第一张专辑《我很张含韵》中的曲目。曲风欢快，朗朗上口，后有多位歌手翻唱。这首歌创作于2005年5月，因歌词励志色彩浓厚，曲调朗朗上口而风靡一时，成为选秀主题曲的巅峰之作，至今仍传唱不衰。

  
歌词下方是 _想唱就唱钢琴谱_ ，希望大家喜欢。
