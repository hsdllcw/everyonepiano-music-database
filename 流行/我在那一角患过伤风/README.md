这是一首有很奇怪名字的歌，出自一张叫做《只能谈情，不能说爱》的特殊唱片。之所以说《只能谈情，不能说爱》是一张特殊唱片，是因为它是一本同名小说的配乐概念唱片——这是一个新名词。  
  
本来《只能谈情不能说爱》在香港出版，购买者大多是作者的支持者，作品知名度不高，并没有吸引到内地同胞的注意。但是沉寂一两年后，冯曦妤的《我在那一角落患过伤风》逐渐在内地通过互联网广泛传播，深受大众的喜爱，2009年更成为步步高音乐手机的电视广告主题曲，在湖南卫视高频率地播出，使歌曲知名度更上一层楼。而由于网络上对歌曲出处的错误解读，使原作和主题曲至今依然默默无闻。
