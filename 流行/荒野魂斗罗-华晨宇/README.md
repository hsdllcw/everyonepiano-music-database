是华晨宇和自己的好友在《歌手•当打之年》演唱的曲目。虽然在乐器上只用了一把吉他，却用这极简的方式造就了极具震撼的舞台效果。两个“中二”少年，戴着略显时代感傻傻气息的墨镜，就坐在那里，上演了一段无畏的荒野之战。
