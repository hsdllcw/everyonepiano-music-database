_等一分钟_ 是徐誉滕演唱的一首流行歌曲，由徐誉滕填词、谱曲、制作，发行于2007年3月2日。收录于徐誉滕2008年10月22日发行的专辑《滕•爱》中。

歌曲MV根据真人故事改编，讲述了一个男孩和女孩相爱了。男孩很爱女孩，女孩很漂亮，男孩突然有一天对女孩说：我要走了云，去我一直想去的地方美国。有一天男孩跑到女孩工作的画室去找她，男孩拿着女孩最喜爱的花，可是当他开门的那一刻他整个人都傻了，女孩正裸着身子，做别人的模特。男孩跑开了，想躲避这一切，可是女孩追上来了，听我说好么，可是男孩打了女孩一巴掌，不要脸。女孩：听我说好么就一分钟，给我一分钟。男孩走了，去了他一直想去的地方美国。女孩去送他，女孩说：求你别走，严。男：我不想在看到你。女：为什么，为什么就不能为我停留一分钟？我的心好痛，严你知道吗？男孩在国外突然收到女孩的信，严，我走了，带着我的遗憾走了，这些钱是在你出国前，我做模特为你攒的费用，它不脏。永远爱你的云。男孩真的后悔了，想起女孩的时候对着女孩的相片哭泣。可是这一切都晚了。不能在挽回了。
