

《 **哭泣的月亮** 》，越南语《Vang Trang Khoc》，由越南著名青年音乐家阮文钟（Nguyen Van
Chung）创作，被誉为2007年最伤感的网络歌曲，荣获2007年冠军曲，在东南亚七国、中国、美国（主要越裔苗裔等）都有翻唱。

中文翻唱版为七月七组合的《月亮海》。

歌词下方是 _哭泣的月亮钢琴谱_ ，大家可以免费下载学习。
