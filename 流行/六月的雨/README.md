

六月的雨是电视剧《仙剑奇侠传》的插曲，是专门为剧中女一号灵儿所写，由男主角胡歌倾情演唱。 胡歌用李逍遥的感觉，唱出了对爱的等待，对灵儿的爱和内心的寂寞。

从这首歌，就可以想起逍遥和灵儿曲折永恒的爱情，想起逍遥为爱而等待的痛苦，想起灵儿的痴情和付出……
一句句，都是逍遥对灵儿的表白，执着与痴情让我们体会到爱情的真谛。融入电视剧，就更加显得浪漫温馨。
