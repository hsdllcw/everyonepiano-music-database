

蒋婴2019推出流行单曲《 **酒醉的雨滴**
》由杰哥哥/思雨词曲创作，上海雨陌文化传媒有限公司推广发行。感人的歌词，加上醉人的歌喉和美妙旋律，无形中唱出了女人的痴情！

“漫长的黑夜里，寒风又呼呼刮起，独自落下伤心的泪，伤心落泪都是为了你，酒醉的雨滴把我的回忆勾起”，好伤感！生命中，有太多的事情身不由己，有太多的无奈心不得已。看不透的伪装，正如猜不透的人心。弄不明的感情，正如读不懂的心灵。与其多心，不如少根筋；与其红了眼眶，不如笑着原谅。人生一世，糊涂难得，难得糊涂。活得太清楚，才是最大的不明白。人知足就会快乐，心简单就会幸福。

《 _酒醉的雨滴_ 》曲谱由 **EOP** 大佬[
**M小冯哒**](https://www.everyonepiano.cn/user-174829.html)给大家编配，曲子有些难度，学过《[
**魔鬼训练营**](/Sale.html)》的小伙伴，要勤加练习哦！
