

《 **我曾用心爱着你**
》是潘美辰的一首经典歌曲，1988年9月发行，收录于她的个人专辑《不要走不要走》中。这是一首激情励志、波动情感的国语情歌，曾在港台大陆引起一时轰动。

歌词描述了一段感情的历史，是说两个人在经历相同的感情后有着不同的感受。“你总是如此如此如此的冷漠，我却是多么多么多么的寂寞”一语道破爱情结束后所产生的敌对，但是从整首歌曲中理解，其实并没有任何的敌对关系，反而更多的是在乎。我想只有歌手本身才能够体会到歌曲中那份真情所在的爱情。也许对于爱情，潘美辰有自己的见解，才会把歌曲表达的如此完整。
