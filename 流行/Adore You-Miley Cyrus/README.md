

**Adore You** 是美国新生代女演员兼女创作歌手Miley Cyrus的歌曲。  
  
当很多人还在纠结Adore You和SMS哪个是Bangerz的三单时 ，Miley Cyrus突然公布 _Adore You_
确定是Bangerz的三单在1.6派台，于13年12月17号开始派主流pop台，正式MV将于12月26号首播。
