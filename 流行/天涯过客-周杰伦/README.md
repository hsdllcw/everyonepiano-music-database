

**天涯过客**
是周杰伦全新创作专辑中的歌曲，由周杰伦作曲。经典周杰伦式的中国风，搭上周杰伦绝美如诗的歌词，以各种景写情，将思念之情、爱情的遗憾、等待、各种细腻的情绪写得含蓄却又丝丝入扣。

  
歌词下方是 _天涯过客钢琴谱_ ，希望大家喜欢。
