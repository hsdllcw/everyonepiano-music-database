

**拜无忧双手简谱** 和五线谱完全对应。

_拜无忧_ 为萧忆情Alex演唱的歌曲，收录在萧忆情首张个人原创专辑《萧音弥漫》中，专辑于2017年2月18日正式发行。且此曲获得2017年国风音乐年度大赏
“最佳曲作”奖项。

萧忆情Alex，男，湖南郴州人，微博音乐人，中国原创音乐基地推荐歌手，因其独特的嗓音和风格受到了广大粉丝的喜爱。
