

《 **这个世界** 》由蔡蓝钦词曲作品，在蔡蓝钦身后发行。收录在1987年专辑《这个世界》中，由飞碟唱片发行。

“在这个世界，有一点希望，有一点失望，我时常这么想。在这个世界，有一点欢乐，有一点悲伤，谁也无法逃开。我们的世界，并不像你说的真有那么坏，你又何必感慨。用你的关怀和所有的爱，为这个世界，添一些美丽色彩。”短短几句词，却唱得那么深刻。对于当下身处环境的乐观，对于未来更美好生活的无限憧憬，你都可以听到蔡蓝钦用那么真诚的声音在唱着。

这个世界有太多的无可挽留，有太多的对失去才华的痛惜。庆幸的是，我们还可以在他们留下的音乐里，继续着自己的青春，或者继续回味着自己的青春。蔡蓝钦的《
_这个世界_ 》，希望给你一些鼓励，一些勇气。

歌词下方是 _这个世界钢琴谱_ ，大家可以免费下载学习。
