

《 **说好了不见面**
》是小贱中板情歌清新二部曲温柔发送，你睡醒之后……我们说好了不见面。秋高气爽，蓝天白云，春夏秋冬，瞬息万变。如爱情，如人生。还好季节的变换。

作者用这首歌表达对爱人的思念与无奈，这首歌处处都会流露出思念的情感，这也正是这首歌的意境所在 。
