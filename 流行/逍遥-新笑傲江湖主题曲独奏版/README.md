

《 **逍遥** 》是2013年版《笑傲江湖》的主题曲，由剧中男主角霍建华演唱。  
  
2013年版《笑傲江湖》由于正工作室与华夏视听环球传媒集团有限公司等单位共同出品，霍建华、陈乔恩、袁姗姗、陈晓、杨蓉领衔主演，胡意涓和黄俊文执导。故事讲述华山派大弟子令狐冲，生性豁达不羁。偶遇剑宗高人风清扬授以“独孤九剑”，又意外尽得五岳各派剑法精髓，引致师父岳不群猜疑，藉口逐出师门。不群内里奸险非常，为独得《葵花宝典》，竟设计令武林各派自相残杀，唯最终作法自毙。而令狐冲最终则辗转练成“吸星大法”，成为恒山派掌门。本剧于2013年2月6日起在湖南卫视首播。  
  
霍建华，台湾著名演员。年少时热爱唱歌，2002年为了得到演唱电视原声带的机会接下第一部电视剧《摘星》，正式踏入演艺圈。2004年开始转往内地发展。荧幕形象多变，戏路宽广。出道至今主要作品有《海豚湾恋人》《千金百分百》《天下第一》《屋顶上的绿宝石》《胭脂雪》《仙剑奇侠传三》《新一剪梅》《一一向前冲》《唐琅探案》《怪侠一枝梅》《倾世皇妃》《笑傲江湖》等。

### 逍遥的歌词：

**演唱：霍建华**

岁月催人老，名利都忘掉

一壶浊酒把梦醉倒

生死也寂寥，贪一个拥抱

管他迟与早，放爱去逍遥

天那么高，两岸青山围绕

尝遍人间多少味道

爱恨滔滔，全都一笔勾销

只想好好把握今朝

你那么好，我用一生祈祷

带着你江湖里逍遥

只要心还跳，就有我逗你笑

牵着你慢慢变老
