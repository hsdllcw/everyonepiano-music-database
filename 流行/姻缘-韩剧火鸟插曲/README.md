

**姻缘** 是由李承哲演唱的。李承哲 Lee Seung Chul ，韩国流行音乐歌手，属于复活这个音乐组合。

另外，《 _姻缘_ 》是韩剧《火鸟》中的插曲。

_姻缘_ ，也许你听不懂它的涵义，但静静听它的旋律，听它哀怨的曲调也不失为一种享受。
