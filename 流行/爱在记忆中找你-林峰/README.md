

《爱在记忆中找你》是歌手林峰的首张粤语大碟，该唱片于2007年11月23日由英皇娱乐有限公司发行，发行3周即获IFPI金唱片销量，并受封当年的「销量新人王」。同名单曲《爱在记忆中找你》为该碟主打歌。
