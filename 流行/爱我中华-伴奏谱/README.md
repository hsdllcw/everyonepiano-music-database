

《 **爱我中华** 》由乔羽作词、徐沛东作曲，最初由歌手韦唯演唱。后宋祖英将此曲唱红。

《 _爱我中华_
》的歌词,表达了强烈的民族情、爱国情。我国是多民族的社会主义国家,实行的是民族自治制度,大杂居小聚居是我国少数民族分布的特点。各民族在中国共产党的领导下建设社会主义,各民族团结起来共同发展、共同繁荣,体现了中华民族的崛起与中国人民对祖国强烈的热爱和赞美之情,也表达了作者的爱国之心。
