

**走马双手简谱** 和五线谱完全对应。

_走马_ 为陈粒首张专辑《如也》中的曲目。

专辑发行时间为2015年2月2日。歌曲《走马》清新靓丽，鉴定了陈粒演唱的曲风，形成自己的风格。有人认为，这是一种抽象式的名字, 歌词与名字并没有多大的关联,
且歌词多是口语化,
更多的是表达一种状态与态度。还有人认为这是一段走出单相思的故事。鼓起勇气去追自己喜欢的人，但是对方没有任何反映，然后自己归结自己辨析，最后总选择放弃。

歌词下方是 _走马钢琴谱_ ，大家可以免费下载学习。
