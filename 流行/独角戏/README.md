《 _独角戏_ 》是台湾女歌手许茹芸的代表歌曲。所属专辑《如果云知道》。  
  
你爱的他一直不忘往昔之情，你的情敌叫做回忆。你们处在同一个空间，却活在两个世界。两个人的感情，最终剩下你在苦苦支持。凄美的音乐加上完美的芸式唱腔，使『独角戏』成为被传唱最广的芸式情歌。  
  
许茹芸，本名许宏琇，1974年出生于台湾。国光艺校音乐科毕业，她自小研习音乐，出道十六年，已发行二十三张个人大碟，实乃多产歌手。因其自幼研习音乐，因此她不但演唱国语歌，还演唱粤语歌，并且进行词曲创作、为朋友写歌等，实属歌坛能手。因其自创空灵缥缈“芸式唱腔”，故江湖人称芸妹妹。主要代表作品《
_独角戏_ 》、《泪海》、《日光机场》、《如果云知道》等。
