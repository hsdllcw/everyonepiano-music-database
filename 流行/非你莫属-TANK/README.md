

《 **非你莫属** 》是线上点播人气排行榜冠军。收录于TANK第二张全创作专辑《延长比赛KEEP FIGHTING》，KEEP
FIGHTING!~继续飞越的梦想，音乐自我竞赛~TANK无限延长 ，2006销售排行铃声下载线上音乐点播新人三冠王TANK 。

Tank演唱的《 _非你莫属_ 》不同于林依晨的深情感动。作为2006年台湾偶像剧《东方茱丽叶》片尾曲。

TANK，2005年迅速走红的台湾歌手，擅长词曲创作、吉他演奏、演唱以及舞蹈，其歌曲创作的能力让他成为各大唱片公司争相指定邀歌的对象，堪称创作、演唱、舞台魅力三项全能的稀有新人。签约华研国际音乐，出版多部音乐专辑，获得多项音乐大奖。2012年9月和女友郑智雅在台北举行婚礼。

歌词下方还提供了 _非你莫属钢琴谱_ ，欢迎感兴趣的朋友下载使用。
