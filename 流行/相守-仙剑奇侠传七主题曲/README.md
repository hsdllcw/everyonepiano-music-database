

作为《仙剑奇侠传七》的游戏主题曲，《 **相守**
》秉承了仙剑音乐的仙侠风骨与古典曲风，继续为大家讲述仙剑世界里那些感人至深的江湖情长。御轻风、执长剑，踏遍时空、越尽轮回，依旧不能忘却的是与你相遇的惊鸿一面。岁月山海皆可平，只愿生生世世相守相伴。

而这首缠绵至深的歌曲经由周深实力演绎，以充满故事感的嗓音，深情吟唱，更是将音乐诠释得淋漓尽致。为《 _相守_
》作曲的实力制作人陈致逸表示，“创作这首歌的灵感，是在了解了仙剑七的游戏剧情后，被游戏中故事情节所感动，算是有感而发”。
