_龙卷风_
是创作型女歌手邓紫棋的翻唱歌曲。除了重新编曲以外，G.E.M.更设计了一段RAP加入其中，用的是周杰伦另一首歌《安静》中的一段歌词并加以修改，让这首全新版本的《龙卷风》呈现出与原曲完全不同的风格。

在湖南卫视2014年03月07日《我是歌手第二季》第九期的舞台上，邓紫棋《龙卷风》在表演的过程中淋漓尽致的释放自己，这对她来说也是代表着一个全新的开始。
