

《 **借口**
》是周杰伦2004年发表的专辑《七里香》的典型周式抒情歌曲，延续了周氏情歌的特点，是周式情歌的代表作之一。周杰伦（1979年1月18日－）中国台湾华语流行歌手、著名音乐人、音乐创作家、作曲家、作词人、制作人、杰威尔音乐公司老板之一、导演。近年涉足电影行业。
