

《 **动物世界** 》是薛之谦针对于社会感情观所存在的问题而创作的，是薛之谦比较满意的作品。收录于薛之谦2017年11月28日发行的专辑《渡 The
Crossing》中。2017年6月21日，该曲获得全球流行音乐金榜上半年二十大金曲奖。

歌曲采取了类似奏鸣曲的曲式，主歌部分几乎就用一架钢琴来运转，之后除了弦乐的引入之外，还用了急板或变奏曲式等方式，加强了作品的层次感，做到了简约而不简单。不过，虽然整首歌曲的旋律很有艺术感，但郭顶的聪明之处，就在于副歌部分的创作，依然保留了流行歌曲应该有的质感和流畅，也让薛之谦最后可以将力气集中运用到这个部分，将作品的高潮推送出来。

该曲与李荣浩的《野生动物》类似，两者写的都是欲望，只不过后者最后着点在情爱，而《 _动物世界_ 》最终则在扭曲的爱情观里，聊起了人性。
