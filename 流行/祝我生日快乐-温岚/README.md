

《 **祝我生日快乐** 》收录于温岚的专辑《温式效应》中。

  

首波主打歌《 _祝我生日快乐_
》原是周杰伦广告主题曲，他特别将歌曲由之前《叶惠美》专辑抽掉，温岚更以内敛唱法诠释，“祝我生日快乐”歌词中唱着“我要谢谢你给的你拿走的一切，热恋伤痕，幻灭重生，祝我生日快乐”在描述一个女生决定挥别旧有恋情的所有回忆，勇敢重生的心情。

  

温岚在《 _祝我生日快乐_
》的音乐录像带中，展现了高难度的演技，先是挑战难度最高的“哭戏”，接着演出激烈的“吻戏”，在新歌“祝我生日快乐”的音乐录像带中又哭又笑，让温岚直呼真的是“演技大考验！”尤其是“吻戏”的部分，因为有多个亲吻镜头，让首次演出“吻戏”的温岚尝试到了长达八个钟头的激吻，让温岚直呼：“真是难忘的屏幕初吻经验。”也由于亲吻戏长达八个钟头，如此的激吻让温岚因此在拍戏的隔天，嘴唇周围竟然都“脱皮”了，上妆时伤口疼痛不以，让温岚心有余悸的表示：“下次不敢了！”
