

《 **我不愿让你一个人**
》是五月天2011新专辑《第二人生》中一首新单曲。作为新专辑首波主打歌，《我不愿让你一个人》借2012的末世传说特别打造了明日版和末日版，传递了如果还有明天以及如果没有明天的两种感触。
