_萤冢_ 是Calvaria在2015年4月26日发布的专辑《Anonymous》中的一首单曲。

某个村庄的入口，有一片名为“千本樱”的樱树林，每到夏天，在分割村庄和樱林的溪流中，美丽的妖精都会和萤火虫一同到来。

以《萤冢守》为名的妖精，会静静地守护萤火虫——那些在夏天结束就会死去的萤火虫们。

据说（萤冢守）为了一个目的而经历轮回。

每次完成守护任务就会长眠的妖精，与一位少女在来年夏天立下相会的约定。

但是，由于那年生祭的仪式（那个少女），那片树林的时间停止流逝。（此句感觉远超我理解范围，直译那片森林所流逝的时间被其保留了下来）

这是“被樱花林囚禁的妖精，与等待夏日的少女，永远没有终点约定”的故事

一位贵族的女儿不愿同父亲去阿谀谄媚其他王宫贵族，在宴会当晚独自一人逃到了森林，一只萤火虫将少女带到了森林深处，在那里发现了一个叫螢冢的地方，这个地方只有每年夏天萤火虫出没的时间才能进入，在那里少女遇见了一位穿着古老服饰的男孩儿

男孩有着一头湛蓝的白发，双眸如同璀璨的钻石般耀眼。他向少女伸出手，示意她跟上。少女握住了男孩的手，内心不由得有些心安。男孩领着少女向前走，一直走到了一颗古树前。

古树已经枯死，干枯的枝丫似在无声的哭泣，可神奇的是，古树的中心仍散发着盎然生机，男孩用手指指了指那棵树，用着生涩的话语解释着古树的由来，随后牵着少女的手，将少女牵至树根旁。

少女好奇的用手戳了戳，古树被触摸到的地方十分柔软，好似婴儿般娇嫩的肌肤，少女不禁入了迷，就在她愣神之际，一只萤火虫飞了过来。只见它光芒暗淡，似是到了烛末之年，萤火虫晃晃悠悠的飘到少女手指的地方，不可思议的一幕就此发生。那萤火虫竟“穿”入了树内。

男孩在一旁，怯生生的解释着：“这是萤火虫的坟墓，老去的萤火虫为了不拖累正群萤火虫的光辉，自甘来到这里，生得寂寂无名，死去无人问津，但它们的尸体为这棵树提供了养分，人生也正如这样，总有人为你的生活付出代价，可他们却不被你理解，乃至怨恨……”男孩的话很轻柔，如一阵风吹入了少女的心房。

两行清泪自少女脸颊滑落，留下两道泪痕，低低的哭声传来，少女依靠在树旁埋头痛苦，可带她来到这的男孩却消散不见，原地只留下一堆萤火虫的尸体。。。
