

任然《 **后继者** 》，悦耳的旋律和任然温暖的嗓音完美诠释着歌曲的情感。

分手后谁是你的后继者？  
你又是谁的后继者？

感情中从来没有谁赢谁输，分手后只有伤感和无奈，要学着去释怀，即使有时候很难。

由音乐才女葛雨晴、音乐全才Xun和金牌制作朱鸽组成的铁三角团队，给这首《 _后继者_
》从创作和编曲带来了顶级的质量保证。任然也在自己擅长的和声部分捉刀，并且与Xun共同演唱了和声，为歌迷送上温暖的音乐大餐。
