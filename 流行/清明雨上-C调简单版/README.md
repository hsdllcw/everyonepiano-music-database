

**清明雨上**
是许嵩作词作曲并演唱的一首脍炙人口的国风悼亡歌曲，曲风不伤大雅，开头弦乐一出就是淡淡的忧伤感，歌词婉约动人，流露出真挚的情感，就是一首古色古香的诗。

许嵩慵懒的声音配上古筝的声音温婉尔雅，细腻柔情。主歌把故事唱给你听，副歌高潮迭起，颇有宣泄之感，情绪非常饱满，所有的愁绪都被雨打湿了眼眶。这是一首很有人文情怀的歌，如诗如画。创作中许嵩还吸收了朋友的修改意见，反复斟酌，制作历程蔓延2个多月，08年此歌发表于网络后传唱度极高。
