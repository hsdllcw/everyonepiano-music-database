

《半兽人》是2002年周杰伦为《魔兽争霸III：混乱之治》作代言而创作的，因此音乐中插入了一些游戏中的音效（例如军队行军的脚步声）。歌词的主题是半兽人种族“兽化”和“还原为人”的过程。音乐基本上是电子的，但合唱部分中管弦乐的运用也较多。有唱也有说唱。
