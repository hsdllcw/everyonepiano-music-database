_飞云之下_ 是韩红和林俊杰演唱的一首歌曲，由易家扬作词，林俊杰作曲。

此曲给人的感觉是非常温柔且平和的，像是被抚摸心口，很轻但治愈效果极佳。或许总会在某一时刻敏锐接触到生活的锋芒，但听完这首歌之后却觉得，哪怕在高空，也可以是平稳的，哪怕乌云之下，也不需要惊惶害怕。飞云之下，亦可感到安心。
