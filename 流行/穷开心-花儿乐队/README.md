《 _穷开心_
》从北京传统戏曲中寻找灵感，相声、曲艺与流行音乐元素结合在一起，以诙谐幽默的口吻传达出世人对开心快乐人生的美好期待。现在80后的年轻人大多对曲艺不感兴趣，花儿用这样一种方式将传统文化推广到年轻人中去。
