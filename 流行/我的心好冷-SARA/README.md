me》、江映蓉《一起来挑麦》等一系列的歌曲，刘佳的作词刻意用平实的语言方式，很多人在城市里过了春夏秋冬，渐渐被生活磨砺地失去了本身的棱角，不再敢表达自己，不在敢诉说感情，不再敢大胆追求明天的美好，创作这首歌正是想让那些人找到失去的勇气，寻获生活的真谛。
