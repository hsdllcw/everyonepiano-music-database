_兄弟抱一下_
是庞龙出的一首单曲，由蔡龙波作词谱曲，贾轶男编曲，由庞龙演唱。歌曲刚刚上线推出，就荣登多个音乐榜单的排名，并再度引领“怀旧风潮”受到全国媒体和歌迷追捧。

此曲词曲朴实，深度提炼的朋友情感，编曲上由制作人、吉他手贾轶男负责掌控歌曲方向，抽简去繁，保留最能表达纪念青葱岁月的吉他声，直抒歌曲内在精神，庞龙也凭借完美唱功将歌曲《兄弟抱一下》演绎的真实到位。

歌词下方是 _兄弟抱一下钢琴谱_ ，希望大家喜欢。
