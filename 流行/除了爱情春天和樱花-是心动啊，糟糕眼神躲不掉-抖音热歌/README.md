

“是心动啊，糟糕眼神躲不掉”，又是一首暴击单身狗的歌曲。

这首歌曲是抖音一个小姐姐改编的，用的是《 **除了爱情春天和樱花** 》这首歌曲的伴奏。

据悉，小姐姐为现男友（李现）改编了这首IU超甜的歌，看的出来小姐姐是非常喜欢李现。李现在电视剧《亲爱的，热爱的》中饰演韩商言一角，韩商言是非常有责任心，对爱情很专一的男主，是很多女生心目中的理想男友，这部剧火了之后，很多女生都称李现为现男友了，表达对他的喜欢和支持。

目前这首改编的歌曲只有几句歌词，没有完整版的，只有期待之后小姐姐会不会发布完整版的歌曲了。

歌词下方是 _除了爱情春天和樱花钢琴谱_ ，大家可以免费下载学习。
