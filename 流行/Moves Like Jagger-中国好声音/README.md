

美国选秀节目The Voice的评委Maroon 5的主唱Adam Levine联手另外一位评委Christina
Aguilera推出了他们合作的单曲"Moves Like Jagger"，并且在The
Voice的舞台上演了现场版的大首播，这首歌曲同时也向滚石乐队（The Rolling Stones）主唱Mick Jagger致敬。

中国好声音丁少华，周礼虎共同演绎过。
