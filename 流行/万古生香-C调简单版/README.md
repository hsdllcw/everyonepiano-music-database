

**万古生香** 是2020年哔哩哔哩拜年祭中的一首为中国古代著名女性群体创作的古风歌曲，由中文 VOCALOID虚拟歌姬洛天依演绎。

该曲PV已于2020年1月24日22:54在bilibili上映。2020年1月25日15:08正式成为传说曲，用时0天16时14分。成为最快达成“VOCALOID中文传说曲”称号的曲目。
