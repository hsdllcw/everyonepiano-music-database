

**说了再见**
收录在周杰伦第十张个人专辑《跨时代》，是传统的“周氏”情歌，曲风舒缓、委婉，很有周杰伦的味道，由古小力和黄淩嘉作词、周杰伦谱曲、钟兴民编曲，是李连杰电影《海洋天堂》中的歌曲。
