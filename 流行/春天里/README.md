

《春天里》是中国大陆歌手汪峰演唱的歌曲。在《春天里》，听到了汪峰自醒式的回忆和对生命的忧虑。他甚至大胆地用极度暗哑而绝望的声音，来营造和万物复苏的春天的巨大反差，这一切听起来都充满了震撼。正是这直面的寂寥，却透出了难得的真诚和勇气，在漫山遍野的绿色中感受到了巨大的荒凉。2010年11月，一组农民工组合旭日阳刚因在《星光大道》中演唱《春天里》而迅速走红。
