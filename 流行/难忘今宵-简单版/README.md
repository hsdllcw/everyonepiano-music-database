》原由中国内地女高音歌唱家李谷一演唱。1984年之后，其作为春晚的结束歌曲一次次地回荡在春晚的舞台上，迄今为止已经陪伴春晚走过N多个春秋。

歌曲将家国情怀、骨肉亲情相融通，通过歌声将美好祝愿传递到千家万户。歌如其名，那年的“今宵”成为温暖的永恒记忆。

》的音律舒缓、悠扬，词意亲切、邃远，易于贴近观众的心，易于上口传唱，更具催动情感的张力和渗透力，既道出了盛筵将散，深情告别时的真挚祝福，又表达了对来年再相会的无限期盼和良好祝愿。可以说，《难忘今宵》作为这个时代的印记已经深深的印在人们的脑海，无论时代怎么发展，它都会成为人们心中永恒的旋律。
