

**Unity** 是一首电子音乐，出自德国音乐人TheFatRat（胖鼠）的EP《Unity》。

搭配律动的bass line，第一次听这首歌就能感受到TFR为《Unity》
注入的欢乐基调。灵动的流水音效搭配轻重兼具的kick，8比特的音色，再加上胖鼠完美的将传统乐器糅合而入，都使这首歌显得灵动而饱满，让人身处想象中的奇妙世界，思绪纷飞，放空自我。而作为胖鼠的成名曲，《Unity》
可谓细节满满，每一次的和弦走向以及鼓组编制都能让人眼前一亮，从而发现不一样的“角落”，一向活泼的风格搭配正能量的歌词，仿佛能让人洗涤尘埃
沉浸而入，脱离繁复的现实世界，音调拉高音色纷呈加入那一刻，就好像跃出窒息的深海，走入无人之境。
