

**舍不得**
是女歌手内地弦子演唱的一首歌曲，由马嵩惟作词、张正宗作曲，收录在弦子2008年4月发行的专辑《不爱最大》中。2009年该歌曲获第16届东方风云榜
“十大金曲奖”及百度娱乐沸点“十大金曲奖”。

  
这首歌延续了弦子上张专辑《沿海地带》《风筝》等情歌浪漫悲伤的风格，也让弦子从一个天真女孩，成长为一个能够笑着面对爱情过往的新时代坚强女生。

  
《 _舍不得_
》唱出所有人的心声，扬着曲风婉转的唱腔，深刻勾画出一个凄凉的故事，近乎完美的音乐流淌至内心深处。很感人，很心痛。而歌词更是唱到每一位听众心坎中。每一个动作，每一句对话，都在脑海里呈现出它的影像。

  
歌词下方是 _舍不得钢琴谱_ ，希望大家喜欢。
