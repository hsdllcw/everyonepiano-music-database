

**心动双手简谱** 和五线谱完全对应。  
_  
心动_ ，林晓培原唱歌曲，出自张艾嘉执导电影《心动》，由才子林夕作词，才女黄韵玲作曲。陈洁仪在2011年专辑《重译》中演唱了此歌曲。  
  
电影《心动》由金城武、梁咏琪和莫文蔚主演，张艾嘉称故事有着自己过去的影子。本剧发生于1970年代至90年代；倒叙方法描述金城武、梁咏琪（中学时期的张艾嘉）分合不定的浮萍爱情，莫文蔚、梁咏琪也有一段友情、爱情交织的复杂情感，甚至夹杂三角恋曲。  
  
歌词下方是 _心动钢琴谱_ ，希望大家喜欢。
