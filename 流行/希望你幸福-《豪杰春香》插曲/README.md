

《 _希望你幸福_ 》是林亨柱演唱的，作为2005年KBS剧《豪杰春香》插曲。  
  
林亨柱（lim hyung joo） 韩国艺人 生日： 1986 年 5 月7 日
“亚洲美声小王子”林亨柱15岁已是美国茱莉亚音乐学院的先修班学生，自小获奖无数.他的魅人气质与完美歌喉眼下风靡韩国，其影像和声音充满了韩国各类媒体。
