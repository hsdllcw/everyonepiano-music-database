_Youngblood_ 是美国朋克乐队绿日乐队(Green Day)的新单曲，于2016年10月7日发行。

绿日乐队(Green Day)，又称年轻岁月，是美国的一支朋克乐队。由Billie Joe Armstrong担任主唱，Mike
Dirnt担任贝斯手，Tré Cool担任鼓手。2014年10月9日，绿日被提名入围2015年摇滚名人堂。
