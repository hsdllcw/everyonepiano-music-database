_第一次告白_ 是由中国男子演唱组合TFBOYS演唱的一首歌曲，单曲于2019年8月5日发行。

歌曲前奏由清爽的电吉他引入，节奏由弱到强，层次鲜明，旋律由主歌的深情过度到副歌的活泼，展现了独特的变化。无论是前奏还是副歌都有着鲜明的记忆点，高低音跨度比较大，很好地展现了三个人不同的嗓音特质。

歌词描写了首次鼓起勇气告白的少年心事，饱含了对于浪漫爱情的幻想和期待。其中“第一次告白，不在乎结局是好坏”、“不担心会错会伤悲会失败，青春不怕一切等待”等语句唱出了一往无前的无畏态度，鼓舞着年轻人大胆地去追求自己的爱情。整首歌在TFBOYS三个人日渐成熟的嗓音演绎下，呈现出迷人的反差效果。
