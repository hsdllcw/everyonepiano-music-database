

**血、汗、泪** ，英文为Blood Sweat &
Tears，是由韩国流行男团防弹少年团录唱的一首流行舞曲。且2016年10月，该歌曲被《公告牌》杂志评为“防弹少年团十大必听歌曲”之一，位列榜首。

《 _血、汗、泪_
》蕴含青少年在青春期陷入诱惑时的矛盾与成长，是一首充满活力的轻快歌曲，该作品选用的是Moombahton和陷阱音乐，也就是雷鬼音乐和浩室电子音的中低拍速风格，曲风独特又不突兀，稍微减弱了防弹少年团以往展示给听众们强烈的粗犷形象，重点是加入了性感魅力。而带走喻指他们一切的“血、汗、泪”、“明知圣杯有毒也要将其喝下”这样的歌词，展现了少年们陷入了无法拒绝的诱惑以及混乱的心境，表达人生会经历恶的影子、他人和外部世界的矛盾等成长的过程。
