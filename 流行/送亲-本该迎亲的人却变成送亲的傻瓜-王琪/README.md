

**写歌的人用脑、作曲的人用情，歌唱的人用心** ，而这个人就是小编最欢的一位原创音乐人—— **王琪** 。

_送亲_
是一首故事形的歌曲，听过让人泪水婆娑，到底是有怎样的悲痛经历，才能写出如此感人的歌曲？如果你也听哭了，建议发给你的女（男）朋友一起听，如果她（他）不哭，那么可以选择把她（他）打哭。

《 **送亲**
》写的是住在草原的一对儿青年男女，他们青梅竹马、两小无猜，一起长大互生爱慕。本以为两人会走进婚姻的殿堂，可是长大后女孩儿却不辞而别，匆匆去到城市里打工，为的是挣钱给母亲治病，可是母亲的病却越来越严重，残破的家庭再也无法支撑母亲高额的医疗费。

女孩儿打工的饭店老板知道了这个消息，他表示愿意出钱给女孩儿母亲治病，但他要求女孩儿嫁给他出车祸后双腿残疾的弟弟。女孩儿哭了，经过内心一番挣扎最后还是同意了这门亲事。等女孩儿再一次回到草原的时候，就是出嫁之时，她见到了心心念念的男孩儿，什么都没说只是默默的流泪。

出嫁的那天迎亲的队伍是一排很气派的车队，而送亲的人群中最显眼的就是男孩儿，一个人骑着马不远不近的跟在女孩儿婚车的旁边。

到了最后离别的时候女孩儿降下车窗对男孩儿说了句，就送到这里吧……

在现实面前爱情有时显得很脆弱，但是真正的爱情不管经历山崩地裂，还是海枯石烂，心底的那份儿情、心里住的那个人却永远也不会改变！
**如果不能娶你回家，那送你出嫁，就是我最后的告白……**

“ **本该迎亲的人，却变成送亲的傻瓜**
”，这一句唱出了多少无奈。原本相爱的两个人，却抵不过命运的捉弄，最终只能选择互相放手。愿我们只听曲中事，不做曲中人，愿屏幕前的你，三冬暖、春不寒、天黑有灯、下雨有伞，愿你人生路上永远有爱人相伴...
