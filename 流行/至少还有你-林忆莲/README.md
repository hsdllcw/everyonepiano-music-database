

**至少还有你双手简谱** 和五线谱完全对应。

_至少还有你_
是香港女歌手林忆莲于2000年1月发行的专辑《林忆莲's》中的一首主打歌曲。是步入21世纪以来最为流行的情歌之一，也是无数女孩子K歌必点的歌曲。《至少还有你》，获得1999-2000新加坡M93.3醉心龙虎榜年度冠军；2000台湾地区KTV点播排行榜29周冠军情歌；第一届ACMA亚洲中文金曲奖：年度金曲，最佳制作奖，最佳作词奖。

歌词下方是 _至少还有你钢琴谱_ ，希望大家喜欢。
