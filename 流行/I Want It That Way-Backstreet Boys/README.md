

《 **I Want It That Way** 》是美国著名男团组合Backstreet Boys（后街男孩）演唱的歌曲。是Backstreet
Boys的标志性的歌曲。

  

《I Want It That
Way》这支单曲横扫了全球，还被美国权威的《滚石》杂志选为《最伟大的100首金曲》中的第10位。成为中板情歌中经典中的经典。《滚石》杂志评价这首歌是”超越古典的金曲”。该歌曲至今还被许多人传唱。
