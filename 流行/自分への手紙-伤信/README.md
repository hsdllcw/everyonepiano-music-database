

《 **自分への手紙** 》是由西村由纪江作曲的。陈奕迅的《 _伤信_
》的旋律就来自《自分への手紙》。西村由纪江，日本当代女钢琴家及作曲家，西村由纪江三岁开始习琴，曾赢得由山叶所主办的Junior Original
Concert音乐大赏，大学时期就读於桐朋大学的音乐系钢琴组。西村由纪江于1986年出道，发行首张个人作品专辑《Angelique》，至今已发行逾廿张专辑。

  

人们常有“第二眼”美人之说，内秀、矜持等特质虽不似外在的惊艳那般驱动着“love at first sight”，但只要用心体会，自然也会觉出其可贵来。《
_自分への手紙_ 》可算是这样的“第二眼美人”式的专辑了。

  

一遍听下来，似乎缺少明显的高潮起伏，亦缺乏绝对经典的作品，但是细细品味，你就会发现《自分への手紙》亦是可贵之作。
