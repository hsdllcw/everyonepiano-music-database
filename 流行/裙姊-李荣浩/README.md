_裙姊_
是由黄伟文、李荣浩共同作词，李荣浩作曲、制作并演唱的歌曲，发行于2017年8月1日。为了向举世闻名的中国京剧艺术大师梅兰芳致敬，李荣浩特邀亚洲知名作词人黄伟文，两人创作了这首中西风格混搭的歌，并采用梅大师的乳名——
_裙姊_ 作为歌名。  
  
李荣浩，中国流行男歌手、音乐制作人、吉他手。2013年9月16日发行个人首张原创专辑《模特》，凭借这张专辑提名第25届金曲奖五项大奖并获得最佳新人奖，实现了从制作人到歌手的转型。
