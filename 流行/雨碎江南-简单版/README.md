

**雨碎江南双手简谱** 和五线谱完全对应。  
  
_雨碎江南_ 是由墨明棋妙原创音乐团队创作的曲子，由河图作曲，后被多个音乐团队引用来作词。雨碎江南原为河图二胡独奏，后经丢子改编为民乐合奏雨碎江南·印象。

西楼月满 油灯下忆长安 敢问公子何时还

你可晓依夜里不成眠 点灯磨砚 提笔徒记飞雪已飘然  
念白：一卷洛书 上起尧舜烬 五千年兴叹

月韵上元 花灯下舞长剑 念一身功名若等闲  
隶书走了笔檐 宣纸染就长篇
