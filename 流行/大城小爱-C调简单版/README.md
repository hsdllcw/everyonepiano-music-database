

《 **大城小爱** 》是王力宏的代表作之一，收录在其专辑《盖世英雄》中。歌曲用了平缓流畅的旋律和简单的和弦走向勾画出了哪些城市青年朴素的爱情。

歌词有一些口语化，但不乏诗意，很淡很朴素，没有任何矫揉造作，写的是很多人内心情感的自然流露，足以温暖人心。然，这此曲最大的贡献是给了广大城市青年一首能唱给爱人听的歌。整首歌音域不宽，在大多男声的驾驭范围内。可以在KTV唱，也可以婚礼唱，可以说是超级亲民了。
