

**冬天里的捉迷藏** 是韩国Core Centens Media于2009年推出的女子流行演唱组合T-
ara的单曲。T-ara（티아라）由全宝蓝、李居丽、朴昭妍（朴素妍）、咸䨒晶（咸恩静）、朴孝敏、朴芝妍（朴智妍）6名成员组成。名称T-
ara源自英文“Tiara”，指想成为歌谣界的“女王”。

  
歌词下方是 _冬天里的捉迷藏钢琴谱_ ，希望大家喜欢。
