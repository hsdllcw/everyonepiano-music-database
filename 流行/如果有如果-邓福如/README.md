

**如果有如果**
是歌手邓福如演唱的一首歌曲。邓福如（阿福），1987年6月20日生于中国台湾新竹市，华语流行音乐女歌手、已解散乐团LazyBomb主唱，现为丰华唱片旗下歌手，经纪公司为大鹏传播。2011年发行首张专辑《原来如此》
。2012年3月7日，推出个人首张EP《你好，哈波尼斯》 。2013年6月7日发行第二张专辑《天空岛》，专辑主打歌曲《Me&U》 。

  
歌词下方是 _如果有如果钢琴谱_ ，希望大家喜欢。
