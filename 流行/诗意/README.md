

邓丽君（1953年1月29日－1995年5月8日），一代歌后，台湾歌唱家。祖籍河北，出生于台湾。是一位在全球华人社会极具影响力的歌手，亦是20世纪后半叶最负盛名的华语和日语女歌手。她与生俱来的完美音色演绎出无数传唱至今、余韵绕梁的歌曲，聆听她天籁的歌声是种美妙的享受。生前发表国语、粤语、闽南语、英语、日语、印尼语等歌曲3000多首。其代表作品甜蜜蜜、小城故事、夜来香、但愿人长久、月亮代表我的心、我只在乎你、又见炊烟、千言万语、你怎么说等歌曲，早已成为无数华人心中永恒的经典。据统计邓丽君的唱片销售量已超过4800万张。
