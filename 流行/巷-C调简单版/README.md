

巷是由虚拟女性歌手洛天依演唱，沨子填词，Someboy编曲，于2012年8月4日正式在哔哩哔哩弹幕视频网站发表的国风网络歌曲。

自面世以来就以其具有浓浓民国风的词曲、唯美的意境和美妙的歌声广受好评。截至现在已有8.68万次观看，2764人收藏。另有孤独流泪1创作的同人小说。
