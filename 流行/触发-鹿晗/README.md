

**触发双手简谱** 和五线谱完全对应。

_触发_ 是由鹿晗演唱的歌曲，收录于鹿晗2017年06月27日发行的专辑《I》中。

极为纯正的美式风格和性感唱腔，典型的Future
bass风格，跳跃而充满磁性的唱法与背景绵长的合成器音色形成巧妙的对比，男性荷尔蒙爆表。而每一段Flow恰到好处的强弱顿音处理，更是彰显出鹿晗独有的音乐气质。制作人Rumours
和Phil 作为当前电音界的领军人物，此次与鹿晗的合作更是为我们带来最正统最前沿的电子音乐。

充满张力的音色中隐藏着收敛又坚定的进攻性，荷尔蒙爆表的性感唱腔中却也稳稳地存留一丝细腻与温柔。像是耳语，亦像午夜寂寥时不经意的撩动。年轻的自我，势均力敌的较量，我们屏住呼吸，因为一切一触即发。
