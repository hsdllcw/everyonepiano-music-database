

**The Truth That You Leave** 是Pianoboy的钢琴曲。整曲洋溢着忧伤的气氛。听起来很有感触，所以小编特意重新制作了 _The
Truth That You Leave原版简谱_ 以及五线谱。

这首曲子存在着很多的意义，至今仍有无数的人为它填词，但一直都没能有合适的歌词。虽然只是在网络上传播，但其知名度也不容小觑。
