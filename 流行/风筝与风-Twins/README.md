风筝与风是收录于Twins《Happy Together
(新曲+精选)》中的歌曲，发行于2002年11月1日，由伍乐城作曲，林夕作词。在2010年3月26日发行的新曲+精选专辑《人人弹起》中再次收录了这首歌。同时，这首歌也是松日MP3广告主题曲。
