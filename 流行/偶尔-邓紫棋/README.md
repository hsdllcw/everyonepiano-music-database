

**偶尔** 是邓紫棋填词、谱曲、编曲并演唱的一首单曲，收录于EP专辑《偶尔》中。该歌于2014年获得第37届十大中文金曲颁奖音乐会优秀流行国语歌曲铜奖。

此曲将邓紫棋最真实、最不经修饰的情感通过她最钟爱的钢琴表达出来。完全是邓紫棋自弹自唱并且完全不经剪辑的录音作品，实力不容小窥，证明了歌曲的纯度和歌手的实力。而《
_偶尔_ 》的歌词亦有哲理，邓紫棋的唱功搭配淡淡的钢琴韵将歌词中的故事表达。
