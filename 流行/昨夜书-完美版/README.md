

《昨夜书》是音乐人许嵩在央视大型文化节目《经典咏流传》第四季第9期中登台演唱的歌曲。其中主歌部分歌词出自南宋名将岳飞词作《小重山•昨夜寒蛩不住鸣》，全曲由许嵩创作。

正是这首词存在着悲愤的情绪，因此在名声上就远远不如岳飞的另一首《满江红》有名了。但许嵩唱这首歌曲，却恰好能够把岳飞的心声给表达出来，也是让观众们认识到岳飞有血有肉的一面：他同样是人，也会有烦恼，也存在着自己无法做到而迷茫的一面。但即便如此，从许嵩的歌声里面，从岳飞写下的词里面，还是看得出岳飞有着气势昂扬的一面：惊回千里梦，已三更。这说明岳飞有着打破迷茫的念头，想要重新建立自己的信心。
