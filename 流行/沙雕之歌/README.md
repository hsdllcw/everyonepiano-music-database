

抖音魔性洗脑的歌曲越来越多，比如这首《 **沙雕之歌** 》就格外的洗脑。

《 _沙雕之歌_ 》由沙雕少年可乐君与言和合作，于2019年8月发行。

此曲在抖音上很火，
很多视频的背景音乐都用到了这首歌，而且不得不说这首歌的歌词反复重复是非常洗脑的，听下来感觉整个人都沙雕了，旋律也是一直在很多人的脑海里。还没听过的小伙伴，赶紧去感受一下吧。

歌词下方是 _沙雕之歌钢琴谱_ ，大家可以免费下载学习。
