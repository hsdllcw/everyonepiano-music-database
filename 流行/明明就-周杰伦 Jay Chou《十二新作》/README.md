

**明明就双手简谱** 和五线谱完全对应，是EOP教学曲。

_明明就_ 是周杰伦新专辑《十二新作》中的主打歌曲，由中国歌曲排行榜12月5日早10点独家首播，首播信息由微博发出。发布当天相关铃声就遭杰迷疯狂下载。

_明明就_
作为周杰伦全新专辑《十二新作》中第一批首发新歌，无论是首发时间或者歌词、曲风都十分保密，因此关注程度略逊于《红尘客栈》，不过奇特的名字引发网友的猜想，从某些媒体曝光新曲曲风类似《枫》，十分好听。
