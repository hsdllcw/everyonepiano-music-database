

这首《 **童话镇**
》是昼夜大大改编的完美版，小编听到后，感觉非常棒，特意为大家扒谱制作而成，谱子中设置了很多变速，让曲子更加完美，喜欢的朋友可以免费下载。

此曲原是独立音乐人暗杠的原创作品，是一首由斗鱼观众作词，暗杠小发作曲，主播和水友共同创作出的作品。经过好友暗杠小发授权后，帮陈一发儿量身打造了女声版本。
