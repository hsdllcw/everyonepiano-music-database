_君だったら_ 是HAPPY BIRTHDAY2013年发行的专辑《今夜きみが怖い梦を见ませんように》收录的歌曲。

Happy Birthday，日本乐队，2010年2月在新宿成立。2位成员都是1988年出生于东京都的女孩。作品有《嘘でもかわいいと言って》等。

歌词下方是 _君だったら钢琴谱_ ，希望大家喜欢。
