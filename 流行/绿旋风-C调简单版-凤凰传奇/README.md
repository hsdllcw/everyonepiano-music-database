

《 **绿旋风**
》是凤凰传奇为建党九十周年特别打造的献礼作品，也是两位成员杨魏玲花、曾毅在加入二炮文工团后首度推出的军营题材歌曲，为2011年火热的红歌浪潮掀起了一股清新宜人的“绿旋风”。

“花好月圆的年代里，岁岁好风景，一天一天守望着，子夜和黎明，从军的长路上，有多少风和雨，披挂齐整的将士们，侠骨也柔情”这动情的歌词瞬间打动了人心。《
_绿旋风_
》融合清新气息与动感节奏，体现出了凤凰传奇一贯“接地气”。而这样一首精益求精的歌曲，一经推出就获得了不俗反响，在各大网站的试听量节节攀升，媒体和歌迷对凤凰传奇首次演绎军歌也给予了充分肯定。
