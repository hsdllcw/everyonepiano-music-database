

独家记忆出自中国香港男歌手于2008年4月9日发行的同名专辑《 **独家记忆**
》。这是一首典型的“陈式”苦情歌，是不少男生的疗伤之作。第一次听到旋律会很有感觉，很有诗意，画面很丰富。特别是开场的一段钢琴，给人以满是回忆的感觉。

《 _独家记忆_
》的MV拍得很伤感，歌词中表现的是大男人在感情上刻意自我麻醉，对情人离去的事实充耳不闻，只希望留下快乐让自己永远回味。陈小春与片中女朋友则要表现得若即若离，二人眼看要分手却又都不明说，表情暧昧，表演起来颇有难度。
