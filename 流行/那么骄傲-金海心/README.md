

《 **那么骄傲**
》收录于金海心2000年全新同名专辑《那么骄傲》，这张个人风格化浓郁的冷色调专辑带给歌迷的，除了精致的音乐外，还有丝丝的清凉。21岁的金海心，漂亮而又才华横溢，她参与了新专辑大部分歌曲的制作并包办了所有歌曲的和音编写﹔还有三首歌曲是她自己创作的﹔在歌曲《过去》中，她又吹奏起了自己熟悉的长笛……有人称金海心是内地歌坛“天后”的接班人，虽然说这种话还为时尚早，而且金海心本人也一向以谦虚、低调的态度对待这种说法，但人们相信，她有这个实力，她的歌曲充分证明了她自己的实力。人们也越来越喜欢这个歌手。

《 _那么骄傲_
》分别在2000年《明星制造》和青春偶像剧《都是天使惹的祸》中担当主题曲。获得首届音乐风云榜“十大金曲，ChannelV]全球华语音乐榜中榜“神州音乐特别推荐奖最佳歌曲及中央人民广播电台“最受欢迎女歌手奖”十大金曲等奖项。

歌词下方还提供了 _那么骄傲钢琴谱_ ，欢迎感兴趣的朋友下载使用。
