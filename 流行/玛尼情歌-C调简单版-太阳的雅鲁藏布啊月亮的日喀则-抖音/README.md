抖音此段歌词出自歌曲《 **玛尼情歌** 》，由薛鹏作词、作曲，冯丹编曲，上官红燕演唱的歌曲，发行于2018年12月26日。收录于同名专辑《玛尼情歌》中。

歌曲从藏族佛学的宗教理论，解读了一段“三世”的轮回情缘。在西北风盛行的内地歌坛，上官红燕柔韧通透的特色嗓音与歌曲中浓烈的民族元素。可以说，此曲赋有浓厚的民族元素，将藏族佛学的宗教理论与儿女情长结合到了一起，目前在网上被网友翻唱后迅速火了起来。
