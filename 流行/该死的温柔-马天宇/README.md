

《 **该死的温柔**
》是歌手马天宇演唱的流行歌曲。马天宇，中国内地男艺人。1986年7月12日出生于山东德州，2006年通过“莱卡加油！好男儿”出道，2009年毕业于北京电影学院。截止目前发行两张唱片，一张EP，斩获奖杯20多座。

《 _该死的温柔_ 》讲述的是男子在女友提出分手之后的伤心情怀，以及对女友的爱恋。

另外，《 _该死的温柔_ 》词曲作者：秦天，曲风与日本流行歌手幸田来未的《梦のうた》部分相似。
