

**Marry U** 是韩国男子组合Super Junior演唱的歌曲。这首歌发行单曲同时收录在Super Junior的第二张专辑《Don't
Don》中，执行制作人为李秀满。 这首歌是包含当代R&B，以及流行民谣旋律，也有爵士乐和说唱元素的加入。

  
歌词下方是 _Marry U钢琴谱_ ，希望大家喜欢。
