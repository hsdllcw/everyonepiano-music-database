

**谁双手简谱** 和五线谱完全对应。

廖俊涛，网综《明日之子》参赛选手，他唱的歌曲名叫《 _谁_
》，这是一首原创的歌曲，不得不说，他还是蛮有才华的，这首歌也是得到了薛之谦的极力赞赏。廖俊涛也成为了薛之谦战队首位晋级的选手。

而现场版《谁》重新编曲后，变得偏后摇，整体听感更加广阔。

歌词下方是 _谁钢琴谱_ ，大家可以免费下载学习。
