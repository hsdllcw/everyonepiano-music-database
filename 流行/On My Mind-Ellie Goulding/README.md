

**On My Mind双手简谱** 和五线谱完全对应。

_On My Mind_ 是英国创作女歌手Ellie
Goulding的歌曲，2015年9月17日发行。Ellie以惊艳空灵的声音开唱，与无限激流中感受带来微妙情谊。听完她的歌会发现，一向将电音进行到底的“绵羊音”原来也可唱出磅礴大气。强烈的节奏感，狂野的魅力，神秘的热情，奔放的烈爱，时高时低，时重时轻的唱腔给人带来新鲜感。
