_绅士_
收录于薛之谦2015年5月20日发行的EP《绅士》中，后收录于薛之谦2016年7月18日专辑《初学者》中。2016年6月12日，该曲获得第9届城市至尊音乐榜年度二十大金曲奖。

该曲旋律舒缓，歌词深情，让听众听到带有薛之谦特色的“薛氏情歌”，用极限来诠释伤悲的爱情寓言。该曲如一场电影，更是薛之谦”绅士”品格的一个音乐化身。该曲是薛之谦一直擅长的情歌路线。薛之谦在这首歌中表达了一种伤婉的概念，是质感上乘的佳作。
