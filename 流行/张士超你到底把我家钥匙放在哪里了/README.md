_张士超你到底把我家钥匙放在哪里了_ 是上海彩虹室内合唱团演唱的歌曲。

这首歌采用了男女8声部合唱，气势恢宏，旋律高亢激昂，由浅入深，由叙事而抒情，层层递进，直至拷问灵魂。唱到中段向上天祈求时，相信所有听者已经不由自主地跪下；尾声段落还化用了周杰伦《牛仔很忙》中“不用麻烦了”的副歌旋律……然而，如此“史诗作品”，讲述的却是一个找不到钥匙进不了家门的怨念故事。
