

《 **阴天** 》，一首1999年发行的歌曲，然而至今依旧吊打现在好多流行歌曲！

此曲是莫文蔚整个音乐生涯非常具有代表性的经典歌曲。在她的歌声中我们能听到很多的感情故事。

“阴天，在不开灯的房间，当所有思绪都一点一点沉淀”想像一下，在经历了一场由热恋渐渐冷却的爱情后，对爱情失去信心，在一个阴天呆在房间里静静回味这段爱情。

“香烟，氲成一滩光圈，和他的照片就摆在手边”，细品后又能隐约感到对这段爱情的丝丝留恋与不舍。
