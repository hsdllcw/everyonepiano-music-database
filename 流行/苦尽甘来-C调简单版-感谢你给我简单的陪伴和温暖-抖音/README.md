

感谢你给我简单  
的陪伴和温暖  
感谢你为我负担  
这不安和心寒

抖音这段旋律出自歌曲《 **苦尽甘来** 》，由翁梓铭作词作曲并演唱，收录在2020年11月11日发布的专辑《感•知》中。

这首歌一经发行就在网上受到了很多网友的追捧，很多人都非常喜欢这首歌。在这首歌当中，讲述的是在爱情的当中历经的许多失败和绝望，很多次想要对爱情彻底失去信息，但是还会希望有一天会有一个人会给自己爱情的温暖。

_苦尽甘来双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
