

独照是歌手容祖儿国语专辑，于2003年12月17日发行；由名作曲人陈晓娟、才子制作人陈国华及名制作人刘思伟三人，特地联手打造这张'独照'专辑，是Joey容祖儿在台湾发行的第三张专辑。整张专辑以容祖儿一个23岁女孩的心情，来唱出她的爱情故事。
