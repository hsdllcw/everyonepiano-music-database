

**Still** 是日本杰尼斯事务所旗下的男性偶像团体岚的歌曲。岚（Arashi）于1999年11月3日发行首张单曲正式出道，所属唱片公司为J
Storm。自2010年起，岚已两次获得日本金唱片大奖，并连续4年当选为日本最受欢迎艺人，被誉为日本的新一代“国民偶像”。

  
歌词下方是 _Still钢琴谱_ ，希望大家喜欢。
