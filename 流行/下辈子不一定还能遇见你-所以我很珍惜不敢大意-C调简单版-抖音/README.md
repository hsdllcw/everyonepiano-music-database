》，由陈雅森演唱，李守俊作词，张博文作曲。

下辈子不一定还能遇见你，所以在任何关系中，懂得服软，轮流低头，就不会走散，哪有什么天生般配的人，只不过一个懂得包容迁就，一个懂得适可而止，相互珍惜才最重要。
