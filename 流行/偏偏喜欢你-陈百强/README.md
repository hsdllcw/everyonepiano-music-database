

**偏偏喜欢你双手简谱** 和五线谱完全对应。

_  
偏偏喜欢你_ 是香港已故流行歌手陈百强同名专辑偏偏喜欢你中的一首歌曲。该专辑可以说是陈百强一生中比较得意的专辑。入榜即直上销量榜榜首6周之久。

  
陈百强（1958年9月7日－1993年10月25日），英文名Danny
Chan，所以被称为丹尼仔。他生于香港，籍贯广东台山，香港已故流行歌手，1993年10月25日因为逐渐性脑衰竭而撒手尘寰。陈百强与谭咏麟，张国荣和梅艳芳同获认为是1980年代粤语流行音乐的主要歌手，他亦是一位擅长音乐创作的歌手。

  
歌词下方是 _偏偏喜欢你钢琴谱_ ，希望大家喜欢。
