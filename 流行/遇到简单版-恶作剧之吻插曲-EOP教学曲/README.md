《遇到》是台湾版偶像剧《恶作剧之吻》的插曲。由方雅贤演唱。

方雅贤，2004钱柜新秀选拔赛寻找『恶作剧之吻』新声音优胜者。作品有『恶作剧之吻』原声带里的《遇到》和《听见》，『我要变成硬柿子』原声带里的《决定》和《爱上思念》。
