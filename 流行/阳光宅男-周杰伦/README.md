

**阳光宅男** 是周杰伦演唱的一首歌曲，由周杰伦作曲、编曲，方文山作词，收录在周杰伦2007年11月发行的专辑《我很忙》中。

这是一首让人迎向阳光、开心舞动的歌曲。编曲虽然有些大路，但是歌曲的阳光气息却是挡不住的，“不走寻常路”的广告词出现在其中也不显得突兀。

《 _阳光宅男_ 》有着轻快曲风下的一种男生情怀；但是歌曲显得太偶像化，像是为偶像剧的配乐，而不是纯正的周杰伦多风格的创作。

歌词下方是 _阳光宅男钢琴谱_ ，希望大家喜欢。
