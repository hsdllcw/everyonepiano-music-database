

**耶和华是爱双手简谱** 和五线谱完全对应。

_  
耶和华是爱_ 是一首基督教歌曲，出自专辑天父／我心旋律诗歌精选集。该曲旋律悠扬，很适合静静聆听。

  
歌词下方是 _耶和华是爱钢琴谱_ ，希望大家喜欢。
