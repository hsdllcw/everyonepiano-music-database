

**那时候双手简谱** 和五线谱完全对应。

_那时候_ 是方艺谚演唱歌曲。新创作不再拘泥于概念的完整性, 回归到音乐人文面, 用纯粹的回忆呈现旋律和感动。这首歌讲关于儿时的回忆！用慢摇滚的曲风速度,
表现不同角度的状态。在编曲上, 大头与制作人投注了更大的心思与创意, 操控的技巧也愈臻成熟了, 融合多种器乐与各式曲风, 强化了自己的风格！

歌词下方是 _那时候钢琴谱_ ，大家可以免费下载学习。
