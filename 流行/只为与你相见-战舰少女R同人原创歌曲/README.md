_只为与你相见_
是国产手游战舰少女R首支同人原创单曲。共有三个版本，分别是中文版，日文版，钢琴慢版，分别由歌手染音若蔡、Romamontella以及云氤演唱。

战舰少女是一款把军舰拟人娘化的策略养成游戏，题材为“二次元 ACG 和
军事”，拥有其重度的SLG养成游戏的玩法与特点，娘化军舰的建造及强化方式别具一格;同时拥有高度的战术自由性，并且在养成与战斗的过程中充满了不同的策略性以及趣味性，高度符合SLG游戏用户的喜好。
