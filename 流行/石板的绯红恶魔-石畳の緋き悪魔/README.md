

《 **石板的绯红恶魔** 》（日文：石畳の緋き悪魔）由日本音乐团体Sound Horizon演唱，收录在其2007年8月发行的专辑《圣戦のイベリア》。

Sound Horizon（サウンドホライズン）是以作曲家Revo为中心的音乐团体。收录了所谓Pop
Music、摇滚乐、爵士到古典音乐、民族音乐、童谣等各式各样的音乐要素。配合故事性强的风格而各取适才、灵活运用各种乐风印象，难以一言表达其类型。不过在编曲方面也被说有着重金属与重摇滚音乐的深远影响。同时，特别在初期的作品里受到SQUARE（即现在的SQUARE
ENIX）所属时代的伊藤贤治的影响。
