最近抖音上这首歌曲火了，据悉此曲叫做《 **善变** 》，由王靖雯不胖演唱。

时间是怎样改变了我们只有彼此之间才最清楚，那年第一次相遇的画面，那天最后的背影逐渐遥远。你是旅客我是景点，逛过了也可能就永不见相见，怪要怪这世界一草一木都很善变，也包括我们的感情和不停歇的时间。可回忆不这样，过去越远它越清晰。所以故事太容易翻篇，而剧情却牢记脑海里面，也正因如此，人总需要留下纪念，是因为不舍，也为证明一些爱，真实存在过。而这首歌，和我。只是想证明那些，有你的从前。
