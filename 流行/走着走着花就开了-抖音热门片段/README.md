**走着走着花就开了**](/Music-10506-走着走着花就开了-在幸福里感受什么都别说.html "走着走着花就开了完整版")》の完整版谱子。

不用想都知道，这是一首撒狗粮的歌曲，歌词中充满了对单身狗的语言暴力！小编真的不能想象作词者是如何找到这么多浮夸、假象、臆想、不科学的语句来写的歌词。

什么叫“我牵着白云渡过一条河”？白马能牵，白云能牵吗？

什么叫“你的忧伤在雨中慢慢被淹没”？雨水能淹没街道，能淹没忧伤吗？

什么叫“从美丽的清晨 到黄昏和日落”？黄昏是指日落以后到天还没有完全黑的这段时间，时间顺序都错了！

什么叫“我们走着走着花就开了”？是花开的太快，还是你们走的太慢？

什么叫“在幸福里感受 什么都别说“？ 气的单身的小编都要背过气去。
