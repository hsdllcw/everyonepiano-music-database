

**让我欢喜让我忧** 既是歌手周华健的一张专辑名称又是其中单曲名称，而且还作为电视剧《红蜘蛛》的片尾曲。

  
周华健和你一起分享生命中不容错过的欢喜与忧愁，让我们从欢喜中找到自己，经过忧愁发现成熟。正如歌词写到，就请你给我多一点点时间，再多一点点问候，不要一切都带走，就请你给我多一点点空间，再多一点点温柔，不要让我如此难受。

  
歌词下方是 _让我欢喜让我忧钢琴谱_ ，希望大家喜欢。
