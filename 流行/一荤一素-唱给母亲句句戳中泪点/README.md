

一首《 **一荤一素** 》，简单朴实的歌词中，一点一滴的表达对亲人的回忆和思念，听着听着，不自觉地就融入了歌声中，像是有魔力一般，牵引着我们的情绪。

这首歌完全戳中泪点。据毛不易讲到：在我实习的过程当中看到过数百个在我面前死去的病人，但是第一个在我面前死去的，是我本人的亲生母亲，真的自己也是非常的无能为力！在谈到母亲的离世，弹唱《
_一荤一素_
》，画面真的是超级感人！虽然毛不易在讲述这段过去的时候，表情看上去云淡风轻，但是还是能从他的语气里听出一丝酸楚，所以他就把和母亲相处的点点滴滴写到了这首歌里，而且一直到现在，他还会经常给母亲发微信，总是相信母亲能够帮助自己。
