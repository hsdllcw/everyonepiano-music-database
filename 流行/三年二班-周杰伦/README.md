

《 **三年二班** 》，是一首歌曲，同时也是一部电影的中文译名。歌曲出自周杰伦的音乐专辑《叶惠美》，电影则是一部首映于2006年的美国喜剧影片。

  

《 _三年二班_ 》写出许多朋友在年青时的疑问与心声，整首歌让人有一种说不出的感受，勾起了听者许多许多的回忆。

  

周杰伦(1979年1月18日－）中国台湾华语流行歌手、著名音乐人、音乐创作家、作曲家、作词人、制作人、杰威尔音乐公司老板之一、导演，涉足电影行业。周杰伦是2000年后亚洲流行乐坛最具革命性与指标性的创作歌手，亚洲销量超过2700万张，有“亚洲流行天王”之称。他突破原有亚洲音乐的主题、形式，融合多元的音乐素材，创造出多变的歌曲风格，尤以融合中西式曲风的嘻哈或节奏蓝调最为著名，可说是开创华语流行音乐“中国风”的先声。周杰伦的出现打破了亚洲流行乐坛长年停滞不前的局面，为亚洲流行乐坛翻开了新的一页。
