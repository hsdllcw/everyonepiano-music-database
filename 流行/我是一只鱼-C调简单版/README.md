

《 **我是一只鱼**
》是收录于1998年8月1日《爱像太平洋》专辑中的一首歌曲，由任贤齐演唱。歌曲把曾经相恋的男女比作鱼和水，离开了你的我，就好比鱼儿离开了水。原来你是我最大的依靠，原来我的生命里已经不能没有你。

任贤齐，华语流行乐男歌手、影视演员、词曲创作人。他的音乐有时柔美连贯，有时舒缓顿促。他的演唱风格也在不断尝试。可不管怎么变，他的“任氏情歌”依然独树一帜。
