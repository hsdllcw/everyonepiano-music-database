

**需要人陪双手简谱** 和五线谱完全对应。

_需要人陪_ 是王力宏演唱的一首歌曲，由王力宏作词，作曲，收录在王力宏2010年8月发行的专辑《十八般武艺》中。

这是一首偏重内心的歌曲，王力宏写出了都市人内心孤寂的心声；信手拈来的音乐乐章，感觉就像是在音乐餐厅听到一个歌手潺潺诉说都市之人的寂寞。《需要人陪》是以钢琴做主导的慢歌，王力宏凭借自己的煽情功力成功演绎了这首抒情歌曲，不管是澎湃的情感还是收敛的情绪，他都能把声音表现力拿捏的恰到好处。王力宏把自己词创作的这首《需要人陪》唱得很深情很彻骨，“一个我需要梦想，需要方向，需要眼泪，更需要一个人来点亮天的黑”这样的歌词要远比那些华而不实的句子来得更有味道。
