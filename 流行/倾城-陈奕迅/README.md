

**倾城双手简谱** 和五线谱完全对应。

_倾城_ 是香港流行男歌手陈奕迅翻唱的一首粤语歌曲，由黄伟文作词，陈佳明作曲，原唱是许美静。收录于专辑《Concert YY
黄伟文作品展》中，发行于2012年10月03日。

许美静的版本专情而绝情，“依然爱你，但是再见了。”而陈奕迅的版本多情而长情，“再见了，但我依然爱你。”
字里行间没有苦情，也谈不上多伤感，就是一种「决绝」，像寒风刺骨的冷冽、凄婉，穿透人心的冰冷，足以倾倒一座城市。
