

《 **任天堂流泪** 》是香港粤语流行曲歌手古巨基的歌曲。歌词表达了对不可挽救的爱情的依依不舍，一发行便获得了无数的好评。

同时，此曲收录在专辑《游戏-基》中，发行于2003年12月05日。2004年，古巨基凭借该专辑提名第4届音乐风云榜“最佳男歌手奖”。
