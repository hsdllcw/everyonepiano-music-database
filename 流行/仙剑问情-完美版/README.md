

《 **仙剑问情** 》是仙三外传的主题音乐，也是对前三代仙剑游戏精神的完美总结。凄婉悠长的乐声响起，一群白鹤在青天白云间展翅翱翔，烟云缥缈的仙山若隐若现。

2007年，《 _仙剑问情_
》在后作《仙剑奇侠传四》中再次出现，人物琴姬在相应剧情后与众人离开千佛塔，在千佛塔前空地上，琴姬为众人弹唱此曲。2018年，由奇然翻唱该歌曲，并收录于《仙剑•梦圆集》。
