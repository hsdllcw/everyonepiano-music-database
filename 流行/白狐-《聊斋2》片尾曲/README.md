

**白狐EOP** 由网友曳影制作，感谢曳影对EOP的支持！

《白狐》作词：孙红莺（网名：玉镯儿），作曲：李旭辉（网名：枫林）原唱：陈瑞。  
  
歌曲《 _白狐_ 》，原唱陈瑞，后被选为2007年叶璇、孙耀威、陈浩民等主演电视剧《聊斋2》的片尾曲，改由艾轶曼演唱。  
  
歌词的下方还提供了 _白狐钢琴谱_ ，欢迎感兴趣的朋友下载使用。

### 白狐歌词：

我是一只修行千年的狐

夜深人静时 可有人听见我在哭  
灯火阑珊处 可有人看见我跳舞  
我是一只等待千年的狐  
千年等待 千年孤独  
滚滚红尘里 谁又种下了爱的蛊

我爱你时 你正一贫如洗寒窗苦读

能不能为你再跳一支舞  
我是你千百年前放生的白狐

只为你临别时的那一次回顾
