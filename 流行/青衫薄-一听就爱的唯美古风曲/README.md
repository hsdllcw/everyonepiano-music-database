_青衫薄_ 是KBShinya，哦漏，萧忆情Alex以及排骨教主共同演唱的歌曲。此曲中的戏腔可以说是相当唯美，果然是不从错过的好音乐。

歌曲讲述的是一个与少年有关的故事。两个少年自幼一起长大，寒窗十年双双金榜题名。然而一纸诏书改变了他们的命运，赴任之时一人江州一人陇右。一南一北，一东一西，自此相逢成了梦中景。风尘仆仆多少年后，二人偶然相逢，却发现竟已经无言以对。

似乎人和人的交往就是由陌生到熟悉再到陌生的过程。当时光拂过，淡薄的情感会变得浓厚，浓烈的感觉也会渐渐索然。没有谁会一辈子陪着谁，曾经的情感在离别的注视下终归会逐渐沉淀。我们所能做的，似乎也只有珍惜当下。

珍惜现在的每一分时光，以后回首，便会发现，那时候的我们一笑就能忘忧
