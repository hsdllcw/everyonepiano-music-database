

《 **陪你去流浪**
》由薛之谦填词作曲并演唱，于2019年12月9日发行，收录于专辑《尘》。此曲无需华丽的演唱技巧，用最纯粹的倾诉，就像讲述故事一样，直白又感人。

“流浪”对于薛之谦而言意义非凡，因为这首歌的创作时间将近7年之久，背后的故事很励志，足以证明“老薛”是一位具备了匠人精神的唱作歌手。

当年，薛之谦拍戏的时候写了这首歌，不过，前奏部分借用了《千言万语》的歌词和旋律。但是，薛之谦绞尽脑汁，花了六年的时间也没有写出更好的前奏。

两年前，薛之谦终于决定，花钱买下版权，因为“老薛”认为，如果缺少了这个前奏，“流浪”这首歌就无法成立。

外表有点“无厘头”的薛之谦，对于音乐的态度却极其认真，难怪他能成为华语乐坛的“中坚力量”。果然，前奏一响，听众就被带入意境之中，好听到无法反驳。
