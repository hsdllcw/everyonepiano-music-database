

《 **伊斯坦堡** 》是周杰伦演唱的一首歌曲，由周杰伦作曲，徐若瑄作词，洪敬尧编曲，收录于周杰伦2000年11月7日发行的专辑《Jay》中。

此首出自于徐若瑄远赴土耳其时，在伊斯坦布尔为周杰伦所创作的歌词，全首充分表现出，徐若瑄小女人可爱的一面，而歌曲中叙述新一代恋爱的甜言蜜语，让人有一种回到当时初恋的感觉，仿佛让人回到了过去。
