

《 **愛でした。** 》是日本偶像团体関ジャニ∞演唱的歌曲。同时是锦户亮主演的TBS系电视剧《パパドル！》的主题曲。

関ジャニ∞，kanjani ∞
简称关八。日本偶像团体，官方翻译为关杰尼∞。原名関ジャニ8，出道后将8写作∞，意为无穷大。由横山裕、涉谷昴、村上信五、锦户亮、安田章大、丸山隆平和大仓忠义等7人组成。

歌词下方是 _愛でした。钢琴谱_ ，欢迎大家使用。
