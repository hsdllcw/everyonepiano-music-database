_我好想你_
是由歌手潘广益自己作曲、作词并演唱的一首歌曲，收录于专辑《好想再爱你》。潘广益的声音，温柔而充满回忆，是治愈，或是撕裂，无法确定。唯一可以得知的，恐怕只是所有的选择，悲伤或喜悦，都源于内心甘愿。

歌词下方是 _我好想你钢琴谱_ ，大家可以免费下载学习。
