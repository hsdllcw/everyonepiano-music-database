_等下一个他_
是华语流行女歌手习谱予演唱的歌曲。19岁的内蒙古女孩习谱予，阳光、简单而又洒脱，自幼就非常热爱流行音乐和歌唱。曾在美国求学时习谱予接触到了欧美最为流行的电子乐，深受震撼。随后，结识了一群志同道合的音乐伙伴，并将自己的音乐构想不断分享给大家，在一拍即合后，便正式走上了EDM情歌的道路。

歌词下方是 _等下一个他钢琴谱_ ，大家可以免费下载学习。
