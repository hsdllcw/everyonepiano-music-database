

**放生双手简谱** 和五线谱完全对应。

_放生_ 是范逸臣演唱的一首歌曲，收录在专辑《不说出的温柔》，武雄作词，“阿Bert”梁可耀作曲。

在电影《我的野蛮女友》主题曲“I believe"及韩剧《PIANO》同名主题曲"PIANO"中深情动人的演唱，让他成为唱片店里问询率最高的未发片新人。
主演海角七号大热，使歌手出身的男一号范逸臣人气直升，身价翻倍。

歌词下方是 _放生钢琴谱_ ，喜欢的朋友可以免费下载学习。
