恋人心_ 是女歌手魏新雨全新唯美情歌，同时也是电视剧《花千骨》的插曲。  
  
这是一首经典唯美古风歌曲，歌词表达恋人的思念，曾经的唯美只能回忆，留下半颗心给你。曲风温柔舒缓，给人一种安静的相思。静下心来细细品味歌曲能把听众带入境界，可谓一首惊世之作。  
  
而此曲的发布会于2014年7月13日在成都举行。布会众多媒体前来参加，同时魏新雨的圈内好友们著名音乐人高进、好声音歌手王悦等分别录制了VCR，在发布会上将一一送出对魏新雨新歌发布会的祝福，为其新歌加油。这次魏新雨的全新单曲《恋人心》新歌发布会是由红世控股主办，同时也是重磅打造单曲《恋人心》的出品方。
