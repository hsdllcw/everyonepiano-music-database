

**我们的时光**
是1968年首映的英、义合制电影《罗密欧与茱丽叶》（又名《殉情记》）的插曲。此片现被誉为经典，当年亦曾入围不少电影奖项，包括第41届奥斯卡金像奖的最佳影片奖及最佳导演奖，但最后在1969年颁奖礼当晚只赢得最佳服装设计奖及最佳摄影奖。

  
此曲由意大利作曲家Nino Rota创作。他的创作风格比较传统，特别擅长写作优美的旋律，作品广为传播，为大众所喜爱。

  
歌词下方是 _我们的时光钢琴谱_ ，希望大家喜欢。
