

**负重一万斤长大** 这首歌是歌手太一了解韩国幼童性侵事件素媛案之后，有感而发。最近时代少年团也翻唱了此曲。

歌曲讲述了一个被性侵因而未婚先育的小姑娘的内心世界。负重一万斤，不只是意外有的孩子，还有内心的沉重。这一切都不是她应该经历的。高潮部分，在“负重一万斤长大着”这一句童声为主，太一的声音渐渐的融入底音。而平稳的童声下面隐藏着的是痛苦的声音。“为什么凋零了的不只我一个”，将一个小姑娘的故事，变为一个遭受了同样伤害的群体的故事。
