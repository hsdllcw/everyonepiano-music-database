When a Child is Born_ 这首歌名叫《当孩子诞生时》,也有人称《当婴儿降生时》，是电影《人狼之恋》的主题插曲。此歌歌手Johnny
Mathis约翰尼·马蒂斯
有“乡村歌王”的美称，最擅长演唱那些具有典型民族地方特色的优秀歌曲。《当圣婴降生时》是美国电影《人狼之恋》中的主题插曲，也是一首圣诞歌曲。圣菲利普童声合唱团（Libera的前身）在《天使之音3》也演唱词曲，非常好听。
