

**当你双手简谱** 和五线谱完全对应。

_当你_ ，2010年林俊杰重新编曲演唱该歌曲，收录在其个人概念自选集《她说》中。

其实，此曲是王心凌2003年发行的首张个人专辑《Cyndi Begin》中的歌曲，也是林俊杰作曲。在王心凌演唱后深受广大歌迷的喜爱。

歌词下方是 _当你钢琴谱_ ，大家可以免费下载学习。
