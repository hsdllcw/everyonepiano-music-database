

**娃娃脸** 出自后弦2009年发行的同名专辑《娃娃脸》中。这首歌曲在曲风上让人有耳目一新的感觉，比较适合快节奏却又复古的现代都市生活。

《 _娃娃脸_
》一经推出后，便在各大音乐网站迅速抢站推荐位置并点击惊人，连连上榜，位居首位。某大型搜索引擎更是连续登陆在不同全国榜单上，并且《娃娃脸》整张EP的“幻想三步曲”——《娃娃脸》《花甲》《你还欠我一个拥抱》三首歌曲全部上榜，并连续蝉联三甲数月不下。
