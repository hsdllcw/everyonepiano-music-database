

毛不易新歌《 **深夜一角** 》又是一首经典原创，一字一句饱含深情。

毛不易曾经讲过创作这首歌的背景：他当时一个人在外地实习，呆在医院的重病急诊科里，每天都有可能面对一个个生命忽然的消失，心里说不出的压抑。晚上回到独居的出租屋里，空气安静的吓人。身边没有朋友，只能常常给要好的朋友打电话。可是隔着那么远的距离，又缺失了多少温度。加夜班的时候，总会去那家卖炒面的小摊。像是互怜，又像是借一丝温暖，但总填充不了日益空虚的心灵。不知道该如何排解，也不知道未来的路在何方。毕业，就意味着迷茫。

有时候很希望没有人能够听懂毛不易的歌，那么说明你足够幸运。因为听得懂的，那都是过往的经历。
