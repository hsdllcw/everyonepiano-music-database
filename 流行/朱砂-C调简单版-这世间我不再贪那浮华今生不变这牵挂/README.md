

“这世间我不再贪那浮华，今生不变这牵挂”出自任然和Xun易硕成演唱的歌曲《 **朱砂** 》，于2020年9月19日发布。

歌曲以钢琴开场，没有刻意加入太多民族乐器，之后吉他和箫的交错，让整首歌多了宜古宜今的隽永味道，听者如同观看一篇荡气回肠的古风小说，其中意境，值得我们去不断想象和回味。

任然的歌声带着淡淡愁绪和喜悦，没有太多起伏，却诉尽了内心的满足，而Xun易硕成则在富有磁性的演唱中，展现了更多情绪起伏变化，两个人一静一动，男子的归来，女子的等候，都在她和他团圆的一刹那，完整了“尘世间为我（你）点一抹朱砂，生生厮守这天下”的幸福。
