《 _稳稳的幸福_
》是2013年贺岁片《越来越好之村晚》的主题曲，由陈奕迅演唱，是继电影《将爱情进行到底》的主题曲《因为爱情》之后，“张一白+小柯+陈奕迅”的再度合璧。2013年1月23日歌曲正式发布。2013年9月，《辣妈正传》再次以《稳稳的幸福》作为主题曲，由孙俪和张译对唱。  
  
最早导演张一白是想请陈奕迅出演电影，无奈因档期缘故，这一想法最终没能实现，最终陈奕迅改为唱主题曲，以献声的方式参演了张一白的电影。陈奕迅和小柯在反复听过小样后，讨论如何来演绎这首歌，从发音的准确度到情绪的把握，两人在细节上反复推敲，这首歌在香港完成录制。第二天，小柯带着20多轨完整的录音飞回北京，逐句播放给工作室的同事们听，全员投票选出最感人的一次演唱。  
  
电影《越来越好之村晚》讲述的是一段关于老百姓过年的故事，主题曲《 _稳稳的幸福_
》便传递出这种质朴的幸福感。“无论我身在何处，都不会迷途，我要稳稳的幸福……”陈奕迅的这首《稳稳的幸福》唱出了很多人的心声，尤其是女性朋友更想拥有一份安稳踏实的幸福。小柯在配乐上极为简单纯粹的手法，让旋律在平静淡雅中娓娓诉说，在几秒之内就沁入人心。
