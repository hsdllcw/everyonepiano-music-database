

**Say Something** 是新人组合 A Great Big World 首支作品。A Great Big
World是美国纽约的一支“黑马“乐队。由歌手Ian Axel和作曲Chad Vaccarino组成。后美国流行天后Christina
Aguilera也翻唱了此曲。

  
A Great Big World在Thevoice现场表演让这歌 _Say Something_ 走红，迅速爬升至Billboard
和iTunes下载榜冠军的心灵之曲。
