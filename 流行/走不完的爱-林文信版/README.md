_走不完的爱_
，台湾歌曲，选自《林文信12小时学会流行键盘教程》。原是歌手韦唯演唱的一首单曲，该曲收录在1989年发行的同名专辑中。韦唯，1963年9月28日生于内蒙古自治区呼和浩特市，中国内地女歌手、作曲人、国家一级演员。独特的音色和充满异域风情的长相，让很多人记住了她。
