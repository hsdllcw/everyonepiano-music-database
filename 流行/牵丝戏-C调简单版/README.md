

**牵丝戏**
是一首古风单曲，由银临、Aki阿杰演唱。整首歌融合了牵丝戏特有的悠扬节奏，让曲子充满了节奏感和灵气，恍然间看到一个戏偶在僵然而明媚的笑容里陪着灵动的手指翩飞舞跃。

主歌给人的感觉是恩婉而缠绵的，怀念与眷恋都尽数细细的筛入了时光的漏网，随心事挫骨扬灰。银临的声音把这种秀美发挥到了极致。歌声带人恍然间回到了那场灯火葳蕤，听到了扑火后哔哔啵啵的火星声，看到了水袖流云广的轻曼柔姿，也闻到了残存在空气里的，时光灰烬的味道。

副歌部分采用了惊艳的戏腔，阿杰用高吭的嗓音及丰沛的情感，充实了整个故事。彤云雪狂，灯火摇曳，白头翁用尽人生中全部美好的青葱岁月燃烧了戏偶的一寸青丝。而戏偶在茫茫天地间突然有了一份存着人性的牵恋。高歌也无畏，耀美也无畏，他存在的意义就是为了成全。成全少年那个一直走到老的梦想。他的存在，也实现了常人所不能实现的美好，寄托了所有人心里，关于明媚的全部希冀与渴望。
