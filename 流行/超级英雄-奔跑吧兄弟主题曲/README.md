

**超级英雄** 原为邓超多年前私人压箱底的歌曲，原本打算出EP。莫艳琳专门为邓超写的，选为《奔跑吧兄弟》主题曲后他参与了改编。

2014年10月7日晚，浙江卫视《中国好声音》第三季总决赛直播中，邓超意外登场演唱了一首名为《 _超级英雄_
》的歌曲。邓超一开嗓雅痞风十足，歌王风范兼幽默感惹得评委席上的那英不禁起立飞吻。富有感染力的表演令观众沸腾，网友激动刷屏：“恭喜本届好声音冠军邓超！”“超人动作简直太乖了！”而邓超在舞台上献给所有歌手的那番话，被当作格言疯狂转发：“每个人都要成为自己心目中的冠军！”

歌词下方是 _超级英雄钢琴谱_ ，希望大家喜欢。
