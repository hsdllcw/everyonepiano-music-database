

每个人的心中都有一个江湖，那里没有世俗的纷纷扰扰，有的只是轻剑快马，侠肝义胆的快意江湖，这样的江湖也是所有人最向往的，今天给大家推荐的就是侠气十足的古风歌曲《
**明月天涯** 》。

五音JW演唱的这首《 _明月天涯_
》可谓是这类歌曲中的经典之作。五音的歌声真的是少见的有着洒脱不羁的侠气，非常适合唱这种快意江湖的歌曲，尤其是唱到那句“恩仇趁年华轻剑快马”简直太帅了。
