何必执迷不悟，过后怀念光顾。虽然彼此忙碌，但爱依然不止。因你眼泪如雨，为你收起雨露。执迷你的脚步，不悟是否清楚。

所以，两个人的奔赴才有意义，一个人的执迷不悟一文不值。一定要懂得爱惜自己！
