

**甜甜的** 由周杰伦作曲并担任歌曲的制作人，收录在其2007年发行的专辑《我很忙》中。

因为《我很忙》之前发行的那张专辑被人批没有改变，没有进步，于是周杰伦在《我很忙》中这张专辑中力求突破，并专门为专辑创作了这首《甜甜的》
。在录制歌曲时，一开始周杰伦在考虑这首风格甜美的歌曲是否适合自己演唱，但是想到这首歌可以表达自己对学生时代那种单纯感觉的怀念；于是周杰伦决定自己来演唱。

《 _甜甜的_
》这首歌比周杰伦一般可爱的歌更多点俏皮，没有过多的前奏，一开头就唱出来，直接涌现甜甜的味道。《甜甜的》与《阳光宅男》刚好一刚一柔成为摇滚风格的两个呼应篇，但歌曲不时出现的一句“哦”的叫喊效果听起来很不舒服。
