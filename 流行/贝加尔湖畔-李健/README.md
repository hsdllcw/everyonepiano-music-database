_贝加尔湖畔_
是音乐人李健作词作曲并演唱的一首歌曲，收录在李健2011年发行的专辑《李健•依然》中。这是他继发行《音乐傲骨》专辑后，暌违两年发行的第五张全新创作专辑。据悉，为了此次新专辑，李健远涉万里前往英国伦敦，在名为Metropolis
Studio的录音棚里完成整张专辑的后期制作。蓝色男孩（BLUE）、邦乔维乐队（Bon Jovi）、迈克尔•杰克逊、Lady
GaGa等国际巨星的经典专辑就是在这个录音棚里录制的。

歌词下方是 _贝加尔湖畔钢琴谱_ ，希望大家喜欢。
