

**Teenage Dream** 由Katy Perry演唱，收录在其第二张主流厂牌专辑《Teenage Dream》中。凯蒂·佩里（Katy
Perry），1984年10月25日出生于美国加利福尼亚州圣巴巴拉，美国流行女歌手、演员兼词曲创作者。  
_  
Teenage Dream_ 这首歌曲在2010年7月22日
(2010-07-22)作为专辑的第二支单曲发行。接受采访时称，这是她自己亲手操刀制作的单曲，对她来说这支单曲非常特别，“我是在加利福尼亚的圣巴巴拉创作的这支单曲，那里是我的故乡，就像是我所有创作源泉的始发地。”
