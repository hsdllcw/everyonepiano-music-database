

歌曲《 **起风了** 》自推出后就一度成为了大家必听的神曲，甚至网友们一直单曲循环也百听不厌。

此曲表达的是一个少年年少的时候渴望观赏世界之大，在外奔走多年后，对于家乡和家乡的她十分想念，回到家后这里熟悉的一切和她依旧在原地等待，这才知道家乡和她是最好的。于是这位曾经的少年决定好好守护她，不再辜负这份真情。

由买辣椒也用券演绎的《 _起风了_ 》从器乐、编曲、到录音，百分百还原了温暖的日系旋律。

但是实际上，这首歌并不好唱，而由吴青峰演绎的版本则赋予了歌曲空灵美妙的意境，吴青峰雌雄莫变的嗓音，把歌曲里的青春翻涌唱得纯净无暇，不沾染俗尘。

有趣的是，买辣椒也用券的音色和吴青峰恰恰相反，她有着一副中性成熟，带着男声质感的女嗓，两人共同交织出了两版各有千秋，令人回味无穷的《起风了》。
