

**トイレの神様双手简谱** 和五线谱完全对应。

  

_トイレの神様_
是日本女性创作歌手植村花菜的第十张单曲，于2010年11月24日发行，是植村花菜的成名作和代表歌曲。为纪念怀念已故外祖母的感人歌曲，获得大众的强烈支持，及后衍生出同名小说、绘本、电视剧等作品。

  

歌词下方是 _トイレの神様钢琴谱_ ，希望大家喜欢。
