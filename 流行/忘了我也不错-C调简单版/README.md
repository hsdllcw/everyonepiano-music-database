

《 **忘了我也不错** 》由王心凌演唱。钢琴、吉他，加上王心凌的声音，如同铁三角般的主要架构，再点缀以些许的贝斯和打击乐器过门，组成了这首动人的抒情歌曲。

歌曲内容描述分手后由依恋到豁然割舍的心情转变，林夕文字功力丝丝入扣，短短数十字就将女性心情描写得淋漓尽致，为了这首歌曲，王心凌足足等待了快要一年，终于拿到林夕写来的歌词。

心凌非常独特的“气假音”演唱功力，透过这首歌更得到长足发挥，她擅长使用气声的音质唱功来表现假音的音域，这种混杂气声和假音的方式，非常适合表现歌曲中的遗憾、依恋、感叹、与哀伤的复杂情绪。也是让王心凌能够一步由“甜蜜教主”跨上“情歌女神”的独家秘方。
