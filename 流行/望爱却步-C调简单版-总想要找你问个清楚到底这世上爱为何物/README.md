

“总想要找你问个清楚，到底这世上爱为何物”出自歌曲《 **望爱却步** 》，由于洋演唱，发行于2018年7月4日。

歌曲表了一种错过的悔恨，面对爱情或友情没有及时表白，以致于后来陷入深深的相思痛苦之中，如果当初再多一点勇气，也许就会是另一种结局，一念花开，一念花落，年轻人不要怕错，只要我们没有错过，就不算辜负美好的青春。

_望爱却步双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
