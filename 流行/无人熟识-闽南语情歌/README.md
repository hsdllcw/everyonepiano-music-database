_无人熟识_
由曹俊鸿作词，华语女歌手江蕙演唱。一首自由欢快的闽南语歌，而江蕙更是唱出了真情实感，非常好听。出道20多年的台语歌后江蕙，堪称当今台湾乐坛身价最高的女歌手，从以前唱到现在，张张专辑都大卖。因所唱歌曲大部分都是悲伤情歌，与妹妹江淑娜合称“苦情姊妹花”，近年来苦情不再，应称“幸福姊妹。江蕙歌曲共426首，江蕙专辑63张。
