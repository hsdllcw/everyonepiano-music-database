

《爱是你我》是刀郎为印度洋海啸赈灾义演专门创作的歌曲。在2005年元月七日香港“爱心无国界”赈灾晚会上首唱。此歌曲原唱是刀郎、云朵和王翰仪。他说“这首歌虽然写得比较仓促，但希望能够唤起更多人的爱心”。在2010年北京春晚上，小沈阳与妻子沈春阳深情对唱刀郎的歌曲《爱是你我》，又感动了观众，此后此歌搜索量大增。
