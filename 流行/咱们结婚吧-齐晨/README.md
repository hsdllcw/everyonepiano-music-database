

**咱们结婚吧双手简谱** 和五线谱完全对应。

_咱们结婚吧_ 是素有“华语天籁男声”美誉的齐晨于2013年11月发行的全新主打单曲。该歌曲非主流曲风备受歌迷喜爱。

齐晨是华语乐坛偶像与实力俱备的新生代男歌手，颇具时尚气息的天籁男声。嗓音纯净，气质清纯，唱功扎实。可以穿透人灵魂的嗓音直击人心底最深处。

歌词下方是 _咱们结婚吧钢琴谱_ ，希望大家喜欢。
