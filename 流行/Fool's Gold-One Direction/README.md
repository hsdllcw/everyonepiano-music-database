

**Fool's Gold** 是英国组合One Direction的单曲，收录于专辑《Four》中。单向组合（ _One Direction_
，简称1D）是一支来自英国与爱尔兰的男子组合。包括Louis Tomlinson、Harry Styles、Liam Payne、和Niall
Horan四位成员。

歌词下方是 _Fool's Gold钢琴谱_ ，希望大家喜欢。
