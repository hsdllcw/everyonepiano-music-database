

**相同天空下，不同时间里**
是韩国tvN于2012年4月18日起播出的水木迷你连续剧《仁显王后的男人》的原声。《仁显王后的男人》由《吸血鬼检察官》金炳株导演执导，《顺风妇产科》、《搞笑一家亲》、《Coffee
House》宋在贞作家执笔。讲述了朝鲜王朝校理金鹏道（智铉寓 饰）通过时光隧道从朝鲜时代穿越到300年后的现代，遇到了女演员崔熙进（刘寅娜
饰）后发生一连串刻骨铭心的爱情故事。

  
另外， _相同天空下，不同时间里_ 是时间韩国Big Hit
Entertainment在2007年推出的三人男女混合组合8Eight中珠熙的歌曲。成员包括队长李贤、白灿、珠熙，是挑战MBCShow
Vival节目而得到出道机会的男女组合。

  
歌词下方是 _相同天空下，不同时间里钢琴谱_ ，希望大家喜欢。
