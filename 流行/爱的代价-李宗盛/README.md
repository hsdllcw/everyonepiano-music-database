

**爱的代价**
是一首由李宗盛作词作曲，张艾嘉演唱的华语流行乐坛的经典歌曲。歌曲优美的旋律，诗意的歌词带着对逝去的爱的淡淡感伤和对现实的正视，表达了人们心灵深处对爱的希冀和渴望。此外梁咏琪、陈淑桦、刘若英、丁当、五月天玛莎和词曲作者李宗盛等人都演唱过这首歌。

  
歌词下方是 _爱的代价钢琴谱_ ，希望大家喜欢。
