

**我的中国心**
是张明敏演唱的一首爱国主义歌曲，由黄沾作词，王福龄作曲，收录在张明敏1982年发行的同名专辑《我的中国心》以及2005年发行的专辑《明敏龙情》中。1984年，在中央电视台春节联欢晚会上，张明敏演唱了《
_我的中国心_ 》；同年这首歌曲获得了中国音乐协会歌曲编辑部颁发的第三届神钟奖。

  
歌词下方是 _我的中国心钢琴谱_ ，希望大家喜欢。
