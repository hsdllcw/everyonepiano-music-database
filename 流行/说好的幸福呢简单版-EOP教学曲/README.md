_  
说好的幸福呢_
是周杰伦演唱歌曲，收录在其2008年的音乐专辑《魔杰座》中。周杰伦与方文山写出这首今年最遗憾的情歌，当心里问着这句话时，幸福，早已经消失不见了。这是一首令人听了就会想哭的歌，钢琴哽咽着，大提琴低吟着，唱分手的伤，爱情离开的惆怅；那些爱过的感觉都太深刻，都还记得；方文山以疑问句写爱情的不舍与后悔，只是所有的疑问得不到对方的回答，回忆的音乐盒还旋转着，要怎么停呢？只剩下钢琴声，代替了回答。
