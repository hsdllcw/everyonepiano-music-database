

**The End of the World** 是美国著名的女乡村音乐歌手Skeeter Davis
在1963年演唱的歌曲。此曲被评为当年最受欢迎的歌曲，被无数人翻唱过，但无人能及她原唱的魅力，这首歌也是她唯一的传世之作。2009年的电影《美国情事》选择《the
end of the world》作为片尾曲，直抵人心的伤感。

  
歌词下方是 _The End of the World钢琴谱_ ，希望大家喜欢。
