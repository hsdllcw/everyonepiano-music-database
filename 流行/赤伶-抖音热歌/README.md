

戏子不入戏，我却入了你的戏。我国古代戏曲经过上百年的沉淀，到现在依然在坊间流传，可见这些歌曲融入的情感至深，歌词写得动人心魄，酣畅淋漓。抖音上很火的《
**赤伶** 》唱出了台上人与台下人、戏中人与戏外人的伤感离别之情，曲调婉转绵长、悲戚伤情。

_赤伶_ 是古风歌手HITA于2018年9月27日发行的歌曲，收录于《赤伶》专辑。

歌词下方是 _赤伶钢琴谱_ ，大家可以免费下载学习。
