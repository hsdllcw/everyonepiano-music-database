

### 千里之外简介：

歌曲《千里之外》周杰伦找来美声天王费玉清合唱话题十足，首播消息曝光后，不但年轻听众频频打来询问，还有不少的年长的听众打来询问首播时间，可以这次的合作引起不少的关注。写词还是找来老搭档方文山，流畅的文笔帮歌曲加分不少，歌词细腻描写对已逝爱的感触与感伤，文字的描述勾勒不少的的画面。
