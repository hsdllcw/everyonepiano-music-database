

《 **那天以后** 》是韩剧《火鸟》插曲，很好听的一首歌曲。

《火鸟》片 长：26集（韩国原版剪辑）；地 区：韩国；出 品：MBC；首播时间：2004年4月5日-6月29日（月火剧）；接档：大长今。
