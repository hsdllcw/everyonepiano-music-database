

**野孩子双手简谱** 和五线谱完全对应。

_野孩子_ 是杨千嬅演唱的一首粤语歌，作词黄伟文，作曲雷颂德。收录在其2001年9月22日发行的专辑《Miriam》里。

专辑封面以杨千嬅最喜欢，也是幸运色的紫色为主，在限量版中千嬅还亲自绘画的卡通肖像及不倒花，她希望自己可以像不倒花一样，并表示日后再出专辑时还会以不倒花为她的个人标志，比喻自己的歌可以留存长久。

歌词下方是 _野孩子钢琴谱_ ，大家可以免费下载学习。
