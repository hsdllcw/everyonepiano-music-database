《 _你的背包_
》讲述的是一个纯净的但又充满遗憾的初恋故事。男孩女孩彼此相爱，情感真挚而纯洁，但因为年少张狂，或许只是一点点的误会和争吵，却最终导致了两人的各奔东西。女孩与男孩在机场告别，女孩借给男孩一个背包。女孩走了之后再也没有回来，男孩却每天背着这个背包上班。多年后，男孩历经风雨，人世沉浮，走过了少不更事的年纪，才更加体会这段情感的可贵，背包熟悉了男孩的汗，成为他生命的另一半。只是，过去的早已过去，无法再追寻，只有那个背包———那个他不想归还的、装载了他的青春、真情、纯洁、遗憾还有伤感的背包还依稀带来往日的影子，不知不觉，背包就这样溶入了他的生命，并和那些难以忘怀的往事一起。林夕的歌词并不多，情意单纯，寥寥百字，却也道尽了不少的沧桑，即将一个负着背包“情债”的苦主跃然歌内，让人深受感动。
