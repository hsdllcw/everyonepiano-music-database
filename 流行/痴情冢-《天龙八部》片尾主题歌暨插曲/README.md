

**痴情冢EOP** 由网友曳影制作，感谢曳影对EOP的支持！

《痴情冢》是钟汉良版天龙八部插曲，原片尾曲，由贾青（剧中饰阿朱、阿紫）演唱。该插曲在2014年1月4日晚，钟汉良版天龙八部在湖南卫视播到聚贤庄大战时以完整版出现，之前都以短篇插曲形式在剧中出现。  
  
歌词的下方还提供了 _痴情冢钢琴谱_ ，欢迎感兴趣的朋友下载使用。
