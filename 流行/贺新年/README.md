

**贺新年**
是卓依婷演唱的曲子，自幼聪明好学，3岁拜师学歌，5岁登台即红遍东南亚，拥有无数歌迷。歌曲风格各异，以柔情甜歌为主。听众年龄层次跨越度非常广泛，属于大众型歌手。
