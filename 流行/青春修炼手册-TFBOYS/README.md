

**青春修炼手册双手简谱** 和五线谱完全对应。  
  
_青春修炼手册_ 是TFBOYS于2014年7月24日在QQ音乐发布的歌曲。歌曲由王韵韵作词、刘佳谱曲、编曲（前奏借鉴其编曲的歌曲《爱的魔法》）。  
  
TFBOYS作为国内最年少偶像团体，成长便成为了这只团体伴随着的话题。这次的《青春修炼手册》更是直接以调皮搞怪的音乐风格告诉粉丝，原来人见人爱的正太是这样修炼成长的。在新单曲中，TFBOYS仍然以自己的年龄段特有的思考模式讲着他们心中理解的成长到底是什么。而新歌的创作更邀请到了国内知名创作人刘佳倾力打造。整首歌活泼轻快的旋律简单明了，这个夏天，就在这样一首歌里领略这三位少年的成长，也回味属于自己的成长的烦恼吧。  
  
歌词下方是 _青春修炼手册钢琴谱_ ，希望大家喜欢。
