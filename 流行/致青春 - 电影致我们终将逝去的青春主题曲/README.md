

**致青春双手简谱** 和五线谱完全一致，是EOP教学曲。

《 _致青春_
》是电影《致我们终将逝去的青春》主题曲。电影《致我们终将逝去的青春》是赵薇的导演处女作，也是其在北京电影学院导演系研究生的毕业作品，电影的粗剪版被北京电影学院评为99分，新创北京电影学院导演系硕士生毕业作品历史最高分记录。影片改编自著名作家辛夷坞的同名小说，汇集了赵薇圈内众多巨星好友。

《致青春》，由音乐人窦鹏作曲，编剧李樯作词，并且由天后王菲献唱。歌词中一句“这年华青涩逝去，明白了时间”道尽对逝去青春的缅怀与怅惘。王菲在演唱中，声音依旧空灵，更巧妙地弱化了每一句的尾音，让整首歌听起来“菲”味十足，有点当年《寓言》专辑的影子。

2013年4月9日赵薇在微博中公布了《 _致青春_
》的歌词，并配上了王菲酷酷的封面照。王菲随即转发并调侃自己的照片称：看，多么不羁的脸。如果你喜欢这首钢琴曲，那就下载 **致青春五线谱** 吧！
