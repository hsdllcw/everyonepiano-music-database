

《 **给自己的情书** 》是王菲演唱的歌曲，亦是电视剧《婚前昏后》的主题曲。该曲的普通话版为《笑忘书》，均收录于王菲2000年发行的专辑《寓言》中。

这首歌一字一句像是王菲在柔声规劝：不要灰心不要失望不要难过，要爱护自己慰藉自己，使自己变得强壮，变得闪闪发光，才能遇到相衬的人，才能帮助你爱的人。

同时，此曲的钢琴前奏非常感人，是一首慢版舒情歌曲。歌词描述了女人离婚后要独立自强的心声，粤语版的《 _给自己的情书_
