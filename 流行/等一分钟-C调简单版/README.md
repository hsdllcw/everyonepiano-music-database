

**等一分钟**
是徐誉滕自己作词作曲，根据自己的真实故事改编的。流畅的旋律，磁性的男音，居然有些似曾相识的感觉。迅速搜寻着脑海里对伤心情歌的记忆，却发现原来熟悉的是自己曾被伤过的心。

其实，徐誉滕一开始的本质工作并不是一名歌手，而是一名幕后的创作人，专门为一些比较有名气的歌手填词作曲。对于《等一分钟》这首歌来说，很多人都说没有亲身历经过一段刻骨铭心的爱情故事，不可能写得出这样的歌词。的确如此，徐誉滕能写出这首歌来，就是因为感情上受到了伤！可能因为这些词都是自己想对恋人说的话，所以徐誉滕只用了半个小时的时间，就完成了歌词的创作。

而在汶川地震中，曾有一个男孩被埋在废墟中，他一遍遍地听着储存在手机里的《 _等一分钟_
》，熬过了整整77个小时，徐誉滕也因此被邀请加入“跨越太平洋的爱”大型公益演出，回报来自大洋彼岸的爱心。
