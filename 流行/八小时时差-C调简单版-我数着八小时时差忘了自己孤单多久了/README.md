

“我数着八小时时差，忘了自己孤单多久了”出自歌曲《 **八小时时差** 》，是“深情磁嗓”郑鱼的一首失恋情歌，超级催泪！

数着八小时的时差，查着另一座城市的天气，想着城市里的你……如果当初学会珍惜，现在是不是还能牵你的手？
