

“你爱自由胜过爱我，可是我偏偏 偏爱缠绵胜过洒脱”，你能说出这首曲子的名字吗？

这首歌曲的歌名叫做《 **过活** 》，是由 **月野宁** 作词、作曲， **犹八音舍** 编曲， **高鱼** 演唱的一首歌曲。4月7日 **高鱼**
还在抖音发了一个征集此曲名字的视频，引来众多吃瓜网友的献名。

在2020年4月9日歌曲发行，正式名字叫《 **过活**
》，吃瓜群众纷纷表示喜欢，也引来众多高手的翻唱。不同的版本有不同的风格，当然一向追逐美好的小编还是喜欢最靓丽的 **戴羽彤**
的版本，甚至连网站的这首谱子的图片都用到了她的截图。

“ **过活** ”连高学历的小编都很难理解其中的含义，“ **活过** ”到很容易理解，不过当你看到这句歌词：

你的呼吸只是一种经过  
只是我偏偏  
把它当成氧气 **过活**  
越来越难过

是不是就不懵逼，豁然而开朗了？！![](/Public/editor/plugins/emoticons/images/22.gif)

爱情很美好，但男人爱自由，女人爱缠绵，相拥之后，互相成全，才有意义。把一个人的存在当作氧气过活，必定会爱的卑微。接受结局，云淡风轻，偶尔想起，也不失一种美好。

看到这里，是不是单身狗们会长长的舒了一口气：“与其知道一定分手，当初又何必开始？做只单身狗，它不香吗？”
