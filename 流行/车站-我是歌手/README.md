

**车站双手简谱** 和五线谱完全对应。

_车站_
是李健填词，演唱的音乐作品。此曲原歌名是日本歌曲《駅》，是竹内玛莉亚为中森明菜创作的歌曲，被收录在中森明菜1986年12月24日发行的专辑《CRIMSON》中。该曲由竹内玛利亚作曲，俞铮和陈少琪填词。2015年4月3日在《我是歌手》2015巅峰会李健重新填词该曲名为《车站》并翻唱该曲。

悠扬的曲调凝结着歌手李健亲自填词的心意，简单与诗意贯穿始末。在手风琴和吉他的简单伴奏中，沉静温暖的歌声缓缓而出，似乎将现场观众视作了多年的知己好友，不炫技不花哨，却自有一番天然去雕饰的动人滋味，让你不知不觉让你在诗般的意境中道尽人生百味。
