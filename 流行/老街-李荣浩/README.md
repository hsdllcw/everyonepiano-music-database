_老街_ 是李荣浩演唱的歌曲，收录在专辑《小黄》中。致敬周杰伦的歌曲《上海1943》的风格和调调。

从曲调来说，主打歌《老街》旋律朗朗上口，采用了三段体的曲式风格感情也层层递进，一开始的竖琴营造出安静的氛围，又以平缓感人的曲调贯穿其中，舒服的音乐元素编制让我们感受到这位音乐独行侠别样的细腻情感，而李荣浩质朴感性癿嗓音更是锦上添花，将自己对老街的那份说不出的伤感表现的淋漓尽致。

每个人都有对故乡的记忆，思乡的情结犹如人的生物钟一般在某一时刻准时袭来。老街让我们找到了已经离开多年的故乡的感觉，更加真实癿回到我们的眼前，安静纯朴的家乡生活回忆是可以平静任何浮躁的人儿的。老街那种种熟悉的气息，有时像是蹒跚学步时，有时像是第一手牵手的稚气…
