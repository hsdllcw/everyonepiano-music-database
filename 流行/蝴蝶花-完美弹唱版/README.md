

**蝴蝶花双手简谱** 和五线谱完全对应。

_蝴蝶花_
是水木年华演唱歌曲，收录于其《一生有你》专辑中。水木年华，大陆男子民谣歌唱组合，“水木年华”的名字取自于“水木清华”，前后共有四位成员，均是清华大学毕业生，目前由卢庚戌和缪杰组成。

歌词下方是 _蝴蝶花钢琴谱_ ，希望大家喜欢。

### 蝴蝶花歌词：

是否还记得童年阳光里  
那一朵蝴蝶花  
它在你头上美丽的盛开  
洋溢着天真无瑕  
慢慢地长大曾有的心情  
不知不觉变化  
痴守的初恋永恒的誓言  
经不起风吹雨打  
岁月的流逝蝴蝶已飞走  
是否还记着它  
如今的善变美丽的谎言  
谁都得学会长大  
早已经习惯一个人难过  
情爱纷乱复杂  
想忘记过去却总又想起  
曾经的无怨无悔  
谁能够保证心不变  
看得清沧海桑田  
别哭着别哭着对我说  
没有不老的红颜  
谁学会不轻意流泪  
笑谈着沧海桑田  
别叹息别叹息对我说  
没有不老的红颜
