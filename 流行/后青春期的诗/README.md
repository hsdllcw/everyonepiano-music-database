

五月天专辑《后青春期的诗》由相信音乐唱片公司发行于2008年10月，还未正式发片前已破10万销售大关。

### 《后青春期的诗》歌词

当烟雾随晨光飘散  
枕畔的湖已风乾  
期待已退化成等待  
而我告别了突然

当泪痕勾勒成遗憾  
回忆夸饰着伤感  
逝水比喻时光荏苒  
终於我们不再  
为了生命狂欢  
为爱情狂乱  
然而青春彼岸  
盛夏正要一天  
一天一天的灿烂

(阿信口白:  
然后呢一起走吧)

谁说不能让我  
此生唯一自传  
如同诗一般  
无论多远未来  
读来依然一字一句  
一篇都灿烂

让天空解释着蔚蓝  
浮云定义着洁白  
落花铺陈一片红色地毯  
迎接我们到未来  
精彩未完的未来  
~~end~~
