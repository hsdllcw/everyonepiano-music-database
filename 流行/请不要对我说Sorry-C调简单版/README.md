

《 **请不要对我说Sorry**
》由中国内地女歌手何洁演唱。这是一首极富感染力的悲伤情歌，讲述了一个努力去爱但仍然遭遇分手的女生，在放手与回忆中痛苦挣扎的故事。

同时这也是每一个曾经面对过爱情失败的人都会经历的过程。擅长演绎情歌的何洁在录音棚内一气呵成，完整地记录下制作人对这首歌曲演唱情绪要求的细微之处，令听者为之恻隐，丰富细腻的情感搭配钢琴为主调的编曲，娓娓道出爱情故事中的美好与遗憾。

另外这首歌曲在一贯的“减法”企划要求下，编曲依然秉承着一切从简的原则，只保留最基础的旋律与配器，令整首歌听上去简单流畅。
