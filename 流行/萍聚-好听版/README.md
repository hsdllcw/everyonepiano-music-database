

《 **萍聚** 》是一首由李翊君、李富兴演唱的经典老歌，唱出了一代人的回忆，满满的感动！同时，这也是一首感慨即将离别的歌曲，充满了人情味。

歌曲表达的意思是大家相遇而结缘，就不要想太多，珍惜当下的感情，即使分别也留下了美好的回忆。正如歌词所唱：

别管以后将如何结束，至少我们曾经相聚过  
不必费心地彼此约束，更不需要言语的承诺  
只要我们曾经拥有过，对你我来讲已经足够

一首萍聚，平添了多少忧思；一些邂逅，释然了多少缘分；人的一生有许多的回忆，只愿彼此的回忆中有彼此存在就行了，只在乎曾经拥有，不在乎天长地久……记忆里，多收藏别人的好，生活里，多温暖自己的心。总有一天你会明白，你惊艳不了谁的人生，但可以温柔自己的岁月！
