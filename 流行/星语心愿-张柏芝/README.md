星语心愿_
是由高雪岚作词，金培达作曲的。收录于张柏芝的同名专辑《星语心愿》。张柏芝，香港著名演员、歌手。1980年5月24日出生于香港油麻地。1998年被周星驰发掘，成为《喜剧之王》的女主角，因此一炮而红。因其形象清丽秀美，有着四分之一的混血（妈妈是混血），成为90年代末期的一代玉女掌门人。2002年主演韩国电影《白兰》获得韩国电影大钟奖最佳女主角提名，成为首位入围韩国三大电影节的华人影星，令她在韩国知名度大增。2004年凭借影片《忘不了》斩获第二十三届香港电影金像奖最佳女主角（影后）。2011年，张柏芝签约新东主AEG新亚洲娱乐，为期八年，拥有经纪全约，并成立“张柏芝工作室”，进军唱片制作、影片投资等幕后工作。  
  
歌曲《星语心愿》从被唱出，至今仍是KTV中点播率极高的歌曲。
