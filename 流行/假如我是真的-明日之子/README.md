_假如我是真的_
是明日之子3学员洪一诺翻唱邓丽君的经典歌曲。这首歌曲不光词好曲美，而且寓意深远，发人沉思，读白了尘世为人之难和对如水一般人生的愿望，歌曲道出了一个人的心境，很真实，正所谓“曾经沧海难为水”；歌曲，柔婉细腻，干净甜美，让人听了非常的舒服，纯粹是一种享受，就好似涓涓细流浸润在了口渴人的心田，令人难以忘怀！
