专辑《爱到无可救药》中的《 _我以为_
》是品冠最为钟爱的情歌。它是制作期最长、歌词讨论最久、品冠也唱最久的真诚制作。由品冠自己作曲制作，在深刻动人的旋律线中，铺陈爱情的无奈感伤……此外秦辉、范冰冰和王珞丹也有同名的单曲。  
  
《我以为》描绘三角恋情里，最认真却最无力的一方，无限付出却换来无数失望，那份无语问苍天的深刻感受。品冠再三调整尝试，以最真诚的声音表现，诠释一个真挚而无奈的爱情心情。《我以为》前奏即以带着淡淡哀伤的钢琴声进入，慢慢带进歌词故事的开端，品冠娓娓陈述；中段以后节奏与弦乐进来，将无奈与哀愁带到一个更宽阔的境界，故事渐渐走到尾声，无奈感却亦发强烈，到副歌时是几近无声呐喊的低吟。
