

**一路上有你双手简谱** 和五线谱完全对应。

_  
一路上有你_ ，演唱者张学友，出品于张学友1993年个人专辑《吻别》当中。至今保持新加坡唱片销量最高记录（60多万张），
并获得香港电台十大中文金曲奖。张学友是香港乐坛“四大天王”之一，创造了亚洲歌手举办演唱会场次的纪录，并被吉尼斯世界纪录认可，曾三次夺得“亚太区最受欢迎香港男歌星和四次夺得“金曲金奖”。另有周杰伦同名演唱版本。

  
歌词下方是 _一路上有你钢琴谱_ ，希望大家喜欢。
