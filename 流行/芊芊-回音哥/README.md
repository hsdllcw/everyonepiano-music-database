

自上一首《海绵宝宝》后，时隔3个多月,回音哥带来自己的最新力作——《 **芊芊**
》。这是回音哥首次尝试古风歌曲。诗意的词句和古韵的曲调，配上回音哥低沉沙哑混合着慵懒味道的声音，带着听众穿越回千古之前，诉说一段深藏已久的爱恨情仇，流露出淡淡的不舍之情。  
  
《 _芊芊_
》是回音哥即将发行的首张个人专辑的预热单曲，酝酿已久的专辑最终将于六月与大家见面。选择如此具有中国古风特色的歌曲为新专辑暖身，让人对即将发行的新专辑更多了几分期待和遐想。  
  
回音哥，本名不详，中国四川成都人，网络歌手，因在网络演唱歌曲而迅速走红，拥有极富感染力的磁性嗓音。为人低调，只闻其声不见其人
，颇具神秘感。在网络发布的原创歌曲有：《对不起，我还爱你》、《逃》、《一个月的雨》、《海绵宝宝》、《 _芊芊_ 》、《爱与不爱之间》等，受到众多网友的喜爱。
