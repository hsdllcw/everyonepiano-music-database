明天，你好_
是由牛奶咖啡演唱的一首清新风格的歌曲，由王海涛作词、格非作曲。歌曲收录在2011年7月发行的EP《Lost&Found去寻找》中，为专辑中的第二波主打曲。  
  
牛奶咖啡的这首歌曲的灵感来源于当代社会越来越快的生活节奏与人们日益增加的生活压力，是送给伙伴的送别词，充满了眼泪和拥抱的美好回忆。歌曲由词人王海涛填词，华谊兄弟出版发行。  
  
在上张专辑《给你点儿颜色》大获成功后，牛奶咖啡希望通过音乐向人们传达一种更为积极主动的生活态度，同时表达一种心灵上的态度，指引当下的年轻人坚持自己的梦想，坚信黑暗之后就是黎明，坚信找到自我便可以找到未来，坚信内心一定会给自己前进的希望。牛奶咖啡将自己的成长心得借音乐之手合盘托出，于是诞生了这首《明天，你好》。  
  
《明天，你好》是一首送给惜别的伙伴，充满了眼泪和拥抱回忆的清新抒情之作，代表着长大的含义。歌曲通过清新温柔的乐风，向人们传达着一种美好的愿望，积极面对生活，快乐生活。穿插其中的口琴为歌曲增色，悠扬的口琴音色给歌曲蒙上一层淡淡的怀旧气息，仿佛那些离别的情绪重新回到了眼前。  
  
歌词内容来自于词人内心的感受：越美好的事物，越害怕得到，因为美好只在得到的前一秒。这首歌鼓励了那些对“明天”充满希望的人们，鼓励他们对生活充满希望和向往，同时表达了对现在年轻人把每一天过成崭新的、使生活快乐起来的希望。
