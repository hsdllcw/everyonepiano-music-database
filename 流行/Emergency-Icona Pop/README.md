

**Emergency双手简谱** 和五线谱完全对应。

_Emergency_ 是瑞典女子DJ团体Icona Pop于2015年发行的一首歌曲。该乐队是由在斯德哥尔摩长大的Caroline Hjelt和Aino
Jawo于2009年组成。它的成立影响了电气浩室、庞克和独立流行类型的音乐风格。至今，她们已在瑞典、美国、德国、意大利和伦敦等国家表演过，并且受过瑞典及外国媒体的专访。
