

**Faded In My Last Song** 是韩国男子演唱组合NCT的先行分队NCT U的歌曲。

具体是由泰一、JOHNNY、悠太、TEN(李永钦)、LUCAS(黄旭熙)、仁俊、楷灿和志晟参与演唱的。钢琴旋律和Heavy的贝斯、作为铺垫音色的弦乐器和强烈中毒性的副歌相互调和激发出小调氛围，以钢琴作为物件加以活用的歌词描绘出为心爱的恋人做最后一次的演奏的内容，将悲伤的情感发挥至极致。
