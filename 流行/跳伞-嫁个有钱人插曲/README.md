

**跳伞双手简谱** 和五线谱完全对应。

_跳伞_
是由郑秀文、任贤齐等人主演的《嫁个有钱人》插曲，演唱者为香港歌手郑秀文，并收录于所属专辑《温柔》中，发行时间为2002-01-05。所属公司是华纳唱片。

歌词下方是 _跳伞钢琴谱_ ，希望大家喜欢。
