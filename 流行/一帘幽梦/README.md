

歌曲《 **一帘幽梦** 》，由萧丽珠原唱，是1975年琼瑶为电影《 **一帘幽梦** 》所写的同名主题曲。1996年电视剧版《 _一帘幽梦_
》播出，这首歌改由许茹芸演唱，作为本剧的片尾曲。2007年，翻拍电视剧《 _又见一帘幽梦_ 》再次以这首歌作为片头曲，改由同恩重新演绎。
