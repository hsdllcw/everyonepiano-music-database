

**Synchrogazer** 这首歌曲是由上松范康（Elements Garden）编曲，水树奈々(Nana Mizuki)演唱的。

_歌曲Synchrogazer_ 所属同名专辑Synchrogazer的第26首单曲。

另外， _Synchrogazer_ 是电视剧[战争女神的绝唱Shinfogia]的片头曲，包括富士电视台的两部电视剧主题歌在内，收录了3首曲子。
