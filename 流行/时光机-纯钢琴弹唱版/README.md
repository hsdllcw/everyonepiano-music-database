_时光机_
是五月天演唱的一首歌曲，由主唱阿信作词作曲，收录在2003年发行的同名专辑《时光机》中。2003年，五月天凭借该歌曲获得了第十五届台湾金曲奖最佳乐团奖。

这首歌代表着五月天两年跟歌迷不见的感受，其实是怪兽母亲发病后昏迷，阿信写给他的歌。柔美动人的慢歌，配乐非常的婉转，有种深入人心扉的悲伤让人喘不过气来，尤其是一句“最后又是孤单到天明”有种让人落泪的无奈。
