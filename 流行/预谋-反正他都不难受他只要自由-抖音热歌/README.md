

抖音“反正他都不难受，他只要自由”这首歌叫做《 **预谋**
》，原唱者是许佳慧，由代理仁填词谱曲、金泽编曲，于2012年1月4日发行，之后在抖音上被翻唱走红。许佳慧是中国内地女歌手，她在娱乐圈中被称为“魅惑小天后”。

虽然这首歌叫做“ _预谋_ ”，不过讲述的却是关于爱情的故事。这首歌唱出了很多人的心声，也唱哭了很多人，有些情感确实需要一些声音去表达。

歌词下方是 _预谋钢琴谱_ ，大家可以免费下载学习。
