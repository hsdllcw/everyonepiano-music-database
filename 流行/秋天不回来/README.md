

《 **秋天不回来** 》这首歌由王强演唱；发行于2006年12月，该歌所属同名专辑《 _秋天不回来_ 》。在百度流行金曲排行榜十周排名第一。

王强，著名歌手，1997年毕业于武汉同济医科大学药学本科班，2000年获得湖北人民广播电台举办的翻唱大赛获得铜奖。2002年参加内地原创歌曲大赛，获得优秀创作奖。
2003年从湖北某医疗单位辞职出来专职从事音乐创作及歌手。
2005年参与广州著名音乐人胡力《狐狸爱上熊》专辑的制作和宣传（企宣），参与部分歌曲和声。2005年12月28日首次参加中国移动彩铃唱作大赛广东省赛区总决赛，获得最佳原创歌曲奖。如果喜欢这首曲子，下载
