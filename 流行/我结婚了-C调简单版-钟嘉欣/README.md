

《 **我结婚了** 》这首歌于2016年发表在专辑《星空爱情》里，由钟嘉欣作曲且演唱，王祖蓝作词，唱的是待嫁女生的一些心理感受。

结婚对于女生来说是一个很有压力的事情，虽然新时代的女性中，对结婚有新的看法和想法，但从传统意义上说，结婚对女生而言，是离开自己的爸爸妈妈，离开自己熟悉的一切，去往别人家生活的一个经历，为了心爱的人，去往陌生的地方，接受一切未知的事。

这首歌作为钟嘉欣结婚时的一首歌曲，不仅唱的是她自己，也是唱给每一个待嫁的新娘。
