》就是这样一首歌。歌声打开尘封多年的心琐，让久违的情怀重新复活。真爱不需要承诺，只用心灵去触摸，是歌声点燃爱情之火，爱的旋律是永远唱不完的歌。

人海茫茫，人与人之间能够相遇相知，或是相亲相爱，是必然，也是偶然。冥冥之中，自有一种说法或叫缘分的东西。君住江之头，妾居江之尾，同饮一江水，这是地缘。如果因为爱的倾慕，结成连理，那就叫千里姻缘一线牵了。
