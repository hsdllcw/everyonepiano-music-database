

**演员** 是由歌手薛之谦创作并演唱的一首歌曲，收录在其2015年5月发行的EP《绅士》和2016年发行的专辑《初学者》中。

此曲唱腔更趋成熟，与早期擅长的抒情R&B风格不同，《 _演员_
》的唱法对内心感受的细节处理更丰富、更有层次感。这一次，他不仅负责作品的创作，还担任制作人，这是对自己音乐全盘把控的标志。
