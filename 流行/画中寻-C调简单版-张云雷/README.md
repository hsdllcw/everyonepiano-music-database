

由张云雷演唱的《 **画中寻** 》于2021年6月15日上线。张云雷清澈透亮、温雅柔和的嗓音，娓娓道来他眼中的诗画中国，引着我们走入水墨丹青的深处。

《 _画中寻_
》是一首中国风歌曲。在张云雷的演绎中，美是吹皱的湖面，是缥缈的雾气，是文人的撇捺，是坠落的雨滴。精通传统曲艺的张云雷在歌曲中加入戏腔，让歌曲凭添叙事感，把拳拳情意揉碎，镶嵌在颤悠悠的尾音里。张云雷将声画结合成完整的画面呈现给聆听者，可谓“远看山有色，近听水无声”。
