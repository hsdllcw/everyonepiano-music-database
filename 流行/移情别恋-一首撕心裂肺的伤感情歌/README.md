

**移情别恋** ，这首歌词曲创作、演唱都是郭峰，他是才华横溢的创作型音乐人！

正如歌名，这是一首撕心裂肺的伤感情歌。恋爱中，一方移情别恋，喜欢上了别人，真的是让人伤心。但是转念一想，恋爱时对方移情别恋并不是很可怕，总好过结婚之后“家中红旗不倒，外面彩旗飘飘”或“春色满园关不住，一枝红杏出墙来”！有没有道理？
