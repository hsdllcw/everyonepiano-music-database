《 _我愿意_
》是王菲演唱的一首歌曲，由姚谦作词，黄国伦作曲，收录于1994年新艺宝发行的专辑《迷》中，是其早期代表作之一。该歌曲曾获得1994年“全球华语音乐榜中榜中文金曲”和2003年"台北之音百首经典歌曲网路票选第一位"等奖项。  
  
该曲的歌词由姚谦完成，其称，当年填写这首词的灵感竟然来自于八卦杂志上的一组照片——王菲和窦唯在一起时，一早出门，去公共厕所倒夜壶。那时候王菲的艺名还是王靖雯，有制作人向姚谦邀歌，姚谦即以此为灵感，创作了《我愿意》的歌词。作曲由黄国伦完成，其称，写这首歌的初衷是送给上帝的礼物。  
  
《 _我愿意_ 》是一首很深情，感动人的歌曲，同时是朋友K歌的定番曲目，也是个人喜欢的音乐风格之一。歌词提供了免费 **我愿意钢琴谱** ，欢迎大家使用。
