

**算什么男人**
是周杰伦演唱的一首歌，由周杰伦作词作曲。收录在周杰伦2014年制作发行的专辑中。周杰伦创作的词曲作品，歌词是以自己与自己内心对话的方式来写，自己骂自己对于爱情不能好好把握、将心爱的人拱手让人的心情。周杰伦首次用以往没玩过的低沉唱腔，演唱男人内心的极度疼痛。周杰伦与林依晨在MV里饰演一对一起在咖啡厅工作的情侣，两人有不少对手戏，像是咖啡厅里隔着玻璃窗画爱心，就是歌词的写照，“我的温暖、你的冷漠，让爱起雾了。如果爱心画在起雾的窗，是模糊？还是更清楚？”周董的浪漫透过文字、影像与音乐，彻底融化歌迷的心。

  
周杰伦与林依晨在《 _算什么男人_ 》MV中饰演一对一起在咖啡厅工作的情侣，两人有不少恩爱互动。

  
MV中，小两口的生活很甜蜜，不管是一起慵懒地在沙发看电视，还是在厨房打情骂俏，一起玩桌式足球……都尽显浓情密意。然而男主角周杰伦却经常忽略女生的用心，后来女主角林依晨生活中出现了一个贴心的Darren邱凯伟，在女主角被Darren追走了之后，男主角只能徒留懊悔。
