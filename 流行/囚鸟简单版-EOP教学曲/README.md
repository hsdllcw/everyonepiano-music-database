

**囚鸟双手简谱** 和五线谱完全对应，是EOP教学曲，很适合新手练习。

_囚鸟_ 是1996年彭羚所出的这张专辑是一张精品。由张宇、黄舒骏、巫启贤、袁惟仁等名匠共同打造的一张国语专辑，非常耐听。
透过彭羚极富磁性的嗓音，将每一首歌曲唱得娓娓动听。其中的同名歌曲《囚鸟》更是成为一首经典。

歌词下方是 _囚鸟简单版钢琴谱_ ，大家可以免费下载学习。
