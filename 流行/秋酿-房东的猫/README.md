

**秋酿双手简谱** 和五线谱完全对应。

_秋酿_ 是由房东的猫演唱的一首民谣歌曲，由房东的猫填词、谱曲、编曲、制作，收录于同名专辑《房东的猫》中。

房东的猫是由主唱红鼻子小黑和吉他手少年佩组成的快乐民谣小组合，成立于2012年。这是两个来自中南财经政法大学的姑娘，94年出生，长发的叫王心怡，负责主唱，声音清纯干净。短发姑娘名叫吴佩岭，担任吉他手。房东的猫的名字源于她们在大二时的一次咖啡馆演出，当时的她们并未把名字确定下来。老板让她俩取个名字，灵机一动的她们因经常下宿舍楼的时候经常能看见一只猫惬意慵懒的呆在那儿，便取名叫房顶的猫，后来觉得味儿不太对。还是房东的猫更加贴切，名字就这么给定了下来。一切都是那么悠然随性，也很符合他们身上与作品中所散发出来的的特质。

歌词下方是 _秋酿钢琴谱_ ，大家可以免费下载学习。
