_素素之死背景音乐_
是2010年胡歌等主演的大型穿越玄幻电视剧《神话》的配乐。《神话》改编自电影版《神话》，由上影英皇文化发展有限公司出品，成龙担任总监制，唐季礼任艺术总监，导演蒋家骏执导，胡歌、白冰、张世、张萌、任泉、陈紫函、金莎等主演。

全剧剧情采用古今交错的形式发展，讲述了80后青年易小川（胡歌饰）在一次偶然的际遇下穿越回到秦朝，经过一次次的奇遇和历险，逐渐蜕变成为仁心济世的一代大将，谱写了一段感人至深的神话。该剧在于2010年在央视首播，开播后接连刷新央视收视纪录，在网络上也有较高的点击率。
