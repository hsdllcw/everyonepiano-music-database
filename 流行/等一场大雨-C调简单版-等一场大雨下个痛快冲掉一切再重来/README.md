这首歌的歌名叫做《 **等一场大雨**
》，由莫非作词，刘博洋编曲，苏星婕演唱，发行于2021年5月21日。这是平凡我们的一个心路过程，首先是“等雨”，再到“下大雨”，其次“雨过天晴”，最后“又相遇”，错误的爱情，或者不合适的爱情，难免会有些令人失望。

这首歌温柔而治愈，因为总有一个对的人在对岸等你，就让这些事情随着一场痛快大雨冲刷掉吧。
