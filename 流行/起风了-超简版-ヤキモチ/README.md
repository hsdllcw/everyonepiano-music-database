《 **起风了** 》改编自日本歌手高桥优创作并演唱的歌曲《ヤキモチ》。后米果填词，买辣椒也用券演唱。大火后，周深，林俊杰，吴青峰，分别翻唱。

这首歌原版是写给男孩子的。米果的填词也完美的表达的原版的内涵。

歌词的大意是：男孩子年少时追逐梦想，告别家乡和心里喜欢的女孩子，经历了岁月的各种沧桑，始终无法忘怀那份美好。老年了回到家乡，要去看当年的恋人的那种忐忑，复杂，而又无可奈何的伤感的心情。一切的青春年少，美好情怀，都在岁月中无可奈何去了。也很好了演绎了一个老男孩历经沧桑，却依然心存梦想；半生归来仍少年的情怀。

高桥优的原版很好演绎了原歌词的含义和表达的情绪，沙哑而略带沧桑的嗓音。而后面的国内各种版本，展现了各种空灵，唯美。
