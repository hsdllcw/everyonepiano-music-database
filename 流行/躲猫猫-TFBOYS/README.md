_躲猫猫_
是TFBOYS的最新单曲。旋律清新明快，悠扬婉转。TFBOYS的歌曲最大特点就是新鲜年轻，他们唱着00后的感情观，基本上他们歌唱的都是健康向上的正能量，还有些情窦初开式的青涩告白。

### 大家一起躲猫猫：

看到歌名，小编的第一反应竟然不是好奇歌曲本身，而是想起了小时候玩的游戏---躲猫猫。估计好多小朋友的第一反应都应该和小编是一样的吧，嘻嘻。。。

在这无风的夜晚  
快乐变得好简单  
我们一起躲猫猫  
躲到烦恼找不到

简单直白的歌词，让每一个听众都能身临其境，仿佛置身夏夜星空，吹拂着凉爽微风，与三两好友一起嬉戏度过的美好青春时光。

记得千玺曾在微博晒出自拍，还留言称，“我喜欢这种捉迷藏躲在树上大家都找不到我的感觉，可是你们不能找不到我就走了啊，剩下我一个人仰望星空有点方”。

看到这个微博的时候，小编我竟然不厚道的笑了起来，真的是替千玺感到方。估计这首歌曲就是千玺给出的灵感吧。容小编我再笑一会，哈哈哈哈哈哈哈。。。。。

歌词下方是 _躲猫猫钢琴谱_ ，大家可以免费下载学习。
