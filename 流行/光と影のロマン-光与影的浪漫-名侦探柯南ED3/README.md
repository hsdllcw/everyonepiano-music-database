

**光と影のロマン**
是日本歌手宇德敬子的第9张EP。该专辑于1997年05月14日发行，共收录了包括《光と影のロマン》、《エターニティ》以及伴奏在内共计4首歌曲。另外，专辑中的同名歌曲《光と影のロマン》也被用作日本超人气动漫《名侦探柯南》的第3首ED，在第52集-70集播放。

  
歌词下方是 _光と影のロマン钢琴谱_ ，希望大家喜欢。
