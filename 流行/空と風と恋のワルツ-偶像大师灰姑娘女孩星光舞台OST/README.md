STARLIGHT MASTER 04 生存本能ヴァルキュリア》中。

本作是以在Mobage平台上好评配信中的《偶像大师灰姑娘女孩》为题材的音乐游戏。由玩家自己组成队伍，享受“LIVE”的乐趣。“LIVE”内共有5个按钮，配合歌曲的旋律来点击按钮，典型的音游玩法，再配合上华丽的3D舞蹈演出，着实让人眼前一亮。游戏初回收录曲，包括游戏主题曲《とどけ！アイドル》在内，将有20首以上，登场的偶像超过50人。“LIVE”画面可以在2D和3D模式之间切换。2D是可爱的Q版造型，3D可以欣赏华丽的舞蹈，各玩家根据自己的喜好自由选择吧~
