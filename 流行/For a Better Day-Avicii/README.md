

**For a Better Day** 是知名瑞典DJ及音乐制作人Avicii的歌曲，收录于专辑Stories中，发行于2015年8月。

早于之前报道，Avicii正在忙于制作一张全新的录音室唱片《Stories》，据透露，将在Avicii这张新专辑中客串献唱的知名音乐人包括酷玩乐队(Coldplay)主唱克里斯-
马丁(Chris Martin)，绿日乐队(Green Day)主唱比利-乔-阿姆斯特朗(Billie Joe Armstrong)，邦乔维乐队(Bon
Jovi)主唱Jon Bon Jovi以及体制崩溃乐队(System Of A Down)的主唱塞尔吉-坦吉安(Serj Tankian)。

截至到8月31日，已经率先发布了专辑内的3首单曲。

歌词下方是 _For a Better Day钢琴谱_ ，希望大家喜欢。
