

**别问我是谁** 是王馨平演唱的歌曲。且该曲收录于同名专辑中，1993年台湾发行。

  
王馨平凭这张国语大碟《别问我是谁》，正式进军台湾乐坛，第一主打为《 _别问我是谁_
》，这首歌曲令她得到台湾金曲龙虎榜的冠军位置。第二主打为《伴你一生》，这首歌曲成为台湾金曲龙虎榜的劲歌推荐。第三主打为《爱我一生一世好不好》。此专辑台湾销量达60万张，令她成功打开台湾市场。

  
歌词下方是 _别问我是谁钢琴谱_ ，希望大家喜欢。
