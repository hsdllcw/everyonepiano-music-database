_相思花_
是游戏《吉原彼岸花》的主题曲，由歌手Riryka演唱。《吉原彼岸花》（日语：吉原彼岸花）是由MariaCrown公司制作，于2015年9月25日发售的女性向成人文字冒险游戏暨恋爱游戏。2017年10月12日，PS
VISTA版的《吉原彼岸花 久远之契》（吉原彼岸花 久远の契り）正式发售，2018年12月20日发行Nintendo Switch版的《吉原彼岸花
久远之契》。

歌词下方是 _相思花钢琴谱_ ，大家可以免费下载学习。
