

**桃花朵朵开**
是阿牛作曲并演唱的一首歌曲。这首睽违二年全新贺岁力作，有浓浓的年节气氛和震天价响的锣鼓声，最适合一家老小欢天喜地庆团圆时听了！新的一年要听阿牛，桃花保证开呦！

《 _桃花朵朵开_
》简单而热闹，还充满浓浓过年气氛，阿牛用摇曳的桃花树、成双的鸟儿、在树下等故人的痴情人，营造出浓浓的春意和爱意，歌词琅琅上口，在爱情之外包含了“回家、团圆”的意思。阿牛敲锣打鼓、旌旗飘扬般的将这首歌唱得情深款款，将一首爱情为主题构思的歌曲唱得很喜庆，旋律优美、歌词朗朗上口，是一首标准的口水歌，但寓意倒是与春节的氛围蛮相符的。
