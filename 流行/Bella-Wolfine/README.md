_Bella_
是哥伦比亚歌手和作曲家Wolfine在2017年发行的歌曲。此曲已经发行便受到了全球病毒式的盛行。在Youtube上创下了4.5亿的浏览量。同时在Spotify上的流量也超过了2.3亿。可以说，这首歌曲是Wolfine歌唱生涯中最成功的一次。
