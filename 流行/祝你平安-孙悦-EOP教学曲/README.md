

《 **祝你平安**
》是电视剧《都市平安夜》片尾主题曲，当年剧中的原唱是那英和陈琳，其后孙悦把它唱红了大江南北，使全国人民都知晓了孙悦与这首歌，首唱是1994年。

孙悦，1972年出生于黑龙江省哈尔滨市道里区，中国著名歌手。以一首《 _祝你平安_
》崛起于九十年代中期的中国流行乐坛，成为中国内地歌坛的中流砥柱，百变歌后。发行单曲及专辑三十余张。从乡村民谣乐、劲爆舞曲、凄苦抒情歌曲、电子R&B音乐，以及影视均有涉猎。第一个宣布退出音乐颁奖典礼的歌手。

歌词的下方还提供了， _祝你平安钢琴谱_ ，欢迎感兴趣的朋友下载使用。
