萤火是TFBOYS于2016年12月30日发行的单曲。全新单曲的词曲作者Tae Mu,
Jay之前已与TFBOYS合作过《Heart》、《爱出发》和《Love with you》等多首歌曲，深知三少年各自的特色，默契十足。  
  
整首歌旋律浪漫唯美，伴着清爽的电吉他和满含着暖意的鼓声，将三少年磁性、清澈、温暖的音色特质表现的淋漓尽致，奏响了一曲冬日的抒情之音。歌词以“萤火”为主题展开，讲述了在萤火的陪伴下长大的孩子，心中怀抱着萤火般的爱与希望，飞向遥远的未来。动人的故事搭配上TFBOYS清澈的声音，再合着暖意十足鼓声，宛如冬日夜晚的一束光，散发着温暖整个冬夜的热度，为听者描绘了一幅关于希望、陪伴、成长的温暖画面。透过他们坚定的声线，可以看到少年在迷茫时的坚持，黑暗中的坚强，正突破重重阻碍飞向更远的未来。  
  
歌词下方是 _萤火钢琴谱_ ，希望大家喜欢。
