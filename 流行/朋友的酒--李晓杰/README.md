

《朋友的酒》是李晓杰演唱的歌曲，积木鱼作词，积木鱼作曲。
收录在李晓杰2009专辑《老大》中，发行于2009年12月13日。酒是男人之间释放豪迈与真挚的桥梁。朋友如酒，愈久愈浓愈香愈醇。一杯酒一辈子，朋友的酒，包含着千言万语。
