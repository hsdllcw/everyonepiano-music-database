

**Charlie Brown**
是英国摇滚乐坛进入新世纪之后崛起的最受欢迎的摇滚乐队之一Coldplay的歌曲。Coldplay有四个成员，乐队最早成形于伦敦大学学院
(UCL)时期。团员包括Chris Martin（主唱、键盘、吉他）、jon buckland（主奏吉他）、Guy Berryman（贝斯吉他）及Will
Champion（鼓、合音、其他乐器）。Coldplay乐队成军之后的每一张专辑都不断进步，一步一步地扩大他们的影响，并勇夺格莱美等多项大奖。

  
2011年10月19日，Mylo Xyloto在日本提前发行，增加了乐队在2011 Glastonbury音乐节表演的 _Charlie Brown_
、Life is For Living和Every Teardrop is A Waterfall。
