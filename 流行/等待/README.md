

《等待》作为电视剧《汉武大帝》片尾曲其实是有一定内涵的。明知辉煌/过后是暗淡/仍期待着把一切从头来过。无非就是一个帝王明知达到鼎盛巅峰后就会走下坡路，却还是要冲向巅峰。  
了解一下汉武帝的生平就会了解一些的。重点关注武帝后期磐尽全

国之力与匈奴决战，迫使匈奴西迁（最终西迁的匈奴人摧毁了罗马帝国），保证大汉百年边疆无事。但国库空虚民生疲敝，帝国烽烟四起。换太子，杀皇后，防止重现母权过大，外戚干政，发布轮台诏（罪己诏），奠定20年后孝宣中兴。

《等待》大概就是这么一个意思，他的爱是对大汉帝国的爱。
