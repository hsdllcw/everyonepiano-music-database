

**一点点双手简谱** 和五线谱完全对应。

_一点点_ 是周杰伦演唱歌曲，收录在其2016年专辑周杰伦的床边故事，于6月24日正式发行。

作为华语乐坛的传奇，周杰伦在音乐上就从未停止探索与创新。中国风与嘻哈结合，戏曲与饶舌MIX等等，这些内容上的尝试让周杰伦收获华语圈各大重要的音乐奖项;而在音乐载体上的选择，他也更注重与流媒体平台的合作。

歌词下方是 _一点点钢琴谱_ ，希望大家喜欢。
