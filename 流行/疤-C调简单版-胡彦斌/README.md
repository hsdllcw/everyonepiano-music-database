

《 **疤** 》是一首洒脱的胡氏情歌，由甘世佳作词，胡彦斌作曲，谷粟编曲，胡彦斌制作并演唱。

忧伤的歌词就像一篇回忆录，以叙事的手法梳理情感的伤疤。正如歌中「最多留下淡淡的一道疤，在临睡前会有些痒/忍住它/放下它/无视它/忘掉它」与「最多留下淡淡的一道疤/坚硬的壳
就像盔甲/可是她/随便她/什么她/哪个她」这“淡淡的”一道疤，胡彦斌演绎动情却不撕心裂肺，用这份柔情渗入人的心底，用这份坦然去抚慰那颗伤痕累累却又“逞强”的心。

歌曲旋律灵动轻缓，带有淡淡的忧伤，歌词里把过往情感比喻成一道浅浅的伤疤，搭配胡彦斌富有磁性的嗓音，慵懒的唱法，把心中的过往情感诠释的淋漓尽致。通过歌曲让人知道，伤，虽然使人疼痛，但它同时也教会人坚强和珍惜，会结成疤，从而使人更加勇敢的前行，这就是成长。
