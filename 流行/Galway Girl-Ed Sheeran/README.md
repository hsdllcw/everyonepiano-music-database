_Galway Girl_ 是由英国新生代歌手Ed Sheeran (艾德•希兰)演唱的一首歌曲，是Ed Sheeran (艾德•希兰)2017年新专辑《÷
(Deluxe)》中的第6首。该张专辑名称读作“divide”，于2017年3月3日通过Asylum
Records和大西洋唱片公司发行。作为一个优秀的唱作人，艾德•希兰不只为自己创作，还为许多歌手写过歌。
