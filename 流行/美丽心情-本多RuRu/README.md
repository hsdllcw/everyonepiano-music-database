《 _美丽心情_ 》是台湾歌手本多RuRu演唱的一首歌曲，这首歌收录在她的同名专辑中。原曲为日本歌手中岛美雪的《帰省》。2001年发行。

RuRu唱出多年奋斗初衷“美丽心情”。因为RuRu这么一段与常人不同的历练，让维京总监姚谦为她量身打造这首专辑主打“美丽心情”，歌词中提及“多雨的冬季总算过去
天空微露淡蓝的晴 我在早春清新的阳光里 看着当时写的日记”、“原来爱曾给我美丽心情
像一面深邃的风景”正是RuRu的这一阵子心情写照，而这首歌的原曲是中岛美雪的歌曲，也正映衬着RuRu中日对照特殊经历。

正如美丽心情所唱：“只有不后悔去做，明天才有美丽心情”。希望大家在知道RuRu的奋斗故事后，也以同样温暖的心感受RuRu的歌——“美丽心情”！
