不怕不怕_ 是由新加坡流行歌手郭美美在2006年翻唱的一首歌，原作是摩尔多瓦裔罗马尼亚乐团O-zone的《dragostea din
tei》（又名“麦阿喜”）。此曲吸引了消费大众，电台点播不断，销售成绩也一片看好。而2006年发行同名个人专辑《不怕不怕》，深获好评，并获得全球华语歌曲排行榜、9+2音乐先锋榜等奖项的新人奖。
