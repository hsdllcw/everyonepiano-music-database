

**If I Lose Myself**
是Rihanna是美国流行摇滚乐队OneRepublic的歌曲。OneRepublic，中译名“共和时代”，曲风走流行、独立摇滚的路线。2006年天使专辑《Dreaming
Out Loud》诞生，主打《Apologize》打破各国流行榜纪录。2009年新专辑《Waking
Up》再次获得乐评人及广大乐迷肯定。2013全新专辑《Native》发布。

_
