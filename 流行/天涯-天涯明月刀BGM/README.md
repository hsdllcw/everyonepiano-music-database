_天涯_
此曲出自3D武侠题材MMORPG网络游戏《天涯明月刀》。游戏于2016年07月01日在中国大陆正式公测，由《天涯明月刀》小说改编而成。该作有丰富的社交系统，在游戏中玩家可以拜师和收徒，可以参加帮会，和帮会成员参加任务活动，还可以加入联盟、盟会，体验不同阵营之间的战斗。

歌词下方是 _天涯钢琴谱_ ，大家可以免费下载学习。
