

《 **写给我第一个喜欢的女孩的歌**
》是中文VOCALOID著名UP主ilem的第26首投稿作品，用VOCALOID制作，洛天依演唱，在2015年12月24日22:09发布于中文弹幕视频网站bilibili，为圣诞贺岁曲。

本曲已于2017年2月3日02:44达成Vocaloid中文传说曲。是ilem的第8首传说曲。

本曲是ilem在多次梦见初恋女孩后，写给她的圣诞歌曲。曲调简单，却蕴含年少青涩的心绪，混杂着如今回首过往的感概与怀念，令人听之动容。

_写给我第一个喜欢的女孩的歌双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
