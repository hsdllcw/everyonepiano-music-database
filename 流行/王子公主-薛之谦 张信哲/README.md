

**王子公主双手简谱** 和五线谱完全对应。

_王子公主_
，薛之谦和张信哲同台演绎的动人情歌，用最感人的声音唤醒金曲。歌曲原唱是张信哲，收录在其2000年6月2日发表的专辑《信仰》中。该曲由李宗盛填词，林一峰谱曲。

歌词下方是 _王子公主钢琴谱_ ，大家可以免费下载学习。
