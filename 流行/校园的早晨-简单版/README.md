

《 **校园的早晨** 》是王洁实、谢莉斯演唱的一首校园歌曲，由著名作曲家谷建芬作曲、诗人高枫作词。

此歌的创作灵感是源于辽宁大学校园——高枫先生晨起到辽大散步，看到校园内的晨读景象，便触发了他写下了这首著名的校园歌曲。此曲在二十世纪八十年代曾经风靡一时。

另外，《 _校园的早晨_ 》旋律清新活泼、优美动听，也是它的灵感来源地辽宁大学的校歌。
