

“天天想你，天天问自己，到什么时候才能告诉你”，歌曲《 **天天想你** 》是张雨生歌唱生涯的代表作品之一，发行于1988年11月。

张雨生的这首歌据说是献给妹妹的，他们兄妹的感情很好，但是妹妹却在一次郊游中溺水身亡了。雨声说他妹妹很喜欢唱歌，雨声做音乐也是为了实现它妹妹的梦想！
