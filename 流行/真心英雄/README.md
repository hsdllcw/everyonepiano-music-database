

《 **真心英雄** 》是成龙，周华健，李宗盛，黄耀明合唱的经典曲目。

该歌曲是李宗盛1993年作品。这首歌最早是收入在《滚石九大天王纵夏欢唱12出好戏》合辑里面的。原唱：成龙&周华健&黄耀明&李宗盛。粤语版：《在晴朗的天空下》。

我想很多人应该都非常熟悉这首歌。不仅仅是旋律，其中的歌词也印象深刻吧。现在再次听这首歌，还是会受到很大鼓舞。歌词下方有提供 _真心英雄钢琴谱_
，欢迎大家使用。
