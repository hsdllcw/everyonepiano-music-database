

**老街** 出自李荣浩2010年9月9日发行的专辑《小黄》。李荣浩质朴感性的嗓音，加上朗朗上口的旋律，将自己对老街的那份说不出的伤感表现的淋漓尽致。

从歌词意境来说，从《 _老街_
》的“巷尾老爷爷卖的热汤面，味道弥漫过旧旧的后院”，到《小黄》中黯然神伤的“星星在闪烁，小黄在梦中”都是以柔情为主题，带给了我们颇多的感动。歌词，通俗易懂。“汤面”“炊烟”“街角”“天空”这些貌似简单的字眼却带给了我们浪漫温暖的意境，将很多感人的场景呈现于眼前，更是我们心中的画面，让人回味无穷。
