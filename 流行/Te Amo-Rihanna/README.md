

**Te Amo** 是蕾哈娜Rihanna专辑《Rated R》的一首歌曲。标题意为我爱你，MV里蕾哈娜与法国名模Laetitia
Casta上演蕾丝恋，微妙的感情间显露着阴影。该歌曲成绩差强人意，但在欧洲成绩斐然。

  
歌词下方是 _Te Amo钢琴谱_ ，希望大家喜欢。
