_舞娘_
是中国台湾女歌手蔡依林演唱的一首歌曲，由澳大利亚电音组合NERVO作曲，陈镇川作词，收录在蔡依林2006年5月12日发行同名专辑《舞娘》当中。2006年10月该歌曲获得第六届全球华语榜中榜最受欢迎金曲奖；2007年1月获得第十三届华语榜中榜港台年度最佳歌曲奖。

此曲不仅在音乐录影带中，蔡依林令人惊艳的舞蹈表现十足性感，而且这首歌曲的曲调也是充满异国情调的曲风，流畅且利落的节奏使歌曲悦耳动听，动感十足。
