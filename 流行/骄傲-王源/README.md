

**骄傲双手简谱** 和五线谱完全对应。

_骄傲_ 是王源演唱的单曲，由王源作词，赵英俊作曲，是2017年11月20日发布的单曲。

站在人生的岔路口，前方是理想，回首是母亲不舍与殷切的目光。离开家门时的情景历历在目，心里不舍，却故作洒脱，每天为学习和工作匆匆忙碌，对母亲的陪伴少之又少。少年厚重而细腻的内心情感一字一句间被缓缓道来，暗含愧疚又渴望成长，希望有一天变得更加坚韧强大，成为母亲心中引以为荣的骄傲。这是来自一个17岁少年对自己，也对母亲的承诺，亦是少年王源送给母亲的一份珍贵礼物。

歌词下方是 _骄傲钢琴谱_ ，大家可以免费下载学习。
