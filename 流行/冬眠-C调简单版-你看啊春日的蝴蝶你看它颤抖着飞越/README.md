这首歌曲叫做《 **冬眠** 》，由司南演唱，收录于2019年10月28日发行的专辑《冬眠》。整首歌都有一种清冷、忧郁的美。

此曲所采用的意象极佳。春日的蝴蝶，感受到春的生机；仲夏的弯月，感觉到夏夜的宁静，也似乎有那么一丝丝伤感呢；秋末的落叶，感受到秋季的萧瑟，树叶告别大树正如她告别自己，充满着忧伤与不舍；冬至的白雪，因为秋叶的离开，只剩下树独自领略冬季的孤独与寒冷，也正如她离开自己后，心中的孤独与想念。四种意象分别选取四季的特别之处，像是在平铺直叙一年的变化，也似娓娓道来的一个凄美的爱情故事。
