

**松烟入墨** 由Winky诗演唱。虽说此曲讲述的是墨的制作，但歌词却极美。

而演唱者Winky诗将整首歌唱出一如茶般浓厚却也香醇的气息；又仿佛如清风一般整个清爽的气息扑面而来，没有一点点的累赘；慢慢的静下心来松香、墨香、茶香也就扑面而来，就像一切不开心也都随风飘散了，一切的情绪也就没有了。歌曲也在慢中渐渐的进入到了一定的快节奏，恰恰是这样的处理让我感受到了演唱者的声音中的坚定，就像那些令我们不开心的过去一样虽然难忘却还是让我们坚定的向前方走去。
