

**蜀绣**
是郭敬明填词，刘佳、严艺丹谱曲的一首中国风歌曲，由第二届中国成都国际非物质文化遗产节蜀绣文化传播大使李宇春演唱，收录在天娱传媒制作的专辑《红星闪闪》中，是成都非物质文化遗产节主题曲。

  
2015年央视春晚，李宇春现场演唱了《 _蜀绣_
》。从创意来讲，这首作品将“蜀绣”这种四川工艺瑰宝述之以文字、谱之于旋律，向人们展现了蜀绣的美轮美奂。对于李宇春来讲，与郭敬明的跨界合作，显然也赋予了她更宽的音乐视野。尤其值得一提的是，从《蜀绣》这首作品里，更可以明显感受到她音乐演绎上的进步。不再是青春年少时那种一览无遗的直接，而多了几分往复还绕的立体和层次。

  
歌词“唯美”，有着浓郁的“中国情成都风”，歌词通过一个成都绣娘的视角，将绣娘个人对刺绣、丈夫和家国的深情融入其中，歌词大气又感人至深。开始的呢哝轻吟浅唱到慢慢绽放，如一朵花一样的慢慢开放。有种如沐春风的感觉，像羽毛一样轻轻抚过。歌曲的曲调婉转悠扬，与李宇春时尚的唱腔形成呼应。
曲调旋律悠扬清爽，歌者的声音也温暖悠扬，仿佛听着歌声就可以看见蜀国山岳深处绕指刺绣的女子，人映于画，画现于歌。

  
歌词下方是 _蜀绣钢琴谱_ ，希望大家喜欢。
