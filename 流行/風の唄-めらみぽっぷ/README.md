_風の唄_ ，原曲为信仰は儚き人間の為に，由めらみぽっぷ演唱。

めらみぽっぷ(meramipop)是活跃于同人与商业音乐活动的女歌手，是东方Project同人音乐社团凋叶棕的爱用歌姬，在少女病中则使用Lico的名义，同时拥有个人社团Cosmopolitan。擅长Pop、Rock风格的音乐。相比低音部分，高音相当强。

歌词下方是 _風の唄钢琴谱_ ，大家可以免费下载学习。
