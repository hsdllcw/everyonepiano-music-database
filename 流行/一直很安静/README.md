《 _一直很安静_
》演唱歌手为阿桑，作词方文山，是一首淡淡感伤的歌。是2004年热播大型神话悲情剧《仙剑奇侠传》的插曲。是对女主角林月如（安以轩饰）的一生感情的诠释，随着电视剧的热播，加上这首歌曲感情细腻、悲伤却很真实，很快就流传了下来。  
  
《 _一直很安静_
》的歌词本是方文山写给其女友的，方文山说有个女生喜欢他，可是他有女朋友，不可能同时交往，所以她一直都不说话，很安静地在一旁；而后结合电视剧《仙剑奇侠传》中女主角林月如对李逍遥的爱恰好也是这样，于是这首歌曲就用来当做该剧的插曲。  
  
这首歌曲是阿桑用略带沙哑的嗓音忧伤地唱着，令观众不禁为之动容。这首忧伤的歌曲专为《仙剑奇侠传》剧中女主角林月如而作。听着这首略带哀伤的歌曲，不少仙剧迷们会情不自禁地想起剧中月如与逍遥的点点滴滴，不能忘记月如与逍遥吃到老玩到老的誓言，不能忘记月如对逍遥默默地爱，不能忘记月如为逍遥做出的牺牲,也正如歌词中所唱：明明是三个人的电影我却始终不能有姓名。这份无名的爱却成为了一个完美干净的句号。这位剧中女主角林月如被无数仙剧迷们所铭记。
