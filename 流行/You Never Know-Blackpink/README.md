

**You Never Know** 是韩国女子演唱组合BLACKPINK演唱的一首歌曲，收录于其2020年10月2日发行的首张正规专辑《THE
ALBUM》中。作为一首凸显出BLACKPINK歌唱实力的曲目，将充满感性的歌词和旋律同简洁的编曲调和，更深刻地传达出歌曲的信息。
