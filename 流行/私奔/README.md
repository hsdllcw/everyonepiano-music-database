_私奔_ 是由歌手郑钧作词、作曲、演唱的歌曲。此歌出郑钧自第六张专辑《长安长安》。先在中国好声音第一季梁博演唱，后又在中国好声音第三季陈直演唱。  
  
梁博，2012年《中国好声音》参赛选手。导师那英的小二班成员，那英组唯一拿到梦想席位的队员。在2012年9月30日中秋之夜，在万众瞩目的梦想巅峰之夜，以往出具的好声音们也到现场一起high翻舞台，还有范冰冰、刘亦菲、吴彦祖、张惠妹等众多大牌明星到场助阵，为好声音们加持、颁奖、与现场观众一起共襄盛举。最终，来自东北的90后小伙梁博，以高支持率获得中国好声音年度总冠军。从盲选时只有那英为他转身，到巅峰之夜的睥睨群雄，他不是“黑马”，是“奇迹”，让我们感动的奇迹。
