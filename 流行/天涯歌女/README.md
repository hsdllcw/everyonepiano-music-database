_天涯歌女_
是1937年中国电影《马路天使》的主题歌之一，由贺绿汀作曲，田汉作词，周璇原唱。周璇以此曲红遍天下，后来甚至以此曲为名演出一部同名电影《天涯歌女》。  
  
曲调采用了江南民歌的音乐素材。而周璇也凭借在此片中的出色表演一举奠定了她在当时上海电影界以及后来中国电影史中不可动摇的地位。
