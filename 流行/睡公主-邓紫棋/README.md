_睡公主_
是2008年10月16日正式发行的，由香港唱作歌手邓紫棋（G.E.M.）填词谱曲并演唱的一首原创歌曲，收录于个人首张同名EP《G.E.M.》中。2009年，该曲获得Neway卡拉OK颁奖礼。

《睡公主》结合了摇滚等多种现代流行音乐元素，歌曲开始给人一种莫名的悲伤感，但是到副歌部分又有种冲出泥潭找到希望，尾声部分又感觉一种释然。
