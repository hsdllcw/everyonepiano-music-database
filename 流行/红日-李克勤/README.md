

**红日双手简谱** 和五线谱完全对应，是EOP教学曲。

《 _红日_
》为李克勤效力于宝丽金唱片公司时期的最后一张大碟，大碟推出时间为1992年夏季。之后他便转投至星光唱片。本大碟内收录了11首歌曲，李克勤亲自为「龙影侠」、「红日」和「你是我的太阳」填词。而改编歌「
_红日_ 」是TVB 电视剧《他来自天堂》主题曲，此歌至今仍深受欢迎，视为「励志歌」经典之作。

另外，与周慧敏合唱的「万千宠爱在一身」中场一段独白最后一句，及翻唱由谭炳文原唱的旧欢如梦，配合李克勤所跳的舞步，都为当时佳话。
