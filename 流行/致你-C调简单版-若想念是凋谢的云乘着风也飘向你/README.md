

抖音上很多视频的背景音乐都用到了这首“若想念是凋谢的云，乘着风也飘向你”，触动了不少人的心弦。据悉，此曲叫做《 **致你** 》，由yihuik苡慧演唱。

相爱是一场搜证游戏，在生活的各个地方，都会留下痕迹。分手后，我们试图拼命抹掉这些痕迹，拉黑、删除、扔东西、甚至搬家……然而，我想告诉大家的是：抹不掉没关系，放不下也没什么大不了的。因为伤痛再痛也抵不过时间，痕迹再深也会被下一个对的人抚平。总有一天，你会笑着说出这些过往。所以，要用最大方的态度去迎接开始的所有，自然也要坦然的接受任何结局。
