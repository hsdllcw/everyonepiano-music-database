

**雪之华**
雪之华是韩国KBS2004年出品的月火偶像剧《对不起，我爱你》的主题曲。该剧由李亨民执导，李庆熙执笔，苏志燮、林秀晶、郑京浩、徐智英等人联袂主演，以收养为主题，描写了一个关于被抛弃孩子悲苦人生的心碎故事。

同时， _雪之华_
（雪の华，ゆきのはな）由韩国著名男歌手朴孝信演唱。最初此曲是中岛美嘉在2003年10月1日发行的第10张单曲。也是中岛美嘉亲自出演的明治「boda」「galbo」的广告歌曲。中岛美嘉凭借该曲获得第45回日本唱片大奖的金奖。

该曲由Satomi作词，松本良喜作曲及编曲。

歌词下方是 _雪之华钢琴谱_ ，希望大家喜欢。
