

**囚鸟**
是彭羚同名专辑《囚鸟》中的作品，无疑也是彭羚所有国语歌曲中流传最广的一首，表达的是都市中感情的冷漠和缺失，爱越来越少，孤独的日子里，每一个人都像是囚鸟一样，渴望着爱人能够给自己一点微薄的爱，却不可得，自己又不忍挣脱，只能卑微的继续，唯一和自己作伴的只有自己流淌的眼泪。

“我像是一个你可有可无的影子，冷冷的看着你说谎的样子。这撩乱的城市容不下我的痴，是什么让你这样迷恋这样的放肆？”这样一个“都市怨妇”的形象，是经典，也是彭羚迟迟突破不了的桎梏。这首歌流畅的旋律，哀怨的歌词，尤其是彭羚张弛有力的亮丽高音，都给人留下了深刻的印象。
