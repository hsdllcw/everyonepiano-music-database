

**爱如空气** 是孙俪演唱的一首歌，收录于其2006年发行的专辑《小小的梦想》之中，同时也是其主演的热播电视剧《幸福像花儿一样》的主题曲。

  
歌词下方是 _爱如空气钢琴谱_ ，希望大家喜欢。

### 爱如空气歌词：

时光冲淡往事鲜艳褪去

我们之间的爱轻得像空气  
而我依然承受不起
