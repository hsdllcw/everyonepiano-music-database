

如果云层是天空的一封信  
能不能再听一听 听你的声音  
就算是探秘 是贪玩而已

此段歌词出自任然演唱的《 **无人之岛** 》。

《 _无人之岛_
》，谐音“无人知道”，自己内心最深处的东西也就只有自己能够触摸到。想忘记，却永远无法忘记。曾经的片段记忆永远都会躲在那个角落里，不愿出来又渴望被发现。

任然曾说过，这一首唱的是 “你是我，我亦是你。消失或存在
都在我记忆最深处，不灭不散”。对于没能在一起，那还不如把当初的美好收藏起来，放在心里。错过了终是失去！
