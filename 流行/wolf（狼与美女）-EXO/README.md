

《狼与美女（ **wolf** ）》是EXO首张正规专辑《XOXO(Kiss&Hug)》主打曲，分中文版和韩文版。

《 _wolf_ 》是一首Dub Step与Hip
Hop相结合的舞曲，野性并充满力量的节奏旋律，如童话般的故事细节与有趣的歌词，加以独特的编舞而成，整首歌散发出别样的魅力。

歌词下方还提供了 _wolf钢琴谱_ ，欢迎感兴趣的朋友下载使用。
