_突然想爱你_
是许茹芸创作并演唱的一首歌曲，收录于专辑《如果云知道》，上华唱片于1996年9月8日发行。粤语版为《留低锁匙》，由李敏作词，收录于《茹此精彩13首》中。

爱上一个人只在一瞬之间，而且这种爆发的情感往往是一发不可收拾的。旋律不算复杂但很优雅，高潮部分琇琇唱得撕心裂肺，藏不住的爱恋破堤而出。

歌词下方是 _突然想爱你钢琴谱_ ，喜欢的朋友可以免费下载学习。
