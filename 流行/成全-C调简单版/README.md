

《 **成全** 》是由陈小霞作曲，刘若英演唱的歌曲，收录在专辑《年华》中。2016年9月18日，林宥嘉在江苏卫视《蒙面唱将猜猜猜》中进行翻唱。

刘若英的歌，就像她的名字，温婉醇厚，用自己的方式诉说故事，这首《 _成全_
》，被她唱出一种无所畏惧的勇敢。虽然每次听都会有心酸酸的感觉，唱出来的真诚以及唱出来的情感，似乎是流经了那悠久的岁月，得到的是无数人的共鸣。
