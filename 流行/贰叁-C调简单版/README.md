

继年度话题之作《大碗宽面》引爆全民视听讨论热潮、收获市场与专业一致好评后，环球音乐联手顶级潮流音乐人吴亦凡，重磅呈现恸人歌曲《 **贰叁**
》，用感人至深的音乐创作，温 暖乐迷的秋末冬初。

近年来不断在创作中挑战自我的吴亦凡，此番在《 _贰叁_
》中再度加入古风元素，新歌从曲调到词作都洋溢浓浓的中国风，听来余韵悠远，让人回味无穷。动人旋律框架下，歌曲缓缓流淌西洋乐器竖琴声，中国民乐乐器古筝、箫带入人声，然后用琵琶弹拨点缀……副歌说唱部分融入吴亦凡擅长的R&B节奏，加上中国鼓、笛子等贯穿其间，让“吴亦凡式”中国风更具特色。在歌词部分，吴亦凡用古文结合白话的特别创作模式，真挚讲述了一段穿越生死的凄美恋爱故事，将感情中的思念、无奈与遗憾悉数铺展，让人听来怅然，更感受到吴亦凡创作时的别样用心。
