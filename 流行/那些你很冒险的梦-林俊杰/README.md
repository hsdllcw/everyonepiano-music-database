是林俊杰演唱的歌曲，由王雅君作词，林俊杰作曲，收录在林俊杰2011年12月31日发行的专辑《学不会》中。2013年2月，该曲获得马来西亚one
FM劲爆30排行榜年度三十大金曲第15名。

客观来说，此曲旋律并不独特，加上钢琴与弦乐又是种抒情常态，所以整首歌的辨识本不高，但副歌部分加进的Band提升了情绪层次。由王雅君所作的歌词居然出现了“摺纸飞机，碰到雨天，终究会坠落”、以及“你却不想懂，只往反方向走”这样消极的字眼，令人意外，也为歌曲创造了一些可讨论性。这首歌是林俊杰式的悲伤情歌，关于逝去的爱，每多想一次就多痛一次，但是再也回不去那些美梦。
