

**缺口** 为电影《等一个人咖啡》的主题曲。由音乐鬼才庾澄庆演唱，小说鬼才九把刀填词和其黄金搭档木村充利作曲。

  
电影《等一个人咖啡》为新锐导演江金霖的首部剧情长片，由九把刀和柴智屏监制，是九把刀继《那些年，我们一起追的女孩》后，
“爱情三部曲”的第二部作品。影片于2014年8月15日在台湾、香港地区正式上映。

  
歌词下方是 _缺口钢琴谱_ ，希望大家喜欢。
