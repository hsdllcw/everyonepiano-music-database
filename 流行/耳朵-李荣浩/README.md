

**耳朵** 是由文炅作词，郑宇界作曲，李荣浩演唱的歌曲，收录于李荣浩2018年10月17日发行的同名专辑《耳朵》中。

《 _耳朵_ 》的情绪相比较专辑中的《我知道是你》没那么浓郁，吉他摇滚的成分相对减少，是用了“安抚”的手段来激发耳朵听更内心的声音，若是 Bass、Drum
在这样一首歌中出现，那显然是对耳朵的“刺激”而非激发。

深爱吉他摇滚的李荣浩拎得清轻重缓急，《耳朵》的动人全然来自李荣浩“收”着的监制想法，以及音乐人郑宇界和已逝词人文炅在词曲层面的真挚表达。
