

你骄傲的飞远  
我栖息的夏天  
听不见的宣言  
重复过很多年

最近小编在刷抖音的时候，总能听到这段歌词。据悉，此曲叫做《 **飞鸟和蝉** 》，由任然演唱，以此献给为情所困的人！另外，网站还为大家提供了《[
**无人之岛**](Music-11338-无人之岛-任然.html "无人之岛")》钢琴谱，喜欢的也可以下载。

一只自由浪漫的飞鸟，碰上了一只微小柔弱的蝉，一个是飞过的地方即是美丽，一个是只有树干为栖息地；对飞鸟来说，你只是一位万处风景的其一，对那只蝉来说，你既是它一整个夏天，也是它的一生。

作为一首流行歌曲，尤其对于听众的听感而言，除了要有入耳的旋律外，还需要保证一定的创作理念，才能显得更为上乘。很明显，《 _飞鸟和蝉_
》从歌名入手，简单明了的告诉大家这是一首阐述跟离别相关的情歌。从歌词来看，深度、有内涵，脱离了情情爱爱的感伤，以更为隐喻的方式告诉大家相遇离别终有时。  
  
任然的音色较为深沉，很容易把听众的情绪带入到歌词里，而后搭配精炼且高度入耳的旋律，非常适合听众哼唱。  
  
“你说青涩最搭初恋  
你说空瓶适合许愿”  
  
人生总有不期而遇的温暖和生生不息的希望，终有一天，你会是某人的限量版快乐与守候。
