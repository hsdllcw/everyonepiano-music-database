作曲，102位歌手联袂献唱，将历史、国风、英雄等元素浓缩在歌曲中，以引起玩家感情上的共鸣，给人以美的享受，乃至于心灵的震撼。该曲为《 **王者荣耀**
》五五开黑节100英雄官方献礼歌，于2020年5月2日全网上线，并登陆游戏客户端。
