_  
年轻的战场_
是2006年莱卡•加油好男儿主题曲。关于原唱，目前有较大争议。目前主要演唱的版本有兄弟联、张杰、孙楠、张恒远。加油！好男儿是一档由东方卫视主办的全国男性选秀节目，类似湖南卫视的“超级女声”和“快乐男声”、央视的“梦想中国”。但此节目并不只注重于声乐，而是以参赛选手全面素养来确定名次，包括歌唱、演技、机智、人气（通过短信投票）等等。

  
歌词下方是 _年轻的战场钢琴谱_ ，希望大家喜欢。
