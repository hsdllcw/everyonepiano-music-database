

“于是那一刻心动，我开始心动，彼此懵懵又懂懂”，是抖音上的一首甜美小情歌。歌曲的名字叫做《 **那一刻心动** 》由任子墨演唱，发行于2021年4月10日。

每一段感情都有一首歌曲可以代表，在遇见你的那一刻，心动便是唯一符号。这是一首温馨、温暖的歌，以简单的旋律营造甜蜜情怀，爱情像海岸线的玫瑰、酸甜口的汽水，沉浸其中的女孩儿都会问“你喜欢我什么～”。其实，哪有缘由，只因是你。
