_魔法城堡_
是TFBOYS于2014年3月13日发行的单曲，由冷子夕作词作曲，TFBOYS演唱。收录在同名专辑《魔法城堡》中。为电影《洛克王国3圣龙的守护》推广曲。

2014年4月，TFBOYS凭借《魔法城堡》获得音悦V榜中国内地最具人气歌手奖和音悦直播人气歌手。

这首全新单曲是为TFBOYS量身打造的清新治愈系歌曲，曲风梦幻唯美，整首歌曲充满了神秘的气息。在这首新作品，展开一次华丽冒险，会给歌迷带来多少耳目一新的感觉。

歌词下方是 _魔法城堡钢琴谱_ ，希望大家喜欢。
