

《冬日恋歌》，又译《冬季恋歌》是韩国KBS电视台四季系列（蓝色生死恋系列）的第二部曲，于2002年1月初至2002年3月其间播放的感人爱情电视剧，由尹锡湖执导，裴勇俊、崔智友、朴容夏及朴率美等出演。
