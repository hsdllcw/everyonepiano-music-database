

**迟到** 是华语一代巨星刘文正的原唱歌曲，收录于其1981年的专辑却上心头里。  
  
_迟到_
是什么？经典怀旧风了解下，论先来后到的重要性了解下。曾经有人说过，两个人的出场顺序很重要，有的遇到了就是一辈子，不管是否是最好的。有的错过了即使是最好的也是错过了，只能怪在最不懂爱情的年纪遇到了最好的你。  
  
歌词下方是 _迟到钢琴谱_ ，大家可以免费下载学习。
