

《 **等不来花开** 》由内地男歌手pro演唱，发行于2022年。

整曲歌词很伤感，但旋律却非常欢快，形成了鲜明的对比。就像歌中所唱，“我们的爱就像秋叶等不到花开，我们的爱就像风中漂浮的尘埃”，这样的爱是没有结果的，因为到了秋天，叶子就要落下，根本不可能等到花开。而空中漂浮的尘埃，更说明爱情的无果。所以，在爱情中，我们缺少的并不是一个异性，而是能和自己产生共鸣的灵魂。

歌词下方是 _等不来花开钢琴谱_ ，大家可以免费下载学习。
