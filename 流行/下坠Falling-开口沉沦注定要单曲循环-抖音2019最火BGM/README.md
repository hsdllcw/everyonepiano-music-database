

**下坠Falling双手简谱** 和五线谱完全对应。

有一首歌曲被很多网友表示第一句就非常好听，在抖音上也很火。而歌曲制作也非常棒，一听就让人沉醉，此曲就是Corki演唱的《 _下坠Falling_
》。歌词讲述的是男女之间的感情，其实是抒情的感觉，但却很有节奏感，绝对是一首不能轻易错过的好歌，稳稳地抓住了听众的耳朵。

歌词下方是 _下坠Falling钢琴谱_ ，大家可以免费下载学习。
