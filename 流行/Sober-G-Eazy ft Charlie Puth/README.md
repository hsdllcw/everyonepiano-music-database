

**Sober双手简谱** 和五线谱完全对应。

_Sober_ 是G-Eazy与Charlie Puth一同合作的歌曲。

G-Eazy是一个独立的白人说唱歌手，他与Macklemore一样，都是来自美国南部且发展迅速的白人说唱歌手，但G-
Eazy对自己更加严格。他的说唱十分清晰，他的作品通过律动精心直接地阐述着个人的想法。而Charlie
Puth因翻唱多首经典歌曲在YouTube上获得关注与赞誉，虽然以这样“接地气”的方式成名，但其实除天赋以外，Charlie
Puth一直接受正统的音乐训练与熏陶，伯克利音乐学院科班出身的他功底深厚，自幼研习多种乐器，练得一手专业的钢琴技法，嗓音干净悠扬，高音透彻，低音沉稳，再加上出众的创作才能，即便在更新迭代迅猛而无情的欧美乐坛，这样的音乐才子也实为不可多得。
