_莲花_ 是李玉刚演唱的一首歌曲，由李玉刚作词作曲，收录在李玉刚于2014年12月11日发行的专辑莲花中。

此曲充满禅意，表达了李玉刚对禅意的诠释——“一念一清净，心是莲花开”。彼岸莲花的遥不可及，却已经开在脑海心间，相伴相生，从此结缘，因此开始。歌曲以佛学思想作为艺术上的指导，融合了中国文化元素与New
Age音乐的曲风，以男女双声腔的唱功和静雅的曲辞，创作出涤荡心灵的治愈系歌曲，令听者审视内心，感悟自我。
