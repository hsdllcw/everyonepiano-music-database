

《 **跟往事干杯** 》收录于姜育恒的同名专辑《跟往事干杯》之中，是姜育恒的代表歌曲之一。

姜育恒，1958年11月15日生于韩国首尔，祖籍山东荣成。韩籍华裔，中国台湾著名男歌手。主要代表作有：《再回首》、《 _跟往事干杯_
》、《驿动的心》、《爱我你怕了吗》等。1984年开始发行唱片，以一首《再回首》而闻名，此曲曾被王杰，李翊君等明星翻唱。至今姜育恒在两岸三地的人气依然不减，是华语乐坛上的一颗“永恒之星”。
