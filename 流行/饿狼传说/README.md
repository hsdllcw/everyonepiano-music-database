

《 **饿狼传说**
》是香港歌手张学友发行第十四张粤语专辑，唱片制作人欧丁玉，宝丽金唱片公司制作发行。这张唱片亦被认为是一张类似EP类型的粤语唱片。张学友在此张专辑的MV开始尝试利用电脑技术制作，融合数首主打歌曲，包括《
_饿狼传说_ 》、《非常夏日》等，上市时收到了很好的效果。唱片被分为数种版本，市场上常见的分别有正式版，早期的万宝路红版（Marlboro Red Hot
Hits），香港本地EP版（收录四首歌曲）以及日本EP版本（收录四首歌曲）。
