_Amerika_ 是德国工业重金属乐队Rammstein的歌曲，收录在其专辑Reise,
Reise中。该乐队由德国东部地区一群厌倦了工厂生活的无产阶级组成。他们秉承了德国电子乐的一贯作风——无感情、冷冰冰、有着金属般的质感。他们通过吉它采样的方式进行创作，完全用德语演唱，背景里常常穿插了阴冷而飘渺的伴唱，类似中世纪的感觉，他们的音乐混杂了工业和电子成分。
