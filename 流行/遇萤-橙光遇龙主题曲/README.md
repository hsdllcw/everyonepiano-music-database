

**遇萤**
是由YT君制作的橙光文字游戏《逆袭之星途闪耀》的剧中剧《遇龙》的主题曲，由CRITTY演唱，执素兮作曲，顾聆落作词。首唱版由只有影子演唱，男声版由宗顺康演唱。

《遇龙》是《逆袭之星途闪耀》里女主拍摄的电视剧。《逆袭之星途闪耀》中只有《遇龙》的部分剧情，作者就又另外制作了一个纯剧情完整版的无选项游戏。《遇龙》为原创剧情，目前不授权任何小说和游戏改编。由橙光游戏《逆袭之星途闪耀》改编的电视剧《逆袭之星途璀璨》中，《遇龙》的流萤和夏侯雪由宋轶和黄婷婷分饰。

_遇萤双手简谱_ 是网友提供的，由网友 **1113017186** 制作，感谢1113017186对EOP的支持！
