

**两个世界双手简谱** 和五线谱完全对应。

_两个世界_
是台湾知名男歌手张克帆所演唱的歌曲。张克帆，90年代初台湾少年偶像组合红孩儿的主唱成员。在1995年至2000年间出版七张个人唱片专辑，曾入围第七届台湾金曲奖最佳新人奖。后因个人财务问题短暂离开演艺圈，现主要从事谐星通告艺人以及主持人和演员的工作，也曾为数部电视剧配唱主题曲。业余时间进行歌曲创作。

歌词下方是 _两个世界钢琴谱_ ，希望大家喜欢。
