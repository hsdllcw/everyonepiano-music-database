

李荣浩第七张专辑首波覆古港风单曲《 **纵横四海**
》于2021年7月11日上线。这首歌无论编曲、还是作曲，都极具覆古港风色彩。无论世代怎麽变化，眼前的世界多么新潮摩登，永远不变的是我们热血青春一腔奋勇的心，纵横四海，时光更叠，唯有感动与痛楚如初，永至我们内心深处。

《 _纵横四海_
》由李荣浩作词、作曲，以充满覆古曲调重现90年代特有的港风魅力。不断环绕在耳边的副歌“天南地北，太多种假如，可再狂蜂的浪蝶也孤独，天南地北，摩登的建筑，也无非一早一晚一归宿”不禁让人想起
--
深夜时分，结束一天工作，走出办公大楼，擡头望天，看到眼前世界的潮流与摩登，好像跟自己显得格外不搭，但好像又与自己息息相关。面对世代的变化、流行一次又一次的轮回，我们最应该的是取悦自己，保持一开始的初心，坚守随性潇洒的人生态。
