天生歌姬A-Lin第四张专辑【寂寞不痛】首波推荐主打歌----「 _给我一个理由忘记_
」由新锐音乐人游政豪作曲、资深作词人邬裕康填词，整首歌唱出人在面对爱情消逝时的撕心之痛，情绪由收而放，渐浓渐烈，宣洩释放了积压在心底的眼泪，是2010年最能抚慰人心的一首情歌。

黄丽玲，艺名A-Lin，出生于1983年9月20日，台湾女歌手，台东阿美族原住民。出道时被封为“天生歌姬”，嗓音独特浑厚，曲风多元丰富。歌词下面是
