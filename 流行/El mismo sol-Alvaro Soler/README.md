_El mismo sol_ 是西班牙歌手兼流行音乐作曲家Alvaro Soler于2015年5月8日正式发布的个人单曲。后收录在专辑《Eterno
Agosto》中。且此曲发布后，迅速获得全欧洲的关注和巨大的人气。随后与珍妮佛.洛佩兹(Jennifer Lopez)合作制作西班牙语/英语双语版的《El
mismo sol(Under the Same Sun)》，在美国，英国以及全世界地区发行。
