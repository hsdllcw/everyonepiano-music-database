

我到底要怎么做才能够被你偏爱  
是否情话要足够动听才能叫例外  
而你置身事外  
不动声色就将我击败

抖音上这段伤感的旋律触动了众多听者的内心。据悉，此曲叫做《 **偏爱和例外** 》，由烟(许佳豪)演唱，于2021年10月8日发行。

日本著名文学大师村上春树曾说：爱情中最好的心态，大概就是深情而不纠缠。我们总会在因缘际会之下遇见一个人，爱上一个人，但很多时候，未必能够走到白头。于是总是忍不住地去在感情里面卑微付出，放低姿态，只求得他的一个回眸，一个温柔的眼神。当你在爱情里放弃了自尊，放弃了一切，这份爱就早已经变了质。因为他的心从始至终都不在你这里，所以怎么牺牲也无济于事。

所以，好的爱情该是高傲地昂起头，你来我欢喜，你走我也不挽留。不为他的离去而遗憾，不将就，也不纠缠。
