_简单爱_ 是周杰伦第二张专辑《范特西》第三支歌曲，由周杰伦作曲、徐若瑄填词，歌词简单易懂、旋律朗朗上口，2001年9月14日发行。  
  
初恋般的青涩与甜蜜，单纯且没有杂念的感觉，绝对的纯粹的恋爱，抒发恋爱中的男孩那种甜蜜的心情，曲风和节奏很容易上口。  
  
简单爱是J式节奏情歌的代表作，表现了杰伦天真烂漫的一面。简单的编曲，以弹拔乐和打击乐，构造出一个轻松愉快的初恋画面，听这种歌曲，又一种说不出的愉悦感。  
  
歌词下方是 _简单爱钢琴谱_ ，希望大家喜欢。
