_你不知道的事_ 出自王力宏专辑《十八般武艺》，由王力宏作曲填词，也是王力宏电影导演处女作《恋爱通告》的主题曲。  
  
《你不知道的事》无论是旋律还是配合着电影婉婉道来的歌词，都让人印象深刻。这首歌充满浓浓的古典味儿，词曲深刻真挚，每颗音符都重击内心；唯美而深情的旋律，简单的几句歌词，就让人深深掉入爱情的漩涡里，猜疑、无奈、不安、彷徨，因为爱，在背影守候，无怨无悔，为了爱，做了许多《你不知道的事》。
