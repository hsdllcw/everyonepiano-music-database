歌曲出自中国内地女歌手格格演唱的代表作《 **火苗** 》，由何沐阳作词、作曲。收录于2010年6月13日发行的专辑《雪莲》中。

歌词朗朗上口，唱进了心坎，果然是一首一折不扣的“洗脑神曲”！听完后，感觉空气中都充满了爱的味道！作为万年单身狗的小编突然也想恋爱了，有意者，请到EOP官方QQ群详谈，呜呜~~~

额。。突然跑题了！尴尬！
