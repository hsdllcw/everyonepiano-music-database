

2018许嵩第七张词曲全创作专辑《寻宝游戏》第三支曲目《 **如约而至** 》正式发表。

《 _如约而至_ 》由许嵩包办词曲创作及担任音乐制作人，以Folk-
Rock的曲风来呈现。在歌曲中演奏的电吉他是摇滚乐传奇人物约翰•列侬在beatles乐队期间所用吉他的相同版本，重现1960年代surfguitar音响效果，具有鲜明的复古质感。Blues-
Rock风格的鼓律动明快有力，在节奏摆荡中消解了歌词里隐含的沉重，达到了举重若轻的效果。歌曲旋律回归至简，如水般流淌。许嵩对这首作品的演唱也天然无雕饰，如同面对面的轻语，传递出内心深处的宁静与释然。
