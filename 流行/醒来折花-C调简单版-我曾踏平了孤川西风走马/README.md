

抖音“我曾踏平了孤川西风走马”出自歌曲《 **醒来折花** 》。此曲由子尧演唱，发行于2020年7月10日。

这是一首中国风、讲述思愁的歌曲，融合草原元素，给人带来耳目一新的感受。依旧由刘涛参与词曲与歌曲创作。刘涛把握着曲式走向，子尧狂放与不羁的演唱。可谓是相得益彰。

_醒来折花双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
