_真心话太冒险_ 出自TFBOYS的同名专辑真心话太冒险。此曲由王燕冰、刘佳作词，刘佳作曲，于2016年2月22日湖南卫视元宵喜乐会首唱。

一直以来，TFBOYS一直在尝试不同风格的音乐作品，从清新可爱到缓慢抒情，从热血羁绊到追梦励志，2016年他们继续寻求创意打破陈规，带来全新风格的作品《真心话太冒险》。歌曲以朋友间常玩的游戏“真心话大冒险”为引线，歌词真实表现了TFBOYS三人平日的喜好和对对方的看法，以诙谐幽默的吐槽方式呈现TFBOYS三小只的日常和满满的兄弟情谊，形式十分新颖创意。同时，歌曲欢快轻松的旋律更是让人听后忘却纠葛烦忧，心情暖暖。这首歌曲叙述了他们三个人的真实故事，却也是我们每个人和朋友的真实写照，希望能让大家更加珍惜朋友之间的友情，多对朋友说说真心话。
