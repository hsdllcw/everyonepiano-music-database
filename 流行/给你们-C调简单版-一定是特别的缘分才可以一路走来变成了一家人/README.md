

在婚礼上我们常常可以听见张宇演唱的《 **给你们**
》这首歌，无论是在高大上的酒店还是淳朴的农村小舞台。每当婚礼上响起这首歌时，我们常常会有些触动，感慨爱情的不易，畅想未来的生活。

而张宇沧桑的声音恰恰更能表达一对新人经历种种最终走到一起的不容易。歌曲前面的悠悠缠绵表达了喜结连理的害羞与不确定，不确定两个人的未来是什么样的。后面突然慷慨激昂，表明两个人携手共进，同甘共苦，互相付出，不计回报，一起创造婚后的甜蜜生活。由此可见，这首歌很适合在婚礼上演唱的，特别是有些经历的人，更能体会歌曲中的人生哲理和正得不能再正的爱情观。
