

**人鱼的眼泪双手简谱** 和五线谱完全对应。

_人鱼的眼泪_ 是EXO《XOXO(Kiss&Hug)》专辑里收录的一首歌，原名Baby don't
Cry。这张专辑是EXO的首张正规专辑，因此分别于韩国和中国活动的EXO-K和EXO-
M也将合体，以12人的形式往返于中韩两地进行活动。新专辑在2013年6月3日上市。Kiss为韩文版，Hug为中文版。

歌词下方是 _人鱼的眼泪钢琴谱_ ，大家可以免费下载学习。
