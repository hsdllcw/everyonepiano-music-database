

**U.F.O.** 是英国新一代乐队Coldplay的歌曲，出自其专辑Mylo
Xyloto中。Coldplay是英国摇滚乐坛进入新世纪之后崛起的最受欢迎的摇滚乐队之一，他们秉承了英式摇滚乐队一贯的风格，成为了英国新一代乐队中的杰出代表。1996年时在UCL（University
College
London）四人相遇，两年后决定合组此乐团。Coldplay乐队成军之后的每一张专辑都不断进步，一步一步地扩大他们的影响，目前他们正处于事业的快速上升期。

歌词下方是 _U.F.O.钢琴谱_ ，希望大家喜欢。
