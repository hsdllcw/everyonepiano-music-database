

《 **特别的人** 》是方大同填词、谱曲并演唱的一首慢板抒情歌曲，收录于方大同2014年4月11日发行的专辑《危险世界》中。

《 _特别的人_
》是一首以弦乐及缓拍琴声为主的情歌，一首会在心底不断回荡的旋律。歌曲表达了每个人在生命之中，都在等待一个特别的人，我们总是要在了解了爱与被爱之后，才能清楚看见那个完整自己的人。这首歌邀请到了Michael
Jackson、Justin Timberlake、Benjamin
Wright等多位弦乐编曲家来编排歌曲是弦乐部分。让人在听过主歌第二段和副歌部分之后都会有一种不绝于耳的感觉。
