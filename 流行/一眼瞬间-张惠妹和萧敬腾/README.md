_一眼瞬间_
是由邬裕康作词，曹格作曲，张惠妹、萧敬腾演唱的歌曲，该歌曲收录于张惠妹2007年发行的专辑《Star》中。这张专辑不但有动人深刻的万人传唱情歌，经典传颂的男女对唱歌曲，还有复古时尚的舞曲节奏以及细腻单纯的小品佳作。2008年凭借该专辑张惠妹入围第19届台湾金曲奖的最佳年度歌曲奖、最佳国语专辑奖两个奖项。
