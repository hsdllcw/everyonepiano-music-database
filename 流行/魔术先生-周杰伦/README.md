

《魔术先生》是著名歌手周杰伦的专辑《魔杰座》里的一首主打歌曲。由方文山作词，周杰伦作曲的《魔术先生》在华丽的美式纽奥良曲风优雅出场，整首歌的编曲以充满新奥尔良爵士风味乐团演出的方式呈现，如魔术般的绚烂夺目。
