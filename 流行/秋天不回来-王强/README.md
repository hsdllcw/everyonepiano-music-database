

**秋天不回来双手简谱** 和五线谱完全对应。

_  
秋天不回来_ 这首歌由王强演唱；发行于2006年12月，该歌所属同名专辑《秋天不回来》。在百度流行金曲排行榜十周排名第一。

  
原唱为韩国Click-B的歌曲To Be Continued，收录于2002年1月发行的专辑너에게/为你中。Click-
B是韩国演艺公司DSP于1999年推出的男子7人乐队（Boy band）。他们以摇滚结合舞曲的音乐风格迅速占领市场，并标榜自己华丽的外表以及惊人的实力。

  
歌词下方是 _秋天不回来钢琴谱_ ，希望大家喜欢。
