

执着于音乐梦想道路的张杰，对梦想的付出众人皆有所感悟。而张杰自身也一直希望能吟唱一首激昂却感人肺腑的歌曲，让每一个聆听着内心平静而充满力量。张杰说，在这个充满着不安的时代，焦虑和失落总是在脑海间彷徨，面对灾难人类是如此渺小，谎言让一切显得不堪，然而最终却有一种永恒的力量支撑我们——这，就是爱，至善至美的爱。
