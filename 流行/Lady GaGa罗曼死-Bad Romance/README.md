

由Lady GaGa和RedOne共同撰写的《罗曼死》( **Bad Romance** )是收录于GaGa改版EP专辑《名利怪物》(The Fame
Monster)中的首波主打单曲。其音乐录影带发布于YouTube上短短只有三个月，点击率已突破一亿大关，并引起全球模仿潮。直至2011年10月21日，该歌曲MV在影音网站YouTube的全球点击率已突破4.2亿次。
