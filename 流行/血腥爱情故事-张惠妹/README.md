

《 **血腥爱情故事**
》是张惠妹演唱的歌曲，由Hush作词，Jonnic作曲，收录于张惠妹2015年发行的专辑《阿密特2》。2015年，HUSH凭该曲提名第16届华语音乐传媒大奖颁奖礼最佳作词人奖。

《 _血腥爱情故事_
》由钢琴开启，张惠妹展现了大开大合的唱法、无痕切换的超高假音，并传达了宣泄式的爱情宣言，很适合作为选秀比赛的曲子。该曲在曲子的编排上，如同小说般的起承转合。歌词部分，“就让我紧跟着你起承转合，让我为你写一本恐怖小说。越血流、越手酸、心越空、肉越痛，千刀万剐的感情才生动”，激烈且充满张力，字字见血，令听众过瘾。
