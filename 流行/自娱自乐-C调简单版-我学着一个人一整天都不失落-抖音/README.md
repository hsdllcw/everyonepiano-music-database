

近期刷抖音的朋友有没有被这段旋律吸引了“我学着一个人一整天都不失落，就让我喝醉了，唱着歌自娱自乐”。据悉，此曲叫做《 **自娱自乐**
》，由金志文演唱，发布于2021年8月29日。

《 _自娱自乐_
》描述着成年人对生活中各种烦恼“自娱自乐”式的排解。音乐响起的那一刻似乎可以暂时放下盔甲，肆无忌惮的跟着音乐跳动，找寻生活中的色彩，学会自娱自乐大概是作为一个成年人努力活着的美好方向吧。
