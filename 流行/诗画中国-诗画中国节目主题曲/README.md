

《 **诗画中国** 》由许嵩和谭维维演唱、许嵩作词作曲及制作，是2022年中央广播电视总台推出的大型文化节目《诗画中国》的同名主题曲。

诗写画意，画传诗情。该作品从寻访诗人画家的角度起笔，带着真挚的情感与忘我创作、书绘山河的作者们进行跨越时空的对话，在许嵩与谭维维娓娓道来、层层递进的演绎中，歌咏了在诗与画中典藏的神州之美。
