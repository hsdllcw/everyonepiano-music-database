

抖音“你会长出羽翼，拥抱星河万里，将这世间温柔都尽收眼底”出自歌曲《 **星河万里** 》由95后音乐人Rom邢锐演唱，发行于2021年。

最近听过一段很温暖的话：“希望在你丧失表达欲的时候，能有人温柔的接住你的疲惫。”成年人的世界，懂比爱重要，可是没有爱，何来懂！
