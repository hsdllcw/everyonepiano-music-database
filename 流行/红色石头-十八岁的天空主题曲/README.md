

**红色石头** 是电视剧《十八岁的天空》主题曲。歌手：李智楠,此曲收录于他2006年1月20日发行的专辑《简单的一天》。  
  
李智楠，国内著名演员，2000年赴日本留学，2002年凭青春偶像剧《十八岁的天空》一举成名，之后又接连出演了多部影视剧。音乐代表作品《红色石头》、《一个人的寂寞》《起跑天堂》等。电视剧《十八岁的天空、《幸福最晴天》、《千山暮雪》、《烽火佳人》、《十八岁的天空2》等。其中《烽火佳人》未播先火，预计2013年播出。
