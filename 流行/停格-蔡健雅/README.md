_停格_
是2015年，蔡健雅在推出个人最新专辑之前，抢先推出了为电影「澳门风云Ⅱ」所谱写的最新歌曲。蔡健雅的知性气质和略带沙哑的感性嗓音一直是都会情歌的最佳代言人，她的许多经典歌曲更成为都会男女们遭逢情伤时必听的疗伤歌曲。

停格，此曲描述的是相爱却注定无法相守的悲剧爱情，然而彼此拥有最美的一刻却将永远在心 中停格，即使花儿枯了，心脏停了，这份记忆也永远不会腐朽。
