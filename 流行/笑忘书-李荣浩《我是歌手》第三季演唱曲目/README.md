

**笑忘书双手简谱** 和五线谱完全对应。

《我是歌手第三季》第四场录制落下帷幕，第三场折戟而归的李荣浩卷土重来，返场深情演绎天后王菲的经典情歌《 _笑忘书_ 》。

李荣浩高难度挑战天后王菲的《笑忘书》，因有多年音乐制作经验傍身，李荣浩对于歌曲的拿捏十分精准到位，循序渐进的情绪处理，琐碎的呢喃、自言自语的絮叨，在他沙哑低沉的歌声中全都灌注成触动心声的涟漪。坦然淡定的吟唱中有曾经沧海的缠绵悱恻，也有孤独高歌的千回百转，但更多的是笑忘过去迎接新生的坚定，不仅把林夕在歌词中隐藏的情意表达得淋漓尽致，也将这首深深打上王菲烙印的《笑忘书》唱出了别样滋味，少了一份原版的幽怨婉转，却多了一份男儿的率性洒脱。

歌词下方是 _笑忘书钢琴谱_ ，希望大家喜欢。
