

**爱转角**
是台湾偶像剧《转角遇到爱》的主题曲。《转角遇到爱》是一部带有轻松喜剧色彩的偶像剧。由徐熙媛，罗志祥，陈至恺，范明等两岸三地演员主演。主题曲由罗志祥演唱。

罗志祥（Show Lo），亚洲超人气偶像天王，有着亚洲舞王之称。是一名集歌手、舞者、演员、主持人等多方位发展的的全能艺人。
