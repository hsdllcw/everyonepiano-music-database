_青石巷_ ，此曲由魏琮霏创作。同时也是休闲益智游戏《别踩白块儿2》中的背景音乐。

关于歌谱，别踩白块里的青石巷是节选，而且只是原曲的前半部分，后半部分因为曲调非常诡异恐怖在一开始发行就被剪掉了，完整的歌谱一直没有流传出去。

魏琮霏生前是一个专门为葬礼设计钢琴曲的人，当时的人都称他“引路人”谱出的音乐多是现场即兴演奏的，所以留下来的乐谱非常之少，青石巷就是这少之又少中的一首，而魏琮霏也因为这首歌一病不起，直到离开人世。
