_星辰_ 是电视剧《择天记》的片头曲，由刘畅作词，谭旋作曲，张杰演唱。

歌曲曲风热血激昂，词风豪气万丈，开启逆天改命新世界。此次《星辰》的创作邀请到《凉凉》的词曲原班人马，知名影视音乐制作人谭旋以及国内知名作词人刘畅联手打造。强强联合，打造热血主题曲《星辰》。张杰用自己独具特色的演绎方式，向听众展开一幅少年意气的青春画卷。歌词单刀直入，径直对准“逆天改命”这一主题。
