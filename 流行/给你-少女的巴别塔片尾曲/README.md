_给你_
是由SNH48成员易嘉爱自编自唱的原创曲目，于2015年5月30日《我的太阳》易嘉爱生日特别公演作为特别节目首次披露，随后作为SNH48首部大型纪录片《少女的巴别塔》片尾曲受到广泛好评。正式版音源于同年7月21日在微博首发。

用最真实纯粹的情感带入演唱，描写每个人在成长过程的青春岁月里，所经历的烦恼、挫折以及收获的友情。"要给你依靠不只是随便说说而已，相信彼此是最美好回忆"~

歌词下方是 _给你钢琴谱_ ，希望大家喜欢。
