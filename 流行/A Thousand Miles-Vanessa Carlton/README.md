

**A Thousand Miles** 是美国最卖座电影[小姐好白]的插曲。演唱者 _Vanessa Carlton_
（瓦妮莎卡尔顿）是位天才音乐少女，8岁就完成生平的第一部作曲作品。这首歌曾连续数周雄踞欧美流行音乐榜首的位置，是其所演绎歌曲中经典之经典。  
  
歌词下方是 _A Thousand Miles钢琴谱_ ，希望大家喜欢。
