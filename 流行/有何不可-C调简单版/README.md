

《 **有何不可** 》是由许嵩作词、作曲并演唱的一首情歌。

在该曲中，许嵩用充满弹性的咬字，跳跃、轻松的律动感，清新略带慵懒的嗓音，完美了诠释出了“只想为你唱首歌，没有什么风格，为你放弃世界有有何不可”这样的情感，让人感觉有一股莫名的暖流缓缓地融化了心，同时也有了一些浪漫，让人只想就这样下去。

_有何不可双手简谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
