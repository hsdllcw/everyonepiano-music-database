

“太多的理由，太多的借口，别再问我难过时候怎么过”，在2002年《乌龙院闯情关》这一部电视剧播出的时候，陈冠蒲演唱的这首《 **太多**
》真的是红遍了大江南北。

此曲真的是深入人心，因为这首歌是对爱情的一种描述，当时的陈冠蒲也没有想过自己会因为这一首歌而红遍大江南北，直到现在其实这一首歌都还会徘徊在我们的耳边，独特的嗓音，优美的旋律，真的做到了百听不厌的那种。

后来《 _太多_ 》这一首歌也成为了很多电视剧的歌曲，就连大家都非常喜欢的《倩女幽魂》、《新蜀山剑侠》都选用了这首歌。
