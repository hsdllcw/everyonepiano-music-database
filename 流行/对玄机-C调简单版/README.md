

**对玄机** 是由任然和王梓钰演唱的一首歌曲，由偏生梓归作词，殇小谨作曲。

创作背景：  
她一生，有关风月之事，回首之间最为坦荡；她一生，似短又长，笔下提来苦痛，是爱非恨。  
她是唐朝女子中最为明朗清疏的一颗星，却也是坠落最快的一颗。  
她曾被所慕之人介与好友为妾，又因为正室所不容而独留于道观。她这一生爱的热烈奔放，因而眼中也分外不容沙。  
她因肆意纵情名扬长安城，却也因疑似妒杀奴婢而落幕于长安城。  
后人对她的评价褒贬不一，却独不敢否定她的一身才情。  
鱼幼薇，道号玄机，卒于极刑，享年二十又七。鱼幼薇最早爱的是温庭筠，但温庭筠那时年老不敢担负她的未来，便把鱼幼薇介绍给自己朋友李亿，鱼幼薇惨笑，假戏真做，为的便是赌这一口气，而李亿娶鱼幼薇为妾后处处受到正妻刁难。  
后爆发战乱，李亿把鱼幼薇留在寺庙中许下承诺，数年后定来接她。但鱼幼薇多等了数年，迟迟不见李亿，“呵！”鱼幼薇惨笑。  
自此鱼幼薇改名为“鱼玄机”整日放纵自己，纸醉金迷，她恨，她的爱原来一开始就没有结果。她的命运如此多磨，便在不相信世间情爱，再不相信。。。鱼玄机在外人看来不知廉耻，似青楼女子一般，但谁又知道她鱼玄机曾经也叫鱼幼薇呢？  
谁又知道她鱼幼薇曾想愿许一人偏爱，尽此生之慷慨呢？  
没有，世人只看到她是风尘女子鱼玄机，却不知她曾是痴心女子，鱼幼薇。
