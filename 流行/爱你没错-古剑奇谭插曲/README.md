

**爱你没错双手简谱** 和五线谱完全对应。

_  
爱你没错_ 是张信哲最新演唱的一首歌曲，由代岳东填词，曾缔、谭旋谱曲，Mac
chew编曲，同时也是2014年电视剧《古剑奇谭》的插曲，在欧阳少恭回忆巽芳之间的感情时放出。

  
歌词下方是 _爱你没错钢琴谱_ ，希望大家喜欢。
