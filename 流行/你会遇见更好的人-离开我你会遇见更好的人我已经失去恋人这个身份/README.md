郭聪明新单曲《 _你会遇见更好的人_
》，一首会刺痛神经的小情歌更贴切这个季节，虽然有时爱情带来的伤痛无可避免，但是一定会有治愈你的人来到你身边，即使分开也彼此祝福，因为我们都会遇见更适合的人。
