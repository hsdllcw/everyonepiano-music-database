

在《我是唱作人》总决赛上，梁博带来了一首原创作品《 **我不知道** 》，歌曲以一种温柔视角来对待当下网络暴力的不良现象。

梁博低沉而沙哑的嗓音温柔地娓娓道来，把世间的烦恼、焦虑、无奈、孤独、不舍，描绘得五味杂陈挠心抓肺，希望那些孤独和焦虑的灵魂，一觉醒来变成小孩，忘记曾经经历的悲伤和无奈。
