

**祝你一路顺风** 最初是吴奇隆送给即将服兵役的陈志朋的，而多年之后陈志朋的《舍不得》专辑中，他自己翻唱了这首歌，特别提到了翻唱当年吴奇隆送给他的那首《
_祝你一路顺风_ 》。“当时我要去当兵，吴奇隆唱的时候更多是感动、伤感，现在我要回送给他温馨、温暖的感觉。”

  
词：吴奇隆 曲：李子恒  
歌手：吴奇隆 专辑：追风少年
