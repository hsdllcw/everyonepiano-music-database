

《 **故乡** 》是许巍演唱的一首歌曲。收录在专辑《那一年》中。这首歌是一首摇滚风格的歌曲。歌曲展现了许巍高超的唱功及创作功力。

很多人以为这首歌是许巍写给自己家乡的，其实这首歌是他写给一直默默支持鼓励他的妻子袁枫，说袁枫在他的心里就像故乡一样！
在许巍最潦倒，贫困，疾病的时候，妻子对他的不离不弃，悉心呵护！让许巍度过了人生中最黑暗的时候。一句“你是茫茫人海之中我的女人”够爷们，够霸气，绝对算得上歌曲里最像情书的一首歌！

《 _故乡_ 》整首歌意境沧凉，憾人魂魄！尤其是前奏的吉他弹奏，层层递进，把人带进一种悲凉的境界！这也是许巍左其顿的巅峰之作！
