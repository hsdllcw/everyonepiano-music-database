

抖音“还不能够从你的梦里苏醒，我还没有缝补受伤的躯体”出自歌曲《 **问风** 》。《问风》是金渔演唱的歌曲，由李伦序作词，李伦序作曲，季志豪编曲。

歌词要表达的是一个女孩子默默地深爱着一个男孩子，却没有得到回应或者说只是她单方的感觉。男孩子并不喜欢她，而女孩深受伤害。最后算是一种劝诫，告诫女生一定要懂得自爱，不要毫无保留地掉进爱情的漩涡里，不要总是为情所困。

这是一首有些小伤感的情歌。失恋后情绪低落，用歌曲来寄托思念，同时也表达了内心对于这份感情的无奈。
