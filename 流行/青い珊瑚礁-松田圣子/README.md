

《 **青い珊瑚礁** 》是日本歌手松田圣子演唱的歌曲。

松田圣子（Seiko
Matsuda）（1962年3月10日－），日本福冈县久留米市人，是日本知名的女歌手与女演员。1980年以单曲《裸足的季节》（裸足の季节）出道而备受瞩目。纵横歌坛20年盛名不衰，在日本乐坛排行榜上曾创下连续24张单曲都拿下第一名的纪录。至2006年前无人能破。松田圣子有过两次婚姻，因为嫁给神田正辉而从夫姓神田，离婚后并未改回。他们所生的女儿神田沙也加亦为歌手与演员。

歌词下方是 _青い珊瑚礁钢琴谱_ ，欢迎大家使用。
