

孙耀威1994年的歌曲《 **爱的故事上集** 》，是一首日文的改编歌，原是1993年The 虎舞竜的主唱高桥恭司（George
takahashi）的作品，据说背后还有一个动人的故事。至今孙耀威的《爱的故事上集》仍是最流行的经典版本。

这首歌节奏轻柔流畅，孙耀威用细腻的声线情意绵绵地道出了懵懂少年的恋爱故事。这段故事，有夜色，有徘徊，也有迟疑和爱意。然而人们的痴心总是无疾而终。等着等着听歌的人也白了头。
