

**Oh My God** 是韩国女子演唱组合(G)I-dle的歌曲，收录在其第三张迷你专辑《I Trust》中。

“这张专辑的目标是集中一个人能感受到的所有情感”。同时，“I
trust”也传达了这样一个信息：尽管其他人可能称我为魔鬼，但我仍将以自己的方式展示我的清白，”Soyeon说这张专辑以自信为中心，首先要相信自己才能带来自信。”
