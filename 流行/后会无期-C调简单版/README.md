

《 **后会无期** 》是徐良和汪苏泷演唱的一首歌曲，收录在徐良2011年发行的专辑《不良少年》中。

此曲流畅且具有超强传唱性的旋律，同时徐良对节奏的良好把握度，也让人眼前一亮。而且徐良的签约公司美妙音乐给予了他充分的空间，并没有大刀阔斧地对他的音乐进行更市场化的精良包装，这反倒保留了徐良音乐中的最宝贵的淳朴面貌。

这歌曲表达一种情绪，徐良和汪苏泷与一个女主角一同出境演绎这首有点抽象的MV。MV远赴台湾苗栗取镜，徐良一身黑色造型酷劲十足，汪苏泷则身着从未尝试的荧光色系蕾丝做装饰的衬衫，导演陈映之力求把这只网络上深受歌迷喜爱的歌曲通过视觉完美呈现出来。
