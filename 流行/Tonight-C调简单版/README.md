

**Tonight** 是韩国超人气组合BIG
BANG的歌曲，发行于2011年2月24日，收录在其第四张同名迷你专辑《Tonight》中。天才般的团队，神人般的节奏，好听到无可挑剔！

MV在美国拉斯维加斯拍摄，通过拉斯维加斯繁华夜景下倍显寂寞的Big Bang成员，演绎出了《 _Tonight_ 》隐藏在华丽旋律下的悲伤情感。Big
Bang的第四张迷你专辑在24日推出后就席卷了韩国国内所有音乐排行榜，不到一周时间里销量就轻松突破了10万张。
