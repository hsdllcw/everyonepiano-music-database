

杨宗纬歌曲《 **其实都没有** 》写出了自我独白的心灵鸡汤，发现把自己掏空、在“其实都没有”的状态下调整归零，才会拥有更多、看到学到更多！

在这首歌中，杨宗纬用“自白式”的创作填词，把这几年自己因为工作来往不同都市游走的疏离及心情，忠实不掩饰的表达出来。杨宗纬表示，因为歌唱，让自己来往不同的城市间，接触许多不同的人与关爱，心灵的丰盛与归零就在每一次移动瞬息万变。他说，当写下《
_其实都没有_ 》这首歌时，他期许自已能更洒脱。

歌曲没有复杂的乐器、过多的和弦、大的起承转合、分明的层次，也没有转音、没有飙高音，而是用音乐给他最原始的爱与力量，哼唱出一个故事，一幅画，一段心路历程，唱出了歌手对每一座所走过城市的观察、记录，把生活在城市的人们的心情表达出来。
