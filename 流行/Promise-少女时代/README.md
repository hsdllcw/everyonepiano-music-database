_Promise_ 收录在少女时代正规四辑《I Got A Boy》中R&B风格的歌曲，是一首中拍抒情歌曲，甜蜜旋律非常符合少女时代甜美嗓音的曲子。

少女时代（Girls'Generation）是韩国SM娱乐有限公司于2007年推出的女子流行演唱团体，由金泰妍、郑秀妍、李顺圭、黄美英、金孝渊、权侑莉、崔秀英、林允儿、徐珠贤9名成员组成。

歌词下方是 _Promise钢琴谱_ ，希望大家喜欢。
