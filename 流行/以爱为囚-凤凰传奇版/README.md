《 _以爱为囚_
》原曲来自于俄罗斯，带有一丝悲伤之感。2022年，凤凰传奇改编版《以爱为囚》，唱出了新感觉，很多网友听完都感叹，这哪是以爱为囚，明明是刑满释放！

《以爱为囚》歌词表达了爱情是心甘情愿的囚牢，陷入其中无法自拔的，就如同牢笼一般的痛苦，让人无法摆脱，却又是自愿的选择。

对于凤凰传奇版《以爱为囚》，他们在咬字、节奏和轻重音上都做了调整，一开口所有负面情绪都被吹散了，尤其是曾毅的rap。总之。听完后感觉佛光普照，唱出了爱的坦坦荡荡，爱的互相救赎！
