

“那就这样吧，再爱都曲终人散啦，那就分手吧，再爱都无需挣扎”原是动力火车1999年演唱的歌曲《 **那就这样吧**
》。后来，张一山翻唱了此曲，并再次爆火起来。

从歌词，不难看出，这是一首很凄美的情歌，歌词都有着浓浓的故事感以及情景感。

所以，小编觉得这首歌的背后，一定有很多辛酸和无奈。也许是因为男孩与女孩出现了情感问题，亦或者一方喜欢上了别人而造成了分手。“那就这样吧”简单的一句话，却道出了特别多的心酸与无奈，以及不舍和念念不忘。

最后，送大家几句话共勉：  
错的人迟早会走散，  
对的人迟早会相逢；  
珍惜所有的不期而遇，  
看淡所有的渐行渐远；  
失去的都是配角，  
留下的才是人生！
