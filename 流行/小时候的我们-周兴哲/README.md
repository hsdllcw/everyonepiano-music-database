_小时候的我们_
是由周兴哲、吴易纬作词，周兴哲作曲并演唱的一首歌曲，收录于同名专辑《小时候的我们》发行于2020年1月7日。这张专辑想回到每个人童年中最单纯、最直率的时刻，有些感受会随着年纪慢慢成长、慢慢变成熟，有时候会让人忘了那第一次的感动，而这张专辑就是想带听众找回那第一次的感动！就像第一次谈恋爱、第一次失败、第一次写下一首歌，让音乐为自己说话，回到那样的感动。
