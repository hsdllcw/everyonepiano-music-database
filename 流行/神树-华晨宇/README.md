

《 **神树** 》是华晨宇作曲并演唱的歌曲，由裴育作词，于2020年4月19日发行，收录于华晨宇专辑《新世界NEW WORLD》中。

歌曲以极简泠冽的风格，勾勒出万籁俱寂、海啸山崩的末日图景，在阵阵寒意中发出一声声不肯屈服的“Never Give Up”，让“希望”的种子冲破土壤。

2019年11月15日，《 _神树_
》首次在“2019华晨宇火星演唱会”中演出。2020年3月13日，该歌曲于《歌手●当打之年》第六期节目中电视首唱，Live版本发表于网易云音乐。
