《 **面具** 》是香港流行歌手许廷铿的一首粤语歌曲。收录于专辑《蓝调》中，发行于2012年01月12日。

英式摇滚的《 _面具_ 》，虽然在许廷铿的演绎下，缺少点神经质和摇滚独特的狂野，但扎实的声线，以及老练的衔接，也能让它成为一首不错的粤语硬情歌。
