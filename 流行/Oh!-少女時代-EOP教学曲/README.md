

**Oh!双手简谱** 和五线谱完全对应，属于EOP教学曲。

_  
Oh!_
是韩国组合“少女时代”的第二张专辑的主打歌曲曲目，由知名音乐人Kenzie创作，也是专辑的名字。发行时间：2010年01月28日，在朝鲜预售近15万张。成绩超乎预期。而与专辑同名的主打歌〈Oh!〉，在1月25日网络上首次发布后，十分钟之内就在Real
Time音乐排行榜站上榜首。

  
歌词下方是 _Oh!钢琴谱_ ，希望大家喜欢。
