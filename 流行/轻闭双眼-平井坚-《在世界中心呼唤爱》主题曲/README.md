

《 **轻闭双眼** 》（瞳をとじて）是平井坚的第20张单曲。于2004年4月28日由DefSTAR
Records发行。是电影《在世界中心呼唤爱》的主题曲，销量超过100万，并排名当年Oricon公信榜的年榜冠军。

《 _轻闭双眼_ 》获得2005年（第四届）MTV日本音乐录像带大奖的「最佳男艺人音乐录像带奖」和「最佳电影音乐录像带奖」。

平井坚于2005年的第55回NHK红白歌合战中演唱此曲，并且在压轴前一位的重要位置演唱。在此曲的音乐录像带中，女演员奥田惠梨华出演。台湾歌手游鸿明曾翻唱此曲，曲名为《我可以》。这张专辑中还收入了翻唱老鹰乐队的歌曲《亡命之徒》（Desperado）。
