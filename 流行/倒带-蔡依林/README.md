_倒带_ 是蔡依林演唱的一首歌，由周杰伦填词谱曲。收录在蔡依林2004年2月27日发行的专辑《城堡》中，是电视剧《求婚事务所》的插曲。  
  
《倒带》有着“英式摇滚”的曲风，2006年在香港演唱会时，蔡依林并没有在台上说起感情事，但仍选唱了两首由周杰伦写的歌，尤其在演唱《倒带》的“过去甜蜜在倒带，只是感觉已经不在”时，她一时感触哭成花脸，这首歌充满了蔡依林的感情。  
  
在2004演唱会上周杰伦演唱倒带，并改词“我在幸福的门外，却一直都进不来”改唱为：“我想你应该明白 却一直都进不来
”，“而我对你的期待，被你一次次摔坏”改唱为：“过去你给的期待 被我一次次摔坏”。  
  
歌词下方是 _倒带钢琴谱_ ，希望大家喜欢。
