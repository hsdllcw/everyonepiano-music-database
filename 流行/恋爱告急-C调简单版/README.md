

《 **恋爱告急** 》是鞠婧祎演唱的歌曲，由符酷作词，胡臻作曲编曲，收录于同名专辑《恋爱告急》中，发行于2019年11月8日。

这张迷你EP是自2019年6月单曲《孤独与诗》以后，鞠婧祎以全新主打风格带来的专属音乐作品，此前，鞠婧祎带来了诸多质量上乘的抒情歌曲、影视金曲，这次她则将以富有感染力的“偶像”曲风回归，活力满载的青春恋爱主题，集结了中国和海外的优秀艺术家为其量身打造的两首全新单曲《恋爱告急》、《虹》，两种气氛完全不同的歌曲表达鞠婧祎一路积极向上、不断追求突破的心境。
