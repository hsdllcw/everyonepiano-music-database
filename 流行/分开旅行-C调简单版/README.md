

**分开旅行** 由刘若英、黄立行演唱，想表述的就是两人关系就像一场分开的旅行，就算是天各一方，若是有爱纷飞千里都心系对方，若是无情两两相依也是各怀鬼胎。

《 _分开旅行_
》与刘若英一贯的曲风有很大分别，歌曲旋律带英伦摇滚的风格。刘若英不同以往的演绎和尝试，给人以桀骜不驯的印象。刘若英的歌声、戴爱玲的声乐、黄立行的歌声与饶舌，三人的声音让该曲更显生动、特别
。歌词描述一对情侣于旅行的目的地有不同意见，男的要去洛杉矶，女的却想去巴黎，最后决定分开旅行，各自飞向自己想去的地方。歌曲点出了爱情中的症结，像刘若英和黄立行所唱的那样，让人心悸的其实不是旅行地点的分歧，而是想挣脱彼此的念头。
