

**美しい稲妻双手简谱** 和五线谱完全对应。  
  
《美しい稲妻》是日本女子偶像团体SKE48的歌曲。  
  
SKE48由秋元康担任制作人，于2008年出道，以名古屋荣为中心活动的日本大型女子偶像团体。
与AKB48并不相同，特征是成员主要出生于当地（特别是爱知县）。  
  
下面提供了 _美しい稲妻钢琴谱_ 和歌词，希望大家喜欢。
