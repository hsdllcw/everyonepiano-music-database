

**逍遥叹** 是2005年根据国产单机游戏《仙剑奇侠传》改编的热播玄幻神话电视剧《仙剑奇侠传》的插曲之一，由陈宇任作词作曲，屠颖编曲，演员胡歌演唱。

  
《 _逍遥叹_
》歌曲MV时长05：27，歌曲伴随着男女主角间曲折离奇的动人爱情故事，给人以极大的震撼，故事的情节做到了首尾呼应，有章有序，丝毫没有让人觉得有凌乱的感觉。

  
歌词下方是 _逍遥叹钢琴谱_ ，希望大家喜欢。
