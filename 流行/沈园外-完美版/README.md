

《 **沈园外** 》这首歌是最近突然火起来的一首歌，由爱写歌的小田作曲，阿YueYue、戾格
、小田音乐社共同演唱，目前这首歌在网上的翻唱度很高，深受大家的欢迎。

这首歌背后的故事是讲述了唐婉、陆游、赵士程三人之间的爱情故事。都知陆游沈园钗头凤，可谁又知此后唐婉几年后郁郁寡欢离世。都知陆游为孝义与唐婉和离，可谁又知赵士程为娶唐婉扛了多少的压力。世人都知陆游唐婉情深义重却爱而不得，可谁又知唐婉死后赵士程终身未娶。其实沈园不如不遇，一遇的终是苦了三人，一遇误终身。
