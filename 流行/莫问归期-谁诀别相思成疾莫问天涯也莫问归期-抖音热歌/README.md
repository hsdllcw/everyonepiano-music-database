

抖音“谁诀别相思成疾莫问天涯，也莫问归期”是蒋雪儿演唱的《 **莫问归期** 》。由堇临作词，刘涛作词、作曲。

心以动，奈何情以远，离别过后，何处相遇，相思成疾，无药能医，也不在问归期，粗茶淡饭冷暖自知，心亦如此莫问归期。
