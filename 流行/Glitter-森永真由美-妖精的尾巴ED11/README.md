

**Glitter双手简谱** 和五线谱完全对应。

《Glitter》是日本歌手森永真由美演唱，日本动画《妖精的尾巴》（FAIRY TAIL/ フェアリーテイル）的第十一首片尾曲。

由东京电视台出品，A-1
Pictures和SATELIGHT负责动画制作的长篇电视动画《妖精的尾巴》改编自真岛浩创作的同名魔法题材少年漫画。作品讲述了魔导士公会“妖精的尾巴”的成员露西、纳兹、哈比、格雷、艾露莎和他们的众多伙伴们在这个魔法世界中的一系列热血故事。

歌词下方是 _Glitter钢琴谱_ ，欢迎大家使用。
