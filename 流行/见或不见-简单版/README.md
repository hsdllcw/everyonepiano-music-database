_见或不见_
是2010湖南卫视自制女人穿越宫廷大戏《宫锁心玉》当中的片尾曲，由当红人气影视男星（何晟铭）亲临演绎。唱出了四阿哥（何晟铭饰）对晴川（杨幂饰）的种种爱慕情怀。

《宫锁心玉》，又名《宫》，是《宫》系列的第一部。该剧讲述一个现代少女洛晴川，穿越到清朝，经历古代的宫廷生活，经历一番“宫心计”和男女情爱之后，回到现实中，更加懂得珍惜当下的故事。

歌词下方是 _见或不见简单版钢琴谱_ ，希望大家喜欢。
