

**May It Be** 是彼得杰克逊导演的电影魔戒的主题曲，且由Enya演唱。 _may it be_
获得了极高的学术荣誉和商业地位,发行后在德国单曲榜成为了冠军。在多国下载榜也位居后座,之后获得奥斯卡<最佳原创歌曲>金球奖<最佳原创歌曲>格莱美奖<最佳电影歌曲>提名。
