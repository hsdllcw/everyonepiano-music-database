

薛之谦新歌《 **洛城** 》于2021年12月31日上线，悠然的声音配上明朗的音乐节奏，演绎出江湖里洛城中的前尘悲歌。

英年侠气，遍洒古都。恨东风，不惜世间英雄。笑谈中，死生同，一诺千金重。这首歌曲当中表达的是侠气，是情谊，也是城市，是岁月。正如歌曲的名字一样，《 _洛城_
》唱尽了这座音乐“洛城”中的每一个侧面，有侠士往来红尘内外的一腔豪情，有英雄剑吼西风、渴饮敌血的报国壮志，亦有红颜镜中落泪、哀叹年华的凄美婉转。只是凡此种种，皆已淹没在岁月的长河之中，薛之谦将音乐作为开启时空的钥匙，带领听众回到那个充满故事的“洛城”。
