

时代少年团被邀请演唱了献礼歌曲《 **青春走在新征程** 》！歌曲于2021年6月28日发行。温暖蜜嗓，一起发扬时代正能量，根正苗红的少年们啊！

凭借着这两年的努力，时代少年团终于打破了TF二代团的旧印象，成为了内娱当红一线顶级偶像男团组合。

近期的他们，先后在《东方风云榜》和《TMEA音乐盛典》上拿到最佳团体奖和年度十大金曲奖，并且凭借舞台表现力，进一步提高了组合的知名度。

有了代表歌曲，有了经典舞台，还有了来自华语乐坛的奖项，如今再加上演唱献礼歌曲的主流认可，时代少年团的演艺事业，完全可以说是一片大好。
