

君不见

君不见  
妾鼓瑟绵绵

歌词出自时间胶囊演唱的古风歌曲《 **青丝** 》。初听便被它的戏腔和声音打动，无法自拔。

主唱用人声代替二胡，可以称得上亦真亦假。单曲循环了一整天，尤其喜欢戏腔和唱词。为所爱千娇百媚茶饭不思的女子，本质上和为信念苦战不息的勇者一样浪漫。“丝”“思”同音，“青丝”即为“情思”之音。

而几句“君不见”更是唱出了一个女子的痴情与哀怨，莫名觉得哀愁，只愿“心悦君兮君也知”！
