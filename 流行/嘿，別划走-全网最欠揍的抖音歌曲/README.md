“哈哈哈哈哈打不过我吧，没有办法我就是这么强大，哈哈哈哈哈追不上我吧，啦啦啦啦啦啦啦啦啦啦啦，哈哈哈哈哈被我打败啦，全都一起上吧我根本没在怕。”
