

2021年7月17日薛之谦生日之际，他又带着新歌来了——作为薛之谦第十二张专辑的首波主打爱情哲思单曲，《 **变废为宝**
》正式上线，歌曲由薛之谦、茶茶共同作词，茶茶作曲。

年轻时的爱情，总有自不量力的美，粉身碎骨，《 _变废为宝_
》恰恰与之相反，不冲动、不巧饰、不煽情，是经历过疾风烈雨后的宁静与娓娓道来。编曲的细腻心思在平静的主题下悠扬释放，在钢琴与吉他的传统架构里，用氛围去把握“等到世间万物睡个好觉
再把你拥抱”的婉转情愫，在游刃有余的人声中，激烈的情感被慢慢回收，电子音效的点缀与合成器制造出的空间感形成互文，以反转情感疏离的暗涌。

爱情宇宙中的温度和星光，屏息守护，是我谨小慎微的希望；尘世纷扰，顿然窥见你双翼丰满的羽毛。身在其中而脱离了主观视角，以他者之感官与意识，见微知著，一边极力克制自省，一边撕开本质伤疤。
我们每个人心里都有一个美好的世界，没有世俗和争执，这样美丽的爱情才能被保存。爱情是随着世界的质感一起变化的，所有美好的事物都慢慢地躲在希望里，变废为宝也是一种向往。

如果上一次的《天外来物》歌唱了奋不顾身的自我奉献，那这一次的《变废为宝》则是看破世间冷暖的大爱无边。
