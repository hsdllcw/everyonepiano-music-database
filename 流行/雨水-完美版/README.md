《 _雨水_ 》出自音阙诗听以节气打造的音乐专辑《二十四节气》，由音阙诗听李佳思演唱，2020年4月4日发行。

雨水，是二十四节气之第2个节气。其含义是降雨开始，降雨量级多以小雨或毛毛细雨为主。俗话说“春雨贵如油”，进入雨水节气，我国北方地区尚未有春天气息，南方大多数地方则是春意盎然，一幅早春的景象。

而此曲以淅淅沥沥的雨声开始，瞬间将听众带入到雨水季节的氛围中。
