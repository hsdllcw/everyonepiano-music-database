_Santa's Coming For Us_ 是Sia圣诞专辑Everyday is Christmas中的首支单曲。此曲由Sia和Greg
Kurstin合作完成。

Sia的嗓音有种扭曲的质感，沙哑时像刀锉，嗞嗞作响，高音嘹亮引人入胜，即便有几个小破音，也成了歌中真挚情感的佐料，再加上澳洲人独特的英语发音，让她的歌声别具一格。
