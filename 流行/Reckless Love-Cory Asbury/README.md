_Reckless Love_ 是美国基督教音乐家Cory Asbury演唱歌曲。发行于2017年10月27日，作为同名专辑Reckless
Love的主打曲目。此曲由Caleb Culver，Cory Asbury以及Ran Jackson共同创作。同时，这首歌曲也成为Cory
Asbury的首张荣登2018年3月Billboard基督热歌排行榜的歌曲。
