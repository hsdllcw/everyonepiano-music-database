

**娃娃脸**
是后弦演唱的一首歌曲，由后弦作曲、作词，收录在后弦2009年发行的同名专辑《娃娃脸》中。这首歌曲在2009年第228期酸酸乳中联榜内地榜中排名第五位。

后弦表示《娃娃脸》的灵感来自自己在网上和朋友聊天互相发送的兔斯基、YOYO猴等卡通表情。

《 _娃娃脸_
》是一首R&B加上电子音乐加上中国风揉和出来的歌曲。这首歌曲在曲风上让人有耳目一新的感觉，比较适合快节奏却又复古的现代都市生活。在这首歌曲里，后弦用鼓的节奏和幸福的唱腔轻轻的描述出许多已经逝去的童趣
，引用了不少童话和动漫中的元素。但作为一首表明时尚态度的歌曲，这首歌曲一方面少了歌者应该有的鲜明人生态度，另一方面在音乐的创造力上，也不够时尚和前卫。

歌词下方是 _娃娃脸钢琴谱_ ，希望大家喜欢。
