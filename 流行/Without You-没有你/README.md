_Without You_ 是美国女歌手、词曲作家Mariah Carey的一首翻唱歌曲。  
  
此歌的原唱是英国著名摇滚乐队Badfinger（坏手指乐队）发行于1970年的一首歌曲，收录在专辑《No
Dice》中。这首歌真正流行始于1972年Harry Nilsson的翻唱。此后有超过180名的艺术家录制过此歌，并多次成为各国音乐排行榜的冠军歌曲。  
  
Without
you这首歌曲包括和音都是由Nilsson自己在录音室完成，是首高难度的曲子。在八十和九十年代，有不少自认实力派唱将的艺人，都认为非得演唱这首「炫技曲」，才能炫耀出自己的火候。包括Air
Supply合唱团和九十年代的天后Mariah Carey，只不过当Mariah
Carey于1994年初，再度把这首歌唱进排行榜时，Nilsson却悄悄的告别了人间。而只有Mariah
Carey将这首歌演绎的炉火纯青的地步，成为经典不朽的传唱。
