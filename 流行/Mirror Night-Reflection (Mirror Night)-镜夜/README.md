Mirror Night_ ，又名 Reflection (Mirror Night)，是《Deemo》初期歌曲，原曲选自V.K克专辑《镜夜》。  
  
《Deemo》是由Rayark
Inc.发行的一款音乐手机游戏，于2013年11月13日发行。该游戏与前作《Cytus》不同的是，《Deemo》是一款不折不扣的钢琴节奏作品，同时还辅以了相当精致的剧情铺垫：小黑人
Deemo 与天降少女的意外邂逅，敲响了指尖下的琴弦之音，推开了童话旅程的序幕。  
  
V.K克，1982年11月18日生于中国台湾省基隆市，配乐作曲家、流行钢琴家。音乐风格：新世纪、POP、交响乐。
