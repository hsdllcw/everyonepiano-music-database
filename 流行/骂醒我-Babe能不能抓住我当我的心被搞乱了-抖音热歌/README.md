

抖音上“Babe 能不能抓住我，当我的心被搞乱了”，是歌手周汤豪的国语歌曲，曲名为《 **骂醒我**
》。收录于《周汤豪同名专辑》。该歌曲由华纳唱片于2010年7月16日发行。

此曲为美式中慢板情歌。简单的Beat加上钢琴跟大提琴的古典气质，最后一点点的电子合成器，让歌曲呈现出不同于现今台湾的流行情歌！

歌词下方是 _骂醒我钢琴谱_ ，大家可以免费下载学习。
