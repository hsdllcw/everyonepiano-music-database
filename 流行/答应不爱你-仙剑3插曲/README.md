

《 **答应不爱你**
》是郑中基在2009年6月18日发的最新单曲，是其所在的公司金牌大风为《仙剑3》所量身打造的一张电视原声带中的一个插曲。由内地新锐音乐人金大洲作曲，堪称郑中基又一经典歌曲。这首歌是写给《仙剑3》里紫萱和长卿的。紫萱和长卿是三世情缘，最后还是分开了。歌词的含义是本来答应不爱对方了，但根本就做不到不爱对方。

郑中基，香港歌手和演员，其人多才多艺，以实力唱功闻名，喜剧表演的演技亦深得人心，摄影技术更是为人所称道。歌曲代表作有《戒情人》《你的眼睛背叛你的心》《无赖》等，影视代表作有《龙咁威2》《老港正传》《越光宝盒》等。
