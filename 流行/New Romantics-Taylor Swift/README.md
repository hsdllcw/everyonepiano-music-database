

**New Romantics** 是美国女歌手Taylor
Swift演唱的歌曲，收录于她的第五张录音室专辑《1989（豪华版）》。泰勒•斯威夫特（Taylor
Swift），1989年12月13日出生于美国宾夕法尼亚州，美国乡村音乐、流行音乐创作女歌手、演员、慈善家。
