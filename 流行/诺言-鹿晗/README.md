

**诺言双手简谱** 和五线谱完全对应。

_诺言_ 是由朱婧汐填词，Santell、Djemba
Djemba谱曲，鹿晗演唱的抒情歌曲，于2015年11月2日首发，同年收录于鹿晗12月23日发行的专辑《RELOADED》中。

《诺言》是一首甜而不腻的情歌，R&B曲风十分清新。歌曲推崇爱，但反对挥霍青春的荷尔蒙，反对如快进一般的恋爱节奏，通过反现代快速规则的浅吟低唱，去解读爱在时间上的意义。以细碎清爽的节奏将相遇到触电的幸福感揉进旋律，像融化掉的糖浆浇在鹿晗的嗓音上，十分悦耳。

歌词下方是 _诺言钢琴谱_ ，大家可以免费下载学习。
