

**搁浅**
是周杰伦2004年发行的专辑《七里香》中的歌曲，是一首典型周氏情歌，延续了周杰伦一贯的风格。平庸而又深情的钢琴乐奏响起了，倍感自责的歌词清晰可闻。曲与词契合所谓是巧夺天工；曲调起伏不大，却越显忧郁，用阴暗压抑的天气衬托出整首歌的气氛无乎不完美，歌曲最后一段的高潮部分竭斯底里般撕心裂肺宣泄着自己地过错。再怎样自我谴责远方的你是否能感应的到，曾经的记忆还是无法抹去。
