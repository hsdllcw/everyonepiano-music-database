

**Dancing Queen**
是瑞典国宝级乐队ABBA发行于1976年8月16日（瑞典）的一首单曲，歌曲收录在乐队第四张录音室专辑《Arrival》和精选集《ABBA Gold:
Greatest Hits》（1992年）中。

  
歌词下方是 _Dancing Queen钢琴谱_ ，希望大家喜欢。
