

蔡琴的《你的眼神》就像一杯岁月的美酒，越是久远，越是香醇。她波澜不惊、低回委婉的歌声，是一种被遗忘了的古老语言，有着一种古典的浪漫，一种优雅的感伤。

### 你的眼神歌词：

象一阵细雨洒落我心底  
那感觉如此神秘  
我不禁抬起头看着你  
而你并不露痕迹  
虽然不言不语  
叫人难忘记  
那是你的眼神  
明亮又美丽  
啊～ 有情天地  
我满心欢喜
