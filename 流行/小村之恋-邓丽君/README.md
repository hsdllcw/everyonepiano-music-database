

**小村之恋**
是华人歌手邓丽君的演唱作品，最早收录于邓丽君1977年12月发行的国语专辑《岛国之情歌第四集—香港之恋》之中。该曲原曲为邓丽君1972年2月21日发表的日语单曲《ふるさとはどこですか》。

  
歌词下方是 _小村之恋钢琴谱_ ，希望大家喜欢。
