

《 **若花怜蝶** 》一听就是《天官赐福》花怜同人曲，由萧忆情和余夏一起合唱的。

余夏的声音非常好听，初听就一见钟情。当然萧忆情的声音也不错，给人暖暖的感觉。这首《 _若花怜蝶_
》也是够甜了，把花城和谢怜的感情都写的很深动。花城给过谢怜的每一个承诺从未落空，他用几百年的虔诚，完整了所有的许诺，他去爱去追求，他活下去的全部勇气都是心上金枝玉叶的贵人，这世间唯一的神明，深情唯花城二字可媲美。
