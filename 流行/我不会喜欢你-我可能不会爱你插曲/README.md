

**我不会喜欢你**
由徐誉庭作词，陈柏霖和王宏恩作曲，陈柏霖演唱，是电视剧《我可能不会爱你》中的插曲，收录在《我可能不会爱你电视原声带》中，于2011年10月18日发行。

  
2012年1月，该歌曲创下华人音乐圈颇具公信力的“KKBOX数位音乐风云榜”单日冠军纪录。

  
歌词下方是 _我不会喜欢你钢琴谱_ ，希望大家喜欢。
