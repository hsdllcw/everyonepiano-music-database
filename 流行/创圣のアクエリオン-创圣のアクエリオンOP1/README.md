

《 **创圣のアクエリオン** 》是日本动画《创圣のアクエリオン》（创圣大天使）片头曲。由AKINO演唱。

《创圣大天使》是由河森正治原作，由SATELIGHT于2005年出品的动画作品。TV动画全26集，现已完结。续作《创圣的大天使EVOL》已开始制作并于2012年起开始放送。

歌词下面是 _创圣のアクエリオン钢琴谱_ ，欢迎大家使用。
