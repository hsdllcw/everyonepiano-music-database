

**Beautiful** 是美国嘻哈及R&B歌手Akon所演唱的歌曲。阿肯（AKON），出生自非洲塞内加尔，
世界杯期间联手美国当红女歌手凯莉•希尔森（Keri Hilson）演唱足球金曲《加油非洲！（Oh
Africa！）》，成为2010年南非世界杯的主题曲。AKON还和迈克尔•杰克逊合唱了许多首歌曲，如"Hold My Hand"，"Wanna Be
Startin' Somethin"。

  
歌词下方是 _Beautiful钢琴谱_ ，希望大家喜欢。
