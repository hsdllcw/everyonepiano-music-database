

台湾爱情偶像电影《 **不能说的秘密** 》由周杰伦、黄秋生、桂纶镁主演。截止2007年底，《 _不能说的秘密_
》在亚洲票房接近960万美元。该剧荣获2007年 44届台湾电影金马奖、最佳原创电影歌曲、最佳视觉效果、年度台湾杰出电影等奖项。
