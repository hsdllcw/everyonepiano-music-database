

**风度双手简谱** 和五线谱完全对应。

_风度_ 是由汪苏泷填词、作曲并演唱的一首歌曲，收录在汪苏泷发行的专辑《万有引力》中。2012年12月3日，该歌获得百度沸点十大金曲奖。

此曲和声部分的女声也让人惊艳，通常汪苏泷都是一人搞定全部和声，而这首歌女声的现声，是《某人》之后的再次尝试，并且加大女声和声比例，让整首歌曲在之前的气质之上，有了更多新鲜的张力。整首编曲大气而不失细腻，Bridge部分的起伏强烈，让听者情绪随之活跃。副歌旋律唯美、动听。吉他弹奏哀而不伤，情感处理十分到位。
