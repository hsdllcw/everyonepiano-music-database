

**Bitter Sweet Symphony** 是The Verve的一首名曲，选自他们1997年09月30日发行的专辑《Urban
Hymns》。由The Verve主唱Richard Ashcroft改编自滚石乐队的《the last time》而成。

  
歌词下方是 _Bitter Sweet Symphony钢琴谱_ ，希望大家喜欢。
