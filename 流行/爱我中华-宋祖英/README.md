_爱我中华_
是为1991年召开的第四届中国少数民族运动会创作的会歌。谱曲时广泛采用广西、云南等地少数民族的音调。歌曲原是二段体。第一乐段曲调较活泼，节奏跳跃，音域较窄。

这首脍炙人口的歌曲多次在中央电视台的春晚中歌唱，还被编入初中生音乐教材，并名列31首被“嫦娥一号”搭载的歌曲之中。该歌曲歌词深刻，充满了爱国主义情怀，受全球华人所喜爱。
