_肆无忌惮_
由薛之谦作词并演唱、张洢豪作曲，二人首次合作挑战pop流行曲风。薛之谦一直在探索情歌的无数种可能，每一次的尝试都凭借真诚拨动听众最柔软的那根心弦。歌曲中的配器繁复而不杂乱，以忧郁的节奏融合钢琴、鼓、合成器、吉他、Bass、等乐器实录以及人声和音营造出极具感染力的哀伤氛围，仿佛一种全新的人声乐器融入了歌曲的编排，天衣无缝的让歌曲充满细节，情绪达到丝丝入扣。整体以蓝调曲风为基调，吉他的铺设让布鲁斯风味愈加浓郁。

值得玩味的是配器中不仅运用到了木吉他增添曲意悠扬，同样搭配电吉他让歌曲具备urban气质，更符合当代人心中爱情具象化的模样。整个歌曲的风格充满了摇曳而规整的冲突，薛之谦在这首歌曲中的vocal表现也十分生动，歌曲的演唱方面假音和转音的娴熟运用更是加重歌曲的伤感浓度。歌词走心而深刻，用最精简的话语刻画了爱情的本质模样。
