

延续上一首歌中关于“成熟”的话题，阿肆与徐佳莹合作的第二首先行曲《 **尽管如此，还是** 》，将目光聚焦于年过三十的晚熟人群。

在这个被认为本该安稳的年纪，他们却仿佛刚刚踏入“后青春期”，心中徒然升起推翻生活的激情，试图在梦想和现实的博弈中维持天平。直白的歌词在吉他恬静的旋律中娓娓道出一段内心独白。那些被大家认为是不合时宜的幻想、不明事理的抱怨在歌曲中得以宣泄，不张扬，也不矫饰。阿肆的声线干净自然，存留着些许少女的率真；而徐佳莹深情细腻的声音中，满是女性柔和的力量。她们像人群中失散的同伴，依靠微弱的雷达信号找到彼此，互相安慰——“理想”与“现实”的对垒是每个人生命中必须面对的课题。年少轻狂时，或许还能凭借一腔孤勇坚定地站在某条路上；可当步入三十而立的阶段，选择却因为无数的顾虑而变得困难起来。但正因为如此，才应该提醒自己：世上本没有标准化的人生，不要因为庸众的目光，为燃烧的赤子之心感到羞愧。就像毛姆笔下的高更，满地都是六便士，他却抬头看到了月亮。
