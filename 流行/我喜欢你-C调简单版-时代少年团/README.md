

《 **我喜欢你** 》由中国内地男子演唱组合时代少年团演唱，于2021年7月26日发布。

这是一首甜甜的小情歌，轻松欢快的小旋律搭配少年清亮干净的音色，给整个夏天带来了一丝甜甜的清凉！

歌曲中一句“ _我喜欢你_
”，包含了多少少年的小心思，我赠你玫瑰，你回我以爱意，少年的爱总是小心翼翼又横冲直撞，直到撞进你的心里，让你心怦怦，如小鹿般乱跳。少女藏不住的小心思，都被这横冲直撞撞得不知所措。于是少年和少女都脸红扑扑。所以，最美好的感情要用最勇敢的声音喊出来，这个夏天，就要大声说：我喜欢你！
