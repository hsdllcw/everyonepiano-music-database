_我的朋友_ 是TFBOYS于2019年7月12日发行的一首单曲。由外国作曲家Adam Nierow/Peter Habib/Makoto
Wakatabe创作，才女词人王韵韵填写中文词。曲风活泼可爱，充满能量。编曲采用大量电子音色，营造出热烈真挚的气氛。歌词描写了朋友之间轻松搞笑，一起拼搏的场景，十分生动亲切，画面感强烈。TFBOYS三个人元气十足的声音很好地诠释了朋友之间的真诚与羁绊。
