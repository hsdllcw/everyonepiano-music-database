

**手放开双手简谱** 和五线谱完全对应，属于EOP教学曲。

《 _手放开_ 》是台湾著名男歌手李圣杰2004年发行第三张专辑《绝对痴心·手放开》中的主打歌。

李圣杰，台湾男歌手，拥有四分之一德国血统，曾为台湾网球“国手”。早年曾用艺名李杰。擅长情歌，代表作《手放开》蝉联近三年KTV点播总排行榜前三名，《痴心绝对》亦为KTV点播排行榜冠军。而《远走高飞》这首歌是李圣杰透过一张发黄的老照片，逐渐了解外公的浪漫故事，也就借由这首歌献给曾经翱翔蓝天的外公外婆。2006年发行的《眼底星空》，及之后发行的《擦肩而过》，更是风靡全台。全新大碟将在2013年与广大歌迷见面，此专辑目前正在筹备当中。

《手放开》很多人都很熟悉，也非常喜欢。是一首旋律很美，很能打动人的曲子。下面是 _手放开钢琴谱_ ，感兴趣的朋友可以使用。
