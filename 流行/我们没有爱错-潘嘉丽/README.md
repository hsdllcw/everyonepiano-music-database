_我们没有爱错_ 是2020年初春，歌手潘嘉丽推出的最新单曲。歌词由新锐词手吴易纬操刀。

5分多钟的曲长，在钢琴与弦乐陪衬下，潘嘉丽转换不同的咬字与口气，在每一段落间，或疑问、或肯定、或倾述，唱出应该充满悲伤却带着相信爱的坚定力量 。

潘嘉丽希望借由《我们没有爱错》这首歌，让“I Love You”这句爱的魔法延续，爬梳聆听者的记忆，让过往的美好现身。
