《 _小野猫_
》最初发行于2016年，一经上线便引爆乐坛，2017年更是风靡整个YY，深受各大主播及众粉丝喜爱，刷屏的都是《小野猫》。而现今，此曲再次走红。

这首歌曲听起来相当活泼，有爱情感，性感十足，但同时又表明了对爱情专一承诺。
