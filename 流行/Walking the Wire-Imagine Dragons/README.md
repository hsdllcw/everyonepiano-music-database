Dragons的歌曲。收录在其第三张录音室专辑Evolve中，发行于2017年6月23日。乐队由主唱丹•雷诺斯（Dan
Reynolds）、贝斯手本•麦基（Ben McKee）、吉他手韦恩•瑟蒙（Wayne Sermon）和鼓手丹尼尔•普拉兹曼（Daniel
Platzman）组成。
