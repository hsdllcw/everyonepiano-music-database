

**我想有个家** 是潘美辰演唱的一首歌曲，由潘美辰作词、作曲，收录在潘美辰1989年发行的专辑《是你》中。《 _我想有个家_
》是潘美辰的代表作品之一。1990年这首歌曲获得了第一届台湾金曲奖年度最佳歌曲奖。这首歌曲以动听的旋律，具直白且写实性的歌词而备受大众关注，潘美辰用独特的嗓音深情演绎了这首歌曲，唱出了对爱的渴望。

  
歌词下方是 _我想有个家钢琴谱_ ，希望大家喜欢。
