

**那些花儿**
是中国台湾流行音乐女歌手范玮琪演唱的歌曲，收录在其发行的专辑《真善美》中。范玮琪用她那抚慰人心的范式唱腔，唱出了一种坚强和柔软融合的质感。2004年该歌曲的获得Music
IN 中国TOP排行颁奖晚会“港台年度金曲”奖。

最初，此曲是中国内地男歌手朴树的歌曲，朴树亲自作词作曲，收录在其1999年发行的首张个人专辑《我去2000年》中，同时也是2002年电影《那时花开》的片尾曲。

《 _那些花儿_ 》是首忧伤而又温暖的作品，朴树和范玮琪二人都把这首歌当作自己的“保留曲目”。

歌词下方是 _那些花儿钢琴谱_ ，希望大家喜欢。
