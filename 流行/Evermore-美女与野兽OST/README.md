_Evermore_ 是美国华特•迪士尼影片公司出品的2017奇幻片《美女与野兽》的插曲。由Alan Menken创作。

电影根据迪士尼1991年同名动画电影改编，讲述了少女贝儿为解救父亲而与受诅咒化为野兽的王子朝夕相处，最终用真爱战胜魔法的故事。该片于2017年3月17日在美国、中国同步上映。
