_趁早_
是张宇作曲，十一郎作词，中国台湾歌手张宇演唱的歌曲。收录在张宇2005年3月15日发行的第十五张专辑，也是第二张精选辑《男人的好》中。原曲由张惠妹演唱，收录在张惠妹2000年的个人第七张音乐专辑《不顾一切》中。

张惠妹版的《趁早》唱出女人的内敛沧桑，但这次张宇版的《趁早》则将男性的情感宣泄表现的淋淋尽致！绝对又是另一首男性乐迷们上KTV的必K之作！
