

**相爱后动物感伤** 是由陈镇川作词，小安作曲，张惠妹演唱的歌曲。该曲收录于张惠妹2009年发行的专辑《阿密特》中。

《 _相爱后动物感伤_
》以一种含蓄的方式，表达了在现代社会中，有太多太多的人，只是因为一时的寂寞或无聊，在明知不爱对方情况下，与其交往，甚至发生了关系，但，当再度遇见了另一个更加喜欢的对象时，便极欲摆脱原本交往的人，而在不愿或无法离去的情况下，便只能做出越轨的行为，使双方难堪。工商发达的今天，有多少人藉由偷情，逃避压力与独自工作的孤独？有多少人藉由这种伤害自己、伤害别人的方式，寻求一时的快感？这或许解决了一时的空虚，却永远不可能，真正的解决问题。
