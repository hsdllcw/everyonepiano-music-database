_热带雨林_
是方文山填词，周杰伦谱曲，S.H.E演唱的伤感抒情歌。收录于2002年1月29日发行的专辑《青春株式会社》中，该曲是专辑的主打歌之一。2003年1月18日，该曲获得Hito流行音乐奖-
十大华语歌曲奖。

歌曲采用柔美、温和的旋律，让人容易回忆起往事，采用伤感而深入人心的歌词，易引起听众的共鸣。歌曲通过三人的完美和声，表达出了情人之间感情受困如置身梦境，迷失在热带雨林的感觉。
