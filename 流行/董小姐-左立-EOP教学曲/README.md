

《 **董小姐** 》是民谣歌手宋冬野的原创歌曲，收录在2012年12月发行的音乐专辑《摩登天空7》中。

宋冬野，中国民谣歌手，音乐创作人。他以清新的弹唱风格，木质般的嗓音，尖锐地反映社会现象的原创歌词，温暖并强烈的新民谣曲风获得许多文艺青年的青睐。
2012年签约摩登天空。代表作品：《莉莉安》、《安和桥》、《年年》、《嘿，裤衩儿》、《抓住那个胖子》、《佛祖在一号线》、《 _董小姐_ 》等。

2013年6月29日晚，湖南卫视2013《快乐男声》首播，而一位抱着吉他上阵的唱吧歌手左立成为了全场焦点，来自凤凰古城的他为女友深情演绎了一首《董小姐》沁人心脾、令人沉醉、催人潸然，感动了全场。左立轻轻哼唱着：“董小姐，你从没忘记你的微笑，就算你和我一样，渴望着衰老……”他的歌声甚至让评审陶晶莹想起了美妙的初恋，让她感动万分。值得一提的是，随着节目的播出《董小姐》一曲立马上升到微博话题榜的前几位，有不少网友被左立的歌声感染，而这首歌的原创作者宋冬野更是在微博送出了对左立祝福。最终，左立凭借其歌曲成功晋级长沙唱区10强。

《董小姐》是一首很淳朴，让人感动的歌。其中有一句歌词“爱上一匹野马，可我的家里没有草原”让很多朋友印象深刻。歌词下方是 _董小姐钢琴谱_ ，希望大家喜欢～
