

**天龙八部之宿敌**
，简称宿敌，是2011年许嵩成为了网络游戏《天龙八部3》的年度代言人后，受搜狐畅游邀请，为该游戏创作的中国风歌曲，创作构思源于金庸先生的《天龙八部》。

许嵩此次创作依旧词曲创作一手包办，再一次凭借其个人中国风词曲特质而为华语乐坛此类别开了另一扇门，令整首歌许式中国风特色完整而又细节毕现，使《
_天龙八部之宿敌_
》古韵弥漫。而编曲老师蔡庭贵在编曲的把握上，则让古今乐器糅合一体，却又集大气而柔婉的演绎，如余音绕梁者欲三月不绝。而歌词部分，虽为代言歌，但内容上却并不拘泥于天龙八部原著。台湾混音师林哲民驾驭最后一关，使得整首歌音乐形态层次感一一带出，仿佛真如歌曲里讲述的三月春花渐次醒。
