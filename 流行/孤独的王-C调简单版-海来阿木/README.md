

中国内地最具实力音乐唱作人海来阿木，歌曲《 **孤独的王**
》一经推出便好评不断，歌中听海来阿木沉静的坦露心声，就像是一个知己，告诉你他的经历，和你道出他的心事。

越是强大的人，越是有自己软弱的一面，海来阿木亦是如此。但无论有多艰难，他依然微笑面对，勇往直前，最终用实力登顶歌坛。

其实《 _孤独的王_
》是海来阿木唱给自己的一首歌，“你笑我看不破镜花水月，我叹你不懂我情深义重，奈何人间以千壶掩寂寞”，海来阿木用歌声唱出了独特的戏韵，致敬自己的过往！
