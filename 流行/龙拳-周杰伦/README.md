

《 **龙拳**
》出自周杰伦的专辑《八度空间》。该曲在填词上紧扣中华文明的地域特色，并以龙拳的开天辟地暗喻了中华民族的日益崛起。其次，《龙拳》的曲风为国际上流行的HIPHOP风格，容易引起共鸣。

该曲与《双截棍》、《刀马旦》等歌曲一脉相承，充满浓厚的中国味道
。听众听这首歌的时候画面会飞跃至充满浓厚中国味的“龙拳”，周杰伦仿佛化身为东方之龙，时而跳跃黄河两岸，时而登上泰山顶峰。整首歌“宫廷式鼓点”强力敲打着，周杰伦用中国乐器玩出了摇滚。
