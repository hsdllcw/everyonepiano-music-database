_Sorry_
，既可以理解为“道歉”，也可以理解为“遗憾”。教主的这首歌有两层表达意思。其一，是对去年自己的“抱歉”，对粉丝的“抱歉”。想以这首歌告诉我们今年他一定会更好！其二，是教主回看一路坎坷的2020，为全人类而感到遗憾。毕竟今年全球70亿人都不容易。但更多的应该不仅仅只是“遗憾”。教主也许是想借此，鼓励各位放下去年的一切，一路向前，砥砺前行！
