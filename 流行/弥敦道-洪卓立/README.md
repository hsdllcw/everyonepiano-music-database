《 _弥敦道_ 》是香港歌手洪卓立演唱的歌曲，收录于洪卓立《Go》专辑中，由颉臣作词，谢杰作曲。弥敦道（英语：Nathan
Road）位于香港九龙，连接旺角与尖沙咀这两个主要商业区，是香港最著名的街道之一。

洪卓立，香港歌手。出生于香港，2006年参加「英皇新秀歌唱大赛」获得亚军从而晋身乐坛。代表作有《爱·无胆》、《直间恤衫》、《弥敦道》等。
