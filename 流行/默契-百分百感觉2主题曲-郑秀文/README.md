

**默契双手简谱** 和五线谱完全对应。

_  
默契_ 是华语乐坛天后级女歌手郑秀文演唱的一首歌曲。也是电影《百分百感觉2》的主题曲。郑秀文（英文：Cheng Sau Man、 Sammi
Cheng，1972年8月19日－），香港出生，籍贯广东汕头市，母亲为印尼华侨。郑秀文在乐坛、影坛均具有举足轻重的影响力，其世界巡回演唱会总场次更突破150场、全球的唱片总销量超过2,500万张，并多次引领流行，拥有“百变天后”之美誉。

  
歌词下方是 _默契钢琴谱_ ，希望大家喜欢。
