

**突然** 是日本乐队组合FIELD OF
VIEW单曲及同名歌曲，于1995年7月24日发行。该专辑共收录了《突然》，《きっと离れていても》，《突然》（原装卡拉OK版）三首歌曲。其中，专辑中的同名歌曲《突然》也被选作宝矿力水特1995年的广告歌在日本多家媒体播放，并被Zard、织田哲郎等歌手翻唱。由中国台湾歌手任贤齐演唱的《永不退缩》亦是根据这首歌曲改编。

  
歌词下方是 _突然钢琴谱_ ，希望大家喜欢。
