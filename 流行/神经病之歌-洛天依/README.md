_神经病之歌_
是ilem于2014年12月30日于bilibili发布的Vocaloid原创合唱作品，本曲拥有着精神病至极的歌词，循环洗脑的旋律和令人惊异的中毒性。

此歌由UP主ilem首先上传在B站的Vocaloid&UTAU区，使用Yamaha公司以VOCALOID
3语音合成引擎为基础制作的中文Vocaloid声源洛天依和言和演唱[1] （后半段有CRYPTON FUTURE
MEDIA以Yamaha的VOCALOID系列语音合成程序为基础开发的日文声源初音未来和KAITO客串，以及乱入了金馆长的笑声），很快在对应区引起较大的效应。随后，B站UP主Mr.Lemon将其鬼畜了元首和葛平，一时在鬼畜区引发轰动。随后，B站大名鼎鼎的“四大欠王“之一A路人和翻唱红人KBshinya将其翻唱，彻底带动其翻唱的狂潮。
