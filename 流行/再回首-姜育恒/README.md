歌曲《 _再回首_
》由台湾女歌手苏芮演唱，是80年代末期，90年代初期脍炙人口的一首歌曲，曾一时风靡大街小巷，传唱率颇高，被李翊君，姜育恒等众多歌手接连翻唱，其中以姜育恒的版本最红。  
  
《 _再回首_
》的词曲作者分别是陈乐融和卢冠廷。《一切为明天》是1988年10月推出，没有收录该曲。大陆1989年引进这张专辑时，抽掉了3首当时认为有问题的歌曲，换上了3首歌曲，《再回首》便是加入的一首。洪金宝的电影《群龙戏凤》主题曲是苏芮首唱的获奖歌曲《凭着爱》，为了在台湾上映，苏芮又演唱了同曲不同词的国语歌《再回首》，而当时苏芮和飞碟唱片的合约将满，没有发片计划，所以唱片公司不是很重视这首歌，只收录在一张合辑中，1989年底又收录在苏芮的精选专辑《经典》。
