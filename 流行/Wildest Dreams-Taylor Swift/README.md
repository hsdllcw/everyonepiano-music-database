

**Wildest Dreams** 是美国女歌手Taylor
Swift演唱的歌曲，收录于第五张录音室专辑《1989》。2014年10月23日，Taylor
Swift在社交网站Instagram曝光新专辑《1989》标准版完整曲目。

  
歌词下方是 _Wildest Dreams钢琴谱_ ，希望大家喜欢。
