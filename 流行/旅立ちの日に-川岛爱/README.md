

**旅立ちの日に双手简谱** 和五线谱完全对应。

_旅立ちの日に_
，中文译名启程的日子，是目前最流行的日本中学生毕业歌，在日本广为流传。许多日本歌星都唱过这首歌，同名歌曲有多首，其他较为著名的是川岛爱（川嶋あい）的版本。该歌曲的国语版为《分别还有重逢时》。

歌词下方是 _旅立ちの日に钢琴谱_ ，大家可以免费下载学习。
