

**第几个100天**
是林俊杰演唱的歌曲，由姚若龙作词，林俊杰作曲，首发于2009年12月2日，收录在林俊杰2009年12月18日推出的专辑《100天》中。

是一首十分上口、感情十分真挚动人的作品。林俊杰的曲子保持了一贯的优质水准，姚若龙的填词也不过亦不失，但在编曲上太过注重于技巧的着墨，多少落了些窠臼。《
_第几个100天_
》开曲钢琴铺陈做引，弦乐加持的大气似回到了从前，多少给听觉带来了一丝沉重、少了一丝惊艳，而长达半分多钟的前奏减缓了整体节奏的明朗，伴着弦乐，林俊杰的声音略微经过后期处理，没有一丝瑕疵可言，也正是如此，这首歌几乎感觉不到任何现场感，弦乐的加持让整体的节奏变缓，紧凑的情感的冲击力反而被削弱了。
