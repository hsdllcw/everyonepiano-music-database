

抖音“梦断不成归，几分憔悴，锦瑟无端生悔，赢得满行泪”是张晓棠演唱的《 **苏幕遮** 》。金志文的作曲保持一贯的分数，风格很突出，女声在戏曲部分表现出色。

词的那一段，比平时听的戏腔，给人一种更为灵活的感觉，女子唱起戏腔起来要比男子，可能会多一份尖锐，但却不刺耳，这比一般的戏腔，少一分古板，多一份柔情，反而让人感觉很舒服，唱得字字入心，环环相扣，成功的锁住了大家的耳朵。
