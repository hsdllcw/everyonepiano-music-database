《 _自我催眠_
》收录在罗志祥2005年专辑《催眠SHOW》中。抒情的R&B旋律，一贯的Jay曲风，钢琴配上吉他轻轻的扫弦声，空气中都弥漫着哀伤的味道。罗志祥说，原本这张专辑打算向周杰伦邀歌的，但是由于父亲去世而耽搁了。没想到有一天杰伦突然打电话给罗志祥，只酷酷地吐出两句话来：“我帮你写了一首歌，你听听看。”原来杰伦从报道中看到罗志祥强忍悲痛，忙于后事，深深为罗志祥的坚强所感动，于是便特别为罗志祥谱了一首曲子。
