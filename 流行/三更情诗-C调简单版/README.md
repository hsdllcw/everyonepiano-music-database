

**三更情诗**
是由周思涵演唱的一首歌曲。这是首描绘初陷恋情时，将他/她思念到凌晨，依旧坐在桌前为他/她书写情诗的纯情古风恋曲，歌手周思涵抒情而婉转的声线，将一段青春韶华的相思趣事演绎得惟妙惟肖。

人间四月芬芳尽，山寺桃花始盛开，青春里最难忘的便是那一封封彻夜写下的情诗。听阿涵温柔的嗓音，为你抚慰青春韶华里悸动的意难平。
