

**双手简谱和五线谱完全一致，是EOP教学曲。《** 别说我的眼泪你无所谓》是东来东往的成名曲。

东来东往，中国内地男歌手。2003年凭借一首原创歌曲《别说我的眼泪你无所谓》迅速窜红，歌曲网络搜索率达千万，点击下载率逾千万，占据中国百度金曲排行前五强，荣膺2003年度网络最受欢迎男歌手，是近年来内地流行音乐最具影响力的歌手之一。
