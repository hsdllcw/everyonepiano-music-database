

《 **最真的梦**
》是台湾歌手周华健1989年演唱的一首歌曲（代表作），由陈桂珠作词、罗大佑作曲，洪艾伦、罗大佑编曲。收录于1989年11月10日发行的同名专辑《最真的梦》中。

作为专辑的主打歌，此曲旋律优美动人，朗朗上口，是一首非常经典的老歌，其粤语版为香港女歌手陈慧娴所唱的《人生何处不相逢》。
