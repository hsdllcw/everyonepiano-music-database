

刘珂矣，一位禅意古风歌手，每一首歌都是精心打造，极致力求完美。而近期新发布的歌曲《 **闲庭絮** 》更是美到不可方物。

歌曲灵感出自清代文学家沈复《浮生六记》。古语说：“身放闲处，心在静中”。

刘珂矣的歌声有种抚平情绪的神奇力量，淡雅悠然。从之前的半壶纱到如今的 _闲庭絮_
，即使心情再烦躁，听几遍就会忘却烦恼，仿佛手捧一盏清茶看云卷云舒，日升日落时光斗转都不曾觉得。

所以，在难得的休息时光，不妨放下手机，停止玩游戏，好好的陪伴家人、朋友。或是在一壶茶、一本书的时光里，聆听一首《闲庭絮》，好好的陪伴自己的心灵吧。
