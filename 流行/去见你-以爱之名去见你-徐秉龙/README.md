_去见你_
是音乐人徐秉龙的全新单曲。此曲依旧是徐秉龙包揽词曲创作，讲述的是歌者与听众之间微妙的特殊关系，除了以歌迷视角表达感情，也听见了真诚的表演者如徐秉龙最内心的告白。

整首歌行进犹如浪漫情景剧，清新美好，层次分明。看着像是以一位歌迷的口吻揭开序幕，但其实通篇更像是对朋友间的对话。歌曲结尾又回到首段，秉龙一如恋人絮语般的语气，感觉像是“我们”经历的一幕幕生命转场。再见又分开，分开是为了下一次更好的再见。
