

《 **路边的野花不要采**
》传唱至今，歌曲于1972年由马来西亚音乐人李俊雄写给邓丽君，因其曲调轻松俏皮，歌词浅显易懂，配合邓丽君可爱动人的演唱，一经推出很快成为70年代风靡东南亚和全球华人社会的流行歌曲。

歌曲开头描述了一位妻子在殷殷嘱咐即将远行的丈夫“路边的野花不要采”，实则告诫丈夫不要被外面的诱惑所干扰。紧接着，“记着有我天天在等待，我在等着你回来”语言很自然流畅，因而也就更显得真实可信，仿佛就是我们的身边事。

《 _路边的野花你不要采_ 》，喜庆热闹中带着一丝诙谐，叮嘱出门在外要经得住诱惑。真是千古送别，人同此心啊！
