

《 **长安忆** 》是音频怪物演唱的一首非常经典的古风歌曲，后双笙翻唱。原曲是《かざぐるま（风车）》。此歌曲化用了许多文人雅士的诗词曲。

这首歌曲将长安的历史文化和人文美景都展现出来。老妖其实很适合演绎这首历史文化底蕴深厚的歌曲，他的声线厚重低沉，歌声也非常大气，他演唱的这首《长安忆》就能够让我们想起很多有关长安的故事。
