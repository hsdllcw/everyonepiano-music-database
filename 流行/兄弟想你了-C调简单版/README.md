

近段时间以来，在抖音上面越南版的《 **兄弟想你了**
》十分的火，不少网友都纷纷表示被这个魔性的音乐已经深深的洗脑了，听到停不下来。中文版《兄弟想你了》由姜鹏演唱，那越南版是谁唱的呢？

据了解，这首越南版的《 _兄弟想你了_ 》越南名字叫做《Huynh （SON2M
Remix）》，是一位来自越南的歌手翻唱的，之后在抖音上面被一位网名叫做“三斤哥流浪在越南”的抖友分享了出来。

这首歌目前在短视频平台上面特别的火，很多网友都在用这首歌当做背景音乐，这首歌表达了对兄弟们、对家乡的思念之情，也与漂泊他乡的友人产生了共鸣。
