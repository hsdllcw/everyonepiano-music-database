

**借** 是明日之子最强厂牌毛不易在明日之子第11期中演唱的一首原创歌曲，是毛不易歌曲中比较成熟的一首。该歌曲后收录在毛不易的音乐现场专辑《巨星不易工作室
No.1》中。

凭借这首原创歌曲《 _借_
》毛不易再次让所有人为之惊艳。一向对毛不易要求苛刻的华晨宇不仅对毛不易表示盛赞，而《借》的歌词更是让薛之谦感叹到：“你是我见过的年轻一辈中，写词最有灵气的人。
