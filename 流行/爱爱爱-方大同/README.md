

**爱爱爱双手简谱** 和五线谱完全对应。

_爱爱爱_ 收录在方大同2006年12月31日发行的同名专辑爱爱爱中。

2007年，该专辑获得新城国语力颁奖礼“新城国语力优秀专辑”奖项，同名歌曲《爱爱爱》获得新城国语力颁奖礼“新城国语力歌曲”奖项。

这是一首反情歌的作品，词人周耀辉借着一种书写“天花乱坠”的笔触，对该时代只有一种告白，对“不明不白不分好歹，都为了爱爱爱”重演慨叹。

歌词下方是 _爱爱爱钢琴谱_ ，大家可以免费下载学习。
