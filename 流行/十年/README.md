

《 **十年**
》是中国香港男歌手陈奕迅演唱的一首歌曲，由林夕作词，陈小霞谱曲，改编自陈奕迅演唱的粤语歌曲《明年今日》。该歌曲是陈奕迅情歌代表作之一，收录于陈奕迅2003年5月发行的专辑《黑白灰》中，同时还是2005年播出的电视剧《女才男貌》的片头曲。
