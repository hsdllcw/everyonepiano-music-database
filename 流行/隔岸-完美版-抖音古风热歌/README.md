

最近在抖音刷到的很多视频都是以“你呀你冻我心房酸我眼眶一生的伤”这首歌作为背景音乐的，歌曲十分的忧伤。据悉，此曲叫做《 **隔岸**
》，是由姚六一演唱的一首古风歌曲，天作之合两岸隔，彼岸灯火徒观望。

这首歌十分适合古风爱情故事，悠扬深情的歌声一步一步带你沉浸其中，歌词也写的极好，写出了喜欢一个人的深情，每一字每一句唱的都是你。一首深情古风爱情曲，初次相遇便知道是愿意一生追寻的人。而在这世间，爱而不得常让人最难忘怀，有多少心酸，有多少无奈，只剩最后在隔对岸等你。
