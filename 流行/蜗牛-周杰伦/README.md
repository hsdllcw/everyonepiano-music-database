

周杰伦（Jay
Chou）祖籍福建泉州。华语杰出音乐人、亚洲流行天王、词曲创作家、编曲及制作人、MV及电影导演、编剧，2000年后亚洲流行乐坛最具改革性创作歌手，唱片亚洲总销量超过3100万张，其音乐打破亚洲原有的音乐主题形式，开创了多元化音乐创作和
“现代流行乐中国风” 的先河，为亚洲流行乐坛翻开了新的一页。
