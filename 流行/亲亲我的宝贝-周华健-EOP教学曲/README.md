_亲亲我的宝贝_ 是周华健演唱的一首歌曲，由周华健作词、作曲，收录在周华健1990年发行的专辑《不愿一个人》以及2003年发行的专辑《KEEP WAKIN
1987-2002 周而复始》中。  
  
歌声棒的周华健其实词曲创作功力也不差，他在1989年为刚诞生的儿子写下的《亲亲我的宝贝》，到现在小孩都已经满22岁了，却还是让人记忆犹新。这首歌创作的背后，其实有许多妙事。  
  
《亲亲我的宝贝》后来广为传唱，成为很多爸爸妈妈可以唱给小孩听，表达父爱母爱的歌曲。周华健说，“导致后来女儿出生的时候，有很大的压力，觉得必须要再去写一首女儿歌，不然女儿就觉得很不公平，长大以后‘他(哥哥)有歌，我(妹妹)没有歌啊’！”
