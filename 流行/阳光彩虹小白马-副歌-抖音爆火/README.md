

**阳光彩虹小白马**
是大张伟最新专辑《人间精品》中的一首单曲，欢快的旋律搭配暖心的歌词颇受好评。大张伟用自己燥而不乱的朋克方式阐述暖心积极的生活态度：任何时候请相信“一切都会好哒”。而这首新歌的MV也于4月24日正式发布，以手指舞搭配轻松欢快的歌曲氛围，一展大张伟鬼马本性，点燃这个春天的最后一把燥火！

据悉，《 _阳光彩虹小白马_
》的创作灵感来自大张伟在《跟着贝尔去旅行》中的一次录制，当时艰苦的录像环境和内容让大张伟濒临崩溃，而就在这样的境况下他能想到最美好的事物居然就是“阳光彩虹小白马”。除此之外，高压的工作环境时常让大张伟感到焦虑和无助，后来他看到一个电影的截屏，深受触动。上面说道：“心很脆弱，需要你哄它，要轻拍着它，对它说，一切都会好的。”从此以后，大张伟便时常在睡觉前，拍打着心口平静自己，对它说，一切会好的。于是歌词就来了，面对一切的郁闷痛楚焦躁，要记得！你是那个阳光彩虹小白马，一切会好哒！

或许正是因为这些最真实，最真切的感受，才让歌曲一经发布，便引发了广泛共鸣。很多网友听后都纷纷表示：“太治愈系了！被歌曲中的快乐力量感染了。”“这首歌让我想到了自己，这么欢快的歌曲竟然听哭了。”《阳光彩虹小白马》要向大家传递的正是这份快乐和正能量，这也是大张伟神奇的音乐魔法，正是这样走心的创作，让所有的听众从俏皮的歌曲中，感受到实实在在的温暖。在社会焦虑普通存在的当下，生活或许不易，世界或许残酷，我们或许无助，但仍要告诉自己“你是最强哒最棒哒最亮哒最发光哒”！
