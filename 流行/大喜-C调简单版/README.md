

原创国风电音单曲《 **大喜** 》由假寐作词，殇小谨作曲，于瑾编曲，泠鸢yousa演唱，殇小谨制作，于2021年3月5日发行。

MV由知名UP主楽小漫导演，虚拟偶像超高品质演唱会【创世之音】制作团队制作，可以说是虚实结合的现代化MV。

泠鸢是一位来自二次元的虚拟歌姬，以极具辨识度的特殊声线，灵活游走于多种曲风。填词风格充满故事性，具有很高的艺术价值。作曲同样颇具特色。
