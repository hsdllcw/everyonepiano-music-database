

**Always** 由Bon Jovi乐队的主唱Jon Bon Jovi创作，原本为电影Romeo Is
Bleeding创作，由于此电影观众评价较差，于是Jon把这首歌扔到了书架上。后来一位朋友发现了这首歌，让他发行至乐队的精选集《Cross
Road》中，结果十分成功，成为了Bon Jovi最畅销的单曲。

  
邦乔维（Bon Jovi），是当代美国摇滚乐队，少年时期即开始接触摇滚乐。1983年组建同名乐队，逐渐占据市场。以主流硬摇滚、金属摇滚见长。
