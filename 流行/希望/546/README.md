_希望_
是香港女歌手、演员陈慧琳所演唱的一首歌曲，同时也是2005年李英爱主演的青春励志传奇韩剧《大长今》香港版本的主题曲。韩国原版大主题曲名为《呼唤(ONARA)》由任世现作词兼作曲，金智贤、白宝贤及金露姬韩语主唱。  
  
此曲旋律悠扬，每次听到都有一种，空灵轻松的感觉。是小编的大爱之歌。  
  
歌词下方是 _希望钢琴谱_ ，希望大家喜欢。
