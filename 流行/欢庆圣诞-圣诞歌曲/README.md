

**圣诞节**
(Christmas)，这个名称是“基督弥撒”的缩写。弥撒是教会的一种礼拜仪式。圣诞节是一个宗教节。因为把它当作耶稣的诞辰来庆祝，因而又名耶诞节。这一天，世界所有的基督教会都举行特别的礼拜仪式。每年12月25日，是基督徒庆祝耶稣基督诞生的庆祝日，在圣诞节，大部分的天主教教堂都会先在12月24日的耶诞夜，亦即12月25日凌晨举行子夜弥撒，而一些基督教会则会举行报佳音，然后在12月25日庆祝圣诞节；而基督教的另一大分支——东正教的
_圣诞节_ 庆祝则在每年的1月7日。
