_草木_
是魔道祖师薛晓同人曲，由古风歌手若以止白演唱。《魔道祖师》是一部由墨香铜臭所著的原创耽美玄幻小说，主线道侣携手打怪解谜。2015年10月31日于晋江文学城连载，初版于2016年3月1日完结（正文），至2016年9月7日修文完成（正文+外三篇），网络版正式完结。后被改编为网络动画和广播剧。
