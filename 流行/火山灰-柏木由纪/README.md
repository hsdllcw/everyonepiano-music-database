

**火山灰** 是日本女子偶像团体AKB48 Team B及NMB48 Team N成员柏木由纪的歌曲。柏木由纪（Kashiwagi
Yuki），1991年7月15日出生于日本鹿儿岛县所属演艺经纪公司为Biscuit Entertainment。

  
歌词下方是 _火山灰钢琴谱_ ，希望大家喜欢。
