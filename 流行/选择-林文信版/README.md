_选择_ 是林文信12小时学好流行键盘教程基础部分的第11课第1首歌。

此曲原是由林子祥、叶倩文演唱，发行时间为1992年，华纳唱片发行，收录在林子祥同年发行的广东专辑《祈望》中，专辑一共12首歌，而《选择》是其中唯一的国语歌，也是唯一的合唱歌曲。且这首歌曲至今仍然有很高的传唱度，是华语音乐中最具影响力的对唱情歌之一。
