

**Eight双手简谱** 和五线谱完全对应。

_Eight_
是李知恩IU与防弹少年团闵玧其SUGA的首次神仙合作。展现不同于以往的独特魅力，声音极具辨识度，美妙的旋律搭配着醉酒低嗓，听觉享受极其震撼。IU，1993年5月16日出生于韩国首尔特别市，韩国女歌手、演员、主持人。Suga，闵玧其，1993年3月9日出生于韩国大邱广域市，韩国流行乐男歌手、男子演唱组合防弹少年团成员。

歌词下方是 _Eight钢琴谱_ ，大家可以免费下载学习。
