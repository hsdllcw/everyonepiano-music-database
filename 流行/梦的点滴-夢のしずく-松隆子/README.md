

《 **梦的点滴** 》是歌手松隆子(日本）1999年9月22日发布的第八只单曲。收录在专辑《爱恋的日子》之中。

松隆子，日本歌手、艺人。出生于东京都，原名为“藤间隆子”。所从事的“松本幸华”为她在舞台剧上的艺名（舞俑名），是Papado,
Inc.旗下艺人，及隶属于日本BMG唱片公司。2007年12月28日和知名音乐人佐桥佳幸闪电结婚。

歌词的下方还提供了 _梦的点滴钢琴谱_ ，欢迎感兴趣的朋友下载使用。
