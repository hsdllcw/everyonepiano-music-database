最近在抖音上特别火的一首背景音乐。据悉，此曲叫做《 **活该** 》，唱出了多少人的心声！

拼了命去爱，换一句活该，只能躲在无人的地方哭，怕爱我的人心痛，怕不爱我的人笑话。

在爱情中，双方是平等的。恒久的爱，是需要用一生来供养的！它需要用真心来呵护！用真情来打理！虽然，爱，总是让你欢喜让你忧，爱，总是让你牵挂让你愁。虽然，爱，有时也会让你心痛不已，幽恨迷离，痛彻心扉！
