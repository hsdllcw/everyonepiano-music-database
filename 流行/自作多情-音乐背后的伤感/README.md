

**自作多情双手简谱** 和五线谱完全对应。

_自作多情_ 是本兮演艺的歌曲，收录在2012年06月30日音乐专辑《Run Away》。本兮的歌曲大多都是说唱RAP和凄凉悲伤的音乐，
声音很特别，很适合唱可爱型歌曲，但喜欢凄凉、悲伤的音乐风格。本兮作为一名原创歌手，每首歌都是以自己真实经历结合自身感受创作，写出了她不同时段不同的情感体验。毕竟艺术来源于生活，又高于生活。

这首歌节奏轻快，听的过程给人一种激昂愉快的感受，让人充满遐想，但又夹杂着伤感的情绪，把我们带入一种又美又凄凉的意境中，让人不自觉的沉溺于这种舒适安静的旋律细细品味。
