

**陷阱** 是抖音红人王北车的原创歌曲，1995年出生的抖音红人！因一首《陷阱》在抖音上火爆走红。95年的小鲜肉啊！  
  
“安排”这个梗最近火遍全网，前段时间抖音上火了一个刘宇宁，最近又火了王北车，王北车安排也成了流行语，其实“安排”这个词来自于一个句式：你们有你们的摩登兄弟，我们有我们的王北车，王北车出面安排一下吧。王北车安排就是粉丝们要求王北车出来维护一下粉丝们的地位。  
  
歌词下方是 _陷阱钢琴谱_ ，大家可以免费下载学习。
