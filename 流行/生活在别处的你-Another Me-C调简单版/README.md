

毛不易和中岛美嘉梦幻联动，共同推出的新歌《 **生活在别处的你** 》（Another
Me）于2021年7月13日上线。温情旋律中，讲述着不同地域的平凡人物相同的坚持与期冀。

歌名取“ _生活在别处的你_ ”，歌词中也唱着“
一日三餐，奔波忧烦”“你咽下命运的为难，其实已足够勇敢”等日常生活场景与心情，两位歌手的呼应接唱，似乎在诉说着“生活在别处的你，像极了另一个我”。

可以说，此曲一如既往地简单朴实，直击人心。毛不易和中岛美嘉此次破次元的合作，不仅给歌友们带来了惊喜，还给大家带来了好听的治愈系歌曲。听过这首歌后，能让人缓解所有的焦虑，同时可以抚平心灵上的创伤。
