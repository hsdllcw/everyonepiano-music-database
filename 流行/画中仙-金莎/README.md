

**画中仙双手简谱** 和五线谱完全对应。

_画中仙_ 是中国内地女歌手金莎演唱歌曲。金莎（kym），籍贯上海，中国大陆流行人气女歌手和当红女艺人，被誉为“大陆甜歌玉女小天后”。
气质高雅，清纯，金莎的嗓音甜美灵动，清新脱俗，获得众多奖项。

歌词下方是 _画中仙钢琴谱_ ，大家可以免费下载学习。
