

Jolin首次尝试七八O年代甜蜜复古代表作，犹如洋果子般的香甜爱恋，送你日不落的想念，上面画满色彩鲜明的油画，将爱的明信片轻快送达！继《马德里不思议》之后又一年度甜蜜主打歌！

本歌是翻唱《sunshine in the rain》 是瑞士的BWO(BODIES WITHOUT ORGANS) 空壳乐队的乐曲。
