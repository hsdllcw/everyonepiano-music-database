

**听海双手简谱** 和五线谱完全对应。

_  
听海_ 收录于1997年6月张惠妹发行的第二张个人全新专辑《BAD
BOY》，由著名音乐人林秋离、涂惠元继续为阿妹量身打造又一抒情力作。这首歌排在曲目的第五首，发行最初并非是作为主打歌，但因为受到歌迷出乎意料的热烈欢迎，迅速成为流行经典。被如张学友、魏雪漫等歌手多次翻唱。

  
歌词下方是 _听海钢琴谱_ ，希望大家喜欢。
