

抖音“戏子多秋，可怜一处情深旧”，许多朋友都被这首歌的旋律以及优美的歌词深深吸引。据悉，此曲叫做《 **辞九门回忆** 》，最初由冰幽 ，解忧草
演唱。后等什么君翻唱后，再度爆火。

《 _辞九门回忆_
》取材于南派三叔的短篇小说集《老九门》之《二月红篇》，"戏子多秋，可怜一处深情旧"，故事里，二月红的爱人辞世、友人或也离散，身处欢笑之地，却仍是只想吃一口熟悉的面汤。因此这首歌最开始对应的受众，正是喜欢二月红的国风读者们。

但在各平台宣推助力之下，歌曲热度持续爆发，《 _辞九门回忆_
》有了越来越多被歌本身吸引的听众。歌曲把过往情义和家国情仇都描摹进词曲里，等什么君的演绎版本，更多了几份江湖洒脱气息，完美命中国风爱好者的心。有人用歌来演绎自己心中的故事，有人讲述自己关于"戏"的回忆。
