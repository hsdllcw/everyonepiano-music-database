_蝶々结び_
是日本女歌手Aimer的歌曲，收录在专辑daydream中。Aimer，出生于日本熊本县，在2011年9月7日以单曲「六等星の夜/悲しみはオーロラに/TWINKLE
TWINKLE LITTLE STAR」正式出道，隶属唱片公司是日本索尼音乐娱乐旗下的DefSTAR Records，2015年9月移籍到SME
Records。

歌词下方是 _蝶々结び钢琴谱_ ，大家可以免费下载学习。
