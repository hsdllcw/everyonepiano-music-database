

“一声落叶一声秋，跨越万物的邂逅，雨中相拥风中漫游，却无法相守永久”出自歌曲《 **落叶与秋** 》。

《落叶与秋》由刘烁七与梓豪共同演唱，于2021年发行。

歌曲曲风轻松舒缓，十分惬意。欢快的节奏，融入情感饱满的男、女声，让我们在歌里仿佛听到了细腻、甜蜜的爱意。
