

**风往北吹双手简谱** 和五线谱完全对应。

_风往北吹_ 是由易家扬作词，孙楠作曲并演唱的一首歌曲，歌曲收录于孙楠1999年10月01日发行的音乐专辑《南极光》中。

孙楠，中国大陆男歌手、演员、配音、导演，2000年2月年推出专辑《南极光》。迄今为止，其主要成就有中歌榜最佳男歌手，东方风云榜最佳男歌手，音乐风云榜最佳男歌手，全球华语榜中榜最佳男歌手，CCTV-
MTV音乐盛典最佳男歌手以及2015蒙面歌王总冠军。

歌词下方是 _风往北吹钢琴谱_ ，希望大家喜欢。
