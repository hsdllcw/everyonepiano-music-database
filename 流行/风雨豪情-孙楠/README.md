_风雨豪情_
是中国大陆男歌手孙楠的歌曲，收录在其专辑《第一楠主角》中。孙楠的这张新专辑收录了由新近孙楠演唱的13首影视主题曲/插曲。孙楠自己透露：“加盟华纳唱片后，我的歌路以及音乐的制作、包装都会改变，从这个意义上来说，《第一楠主角》是一张承上启下的唱片，但这也并不说明我以后就不演唱影视歌曲了，一切随缘。”

歌词下方是 _风雨豪情钢琴谱_ ，希望大家喜欢。
