

2014年3月8日，马航MH370与我们失去联系。音乐牛人马健南打造的公益歌曲《 **马航去的地方**
》,带领小伙伴们为马航祈福,一起继续等待奇迹的降临。但不幸，马来西亚总理在2014年3月24日晚宣布马航370航班坠毁于南印度洋。

  

2014年度听到最令人悲痛的消息莫过于马航飞机的失联，当官方宣布马航客机确认坠毁后，无数网友失声痛哭，为了纪念逝去的200多号人，歌手马健南创作了一首歌曲《马航去的地方》，倾情演绎催人泪下。

  

马航失联十四天来，关于它的一点点消息都牵动了全世界的目光，所有人都希望奇迹能够发生。在深圳，一位叫做马健南的原创歌手，录制了一首原创的公益歌曲——《马航去的地方》，希望通过音乐，向失联的亲人们传递生命的希望。

  

歌词下方是 _马航去的地方钢琴谱_ ，欢迎大家使用。
