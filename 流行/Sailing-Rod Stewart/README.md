

**Sailing双手简谱** 和五线谱完全对应。

_  
Sailing_ （航行），电影《哥伦布传》的主题曲，也是Rod
Stewart（洛·史都华）的成名之作。Sailing这首歌，朴实无华、意境悠远，既有心情的宣泄，又带点淡淡的忧伤，透出一股苍茫，给人以力量。

  
歌词下方是 _Sailing钢琴谱_ ，希望大家喜欢。
