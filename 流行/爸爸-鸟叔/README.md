

**爸爸** 是收录在2005年鸟叔(PSY)发行的《Remake&Mix
18号》专辑中的歌曲。该歌曲是讲述孩子长大后，写给为了家人而奉献自己的父亲的一封书信。此次公开的中文版，结合了郎朗细腻、完美的编曲，加上鸟叔(PSY)的韩文说唱，配上中文副歌部分，将把韩国粉丝受到的感动传达给中国粉丝。

鸟叔(PSY)跟郎朗在2014年9月19日举办的“2014仁川亚运会开幕式”上首次见面。在开幕式表演中，鸟叔(PSY)和郎朗合作了《Champion》。尽管在
舞台上鸟叔(PSY)和郎朗语言不通，但仍通过“音乐”来交流并成为好朋友，以至于后来还合作了《 _爸爸_ 》中文版。

由于这次舞台是鸟叔(PSY)的首支中文单曲《爸爸》的首秀舞台，也是鸟叔(PSY)在“ALL NIGHT STAND 2014”后的首个舞台，而受到大众关注。

歌词下方是 _爸爸钢琴谱_ ，希望大家喜欢。
