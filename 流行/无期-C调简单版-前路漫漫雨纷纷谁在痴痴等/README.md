

前路漫漫雨纷纷，  
谁在痴痴等，  
任其心头千般恨，  
不做负心人。

光头华夏一曲《 **无期** 》唱出了多少人的无奈与痛苦。如果说遗憾是一种美，那么这份爱便是，优伤、心酸、甜美、思念的美。

有一种无奈，叫做回头无望，你没有遇到能你感同身受的人，经历过才能懂，寂寞时的爱太残酷。

而最难过的莫过于当你遇上一个特别的人，都明白也许不可能，或早或晚，你不得不放弃。爱恨就在一念之间！错过了，都无法再回头了！就像馊了的酒，味道变了。
