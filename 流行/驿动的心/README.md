

姜育恒，台湾歌手。20世纪80年代初，姜育恒孤身从汉城赴台湾发展，一首《再回首》红透半边天，《驿动的心》、《一世情缘》、《再回首》、《多年以后》、《跟往事干杯》、《别让我一个人醉》、《梅花三弄》、《爱我你怕了吗》、《女人的眼泪》、《爱的痕迹》等多首歌曲脍炙人口，然而之后他投资做生意失败，负债累累，个性忧郁的他再次走上歌坛。

### 《驿动的心》歌词：

曾经以为我的家  
是一张张的票根  
撕开后展开旅程  
投入另外一个陌生  
这样飘荡多少天  
这样孤独多少年  
终点又回到起点  
到现在我才发觉  
哦~  
路过的人我早已忘记  
经过的事已随风而去  
驿动的心已渐渐平息  
疲惫的我是否有缘  
和你相依
