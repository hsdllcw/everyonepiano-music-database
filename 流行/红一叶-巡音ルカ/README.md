

**红一叶钢琴曲**
古风味浓厚，曲调哀婉而又忧伤，电子与古典乐器的交融将秋天的萧瑟与红叶下的思念之情表达得淋漓尽致。此歌受到诸多音乐爱好者的喜爱，且被尊称为“神曲”。
_红一叶_ 原唱是初音ミク、巡音ルカ。

该《 _红一叶_ 》为巡音演唱的版本，巡音所唱的版本更广为流传。
