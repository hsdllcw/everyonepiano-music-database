

**弈剑听雨阁** 是一首古风歌曲，由知名古风音乐唱作人小曲儿演唱。

这个曲子的歌词写得非常美，“纵马山川剑自提，身前身后事，付一笔流水落花随意”有一种笑傲江湖、相当洒脱的感觉。

小曲儿，中国原创音乐基地热门歌手，平纱落雁原创音乐团队成员，在“古风圈”颇负盛名，2012年发行的个人首张原创古风专辑《曲倾天下》倍受好评。
