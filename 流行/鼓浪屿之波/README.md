

《 **鼓浪屿之波** 》是一首歌唱祖国统一的音乐作品，由著名作曲家钟立民作曲，张藜、红曙作词。1982年著名歌唱家李光曦首唱。且此曲已成为厦门的品牌之歌。

作为一支思乡曲，《 _鼓浪屿之波_
》的歌词试图表达的是，由于海峡的阻隔，旅居大陆、乡愁难以舒解的台湾同胞在厦门鼓浪屿登高远眺的情景，贯穿其中的是盼望海峡两岸和平统一的主题。歌词中提到的日光岩是厦门的象征，也是鼓浪屿最高处。歌词用日光岩暗含登高眺望、盼望宝岛早日回归团圆之意，也唱出了海峡两岸同胞期盼统一和浓浓的思乡之情。
