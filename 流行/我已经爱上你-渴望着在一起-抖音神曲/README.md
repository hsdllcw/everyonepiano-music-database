

抖音最近火了一首歌曲，有句歌词为：我已经爱上你渴望着在一起。此曲原名为听闻念阳凡懿，后更改为 **我已经爱上你** 。

正如歌词所言“可是我们的差距，有着很大的距离，只能故意骗自己，骗自己放下你”，两人相爱却无法相守，这里小编希望天下有情人终成眷属。
