

**心太软**
是中国台湾男歌手任贤齐的成名作。歌曲以一个劝解者作为主要视角，呈现出一位痴情者为爱毫无保留的付出，并在夹叙夹议间呈现出一些爱情的小哲理。这首歌曲不仅歌词通俗易懂，而且旋律也非常简单流畅，尤其“心太软”生动贴切的形象，更助长了歌曲的流行势头。

《 _心太软_
》传遍了大江南北，其实它的旋律很简单，唱的也只是很常见的一种“鸡肋”爱情心态：对于已近绝望的爱情，食之无望，弃之不忍，但它却触动了城市男女最柔软的心底。

《心太软》不是阳春白雪，整首歌里没有一丝一毫高雅的态度；  
《心太软》也不是下里巴人，在歌曲里找不到有任何可以被斥之为“庸俗”的卖点。

歌曲中诸如“相爱总是简单，相处太难”这样的爱情小哲理，堪称都市文化的代表。这首歌曲以最简单的词汇表现出一定的现实规律，是它成为流行经典的因素之一。《心太软》看似是一首简单的小情歌，就在这一深一浅的平衡中，尽显创作过程中的难度。
