

**祖国不会忘记** 是由张月谭作词，曹进作曲，黄鹭演唱的歌曲，创作于20世纪90年代。2001年，该曲获得第八届精神文明建设“五个一工程”入选作品奖。

整体上，《 _祖国不会忘记_
》拥有着磅礴大气而又不失朴实真挚的风格。在铿锵有力的前奏中，歌曲波澜壮阔的气势喷薄而出。歌曲的第一部分，节奏前密后疏，富有动感，充满活力，表现出战士们勇往直前的攀登精神；歌曲的第二部分，节奏拉宽音域提高，抒情中不乏坚定，表现战士们无私奉献的乐观精神。张月潭的歌词饱含对航天科技工作者的敬仰、对航天事业的热爱；而曹进的歌谱则在雄浑中蕴含着深情。
