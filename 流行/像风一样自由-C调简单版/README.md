

**像风一样自由** 这是许巍对生活的呐喊，就像风抓不住，雨截不停一样。

自由的意义是什么？是不用思索生活琐碎，淡然面对柴米油盐；是不用背负人间责任，笑谈生活无常；或者是不用肩扛亲人朋友，独自前行。那不是自由，是逃避。世间哪有真正的自由，自由的背后往往背负更多：世间形容词只有对立才会存在，就像有高才有矮，又胖才有瘦，所以，自由的背后是责任。

每次在心情低落的时候听到这首歌都会有种豁然开朗的感觉，好像可以敞开心扉去接受一切。听许巍的歌是一种享受，他让我平静，慢慢的思考人生，不慌张不焦虑。我也许是不曾历经沧桑，但我向往风雨过后依然
_像风一样自由_ 。
