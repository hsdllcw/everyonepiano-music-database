

**做我老婆好不好双手简谱** 和五线谱完全对应。

_做我老婆好不好_ 是徐誉滕演唱的一首流行歌曲，由徐誉滕填词、谱曲、制作，收录于专辑《爱若去了》中，发行于2007年10月22日。

从《等一分钟》，到《茉莉花开》和《流星飞呀》，然后到向大家介绍的这首《做我老婆好不好》，他始终是在用自己清凉纯净的声音在像我们讲述一个又一个故事。

歌词下方是 _做我老婆好不好钢琴谱_ ，希望大家喜欢。
