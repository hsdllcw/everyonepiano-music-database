

2021年12月25日，时代少年团发布了歌曲《 **有你** 》。此曲由时代少年团队成员七人全部参与作词，精心制作，收录于同名专辑《有你》中。

“因为有你，每一片雪花都有意义”这句歌词戳中了无数粉丝的心。音律悠扬字符旋转谱写新乐章，琴声流泻指尖掠过黑白变幻七彩。《 _有你_
》整首歌节奏十分欢快，让大家在这个寒冷的冬天感受到了不一样的暖意。

_有你钢琴谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
