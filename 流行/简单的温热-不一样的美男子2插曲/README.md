

**简单的温热双手简谱** 和五线谱完全对应。

_简单的温热_
是电视剧《不一样的美男子2》的插曲，由尤起胜作词作曲，全能唱作才子徐良受邀献声，深情饱满极富青春态度的徐良与OST“青春•科幻”的音乐风格携手共进。

《不一样的美男子Ⅱ》是由芒果影视、春秋风云影视策划有限公司联合出品的情感励志剧，丁仰国导演执导，张云龙、阚清子领衔主演，陈奕，崔绍涵、蒋羽熙、谢彬彬、方逸伦等联袂主演，汤镇宗、张佩华、郑家榆特别出演。

该剧讲述了双胞胎姐妹初夏和萧瑾，因意外而重逢，却阴差阳错的爱上了同一个男生关昊，在历经重重磨难和生死考验后，初夏带着友情和爱情走向一段别样历程的故事。

歌词下方是 _简单的温热钢琴谱_ ，希望大家喜欢。
