

《星座书上》简单版很简单，双手简谱和五线谱是完全对应的，是EOP教学曲，很适合新手练习。

《星座书上》，具有简洁的编配，颇具演唱难度的旋律，有骨骼的词。延续歌迷们所谓"V式"浪漫词风，旋律线简洁却不缺乏技巧性，陈述部和副歌之间的强烈对比有着极大的情感冲击力。Vae的编曲以及制作令人佩服。演唱方面，其歌迷一直称其为“灵魂派”——用心歌唱的孩子最动人。
