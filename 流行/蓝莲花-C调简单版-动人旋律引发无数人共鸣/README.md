

对于80后的人来说，许巍可以说是天才一样的人物，他的很多歌都是大家的最爱。特别是这首《 **蓝莲花** 》，在KTV的时候，总是会发生抢麦的故事。

《 _蓝莲花_
》歌唱的是一种坚定不移追求梦想的宝贵精神。这首歌曲的歌词没有豪言壮语，曲调亦无跌宕起伏之处，整首歌以一种平静祥和的口吻唱出来，但却震撼了千万人的心灵。

了解许巍的人都知道，其实他特别的崇拜玄奘大师，也比较喜欢佛学文化，这首歌如果你仔细品味的话会发现它的禅意深远，而且也蕴含着许多人生道理。玄奘当年为了取得经书而百折不挠的精神让人钦佩，蓝莲花象征他也未尝不可。

其实一首歌被写出来之后被听众听到耳朵里面的那一刻起，它就不单单只是一首歌了，因为每个人的故事不同，感悟自然也就不同，可能每个人对于蓝莲花的理解都不一样吧，对于许巍来说，它象征着玄奘，那你呢？
