

**你是我的玫瑰花双手简谱** 和五线谱完全对应。

_你是我的玫瑰花_ 是庞龙演唱歌曲，由大鹏作词作曲，毕晓笛、李杰编曲，收录于其2005年9月6日专辑《你是我的玫瑰花》中。

歌词下方是 _你是我的玫瑰花钢琴谱_ ，希望大家喜欢。
