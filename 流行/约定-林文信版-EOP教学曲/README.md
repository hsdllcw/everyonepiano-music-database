

《 **约定** 》收录于台湾知名女歌手周蕙1999年出道的首张个人专辑《周蕙精选》，当年选歌的时候，制作老师在KTV听到了王菲的粤语版《 _约定_
》，就推荐给了公司，公司非常大胆的决定让初出茅庐的周蕙来翻唱这首歌，于是邀请姚若龙陈小霞重新填词编曲，果不其然，这张专辑凭借《约定》的国语版，以及《不想让你知道》《风铃》等多首脍炙人口的好歌迅速红遍大街小巷，创下100多万张的惊人销售量、占居音乐排行榜首位过半年之久,而周蕙也因歌声清澈纯美好一举成名，与同期新人蔡依林、萧亚轩、孙燕姿迅速成为当年炙手可热的四小天后。周蕙的事业在稳定中前进。全新专辑《自己的房间》已于2011年10月18日发行。

  
天籁女声，与世无争的微笑，仿佛只想安静的唱歌，恬静到几乎透明。她用自己的歌声感动了音乐世界，独有的魅力征服了无数歌迷。蕙儿的每一首歌，闭上眼睛，用心听，心就慢慢平静下来。每一次情感上的经历都能在蕙儿的歌中找到安慰。蕙儿的声音总是这样，让你在遗忘很久后的某一个时刻突然记起，然后感动的一塌糊涂。

  
而这首 _约定钢琴曲_
则是林文信根据周蕙的《约定》改编而来，但仍然不失原曲的味道。同时，双手简谱和五线谱完全对应，也是EOP教学曲。所以比较适合钢琴初学者练习。
