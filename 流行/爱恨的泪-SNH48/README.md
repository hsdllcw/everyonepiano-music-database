_爱恨的泪_ 是SNH48 TEAM SII的公演《不眠之夜》中的曲目。原曲为AKB48的《てもでもの涙》。

2015年1月31日，SNH48年度金曲大赏演唱会上由袁雨桢、莫寒演绎。

2015年12月26日，SNH48第二届年度金曲大赏演唱会上由冯薪朵、陆婷演绎。

2017年1月7日，SNH48第三届年度金曲大赏演唱会上由李艺彤、黄婷婷演绎。
