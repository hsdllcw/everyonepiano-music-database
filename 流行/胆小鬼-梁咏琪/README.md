_胆小鬼_ 是梁咏琪演唱的一首歌曲，收录在专辑《梁咏琪 gigi》当中。  
  
同名专辑《梁咏琪 gigi》，可说GIGI将音乐想像成生活，于生活中的酸、甜、苦、辣，音乐里也会有所感受，一起分享GIGI的喜、怒、哀、乐。
这张专辑请到李偲菘、张洪量、黄韦皓、陈伟为制作人，而姚若龙、李偲菘、陈小霞、张洪量、潘协庆、十方等人，为GIGI量身订做词曲，这张生活纪事的音乐写真，轻快明亮的folk、快捷奔放的rock、抒情温馨的love，在这样的音乐里，就好似看着GIGI的情绪在空气里呼吸，看着GIGI这样的在面对生活。这就是GIGI的快适生活音乐。  
  
相信喜欢梁咏琪的人一定很多，喜欢她的《胆小鬼》的人也不少。这首歌个人觉得是一首甜美，可爱的歌。
