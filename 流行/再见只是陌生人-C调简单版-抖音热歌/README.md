

**再见只是陌生人**
由歌手庄心妍演唱。时间带走了年少轻狂，也让我们懂得了人走茶凉，经历了流年聚散，看透了世间冷暖。爱情这东西，只有爱不爱，没有合不合，没有人天生为你准备，磨合的过程固然痛苦，但过后的融洽却更有意义！

“我们爱得难舍难分，爱得奋不顾身，为何再见只是陌生人”，时过境迁，情只是一段回忆。陌生人变成熟人很简单，但是熟人一旦变成了陌生人，就再也回不去，那种深爱后的陌生，那种，从无话不说变成无话可说的痛，任何语言，都表达不了

《 _再见只是陌生人_ 》歌曲的整个情绪开始向着更加洒脱的方向发展，主歌与副歌的黑白向彩色的转换，也完成了歌曲想表现的情绪转换。
