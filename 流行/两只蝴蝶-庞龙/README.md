《 _两只蝴蝶_
》是中国内地男歌手庞龙演唱的一首歌曲，由牛朝阳作词作曲，周平亚制作，2004年收录在《两只蝴蝶》同名专辑，同时也是电视剧《281封信的》的插曲。2009年音乐人周亚平凭借该歌曲获得“亚太文化产业杰出人物奖”。  
  
《 _两只蝴蝶_
》的词、曲都是由牛朝阳创作的，《两只蝴蝶》的副歌部分有三个音符是一位叫杨桦的女孩帮牛朝阳写的，虽然事隔多年，但牛朝阳觉得必须对这个女孩子表示感谢。《两只蝴蝶》是在1996年写的，当时牛朝阳还是鸟人公司的歌手，他认识了一位叫杨桦的女孩，杨桦经常来听牛朝阳弹琴唱歌。有一次，当牛朝阳写到这首歌副歌部分，歌词是：我和你缠缠绵绵翩翩飞，飞越这红尘永相随，但“永相随”三个字的旋律牛朝阳写了好几种都觉得不好，于是就叫杨桦随意接着唱，杨桦就按照自己的感觉唱了这三个音，牛朝阳当时觉得很好，就记下了谱子，就这样定稿了。  
  
《两只蝴蝶》是一首典型的、经典的、凄美的爱情歌曲，纯情的歌词深入人心，让每个陷入爱河的男男女女感动不已。尤其当演唱到“等到秋风起，秋夜落成堆，能陪你一起枯萎也无悔”时，歌者庞龙早已潸然泪下，听者也莫不唏嘘不已。带有内在律动的美丽的旋律，极强的上口感，活跃而自然的画面感。
