

**牵丝戏双手简谱** 和五线谱完全对应。

_牵丝戏_ 是由银临、Aki阿杰演唱，并于2015年推出的古风单曲。通过描绘傀儡翁与牵扯一生的傀儡之间的相伴、别离，来诉说一段牵恋。

银临是原创音乐人，代表作《锦鲤抄》和《腐草为萤》。

歌词下方是 _牵丝戏钢琴谱_ ，喜欢的朋友可以免费下载学习。
