

**瞬间双手简谱** 和五线谱完全对应。

_瞬间_
是邓紫棋填词、谱曲并演唱的歌曲，于2015年9月17日首发，同年11月6日收录于邓紫棋发行的专辑《新的心跳》中。2015年9月26日，该曲获得全球中文音乐榜上榜周冠军。

瞬间在表现上更趋丰富，口气和声音情绪越发婉转并充满质感，曲风与《多远都要在一起》风格相似
，歌曲朗朗上口，歌词直击人心。歌曲属于最简单的芭乐，歌词写得华而不实，令人听下来并没有什么感触。bridge的旋律走向有点特别，最后一段副歌的清唱算是唯一亮点。

歌词下方是 _瞬间钢琴谱_ ，大家可以免费下载学习。
