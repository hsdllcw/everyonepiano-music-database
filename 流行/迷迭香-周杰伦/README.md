

**迷迭香**
是周杰伦演唱的一首歌曲。由方文山作词，周杰伦作曲，钟兴民编曲，收录在周杰伦2006年发行的专辑《依然范特西》中。2007年，这首歌曲在无线音乐榜上获得第二季冠军。

《 _迷迭香_
》的曲风不同于周杰伦以往的歌曲，周杰伦在成名后并未保守地止步不前反而大胆尝试，勇气可嘉。歌曲以慵懒的鼓声敲开夜的薄纱，在充满异国情调的曲风烘托下，周杰伦的唱腔也变得很温柔。《迷迭香》在曲风和词作上显出诚意，但这样的异域风情，让人不由联想起《她的睫毛》。

另外，歌曲的MV由周杰伦亲自执导。MV中周杰伦与女舞者一同跳起“伦巴”舞。此外周杰伦的外婆也入镜拍摄，在MV中回忆一段过往恋曲。

歌词下方是 _迷迭香钢琴谱_ ，希望大家喜欢。
