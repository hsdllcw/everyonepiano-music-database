

抖音“还记得那年那月那花香 还记得那时那里那姑娘”出自歌曲《 **那年•年少**
》。《那年•年少》由宋宇宁作词、作曲，宋宇宁演唱，发行于2020年11月13日。

如果你看了《好声音》第九季的节目，就应该知道，其实这首歌当时是宋宇宁的参赛作品，并且还获得了当时现场观众的大力支持。没想到今年它再度火了起来。作为好声音第一位进入决赛的原创歌手，宋宇宁以《
_那年•年少_ 》的震撼舞台表现征服全场，一举闯入五强行列。“年少多轻狂，天高欲何妄。”歌曲淋漓尽致传递着少年那股顽强自信、永不言败的骄傲。

宋宇宁人歌合一的完美演绎，获得导师李健极高评价：“完全绽放了自己，像歌词一样。”酷狗歌曲评论区4000＋热评中，网友各种花式夸奖：“简直发现了一个宝藏、戏腔高音绝绝子、神仙颜值拥有神仙声音、好听到爆炸”。
