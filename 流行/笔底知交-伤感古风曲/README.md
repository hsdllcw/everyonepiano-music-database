_笔底知交_
是由锦衣小盆友作词，银临作曲，灰原穷编曲，KBShinya和哦漏演唱的一首歌，收录于KBShinya和哦漏的合作专辑《K.O》。后银临和少司命重新翻唱了此曲。歌曲真的很虐心，画师和君王的约定因为山河而成为生死永别，他们的友谊有始无终
，君王从此再无像画师一样的笔底知交，只得在批阅奏折时用画师送给他的印来留念。或许这就是超度了时空生死的友谊吧！
