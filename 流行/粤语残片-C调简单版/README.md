

《 **粤语残片** 》是香港流行男歌手陈奕迅的一首粤语歌曲，由周博贤作词，C.Y.Kong（江志仁）和陈奕迅作曲，陈奕迅演唱。收录于专辑《What's
Going On…?》中，发行于2006年11月23日。

《 _粤语残片_ 》的国语版是《白色球鞋》，收录于2006年11月23日发行的专辑《认了吧》中。

《粤语残片》讲了搬家时发现夏季衬衣回想起当时的初恋，但现在即使经历再多恋情也不会有与初恋相识时的动心。虽然该歌只有现场版MV，但MV中Eason极为动情几乎快要哭出来地演唱相当震撼和感人。
