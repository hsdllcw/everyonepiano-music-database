

《 _不值得_ 》是一首比较伤感的歌曲，由梦飞船演唱，在酒吧和咖啡店经常能听到。粤语版《不能爱恋 只能暗恋》由香港艺人应昌佑演唱。  
  
梦飞船组合，成员有苏志城、刘瑞聪和林毅心。  
  
歌词的下方提供了 **不值得钢琴谱** ，欢迎感兴趣的朋友下载使用。
