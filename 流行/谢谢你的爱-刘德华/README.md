

《谢谢你的爱》这张专辑是刘德华在宝艺星最后的作品，“谢谢你的爱”是一首风靡全国的名曲，全球销量过百万，也为刘德华囊获了多个音乐大奖，致使刘德华跳槽华纳后还专门从前唱片公司要了这首歌作一个粤语版。

刘德华
大中华区乐坛和影坛巨星，华人娱乐圈影、视、歌多栖发展最成功的代表之一。作为歌手，他是香港乐坛“四大天王”之一，吉尼斯世界纪录大全中获奖最多的香港男歌手；作为演员，他参与演出的电影已经超过140部，先后获得三届香港电影金像奖最佳男主角奖，两届台湾电影金马奖最佳男主角奖。曾四次登上央视春晚舞台，享誉全球华人社会。
