

**类似爱情**
是萧亚轩演唱的一首歌，由易家扬填词，小安谱曲，收录在萧亚轩2008年发行的专辑《三面夏娃》中。2008年6月20日，该曲获得Topinmusic-
IN乐榜港台彰榜牌两周冠军。

《 _类似爱情_
》被许多歌迷奉为专辑中最为感人至深的一首难得的佳作，除了上佳的词曲表现，Elva真情流露、用情至深的演唱更是让人耳目一新、感动升级。虽然是萧亚轩较少尝试的民谣小品，感觉温柔感性。萧亚轩成功将歌曲中“我们两个人陌生又熟悉”那类似爱情的暧昧感唱得丝丝入扣，充分表达出爱情来临时的甜蜜。

歌词下方是 _类似爱情钢琴谱_ ，希望大家喜欢。
