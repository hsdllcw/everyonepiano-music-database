_山鬼_
是原创音乐人Winky诗的曲子。Winky诗，偏好词、曲、编、唱、后期、PV/MV一手包办。风格多变，国风、和风、Jazz、流行、民乐等都喜欢玩。名字中的“Winky”源于小诗最喜欢的小说《哈利波特》中家养小精灵闪闪的英文名，虽然小诗并不是很中意这个角色，然而却喜欢这个很跳跃的英文单词；而名字中的“诗”源自于小诗在“弃文从乐”前最爱创作的现代诗。
