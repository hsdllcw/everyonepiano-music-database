

《 **慢慢**
》原唱张学友。张学友，1961年7月10日生于香港，是一位在亚洲地区和整个华人社会具有影响力的实力派音乐巨星和著名电影演员，香港乐坛“四大天王”之一，在华语地区享有“歌神”的称誉。王乃恩2012年参加节目《中国好声音——The
Voice of China》成为刘欢团队的一名学员，在中国好声音舞台翻唱《 _慢慢_ 》。
