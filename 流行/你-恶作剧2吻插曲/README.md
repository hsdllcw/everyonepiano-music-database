

**你**
是中国台湾偶像剧《恶作剧2吻》的插曲。《恶作剧2吻》是台湾偶像剧《恶作剧之吻》的续集。该剧讲述的是一个单纯开朗的高中女学生——湘琴，自从在开学典礼上看见代表新生致词的江直树後，便不由自主地喜欢上这个号称IQ200的超级天才少年，而后努力展开追求的故事。

且歌曲 _你_ 由林依晨演唱。林依晨（Ariel Lin），1982年10月29日出生于台湾省宜兰县，中国台湾女演员、歌手。

歌词下方是 _你钢琴谱_ ，希望大家喜欢。
