

《 **遗憾** 》原是许美静的经典作品之一。此次，由曾经为齐秦、迪克牛仔、谢霆锋等人担任过制作人，并打造过多首脍炙人口的歌曲的音乐人Peter
Wong担任制作人，为李代沫倾力打造。

李代沫透露：“《 _遗憾_ 》这首歌曲选择在圣诞节时发布，也是真心的希望这首歌曲能在圣诞节陪伴着大家，让大家在今天不会孤单。

《 _遗憾_
》略带忧伤的词曲，经过李代沫的演绎，仿佛细细的讲述了爱情中的百转千回。李代沫就像你的知己一样，将你生活与爱情中种种的遗憾与心碎用真情唱出，在圣诞节来临时，为你送上暖意，安抚你为种种遗憾而失落的心。该钢琴曲为
**遗憾弹唱版** 。
