

**闹够了没有**
由赖伟锋作词作曲，赖伟锋演唱，收录在其2013年发行的专辑《写给少年回不去的爱》以及2013年7月发表的EP《不赖》中。该曲写出了所有还在单恋着的人内心的纠结和与煎熬，受到业内人士及广大网友的一致好评。

  
《 _闹够了没有_
》MV是《不赖》mv的姐妹篇，在内容上二者有着密切的关联。《闹够了没有》MV讲述了当下“备胎进行时”的委屈和纠结，所有还在单恋着的人内心的纠结和与煎熬。

  
歌词下方是 _闹够了没有钢琴谱_ ，希望大家喜欢。
