_漂洋过海来看你_
是电视剧《漂洋过海来看你》的主题曲，由李宗盛作词作曲，朱亚文、王丽坤演唱。发行于2017年03月22日。歌曲以小切口植入女人内心故事的情歌，忧伤得无以复加，画面感极强。
