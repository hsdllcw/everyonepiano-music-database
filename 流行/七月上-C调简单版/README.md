

**七月上**
是Jam演唱的一首歌曲，由Jam本人作词作曲，于2015年4月17日发布，收录在专辑《七月上》中。歌曲发布后受到许多人的关注与喜爱，成为新时代流行歌曲。

七月上这首歌没有真实的故事依据，不过歌词本身讲的就是一段故事。意思大概就是主角爱上一个给他（她）如七月般感觉的人，也可能是在七月里遇到的，主角勇敢追求，想要了解对方的内心深处，但这可能是一场苦恋。因为害怕失去而不敢去拥有，更宁愿对方从来没有出现过；因为心疼他，害怕自己的出现会伤害他，最后在离开后，仍然会选择在背后默默的爱着他...

其实从《 _七月上_
》里听出了爱情义无反顾中不一样的味道，也表达了Jam自己的一种感情看法。或许一句大漠边疆，让爱情从恋恋红尘变成了黄沙滚滚，或许一句乘风破浪，把爱情写成了飞蛾扑火的执着。
