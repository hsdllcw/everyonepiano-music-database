

**安和桥双手简谱** 和五线谱完全对应。

_安和桥_ 是民谣歌手宋冬野演唱的一首歌曲，由宋冬野作词、作曲，收录在宋冬野2013年发行的专辑《安和桥北》中，是该专辑的主打歌曲。

由于宋冬野对这首歌的感情太深，所以已经无法做到从编曲制作的等等其他方面去评述它，宋冬野凭借它斩获2013年”第三届阿比鹿音乐奖“年度单曲奖”。

歌词下方是 _安和桥钢琴谱_ ，希望大家喜欢。
