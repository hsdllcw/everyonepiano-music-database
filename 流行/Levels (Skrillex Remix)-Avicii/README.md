

**Levels (Skrillex Remix)** 是瑞典house DJ Avicii的电音作品，发行于2011-10-28。Avicii 原名Tim
Bergling，中文译名：艾维奇。1989年9月8日出生于瑞典的斯德哥尔摩省。自小迷恋音乐的他对电音舞曲情有独钟，受到了Daft
Punk的熏陶，加上瑞典地区发达的电音产业让他萌生了成为DJ的念头。

  
歌曲 _Levels (Skrillex Remix)_ 中的人声部分采样自Etta James的《Something's Got a Hold on
Me》，而同样采样了Etta James这首歌曲的还有Flo rida的近期热单《Good feeling》但实际上《Good
feeling》是直接采用avicii中的剪辑加以修改。歌曲在多个国家的单曲榜中登上榜首。
