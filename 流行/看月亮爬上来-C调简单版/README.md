

《 **看月亮爬上来** 》由张杰演唱，收录在张杰2009年发行的概念专辑《穿越三部曲》和2012年发行的《One Chance 张杰新歌+精选》辑里。

这是一首白色的歌。清新、单纯、美好、简单。所以张杰站在音乐的原点演绎这首歌，他将标志性的嘹亮高音放在一边，还原到最真实最原始的嗓音状态，采用中国传统的五音唱法，将《
_看月亮爬上来_ 》演绎成一首和他以往作品完全不同的优美月亮情歌。

轻快悦耳的民谣小调，似曾相识又久违了的亲切。最简单的旋律和歌词配合出来极美的韵味与意境。同时借由巧妙高超的编曲做最丰富的变化，营造出了“小夜曲”般的优美意境；而ACAPELLA（用人声模仿器乐）+多声部人声和声变化，更加突出张杰的唱功。
