孤独的足迹_ 是韩剧《一枝梅》中的一首纯音乐插曲。也有翻译为《孤独的脚印》的。  
  
简短又意味深长，好像把一枝梅的恩怨情仇总结到一起了。此曲里没有什么转折，只是完全的孤独、悲伤，也带着一点点的遗憾，想抓住，却抓不到的感觉，很美，就像主角带着恨意，寻找亲人、当义贼，只为了家人，那本来无辜的家人...一路坎坷，离挚爱而去、只为亲人，付出的代价太多太多，最后，却不忍心杀了那个所谓的“王”、不知所踪，世人每每想起时，都会感叹...听了这首曲子，我又想了那种情绪，多少爱恨情仇，为了不暴露身份，最爱连他的真面目也没看过，至于真正的故事是不是这样，已经不重要了......  
  
韩剧SBS水木剧《一枝梅》由李准基、韩孝珠领衔主演，主要讲述朝鲜中期的侠盗一枝梅与当权者和腐朽的社会制度进行抗争的故事，他专门惩罚有钱有势，有武力的贪官。由于不知道他的来历，他成为朝廷官员们最痛恨的罪犯，也是穷困百姓们的希望。偶然的机会他与卞植的女儿恩彩相遇，并陷入爱情。剧中通过当时朝鲜王朝的种种现象来讽刺当今社会存在的官场与黑势力勾结、高官子弟享有特权、作弊风气等社会现象。
