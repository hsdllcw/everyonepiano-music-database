

**最近双手简谱** 和五线谱完全对应。

抖音有一首歌曲让人听起来非常有感觉，歌词是 “你想要的,我却不能够给你我全部”，此曲叫做 _最近_ ，原为李圣杰演唱，后经抖音王小帅翻唱再次走红。

同时，网站还为大家提供了《[你那么爱她](Music-2220-你那么爱她-李圣杰-林隆璇.html "你那么爱她")》的曲谱下载
