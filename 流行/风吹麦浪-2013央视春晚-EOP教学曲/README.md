

**风吹麦浪双手简谱** 和五线谱完全对应，是EOP教学曲。

《 _风吹麦浪_
》这首歌写得更多的是一种怀念，但怀念里没有伤感，只有温馨的记忆，表现了这个喧嚣的商业城市中人们对往日那份单纯而美好的情感的留恋，呼唤人们记忆深处最纯净的声音。该歌曲被选入2013年央视蛇年春晚歌舞类节目，李健与孙俪联袂献唱。

“2007年度北京流行音乐典礼”2008年1月23日晚在北京奥体中心举行，一向低调的唱作歌手李健凭借专辑《想念你》中的歌曲《风吹麦浪》夺得最佳作词大奖，显示了这位创作才子不俗的功力。

《风吹麦浪》这首歌从头至尾贯穿着一种远离城市喧嚣的轻柔欢快之感，其清新质朴的质地确实有着一种田园生活独特的情调。歌曲中抒发的本意是在怀念从前的一段感情，但是在歌词之中我们没有发现是带着伤感的情绪来回忆的。反而是这种平和的节奏使得我们相信，这段歌词中的“感情”曾经带给作者更多的是快乐而不是痛楚。

这首歌很好听，相信很多人都喜欢，歌词下面提供了 _风吹麦浪钢琴谱_ ，喜欢的可以下载。
