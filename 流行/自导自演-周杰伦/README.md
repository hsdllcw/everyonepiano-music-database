

《 **自导自演** 》是一首R&B曲风的歌曲，由周杰伦演唱，收录在其2010年5月18日发行的专辑《跨时代》中。

歌词第一句“巷口甩尾甩开了过去的熟悉”就充满想象力，颇有意识流的味道，配合上周杰伦的唱腔正好。

接着，“你我距离就像打滑和那飘移”、“车灯闪了一下代表我还在意”、“喇叭按了两声代表我会离去”和“呼啸而过的引擎声”等歌词中的汽车用语在周董笔下都成了具有特定意味的事物，主角在车上一举一动都是过去感情的倒影。

作为离别和驾驶题材的歌曲，《 _自导自演_ 》比《飘移》《一路向北》更值得玩味。
