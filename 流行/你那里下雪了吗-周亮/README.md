你那里下雪了吗由胡新海作词，吴颂今作曲，周亮演唱，收录在周亮1995年8月8日发行的专辑《你那里下雪了吗》之中。

2002年2月，卓依婷在专辑《化蝶》之中翻唱了《你那里下雪了吗》。

2006年10月，张芯在专辑《花蝶恋》之中翻唱了《你那里下雪了吗》。
