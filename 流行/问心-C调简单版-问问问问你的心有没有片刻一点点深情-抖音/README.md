

抖音“问问问问你的心，有没有片刻一点点深情”出自歌曲《 **问心** 》由就是南方凯（项学凯）演唱，执白作词，魏传迪作曲，发行于2021年11月30日。

在我们相处的时间当中是否有一刻钟的时间你对我是深情以赴呢？其实，很多时候，一首歌曲的迅速走红跟歌词和旋律有很大的关系。如果一首曲子能引起大家的共鸣，触动大家的内心，那么自然而然就会被大家认可。正如这首《
_问心_ 》，“问问问问你的心，有没有片刻一点点深情”，一句话就直击内心，让人回味。
