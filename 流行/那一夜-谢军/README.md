

**那一夜**
是中国内地男歌手，音乐制作人谢军的代表作。2007年某网站评选出十大恶俗网络歌曲，《那一夜》位居榜单首位。同年，中国音乐家协会召开"抵制网络歌曲恶俗之风"座谈会，《
_那一夜_ 》遭到与会代表的痛批。

  
歌词下方是 _那一夜钢琴谱_ ，希望大家喜欢。
