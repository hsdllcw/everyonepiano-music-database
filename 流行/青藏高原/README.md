

长江黄河的发源地、圣洁的雪域高原、丝绸之路的延伸、神秘的宗教文化、灿烂的历史、鲜活的风土人情、壮丽的山河……  
《青藏高原》赞美祖国大好河山，期盼世界和平，人类安康。也成就了后来人对西藏的向往。由歌唱家李娜演唱。她也凭借此歌达到了她歌唱事业的巅峰。
