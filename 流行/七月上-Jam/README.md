

**七月上双手简谱** 和五线谱完全对应。

_七月上_ 是Jam演唱的一首歌曲，由Jam本人作曲，小飞作词，于2015年4月17日发布，发布后受到许多人的关注与喜爱，成为新时代流行歌曲。

Jam，湖南长沙籍90后民谣音乐人，其作品主要在荔枝FM，网易云音乐。

歌词下方是 _七月上钢琴谱_ ，大家可以免费下载学习。
