_谁动了我的琴弦_
是周笔畅演唱的歌曲，由柯肇雷作词作曲，收录于周笔畅首张同名个人专辑中。是周笔畅首张专辑的同名主打歌，同时也是小柯老师送给她的礼物。在这首古今交错、中西合璧的歌曲中，小提琴与琵琶交织铺陈，流行唱腔演绎古色词境。笔笔用略带沙哑而又充满质感的音色，赋予了整首歌丰富的画面感。

歌词下方是谁动了我的琴弦钢琴谱，希望大家喜欢。
