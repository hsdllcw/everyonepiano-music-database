

**泥销骨** 是南风ZJN演唱的一首歌曲，同时也是《魔道祖师•忘羡》同人歌 。

《魔道祖师》是中国网络作家墨香铜臭（xiu）创作的耽美、玄幻、修仙小说。小说以魏婴（魏无羡）和蓝湛（蓝忘机）之间的恋情为主线。故事背景设定有仙门百家，以云梦江氏、姑苏蓝氏、兰陵金氏、岐山温氏、清河聂氏五大世家的兴衰、相互之间的斗争为主。五大世家中，岐山温氏实力最为强大，一家独大，压迫其它家族。有作者评价小说为中国大陆耽美文中的“现象级作品”，而自2018年起，小说相继改编成广播剧、动画、电视剧，皆大受欢迎。
