Shawn
Mendes给人的印象就像是无公害水果。除鲜嫩可人，各种养眼，还安全无毒副作用。尤其那亲切而又略显矜持的笑容，更像邻家男孩。他的声线干净如晨露，让人感觉纯静，却又隐带甘甜。看似舒缓，但激情爆发处，有着让人欲罢不能的节奏感。2017年11月20日，获得全美音乐奖最受欢迎当代成人歌手。
