

**パプリカ双手简谱** 和五线谱完全对应。

_パプリカ_ ，中文译为红辣椒，是由日本小学生组合Foorin发行的一张单曲，该单曲由米津玄师担任词曲作者并兼任制作人。

2018年7月19日，YouTube上首先释出了该单曲的舞蹈音乐录影带；随后在8月10日发布了名为“世界观音乐录影带”的视频。舞蹈由辻本知彦和菅原小春负责编排。

NHK在8月到9月期间于其节目《大家的歌》中播放该单曲。单曲表演的舞台由Foorin的舞蹈构成，同时2020年夏季奥林匹克运动会吉祥物Miraitowa和Someity也会出现。同时该单曲也成为NHK的2020年东京奥运会应援曲。

歌词下方是 _パプリカ钢琴谱_ ，大家可以免费下载学习。
