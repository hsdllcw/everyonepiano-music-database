

**Gotta Be You** 是英国当红男子流行组合One Direction 的首张专辑 Up All Night 的第二曲主打歌，这是继第一曲主打歌
What Makes You Beautiful
之后的又一首大热歌曲。该歌曲走抒情路线，旋律朗朗上口，歌手的演唱也富有真情实感，使其牢牢抓住大批少女粉丝的心，被粉丝广为传唱。 _Gotta Be You_
在各大广播电台播放时收听率大增，并在欧美歌曲榜单中获得不错的成绩。

  
One Direction 是一个英国五人男子组合，成员包括：Liam Payne，Louis Tomlinson，Niall Horan，Harry
Styles和Zayn Malik。他们的平均年龄在18岁。他们作为个人选手通过了The X
Factor（又名英国偶像，一档英国选秀节目）2010年第七季的选拔赛，进入了训练营却没有成功进入男生组评委之家。在评委Nicole
Scherzinger的提议下，这五个青少年于2010年7月23日组成One Direction组合，一路晋级。最终，One
Direction组合成为比赛的季军，之后签约唱片公司Syco Music。
