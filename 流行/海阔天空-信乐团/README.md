

台湾首部艺人自传偶像剧"死了都要爱-信乐团的故事"片头主题曲： _海阔天空_ 。  
  
描写信乐团团员们心声《 _海阔天空_
》是首不俗之作。在简单而带点伤感的编曲下，歌者们沧桑的歌声更见动听，正如笔者在另一个专栏所言，乐迷无须看任何画面，脑海已不期然浮现起一幕幕年轻人奋斗的画面。  
  
歌词的下方还提供了 **海阔天空钢琴谱** ，欢迎感兴趣的朋友下载使用。
