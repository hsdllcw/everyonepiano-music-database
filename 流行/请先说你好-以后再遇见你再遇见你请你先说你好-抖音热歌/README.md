

最近抖音上有个老奶奶带着老伴的遗像坐缆车的视频让人非常感动，背景音乐有一句歌词是以后再遇见你再遇见你请你先说你好。此曲叫做 **请先说你好**
，由贺一航演唱。

曾经以为今生能共度，不想却一别两散在人海。自和你别后，苦楚多欢乐少，寂寞在心上筑了巢，若能再见，情思如潮在心里翻腾成海，盼你能先说一声你好，让我偷来时间扼杀拥抱的冲动。贺一航新歌《
_请先说你好_ 》，情到深处情难控，略带嘶哑的感性吟唱爱之殇。
