

**于是** 是邓紫棋演唱的一首歌曲，由邓紫棋作词、作曲，Lupo
Groinig编曲，首发于2015年11月3日，收录于邓紫棋2015年11月6日由蜂鸟音乐发行的录音室专辑《新的心跳》中。

《 _于是_
》带着无奈，必须要让自己坚强。这首歌跟《单行的轨道》走向相似，而风格完全不一样。在做一个选择时，因为什么，于是你这样，于是你离开了，于是我也要适应，因为你走了时间不会停下来，于是我必须要抹干我的眼泪，必须要站起来，是一个“带着无奈，必须要让自己坚强”的感觉。
