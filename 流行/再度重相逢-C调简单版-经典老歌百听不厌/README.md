

**再度重相逢** 是中国台湾男歌手、音乐人伍佰作词、作曲并演唱的一首歌曲，收录于伍佰2003年12月5日发行的专辑《泪桥》中。

《 _再度重相逢_
》是一首温暖、深情，阐释了如何去爱——“简单爱你心所爱”的摇滚抒情哲理曲。意境通脱淡然、清新洒脱，如阳光下的一道清冽甘泉，沁人心脾、让人顿悟；如微风抚摸空悬天际的一抹彩虹，令人心驰神往。
