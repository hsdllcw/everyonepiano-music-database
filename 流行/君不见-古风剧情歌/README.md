

**君不见双手简谱** 和五线谱完全对应。

_君不见_ 是一首古风剧情歌，由艾辰/璟年/屿汐/麓七一同演绎。剧情阐述：我一直以为，有了青梅便会结了白头。 我一直以为，骑了竹马便会携手天涯。
我曾以为，我会立下显赫战功，封侯拜相，然后风风光光的将她迎娶归家。 可却忘了，世事难料。犹记得那年，她拿着红灯，说待君归来，非君不嫁。

歌词下方是 _君不见钢琴谱_ ，大家可以免费下载学习。
