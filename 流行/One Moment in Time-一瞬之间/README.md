_One Moment in Time_ 是1988年汉城奥运会美国队队歌，是众多歌迷心中最经典的奥运歌曲。此曲由Whitney
Houston于1998年演唱并发行，并于2000年收录于《The Greatest Hits》。  
  
惠特妮·休斯顿（Whitney
Houston）是屈指可数的全球流行天后之一。她是极具实力、充满魅力的超级巨星，在歌唱艺术的领域，她是至高无尚的巅峰。可惜的是，近些年来，惠特妮·休斯顿深受“毒”害，难以自拔，最终给自己也给广大歌迷造成了无法弥补的损失。其实她一直都很清醒自己的所作所为，她在一次接受访谈时就曾说过：“最大的恶魔是我自己。我是我最好的朋友，也是我最坏的敌人。”可惜，最终葬送自己的就是那个恶魔的自己。
