

**落叶归根** 是由王力宏倾情演唱的一首经典歌曲。

  
睽违一年半，在歌迷的千盼万待下，王力宏终于在2007年7月13日发行他的全新国语新专辑《改变自己》；这次首波主打歌" _落叶归根_
"受到拍电影"色戒"启发影响，除了有着小提琴陪衬优美旋律的复古曲风，富有深度及浓厚文学气息的歌词，更让人觉得跟以往的王力宏有很大的不一样！力宏说当初在创作这首歌的时候正好是他在拍电影的时候，入戏很深的他，坦言那段时间都是被他所演的文人青年角色"邝裕民"附身，骨子里都是留着邝裕民的血液，因此所想的所思考的方向都是受"邝裕民"影响，而且有时候下了戏因此所写出的歌词自然有一股文学味道；力宏笑说，邝裕民"上身"的那段时间，连爱听的音乐都是非常的"复古"！而新歌"落叶归根"写好的时候，力宏有拿给导演李安"试听"；力宏开心的说，导演很喜欢这首歌，听完后微笑点头，直说好听！

  
王力宏"落叶归根"MV大手笔请到曾入围奥斯卡金像奖的知名摄影师罗德里哥普雷多(RODRIGO
PRIETO)友情跨刀掌镜！罗德里哥是电影"色戒"的摄影师，曾经与李安导演合作过电影"断背山"的罗德里哥普雷多因拍摄此片而获奥斯卡金像奖提名；而他也曾为无数好莱坞巨星掌镜，如拍过布莱德比特的"火线交错"、安洁莉娜裘莉的"亚历山大大帝"，更拍摄过饶舌天王阿姆的半自传电影"街头痞子"，对于音乐与镜头的关系处理功力，可说炉火纯青，处理到恰到好处！而力宏这次因为拍电影而与罗德里哥因结缘，两人从对电影音乐到艺术的观念无一不谈，话题相同，成为很聊的来的好友，颇有"英雄惜英雄"的感觉！也因此力宏与罗德里哥提议可以合作拍之后新专辑MV
时，罗德里哥便二话不说爽快答应，也因此促成这段佳话！

  
落叶归根是一首很好听的歌曲。歌词下方是 _落叶归根钢琴谱_ ，希望大家喜欢。
