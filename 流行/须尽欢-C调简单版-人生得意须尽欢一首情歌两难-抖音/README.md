

抖音上有一首很好听的抒情歌曲，其中有句歌词是这样的“人生得意须尽欢，一首情歌两难”。据悉，这首歌曲叫做《 **须尽欢** 》，由宝藏男孩“渡”倾情演绎。

《 _须尽欢_
》仿佛是一位孤独的诗人，在情爱里孤独吟唱，“情爱两难，一别两宽，人生须尽欢”。史宾塞曾说过：“时间的限制，不仅仅是因为生命的短暂，更是因为人生观的变化”。不管世事风云变幻，守住一颗心，不管风雨风雨，依然淡然自若，抛弃了物欲的束缚，抛弃了心灵的羁绊，那么，幸福就在身边，快乐的人生也永不逝去。
