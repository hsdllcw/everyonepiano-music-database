

《 **望月** 》是中国香港流行乐男歌手张学友于1996年1月发行的一首经典单曲。歌曲写的是一个历经坎坷的人举杯望月的感慨。

“狼叫”“雪飘”说明环境很恶劣，也是对社会环境恶劣的描写。之后描述了人生旅程，情逝爱消，只身一人对月苦笑，竟是无奈。“人在变，兽与妖”比喻人心险恶，变化无常，表面坦诚相待，内心却不知在奸笑盘算什么。双手直举，像狼战斗时摇晃尾巴一样，不向天公命运屈服。情爱消散，只是俗世常态，不堪风雨洗礼。只有抗争的浪潮是无法被抹去的。由开始的对世事不公的质问和无奈，到后来向不公和命运的挑战，最后是看破红尘的一种境界。所以才会有向将进酒一样，人生苦短，不如及时享乐。在我看来，这首歌词作者是赋有诗意的，很像李白中后期的风格。这首词写的真的非常有深度。
