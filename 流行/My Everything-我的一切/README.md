

《My
everything》是李敏镐的个人单曲。李敏镐2006年以《秘密的校园》正式出道，2009年因饰演《韩版花样男子》中的具俊表红遍亚洲，2011年出演韩版《城市猎人》的男主角李润成人气飙升，演技也日益成熟。曾获韩国百想演技大赏最佳新人奖。
