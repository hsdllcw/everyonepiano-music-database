

**答案双手简谱** 和五线谱完全对应。

最近被抖音一个喷香水的短视频给迷住了，背景音乐是：爱就像蓝天白云晴空万里，这是什么歌呢？其实这首歌叫《 _答案_ 》,是由杨坤和郭采洁演唱的。

此曲运用R&B的慢板曲风，在调侃与哼唱之中，将对爱情的纠结与体会表现出来。爱情的形式纵然有很多种，而每一种存在故有其意义与道理。就如同誓言一样，或许某一天变成了谎言，而个中意义，只有当局者才能感受到。郭采洁和杨坤用味道、天气、美酒与咖啡等比喻爱情。将抽象的概念变得具象化，引发听众更多共鸣与感受。由此不难看出，爱情的变化仍旧是无规律可循的，但无奈之间却有属于自己的权衡标准，正如歌词所言“每个人有每个人不同的计算”。男女对唱的歌曲不胜枚举，然而杨坤和郭采洁的这一对组合则显得更加抢眼。将帅大叔和美萝莉mix会有奇妙的化学反应。杨坤标志性的沧桑嗓音与郭采洁甜美的歌喉相得益彰。当朗朗上口的旋律搭配当华语音乐市场日渐低迷的今日，相信这首《答案》有好成绩。
