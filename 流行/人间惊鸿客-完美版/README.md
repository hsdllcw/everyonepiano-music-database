

最近在抖音上有一首歌特别火，不仅受到了很多网友的喜爱，还被翻唱了很多版本，歌曲的中文歌词意思是：“叹人间一去风流唯少年，将白驹轻踏寻山海作诗篇，醉花间几回梦中忆风月”。

据悉，歌曲的名字是《 **人间惊鸿客**
》，由叶里原唱，在2020年11月04日发行，少年身姿翩若惊鸿，宛若游龙，行走人间引出无数跌宕故事，《人间惊鸿客》一个瑰丽的侠客故事。

_人间惊鸿客曲谱_ 由 **EOP** 大佬[
**M小冯哒**](https://www.everyonepiano.cn/user-174829.html)给大家编配，曲子有点难，初学《[
**魔鬼训练营**](/Sale.html)》的小伙伴可先行绕过哦！
