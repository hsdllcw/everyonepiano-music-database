

**桃花结双手简谱** 和五线谱完全对应。

_桃花结_ 是一首旋律略有忧伤的音乐歌曲，该歌曲由90后海归女歌手王诗安演唱，《桃花结》是由唐嫣、刘恺威、杨蓉、杨祐宁主演的大时代偶像巨制千金女贼的片尾曲。

歌词下方是 _桃花结钢琴谱_ ，希望大家喜欢。
