

**日月凌空**
是2007年央视涿州影视古装大戏《日月凌空》的主题曲。由姚贝娜演唱。《日月凌空》由四川作家吴因易编剧，刘晓庆主演，关于“一代女皇”武则天和巴蜀才女谢瑶环相处的故事，刘晓庆二度饰演武则天，从40岁演到82岁，剧本40集，拍成长达62集的电视剧。其他演员有焦恩俊、马晓伟、黄圣依、杨子。

  
歌词下方是 _日月凌空钢琴谱_ ，希望大家喜欢。
