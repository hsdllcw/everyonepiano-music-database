

**你好双手简谱** 和五线谱完全对应。

_你好_ 是初音未来所演唱的一首歌曲。初音未来（初音ミク/Hatsune Miku），是2007年8月31日由CRYPTON FUTURE
MEDIA以Yamaha的VOCALOID系列语音合成程序为基础开发的音源库，音源数据资料采样于日本声优藤田咲。

这首歌曲个人感觉写的是老是与学生之间的一些事情。早上看到老师，微笑，说你好等等，歌词非常简单，中文与日文混合，朗朗上口。曲调依旧非常轻松。弹奏这首曲子的时候还是要多注意速度，特别是学习电脑键盘钢琴的朋友。

歌词下方是 _你好钢琴谱_ ，大家可以免费下载学习。
