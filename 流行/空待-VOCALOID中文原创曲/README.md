

**空待**
是王朝于2013年4月18日投稿至bilibili的VOCALOID中文原创歌曲，由洛天依和王朝本人演唱。殿堂曲，截至2020年4月28日已有84.74万次观看，3.46万人收藏。

本曲由咏吟轩中著名组合朝南组（伯南和王朝）主要制作，其他参与制作的社团与组织有Days幻梦年华乐团、捂脸社、忆水寒工作室、平沙落雁团队等。本曲同时也是王朝第一首真正意义上的天依原创曲；B站投稿上用了一个很长很长的标题：王朝、洛天依、空待、咏吟轩、原创、中国风、对唱、DAYS幻梦年华乐团、闲人工作室，而从此之后王朝的稿件标题都是这种风格。

本曲主要讲述了一个凄美的爱情故事，以精致的歌词和优美的戏腔唱法获得了很多人的喜爱，并获得了许多翻唱。
