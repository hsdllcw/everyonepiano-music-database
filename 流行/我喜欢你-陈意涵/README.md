

**我喜欢你双手简谱** 和五线谱完全对应。

_我喜欢你_
是由陈意涵作词作曲的一首原创单曲。陈意涵不仅颜值在线，更是在演唱、创作、乐器演奏方面可圈可点。在《创造101》节目中，陈意涵以其活泼的台风和青春活力的外表给大家留下了深刻的印象，她也因此被称作为“全糖初恋脸”、“wink狙击手”。  
  
歌词下方是 _我喜欢你钢琴谱_ ，大家可以免费下载学习。
