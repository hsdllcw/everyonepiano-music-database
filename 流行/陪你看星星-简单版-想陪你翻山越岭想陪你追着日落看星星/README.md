

“想陪你翻山越岭，想陪你追着日落看星星”出自歌曲《 **陪你看星星**
》是由谢洽仑作词、作曲，陈子晴演唱的歌曲，发行于2021年6月12日。收录于同名专辑《陪你看星星》中。

“星星会暗，月亮会有一天找不到踪迹，即使没有遇见在银河，和你可以相遇的本身就很浪漫了。”
月亮虽然遥远，但某一刻真的照亮过我，如果喜欢的人能给你前进的动力，即使并不能够在一起也是值得的，那才是真正的喜欢，要知道有些人不属于自己，但是遇见了也弥足珍贵。希望每晚都会有一个人愿意陪你看星星，愿意陪你做任何事。
