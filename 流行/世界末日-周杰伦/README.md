

**世界末日** 是周杰伦演唱的一首受欢迎的歌曲之一。

  

周杰伦，中国台湾人，祖籍福建泉州，毕业于台北淡江中学，华语流行歌手、著名音乐人，2000年后亚洲流行乐坛最具革命性与指标性创作歌手，唱片亚洲总销量超过3100万张，有“亚洲流行天王”之称，其音乐突破原有亚洲音乐主题、形式，融合多元的音乐素材，开创了华语流行音乐“中国风”的先声，为亚洲流行乐坛翻开了新的一页。

  

歌词下面是 _世界末日钢琴谱_ ，喜欢的可以下载。

### 世界末日歌词：

想笑来伪装掉下的眼泪。

点点头承认自己会怕黑。

我只求能借一点的时间来陪。

你却连同情都不给。

想哭来试探自己麻痹了没。

全世界好像只有我疲惫。

无所谓反正难过就敷衍走一回。

但愿绝望和无奈远走高飞。

天灰灰会不会。

让我忘了你是谁。

夜越黑梦违背。

难追难回味。

我的世界将被摧毁。

也许事与愿违。

累不累睡不睡。

单影无人相依偎。

夜越黑梦违背。

有谁肯安慰。

我的世界将被摧毁。

也许颓废也是另一种美。
