

有些歌，能揭开你埋藏已久的伤疤；有些歌，能回味你人生路上的坎坷；有些歌，能在你累的时候给你前行的动力。而音阙诗听音乐社演唱的《 **空山新雨后**
》，就是这样一首曲子。歌曲由锦零演唱，假寐作词，殇小谨作曲。正式发行于2019年3月16日。

音乐就像人生的背景，在你年少时循环的某首歌曲，多年后走过某一个街角再度响起，熟悉的声音是否也曾将你带回那段岁月。
