_被风吹过的夏天_
是金莎和林俊杰演唱的一首歌曲，由冯欣慧作词，林俊杰作曲。收录在金莎2005年发行的专辑《空气》中，同时这首歌曲也是林俊杰2005年发行的专辑《编号89757》赠送的的预购单曲。2005年这首歌曲获得了第三届东南劲爆音乐榜颁奖典礼内地十大金曲奖。

这是一首甜蜜的对唱情歌。此曲一经推出就在各大排行榜上位居榜首，成为一首广为传唱的对唱情歌；而金莎更是凭借此曲为人所熟识。
