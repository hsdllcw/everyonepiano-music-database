《 _红豆_ 》是一首耳熟能详的歌曲，该曲在王菲的专辑《唱游》中是一首特色最弱的歌曲，但也是最红的一首歌曲。

歌曲的流行除了旋律的动听之外，林夕细腻、缠绵的笔锋也为歌曲增色不小；而王菲的声音听起来有些弥散，但是形散而神不散，特别是中间饱含的情感，流露自然，不愧为一首情歌的典范。《
_红豆_ 》可以说是“后王菲时代”最具流行性质且广为传唱的歌曲代表。
