_月出_
由音乐制作人徐梦圆作曲，中国内地女歌手双笙演唱的一首歌曲。歌名《月出》出自诗经•陈风。歌曲上部写的是将军战死沙场亡魂归来，得见朝思暮想的佳人，却已阴阳永隔不忍思量的场景。下部描写佳人孓然一身月下思君，此时将军亡魂吟咏起“陟彼高冈，我马玄黄”（诗经•卷耳），把故事推向高潮也是结局。
