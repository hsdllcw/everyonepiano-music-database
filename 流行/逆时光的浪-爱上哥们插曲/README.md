_逆时光的浪_ 是Bii毕书尽继《I Wanna
Say》后另一首对于自己内心深处最真实的告解的自创曲，收录于《势在必行3-心时代》专辑。同时，此曲也是浪漫爱情剧《爱上哥们》的插曲。

毕书尽（Bii \
필서진），1989年7月7日出生于韩国首尔，中国台湾籍流行男歌手。略显沙哑的嗓音，曲风也带点英式摇滚，毕书尽被不少粉丝亲切地笑称为“冒牌韩星”、“偶像包装里是一颗沧桑的心”；在每一张专辑制作中，毕书尽充分地表露出自己的创作才能，也让众多媒体音乐人对他的音乐路充满了期待。
