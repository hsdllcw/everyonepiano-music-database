

**阿婆说双手简谱** 和五线谱完全对应。

_阿婆说_ ，于2017年5月20日零点，由陈一发儿演唱的女声版上线。

此曲是独立音乐人暗杠的原创作品，创作于2016年10月。歌词部分由暗杠及暗杠的听众古道背棺人共同完成，演唱、作曲、编曲、器乐演奏、和音编配、录音混音等工作由暗杠独立完成并发行的录音版本于2017年2月11日首发。

颇有江南韵味的绵缓旋律，搭配触人心弦引人共鸣的歌词情景，融入暗杠温暖倾诉般的唱腔，该版本一经上线即获得广泛好评，云音乐评论迅速破万，广大听众纷纷在歌曲评论区留下自己和阿婆的故事，情真意切，感人至深。

歌词下方是 _阿婆说钢琴谱_ ，大家可以免费下载学习。
