

《 **粉色海洋** 》是周杰伦《最伟大的作品》专辑中收录的一首新歌，由周杰伦编曲、方文山填词，歌曲风格为Pop。整体风格非常甜美，有成为爆款的潜质。

值得一提的是，这首歌曲最开始的童声是由周杰伦的儿子Romeo献声。Romeo的声音虽然稚嫩，但是非常俏皮可爱。
