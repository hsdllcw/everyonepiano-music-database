

这首歌的歌词是翁倩玉的父亲翁炳荣在五十岁生日那天特别送给爱女的。翁炳荣曾是台湾中视第一任节目经理，他为爱女创作过很多脍炙人口的好歌。这首《祈祷》改编于1975年，是翁倩玉主演的电视剧《爱的旋风》的主题歌，台湾丽歌唱片公司出品。卓依婷、王杰和王韵婵、罗时丰等歌星曾翻唱此歌。
