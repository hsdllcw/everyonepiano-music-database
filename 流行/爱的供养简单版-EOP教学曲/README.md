

这首爱的供养比较简单，适合练习，属于EOP教学曲。上面的简谱和五线谱是完全对应的。

热门歌曲《爱的供养》为穿越清宫剧《宫锁心玉》、《宫锁珠帘》片头曲（主题曲）。由杨幂
或何晟铭&向鹰，何晟铭&杜淳（三个版本）演唱，曲风各有不同，深受粉丝的喜爱。随着《宫锁心玉》的热播，该曲也越来越多地为大家所传唱，分别融入了对亲情，爱情，皇权的诠释。
