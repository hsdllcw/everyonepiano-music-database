

《 **相思的债** 》是由绿柳作词，陈伟作曲，陈瑞演唱的歌曲，发行于2009年1月20日。

有时候，喜欢一首歌，不是因为歌唱的有多好听，而是因为歌词写的像自己。

素弦声断 泪湿香腮  
一生徘徊 走不出这相思的苦海  
你闯进了我的心里 真情灌溉  
我却为你万千感慨 默默等待

寥寥几句就唱出了多少人心中的相思和无奈！相思是回忆过去的甘与甜，是缅怀曾经的喜与乐，是思忆以往的痛与苦，是祭奠过往的悲凉与不公允。

《 _相思的债_ 》用简单朴实的歌词，让我们回想起了那些关于这首歌的故事。而你想到了什么呢？
