简约的钢琴，素雅的文字，张碧晨的声音，不施粉黛，轻轻落下，用最真诚的歌声，把这些歌，唱给了自己生命中最重要的亲人。

生在天地之间，浮华一生皆如一梦，时光终要匆匆逝去，而我们都要在无声的岁月中渐渐苍老。人生，就是一路减法的过程，伴着哭声降临世间的开始，随着时间的成长与流逝不得不一一告别生命中的至亲至爱。“见与不见/不可再见/才叫人间”，淡淡安静的旋律流淌着张碧晨对于生命的体验，清脆的琴键声敲击出词藻里深深的情意。“生命依旧向前/你在过的点点/伴我每一天”，撕撕裂裂充满质感的声音唱出了内心炽热的想念，随着岁月辗转，儿时的欢声笑语沦为脑海的点点回忆。张碧晨希望通过这首自己创作的音乐作品《见与不见》，让更多听众明白时间的珍贵，从而珍爱身边每一个举足轻重的人，珍惜当下每一个正在行进着的『你在过的点点』，不负自己，不负余生。
