

**相守双手简谱** 和五线谱完全对应。

_相守_
是栗先达作词并演唱的歌曲，于2016年4月10日发行。这首歌温馨恬淡，相对于《知足》也是有过之无不及。非常坦然安逸的旋律，相信大多数人一听就魔性了，不要错过。

相守是指亲人之间相互厮守、相互守候的生活形式。由相识、相知、相恋到相守，是多久远的过程呢？此生若能寻一人相守到白头，就随意择一城终老呗。

歌词下方是 _相守钢琴谱_ ，希望大家喜欢。
