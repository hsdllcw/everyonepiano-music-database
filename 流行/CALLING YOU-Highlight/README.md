Entertainment经纪公司。2017年2月24日，原BEAST五位成员组为HIGHLIGHT，正式展开演艺活动。2017年3月20日，以HIGHLIGHT组合推出首张迷你专辑出道。2017年5月29日首张迷你专辑的改版专辑《CALLING
