

《 **烟花三月**
》是2009年歌手童丽演唱的歌曲。童丽柔美的歌声把江南烟雨景色表现的淋漓尽致。就好比一幅幅美妙的图画呈现在我们的眼帘，如诗如画却又带有淡淡思愁。

童丽的作品大多都有一些对凄凉爱情诠释的诗词乃至一些女人独自彷徨失落的场景，或许这正是大家喜欢她作品的原由。听着童丽的歌曲，感受着诗词中的史册，让人醉在其中，深深感受到了诗与歌带给大家的快乐。
