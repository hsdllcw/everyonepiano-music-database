

《 **飞羽**
》由业界顶尖音乐制作团队为《梦幻诛仙》量身打造而成。关于其演唱者，经多方查证，此曲有两个版本：一个是崔琰和李东燕的官方发布版，另一个是崔琰和董贞的前期试录版。

本曲打破了中国风三古三新（古辞赋、古文化、古旋律、新唱法、新编曲、新概念）的固化模式，在作词部分融入了《梦幻诛仙》故事背景，采用了更为意境婉转的作词方式撰写，赋予了《飞羽》一段绝美的爱情故事；在作曲部分，《
_飞羽_ 》更充分结合了《梦幻诛仙》浓郁的仙侠风格，一曲三叹，曲意悠长，并突出了《梦幻诛仙》中使用飞仙系统可实现无限翱翔的畅快自由。
