

**追梦赤子心**
原为中国GALA乐队的代表作，此后该歌曲被选为2013快乐男声的主题曲。收录于2013快乐男声全国10强与复活者共同打造的合辑《追梦敢不敢》中。新版《追梦赤子心》由2013快乐男声全国十强选手共同演绎，并由知名导演宁浩执导的主题曲MV，呈现出了不一样的风格。

《 _追梦赤子心_ 》中振奋人心的歌词、高亢的音调、嘶吼的声音，以积极向上的基调、青春激昂的姿态呈现给各位观众，曲调之中多了一份高亢、摇滚的感觉。
