

新生代实力歌手卓舒晨的歌曲《 **走着走着花就开了** 》，歌词简单细腻，契合着甜蜜爱情的主题，就像甜甜的白巧克力一样，每一句都让你感受到那份纯净真诚的心。

走着走着就散了，是指那些不懂珍惜，不懂包容对方的人，既然刚开始爱得如此深，但到最后还是只有散了。

而这首《 _走着走着花就开了_
》，是指两个相爱的人始终相信随着时间流逝，悲伤可以忘记，爱恨可以放下，无论怎样的寒凉，都会在春天到来时终结，只要心存美好，时光便可不老，心若释然，人间安好，待到峰回路转处，便是岁月花开时，愿时光不老，静等花开。
