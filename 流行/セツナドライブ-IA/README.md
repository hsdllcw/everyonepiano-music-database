_セツナドライブ_ 是日本VOCALOID3的虚拟歌姬IA的歌曲。

IA原声声源提供者为Lia。充满通透性口气清晰的高音域是其特色，在摇滚和节奏性舞曲较能表现其速度感，是使各式各样的声音成为可能的虚拟歌姬软件。 IA
