

繁华唱遍是纳兰寻风于2019年2月4日投稿至bilibili的VOCALOID中文原创歌曲，由乐正绫演唱。传说曲，截至现在已有298.70万次观看，14.32万人收藏。

本曲首次登场于哔哩哔哩2019拜年祭。昆曲的曲风富有江南水乡气息，歌词描写了一位自幼时起命运便与戏曲息息相关的阿婆形象，借过往和当下的对照，展现出历史的沧桑感，暗含着戏曲等传统文化之传承的命题。

me!》并列成为第211期改版后第十八首破百万分的歌曲。
