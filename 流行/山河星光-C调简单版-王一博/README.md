

王一博国庆献礼《 **山河星光**
》，是将山河写成一首歌，以悠扬动听的青春旋律，生动优美的画面语言，讲述温暖动人的家国情怀，在佳节之际，献礼伟大事业、伟大人民、伟大祖国。

此曲由中国青年报•中青在线联合有关机构策划出品，由林乔、刘恩汛作词，唐轶作曲，林乔担纲制作人。吉他和西洋弦乐组成的旋律，悠扬律动富有青春气息，诗意的歌词充满画面感，构建出宏大的山河场景和温情视角，家国情怀的主题也在起伏悠扬的旋律中得到彰显。

而王一博的音色一如既往充满少年青春特质，可柔可刚可甜可盐，在清澈又动人的歌声中，云海，白鸽，朝阳，微风，花香，都与万家灯火无缝连接；随着旋律和唱词的递进，绝美画卷凝聚成流金岁月，英雄的轮廓和时代号角，都在中华民族的辽阔伟业中闪耀光芒，于是又开始用歌声追索历史长河，直至梦想照进现实，“盛世的歌在新的百年传唱”。
