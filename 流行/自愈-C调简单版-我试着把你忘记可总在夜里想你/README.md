

“我试着把你忘记可总在夜里想你，悲伤的情绪随时钟转不停”出自歌曲《 **自愈**
》。《自愈》是由穆哲熙作词、作曲，穆哲熙演唱的歌曲，发行于2021年4月14日。

真的所有感情受的伤只能自愈。而时间是治愈感情的良药，很多朋友在感情受伤后都希望可以尽快的从痛苦中走出来，可有的时候越想走出来，越难以让自己走出来，最后只能让留给时间来让自己慢慢疗伤。
