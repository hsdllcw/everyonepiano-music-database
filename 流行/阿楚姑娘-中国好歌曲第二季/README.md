

**阿楚姑娘** ，《中国好歌曲第二季》西安的唱作人梁凡创作的一首歌曲，得到评审好评并加入刘欢导师的原创大碟。

梁凡的声音很质朴，很干净，所谓思乡之情，深情而直扣心门。如青山涌泉，深山流水，怀揣片片翠绿散发缕缕幽香。隐约在远方有一个叫阿楚的姑娘，在月光下等着她的心上人归来。故乡，大概就是年少时拼命想逃离的地方吧。大雪纷飞里西出阳关无故人的景，风吹过的口红欲盖弥彰。

后来，袁娅维在《歌手》舞台上重新演绎了此曲，充沛的感情加上饱满的嗓音，直击听众灵魂深处，当场表演艳惊四座，也让《 _阿楚姑娘_ 》加入了众多听众的歌单。
