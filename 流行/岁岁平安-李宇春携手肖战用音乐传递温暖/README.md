在音乐部分，《 _岁岁平安_
》用了民歌的方式，来组织旋律，推进情感，让人想起每个人记忆中的旧日时光，浮生千山。最后以来自古诗词的“我住长江头，君住长江尾，同根同源，共饮一江水”作为高潮，点出赤子衷怀。
