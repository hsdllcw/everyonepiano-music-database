

最近，抖音上又火了一首歌，歌词为只是你太粗心大意忽略了我的感受的歌曲，小编在刷抖音的时候经常听到有博主翻唱这首歌。这首歌叫做 **心安理得**
，由王天戈演唱，王天戈的嗓音很独特。

王天戈用自己独特的嗓音完全的唱出了这首歌想要表达的忧郁感觉，这首歌的歌词主要表达的意思是对待没有结果或者说是无疾而终的感情的留恋和放手后的释然，算是一首伤感情歌吧！

当一个人熬过了最艰难的时候，就不想再去寻找任何依靠。沉默不是因为词穷，而是因为心空。不去怨，不去恨，淡然一切，往事如烟。
现实有多残酷，你就该有多少坚强！放弃不是因为输了，而是因为懂了， 走了这么久， 发现唯一靠得住的是自己。
