《 _一定会_
》——全世界最渴望得到的肯定句。因为疫情瞬息万变的世界状态，曾经以为的理所当然，却可以在瞬间充满不确定性，这些都让我们内心的不安逐渐侵蚀着寻找亮光的动力。面对这些不确定因素时，我们会怎么做呢？歌词给出了回应“暂时收好羽毛待飞/到时我一定会如期而至陪你哪都无所谓/一定会一定有我陪你飞”，伴随着节奏感极强的旋律，林俊杰用丝滑的高音呐喊着：相信这场暴风雨过后一定一定会有更美的下一段故事。
