

时代少年团新EP《乌托邦lll • 侠》的先行曲《 **绝配** 》于2022年7月25日在各大音乐平台上线。音源发布之后，热度直线上升，瞬间成为热门歌曲。

歌曲描述了朝气蓬勃的年纪里最纯粹最真实的友情，展现了夏日里肆意青春、明快少年的形象。虽然有时候会闹脾气，但会立刻和好，因为他们是朋友，在一起就是完美的绝配。

MV再现了校园生活中朋友相处的小小细节，记录下课桌前、书本后、球场上、篮筐边多姿多彩的画面。看同一本书，上同一节课，听同一首歌，一起长大，搞怪出糗也有不言而喻的欢乐。

《 _绝配_ 》是属于时代少年团的故事，也让每一位听众都能共鸣有人作伴的青春。 在你心中，属于绝配的瞬间，又是谁在相伴呢？
