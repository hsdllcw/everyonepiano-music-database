

《 **青いベンチ** 》 是日本组合テゴマス（Tegomass）的名曲。是Tegomass发行的第四张单曲。  
  
テゴマス（Tegomass），NEWS成员手越佑也和增田贵久组成组合Tegomass。Tegomass是取两位成员名字Tegoshi和Masuda的前一个单词组合而成。两位都是日本johnnys旗下的成员。  
  
两个人在Jr时代就被喜多川就音质和唱功夸奖过，所以两个人一起唱歌的机会增多。组成NEWS后，在专辑「touch」里收录的「ずっと」（一直）和在DVD「NEWS
NIPPON 0304」中收录的「ありがとう・今」等都是他们的原创作品，两个人一直作为重视演唱的组合活动。  
  
2006年11月，作为「Tegomass」，在瑞典发行了单曲『Miso
Soup』（味增汤）。这是杰尼斯事务所首次在欧洲出道的组合。这是因为NEWS的一些歌曲，比如「红く燃ゆる太阳」「サヤエンドウ」「梦の数だけ爱が生まれる」是由瑞典的作曲家提供，并在瑞典国内反响良好，而且考虑到了两人的唱功，做出的决定。2006年12月，在日本出道，和NEWS的活动并行。  
  
歌词下方提供了 _青いベンチ钢琴谱_ ，欢迎大家使用。
