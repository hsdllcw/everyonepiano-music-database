

《 **领悟**
》这首歌收录于辛晓琪1994年同名专辑《领悟》，由李宗盛制作，这首歌还有李宗盛、林忆莲、周华健、杨宗纬、郁可唯、关喆、常石磊、张杰等翻唱版本。

  

近来《 _领悟_ 》这首歌不断在中国好声音，声动亚洲等舞台上被关喆，黄中原，孙伯纶等选手反复唱响。在《我是歌手》当中更是被辛晓琪再度完美演绎。

  

出道多年的辛晓琪凭借《领悟》《味道》《走过》等经典情歌陪伴了一代人的成长，她的歌曲透过独有的内敛歌声和真挚情感，诠释着世间爱情。

  

歌词的下方还提供了 _领悟钢琴谱_ ，欢迎感兴趣的朋友下载使用。
