

近段时间小编在刷抖音时，时常能刷到一首非常好听唯美的歌曲，歌曲是粤语版的，很多人可能听不懂，其中第一句歌词大意是“可不可不要这么样徘徊在目光内，你会察觉到我根本寂寞难耐”，据悉，此曲叫做《
**暗里着迷** 》，由刘德华演唱。歌声优美，给人一种扣人心弦的感觉，不少网友被曲词深深吸引。

《 _暗里着迷_
》被华仔称为是对自己半生非常重要的歌，在歌词的字里行间透露着对某个人的深深的无言的爱恋，也许是造化弄人，这份感情未能公开，有情人难成眷属，只能暗自着迷。
