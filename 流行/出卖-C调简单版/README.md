

《 **出卖**
》是那英演唱的一首歌曲，由林夕作词，周传雄作曲，收录到那英专辑《心酸的浪漫》中。歌词无动于衷却表达了百转千回的善感，是一首婉转、舒缓的慢板抒情歌曲。

歌词的大意是，他从来不拒绝，所以自己以为一起就是爱，原来自己在意的比生命还重要的爱情其实什么都不是。一直的付出，他的世界就是自己生活的全部，最终自己放弃一切去努力追求的原来只是自己的自以为是。感情像个闹钟，按一下就停，那么多年得意忘形闭起了眼睛。到最后
一切的生活都变了，那些以为的一切都不存在 ，爱的那个人原来从来没有爱自己。
