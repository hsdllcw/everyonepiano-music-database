

**漩涡**
是2000年推出的粤语歌曲，由彭羚和黄耀明演唱，歌曲旋律另类，特殊的音效非常惊艳。《漩涡》是香港乐坛“垃圾五部曲”的第三首歌。在2001年1月23日，获得第1届华语流行乐传媒大奖十大华语歌曲的奖项。

《 _漩涡_ 》里的两人，你情我浓，打得火热。既然两人互相爱着对方，又为什么要折堕呢？

这大概就是因为两人的爱情，是见不得光的。

在《漩涡》里的二人，为什么恋情见不得光，黄伟文并没有刻意点出来，他要写的，只是两个人过得今晚，仿佛就没有明天。对于男人而言，这是一场“红颜祸水”；对于女人而言，这是“贪欢惹的祸”——反正，对于在情感漩涡里的二人，今晚上发生的事，绝对不是什么好事。

但是两人都无法控制自己的欲望。女人唱出了“来沉没在我的深处吧”——这赤裸裸的描述出情欲的细节，仿佛可以让听众幻想出一个女人抱着男人的头，慢慢往身下延伸的呻吟……再加上“来拥抱着我，形成漩涡，扭曲那万有引力倒海翻波”的大量“动作”描写——特别是那句“倒海翻波”，将这对男女充满了欲望的狂野一夜描述得淋漓尽致。

所谓优雅的情色，大概也就是这样的程度了。黄伟文要写的，是两人既然明知没有明天，于是就将今晚这仅有的时间用到尽的那种疯狂和沉沦。沉溺在情欲里的两人，只希望“直到这世界彻底瘫痪，剩下自己在游玩”。

可惜世界不会贪欢，世上万物也没法向心公转。女人逾越了理性，超过自然；瞒住了上帝，让男人到身边；而男人也明知这是一场红颜祸水。所以在这一夜后，陷入漩涡的两人又会及时抽身，回归到属于自己的世界里。他们最终还是会重新走到理性的那一面，形成的漩涡，最终也只能静悄悄的沉没湖底，欣赏月圆。
