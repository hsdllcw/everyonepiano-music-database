_面影草_
是日本Rejet公司开发的PC平台乙女游戏剑为君舞（剣が君）中的曲子。剑为君舞发售于2013年12月19日，内容主题为恋爱，玩家人数为单人。且一共有六位可攻略角色，每个人物路线分别为君线和剑线，君线为和魂线和幸魂线，而剑线为荒魂线和奇魂线。

歌词下方是 _面影草钢琴谱_ ，大家可以免费下载学习。
