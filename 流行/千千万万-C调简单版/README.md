

深海鱼子酱的这首《 **千千万万**
》，可谓一夜爆红。每个人都从这首歌里听出了自己的故事。刚开始的时候，还会有些纳闷，为什么要重复千千万万万万千千。后来才知道，“千千万万无法诉说的思念，犹如落日一般，余霞成绮。”

记得宫崎峻曾经说过：当陪你的那个人要下车时，即使你再不舍，也要心存感激，挥手告别，每个人的故事的开头都是具极温柔的，但往往故事的结尾，都配不上整个开头，记住，每个意难平的结果，都是我们最好的结局。你的存在是我的幸运，你的陪伴是我的荣幸，你的离开是我的千千万万。心怀感恩，每一次美好的相遇都是最美的回忆，即使结局让我们泪流满面，那也是一种成长。
