

《 **天气的错** 》是由郑伊健演唱的歌曲，出自专辑《初夏之恋（粤）》。

虽然天气没有我们想象中的那么好，但是我们依然要学着去适应。歌名看似是在说天气，但其实包含着人生的哲理。

在生活中，我们都会或多或少遇到不如意的地方，不可能每件事情都能够称心如意，但是我们应该学会自我调整，学着去和解，学着告诉自己怎么样才能够克服这些不称心的地方。
