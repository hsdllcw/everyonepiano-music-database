_玻璃之情_ 是香港著名演员、歌手和音乐人张国荣的歌曲。收录于2003年7月8日发行的音乐专辑《一切随风》中。这首歌由林夕作词、张国荣作曲。

张国荣在毛舜筠的访谈中说过，他唱歌前都要给这首歌编一个故事，唱的时候脑海中浮现故事中的画面，这种画面感会使他唱得更加投入。用他的话说就是，他的每一首歌都投入了好多感情在里面。我想这才是他能把歌诠释得动人的原因吧。再好的唱功、再完美的声线，也抵不过如此专注的声音。
