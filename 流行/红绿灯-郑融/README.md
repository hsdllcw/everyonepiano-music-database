

**红绿灯** 是香港女歌手郑融的曲目，收录在在其专辑Honey中。此曲推出后获各界一致好评，成为郑融到目前为止唯一一首四台冠军歌，同时令到郑融人气急升。

  
《 _红绿灯_
》在年尾乐坛颁奖礼横扫四台金曲奖，分别夺得新城劲爆颁奖礼新城劲爆歌曲、叱咤乐坛流行榜颁奖典礼专业推介十大歌曲、十大劲歌金曲颁奖典礼金曲奖及十大中文金曲颁奖音乐会十大中文金曲，成为香港乐坛金曲代表作之一。

  
歌词下方是 _红绿灯钢琴谱_ ，希望大家喜欢。
