中华民族千年传承下的二十四节气，每一个传奇都轰轰烈烈。为此，音阙诗听写了二十四首歌，来提醒人们节气的到来。里面记载着二十四个发生在节气里的故事。而《大寒》所记述的是斩尽大寒冻雪三尺/弥留之际/回望一生/看破一切/尘归尘土归土。
