

**鸿雁** ，呼斯楞版权作品、代表作。由额尔古纳乐队演唱，是电视剧《东归英雄传》的插曲，歌曲浑厚悠远，深具蒙古族音乐特色。歌曲《 _鸿雁_
》乌拉特民歌，著名词作人吕燕卫先生作词、音乐大师张宏光编曲，呼斯楞演唱。

  
当我们听到呼斯楞清澈的嗓音演绎此曲，不难产生共鸣，他的歌声里有你的乡愁，有你的成长，于是家乡就成了每个人心底最柔软最美好的缱绻。同时，这也是一个寄情于草原的男人自语。一个人，一杯酒，独对苍天，想一想曾经的过往，用力的遥望未来，那里是不是真的有彩虹挂在天堂。悠远蜿蜒，直抵内心。

  
歌词下方是 _鸿雁钢琴谱_ ，希望大家喜欢。
