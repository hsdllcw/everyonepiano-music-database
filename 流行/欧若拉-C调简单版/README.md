

**欧若拉**
为中国台湾女歌手张韶涵演唱的一首流行歌曲。歌曲与歌词水乳交融，配合得密丝合缝，最大化地传递出了创作者想表达的情感。深具异国风情的编曲，加上作词家施立别出心裁的歌词，呈现出张韶涵直接的声音穿透力，让听者仿佛感受到声音的光芒。

2005年该歌曲获得第五届百事音乐风云榜十大金曲奖；2006年先后获得MusicRadio中国TOP排行榜年度金曲和HITO流行音乐榜颁奖典礼十大华语歌曲等奖项。2008年9月张韶涵在中央电视台中秋晚会上演唱该歌曲。
