_我们都一样_
由苟庆、曹轩宾作词、曹轩宾作曲、沙维琪编曲、张杰演唱，收录在张杰2008年发行的《明天过后》专辑中。2008年7月26日在张杰杭州个人演唱会上曝光首唱。2012年重新混录后期制作后收录进当年发行的《One
Chance 张杰新歌+精选》专辑中。  
  
此曲虽然朴实但是唱出人内心最脆弱也最温柔的那一方角落，在不经意间、霸道及不容妥协地牵扯出我们的一些封存回忆、过往，及我们共同的心事。因此，当我第一次听到它时，我以为，被如今所有众多音乐湮灭了平庸及淡然的今天，不会再有悸动，可是，没想到，那依旧灼热及跳动的心，还会在一刹那间灌满了清澈及鼓励的感动。  
  
歌词下方是 _我们都一样钢琴谱_ ，希望大家喜欢。
