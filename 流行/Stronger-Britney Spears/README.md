

**Stronger** 是美国女歌手Britney Spears第2张录音室专辑《爱的再告白》（Oops!... I Did It
Again）中的一首歌曲，于2000年11月13日作为专辑的第3支单曲发行。

  
歌词下方是 _Stronger钢琴谱_ ，希望大家喜欢。
