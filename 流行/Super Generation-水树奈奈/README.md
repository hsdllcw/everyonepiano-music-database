

**Super Generation**
是水树奈奈第13张单曲CD。2006年1月18日发售，商品编号KICM-1156。在Oricon公信榜周排行榜上，最高排行第6位。单曲“SUPER
GENERATION”是水树奈奈第一次作曲。

  
另外， _Super Generation_ 也是朝日系综艺节目《やぐちひとり》片尾曲。由藤田淳平（Elements Garden）编曲。

  
歌词下方是 _Super Generation钢琴谱_ ，希望大家喜欢。
