

**太委屈**
是台湾艺人陶晶莹的经典代表作，讲述了情人出轨时的委屈心情。这首抒情强打，让你重见最犀利的陶子，最女人的心情，唱出痛心无奈的情殇觉醒，是KTV必练指定曲。

陶子诠释的方式较为内敛，情感投入颇深，将个中细微的情绪掌握得极佳，很让人感动。

2001年《 _太委屈_ 》获得第一届全球华语歌曲排行榜二十大金曲。
