虽然这首歌在李宇春的歌曲里算不上最出色，但歌词和曲调都很朴实，曲调舒缓、轻柔，作为公益歌曲，李宇春朴实的感情和演绎，让听众们有了更多共鸣。歌曲鼓励了在困境中的人们，使人们能鼓起勇气坚强面对困难。该曲也受到了当年各大校园的追捧。
