_Don't Wanna Know_ 是美国摇滚乐队Maroon 5与Kendrick Lamar合作的新歌，于2016年10月11日发行。

魔力红（Maroon
5）是成立于1994年的美国摇滚乐队，由亚当·莱文、詹姆斯·瓦伦汀、杰西·卡麦可、米基·麦登和莱恩·度赛克组成。2011年9月1日，魔力红与克里斯蒂娜·阿奎莱拉合作的曲目《Moves
Like Jagger》获得该周公告牌Hot 100单曲榜榜首位置。
