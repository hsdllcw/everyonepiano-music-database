《 _青花瓷_ 》是方文山作词，周杰伦作曲并演唱的歌曲，收录于2007年11月2日周杰伦制作发行音乐专辑《我很忙》中。  
  
2008年2月6日周杰伦在中央电视台春节联欢晚会上演唱该歌曲。2008年7月5日该歌曲在第19届台湾金曲奖颁奖典礼上获得年度最佳歌曲奖、年度最佳作词奖和年度最佳作曲奖三项奖项。  
  
论意境，《 _青花瓷_
》宛然一出烟雨朦胧的江南水墨山水，水云萌动之间依稀可见伊人白衣素袂裙带纷飞；论词句，《青花瓷》却是一幅笔端蕴秀临窗写就的素心笺，走笔曲折只因心似双丝网，中有千千结；论曲调，《青花瓷》仿佛微风中静静流淌石上的山泉溪涧，清泠透亮而又蜿蜒回环多有不尽之意；这三者叠加至一处，《青花瓷》一曲正如其名，恰似那“自顾自美丽”的青瓷极品，洗尽铅华，古朴典雅，清新流畅。古筝撩拨，牙板清脆，琵琶淙淙，《青花瓷》的中国风分外动人。周杰伦用“素胚”“仕女”“汉隶”等系列词汇描摹了传世青花瓷的风采，周杰伦的唱腔柔情而古朴，略带江南戏曲的雏形，绝妙填词配复古音乐，构成了一阕R&B佳作。  
  
《青花瓷》曲调温柔委婉、淡雅脱俗，以至于其脑海中浮现的全是烟雨江南的画面。即使是普通人，听这首曲子也可以想像得到烟雨下的江南美景吧。真的是一首很美，很有画面和意境的曲子。
