

《 **忘记你我做不到**
》在台湾，香港，新加坡，马来西亚等地都取得了非常炫目的销量。在发行此张唱片后，传媒亦有称张学友为华语乐坛的销量保证，乐坛前辈谭咏麟更是表示张学友是香港乐坛的良心。

《 _忘记你我做不到_ 》这首歌放在第三首，可能学友和欧丁玉也觉得这首安安静静的小调的确是上品，对它抱有期望。
