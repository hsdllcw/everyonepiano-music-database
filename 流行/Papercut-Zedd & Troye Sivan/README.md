

**Papercut双手简谱** 和五线谱完全对应。

_Papercut_ 是电音舞曲制作人Zedd和南非裔澳大利亚人Troye Sivan一同合作的单曲。且该曲收录在Zedd第二张录音室专辑True
Colors中，于2015年5月19日发布。

DJ Zedd（安东•斯拉夫斯基），俄罗斯和德国混血儿，1989年9月2日出生于俄罗斯东部，其艺名Zedd。他主要制作的音乐类型是Electro
House。其最著名的作品是与Foxes合作的《Clarity》，这支单曲在公告牌百强单曲榜上最高登上第8位，并获得美国夜店舞曲榜的冠军。捷德还凭借这首单曲获得了第56届格莱美奖“最佳舞曲唱片”的奖项。

歌词下方是 _Papercut钢琴谱_ ，希望大家喜欢。
