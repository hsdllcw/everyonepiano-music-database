_方圆几里_
是薛之谦填词、谱曲和演唱的国语歌曲。同时也是爱情微电影《最熟悉的陌生人》主题曲。为了能与微电影剧情丝丝入扣，薛之谦将自己代入电影中的角色去寻求灵感。尽管薛之谦创作的时候很辛苦，经常推翻重来。但是经过闭关一个半月苦思，他最终创作完成了这首歌曲。

这首歌除了表现微电影中人物的内心起伏，与情节主题融为一体。同时也是薛之谦献给当今默默支持自己，一路陪伴在自己身边的亲人、朋友、长辈、粉丝的专属独白。

歌曲还有一个吉他版，电影原声带版和吉他版都收录在薛之谦2013年发行的专辑《意外》中。

歌词下方是 _方圆几里钢琴谱_ ，大家可以免费下载学习。
