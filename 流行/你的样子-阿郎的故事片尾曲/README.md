_你的样子_
是歌手罗大佑演唱的一首歌曲，收录在专辑《爱人同志》中，是电影《阿郎的故事》国语版的片尾曲，很经典的电影，很经典的歌曲。它的粤语版是《也许不易》，由李健达填词演唱。

罗大佑，1954年7月20日出生于台湾台北，是台湾省苗栗县的客家人，祖籍广东梅州市区梅县区，台湾地区创作歌手、音乐人、纵贯线乐队成员之一。
