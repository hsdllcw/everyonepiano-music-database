_前世情人_
是周杰伦演唱的歌曲，其单曲MV由周杰伦与女儿共同创作，“有个女孩叫Hathaway，在她四个月大时弹了一段旋律，我引以为傲，她...是我女儿”，周杰伦将女儿比作前世情人，透露出周杰伦初为人父的心路历程。

歌词下方是 _前世情人_ 钢琴谱，希望大家喜欢。
