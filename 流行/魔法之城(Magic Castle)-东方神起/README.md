

**魔法之城** (Magic Castle)是韩国的二人组合더_클래식（The
Classic）的出道单曲，于1994年6月30日发行。由组合成员创作并演唱。共有两个版本（儿童版、共同版）。为韩国经典曲目。  
  
《 _魔法之城_ 》是韩剧《想你》插曲（2012年），朴有天在2013年湖南台元宵喜乐会上演唱过这首歌。  
  
歌词的下方还提供了 _魔法之城钢琴谱_ ，欢迎感兴趣朋友下载使用。
