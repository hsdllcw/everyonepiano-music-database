《 _芒种_
》凭借其洗脑的旋律爆红网络。歌曲发行于2019年6月6日，来自于音阙诗听音乐社，原唱是赵方婧。因其特色的旋律和独具意境的歌词，而被很多明星和艺人所传唱。

此曲将传统民乐与流行电音相融合，婉转的女声真假音切换自如。《芒种》一开头跳跃轻盈的女声吟唱十分抓耳，而间奏部分一段小提琴与古筝的对话，更是为作品增色不少。创作者在欧美日韩文化影响下的流行音乐之中，融入民乐、古乐等中国元素，使其更符合浸润于传统诗词古典文化的中国人审美。歌曲浓郁的中国风旋律十分洗脑，令人忍不住跟着哼唱。
