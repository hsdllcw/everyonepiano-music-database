

抖音“终于你我的爱埋葬在，哭泣的站台”出自歌曲《 **哭泣站台** 》。《哭泣站台》由王小帅演唱，于2021年11月10日发行。

在车站里离别的人们，最痛苦的不一定是送的人，也不一定是别的人，而是强迫自己先转身的人，感情亦是如此。

《 _哭泣站台_
》词曲均由王小帅独立完成，他一如以往温柔且清新的声线表达、在编曲上也加入场景化的元素使得歌曲更加饱满，更能烘托出离别的氛围感。送走的不只是“你”，还有一段一去不复返的青春和爱情，但也恰恰是这个站台，让我们即将失去的爱情，在这个时刻来一场仪式感告别。
