相信大家在浏览抖音的时候都刷到过这段治愈音乐。据悉，此曲叫做《 **风吹一夏**
》，由DP龙猪、Swei水、Rays陈袁演唱，发行于2021年6月10日。愿大家借着夏天的名义，把爱恋变成热恋BGM。

《风吹一夏》从来都不是夏日限定，存在本身便是浪漫的无限复刻。循环于所有季节，藏匿在每个角落。

歌词下方是 _风吹一夏钢琴谱_ ，大家可以免费下载学习。
