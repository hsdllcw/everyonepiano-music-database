《 **等风停**
》是罗云熙首次与秦昊合作的作品，整首歌曲的旋律印刻着秦昊式温柔的音乐表达，与罗云熙的柔和唱腔完美契合。作为第二张个人专辑的首支主打曲，温暖中略带感慨的旋律承载着饱满厚重的音乐情绪，为这首歌带来了独具一格的故事感。

《 _等风停_
》在等风，亦在等风停。人生悲喜无常。总有些等待，一些别离。你若来，我就是陪你飞翔的翅膀，你若走，我就你背后永远最暖的光。不管何时，累了回头望，我终会在那里。
