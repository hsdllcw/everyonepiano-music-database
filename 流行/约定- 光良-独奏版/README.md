

**约定双手简谱** 和五线谱完全对应，是EOP教学曲。

_约定_ 是由光良与种子音乐合作推出全新创作纪录辑。

2006年，光良以感恩的心情推出这一张集合了光良创作经典的演唱演奏全新纪录辑“约定”，集合了他创作以来的各种情感，这些歌不仅仅是代表着光良的成长，也代表着我们与光良一起经历成长的岁月。

以“ _约定_
”为名，这张创作纪录辑不但是光良送给我们的礼物，也是光良送给自己的礼物，藉着这些歌曲与这些回忆，往后回顾，向前展望，约定每一次的感动，都是光良。光良说，“这一路上我持续创作，持续唱歌给大家听，谢谢一路上大家的肯定与认同，我还会继续创作，继续写好听的歌，继续感动每一个人。我要大家陪我一起走下去。这是我和大家永远的‘约定’！”
