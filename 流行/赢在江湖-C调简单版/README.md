

**赢在江湖**
是乐乐作曲、姜鹏演唱的一首歌曲，发行于2018年5月14日。说到底，江湖之大，沧海高山，自由自在，也难免“人在江湖，身不由已”，就像歌词唱的，赢得了自己，才赢得了江湖。
