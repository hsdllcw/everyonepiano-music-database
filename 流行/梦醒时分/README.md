

**梦醒时分**
，这首歌是台湾唱片工业史上首张销量破百万专辑《跟你说听你说》的主打歌曲，被伍佰，林忆莲，王菲，梁静茹，周华健，迪克牛仔，高胜美等歌手翻唱过。

歌名： _梦醒时分_  
作词：李宗盛  
作曲：李宗盛  
演唱：陈淑桦  
专辑：跟你说听你说
