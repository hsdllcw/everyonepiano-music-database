

**岁月神偷旋律谱** 是根据汪苏泷与刘乐瑶在《中国新声代第五季》演唱版制作的。娓娓道来岁月的故事，给人一种“归来仍是少年”之感。

该曲原由由金玟岐作词、作曲，金玟岐演唱，发行于2014年2月18日。歌曲旨在表达要珍惜眼前人，时光永远向前流，不会倒转，所以要珍惜现在拥有的一切，诚如歌词中写到的“岁月是一场有去无回的旅行”。

歌词下方是 _岁月神偷钢琴谱_ ，大家可以免费下载学习。
