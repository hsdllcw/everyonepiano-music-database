

**刚好遇见你** 作为李玉刚于十年节点的诚意之作，与李玉刚之前的歌曲曲风颇有不同，简单的歌词描绘出清新温暖的画面，吉他声显得意味深长。

歌曲旋律并不复杂，李玉刚将其出道十年的心路历程融于明快旋律之中，情感真实而明澈。《 _刚好遇见你_
》歌词简单浅白，朗朗上口，李玉刚没有以女声演绎，而是换成高音低音轮流切换，行云流水地倾诉完歌曲的感情。

该曲MV以李玉刚的一句“行走就是圆梦，回来亦是前行”开始，以一男一女两个身影贯穿全片。MV开头，两个身影相见、相靠、共舞，然后登上火车，游历城市。其间穿插了李玉刚着戏装和写书法的画面。MV后半段则以李玉刚在雨中独舞为主，承接书法片段中李玉刚写下的“莲”字，呈现出莲花开放的画面。最后，MV以李玉刚在雨中看向前方的画面作为结束。
