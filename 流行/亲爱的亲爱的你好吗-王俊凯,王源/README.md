

**亲爱的亲爱的你好吗双手简谱** 和五线谱完全对应。

_亲爱的亲爱的你好吗_ 这首歌是王俊凯和王源也就是凯源的合唱，公司时代俊峰没有发布。这首歌虽没有正式发行，歌曲秒拍视频却被网络疯传，非常受大众喜欢。

亲爱的亲爱的你好吗是来自TFBoys组合的王源王俊凯所唱，是二人练习声乐课的时候唱的歌。

歌词下方是 _亲爱的亲爱的你好吗钢琴谱_ ，希望大家喜欢。
