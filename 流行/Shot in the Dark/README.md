Slade）、艾克索•若斯（Axl
Rose）。AC/DC善于乱世作战，他们来自金属乐版图上并不明显的澳洲，但就是他们成为摇滚历史上最简单但是最可靠的乐队。几十年来他们一直在做着一件事情——捍卫摇滚乐，最好的印证就是吉他手安格斯•扬那一身不变的校服，无论潮流如何改变，乐队成员怎么变动，他的这一身永远是乐队的标志。
