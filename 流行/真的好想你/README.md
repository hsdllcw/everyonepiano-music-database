

周冰倩1991年录制《周冰倩二胡高胡专辑》,发行量超过十万。同年东渡日本,发展音乐事业，1993年在日本第一艺能公司出专辑《Passing
Love》。获得多种音乐大奖,被誉为"上海刮来的飓风"
。1996年参加中央电视台元宵晚会，一曲《真的好想你》唱遍大江南北。收录于1996年发行的《真的好想你-周冰倩珍藏精选》专辑中。
