

《 **恋无可恋** 》是古巨基演唱的一首歌曲，由林夕作词，方皓玟作曲，于2012年9月5日正式发布，收录在专辑《告别我的恋人们》中。

《 _恋无可恋_
》是一首悲情情歌，讲述的是那种迷糊在友情与爱情之间的情感，古巨基唱得动容，不仅仅是个人情感经历的浓缩，也在于歌词内容的精炼以及深刻，正是多亏作词林夕用有限的字数表现出了一段情感深刻的情怀，《恋无可恋》字字句句入心入肺。

同时，古巨基在《恋无可恋》中欲哭又止的哽咽声，也让歌曲有了可以启动听者哭点的感动。
