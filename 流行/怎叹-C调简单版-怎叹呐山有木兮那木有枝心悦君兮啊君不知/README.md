

“怎叹呐山有木兮那木有枝，心悦君兮啊君不知”出自歌曲《 **怎叹** 》。《怎叹》是由Listen
Dream作词、作曲，郑鱼演唱的歌曲，发行于2021年6月2日。

歌曲讲述的是春秋时代一名越人摇船者，在楚王母亲的弟弟坐船游玩之时，借歌声倾述其“心悦君兮君不知”的情感的故事。歌曲用绵言细语的表达，分享了这份略有遗憾的爱意，与温柔。
