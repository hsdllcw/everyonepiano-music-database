

**Papillon** 是著名的新世纪音乐风格乐队神秘园（Secret Garden）的一首曲子。乐队由两位才华出众的音乐家组成：Rolf Lovland
（罗尔夫•劳弗兰）和Fionnuala Sherry
（菲奥诺拉•莎莉）。乐队成立于1994年，其音乐溶合了爱尔兰空灵飘渺的乐风以及挪威民族音乐及古典音乐，乐曲恬静深远，自然流畅，使人不知不觉便已融入其中。

且 _Papillon_ 此曲收录在其第一张专辑，神秘园之歌Songs From A Secret
Garden中。这张由宝丽金挪威公司制作，在菲利普厂牌下发行的唱片一经推出即获得空前成功。截止97年底唱片已发行超过65万张并登上美国Billboard音乐排行榜达一年之久。这张专辑风格比较统一，乐曲舒缓柔美，充满了思忆与郁忧，不经意之间流露出些许的忧愁，令人不得不沉醉于其中。
