_等你的季节_ 是电视剧《步步惊心》插曲，也是演员刘诗诗的第一首单曲。在2012年湖南卫视的跨年演唱会上吴奇隆和刘诗诗改编演绎了男女合唱的版本。  
  
作为小编的我，每次听到此曲都有一种悲伤的感觉。而步步惊心本身就是一个悲剧！所以听到此曲就有一种要哭的冲动，感觉就是若曦对四爷唱的，有一种哀怨的情愫。
