

《 **葫芦兄弟** 》是上海美术电影制片厂1986年出品的动画片，是国内原创最好的动画片之一
，该动画系列自1986年播出以来，一直大受广大观众，而其中的葫芦娃尤其是少年儿童们的喜爱。七只神奇的葫芦，七个本领超群的兄弟，为救亲人前赴后继，妖精们法力无边，葫芦娃大显奇能，一场场较量扣人心弦。
