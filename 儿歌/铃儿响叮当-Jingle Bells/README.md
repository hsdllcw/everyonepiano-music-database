铃儿响叮当_ 是詹姆斯·罗德·皮尔彭特所创作的一首歌曲，旋律轻快，是一首快炙人口的经典圣诞歌曲。起初，此曲名为“One Horse Open
Sleigh”。两年后，这首歌再度公开发表，正式命名为Jingle Bells(The One Horse Open Sleigh)。  
  
每每听到此曲，就仿佛把我们带到了一个美丽圣洁的冰雪世界，我们同歌曲中的小朋友一起感受了滑雪的快乐。当我们真正融入到大自然的怀抱中，就会感到无比快乐。
