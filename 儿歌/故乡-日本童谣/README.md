《 _故乡_ 》，日文为《故郷（ふるさと）》，作曲者是冈野贞一。该曲曲调优美，略带伤感，1914年作为当时小学校学生音乐课修习的歌曲之一。后一代代传唱至今。

歌词里描写了一个人的回忆童年时光的美好，独自外出打拼的辛酸，想起身在故乡的父母已经两鬓白发，自己又何时才能回到那个令人魂牵梦绕的故乡。
