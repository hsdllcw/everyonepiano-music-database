

英国童谣-新年好，新年好呀新年好呀，祝贺大家新年好。我们唱歌我们跳舞，祝贺大家新年好。  
  
《 _新年好_ 》改编自朝鲜电影《卖花姑娘》主题曲的一首儿歌，原曲是英国一首非常著名的曲子《Happy New Year》。  
  
该曲在中国是家喻户晓的儿歌。小朋友们会唱，大人们也会唱。每每过年，大街小巷就会听到洋溢着欢乐和幸福的《 _新年好_ 》旋律。
