

《卖汤圆》是台湾著名民谣，每逢正月十五元宵节这个吃汤圆赏灯的节日该歌就大为流行，该歌被曾被中、港、台多位不同的歌手演绎以下介绍的是卓依婷版本的《卖汤圆》。

一碗汤圆满又满  
三毛钱呀买一碗  
汤圆汤圆卖汤圆
