

《卖报歌》是音乐家聂耳三十年代创作脍炙人口的儿童歌曲。2/4拍，四乐句组成的一段体，五声F宫调式。
《卖报歌》曲调明快、流畅，并以朴实生动的语言，辛辣诙谐的笔调，深刻地描述了旧社会报童的苦难生活及对光明的渴望。

### 卖报歌歌词：

啦啦啦！啦啦啦！我是卖报的小行家，  
不等天明去等派报，一面走，一面叫，  
今天的新闻真正好，七个铜板就买两份报。  
啦啦啦！啦啦啦！我是卖报的小行家，  
大风大雨里满街跑，走不好，滑一跤，  
满身的泥水惹人笑，饥饿寒冷只有我知道。  
啦啦啦！啦啦啦！我是卖报的小行家，  
耐饥耐寒地满街跑，吃不饱，睡不好，  
痛苦的生活向谁告，总有一天光明会来到。
