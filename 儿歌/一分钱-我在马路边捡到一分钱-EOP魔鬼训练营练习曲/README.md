一分钱_ ，又可以成为我在马路边捡到一分钱，是由潘振声作词作曲的一首儿童歌曲。  
  
1964年的一天，中央人民广播电台《小喇叭》节目组向潘振声约稿，请他写一首赞扬“好孩子”的歌。收到约稿信后，潘振声很快又想到那些温馨的场面。此时正是硬币在社会上大量流通的时候，他联想到孩子们的拾金不昧和回家时与警察叔叔告别的情形，便创作了歌曲《一分钱》。  
  
歌词下方是 _一分钱钢琴谱_ ，希望大家喜欢。

### 一分钱歌词：

我在马路边捡到一分钱，  
把它交到警察叔叔手里边，  
叔叔拿着钱，  
对我把头点，  
我高兴地说了声：叔叔，再见！
