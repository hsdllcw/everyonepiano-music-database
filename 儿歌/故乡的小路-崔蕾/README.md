_故乡的小路_
是由陈克正作词，崔蕾作曲的一首歌曲。该曲目的节拍十分奇特，先是以两小节4/4拍节奏的乐句作为开头，后变为前一小节是3/4拍，后一小节是4/4拍，该曲目现已被收入为人民音乐出版社出版的五年级上册音乐书中的一首演唱歌曲。

歌词下方是 _故乡的小路钢琴谱_ ，大家可以免费下载学习。
