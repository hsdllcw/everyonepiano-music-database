_感谢_
是一首幼儿园毕业感恩歌曲，表达了深深的情谊。在流畅舒缓的旋律中将对生命中的感谢娓娓道来，涤荡心灵，令这份意境感染听众，呼唤更多人珍惜并且怀着一颗感恩的心对待一切。

歌词下方是 _感谢钢琴谱_ ，希望大家喜欢。
