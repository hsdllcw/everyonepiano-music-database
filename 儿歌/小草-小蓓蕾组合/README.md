

**小草双手简谱** 和五线谱完全对应。

_小草_ 是一首小蓓蕾组合翻唱的歌曲，由向彤、何兆华作词，王祖皆、张卓娅作曲，收录在小蓓蕾组合发行的专辑《儿歌曲库01》。

歌词下方是 _小草钢琴谱_ ，希望大家喜欢。
