_两只老虎_ 是根据一首在法国叫做《雅克兄弟》的歌曲重新填词改编成的儿歌。

此曲曲调简洁优美，朗朗上口，甚至在民国时被用来做国歌的曲调。1926年7月国民革命军誓师北伐，为了鼓舞士气动员民众，黄埔军校政治教官、国民革命军政治部宣传科科长邝墉，根据儿歌《两只老虎》的曲调重新填了词，改名为《国民革命歌》。这首歌经当时的政治部主任邓演达、副主任郭沫若同意，作为国民革命军军歌。
