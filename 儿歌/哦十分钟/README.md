

歌曲《哦，十分钟》是一首2/4拍，D宫调式，二段体结构的儿童歌曲。曲调欢快、活泼，语言形象生动。歌曲通过对孩子平日课间十分钟的描写，表现了孩子们在紧张的学习后自由欢乐的心情。第一段奏紧凑，曲调反复出现了活蹦乱跳的孩子们在课间轻松，快乐的形象和喜悦的神情。第二段音调转为舒展，第一句与第二句运用旋律的模仿手法，四度、三度的跳进。“哦，十分钟”唱出了孩子们对课间十分钟的由衷欢迎。
