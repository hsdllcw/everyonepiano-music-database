_别看我只是一只羊_
是广东原创动力文化传播有限公司出品的动画作品喜羊羊与灰太狼的主题曲，由古倩敏原唱。此动画由黄伟明、赵崇邦导演，黄伟健编剧，作品于2005年8月在杭州电视台少儿频道首播，该剧以羊和狼两大族群间的争斗为主线讲述了一系列的故事。  
  
此曲旋律轻快，是一首脍炙人口的儿歌，加上动画喜羊羊与灰太狼的热播，这首曲子，更是受到广大小朋友的青睐。
