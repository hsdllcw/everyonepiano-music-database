

**雨花石双手简谱** 和五线谱完全对应。

_雨花石_ 是首经典的儿歌，优美旋律，表达了对和平生活的热爱与珍惜。对先烈的敬仰，对祖国的爱恋。

歌词下方是 _雨花石钢琴谱_ ，希望大家喜欢。
