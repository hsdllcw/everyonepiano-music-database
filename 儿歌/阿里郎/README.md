_阿里郎_
（又译阿里岚等）是著名的朝鲜族民歌，它在不同地方有不同版本，最常听到的是流行于韩国京畿道一带的本调阿里郎，这版本是因为它成为1926年同名电影的主题曲（该电影是朝鲜第一出剧情片）。后来该曲在2000年悉尼奥运会期间被用作韩国与朝鲜代表团的进场音乐。  
  
《阿里郎》这首曲子，应该很多人听过吧。个人而言，是朝鲜民歌中最熟悉的一首啦~ 是一首很有民族韵味的音乐。
