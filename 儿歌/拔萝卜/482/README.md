

**拔萝卜** 是传唱很广的儿歌，也是伴随很多人一起成长的经典儿歌。小蓓蕾组合曾演唱，并收入在《儿歌曲库04》专辑中。  
  
_拔萝卜_ 这首钢琴曲，节奏欢快，简单易学，很适合孩子练习。
