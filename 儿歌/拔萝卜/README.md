

《拔萝卜》这个故事最早是由俄罗斯的阿尔克谢·托尔斯泰所写，是一个真的很幽默的民间童话，很适合五岁以下或者刚刚开始学习阅读的孩子。故事的结尾还有一个意想不到的结果。《拔萝卜》版权已售出20种语言。著名儿童文学博士，儿童文学推广人王林，著名儿童文学博士，儿童文学理论家谭旭东联袂推荐。
