

**月牙船双手简谱** 和五线谱完全对应。

_月牙船_ 这首儿歌讲述了我们和台湾小朋友一起划进日月潭， 虽没钓到水晶宫里的珍宝，却载满月光一船，歌声一船。对宝宝来说是不错的儿歌。

歌词下方是 _月牙船钢琴谱_ ，希望大家喜欢。
