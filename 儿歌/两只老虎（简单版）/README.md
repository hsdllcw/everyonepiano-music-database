

"老虎版"的歌词最早出现可能要早于民国三十二年(1943),因为曹聚仁的《蒋经国论》第五节《新赣南如此》就有提到当年唱了"两只老虎"的儿歌.
蒋经国先生就很喜欢这歌,似乎还是倡导了"三只老虎"的版本.
"许多回忆录都提到，每当蒋经国遇到重大问题，总要率领群众唱这首儿歌，一来激励群众，同时也鼓舞自己然而"老虎版"歌词出自何人,虽然有幸找到所谓的起源但都经不起推敲。
