

**小白船双手简谱** 和五线谱完全对应。

_小白船_
，原名半月，创作于1924年，是一首深受小朋友喜爱的朝鲜童谣，由朝鲜作曲家尹克荣作词、作曲。被译成中、日两国语言，在中国与朝鲜半岛广为流传。1950年被译为中文。歌手范玮琪也有同名的一首歌。1950年被译成中文传入中国，最初收在解放初出版的《外国名歌300首》中。此曲旋律优美，在韩国一般都当做安魂曲使用，而在中国则是著名儿歌。被誉为“东方圣歌”。

歌词下方是 _小白船钢琴谱_ ，希望大家喜欢。
