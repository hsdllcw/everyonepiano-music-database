

这首童声独唱的低幼歌曲描绘了一位幼儿天真地唱数桥下游鸭的情景，形象地表现了幼儿们的咿哑学语及长辈们对其日渐成长的期望。作品歌词亲切生动，饶有儿歌情趣。曲调活泼甜嫩，曲式浅简，节奏欢快，念唱结合，易学易唱，受到儿童们的欢迎喜爱。
一首经典的80年代的儿歌，如今还是很受儿童的喜爱。可能主要是因为起歌词通俗易懂，具有趣味性。
