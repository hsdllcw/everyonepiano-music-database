_  
啊哈！黑猫警长_
，由蔡璐作词作曲，是动画片《黑猫警长》的主题歌。这是难得的国产动画片精品。1987年上海美术电影制片厂摄制完成的4集动画片《黑猫警长》，一时间，主题歌《啊哈！黑猫警长》广为传唱。  
  
歌曲首先通过对黑猫警长威武矫健的形象的描写，生动地表现出黑猫警长作为侦探家的机灵与精明。顿挫有致的节奏。活泼健朗的音调，突出了黑猫警长富于朝气与机敏的性格特征。
