

《 **桃太郎** 》是日本的一首经典童谣。作曲者是冈野贞一。

《桃太郎》是日本著名民间故事，讲述从桃子里诞生的桃太郎，用糯米桃子收容了小白狗、小猴子和雉鸡后，一起前往鬼岛为民除害的故事。在日本，冈山县、香川县、爱知县和奈良县都主张《桃太郎》的故事发祥于自己的地域，其中冈山县冈山市作为《桃山郎》的缘因之地最为知名。
