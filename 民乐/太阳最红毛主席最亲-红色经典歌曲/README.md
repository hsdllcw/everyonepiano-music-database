

### 简介：

《太阳最红毛主席最亲》，是一首红色经典歌曲。1976年9月9日，毛泽东主席逝世，全国人民沉浸在一片悲痛之中。出于对伟大领袖的怀念和崇敬之情，付林写下了《太阳最红，毛主席最亲》这首词，后来由王锡仁谱了曲。这首歌从歌词到曲调都自然平和、朗朗上口，朴实得就像说话一样，表达了老百姓怀念毛泽东主席的真情实感，非常受老百姓的喜欢，至今久唱不衰。
