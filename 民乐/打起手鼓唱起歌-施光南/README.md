

**打起手鼓唱起歌**
是一首富有浓郁的新疆民歌特色的歌曲，由施光南作曲，韩伟填词。这首歌曲节拍欢快、曲调优美，为人们展现了一幅美好生活的动感画卷，表达了人们心中的喜悦与憧憬，流露出了人们热爱家乡、建设祖国的一片豪情。

  
歌词下方是 _打起手鼓唱起歌钢琴谱_ ，希望大家喜欢。
