

**山不转水转**
是由华语流行女歌手那英演唱的一首歌。那英是一个不断求新、求变、不断超越自我的歌者，她的嗓音具有像欧美黑人歌手的磁性和爆发力。她敢于大胆改变自己的歌路和风格，从以前的《山沟沟》、《雾里看花》，到后来的港式乐《愿赌服输》、《征服》、《相见不如怀念》、《心酸的浪漫》等音乐风格不停变化且还能让歌迷们认同。她直率，爽朗的个性也是众人皆知，同时也不失女人的细腻和坚韧的包容力。

  
歌词下方是 _山不转水转钢琴谱_ ，希望大家喜欢。
