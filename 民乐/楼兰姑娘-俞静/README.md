_楼兰姑娘_ 是一首俞静演唱的以新疆古城楼兰为背景的著名歌曲。歌曲是付林作曲，杨海潮、付林填词，出品年代是1994年。

1994年中央电视台全国第六届青年歌手大赛以《楼兰姑娘》获第二名。1995年中央电视台MTV大赛《楼兰姑娘》获铜奖。
