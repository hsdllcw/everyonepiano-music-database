

《 **学习雷锋好榜样**
》是由作曲生茂、作词洪源二人共同创造的歌曲，20世纪50年代末60年代初影响较深的歌曲。歌曲激昂、振奋。在每年的三月五日，各大学校会举行大型的活动——“3.5学雷锋日”在这一天学生会到各个社区义务清扫、访问老人等，《
_学习雷锋好榜样_ 》也就自然成为了主题曲。
