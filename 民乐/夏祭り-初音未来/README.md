_夏祭り_
是首日本民歌，JITTERIN'JINN在1990年首唱。在1999年的精选《8-9-10》内有收录。在以下动漫出现过：太鼓达人OST、乐动舞指、jubeat、初音有翻唱、今日的5年2班第六话ED、2013年7月新番《丧女》第六集ED。

歌词下方是 _夏祭り钢琴谱_ ，希望大家喜欢。
