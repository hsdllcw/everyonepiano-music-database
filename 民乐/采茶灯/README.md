

一九九六年，民乐《采茶灯》在美国亚特兰大奥运会上作为中国体育代表团入场曲目之一而名扬海内外。二00五年被列为福建省首批非物质文化遗产保护项目，是龙岩乡土文化的杰出代表，后又作为"口头文化遗产"被联合国教科文组织收藏。据称，在这个世界上，凡有龙岩人聚集的地方，都传唱着自己的歌谣——龙岩"采茶灯"。

### 《采茶灯》歌词：

百花开放好春光， 采茶姑娘满山岗。

手提着篮儿将茶采， 片片采来片片香，

茶树发芽青又青， 一棵嫩芽一颗心。

轻轻摘来轻采采， 片片采来片片新。

采到东来采到西， 采茶姑娘笑眯眯。

过去采茶为别人， 如今采茶为自己。

采满一筐又一筐， 山前山后歌声响。

今年茶山好收成， 家家户户喜洋洋。
