

**月亮走我也走**
，是一首创作于二十世纪八十年代的爱情歌曲，由胡积英谱曲，瞿琮填词。此曲反映了姑娘对即将入伍的自己的恋人的依依不舍之情和殷切期盼之意，歌颂了真挚纯洁的爱情。

  
歌词下方是 _月亮走我也走钢琴谱_ ，希望大家喜欢。
