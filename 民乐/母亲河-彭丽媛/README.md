

**母亲河**
是中国著名女高音歌唱家彭丽媛演唱的曲子，由王志信作曲。彭丽媛，1962年11月20日出生于山东菏泽郓城，中国第一位民族声乐硕士，国家一级演员，享受国务院颁发的“政府津贴”。1985年7月加入中国共产党，是中国人民解放军文职干部，正军级待遇。

  
歌词下方是 _母亲河钢琴谱_ ，希望大家喜欢。
