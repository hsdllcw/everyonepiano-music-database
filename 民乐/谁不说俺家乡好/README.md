

《谁不说俺家乡好》是拍摄于1961年的电影《红日》中的插曲。影片取材于1947年的“孟良崮之战”，这场闻名中外的惨烈战役对交战双方有着存亡续绝的意义。1947年5月13日到16日三天的时间里，72万军人在孟良崮决一死战。多年后在孟良崮树起了一座永久性纪念物——孟良崮战役纪念碑，以铭记在战役中为了解放战争胜利不惜牺牲的人们。
