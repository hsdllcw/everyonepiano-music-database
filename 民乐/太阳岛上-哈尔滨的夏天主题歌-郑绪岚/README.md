

**太阳岛上** 歌曲创作于1980年，由邢籁、秀田、王立平作词，王立平作曲，郑绪岚演唱，朱逢博等翻唱过。

  
《 _太阳岛上_ 》是纪录片《哈尔滨的夏天》主题歌，一曲《太阳岛上》使黑龙江哈尔滨太阳岛成为全国人民向往的风景名胜。

  
歌词下方是 _太阳岛上钢琴谱_ ，希望大家喜欢。

美丽的太阳岛多么令人神往

我们来到了太阳岛上  
小伙们背上六弦琴

心爱的猎枪

甜蜜的喜悦挂在姑娘眉梢  
带着真挚的爱情  
带着美好的理想  
我们来到了太阳岛上  
幸福的生活靠劳动创造
