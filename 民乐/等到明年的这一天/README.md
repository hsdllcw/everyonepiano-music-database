

朱明瑛祖籍临邑县德平镇，原东方歌舞团演员，著名歌舞表演艺术家，1986年被中国文化部授予“国家一级演员”称号。专攻亚洲、非洲、拉丁美洲歌舞艺术，兼学中国古典舞、芭蕾舞、民间舞和钢琴。能歌善舞、风格迥异、豪迈奔放、声情并茂的表演艺术深受国内外广大观众的赞赏，1985年由文化部公派自费赴美学习现代音乐和文化产业。
