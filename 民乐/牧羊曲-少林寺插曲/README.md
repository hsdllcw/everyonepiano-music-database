“日出嵩山坳，晨钟惊飞鸟，林间小溪水潺潺，坡上青青草”这首《 _牧羊曲_
》至今都还被很多专家认为是中国流行乐坛的经典之作。郑绪岚用她细润甜美的嗓音让一个千娇百媚风情万种的女儿态活生生的呈现在面前，也让铁骨铮铮的嵩山充满柔情一片。
