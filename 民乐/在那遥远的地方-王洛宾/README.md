

**在那遥远的地方双手简谱** 和五线谱完全对应。

_  
在那遥远的地方_
的曲调源于哈萨克族民歌，是西部歌王王洛宾创作于1938年到1939年之间的一首作品。它是世界范围内传唱最广的华人歌曲之一，也是王洛宾所有创作歌曲中艺术评价最高的，被赞为“艺术里的珍品，皇冠上的明珠”。

  
歌词下方是 _在那遥远的地方钢琴谱_ ，希望大家喜欢。
