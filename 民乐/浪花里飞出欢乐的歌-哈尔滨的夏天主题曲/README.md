_  
浪花里飞出欢乐的歌_
为1980年电视纪录片《哈尔滨的夏天》主题曲，由关贵敏演唱。本歌曲抒情，优雅动听，富有诗情画意。随着我国舞蹈事业的发展，本歌曲被制作成舞曲，用作交谊舞，广场舞，在我国民间广泛流行。

  
歌词下方是 _浪花里飞出欢乐的歌钢琴谱_ ，希望大家喜欢。
