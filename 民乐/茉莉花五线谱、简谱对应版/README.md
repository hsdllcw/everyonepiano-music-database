

### 茉莉花简介：

中国民歌，在国内以及国际具有较高知名度，在中国及世界广为传颂。起源于南京市六合区的民间歌曲。由著名军旅作曲家何仿采自于六合的民歌汇编整理而成。此歌曲多次在香港回归祖国政权交接仪式、雅典奥运会闭幕式等重大场面上演出。

在中国及世界广为传颂，系江苏民歌。起源于南京六合一带的民间歌曲，由著名军旅作曲家何仿将采自于南京六合一带的民歌汇编整理而成。

### 茉莉花五线谱、简谱对应版歌词：

好一朵茉莉花，好一朵茉莉花，  
满园花草，香也香不过它；  
我有心采一朵戴，  
看花的人儿要将我骂。  
好一朵茉莉花，好一朵茉莉花，  
茉莉花开，雪也白不过它；  
我有心采一朵戴，  
又怕旁人笑话。  
好一朵茉莉花，好一朵茉莉花，  
满园花开，比也比不过它；  
我有心采一朵戴，  
又怕来年不发芽。
