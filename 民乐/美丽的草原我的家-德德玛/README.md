_  
美丽的草原我的家_
是一首中国20世纪70年代的蒙古族歌曲，由火华作词、阿拉腾奥勒作曲。1978年由著名女中音歌唱家德德玛演唱，受到歌唱界的极大关注。《美丽的草原我的家》于1980年被联合国教科文组织以“世界优秀歌曲”编选入教材，并收录于《亚太歌曲集》中。

  
歌词下方是 _美丽的草原我的家钢琴谱_ ，希望大家喜欢。
