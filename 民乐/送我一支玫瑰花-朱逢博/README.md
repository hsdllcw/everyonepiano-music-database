

**送我一支玫瑰花**
，是一首新疆维吾尔民歌，最早由歌唱家朱逢博1970年演唱，歌曲得以流传。该曲旋律优美、舒展、活泼、深情，充分表达了青年男女对爱情的坚贞执著追求。

  
歌词下方是 _送我一支玫瑰花钢琴谱_ ，希望大家喜欢。
