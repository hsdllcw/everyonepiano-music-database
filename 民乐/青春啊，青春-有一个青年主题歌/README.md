

**青春啊，青春**
是中央电视台第一部实景拍摄的单本电视剧《有一个青年》的主题曲。该片改编自张洁所著同名小说，由蔡晓晴执导，张铁林、方舒、齐士龙、陈浥、朱小明、沈丹萍等领衔主演。该剧讲述了一个在“文革”中失去正常学习机会的知识贫乏的青年不甘现状、勇于追求，终于取得科研成果，并收获爱情的故事。

  
歌词下方是 _青春啊，青春钢琴谱_ ，希望大家喜欢。
