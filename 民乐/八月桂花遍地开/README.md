

**八月桂花遍地开** 是由大别山民歌《八段锦》的曲调改编而成，传唱至今，经久不衰，它的诞生地有多种版本。从上世纪80年代开始，《 _八月桂花遍地开_
》的诞生地引起了各地之争，有江西说，四川说，湖北说，安徽说和河南说。
