_半个月亮爬上来_
，王洛宾创作，合唱由蔡余文改编。属于无伴奏合唱。此曲合唱短小、精练,富有合唱音乐所特有的协调感。歌曲表现了青年人的爱情生活。全曲采用带再现的三段体写成。第一段与第三段曲调平稳,充满夜晚的宁静气氛。大调式的密集和弦,使各声部产生均衡的音响。中段旋律起伏较大,旋律分别由男高音、男低音及女高音唱出,“请你把那纱窗快打开”的句子,表现小伙子的焦急心情。第三段仍回到第一段宁静的气氛中。似在温馨的夜色里,年轻人尽情享受着爱情的喜悦。  
  
王洛宾，长期在西北地区从事民歌搜集、研究和音乐创作,改编了大量民歌。此曲本是由作曲家王洛宾根据西北地区民间音调创作的一首歌曲,因为曲调优美,意境深远,易于学唱而流传全国,常被误认为青海民歌,原作者反而被人们遗忘。
