

**在那桃花盛开的地方**
是董振厚原唱，邬大为、魏宝贵作词、铁源作曲的歌曲，收录在1980年发行的同名专辑中。该曲在1987年获得全国“青年最喜爱的歌”评选一等奖。

  
《 _在那桃花盛开的地方_
》唱出了军民之间的情意，曾两次登上CCTV春节联欢晚会的舞台。1984年，蒋大为带着《在那桃花盛开的地方》这首歌首次登上了中央电视台春节联欢晚会的舞台，这首歌很快在全国流传开来。歌唱家蒋大为用他那嘹亮的歌声感动了几代人。
