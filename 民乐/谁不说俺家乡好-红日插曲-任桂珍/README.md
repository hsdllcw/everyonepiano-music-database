

**谁不说俺家乡好** 是一首著名中国歌曲，由肖培珩作词，吕其明作曲。该曲原为电影《红日》的主题曲，上映于1963年。内容描写对于山东沂蒙山地的歌颂。

  
2007年，中国发射的第一颗人造月球卫星“嫦娥一号”升上太空后，《 _谁不说俺家乡好_
》与《爱我中华》其它由全国民众选出的三十首歌曲一起响彻太空，并且名列三十首歌曲之首。

  
歌词下方是 _谁不说俺家乡好钢琴谱_ ，希望大家喜欢。
