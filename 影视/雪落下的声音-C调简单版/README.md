

我慢慢地听，雪落下的声音，  
闭着眼睛幻想它不会停

这首歌听起来舒缓如落雪的歌曲，出自陆虎作曲并演唱的《 **雪落下的声音** 》。同时它也是电视剧《延禧攻略》的插曲（由秦岚演唱）以及片尾曲。

虽被许多人翻唱。但是小编最喜欢的还是周深演唱的版本。他的声音细腻干净，声不醉人人自醉。周深不是那听雪者，他就是那片雪花。听雪者远不及雪花了解自己内心的孤独和无助，只有雪花自己懂得。而周深的版本上线后，就高居新歌榜榜首，这也是网友们对周深的一种肯定。
