_偏偏_
是汪苏泷、迪丽热巴首次合作，共同携手希瓜音乐•谭旋音乐工作室，为电视剧《三生三世枕上书》打造的片尾曲。迪丽热巴奶萌的嗓音加上汪苏泷充满辨识度的嗓音让这首歌更添特色。歌曲由栾杰作词，谭旋作曲，雷立编曲。

这是一首脍炙人口的甜美古风对唱情歌，歌曲将男女之间的爱恋浓缩，字字句句流露着幸福和喜悦，汪苏泷和迪丽热巴的声音更是为歌曲画上了最高甜的一笔，让凤九和东华的爱情充满了无限生机。
