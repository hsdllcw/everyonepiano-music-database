

**心墙** 是由任嘉伦演唱的歌曲，同时也是电视剧《锦衣之下》的片尾曲。

歌里是言渊的故事，也是任嘉伦的故事。命运的流转，心墙被撕开，繁尘中，弄权一生，以身化剑，愿护她一生周全，携手至白头。时隔多年，任嘉伦的苏音依旧酥到心中，一首慢歌，且听他讲一个慢慢的故事。
