

**爱不老**
是由罗震环演唱的一首歌曲，是电视剧《爱情公寓5》特邀片尾曲。此曲由新兴音乐厂牌安得音乐为罗震环量身定做，动人的歌词和歌手深情的演绎配合得相得益彰，表达着对爱的执着和坚定。
