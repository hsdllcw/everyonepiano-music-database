

**小松树** 原是电影《雷锋》里的插曲，将小松树和儿童进行类比，小松树需要阳光雨露的滋养才能长大，毛泽东思想哺育我们，快快长大。歌曲具有鲜明的时代特点。

《雷锋》是一部由董兆琪执导的电影，讲述主人公雷锋从一个稚嫩的少年变成一名平凡而伟大的解放军战士的成长历程。弘扬雷锋无私奉献的精神。
