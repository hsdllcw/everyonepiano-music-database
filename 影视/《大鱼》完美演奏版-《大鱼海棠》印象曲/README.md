**大鱼双手简谱** 和五线谱完全对应。

这曲《 _大鱼》_
作为动画电影《大鱼海棠》的印象曲，真是整部电影的一个写照。整个故事似乎都在这首曲子里一一道来。不停地寻觅，也不断懂得更多，然而不管怎样最后时间都会把我们带到一个地方。好好品味每句歌词，会觉得意犹未尽，有点像宫崎骏动画的那首《天空之城》。

歌词下方是 _大鱼钢琴谱_ ，希望大家喜欢。
