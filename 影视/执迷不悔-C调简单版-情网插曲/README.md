

《 **执迷不悔** 》是王菲作词并演唱的歌曲，亦是电视剧《情网》的插曲。该曲的粤语版为《执迷不悔》，均收录于王菲1993年发行的同名专辑《执迷不悔》中。

作为同名专辑中的主打歌，《 _执迷不悔_ 》的曲风有灵魂乐式的演唱色彩。该曲是王菲首次尝试歌词创作，而“执迷不悔”更像是她个性的真实写照
。虽然歌曲的曲调较为平凡、编曲普通，但仅仅其透出的那份感觉或信念触动人心已足矣，《执迷不悔》就是这样的一首歌 。

执迷不悔，一种洒脱的生活态度，也可以是半梦半醒之间的一剂自我安慰。这首歌并不是王菲作品中最有名的，但在我看来，这首歌却是最真实的王菲，是王菲内心的写照。
