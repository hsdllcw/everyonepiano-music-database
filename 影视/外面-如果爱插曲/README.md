

**外面**
是电影《如果爱》插曲，由女主演周迅演唱。歌舞片《如果•爱》由陈可辛执导，由周迅、金城武、张学友、池珍熙等人主演，讲述了出身贫苦的女星孙纳，在她努力想要忘记过去一切的时候，她生命中两个重要的男人同时出现，孙纳面临情感抉择的故事。本片亦为第62届威尼斯电影节闭幕影片。

  
歌词下方是 _外面钢琴谱_ ，希望大家喜欢。
