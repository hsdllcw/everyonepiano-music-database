

**从此以后双手简谱** 和五线谱完全对应。

_从此以后_ 是《夏有乔木雅望天堂》的主题曲，由吴亦凡演唱，于2016年7月22日首发。

从此以后是主演吴亦凡从自身角度对片中饰演角色“夏木”的诠释，并亲自填词演唱，曲风清新流畅，旋律温暖抒情。33分钟破百万试听，打破自己创造的记录。

从此以后讲述的正是男主角夏木和雅望之间的一段虐心的年下之恋，在吴亦凡的深情演绎下，显得既清新浪漫，又伤感无比。

歌词下方是 _从此以后钢琴谱_ ，希望大家喜欢。
