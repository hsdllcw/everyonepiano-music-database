

**笑看风云双手简谱** 和五线谱完全对应。

_笑看风云_ 是1993年由黄沾作词、徐嘉良作曲，
郑少秋演唱的一首歌，收录于同名专辑《笑看风云》中。是1994年香港电视剧《笑看风云》的主题曲，平白易懂却直逼人心。影剧《笑看风云》是以黄天主导的商界风云为基础，讲述了黄天与潘仲之间的兄弟情谊以及包文龙和林贞烈之间的爱情故事。

歌词下方是 _笑看风云钢琴谱_ ，大家可以免费下载学习。
