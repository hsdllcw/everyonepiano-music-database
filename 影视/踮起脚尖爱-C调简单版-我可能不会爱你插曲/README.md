

《 **踮起脚尖爱** 》是洪佩瑜演唱的一首歌曲，由小寒作词，蔡健雅作曲，Derek
Chua编曲。电视剧《我可能不会爱你》的插曲，于2011年4月24日发行。

写曲的是蔡健雅，《 _踮起脚尖爱_
》算是蔡健雅少有的婉转慢情歌。歌曲节奏感缓慢而厚重，一停一顿都在计划中。这首歌的词简直像一位痴情大师写的，歌词和电视剧情节十分搭配。
