

**小乌龟** 是由游家豪作词，廖世杰作曲，张栋梁演唱的歌曲，发行于2006年8月25日。同时，此曲也是电视剧《微笑Pasta》的主题曲。

《微笑pasta》该剧讲述一对八竿子打不着的年轻男女，因为街头的一次意外撞击，不小心撞出漫天绯闻，也撞出一段趣味横生的浪漫情缘，两个平行的人生，从此交缠在一起，展开了一段浪漫而不凡的人生旅程的故事。
