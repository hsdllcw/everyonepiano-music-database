

**唯你一生双手简谱** 和五线谱完全对应。

_唯你一生_
是水木年华演唱的一首歌曲。水木年华，中国人文民谣歌唱组合，“水木年华”的名字取自于“水木清华”，前后共有四位成员，均是清华大学毕业生，目前由卢庚戌和缪杰组成。

唯你一生是电影《一生有你》推广单曲。《一生有你》是一部由卢庚戌执导的影片，李健、缪杰主演。

歌词下方是 _唯你一生钢琴谱_ ，希望大家喜欢。
