很多人听到这首歌时，总在不经意间被打动。有人说，整首《 _人世间_
》，清透明亮，有着连接天地的宽广情怀，又有着人间烟火气的温暖浪漫，抚慰着每一个在命运浪潮下浮沉的灵魂，演绎着我们中国人对“家”不变的信仰。
