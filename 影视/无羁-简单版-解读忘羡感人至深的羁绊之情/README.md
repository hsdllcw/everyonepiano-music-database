《 _无羁_ 》是《陈情令》片尾曲。歌名取无拘无束之意，本以为描写的是魏婴那潇洒的性格，看了歌词才知道，其实写的是忘羡之间的感情。

“煮一壶生死悲欢祭少年郎，明月依旧何来怅惘，不如潇潇洒洒历遍风和浪，天涯一曲共悠扬”，古韵十足的歌词搭配荡气回肠的国风旋律，而当肖战清澈透亮、王一博宽厚沙哑的歌声缓缓入耳，再搭配触动人心的歌词和悲伤孤寂的旋律，将温暖动人的情节表达的淋漓尽致。
