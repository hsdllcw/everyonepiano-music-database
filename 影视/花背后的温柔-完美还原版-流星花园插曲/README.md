

新版《流星花园》插曲《 **花背后的温柔** 》，该歌曲是由花泽类饰演者官鸿演唱。同时，此曲也被称为“花泽类之歌”，官鸿哀伤地唱出了他的感情世界。

《流星花园》是根据日本漫画家神尾叶子创作漫画《花样男子》改编的青春言情偶像剧，该剧讲述了董杉菜怀揣着把家里的家厨事业做大做强的梦想来到了明德大学的营养系，却遇到F4之首道明寺，从而展开了一段关于成长和爱情的故事。
