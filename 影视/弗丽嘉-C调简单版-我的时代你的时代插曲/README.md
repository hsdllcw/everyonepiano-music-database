

**弗丽嘉** 由明天作词，都智文作曲，丁当演唱，发布于2021年2月4日，同时也是《我的时代，你的时代》电视剧的插曲。

丁当浑厚有穿透力的嗓音，唱出了对“弗丽嘉”的坚定与深情。弗丽嘉是内心的信仰，是那个给自己力量与勇气走出阴霾的人，他可能是遥不可及的星星，也可能是身边相爱的人。对于艾情来说，弗丽嘉是开导鼓励她成为一个优秀领队的吴白。
