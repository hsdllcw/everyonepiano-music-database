

**轻吻** 是由张鹏鹏、浅紫作词，都智文作曲，苏醒演唱的一首歌曲。该曲是电视剧《我好喜欢你》的插曲。

《我好喜欢你》是由林子平、黄常祚执导，言承旭、沈月领衔主演的都市励志爱情剧。该剧改编自漠兮的小说《交换吧，运气》，讲述了高傲主编陆星成和倒霉鬼设计师童小悠之间的爱情故事，于2020年8月3日在优酷视频播出。
