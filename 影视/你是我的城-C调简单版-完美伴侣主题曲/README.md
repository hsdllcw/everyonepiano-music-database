

周深《完美伴侣》主题曲《 **你是我的城** 》于2022年1月6日上线。舒缓的钢琴开场在周深的歌声中娓娓道来都市情感故事，让人听入迷了！

结合电视剧中的一些名场面，周深的歌声给人感觉既感动又心酸，感动的是他唱进了观众的心缝里，心酸的是剧情中的男女主，为了家庭付出了很多东西，如同剧名一样《完美伴侣》想达到完美这一步，并非易事。

只有相互理解，共同扶持，才能成就彼此，成为所谓的“完美伴侣”。而周深歌词十分细腻地把对家人的情感诠释了出来，每一句唱得都特别接地气，没有华而不实的词语，只有诚意满满的歌词，能达到这种境界的歌手，只有周深，没有之一。

_你是我的城钢琴谱_ 是网友提供的，由网友 **NGERN** 制作，感谢NGERN对EOP的支持！
