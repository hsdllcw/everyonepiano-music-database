

**飘雪双手简谱** 和五线谱完全对应。

_飘雪_ 歌曲原版是中岛美嘉演唱的《雪の华》，随后被朴孝信翻唱歌名为《雪之花》为韩剧《对不起，我爱你》的主题曲。再次被韩雪翻唱为飘雪。

《对不起，我爱你》是韩国KBS2004年出品，由李亨民执导，李庆熙执笔，苏志燮、林秀晶、郑京浩、徐智英等人联袂主演的月火偶像剧。该剧以收养为主题，描写了一个关于被抛弃孩子悲苦人生的心碎故事。

歌词下方是 _飘雪钢琴谱_ ，大家可以免费下载学习。
