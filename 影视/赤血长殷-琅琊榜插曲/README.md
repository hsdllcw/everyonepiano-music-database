

**赤血长殷双手简谱** 和五线谱完全对应。

_赤血长殷_ 由王凯演唱，同时也是电视剧《琅琊榜》的插曲。歌曲还原了小说主角梅长苏的一生。大气磅礴，契合小说剧情。令人闻之落泪。

此曲由于海航作曲，清彦、冰封作词，曲调悠扬中彰显豪迈与坚定，娓娓道尽梅长苏的心路历程：虽是一身病骨，年寿难永，但为扶持明君、昭雪旧冤，他仍旧殚精竭虑、未雨绸缪，最终达成所愿，生动诠释“情义千秋”的主题。

歌词下方是 _赤血长殷钢琴谱_ ，希望大家喜欢。
