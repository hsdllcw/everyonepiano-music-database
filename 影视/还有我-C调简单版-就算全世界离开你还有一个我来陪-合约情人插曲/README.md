

在你心情低落沮丧的时候，永远有一首歌、一个人陪伴左右。

任贤齐演唱的《 **还有我**
》就是这样一首歌曲。暖男的感觉，女孩子很需要这样的人来守候，“就算全世界离开你，还有一个我来陪，怎么舍得让你受尽冷风吹”，真的是好温暖，很治愈！

同时，此曲也是电影《合约情人》的插曲。影片中女主也是被男主这种安全感给打动了吧，这首歌就是很有责任感，完美男人的感觉，羡慕ing。
