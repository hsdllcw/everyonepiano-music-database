

**星月神话**
是中国内地女艺人金莎演唱的一首歌曲，金莎本人填词谱曲并制作，收录于其2010年10月发行的同名专辑《星月神话》中，同时也是2010年播出的穿越情感剧《神话》的插曲之一。此曲和弦规范，非常适合学过[
**EOP魔鬼训练营**](sale-10-【EOP魔鬼训练营】无敌键盘钢琴指法教程-30天成为钢琴大师.html)的小伙伴们弹奏。  
  
而这个MV中主演延续《神话》中易小川的影子，每当女主人公路过熟悉的窗外，眼前都会浮现那段过去的悲惨爱情。为了完整呈现这首柔情味十足的《 _星月神话_
》，MV中也多次浮现了电视剧《神话》中和男主人公的经典片段，这仿佛是易小川又回来了，可闭上眼才发现，一切都是虚幻的。
