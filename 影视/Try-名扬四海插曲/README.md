

**Try双手简谱** 和五线谱完全对应。

_Try_ 是Asher Book的一首单曲。也是电影《Fame》(名扬四海)中的一首插曲。电影根据1980年艾伦-
帕克导演的经典歌舞片《名扬四海》改编。以纽约市的表演艺术高中为故事舞台，描述来自各个阶层和家庭背景的十个青少年，每个人怀着不同的梦想前来应考，经历各种挑战和奋斗之后，有人成功，有人失败，也有人从表演中吸收到人生经验而成长。

歌词下方是 _Try钢琴谱_ ，大家可以免费下载学习。
