

**星空双手简谱** 和五线谱完全对应。

_星空_ 是五月天演唱的一首歌，由石头谱曲、阿信填词，收录在五月天2011年发行的第八张专辑《第二人生》中，是电影《星空》的主题曲。

此曲有着明亮的吉他RIFF，整饬的鼓点，在英伦风味的同时，不失日式J-
Pop的唯美和细腻。歌曲展现了五月天在面对未来的恐惧时，咀嚼着生命中的温暖回忆，由此展现的义无反顾的勇气。
