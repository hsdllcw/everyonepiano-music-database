

《 **突然心动** 》是著名歌手陈晓东的作品之一。同时也是2013年电视剧《兰陵王》的插曲。

这首歌不管是词亦或是曲都温柔委婉，加之陈晓东异常令人心动的声音，似乎能让人一瞬间陷入其中。

《兰陵王》是由东王文化、柏合丽、上影英皇和上海耀客传媒联合出品的古装剧。该剧以一千四百多年前北齐和北周两个王朝相争为背景，讲述了北齐传奇英雄兰陵王金戈铁马的一生，以及其与天女杨雪舞之间的一段唯美动人的爱情故事。
