

电视剧《亲爱的，热爱的》的主题曲《 **无名之辈**
》由音乐制作人陈雪燃演唱。乍一听，好像与这部甜甜的爱情剧背道而驰，但剧作者却将本身的甜爱剧又拔高了一个层次，描摹了一群追梦人的气壮山河！

无名之辈，我是谁？

好像我们一生都在回答这道哲学命题：我是谁，我从哪里来，我要到哪里去……

与一众励志歌曲不同的是，《 _无名之辈_ 》没有大力鼓吹梦想的宏大、壮阔，它在唱每一个无名之辈的心路历程。

也许很累，一生狼狈；  
也许卑微，一生无为；

结尾却是那份豁达和从容。谁生来不是一样的？我曾努力过、拼搏过、灿烂过、热烈过，那么来吧，尽管叫我无名之辈。
