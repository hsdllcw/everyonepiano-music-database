

**只要有你双手简谱** 和五线谱完全对应。

_只要有你_
是孙楠&那英专辑中流行的情歌之一，也是电视剧《少年包青天》系列片尾曲。这首歌由孙楠、那英两个人合唱，由陈颂红填词并谱曲。另外《只要有你》还有邓超、纪丹迪演唱的版本，是《少年包青天3》的片尾曲。

歌词下方是 _只要有你钢琴谱_ ，大家可以免费下载学习。
