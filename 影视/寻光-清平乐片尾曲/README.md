

**寻光** 是郑云龙演唱的歌曲，由董颖达、朱朱作词，董颖达作曲，该歌曲是电视剧《清平乐》片尾曲，于2020年4月7日发行。

聆听《 _寻光_
》，伴随着郑云龙细腻悠远的声线，宋仁宗赵祯一生的线索徐徐铺开，一代仁君在朝堂风云和儿女情长之间的百般挣扎，在郑云龙的歌声中演绎得淋漓尽致，细微地还原了一位君王的隐忍与柔情，令人唏嘘的同时不禁对这惊艳的声音震撼。
