_怀念战友_ 是雷振邦为电影《冰山上的来客》创作的插曲。《冰山上的来客》于1963年由长春电影制片厂
制作发行，由赵心水执导，梁音、阿依夏木、谷毓英等人主演。影片从真假古兰丹姆与战士阿米尔的爱情悬念出发，讲述了边疆战士和杨排长一起与特务假古兰丹姆斗智斗勇，最终胜利的阿米尔和真古兰丹姆也得以重逢的故事。该片曾获1964年长春电影制片厂“小百花奖”的最佳导演奖。
