_你的情歌_
是电影《你的情歌》同名主题曲，由情歌天王Tank惊喜献声作曲。此曲一出，就引爆网路讨论声浪，更被网友封为是继《有一种悲伤》之后新一代的「洗脑神曲」。

Tank舍弃了平常习惯的R&B唱腔，因为太多转音和技巧反而会让这首主题曲没办法感动人心，「唯一能一箭穿心的方式，就是我抛开所有习惯的唱法，简单直接的唱出音符，就像画图有时候一笔成形，比复杂的描绘更清楚明白，让听的人能感受到这份情感的酸甜苦涩。」
