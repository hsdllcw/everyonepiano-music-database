

**我在未来等你双手简谱** 和五线谱完全对应。

_我在未来等你_ 是刘同继《不说再见》后再度与好妹妹合作全新小说《我在未来等你》的同名主题曲。

好妹妹是一支充满实力的民谣组合，他们不仅有着温柔的气质、柔和的唱腔，而且还有着独树一帜的乐队风格。好妹妹的亲切感也是他们的魅力所在，他们有着些许南国清新的味道。

歌词下方是 _我在未来等你钢琴谱_ ，大家可以免费下载学习。
