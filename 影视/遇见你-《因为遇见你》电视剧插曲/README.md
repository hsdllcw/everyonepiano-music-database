

**遇见你双手简谱** 和五线谱完全对应。

_遇见你_ 是牛奶咖啡组合演唱的一首歌曲，由小乔作词，格非作曲。电视剧《因为遇见你》的插曲。

该剧讲述了生活在馄饨店却拥有惊人刺绣天赋、神秘身世的少女张果果如何在时尚圈逆袭的成长励志故事。

歌词下方是 _遇见你钢琴谱_ ，喜欢的朋友可以免费下载学习。
