

**指纹** 是胡歌为电视剧《轩辕剑之天之痕》演唱的插曲。胡歌的声音雄厚且又带走一丝细腻，很容易被带去进故事里面。

《轩辕剑之天之痕》是由上海唐人电影制作有限公司出品，李国立、梁胜权、黄俊文联合执导，胡歌、蒋劲夫、刘诗诗、唐嫣、古力娜扎等主演的古装玄幻剧。该剧讲述了北周后裔宇文拓和陈朝少主陈靖仇由对立到联手，并最终手执上古神器轩辕剑净化妖星，修补天之痕，拯救苍生的故事。2012年7月6日在湖南卫视第一周播剧场首播。
