_冰天雪地_
是何宣林为青春爱情竞技成长剧《冰糖炖雪梨》演唱的主题曲。该剧根据酒小七同名小说改编，讲述了“速滑少女”棠雪在大学校园里重逢“冰球男神”黎语冰，两人在追逐冰上梦想的过程中，收获真挚爱情的故事。目前该剧发行至海外多个国家，包括俄罗斯、日本、泰国、韩国、新加坡、西班牙、美国等。

歌词下方是 _冰天雪地钢琴谱_ ，大家可以免费下载学习。
