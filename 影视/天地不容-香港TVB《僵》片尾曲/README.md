

**天地不容双手简谱** 和五线谱完全对应。

_天地不容_ 是香港TVB《僵》电视剧片尾曲，由胡鸿钧演唱，并本人在《僵》出演。

《僵》是香港电视广播有限公司拍摄制作的时装僵尸电视剧，由郑嘉颖、谢安琪、陈凯琳、陈山聪、关礼杰等主演，陈维冠担任监制。该剧讲述谢安琪饰演僵尸，在荷兰邂逅负责歼灭僵尸的“不死人”，让身边人卷入一场僵尸和不死人的杀戮漩涡的故事。2016年4月11日TVB高清翡翠台播出。

歌词下方是 _天地不容钢琴谱_ ，希望大家喜欢。
