

《 **大鱼**
》词美曲佳，歌手的演唱更抓耳，这着实是一首歌曲和歌者结合完美的经典之作。周深展现嗓音美感的同时，又能把歌里的文化意境一同渲染出来，两者相得益彰，分外出彩。所以，很难说是周深遇见了契合自己的《大鱼》，还是《大鱼》成全了周深。但无论从哪个角度来看，它都是值得去反复聆听和传唱的。

歌曲MV由电影《大鱼海棠》画面剪辑而成。《大鱼海棠》中讲：“爱一个人，攀一座山，追一次梦，不妨大胆一点”。人而为生，生而为乐。孟之汤，酒之醉，难抑心。此心之人，吾爱之至；吾失吾得，风雪相随。春去秋来秋又去，爱不悔，心中系。勇追梦，梦中人，人之心，心之所向，向心尔唯……
