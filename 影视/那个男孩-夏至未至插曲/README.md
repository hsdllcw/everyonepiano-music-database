_那个男孩_ 是电视剧《夏至未至》插曲。由汪苏泷作词、作曲并演唱，发行于2017年06月06日。

作为傅小司与陆之昂的“友谊之曲”，《那个男孩》所展现出的不只是青春里懵懂的青涩，更有属于透明的少年时代的坚定与真挚。那时头顶的蓝天永远是一张寂寞的脸，稚气的少年们幻想着未来，浮云将一切渲染上悲伤的釉质，在天空里发着光，而汪苏泷亦仿佛是站在时光里的歌者，用“理解的目光”，弹奏唱颂出这青涩的乐章。
