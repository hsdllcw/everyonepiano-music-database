

**你在终点等我双手简谱** 和五线谱完全对应。

_你在终点等我_
，天后的声音真的是很有韵味。歌词好美，“被所有的人误解都要理解你，准备好当擦亮你天际的浮云，没有你的地方都是他乡,没有你的旅行都是流浪”。爱的好伟大，期待这部电影，不知是悲是喜，不如听凭天意。

电影《从你的全世界路过》片尾曲《你在终点等我》，华语乐坛神级阵容倾力巨献，王菲献唱，陈小霞作曲，姚若龙作词，张亚东担当制作人。携手路过，相约终点。

歌词下方是 _你在终点等我钢琴谱_ ，希望大家喜欢。
