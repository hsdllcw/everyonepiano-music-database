

**叹云兮** 是古装剧《芸汐传》的片尾曲，由鞠婧祎演唱，发行于2018年7月2日。这首歌也是剧中女主的心理独白，因此歌名亦取谐音“叹芸汐”。

这是韩芸汐给秦王的绝笔情书！随着鞠婧祎的演唱娓娓道来韩芸汐的爱情，不是执手相看泪满眼，而是倾尽一生，换爱人一往无前。正应了庄子那句话，相濡以沫，不如相忘于江湖。可以说，这一曲叹云兮道尽了芸汐的深情执着。
