

任贤齐的 《 **天涯** 》原曲目为日本中岛美雪的《竹之歌》,
该歌意境优美，歌词极富感染力，一经发布，就红遍了大江南北，为任贤齐又一次迎来歌唱事业高峰。同时此曲也是2000年台湾中视电视剧《笑傲江湖》的片尾曲。

歌曲表达了为爱浪迹天涯、苦苦追寻，对爱执着，一往情深、此生不渝的意境！《笑傲江湖》是由赖水清、李惠民执导的古装片，任贤齐、袁咏仪、陈德容参加演出，讲述了任盈盈和令狐冲历遍重重波折险阻，令狐冲和任盈盈终排除万难，逍遥物外，笑傲江湖的故事。
