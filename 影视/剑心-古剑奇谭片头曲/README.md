

《 **剑心**
》是内地男歌手张杰演唱的一首歌曲。由段思思作词、谭旋作曲、王文颖编曲，是2014年古装玄幻电视剧《古剑奇谭》的片头曲，歌曲于2014年6月26日公布。

  

《剑心》由谭旋工作室打造，亚洲爱乐交响乐团担纲乐队演奏。主题曲作为《古剑奇谭》原声最重要的部分，一直是关注的焦点。制作方在歌手人选的方面也是多方考虑，在综合对比多位华语乐坛歌手的实力及人气之后，定下张杰来演唱。

  

歌词下方是 _剑心钢琴谱_ ，欢迎大家使用。
