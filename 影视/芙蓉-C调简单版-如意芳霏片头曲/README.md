

**芙蓉**
是鞠婧祎演唱的歌曲，由符酷作词，胡臻、SHIMA作曲，是电视剧《如意芳霏》的片头曲，于2020年10月21日发行。鞠婧祎细腻柔情的歌声醉人心扉。

《如意芳霏》是由爱奇艺出品、丝芭影视联合出品的轻古风甜爽剧。该剧改编自晋江女频笑佳人的小说《宠后之路》，讲述了傅容因一场预见人生的梦境，决心自强自立掌控自己的命运，与肃王徐晋共同经历一系列人生变故后相知相守的故事。
