

主题曲《 **不可说**
》由“仙侠音乐之父”麦振鸿作曲，独特而虐心的曲风延续了其后浪漫主义的音乐情怀，为电视剧《花千骨》更添神韵。曾作为签约歌手的霍建华声音深情沉稳，而在《陆贞传奇》中与陈晓合唱《心情》的赵丽颖声音空灵柔美，二人的默契合作令主题曲《
_不可说_ 》迸发出别样的韵味。

正如歌中所唱“千丈风波/万般蹉跎/情意都不曾变过”“一世牵绊/一念成祸/还执意一错再错”“不是不可说/是我不敢说/一步一微笑/一步一伤心/一步一劫难/化作飞蛾扑火”“因为爱上你/我才成了我”。MV中：少女花千骨与长留上仙白子画之间的点点滴滴也随之展现：桃花树下的痴痴地等待；为花千骨亲手带上宫铃；绝情殿中二人的恬静生活；白子画对花千骨默默的关怀；花千骨被罚痛受81根销魂钉……同时备受大家期待的神秘莫测的异朽阁、花千骨的妖神造型也在mv中隐隐呈现。
