电音配器打造出的磅礴气势，伴随充满画面想象力的采样间奏，旋律间有一触即发的激情，也裹挟逐梦高远的意蕴。陈雪燃在沉稳和爆发力间肆意游走的嗓音，构建出天高任梦想遨游的极致画面感，也氤氲着直冲云霄、为荣耀拼搏的精神。  
  
负重前行，是一生宿命。何必低头争输赢？既然“生来是鹰“，势必逆风而行。
