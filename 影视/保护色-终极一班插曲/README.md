

**保护色双手简谱** 和五线谱完全对应。

_保护色_ 由苏亦承主唱，Amanda作词，黄声峰做曲，是电视剧《终极一班》的插曲。收录于专辑《终极一班电视原声带》。

该曲是电视剧《终极一班》中主人公送给喜欢的女生的一首歌，以此来表达主人公的爱慕之情。整个曲风缓慢而温柔，歌词没有直接表达出主人公的爱慕，而是借助保护这个词表达出对女生的喜爱却又默默等待的情感。可以这样说，这是一首表达爱意并令人感动的歌曲。
