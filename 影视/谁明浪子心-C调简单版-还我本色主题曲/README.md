

《 **谁明浪子心**
》由王杰演唱，收录在1989年10月1日发行的同名专辑《谁明浪子心》中，是电视剧《还我本色》的主题曲。普通话版本为《她的背影》，由王杰作词作曲并演唱，收录在专辑《是否我真的一无所有》中。

这是一首沧桑的歌，只有经历过生活的磨难的人才能唱的如此动人，王杰绝无仅有的歌声扣人心弦，深深打动我们的心。谁明浪子心？在这个繁杂喧闹的社会中拼打厮杀，谁知道浪人埋藏在心中最柔软一面？谁明白浪人的无可奈何？谁会在意彷徨在街头的浪子？叶子的离开是因为树要成长，因此强装笑颜，因此头也不回。这世界上最高尚的爱是在消失之前不被发觉的爱。浪子带着一颗满是伤痕的心，带着对未来的憧憬默默的走开。心里比谁都明白，自己没有能力为她的未来负责，无奈只好选择离开。漂泊的心，漂泊的人，无法为所爱的她负担。而这首歌，激起了回忆。浪子不应该有感情，一旦有了感情，会让自己陷入痛苦的无底洞。
