_创造回忆_
是流星花园新版中的插曲，由王鹤棣，官鸿，梁靖康，吴希泽演唱。所属专辑《流星花园音乐专辑》。2018年7月23日发布。电视剧讲述了董杉菜怀揣着把家里的家厨事业做大做强的梦想来到了明德大学的营养系，却遇到F4之首道明寺，从而展开了一段关于成长和爱情的故事。
