

**塑料袋双手简谱** 和五线谱完全对应。

_塑料袋_ 是电影《缝纫机乐队》推广曲。由赵英俊作词、作曲，刘洲编曲，缝纫机乐队、Cyan Dragon演唱，发行于2017年09月05日。

《缝纫机乐队》是由他城影业、儒意影业、青春光线、万达影业、乐合影业出品，由董成鹏执导，董成鹏、乔杉、古力娜扎、李鸿其、韩童生、曲隽希等主演的喜剧电影。该片讲述了胡亮的家乡小镇集安，几个背景各异的小人物为了追寻共同的音乐梦想，组建了一支与众不同的摇滚乐队的故事。
