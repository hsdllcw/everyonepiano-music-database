

《 **光的方向** 》作为电视剧 《长歌行》的片头主题曲，由张碧晨演唱，发行于2021年4月26日。

歌曲以吉他、哨笛、弦乐等乐器弹奏营造出了大气磅礴的曲风，从主歌部分的缓和柔情一路高涨至副歌处的激昂，完美表达了女主角的不屈精神和强大信念。

演唱上，我们透过张碧晨饱满且不失爆发力的声线，既感受到了抒情氛围，也在高音诠释中体会到了她的绝佳唱功，张碧晨用可轻盈可浑厚的声音化作极具穿透力、说服力的锲而不舍，将对故事和人物的理解融进其中，以饱满的情绪成功诠释了一段高亢故事。
“我要凭这暗夜里的光，还它与一曲长歌相望”，同时配合上正能量满满的歌词，带给了听众极大的鼓舞和振奋。
