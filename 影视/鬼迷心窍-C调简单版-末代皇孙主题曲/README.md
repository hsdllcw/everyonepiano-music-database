

《 **鬼迷心窍** 》由李宗盛作词、作曲并演唱。同时也是1992年黄日华、周海媚主演台湾电视剧《末代皇孙》的主题曲。

只要自己真爱过的人，每一个人都会将他（她）深藏在心底。现代社会是一个复杂多变的时代，人们的情感受外来因素的干扰比较多，许多人的情感因此而出现了危机。《
_鬼迷心窍_
》这首词虽然未讲明两个人之间究竟由于什么原因而导致分手，字里行间充斥着淡淡的哀愁，但词人依然将浓浓的情意向所爱的人倾诉：“是鬼迷了心窍也好/是前世的姻缘也好/然而这一切已不再重要/如果你能够重回我怀抱。”一个情深似海的人才能说出这样震撼天地的告白，一个情深意切的人才能说出这样矢志不渝的心愿。
