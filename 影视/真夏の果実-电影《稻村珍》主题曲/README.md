

**真夏の果実双手简谱** 和五线谱完全对应。

_真夏の果実_ 是桑田佳祐1990年为自己首部执导的电影《稻村珍》创作的主题曲。歌曲动听，乐曲优美，此歌多次被翻唱。

本歌曲曾由邓丽君和绚香翻唱过日文版，还有中文翻唱版张学友的《每天爱你多一些》，方大同薛凯琪中文翻唱版《 复刻回忆》。

歌词下方是 _真夏の果実钢琴谱_ ，喜欢的朋友可以免费下载学习。
