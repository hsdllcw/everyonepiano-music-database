

《 **旋钮** 》由中国内地创作女歌手刘瑞琦演唱。同时，也是2021年许宏宇执导的都市情感剧《世界微尘里》的片头曲。

《世界微尘里》根剧木浮生的同名小说改编，讲述了艾景初和曾鲤两人的爱情故事。该剧不仅探讨了关于“初老”女孩工作恋爱上的焦虑问题，还打破了传统死板恋爱印象，展现了新时代女性勇敢追爱自由独立的状态，另一方面也传递出爱情不只有一种模样的观念。爱情的迷人之处正是在于它的多种可能性，不论是“清汤锅”“酸汤锅”还是“麻辣锅”，都有可能成为爱情最好的模样。请相信平凡的世界中，总能够找到适合自己“那一锅”的爱情味道。
