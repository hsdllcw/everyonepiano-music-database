

**逍遥双手简谱** 和五线谱完全对应。

_逍遥_
是王青演唱的歌曲，由汪苏泷作词及谱曲，郭超编曲，中国跨次元原创IP《长生歌》推广曲。这曲逍遥真够味儿，把人极力想突破那种状态表现的淋漓尽致！复古的小伙伴可能也会有那种古人情怀，仗剑走天涯。心情不好的时候，一人一剑足以表达所有的复杂情绪，一个字：帅！

该曲由章谋圣担任Bass手。汪苏泷在创作该曲时历经艰辛，光歌名就想了一百来个。起初决定歌名叫《归来》，后来觉得《逍遥》比较合适，寓意做事顺心、逍遥自在。

歌词下方是 _逍遥钢琴谱_ ，希望大家喜欢。
