

**友情岁月双手简谱** 和五线谱完全对应。

_友情岁月_
是郑伊健原唱的一首粤语歌曲，由刘卓辉作词、陈光荣作曲，类似beyond岁月无声，收录在1996年发行的《古惑仔》之人在江湖电影原声带，是电影《古惑仔》的主题曲，有多个翻唱版。

歌词下方是 _友情岁月钢琴谱_ ，希望大家喜欢。
