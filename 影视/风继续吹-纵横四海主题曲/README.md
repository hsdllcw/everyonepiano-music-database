_风继续吹_ 是张国荣演唱的歌曲，由郑国江填词，宇崎龙童Ryudo Uzaki、Youko
Agi作曲。收录在张国荣1983年发行的专辑《风继续吹》中。这首歌是张国荣的成名曲，同时也是他主演电影《纵横四海》的主题曲。

《风继续吹》所描写的意境是：一对恋人，因外力所必须分开时的一段离愁别恨。张国荣在用这首歌里用淡定坦然的方式去表达歌曲中的哀愁气息，那淡淡愁绪已让人心领神会，这就是张国荣在演绎这首歌曲时候让人感觉最舒服、也最佩服的地方：不动声色，不着痕迹，却又尽显风流。

对于这首歌，可以形容为：健康的感伤，浪漫的踟蹰。离别时的缱绻在这首歌里是值得记取与尊重的。这样的态度，不仅是人生感喟，也能看出社会对情感的态度，是感性与理性并重的时刻。什么样的社会样貌，就诞生什么样的情绪，唯有一个向上蓬勃的社会氛围才能够产生如此至情至性的歌曲。
