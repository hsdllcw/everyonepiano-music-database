_树洞_
是欧阳娜娜演唱的歌曲，由吴斌作词，林采欣作曲，田汨编曲。该歌曲也是都市情感剧《小欢喜》的插曲。《小欢喜》描摹出一幅中国式家庭“浮世绘”。该剧设置了三个高三考生家庭，都很具代表性。该剧聚焦的是高考，而更深层的落点是教育和亲子关系。
