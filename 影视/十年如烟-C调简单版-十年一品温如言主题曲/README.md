

“十年如烟，唯愿你安好”电影《十年一品温如言》主题曲《 **十年如烟** 》由中国流行乐女歌手郁可唯献声，温婉绵长歌喉勾勒出言希温衡二人跨越十年的虐恋故事。

MV在郁可唯深情婉转的声线下，言希与温衡十年间相爱与分离的爱情故事缓缓展开。“回想过去，我多希望你可以平安快乐。”“过去答应你的每一句承诺，都在我心里。”MV中言希与温衡相恋、相离，终相守，相爱可抵岁月漫长。影片讲述了言希与温衡跨越十年的爱情故事。正如歌词中所唱“我们十年如烟，不过一笑；谁是谁非，谁太年少；世界很糟，有个你就好。”
