

**周旋** 是2021年张瑶演唱的一首歌曲，由林乔、零作词、都智文作曲，收录于专辑《你是我的城池营垒 电视剧影视原声带》中。

在剧中有着精彩表演的张瑶，她在歌曲中诠释出为爱情周旋的乐观。表达了直来直去的爱情观，拒绝互相猜测彼此做阅读理解，大方沟通才能描绘出想要的幸福模样。
