

《 **念归去** 》是古装奇幻剧《镜•双城》的主题曲，由周深演唱，发行于2022年1月12日。此曲旋律唯美婉转，融入了许多古风特点鲜明的元素。

《 _念归去_
》前奏箫与二胡接连奏响，渲染着哀伤萧瑟的氛围，周深的低吟浅唱，仿佛在向我们娓娓道来男女主人公之间的情爱交织。“问光阴，借一卷，忘不掉的纪念，却不见，却不见，曾以为的缱绻”，歌词处处透着满怀的无可奈何与人物内心的感伤，光阴亏欠、承诺难全、离愁别怨，反复拉扯纠缠着人心的这些遗憾，絮絮轻诉成曲。
