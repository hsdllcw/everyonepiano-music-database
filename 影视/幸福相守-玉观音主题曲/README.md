_幸福相守_ 是电视剧《玉观音》主题曲，由歌手孙楠演唱。歌词表现了对未来对幸福的追求。

《玉观音》是导演丁黑执导的一部电视剧。由孙俪、佟大为、何润东等主演。该剧讲述了缉毒女警察安心如何从一个单纯普通的女孩子最终成长为一名打入贩毒集团内部的坚强战士的故事。剧集于2003年5月27日由北京电视台首播。
