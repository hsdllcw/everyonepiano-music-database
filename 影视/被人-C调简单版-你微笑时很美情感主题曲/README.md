

6月23日，薛之谦新歌《 **被人** 》上线，苦情延续，情歌盛宴。此曲由薛之谦作词，郭冠廷和薛之谦作曲，薛之谦演唱。同时也是电视剧《你微笑时很美》的插曲。

《你微笑时很美》是由裘仲维执导，许凯、程潇领衔主演的生活轻喜类甜宠剧。该剧讲述了被游戏“抢”走男朋友的懵懂少女童谣，因战绩出色收到游戏战队ZGDX邀请，签约成为职业选手的故事。
