

《 **古画** 》是古装言情励志剧《如意芳霏》的插曲，亦是该剧曝光的首只曲目，由剧中女主角傅容的饰演者鞠婧祎倾情献唱。

歌曲以灵动浪漫的钢琴起声，渐入古韵悠长的民乐乐器，循声而来的是鞠婧祎细腻而柔情的嗓音，将水墨纸上一则写不明诉不尽的爱情故事缓缓铺展在听者面前。

歌曲的一字一句似在讲述一对相恋的璧人却人各天涯，无法相见，只能将誊满思念的信笺寄往彼方，聊以慰藉。而“红尘停留在纸上，晕开的思念被融化”的结尾在鞠婧祎宛如清涟的柔美声线演绎下，仿佛留给了听者一个纵有千言万语却无法纾解的思念意境。
