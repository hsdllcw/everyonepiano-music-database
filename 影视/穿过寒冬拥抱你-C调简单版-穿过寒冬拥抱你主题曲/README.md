

“冬日最暖情歌”《 **穿过寒冬拥抱你** 》由王源演唱，亦是电影《穿过寒冬拥抱你》同名主题曲。

歌词由王源与唐恬共同创作而成，两位老搭档的默契合作让每一句歌词都成了冬日最佳表白情话。歌中描绘的爱情，是“穿过大半个城市朝你奔去，想陪你去看冬天的星星”，是“不冷了吧，有我挡一城大雪”，是“爱是你的姓名，我要暖这冰天雪地”，是大雨中给你阳光、寒冬中给你拥抱。正是电影中的爱情给了他们创作灵感，因为疫情的背景，人与人之间的情感显得格外珍贵而特别，王源此次的创作也与他以往的情歌不同，在大时代背景的渲染下，普通人最寻常的情感也变得闪闪发光。他歌声中的人们“尝过刺骨，才懂得温柔。多脆弱，就多勇敢。”
