

**遗憾成全** 是电视剧《爱的理想生活》的主题曲，由殷桃、宋轶、赵今麦、夏若妍以及胡连馨演唱。

《爱的理想生活》是由李江明执导的都市情感剧。该剧改编自朱德庸的漫画《涩女郎》，讲述了五位性格鲜明、经历迥异的都市女性，在经历形形色色的生活困境之后，最终找到自我的故事。
