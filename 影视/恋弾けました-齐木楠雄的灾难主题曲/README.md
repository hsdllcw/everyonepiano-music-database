_恋、弾けました。_
是2017年漫画改编真人电影齐木楠雄的灾难的主题曲。此剧改编自麻生周一的漫画，导演福田雄一，讲述生来就有超能力的主人公高中生齐木楠雄，每天都面临着各种灾难的日常，是一部校园喜剧。

另外，此曲由日本著名乐队ゆず（柚子乐队）演唱。清新二人组创作乐团，由成员北川悠仁与岩泽厚治组成。他们所有歌曲均为两人自己作词、作曲，并参与大部分歌曲的编曲工作。
