

**盛开双手简谱** 和五线谱完全对应。

_盛开_ 是电视剧《猎场》的片尾曲，由韩葆作词，王晓锋作曲，胡歌演唱。

胡歌深情的演唱，道尽了他在《猎场》人生中的酸甜苦辣。歌词句句戳中泪点，让人感受到了无尽的忧伤。

猎场是东阳青雨传媒股份有限公司、浙江影视集团、蓝色星空影业出品的职场题材剧。该剧以都市社会为背景，讲述了郑秋冬职场颠沛、商海沉浮、十年蝶变，成为专业猎头的成长故事。

歌词下方是 _盛开钢琴谱_ ，大家可以免费下载学习。
