

**六月的雨** 是古装玄幻电视剧《仙剑奇侠传》的插曲，由该剧男主角胡歌演唱，演唱出了自己饰演的李逍遥为爱等待的真情真爱。此外，这也是胡歌出道的第一首歌曲。

歌曲以李逍遥告白似的口吻娓娓道来，唱出了李逍遥对爱的执着以及内心的思念与苦楚，配合着浪漫绵长的旋律，让我们体会着爱情的真谛。融入电视剧，就更加显得浪漫温馨。
