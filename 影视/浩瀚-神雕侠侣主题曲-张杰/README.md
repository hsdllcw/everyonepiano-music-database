

**浩瀚** 是由于正作词，谭旋作曲，张杰演唱的一首歌曲。该曲为于正电视剧《神雕侠侣》的主题曲。

  
《 _浩瀚_ 》歌词”，歌词中充斥着“好汉不提当年勇，只想问你懂不懂”等直白词句，网友纷纷表示“这个词太美我不敢看”。
歌词中“为你心动，为你吟颂，一曲高歌诉尽情衷。来势汹涌，去时想通，人生不过一场好梦”更是直接地表露心声，写出了人在江湖的身不由己以及对过往快意恩仇的潇洒释然。实力歌手张杰用他霸气的嗓音、宽广的音域引领我们进入了一个既有腥风血雨，却又充满儿女情长的武侠世界——潇洒坦荡的侠义里，实有荡气回肠的情感爆发。

  
歌词下方是 _浩瀚钢琴谱_ ，希望大家喜欢。

### 浩瀚歌词：

天生我才必有用，  
千万莫欺少年穷，  
败了也要逞英雄，  
不怕世人笑我疯！  
好汉不提当年勇，  
只想问你懂不懂，  
爱恨装得很从容，  
有谁真正能放松？  
四大皆空，色即是空，  
眼里全是，胭脂花红，  
醉在花丛，笑得心痛，  
谁来和我深情相拥？  
为你心动，为你吟颂，  
一曲高歌诉尽情衷！  
来时汹涌，去时想通，  
人生不过一场好梦！
