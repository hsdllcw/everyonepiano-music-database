

**等待双手简谱** 和五线谱完全对应。

_  
等待_ 是一首由韩磊演唱，葛根塔娜作词，张宏光作曲的歌曲，是电视剧《汉武大帝》的片尾曲，该歌曲被收录于韩磊2005年专辑《帝王之声》。

  
歌词下方是 _等待钢琴谱_ ，希望大家喜欢。
