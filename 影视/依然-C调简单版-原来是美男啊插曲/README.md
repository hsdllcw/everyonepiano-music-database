

**依然** ，韩语여전히，由韩国歌手李洪基演唱，同时也是韩国水木剧《原来是美男啊》的原声音乐。

《原来是美男啊》由洪成昌执导，洪氏姐妹编剧，张根硕、朴信惠、郑容和、李洪基联袂出演。该剧讲述了一位淳朴少女偶然代替哥哥女扮男装成为偶像组合的一员，围绕韩国音乐圈展开的故事，展现了偶像组合的成长过程、感情经历以及他们的困惑和烦恼。
