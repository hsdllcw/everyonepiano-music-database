《雷欧•奥特曼》是由圆谷粲、熊谷健导演的一部日本科幻电视剧，首部剧集于1974年4月11日在日本东京播映。该剧的制作路线颠覆了以往的奥特系列作品的传统，一经播映，该剧强烈的写实风格就与前作轻快明亮的风格产生巨大反差，在受众群中造成极大的震撼。在放映初期的平均收视率只有14%，为了应对这样的情况，从中期开始减少了剧中对过于严酷的训练的描写，也会偶尔尝试像泰罗那样的喜剧风格。但最终，该剧还是在年平均收视只有10.9%的悲惨状况下结束播映。而第二期奥特系列的制作也因为该剧惨不忍睹的收视率和当时因石油冲击导致的物价高涨而逼迫停止。
