_缘分一道桥_
是王力宏、谭维维合唱的歌曲。由方文山作词，王力宏作曲，于2016年11月15日首发，16日全网上线。后收录在王力宏2017年12月11日发行的专辑《A.I.
爱》中。这首歌曲也是张艺谋执导的史诗级魔幻电影《长城》的片尾曲。

歌曲MV远赴美国拍摄。这次为了《缘分一道桥》的MV还远从北京空运电影《长城》里的道具，只为拍摄出连贯电影画面的剧情故事。而MV剧情中饰演考古学家的王力宏走进一个被世人所遗忘的房间，只见房内堆放的杂物已满是灰尘，但当王力宏靠近桌子上的滚动条，竟浮出王昌龄作品《出塞》的古诗字句，让王力宏忍不住跟着诗词弹唱起来，转个身倚靠在墙上的镜子也交错出现了谭维维的影像，两人就透过镜子穿越不同的时空见到彼此。

歌词下方是 _缘分一道桥钢琴谱_ ，大家可以免费下载学习。
