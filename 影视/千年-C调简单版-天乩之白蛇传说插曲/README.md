

**千年** 是电视剧《天乩之白蛇传说》的插曲，由廖羽、和汇慧作词，王梓同、和汇慧作曲，金志文、吉克隽逸演唱，将男女主人公跨越千年的爱情纠葛娓娓道来。

而此曲也是金志文与吉克隽逸在阔别好声音舞台之后的首度合作，金志文细腻的嗓音与吉克隽逸独特的声线相呼应，将歌曲中的万般无奈却情深不舍演绎得淋漓尽致，令观众提前领略了一番剧中的相爱却无法相守的至深虐恋。
