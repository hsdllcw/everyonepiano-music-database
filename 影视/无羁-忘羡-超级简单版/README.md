国风音乐专辑》。同时此曲也是电视剧《陈情令》的片尾曲。该剧不仅呈现了传统文化之形，更呈现出中华民族之魂。剧中江氏家训“明知不可为而为之”，取自孔子《论语•宪问》“知其不可而为之”。
作为剧中主人公，魏无羡一生秉承江氏家训，从中能看出本剧想要表达的态度：无论结果与否，能与不能，只要有勇气去挑战自我，突破自我，这也是当代青少年所需要的精神和担当。
