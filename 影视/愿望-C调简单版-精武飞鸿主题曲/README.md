

**愿望** 是全能音乐制作人胡彦斌演唱的一首歌，作词、作曲、编曲、合音、监制全由胡彦斌一人完成，同时也是电视剧《精武飞鸿》主题曲。

喜欢这首歌里嘹亮明媚的笛声，干净而青春，悠扬洒脱着随性，自由自在的欢畅。静静的沉淀心绪，一切都开始变得透明澄澈起来，自在安然如处子般的纯净，一如牡丹般的雍容华贵。这首歌描述的是一种理想的生活状态，没有那么强烈的爱与恨，没有争名夺利，有的只是一种安静的氛围，随时都可以听，不管是对于失恋的人，还是对于失业的人，都很适合！
