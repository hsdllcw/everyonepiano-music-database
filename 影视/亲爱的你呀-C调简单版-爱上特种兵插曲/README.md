

影视剧《爱上特种兵》插曲《 **亲爱的你呀** 》由当代青年男歌手张远温情献唱，将甜蜜恋人间满眼的温柔和爱意缓缓道来。

轻柔婉转的弦乐配合长笛不绝于耳，吉他的铺陈、精准的贝斯和鼓点直击人心，配合张远温情轻柔的吟唱，描绘出沉浸在爱情中甜蜜恋人的满眼美好，仿佛一切微小的事物透过爱情的滤镜都会被放大成无限美好，闪闪发光；风中的情话，沉醉的晚风，化成对你的加倍思念，从此以后，只有亲爱的你呀才是我最温柔的牵挂。
