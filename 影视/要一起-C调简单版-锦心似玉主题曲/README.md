

由周深演唱的电视剧《锦心似玉》主题曲《 **要一起** 》于2021年2月上线！

歌曲运用丰富的配器，营造出浪漫色彩氛围。大气的弦乐与民乐交织，情感层层递进。周深灵动细腻的声线，时而轻快婉转，时而缠绵缱绻。他以独特的唱功和细腻的情感张力，一字一句将感情全数倾泻，诠释出侯爷和十一娘细水流长的爱情故事。

“要三月雨，七月晴，细数星星。要赏春光，画秋景，繁花看尽。要哪怕离千万里，心都呼应。有天白了双鬓也要一起。”歌词透过大量细节扎实氛围，并在副歌处层层递进两个相爱的人要一起的坚定主张，这都是来自于剧情生活与爱情交织的想象。旋律也把爱的深情和坚毅描绘出层次感，把“愿深情不被辜负”诠释得淋漓尽致。
