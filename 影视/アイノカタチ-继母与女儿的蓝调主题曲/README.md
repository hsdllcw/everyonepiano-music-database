

**アイノカタチ双手简谱** 和五线谱完全对应。

_アイノカタチ_ 是日剧继母与女儿的蓝调的主题曲。由MISIA和HIDE演唱。

《继母与女儿的蓝调》改编自樱泽铃的四格漫画作品，描述继母和女儿全心全意的爱和成长的10年间故事。绫濑遥饰演事业心强的岩木亚希子，被带着８岁女儿的单身爸爸宫本良一（竹野内丰饰演）求婚，婚后亚希子专职于家庭主妇，一直拼命努力在育儿和家务事，每天的日常生活中都充满着温柔，有时喜悦、有时悲伤、或有时候的温馨小奇迹。
