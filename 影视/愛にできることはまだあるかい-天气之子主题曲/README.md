_愛にできることはまだあるかい_
是日本原创动画电影《天气之子》的主题曲。该片讲述的是在气候调节失控的时代，被命运捉弄的少年森岛帆高和少女天野阳菜选择各自的生活方式的故事。影片代表日本参与竞争了奥斯卡金像奖最佳国际影片。

歌词下方是 _愛にできることはまだあるかい钢琴谱_ ，大家可以免费下载学习。
