

《良辰美景好时光》插曲《 **顺风**
》由音乐总监金大洲作曲，萨吉作词，赵贝尔演唱。这一次赵贝尔摒弃了以往甜美的歌声，搭配动感的律动，唱出了不一样的感觉，少年少女追逐情感时的美好被完整的诠释。爱上你，就像被爱的箭射中，顺着风来到你的身边，自然又温柔。

《良辰美景好时光》是由秦榛执导，徐璐、林一领衔主演，胡兵、胡耘豪、种丹妮主演的青春励志剧。该剧根据作家翘摇小说《明枪易躲，暗恋难防》改编，讲述了正能量歌手梁辰和计算机系大数据研究方向的大四学生陆景之间互相砥砺，共同成长的励志故事。该剧特邀著名音乐制作人金大洲担任音乐总监。
