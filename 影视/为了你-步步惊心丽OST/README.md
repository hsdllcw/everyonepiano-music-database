

**为了你双手简谱** 和五线谱完全对应。

_为了你_
是由EXO成员金钟大（Chen）、边伯贤（BAEKHYUN）、金珉锡（XIUMIN）演唱的韩剧《步步惊心：丽》的OST之一。同时，边伯贤在该剧中饰演十王子王银一角。

《步步惊心：丽》是韩国SBS电视台于2016年8月29日首播的浪漫奇幻古装剧，该剧根据中国小说《步步惊心》改编，讲述了在日全食现象中灵魂穿越到高丽的21世纪女人解树和四王子王昭之间的浪漫爱情故事，以及高丽宫廷火热的王权竞争。
