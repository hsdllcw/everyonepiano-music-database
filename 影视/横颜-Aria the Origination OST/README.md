

**横颜** 是牧野由依为电视动画系列《水星领航员》第三季《ARIA The ORIGINATION》演唱的插曲。

《ARIA The
ORIGINATION》讲的是以水星Aqua上的观光都市新威尼斯为舞台，围绕着一名以成为领航员为目标的少女水无灯里所发生的一些故事。整个故事风格温柔却又带着一丝的悲伤，再加上画风清新。

歌词下方是 _横颜钢琴谱_ ，大家可以免费下载学习。
