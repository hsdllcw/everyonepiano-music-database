_时间的远方_ 是中央电视台大型纪录片《记住乡愁》的片尾曲，孙楠演唱。

此曲表达出了远行游子的回家梦、思乡情。这首歌还把乡愁的地域性和时间性交织在一起，这从题目中就可以看出来。用“时间的远方”来描述乡愁，既有诗意，又有哲理。陈曦说：“时间的远方并不仅仅指旧日的时光，也指向未来，未来和过去一样都是时间上的远方。”歌词最后写道：“我看见孩子的成长，在心中最软的地方，它呵护了我的晴朗。”
