

《以家人之名》插曲《 **If Rain** 》上线引热议，温柔女声萨吉诠释大爱！

在该剧创作的过程当中，制片人和音乐制作人认为，一定要创作一首使用英文来演唱的歌曲，希望用这首歌曲来表达每一个孩子，都是在成长中经历着失望，却依然被爱着的人的情感。萨吉也一改往日的演唱方法，用最温柔细腻的声音来进行演唱，希望借此能唱出一种关爱着孩子的情感，让每一个人听到歌声里的最温柔的爱。

《 _If Rain_
》，可以理解为如果你感到失望。我们每个人都曾有视为珍宝的日子，希望时光能停在当时。可在你明明还在乎着的时候，它悄悄溜走了，所以你心有不甘的抱怨着，就像雨水打湿毫无准备的你。可就算雨水泪水分不清的时候，你看到心中依然有一束虹光，那是爱你的人为你照亮的引航灯。就算眼泪迷离看不清前方道路，你知道只要沿着那束光继续走，就会找到自己的道路。到那一天，所有的挫折终变成你的太阳，所有的被爱终会变成你的翅膀。
