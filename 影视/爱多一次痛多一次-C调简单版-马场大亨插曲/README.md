

《 **爱多一次痛多一次** 》是谭咏麟演唱的粤语歌曲，收录在其于1993年9月2日发行的粤语专辑《笑看人生》中。同时此曲也是电视剧《马场大亨》的插曲。

该歌曲改编自童安格演唱的歌曲《爱与哀愁》。由于这首歌是华语歌坛填词大师林夕填词，因此其韵味个人感觉大于其国语版。加上校长的演绎与卢东尼的编曲，使其味道更加谭氏，而略微少了一点童氏味道。
