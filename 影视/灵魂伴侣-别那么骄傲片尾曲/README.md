歌曲MV剧情中，田馥甄不断回忆过去两人相处情形，其中一幕是两人在庙会约会的场景。由于这是田馥甄第一次参加日本的祭典，让她感到非常新奇：「觉得很像日剧场景!参加祭典就是要穿得很漂亮，然后约班上喜欢的男同学!」另一场则在屋久岛取景，该地也是宫崎骏动画「魔法公主」中森林的取材之处。为求壮丽景观，一行人跋山涉水才深入至林中，田馥甄更为剧情所需，赤脚踏上溪边碎石，让她笑说：「好像在走健康步道，根本假唯美真疼痛!」
