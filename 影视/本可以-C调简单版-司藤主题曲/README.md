

**本可以**
是影视剧《司藤》主题曲，由摩登兄弟刘宇宁演唱，发布于2021年3月9日。和歌曲《本可以》一样，刘宇宁的歌声总能感受到温情，每次听他唱歌都是不一样的感觉，音色特别，把岁月的沧桑，融进歌声里。

《司藤》是由李木戈执导，景甜、张彬彬、李沐宸、张亦驰领衔主演，吴俊余特别主演的奇幻悬爱剧。该剧根据尾鱼的小说《半妖司藤》改编，讲述了设计师秦放意外坠入山崖，唤醒沉睡数十年的外星后裔司藤，后两人彼此守护共同成长的故事。
