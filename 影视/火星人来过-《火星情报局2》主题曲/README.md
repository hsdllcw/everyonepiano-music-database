

**火星人来过双手简谱** 和五线谱完全对应。

歌手薛之谦回归初心，为综艺节目《火星情报局》潜心创作最新作品《 _火星人来过_
》，正如谦谦本人所说，这绝对不是一首广告歌曲，与以往曲风略有不同，但歌词部分依旧很用心，加入的rap耳目一新，超级带感！

火星人来过是综艺节目《火星情报局2》的主题曲。

歌词下方是 _火星人来过钢琴谱_ ，希望大家喜欢。
