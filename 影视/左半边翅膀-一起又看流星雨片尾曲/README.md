_左半边翅膀_ 是由严艺丹作词，刘佳作曲，许飞演唱的一首歌，收录在许飞2010年发行的专辑《二十六度》中。同时这首歌是偶像剧《一起又看流星雨》的片尾曲。

当钢琴声一点一滴的流泻出来，《左半边翅膀》彷佛缓缓勾勒出一个故事的雏形，牵引着聆听者想起放在心里的那些故事。许飞轻柔的声线领着情绪走进她诉说的画面。吉他跟弦乐一层层的加入，铺排而后的提琴声，突如其来的就划进心里最深处。
