_有种_
是火箭少女101成员孟美岐演唱的歌曲，作为电影《流浪地球》推广曲于2019年1月15日发行。对现阶段的孟美岐来说，此曲简直是最好不过的个人SOLO首单，略带中二的歌词，朗朗上口的旋律，简直是为演唱者美岐本人量身定制。

《流浪地球》根据刘慈欣同名小说改编，影片故事设定在2075年，讲述了太阳即将毁灭，已经不适合人类生存，而面对绝境，人类将开启“流浪地球”计划，试图带着地球一起逃离太阳系，寻找人类新家园的故事。
