

刘德华、易烊千玺惊喜合唱，全“心”演绎《 **还是笨小孩** 》。而此曲也是电影《奇迹•笨小孩》的宣传主题曲，于2022年1月7日发行。

《笨小孩》是刘德华1998年演唱的作品，这首歌承载了很多人的记忆，是一首很励志，很容易让人产生共鸣的歌曲。《 _还是笨小孩_
》就改编自《笨小孩》。歌曲延续了90年代流传至今的经典旋律，加入贴合影片剧情的歌词，通过全新编曲营造出新时代背景之下为生活奋斗的“笨小孩”精神。

歌曲告诉大家：“不管在哪个年代、在哪个环境，也请在希望中保持努力、勤奋，坚持不懈拼博自己的人生，老天自有安排, 因为‘老天爱笨小孩’”。
