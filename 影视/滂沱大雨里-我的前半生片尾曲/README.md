_滂沱大雨里_
是2016年沈严执导电视剧我的前半生的原声音乐。我的前半生是新丽电视文化投资有限公司出品的都市情感剧，该剧讲述了生活优越安逸的全职太太罗子君与丈夫陈俊生离婚后一切归零，在闺蜜唐晶及其男友贺涵的帮助下打破困境，进入职场，在自我成长中走向人生下一程的故事

另外，此曲是李若溪演唱的一首歌曲，歌曲于2017年7月发行。
