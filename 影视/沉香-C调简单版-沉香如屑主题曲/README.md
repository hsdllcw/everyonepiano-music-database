“你在川中哭哭停留，我在尽头为你守候” 曲短情长，轻叹辗转柔情。一开口便让人沦陷！

《 _沉香_
》讲述的是故事中颜淡和应渊生生世世的爱情，这份爱即便是燃尽最后一缕香，妄渡最后一片海，也从未消失过。歌词中那些唯美的元素交织着旋律起起伏伏，钟兴民老师流畅凄美的编曲将这片汪洋的每一朵浪层层叠加，我们听到了张杰歌声中的温柔与坚守，也听到了张靓颖歌声中的心碎与叹息。谎言、恨意、剜心，这些仿佛都在随着那份赠予寂寞地燃烧着，渐逝却渐明，令人怅惋，亦令人沉醉。
