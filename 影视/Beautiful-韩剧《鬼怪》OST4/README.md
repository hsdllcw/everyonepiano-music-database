

**Beautiful双手简谱** 和五线谱完全对应。

_Beautiful_ 是韩剧《孤单又灿烂的神-鬼怪》的第四首插曲，由Crush献唱，旋律清新浪漫。

《孤单又灿烂的神-
鬼怪》是韩国tvN电视台于2016年12月2日首播的金土迷你剧，由李应福执导，金恩淑编剧，孔刘、金高银、李栋旭、刘仁娜、陆星材主演。该剧以东方传说为主题，讲述需要通过找到人类新娘来结束不朽生命的鬼怪与失去记忆的阴间使者以及自称是“鬼怪新娘”的一位“本应死去”的少女的奇妙同居生活。

Crush（申孝燮），1992年5月3日出生于韩国首尔特别市，韩国男歌手。2012年12月7日在《Red Dress》出道，代表作品有《Hug
Me》、《sometimes》、《잠 못드는 밤 (不眠之夜)》等。
