《 _遠き呼び声の彼方へ_ 》是日本1996年圆谷株式会社拍摄的特摄电视剧《迪迦奥特曼》的插曲，由日本的作曲家矢野立美创作。

作为一部经典的特摄剧，整部剧在很多方面都做得十分出色，背景音乐中有很多矢野立美的音乐，为故事的发展添加了相应的色彩。

时而紧促，时而平缓，时而令人紧张，时而催人落泪，让观者很好地感受到剧情的推进，更容易发现故事的意义。

《迪迦•奥特曼》（日语：ウルトラマンティガ，英文：Ultraman
Tiga）是首次引入“形态转换”设定的作品，并且是具有不同能力的三种形式。每种类型都是同一外貌不同的体色变化，而这种模式是制片人笈田雅人与包含万代在内的赞助商，在开发玩具而促销售额的推动下而成立，且也希望奥特曼的既定形象能有创新。
