

**一生有意义双手简谱** 和五线谱完全对应。

_一生有意义_ 是由罗文和甄妮演唱的一首歌，收录于专辑《影视怀旧金曲》中，也是《射雕英雄传之东邪西毒》主题曲。

爱情，永恒的主题。而这首歌曲所描写的也是爱情。在人海之中，遇到了你，人生从此有了意义。当下我们也经常说遇到对的人天天情人节，遇到错的人天天都是清明节，可见两情相悦才是美满爱情的开始。
