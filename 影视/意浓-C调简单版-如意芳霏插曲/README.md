

**意浓** 是由许佳琪演唱的歌曲，由郭德紫毅作词，张净、阿沁作曲，于2020年10月21日发行。同时，此曲也是轻古风甜爽剧《如意芳霏》的插曲。

《如意芳霏》该剧改编自晋江女频笑佳人的小说《宠后之路》，讲述了傅容因一场预见人生的梦境，决心自强自立掌控自己的命运，与肃王徐晋共同经历一系列人生变故后相知相守的故事。
