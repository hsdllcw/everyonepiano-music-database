

古装青春校园网剧《天舞纪》片尾曲《 **都不会变**
》正式上线。此曲由全能音乐人陈雪燃操刀制作包揽作曲、编曲，实力女声叶炫清演绎纠葛在个人命运与世界变迁之间的爱情。

歌曲在恢弘的旋律下构建出风云变幻的天启国背景，融合磅礴弦乐层层丰富奏出错中复杂的爱恨。叶炫清清澈的歌声张力十足在旋律中自由游走，给“都不会变”的呐喊增添了一层笃定。

《天舞纪》根据同名小说改编，由赵锦焘执导，许凯、吴佳怡领衔主演，哈妮克孜、周骏超、李俊辰等主演，讲述了擎天城小霸王李玄与背井离乡来到天启国完成使命的少女苏犹怜在摩云书院共同学习，从欢喜冤家到互相爱慕，最后和一众摩云学子历经挑战，最终迎来和平新纪元的故事。
