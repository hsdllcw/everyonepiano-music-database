

**叮咛** 是毛不易为电视剧《都挺好》演唱的片尾曲。

这首《 _叮咛_
》不但延续了一贯毛不易独有的“毛式”忧郁唱腔，更是由罗大佑亲自“操刀”作词，可谓是令人期待已久。一个是如今华语乐坛的新星唱将，一个是华语乐坛“宗师”级的老前辈，这种实力的组合，光是想一想都让人忍不住要去听一听这首《叮咛》了。
