《 _还在这里_
》，MV中，以那些校园中熟悉的角落、记忆中的课桌、绿草地等美好画面开始。随即男主角程铮一个帅气走过，电影中的故事也开始了。“也曾说了再见，清空痕迹，以为能平息了回忆”，伴随着这句朦胧而又诗意的歌词，动听的男女声合唱悠扬传来
