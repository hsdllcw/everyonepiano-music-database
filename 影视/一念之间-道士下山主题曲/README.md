

**一念之间** 该歌曲由张杰、莫文蔚演唱，董冬冬谱曲，陈曦填词，也为陈凯歌导演电影《道士下山》主题曲。

_一念之间_
，不仅表达了影片中各个角色之间细腻的情感触碰，更对影片的宣传语“一门之隔，两个世界”的观点做出了诠释，就像影片中王宝强饰演的何安，通过道观之门下山入世，从世外清净之地来到充满诱惑的红尘世界，一念之间可能做对可能做错，可能坚守自我，也可能迷失了自己。

歌词下方是 _一念之间钢琴谱_ ，希望大家喜欢。
