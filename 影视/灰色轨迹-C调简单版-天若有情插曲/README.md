

《 **灰色轨迹** 》是中国香港摇滚乐队Beyond演唱的一首粤语歌曲。同时也是1990年刘德华、吴倩莲主演的电影《天若有情》的插曲。

听这首歌的前奏你就像踏进无穷无尽的滴水深渊，让人感到空虚迷茫。而在高潮阶段黄家驹发自内心的深情呼唤：“我已背上一身苦困后悔与唏嘘。”充分表达了迷茫无助的心情。而在此处电吉他的伴奏虽然只是几声轻轻地往复，但却别有一番风味。歌曲的尾奏也十分有味道，不禁让人沉醉其中。
