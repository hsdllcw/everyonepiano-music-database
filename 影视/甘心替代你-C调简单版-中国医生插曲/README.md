

毛不易献唱《中国医生》经典金曲《 **甘心替代你** 》。其细腻又温暖的歌声触人心弦，让人不禁想起中国医护人员挺身而出，挽救无数生命的场景。

2020年至今注定是不平凡的。这场在全国，乃至全世界蔓延的疫情，让我们的生活发生了改变。无数医护人员抗战在危险的第一线，他们的辛勤付出为我们筑起了一道健康的屏障。不怕牺牲，用于奋斗，正是这些可爱的人们才使我们能够平安健康。也正是这场疫情，让我们感受到了大家的团结。

而《中国医生》描述的就是这样的场景：拖着病体依然战斗抗疫一线的院长、满脸印痕久难归家的医生、顶着巨大压力仍旧工作岗位上不下岗的护士、为住户们奔波补给的外卖小哥……一幕幕都让观众感动。让我们向医护人员致敬！
