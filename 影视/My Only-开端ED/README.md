

2022年1月10日，周深最新单曲《 **My Only** 》上线。这是一首纯英文歌曲，被用作时间循环短剧《开端》的片尾主题曲。

《开端》这部剧剧情充满悬疑，节奏紧凑。而周深演唱的这首歌让人感觉到了浪漫，尤其是“You are my
only”，让人印象深刻。歌曲开头以简洁深情的钢琴弹奏，搭配周深空灵且治愈的嗓音，主歌部分轻盈细腻地将情感娓娓道来，副歌部分则以激昂有力的演唱迸发出强大的情感能量。可以说在周深明亮、温暖、治愈的歌声中，让听众感受到了剧作所传达出的温情，更感受到了爱才是最大能量这一主题。
