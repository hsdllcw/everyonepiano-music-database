

**你是对的人**
是戚薇和李俊昊（2PM）演唱的一首歌曲，由戚薇填词，郑宇界作曲。该首歌是电视剧《爱情回来了》的片尾曲，收录于2014年发行专辑《爱情回来了 电视原声带》中。

《 _你是对的人_
》把女生的细腻恰如其分带入，编曲上呈现轻巧的节奏铺陈。而演唱者戚薇和李俊昊两个人的嗓音搭配及情感表达，也把相互的感觉表达得惟妙惟肖，后部分的rap部分设计循序渐进别具匠心，正如恋人之间的呢喃独白，不仅拉开了音乐的起伏，也把甜蜜的氛围一步步的推向高潮。
