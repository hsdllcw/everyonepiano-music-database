

《 **太阳月亮** 》是由王乐达作词、作曲，毛不易演唱的歌曲，发行于2020年12月28日。同时，此曲也是电视剧《流金岁月》的片尾曲。

夙起的月光，温柔而孤独，挑起了漆黑夜晚的灯。在无常的生命轨迹里，岁月往往把她带去不曾想过的地方，但一切无论怎样都难忘。

静静聆听，漫天的金光就像时光海洋在缓缓流淌，泛起的海浪，映射出每一朵遇见的缘分。

毛不易温雅的歌声将故事娓娓道来，把月光缓缓递入你的梦乡……
