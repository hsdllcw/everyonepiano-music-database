

张杰演唱的电视剧《古剑奇谭》片尾曲《 **剑心**
》深得观众心，张杰首次演唱仙侠电视剧主题曲，新鲜感极高，加之他本身高亢嘹亮的声线很适合这种题材。《剑心》伴随着电视剧的热播而广为传唱 ，同时也成为了经典 。

《 _剑心_
》与该剧情搭配的相得益彰，饱含炽热情感的歌声再次虏获了观众们的心。《剑心》剧中重要场景与视觉特效的展现，配合张杰的声音，歌曲全程大气磅礴，令人热血沸腾 。

2015年3月30日，此曲获得第22届东方风云榜音乐盛典年度十大金曲；2015年11月7日，歌曲获得第15届全球华语歌曲排行榜年度20大金曲奖。
