

古装言情剧《玉昭令》的插曲《 **念未别** 》，由张靓作词，邓智伟作曲并担纲制作人，张紫宁演唱。

张紫宁空灵唯美的嗓音和充满倾诉感声线将剧中“端木翠”与“展颜”的虐恋与纠缠的爱情故事缓缓道出，感人至深。“恨只恨天太远遥遥相见，触不得情深缘浅，盼只盼轮回间繁花未歇，待到梦醒时得以成全”字字锥心的歌词与轻缓悠扬的歌曲旋律相得益彰，加之张紫宁的细腻婉转的演唱，深情诠释了剧中男女主人矢志不渝的爱情真谛。
