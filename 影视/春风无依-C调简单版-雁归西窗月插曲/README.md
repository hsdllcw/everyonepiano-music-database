

**春风无依** 是司南演唱的歌曲，由李俊宜、浅紫作词，姜帆作曲，该歌曲是古装爱情剧《雁归西窗月》的插曲，于2021年5月20日发行。

《雁归西窗月》该剧讲述了霸道郡王赵孝谦与率真少女谢小满的乌龙甜怼恋，两人从开始的欢喜冤家到逐渐相知相守，携手历经坎坷，冲破重重阻碍，最终收获了一段浪漫的爱情故事。
