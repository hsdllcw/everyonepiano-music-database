

《 **偏爱**
》随着《仙剑奇侠传3》的播出而迅速增加了传唱度，歌曲沉静婉转且充满爆发力，《偏爱》也将张芸京既拥有男生低沉音色，又拥有女生清澈高音的好嗓子发挥得游刃自如，也使张芸京自己得到情感上的一次爆发。歌曲本身是一首相当工整的抒情摇滚，歌词与曲式相当契合。

歌曲MV中，张芸京本人并未出镜，MV是由《仙剑奇侠传三》的正片中剪辑而来，MV中着重表现了徐长卿和紫萱的相识相恋相知的过程，还描绘了徐长卿与紫萱的三世不忘的羁绊，与歌曲本身中所表达的偏执的价值观所一致。

至2009年底，《 _偏爱_ 》在网络上的搜索视听量高达76,402,788次，张芸京也凭借此曲获得2009年“娱乐沸点”颁发的“年度十大金曲”奖。
