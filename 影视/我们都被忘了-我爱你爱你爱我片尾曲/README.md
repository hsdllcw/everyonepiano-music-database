_我们都被忘了_
是谢安琪演唱的一首歌，由易家扬填词，徐浩谱曲。收录于谢安琪2013年9月20日发行的国语专辑《谢安琪》中，是偶像剧《我爱你爱你爱我》的片尾曲。

此曲说的是伤心难过的失恋故事，讲述当爱情消逝之后我们都被忘了的故事。谢安琪认真的呼吁当你身边有失恋的朋友时，一定要特别关心他们，用爱去陪伴他们度过这个低潮。也希望借由她的歌声能让大家的情感有一个宣泄的管道，能更勇敢。
