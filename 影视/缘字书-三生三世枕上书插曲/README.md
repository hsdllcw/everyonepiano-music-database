_缘字书_
是苏诗丁演唱的歌曲，由段思思作词，谭旋作曲，该歌曲是电视剧《三生三世枕上书》的插曲，于2020年1月21日发行。《三生三世枕上书》是由嘉行传媒承制，杨玄执导，迪丽热巴、高伟光领衔主演，陈楚河、郭品超特别出演，刘玥霏、刘芮麟、王骁、李东恒、袁雨萱、黄俊捷等主演的唯美炽爱仙恋剧。该剧根据唐七同名小说改编，讲述了青丘帝姬凤九和太晨宫尊神东华之间两千年的纠葛情缘故事。
