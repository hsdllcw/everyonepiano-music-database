

《 **一生等你**
》是电视剧《媚者无疆》主题曲，由陈曦作词、陈曦和董冬冬作曲，袁娅维演唱的一首歌曲。“一生一世一期一会”，“一字一句一心一意”，八个“一”字，勾勒出的是对意中人的向往，是此生等你一人的坚定，是对爱的不求回报。

整首歌没有用一件民族乐器，旋律也是始终是相对平缓流畅地进行，但作为一部带着魔幻色彩的古装剧的主题曲，你丝毫不会觉得有任何违和感。在《 _一生等你_
》这首歌里，袁娅维在声线的厚实与柔媚之间做到了完美平衡，气息的运用和转音的处理，始终牢牢抓住了“我以此生等你一人”的丰沛情感。
