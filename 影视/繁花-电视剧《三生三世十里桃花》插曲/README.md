电视剧《三生三世十里桃花》插曲《 _繁花_
》于2月14号情人节前夕正式上线。该歌曲由创作型音乐才女董贞作词作曲，并演唱。歌词以剧中女主素素视角，表达了对男主夜华的深情却不痴情。

《三生三世十里桃花》是由杨幂、赵又廷、张智尧 、迪丽热巴领衔主演的古装神话电视剧，讲述了青丘帝姬白浅和九重天太子夜华的三生爱恨，三世纠葛的故事。

歌词下方是 _繁花钢琴谱_ ，喜欢的朋友可以免费下载学习。
