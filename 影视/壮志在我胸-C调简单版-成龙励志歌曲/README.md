

**壮志在我胸** 是成龙演唱的一首励志歌曲，同时也是电视剧《江湖再见》的主题曲。

歌曲告诉我们：  
无论世事如何变幻，不要丢掉自己的真诚与善良；  
无论风花雪夜还是沧海桑田，都不要丢掉做人的尊严！

有时候我就在想，谁内心不曾隐藏一些暗暗的追求，在多少看似平凡的下面，其实都有一颗不平凡的涌动的心，我们在这样一个世界，在这样一个无所不能的时代，也许谁都能实现心中那个小小的梦想！加油！
