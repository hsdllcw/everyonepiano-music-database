

都市言情励志剧《我的时代，你的时代》片尾曲《 **心动**
》由全能音乐人陈雪燃创作，摩登兄弟刘宇宁倾情演唱，轻快流畅的律动搭配复古合成器，奠定了歌曲甜蜜温暖的基调。

刘宇宁标志性的深情嗓音里添了份轻柔与浪漫，唱给心里如星空般闪烁的女孩。“征服一片纯白无边梦海...直到我登上冠军领奖台/金牌一定要留给你来戴”，这是吴白与艾情的约定，心动是梦想的起点。
