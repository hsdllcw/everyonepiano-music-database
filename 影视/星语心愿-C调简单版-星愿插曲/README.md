

《 **星语心愿**
》是为1999年爱情电影《星愿》而创作的插曲，演唱者张柏芝扮演了该片的女主角。电影《星愿》讲述了既盲又哑的洋葱头向暗恋的护士秋男（张柏芝饰）表白后却意外身亡，他以别人的身份返回人间与秋男相认并道别的故事。

2000年，此曲获得第19届香港电影金像奖最佳原创电影歌曲奖 。2001年，张柏芝重新演绎了Accoustic版，收录在专辑《全新经验（新曲+精选）》里。

《 _星语心愿_
》给人一种新的体验，让普通人感受到盲人的内心世界。歌曲传达了这样一种感情，虽然盲人无法感知到世界的多彩，但他们有一颗明澈的心，有着不放弃的信念。
