

《 **鬼迷心窍** 》是李宗盛在飞机上看见漂亮的空姐有感而发创作的一首歌曲。1993年，李宗盛凭借该曲获得第一届新加坡醉心金曲奖最佳作词奖。

歌曲像是在说一个男人千帆过尽回首往事，一段当年情涌上心头，赫然回望佳人已去欲求不得，遂后晦不迭吟唱哀歌。歌曲感慨“虽然情爱总是让人烦恼”“斩了千次的情丝却断不了”，鬼迷心窍的“我愿意随你到天涯海角”是对深陷情爱难以自拔却又无怨无悔的诠释。
