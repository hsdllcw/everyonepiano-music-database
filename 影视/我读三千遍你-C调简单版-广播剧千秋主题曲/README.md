

**我读三千遍你**
是广播剧《千秋》的主题曲。剧中讲述了晏无师不相信人性本善，却在偶然之下看到重伤濒死的沈峤的故事。这首歌就是晏无师唱给沈峤的心声，太美好了，作词简直神仙！他看着沈峤，从戏弄到深爱。读他三千遍，发现他既不是柔弱的雪也不是漂浮的云，而是皎洁的明月。即使晏无师不相信人心之善，他仍愿意为了沈峤守这世界的温柔，只为了看沈峤无忧无愁的笑。
