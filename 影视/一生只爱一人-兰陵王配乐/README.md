_一生只爱一人_ 是古装青春偶像剧《兰陵王》的配乐。由V.K克演绎。

《兰陵王》由钟澍佳、周晓鹏导演，东王文化、柏合丽、上影英皇和上海耀客传媒联合出品，冯绍峰、林依晨、翟天临、魏千翔、毛林林、胡宇崴等人主演，该剧总投资近一亿人民币。该剧以一千四百多年前北齐和北周两个王朝相争为背景，讲述了北齐传奇英雄兰陵王金戈铁马的一生，以及其与天女杨雪舞之间的一段唯美动人的爱情故事。中国大陆于2013年08月14日在东方卫视、浙江卫视、深圳卫视、云南卫视播出，中国台湾于2013年8月23日在中视上档，该剧还在中国香港、韩国、泰国、日本播出。
