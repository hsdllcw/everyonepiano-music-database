

**遗忘之前双手简谱** 和五线谱完全对应。

_遗忘之前_ 是徐佳莹演唱的一首歌曲，由龚淑均作词，衣睿作曲，是电视剧《女医·明妃传》插曲。

《女医·明妃传》插曲《遗忘之前》，徐佳莹特别以强调韵脚咬字的演唱方式，让听觉更加温婉，也更增添了女人味儿，表达谈允贤面对自己内心情感时的矛盾：明明心里记挂的是一个，视线却离不开另一个人，那种被压抑的情愫，略带伤感，却依然有着甜蜜。揪心的情歌，请小心品尝。

歌词下方是 _遗忘之前钢琴谱_ ，希望大家喜欢。
