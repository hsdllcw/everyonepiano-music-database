

**You're Cold**
是韩国女歌手Heize为韩国爱情剧《虽然是精神病但没关系》演唱的插曲。Heize的声音总是在述说故事似的娓娓道来，是情感的渗透，非常好听。

《虽然是精神病但没关系》由朴信宇执导，赵容编剧，金秀贤、徐睿知主演，于2020年6月20日起在韩国tvN电视台播出。该剧讲述了因生活压力而拒绝爱情的精神病患者监护人，和因天生缺陷而不懂爱情的童话作家相遇后互相抚平伤口治愈伤痛的故事。
