

薛之谦应《胜女的代价》电视剧之约，创作了《 **几个你** 》作为其插曲，
并在《几个薛之谦》专辑中作为首波主打。歌词用素描的手法，细致而真实地写出了想要忘记一个人却不断想起的纠结情 绪，如同倾诉心事，又像自言自语，令人动容。

此曲由薛之谦作词作曲，绝对是他亲生的歌曲，这就是品质的保证。老薛亲生，必是精品。这首歌很有电影的画面感，每次听歌的时候，都仿佛打开记忆的投影机，闭上眼睛，脑海中都会出现各种画面，而且常听常新，每次脑海中出现的都是不同的画面。
