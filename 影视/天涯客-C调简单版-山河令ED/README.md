

**天涯客** 是《山河令》的片尾曲，由张哲瀚和龚俊演唱。

《山河令》是由慈文影视、优酷出品，成志超、马华干、李宏宇执导，小初编剧，张哲瀚、龚俊、周也、马闻远、孙浠伦主演的武侠剧，于2021年2月22日在优酷视频独播。该剧改编自Priest所著小说《天涯客》，讲述了浪迹江湖的鬼谷谷主温客行和离开庙堂逍遥自在的周子舒，因为江湖事牵扯在一起，相识相知的故事。
