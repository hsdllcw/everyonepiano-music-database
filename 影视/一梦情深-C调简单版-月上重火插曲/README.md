

电视剧《月上重火》插曲《 **一梦情深** 》由清彦作词，苏晴作曲，灰原穷、银临共同担任制作，古风歌手银临、王敬轩（妖扬）倾情献唱。

歌曲以极富古典韵味的弦乐拉开帷幕，词情曲意将剧中爱恨纠葛娓娓道来，银临与妖扬的歌声交相呼应，缠绵悱恻，唱尽两人命运眷恋纠缠的一生。

剧中，上官透与重雪芝因缘际会相遇，历经磨难相知，最终携手余生，正如歌中所唱：甘愿梦里沉沦永不分……纵然相聚别离，从来不由你我，惟愿梦里与你共此情深。
