

**你是我所有** 是由刘宇宁演唱的歌曲。同时也是《良辰美景好时光》影视剧片尾主题曲。

《良辰美景好时光》是由秦榛执导，徐璐、林一领衔主演，胡兵特别主演，胡耘豪、种丹妮等主演的青春励志剧。该剧改编自晋江翘摇小说《明枪易躲，暗恋难防》，讲述了正能量歌手梁辰和电竞系大四学生、人气主播陆景之间互相砥砺，共同成长的励志故事。
