

歌曲《 **花愿** 》由谭松韵演唱，同时也是电视剧《锦心似玉》的插曲，于2021年3月1日发行。

谭松韵用她温暖清亮的嗓音以声传情，浅吟轻唱间，将徐令宜和十一娘之间的爱情演绎的跃然于耳。前奏钢琴声缓缓流淌，与弦乐交织，清脆的琵琶声始终贯穿其中，谱写出一篇“锦心流转，流光似玉”的动人恋歌。更表达出锦心夫妇彼此“两情相悦，两袖缠绵，永不厌”的忠贞不渝。
