时至今天说起《泰坦尼克号》，仍有一部分人会莫名的脸红，因为有一些片段让人羞羞哒。。。当朋友问他/她电影讲了什么，对方只清楚的记得那时的小李子，还是帅炸天的小李子。还有一部分人会站出来指责：我从未见过有如此厚颜无耻之人。小李子演技不好吗？温斯莱特演的不好吗？Low,
形而上的low。  
  
小编在此说两句，20年前的电影现在依旧常被大家提起，不仅归功于两位主演的颜值，还是两位牛哄哄的演技。电影也常被进行另类演绎，比如小岳岳河南话版本的《泰坦尼克号》，男女主角成了夹克和肉丝儿，而对话则是这样的：“妮儿，弄啥咧？”“咋咧，滚，跳船。”“海水可凉啦。你看你这个妮儿长得可带劲咧。”  
女主爱自由，男主就是自由，两人看对眼很正常，只是生不逢时，半路就阴阳两隔。  
  
现实中温斯莱特2009年获得了奥斯卡最佳女主角，小李子则开始了长达十九年的奥斯卡影帝长征路，样貌和身材也越来越随心所欲。小编感叹，岁月是把刀，怎么使用得看用的人。  
  
电影火了，男女主角火了，主题曲《我心永恒》和演唱者席琳•迪翁也火了。记得几年前小编看过迪翁的一次采访，她说当时录这首歌的时候，她心情不好，也不喜欢这首歌，所以唱的有些不足，如果现在让她重新录，她会唱的更好。但时间就是这样，只有一次机会。也许正是因为她当时的状态录出来的声音才是大家喜欢的，这首歌才成了经典的经典。  
  
_我心永恒_ （英语：My Heart Will Go On (Love Theme from
Titanic)），又名《我心依旧》和《爱无止境》，电影《泰坦尼克号》主题曲。  
  
此曲由好莱坞主流电影著名作曲家詹姆斯·霍纳（James Horner）一手炮制，加拿大流行天后席琳·迪翁（Celine
Dion）演唱，在公告牌空降冠军，成为公告牌历史第8首空降冠单，后因这首歌曲在美国限制发行，所以在公告牌仅拿2周冠军。具有浓烈民族韵味的爱尔兰锡哨，精巧编排，尽显悠扬婉转而又凄美动人。歌曲的旋律从最初的平缓到激昂，再到缠绵悱恻的高潮，一直到最后荡气回肠的悲剧尾声，短短四分钟的歌曲实际上是整部影片的浓缩版本。  
  
《 _My Heart Will Go On_ 》曾使Celine
Dion在第四十一届格莱美颁奖仪式上独拿两项大奖：最佳年度唱片，最佳流行女歌手。并获得第70 届奥斯卡最佳电影歌曲，为电影泰坦尼克的片尾曲。Celine
Diony也因主题曲《My Heart Will Go On》（我心永恒）而为全世界歌迷所熟知，并一度创下每3秒钟便卖出一张专辑的纪录。
