

《 **不论** 》是由江辉作词，朱海威作曲，阿YueYue演唱的歌曲，发行于2021年9月1日。同时，此曲也是电视剧《周生如故》漼时宜人物推广曲。

剧中周生辰和漼时宜的虐恋，非为虐而虐，更近于悲剧里不可抵抗的宿命。在宿命面前缴械投降，那不叫悲剧那叫苦情；悲剧之所以激发观众的崇高感，正在于人物对宿命的拼命抵抗，然而最终还是失败了。这固然衬托了宿命的强大和无常，但人性的光辉、爱情之唯美纯粹，也因此得到凸显。
