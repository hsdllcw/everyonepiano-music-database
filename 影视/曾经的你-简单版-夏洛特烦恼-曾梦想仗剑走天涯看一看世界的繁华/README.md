我想大抵是我们的回忆里都有一个共同的名字：青春！

歌曲《 _曾经的你_
》是许巍写给自己的，更是写给世间每个人的，被用作喜剧电影《夏洛特烦恼》的插曲。这首充满悲伤情感的歌曲，是一首对青春岁月的怀念之歌和无悔宣言。

曾梦想仗剑走天涯  
看一看世界的繁华

如今你四海为家
