

**枷锁双手简谱** 和五线谱完全对应。

_枷锁_ 是电视剧《柒个我》片尾曲，由高颖作词、作曲并演唱。收录于专辑《柒个我 电视原声带》中，发行于2017年12月25日。

《柒个我》是由华策影视集团出品，张娜担任总制片人，汪文青担任制片人，邓科导演，张一山、蔡文静、张晓谦、高泰宇、范梦、张双利等人主演的中国首部以多重人格为题材的悬疑爱情偶像剧。该剧讲述了拥有七重人格的财阀三世沈亦臻和他的秘密女主治医生白欣欣的治愈系、搞笑、浪漫、爱情故事。

歌词下方是 _枷锁钢琴谱_ ，大家可以免费下载学习。
