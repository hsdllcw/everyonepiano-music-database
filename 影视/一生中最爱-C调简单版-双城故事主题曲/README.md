

《 **一生中最爱**
》是由向雪怀作词，伍思凯作曲，谭咏麟演唱的一首歌，唱出了爱情中真挚与坦诚的力量。同时，此曲也是1992年香港电影《双城故事》的主题曲。

每一次听到这首歌，都能让人想到在电影《双城故事》里面谭咏麟和张曼玉在船上抱着他的时候，每一次看到这里，都会流泪。歌曲旋律和歌词配合得异常出色，再加上谭咏麟极具磁性的唱腔，经典就这样诞生。
