

**最后一次**
是郑秀文演唱的一首歌曲，出自专辑《我们的主题曲》，电影《爱你爱到杀死你》插曲。郑秀文在华语地区享有高知名度，是上世纪90年代中期至2010年代初期，香港很成功的女歌手，她以百变形象和唱跳俱佳著称，被誉为“梅艳芳接班人”
等称号。

《爱你爱到杀死你》是由李力持执导的影片，由香港天王天后黎明与郑秀文合演，是一部轻松幽默的爱情喜剧，电影中还有二人多首经典歌曲。
