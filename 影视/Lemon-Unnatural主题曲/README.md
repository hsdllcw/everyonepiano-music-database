_Lemon_ 是电视剧Unnatural的主题曲，由米津玄师演唱。

Unnatural（日语名：アンナチュラル）是日本TBS电视台播出的医学悬疑剧，由冢原亚由子、竹村谦太郎、村尾嘉昭执导，野木亚纪子编剧，石原里美主演，洼田正孝、井浦新等共演，于2018年1月12日开播。该剧讲述了在“非自然死亡原因研究所”任职的法医三澄美琴探查死者真正死因的故事。
