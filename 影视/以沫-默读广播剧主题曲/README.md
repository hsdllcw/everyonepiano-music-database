

**以沫双手简谱** 和五线谱完全对应。

_以沫_
是广播剧默读的主题曲，由奇然与小魂演唱。《默读》是priest所作的耽美网络小说。小说于2016年6月15日在晋江文学城开始连载，于2017年1月8日全文连载结束，共180章并5篇番外。其作品多正剧向，语言诙谐幽默，行文流畅，风格大气，善于构建各种各样正常不正常的世界观。文笔成熟，能够把对于人生哲理的探究和情节较和谐地融为一体。

歌词下方是 _以沫钢琴谱_ ，大家可以免费下载学习。
