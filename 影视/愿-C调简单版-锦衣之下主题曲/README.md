

**愿** 是周深演唱的歌曲，也是《锦衣之下》电视剧主题曲。

寻着悠扬而带着哀伤的前奏，开口的霎那被触动心弦。犹如谁在耳边徐徐轻诉那愿的魂绕梦牵，梦的往返流连。周深再次表现出他无与伦比的感染力，一贯深情缱绻的歌声与百转千回的吟唱间，仿佛已经随着曲调，一同走过了故事的起承转合与悲欢离别。
而这一切，又随着你的吟唱，化为云烟。

《 _愿_
》叙述着《锦衣之下》陆绎袁今夏的感情之路。MV脉络清晰情节甜蜜，一开始陆绎是清冷孤傲的，对于今夏的恭维讨好并不领情。随着时间的推移，二人在探案日常中，渐渐互相了解，暗生情愫。接着跟随音乐的低吟，外界纷扰，是非不断，二人感情也面临考验，陆绎袁今夏内心矛盾，彼此纠缠，好在最后的他们彼此解开心结，深情相拥。
