

歌曲《 **我会守在这里** 》作为电视剧《以家人之名》片尾主题曲，由萨吉作词、金大洲作曲并担纲制作、毛不易演唱。

歌曲整体真挚而温暖，简单质朴却饱含爱意的歌词，搭配动人悠扬的旋律，加之毛不易娓娓道来的深情演唱，在歌声中温暖治愈直击内心。

成长是一个孤单的旅程，但总有一些眼睛在默默地注视你，你的喜怒哀乐牵动着他们的心，那就是我们的家人。可随着我们的成长，我们会发现在某一个瞬间，突然意识到自己已经是大人了，意识到父辈们其实在渐渐变小。但无论怎样，我们的家人会一直守护在我们身旁，去陪伴、支撑我们继续前行。
