

**一生所爱双手简谱** 和五线谱完全对应。

_一生所爱_ 是电影《西游降魔篇》主题曲，由郑嘉嘉作词，卢冠廷作曲，舒淇演唱。

原版《一生所爱》
在大话西游里在结尾部分出现，经过全片故事情节的发展，至尊宝与紫霞的爱恨交织，观众的情绪已经饱和，古城上夕阳剑客和情人的拥抱，加上这首歌算是一个宣泄点。卢冠廷其人父亲是粤剧演员，卢冠廷本人自小移民，毕业于美国专业的音乐学院，这首歌的编曲不像歌词那样古朴的中国味感觉，反而融合了许多世界音乐，因其复杂性让人百听不厌。
