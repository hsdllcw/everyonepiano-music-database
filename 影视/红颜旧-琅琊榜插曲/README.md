_红颜旧_
由著名演员刘涛倾情演唱，由赵佳霖作曲、袁亮作词，是大型古装传奇电视剧《琅琊榜》的插曲。该曲曲风婉转优美，唱出了一曲凄婉动人的爱情悲歌。此歌为翻唱，歌曲原名《忍别离》，原唱崔子格，收录于专辑《小美人》。

独特的女性化视角使词曲风格自成一派，加之刘涛满带柔情又不失感染力的音色，使人在聆听时不由的随之带入。在这样一部男人戏中，女性角色戏份虽不多，却各具风姿,
红颜旧MV中所呈现的琅琊阁、赤焰战场等场景以及莅阳长公主大殿呈冤、梅长苏跪拜林氏祠堂等等片段画面的闪现。

歌词下方是 _红颜旧钢琴谱_ ，希望大家喜欢。
