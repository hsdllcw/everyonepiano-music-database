

《 **恋爱ing**
》由五月天演唱。同时也是电视剧《门当父不对》的片头曲。“Be动词+动词ing”是英文的现在进行式，五月天始终和音乐恋爱ing，所以也快乐ing！

还记得，小编第一次与五月天结缘，是他们的歌曲《[ **倔强**](Music-11029-倔强-五月天.html
"倔强")》，之后便越陷越深，一发不可收拾。2005年《 _恋爱ing_ 》的发行，让我彻底入坑，紧接着后来的《[
**知足**](Music-1964-知足-五月天.html "知足")》，《私奔到月球》，《离开地球表面》等等，每一首都深入人心。

陪你熬夜聊天到爆肝也没关系  
陪你逛街逛成扁平足也没关系

都说恋爱中的男女智商会降低，净做一些毫无意义的事情。因为是恋爱进行式，所以聊天聊到爆肝、逛街逛成扁平足都没关系。而这些都是恋爱中最常见的事情。

那么问题来了，小编想知道大家恋爱的时候，最喜欢做的都是哪些，欢迎评论区留言哦！

_恋爱ing钢琴谱_ 比较简单，学过《[
**黄金VIP版魔鬼训练营**](/sale-47-EOP魔鬼训练营黄金VIP版.html)》的朋友勤加练习都可以掌握的，加油！
