_不如笑归去_
是电视剧《醉玲珑》的插曲，由陆菱纱作词，胜屿作曲，周传雄演唱。周传雄独特的嗓音带有细腻的情感，搭配古风传统韵味十足的配器，演绎了玲珑夫妇在第二时空虐心爱情的交错纠缠。出生入死相爱相伴，家国天下的情怀，歌词中“琴声碎/曲终心字成灰/浮世来回/人意薄如云水/最怕是负你/这一生千行泪/笑去归/惟愿与你长醉”可以看出元凌（陈伟霆饰）对凤卿尘（刘诗诗饰）爱意的表达及虐心的命运，传递的情感也带动着观众的情绪。
