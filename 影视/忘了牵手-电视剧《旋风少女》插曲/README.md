

**忘了牵手双手简谱** 和五线谱完全对应。

_忘了牵手_ 是内地组合牛奶咖啡为电视剧《旋风少女》演唱的插曲。在电视剧《旋风少女》12集中演唱。收录于牛奶咖啡同名专辑。  
  
忘了牵手由王程浩，富妍作词，格非谱曲。弦律唯美动声，受到人们的喜爱。

歌词下方是 _忘了牵手钢琴谱_ ，喜欢的朋友可以免费下载学习。
