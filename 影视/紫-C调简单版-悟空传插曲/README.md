

电影《悟空传》于2017年7月13日在中国内地上映，而由蔡健雅演唱的电影插曲《 **紫**
》于7月14日正式发行，歌曲以电影中孙悟空与阿紫的凄美爱情故事为创作灵感，诠释心中有爱但却来不及相守的深深遗憾。歌曲在音乐与制作上斥重金打造，田丁作词，配乐人波多野裕介作曲，制作人郭顶加盟，此次也是蔡健雅与郭顶的首次合作，二人细心打磨歌曲，小到每句唱词的表达，精益求精。

歌曲开端便将听众置身于幽静又空灵的环境中，带有明显东方元素的曲调犹如扑面而来，代入感十足。蔡健雅低声浅唱，唱腔冷静而内敛，淡雅中有种感人的温度。她的歌声中带着诗意的哀叹与不及的悔恨，带着丝丝忧伤，轻轻诉说一段悲伤的爱情故事。
