

**劫** 是电视剧《琉璃》的插曲，由王明毓作词，和汇慧&王梓同作曲，音频怪物演唱。

《琉璃》根据十四郎的同名小说《琉璃美人煞》改编，原著以“甜虐”硬核出道。本剧讲述了天生“六识”残缺的少女褚璇玑和离泽宫弟子禹司凤在面临爱情与前世阴谋的双重压力下，携手共对百般腥风血雨，缔造了一段极具色彩的仙侠传奇的故事。该剧在人物设定和剧情走向上贴合原著，画风唯美，服装道具等方面也充满中国韵味。
