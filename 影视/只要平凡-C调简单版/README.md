

“不要神的光环，只要你的平凡”是张杰、张碧晨原唱的歌曲《 **只要平凡** 》。歌曲契合2019新冠疫情，致敬疫情期间每一位伟大的平凡人，泪目。

此曲在致敬小人物的同时，对深刻的生命主题也给出了理解。歌词如“放过对错才知答案”、“跳动心脏长出藤蔓，愿为险而战”等等，在是非对错的暧昧界线中，以“握紧手中平凡”道出关于人生最质朴的回应。歌词娓娓道来，例如其中“没有神的光环，只要你的平凡”一句，让观众在电影和音乐的演绎中，感受到了小人物的真实魅力。

张杰与张碧晨在这首歌曲中完成了一次理性与感性的碰撞结合，将人间的痛苦、无奈、离别一一倾诉，却又蕴含着一份饱经风霜的坚韧，这首歌曲的情感也因此得以升华，更加富有层次感。
