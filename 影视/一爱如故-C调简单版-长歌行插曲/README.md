

《 **一爱如故**
》是摩登兄弟刘宇宁演唱的歌曲，同时也是古装历史传奇剧《长歌行》的插曲。歌曲以皓都与李乐嫣这一对人物的情感戏为主线，描绘出对彼此的执着、付出与不悔，正如歌词中所唱出的“爱一见如故，用尽一生去投入倾注”！

《长歌行》是由浙江华策影视股份有限公司、上海琰玉影视文化传媒有限公司、企鹅影视、上海琰然影视文化传媒有限公司联合出品，朱锐斌执导，迪丽热巴、吴磊领衔主演，摩登兄弟刘宇宁、赵露思、方逸伦等主演的古装历史传奇剧。该剧由夏达的同名漫画《长歌行》改编，这部漫画作品曾荣获中国动漫金猴奖、金龙奖。
