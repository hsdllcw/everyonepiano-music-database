

古装爱情励志剧《嘉南传》片头曲《 **莫离** 》，由剧中女主角嘉南郡主的饰演者鞠婧祎亲自演绎，于2021年10月18日发行。

凭借对角色的感同身受，鞠婧祎用隐含猜忌、试探的声线引出“两颗少年心”。爱藏于心，一生守护，歌词展现了一种双向奔赴的爱情。然而“谁料窗外风动，扰我一卷清梦”，你侬我侬和甜言蜜语，必会经历多舛命运和黑白必争的试炼。凭借生死莫离的坚定心意，两人终于长相守，共赏一帘风雅。而歌曲略带戏腔，很有代入感，扣人心弦！
