《 _怪奇物语主题曲_
》源自Stein早期创作但未使用的作品，其最终用于可作为潜在的商业授权的作品库中。两人与制作人员分享一段较“松散”版本潜在的样本，制作人员认为重制才能与片头画面吻合。Dixon和Stein着手重新修改样本，并根据工作人员的建议，使其“更大、更大胆，并将架构推向高潮”，这个目标最终随着Prophet-5、Roland
SH-2、Mellotron合成器的加入和运用各种音频过滤而得以达成。他们也制作其他长度不同的版本，以方便将主题曲与当时尚未完工的暂定片头整合。
