

电视剧《乔家的儿女》主题曲《 **生活总该迎着光亮**
》由周深演唱。温柔清澈的歌声，似是缓缓道出生活中细小温馨的每一个瞬间，让人感受到家庭与生活的温情，让人不自觉微扬嘴，去体会其中的真情。

歌曲开头“多想一醒来晨风就能抚平忧伤，多想一转身目光就融化掉冰霜......”通过8个“多想”的罗列描绘了一副亲情画卷。

接着，高潮部分“生活总该迎着光亮，再跌跌撞撞也要坚强......”可以说非常好听，周深的完美空灵嗓音唱出了对于美好生活的期盼，同样这样的歌声也像是和一位老朋友久违的谈心，听了之后有一种如释重负的感觉。在他的歌声里，我们能深深地感受到生活的百般滋味以及应对生活的勇气和力量，激励着我们在未知旅途中，无所畏惧，披荆斩棘，勇往前闯。
