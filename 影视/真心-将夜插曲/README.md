

《 **真心**
》作为影视剧《将夜》中的插曲，其有代岳东操刀制作，又有岑思源编曲。在歌曲中富有诗意般的语句与旋律中的相思之情，贴合了影视剧中的情景对白。在庄心妍深情款款的演唱下，用歌声讲述了一个故事。让人在听歌途中，慢慢带入了回忆之中。
