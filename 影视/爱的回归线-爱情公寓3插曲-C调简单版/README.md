

《 **爱的回归线** 》是电视剧《爱情公寓3》的插曲。陈韵若、陈每文演唱，收录在《爱情公寓3 电视原声带》中。

第一次出现在《爱情公寓3》第九集41:52宛瑜展博吵架，只是配乐，无歌词。  
第二次出现在《爱情公寓3》第十集23:17宛瑜离开的时候，是完整的歌曲。  
第三次出现在《爱情公寓3》第十九集40:06子乔拿出手链给美嘉，同样是配乐。  
第四次出现在《爱情公寓3》第二十集05:38美嘉介绍自己的新男朋友:吕子乔，也是配乐。  
第五次出现在《爱情公寓3》第二十四集
37:08曾小贤开始抛硬币到38：25关谷闯进礼堂（第一部分，从开头到“爱凝结了时间”）；从40:57关谷开始向悠悠告白的时候开始到43:45，是完整的歌曲（个别部分有歌词）。  
第六次出现在《爱情公寓5》第一集49：03子乔美嘉领证宣誓，是完整的歌曲。
