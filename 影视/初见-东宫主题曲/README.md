

**初见双手简谱** 和五线谱完全对应。

_初见_
是余昭源与叶里演唱的电视剧《东宫》的主题曲。《东宫》是由唐德影视、优酷、华录百纳、华视娱乐、美浓影视联合出品，李木戈执导的古装言情剧。该剧根据匪我思存的同名小说改编，讲述了本是西州国的九公主曲小枫，因为和亲踏上了中原之路，与当今的太子李承鄞之间发生的感情纠葛的故事。

歌词下方是 _初见钢琴谱_ ，大家可以免费下载学习。
