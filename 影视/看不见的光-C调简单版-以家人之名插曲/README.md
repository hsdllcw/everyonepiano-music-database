

歌曲《 **看不见的光** 》作为电视剧《以家人之名》插曲，由萨吉作词、崔瑟智与南命局共同作曲、赵贝尔演唱、金大洲担纲制作。

夜空在人们常规的认知中，是黑色的。可落日将光藏在里面，星星用微弱的力量拆解掉阴影，月亮又一丝丝把夜的缝隙填进去，总会显现出些许光亮。

正是因为这些一点一滴努力着、藏在夜空中看不见的光，夜空变得不再是让人失明的黑，而是一种全新的颜色，可能这就是所谓“不顺从”的颜色吧！不顺从宇宙赋给自己的黑，总在自己改变着要发出最亮的光。

这样的夜空就像每一个走在低潮中的我们，在对抗逆境的过程中心中蕴藏着光，终究会一点点释放出来，以此来让我们的梦重新发光。
