

**小幸运** 是Hebe田馥甄演唱的歌曲，由Jerry C谱曲，徐世珍及吴辉福共同填词。此外，这首歌也是电影《我的少女时代》的主题曲。

_小幸运_
，此曲在吉他与钢琴的编制下，回忆的曾经就像是每个片段般，揭开青春的序幕；而磅礴的弦乐及鼓声的交织，随着田馥甄的声线起伏，像是诉说一段过往逝去的悸动，同时也带你忆起青春的撼动、重燃年少情怀的那份感动。

###  
青春，遇见你，很幸运：

  
爱上你的时候还不懂感情  
离别了才觉得刻骨 铭心  
为什么没有发现遇见了你  
是生命最好的事情  
  
\-------------------小编最爱，没有之一！！！  
  
歌曲很伤感。作为一个异常感性的小编，对这类歌曲真的是毫无免疫，开口跪，好听到哭（第一次的听的时候真的是哭的稀里哗啦）。  
  
这个世界似乎有这么一个真理---
失去后才知道珍惜。是的，很多人往往都是若干年回过头的时候才明白：为什么当时自己那么任性，为什么得到的不懂珍惜。然而错过了就是错过了。  
  
歌曲，电影相得益彰。就像电影里说的那样，世界太快，总有一首歌，你忘不掉；人海茫茫，总有一个人，守候在角落，等你回首。  
  
青春是段跌跌撞撞的旅行，林真心和校园老大徐太宇正是在这种跌跌撞撞中展开了自己爱情故事。可遇而不可求的感情是每个人都追寻的，却不是每个人都可以拥有的。故事是有灵魂的，情感是有温度的，所以不管现实怎样，都能沉浸在一场美好纯净的梦境当中。笑的尽兴而归，哭的淋漓尽致。  
  
也许这就是每个青春路上相爱的人应该的样子吧——在经历过人生的跌跌撞撞和坎坎坷坷之后，你才明白，你曾经拥有的都是小幸运。

歌词下方是 _小幸运钢琴谱_ ，希望大家喜欢。
