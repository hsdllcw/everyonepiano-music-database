

《 **护花使者** 》是中国香港男歌手李克勤演唱的一首粤语歌曲，收录于李克勤1991年6月9日由环球唱片发行的粤语专辑《雨中街头剧》中。

2018年，《 _护花使者_
》被用在喜剧片《西虹市首富》中，出现在王多鱼签约夏竹后。《西虹市首富》讲述了混迹于丙级业余足球队的守门员王多鱼，因比赛失利被开除离队，却因继承二爷遗产而获得大笔财富，由此引发一系列令人哭笑不得的事件。
