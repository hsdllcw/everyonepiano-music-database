

**千年之约** 是由文颖秋、刘冠帅作曲，潇彬作词，霍尊演唱的歌曲，是电视剧《上古密约》的主题曲。

《上古密约》该剧从上古神话故事中汲取灵感，通过实景采风、特效渲染的方式，构建出完备的、具有浓重国风色彩的神话世界，无论是道具还是布景，都包含着丰富的想象，展现了上古神话的奇幻与瑰丽。剧集以峳国和琅族的争端为始，在抽丝剥茧地叙事中，将上古五正神官以生命为代价守护天下苍生的故事娓娓道来，传递出舍生取义、勇担道义的英雄主义精神
