圣诞歌曲也成了人们最受欢迎的歌曲，这首《 **White Christmas** 》就是其中最广为流传的一首。

Inn）写的。该作品获得了第15届奥斯卡金像奖的奥斯卡最佳原创歌曲奖，Berlin还因此片获得最佳原著故事金像奖提名。

Crosby演唱的版本是世界上最畅销的单曲。
