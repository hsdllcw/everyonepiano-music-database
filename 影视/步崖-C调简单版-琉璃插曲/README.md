

电视剧《琉璃》插曲《 **步崖** 》由中国男歌手李琦深情演绎，段思思、谭旋、李乃刚齐助阵，带着爱情誓言走向地老天荒。

剧中司凤的每次心动都有这首BGM，而此曲就像是在描述司凤的心境。这一生沟沟坎坎，那么多悲欢离合，那么多礼仪规则，于他人来说的障碍在我看来都只是一场修炼。谁说藏于天空的智人感受不到痛，他们不过是看过太多世俗众生的脸，才会心生怜悯，让我们此生能够相遇。且看做山水相依，愿与你不离不弃。

《 _步崖_ 》讲述了一个人对爱情的感叹和坚守。歌词、旋律温柔如水，李琦深情演绎着这段美好的故事，让这段爱情哪怕到天涯海角，也依然有奋力一搏的勇气。
