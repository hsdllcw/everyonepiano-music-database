_赤子_
是电视剧《陈情令》温宁角色主题曲，由于斌演唱。《陈情令》是一部围绕少年成长史展开的仙侠剧，讲述了主人公魏无羡在前世受奸人所害，重生后与故友蓝忘机携手探寻真相，锄奸扶弱的故事。这部剧不仅情节虐人，各个角色更是充满了悲剧色彩。在他们迥异的性格背后，隐藏着不同的创伤经历和处理方式，而这，也成为驱使他们走向不同命运的关键。
