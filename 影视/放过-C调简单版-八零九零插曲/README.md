

《 **放过** 》是电视剧《八零九零》插曲，由马頔演唱，于2021年4月29日发行。

歌曲重点描述了亲情在人生不同阶段，对人心的抚慰作用。纯正民谣的编曲，配上马頔温暖而有厚度的声音，将人们藏在岁月中的心事慢慢抹开，透出亲情在生命中的伤疤和光亮。

我们每个人都可能会因为原生家庭的一些负面影响，去挣扎，去责怪，去选择赌气不原谅。但一旦你做回真实的自己，以成人眼光给童年记忆以新的诠释之后，你就会明白，或许，长辈只是把挣扎和牵挂都埋在心底，那双大手一直在你身后，当你需要的时候，他义无反顾地挡在你的面前，不计得失。

《八零九零》讲述了几个90后年轻人，与一群年近耄耋的老人在一家养老院相遇，在年轻人管理并重建养老院的过程中，双方逐渐重构亲情、友情和爱情的故事。作为一部现实主义题材剧，《八零九零》聚焦当下全社会关注的养老议题，用诙谐轻松的手法，多线交错的讲述，呈现出了温暖治愈的风格。
