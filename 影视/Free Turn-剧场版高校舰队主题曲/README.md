

**Free Turn** 是剧场版动画《高校舰队（青春波纹）》的主题曲。《高中舰队》是由A-1 Pictures负责制作，2020年1月18日上映。

同时，此曲由TrySail演唱。TrySail（日文名：トライセイル）是由日本Music Ray'n
Inc.（ミュージックレイン）旗下所属麻仓桃、雨宫天、夏川椎菜所组成的声优组合。唱片公司是SACRA MUSIC。2015年开始组合活动。
