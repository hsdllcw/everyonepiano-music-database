

**勋章** 出自鹿晗首张个人专辑《Reloaded》，同时也是电影《我是证人》的主题曲。歌曲围绕电影中温暖励志的情感调性进行创作，可谓是正能量爆棚。

《 _勋章_
》的英伦吉他曲美学，管弦乐团的大编制，以及填词人朱婧汐所描绘的罗马神话场景，为整首作品增添了史诗色彩。鹿晗的声音中传递出来的是不卑不亢，不慌不忙。该曲编曲以管弦乐合奏展现海洋的磅礴气势，歌词表达了对英雄主义和追逐梦想的歌颂。
