

《 **遥远的歌** 》是刘惜君为电视剧《你好，旧时光》演唱的片尾曲。曲调温婉动听、清新治愈，将旧时光的温度悉数传递。

刘惜君以温婉、细腻的嗓音娓娓诉说着每个人对旧时光的怀恋。人总喜欢怀念过去，并不是怀念过去的人，而是怀念过去的青春，恰好青春中有那个人。然而，错过的终究回不去，正如歌词中所唱“回不去的留不下的珍藏在心底”。
