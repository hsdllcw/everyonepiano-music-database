

**以风为马** 是叶炫清演唱的歌曲，由张靖怡作词，刘炫豆作曲，该歌曲是电视剧《清平乐》插曲，于2020年4月21日发行。

《清平乐》该剧改编自米兰lady的小说《孤城闭》，以北宋为背景，在风起云涌的朝堂之事与剪不断理还乱的儿女情长之间，还原了一个复杂而真实的宋仁宗。

叶炫清的声音特别的轻柔，特别的深情，而且动听。歌曲所阐述的内容也是令人十分揪心，宋仁宗是一个虚心纳谏的君王，可唯一的几次任性却全给了她，没想到最初的温柔也变成了最后的伤！
