

**余年双手简谱** 和五线谱完全对应。

一曲清歌扣心弦，一腔温柔诉余年。由肖战演唱的电视剧《庆余年》片尾曲《 _余年_
》乘兴而来。歌曲缓缓放出，嗓音抓耳，温柔细腻，倾诉着一段不怨不悔的青春，诉尽一生的信仰。

《庆余年》该剧改编自猫腻的同名小说，讲述了一个有着神秘身世的少年，自海边小城初出茅庐，历经家族、江湖、庙堂的种种考验、锤炼的故事。
