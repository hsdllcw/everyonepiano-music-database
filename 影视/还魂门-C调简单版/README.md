

内地全能歌王胡彦斌与著名作词人林文炫再度默契合作电视剧《老九门》主题曲，为歌迷及剧迷共同开启神秘《 **还魂门**
》。歌曲萦绕着二爷和丫头的绝恋，让人感觉到这份爱的痛和美，很值得品味。

首次与南派三叔合作的胡彦斌胡校长为渲染MV的气氛突出电视剧《老九门》的意境，特意选在中元节凌晨一两点开拍，拍摄场景更是选在了郊区荒无人烟的废弃工厂，现场气氛格外安静阴森诡异。胡校长则在MV中化身成南派三叔小说中的人物，穿越时空，置身民国。与南派三叔塑造的故事情节一样，《
_还魂门_ 》如同开启了老九门的一扇神秘大门，将大家带入到一波三折的故事当中。
