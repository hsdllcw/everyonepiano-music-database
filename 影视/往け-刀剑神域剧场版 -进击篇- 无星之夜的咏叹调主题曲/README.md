无星之夜的咏叹调》的主题曲，由LiSA（织部里沙）演唱。LiSA的歌总是凭借一种激动人心的旋律打动听众，只因为能和听众产生共鸣。用其最摇滚的方式唱出最真挚的感情，有欢喜，有时也有悲伤。
