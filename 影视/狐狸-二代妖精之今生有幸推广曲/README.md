

**狐狸** 是由周以力作曲，薛之谦作词并演唱的歌曲。同时也是电影《二代妖精之今生有幸》的推广曲。

作为薛之谦在2017年发布的最后一首新歌，这首歌延续了薛之谦充满故事感的曲风，通过狐狸和猎人两个孤独影像的若即若离、缱绻试探，影射出人在感情世界里习惯性的猜测防备与自我保护，刻画出情至浓时无限欣喜与痛苦肆虐的交叠画面。
