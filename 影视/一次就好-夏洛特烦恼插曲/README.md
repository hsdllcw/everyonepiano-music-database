

**一次就好**
是电影《夏洛特烦恼》中的插曲，由由董冬冬、陈曦作词作曲，杨宗纬演唱，发行在《夏洛特烦恼电影原声带》中，是电影主人公“夏洛”给喜欢的女生秋雅创作的歌曲。这首歌在电影中共出现过三次，前两次都是夏洛唱给校花听，第三次则是马冬梅唱给病床上的夏洛听。

《 _一次就好_
》本来没有专门录歌，是导演为了满足观众而特意录的。歌曲旋律很自然的把观众带入进电影的情节当中：那一年的教室、夏洛的校服、冬梅的齐刘海、大春的内裤外穿、秋雅的回眸一笑、袁华的一剪梅、梦特的粉红桌布、张扬的龅牙、王老师的考前课堂。听着听着眼眶就红了，1997年，那一年的西红市第七中学。一首很暖心的歌。

歌词下方是 _一次就好钢琴谱_ ，希望大家喜欢。
