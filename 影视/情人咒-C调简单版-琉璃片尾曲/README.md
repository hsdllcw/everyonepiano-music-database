

**情人咒**
是电视剧《琉璃》片尾曲，由郁可唯和阿云嘎两人合作献唱。娓娓道来剧情人物的爱恨甜苦，“不经一番心彻骨，怎知向何处，深藏知觉于肺腑”攻克爱情难关，唯有勇敢坚定！

《 _情人咒_
》歌词、旋律婉转哀艳，代入感十足的声音向我们一点点诠释了所谓“情人咒”就是指人们心中的懦弱与不自信，唯有彼此信任、心意想通，才能无惧风雨侵袭。

“伤筋又动骨的痛苦”“慢热到冷酷的幸福”喜欢这两句的咬字，沉沉地回忆，静静地诉说，若身在其中，无论甜哭，又怎可轻易自拔。阿云嘎的嗓音是上古潺潺趟过的冰泉，本身是清冷寂寥，悠悠自处。郁可唯的声线是透亮涌动的暖流，汇于一起，悄然转变，也像剧中人物，囚于情深又从中互相救赎。
