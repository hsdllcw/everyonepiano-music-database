

**烟雨蒙蒙双手简谱** 和五线谱完全对应。

_烟雨蒙蒙_
是琼瑶电视剧《情深深雨濛濛》台湾中视版片尾曲及内地央视版的插曲，女声版由赵薇演唱，收录于原声带中，男声版由古巨基演唱，亦收录于原声带及其个人专辑《恋恋情深》中。

歌词下方是 _烟雨蒙蒙钢琴谱_ ，大家可以免费下载学习。
