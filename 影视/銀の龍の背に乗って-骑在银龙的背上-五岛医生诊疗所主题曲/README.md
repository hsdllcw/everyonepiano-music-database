《 **銀の龍の背に乗って** 》（骑在银龙的背上）由中岛美雪演唱，是日剧《五岛医生诊疗所》第一、二部的主题曲。范玮琪演唱的《[
**最初的梦想**](Music-1625.html "最初的梦想")》就改编自此曲。

网友经常误以为银龙是指电视剧中，主角那辆叫“银龙”的自行车，“骑在银龙背上”代指骑着自行车穿行于村落之间的乡间医生。但中岛美雪亲自回应过，银龙颜色的来源是医生手里的手术刀。

《五岛医生诊疗所》是由日本富士电视台于2003年7月3日播出的医学、励志类电视剧，改编自山田贵敏的漫画作品《孤岛诊疗所》，该剧以日本南海的一座小岛为舞台，讲述了年轻医生五岛健助和岛上的人们为了保护每一条生命努力拼搏的故事。
