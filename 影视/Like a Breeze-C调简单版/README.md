

歌曲《 **Like A Breeze**
》作为电视剧《以家人之名》插曲，由萨吉作词、黄涌周作曲、剧中人物李尖尖扮演者谭松韵献唱、金大洲担纲制作，曲风清新浪漫，似是一阵爱情的清风自然的吹拂在心尖上，充满了爱意萌生的甜蜜悸动感。

在那么多共同经历的日子里，藏着太多的了解、太多的喜欢，还有太多的摩擦、太多的平淡。它们就像空气，一直围绕在他们身边，习以为常，视为自然。

但就像不知何时突然有一阵微风吹过，把这身边的一切都凝结成了两个新鲜的字眼——爱情。从手牵手的兄妹变成手牵手的恋人，一切都绕了太久，李尖尖以为自己学不会新的牵手，猝不及防的怦然心动来得那么悄无声息却也自然而然。

李尖尖此刻才终于明白，那些渺小的日子、那些窝心的自然、以及那些琐碎的瞬间了，原来都是爱。不必像童话，也不用说什么天长地久，只要你在我眼里，我就在你身边。
