

电视剧《轩辕剑之天之痕》主题曲《 **一吻天荒** 》由胡歌演唱。字字句句中讲述了世间别样的爱恨情仇，再现江湖的豪迈与柔情。

这首歌曲的旋律走向与剧情一样层次分明跌宕，编曲中加入电子乐器配合整部剧的节奏。歌曲的演唱者胡歌也突破以往风格，把剧情中人物的情感深度挖掘和升华，再用他自己独特的方式和唱腔把《
_一吻天荒_ 》这首歌曲表达出来。胡歌录制完成后也表示自己非常喜欢这首歌。
