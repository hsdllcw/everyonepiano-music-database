

**时光正好双手简谱** 和五线谱完全对应。

_时光正好_ 是电影《夏有乔木雅望天堂》的推广曲，由唐恬作词，金大洲作曲，郁可唯演唱。

郁可唯、金大洲、唐恬三人首次合作。此次合作中，制作人金大洲对郁可唯的认真度大为赞赏，配唱时郁可唯仔细揣摩、精准拿捏每个咬字的发声，对歌曲气息的把控更是展现精湛唱功。而填词人唐恬，抢先看完《夏有乔木
雅望天堂》后，被吴亦凡在影片中的表演所打动，仅用一稿就写中《时光正好》的歌词。而听到郁可唯演唱的最终版《时光正好》，唐恬更是表示，“这是我最近听到最好听的一首歌！”。郁可唯则表示：“希望大家听到《时光正好》，能够想起自己青春期里最无可代替的感情。”整个幕后团队，对于《时光正好》信心满满、诚意十足，寄望“时光正好你不来我不老”能在更多人的心里留下青春印记。
