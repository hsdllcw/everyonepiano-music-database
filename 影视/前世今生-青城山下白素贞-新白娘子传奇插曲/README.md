

**前世今生双手简谱** 和五线谱完全对应。

_前世今生_ ，我们也可以叫做青城山下白素贞，名字不同但曲调一样，同是《新白娘子传奇》中的插曲，由左宏元作曲。

《新白娘子传奇》是中国重播次数最多的港台电视剧，几乎每年都重播。好剧配好乐，该剧盛演不衰，每一曲都好听，且耐听。无论是演员，服装造型，法术特效，启用反串大胆创新，边演边唱方式独特，整部剧相当唯美。赵雅芝端庄大方、温柔高贵的白娘子形象以及叶童的反串许仙都已深入人心。
