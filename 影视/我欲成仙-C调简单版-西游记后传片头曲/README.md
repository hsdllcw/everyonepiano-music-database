

**我欲成仙** 是由赵小源作词，三宝作曲，刘欢演唱的一首歌曲，于2000年2月15日发行。同时，此曲也是古装神话电视剧《西游记后传》的片头曲。

《西游记后传》改编自吴承恩所著古典小说《西游记》。该剧主要讲述了唐僧师徒西天取经之后，魔头无天在如来佛祖圆寂后大闹三界，孙悟空带领三界帮助如来转世灵童乔灵儿重返灵山的故事。该剧已于2000年播出并获得第18届金鹰奖电视美术片类最佳奖。
