《 **铃芽** 》（ _すずめ_ ）是野田洋次郎作词、作曲，十明演唱的一首歌曲，于2022年9月30日公开发布。同时，此曲也是动画电影《铃芽户缔》主题曲。

十明（Toaka） 令人印象深刻的嗓音和细腻的歌声给电影带来了全新的感动，野田表示：“在这首歌和十明（Toaka）之间，我感受到了一种无法割裂的羁绊”。

《铃芽户缔》（日语：すずめの戸締まり），是日本动画导演新海诚编剧兼执导，预定2022年11月11日在日本上映的动画电影。本片为新海诚的第八部动画电影，也是他继《你的名字。》、《天气之子》后，第三部牵涉灾难题材的电影。标语为“门的对面，有着全部的时间”（扉の向こうには、すべての時間があった）。故事讲述少女岩户铃芽与以关闭灾难之门为使命的“闭门师”宗像草太，为了关闭灾难之门展开的冒险旅程。
