

“给你的爱一直很安静，来交换你偶尔给的关心，明明是三个人的电影，我却始终不能有姓名”是歌曲《 **一直很安静**
》中让人感触最深的一句。直至今日再听，仍有无限感伤。

《 _一直很安静_
》由中国台湾流行音乐女歌手阿桑演唱，是神话悲情剧《仙剑奇侠传》的插曲。阿桑的歌里表现得更多的是一种遗世独立的孤独感，为自己倾诉，为自己伤感。声音不妩媚，不讨好，很难得。

这首忧伤的歌曲专为《仙剑奇侠传》剧中女主角林月如而作。林月如为了拯救逍遥和灵儿的生命去了，最后阿桑也去了，不得不说令人感伤。忧伤的旋律牵动失落的每一颗，原来只有这样才能理解音乐。她走得很安静，漂泊在安静的最美境界里。
