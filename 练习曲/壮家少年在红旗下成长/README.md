

它本来是叫《壮家少年在红旗下成长》，传唱中也用过《壮家少年热爱毛主席》的名称。它是广西第一首唱响全国的儿童歌曲，是许多人心中难忘的经典。

### 《壮家少年在红旗下成长》歌词：

红艳艳的木棉在南疆开放，  
壮家的少年在红旗下成长。  
教室里操场上好好学习天天向上！  
工厂里田野上学工学农歌声嘹亮。  
啊！  
壮家的少年象春天的花朵，  
象茁壮的秧苗，  
沐浴着灿烂的阳光！  
  
红艳艳的木棉在南疆开放，  
壮家的少年在红旗下成长。  
学政治学文化革命本领党培养，  
心儿红身体壮朝气蓬勃有理想！  
啊！  
壮家的少年在祖国的怀抱，  
在美丽的壮乡，  
肩负着人民的希望！  
  
木棉花迎春风，  
红领巾披霞光，  
壮家少年颗颗心儿向着红太阳。  
毛主席啊毛主席，  
壮家少年在您关怀下幸福成长！
