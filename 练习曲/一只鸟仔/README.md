

**一只鸟仔** 是一首台湾的汉族童谣，因旋律简单，曲调诙谐，内容生动，而流传广泛。

它的唱词通过咏唱四种禽鸟的动作、叫声在儿童内心激发起的美好感情。唱词共四段，每段描写一种家禽或飞鸟，用词准确、平实而又精练，作为儿歌，一方面要求被描写的对象应该是满足儿童的天性，另一方面则应当使用儿童的观察角度及他们的语言来描述这些对象。这两个要求在《一只鸟仔》中体现得非常出色。与歌词相适应，它的曲调也以简洁、清淡的音乐语汇来突出天真、机敏的性格。全曲四个乐句，结构方整，衔接紧凑，以Re(2)、Sol(5)、Do(1)、Sol(5)四个结音体现了起、承、转合的四句头原则。“Sol”虽然是主音，但最活跃的却是“Re(2)”“La(6)”这两个音，在第一、二、四句中，它们均处于中心地位，唯第三句被“Do(1)”所取代。其中，最具特色的音调则是第二、四句尾部的“=”这个音型。

### 一只鸟仔歌词：

透早起来YIDU拐一下拐，一只鸟仔YIDU哮啾啾，  
踮在水沟子YIDU撬一下撬，丢丢铜子YIDU找无巢噢。  
日头落山YIDU刮一下刮，一只水鸡仔YIDU哮呱呱，  
踮在田地子YIDU撬一下撬，丢丢铜子YIDU找无伴噢。  
鸡母仔找子YIDU咯一下咯，一只鸡仔YIDU哮啾啾，  
踮在草埔子YIDU撬一下撬，丢丢铜子YIDU找无母噢。  
鸭咪仔泅水YIDU扎一下扎，一只鸭仔YIDU哮咪咪，  
踮在水沟子YIDU撬一下撬，丢丢铜子YIDU找无伴噢。
