

**明日双手简谱** 和五线谱完全对应，是EOP教学曲，非常适合新手练习。

_明日_ 是《爱情白皮书》演奏插曲。《爱情白皮书》是日本漫画大师柴门文经典名作，并首次同意授权海外拍摄成高质的电影感剧集
，参照日剧唯美的拍摄手法，每一个镜头、音乐都震慑人心，剧中透视爱情感觉的悲喜酸甜。一段围绕五位年青人的爱情故事由无
拘无束、写意的校园生活里发生。由超人气青春班底演绎，有炙手可热的人气偶像余文乐，凭《流星花园》小优一角被受关注的杨
丞琳，加上首度参演戏剧的台湾知名歌手范玮琪，来自温哥华的彭于晏，以及洛杉矶的李元伯，清新亮丽的组合，谱出校园式的青
葱岁月情怀，揉合浪漫调子，将细腻情感娓娓道来。

明日~不要哭,Tomorrow Is Another Day 由张中立演奏，非常好听的轻音乐，如果看过这部电视连续剧的童鞋，相信听到 _明日钢琴曲_
会有很大的感触。
