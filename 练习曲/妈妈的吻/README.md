

**妈妈的吻** 是70年代末，由付林作词、谷建芬作曲的《妈妈的吻》是为了抒发一种来自乡村母子之间的真挚感情、反映底层文化、草根文化而创作的。  
  
这是非常朴素的、中国式乡村的人文情怀。付林说："这首歌曲带来的意境确实触动了人们内心深处的情愫。时代赋予我们的机遇就是要给这些乡村的文化带来一些改变。"
