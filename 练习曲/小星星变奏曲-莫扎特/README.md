小星星变奏曲是莫扎特于1778年所创作的钢琴曲，是一首脍炙人口的名曲。本曲是为钢琴独奏所创作，包含13小节：第一小节为主题，接下来是变奏1到12。原谱中仅有变奏11和12有节奏指示，分别为柔版（Adagio）和快板（Allegro）。

  
本曲为在当时在法国流行的爱情歌曲《妈妈请听我说》（“Ah! vous dirai-je, maman”
）基础上创作的变奏曲。该曲的旋律后来被改编成广泛传唱的童谣《小星星》，因此中国国内也将其改称为《 _小星星变奏曲_
》。但《小星星》的歌词是在莫扎特逝世后才创作的，莫扎特本人对该命名也无从知晓，因此严格来讲该命名并并不确切，仅因为简短而为人所熟知。
