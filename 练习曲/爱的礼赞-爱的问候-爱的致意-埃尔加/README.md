

《 _爱的礼赞_ 》亦译作《 _爱的问候_ 》及《情人的问候》，或《 _爱的致意_
》，是埃尔加献给妻子爱丽丝的求婚曲，原名《Liebesgruss》(Love's Greeting)。  
  
本曲具有典型的“小夜曲”风格，轻松而典雅，是极适用于舞会和沙龙等场合的音乐。
乐曲为典型的ABA三段体加一个尾声构成。A段主题为E大调，小提琴在高音区奏出饱含深情的旋律，恰似来自情人的绵绵情话；乐曲的B段主题转为G大调，柔美的曲调中多了一丝哀怨的情调，经过带有泛音色彩的华丽过门，乐曲又回到A段，经过带有复杂情绪的尾声，全曲在A段主题的变奏形式下逐渐减弱而终了，仿佛是情人还在喃喃私语。  
  
埃尔加（E. Edward Elgar
1857～1934)），英国作曲家、从男爵。1857年6月2日生于伍斯特附近，自幼学习音乐，曾任教堂管风琴师。代表作品有《加冕颂歌》、《威仪堂堂》、《谜语变奏曲》、《杰龙修斯之梦》，是英国音乐开始复兴的标志性人物。
