又称巴哈。1685年出生于德国, 最擅长用严密工整的复调形式来表现人们丰富的内心世界, 并把复调音乐推向了后世无人能够超越的, 完美的顶峰,
被尊称为"西方近代音乐之父"。

那什么是复调呢? 简单来说, 复调音乐是两条或两条以上的旋律线, 有机的结合在一起的多声部音乐。之前学习的教材如大汤, 车尔尼599或者拜厄,
会发现虽然两只手都在弹奏, 可左手更多的是简单伴奏, 右手为主旋律。而在巴赫初级中, 左右手都有可能是旋律,
也都有可能是伴奏，互相呼应。虽然从谱面看音符都很简单, 也没有什么特别的地方, 可弹起来就是觉得很困难。
