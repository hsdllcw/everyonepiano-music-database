

《摇篮曲》，这首常用于小提琴独奏的《摇篮曲》，原是一首通俗歌曲，
勃拉姆斯的摇篮曲作于1868年。相传作者为祝贺法柏夫人次子的出生，作了这首平易可亲、感情真挚的摇篮曲送给她。法柏夫人是维也纳著名的歌唱家，1859年勃拉姆斯在汉堡时，曾听过她演唱的一首鲍曼的圆舞曲，当时勃拉姆斯深深地被她优美的歌声所感动，后来就利用那首圆舞曲的曲调，加以切分音的变化，作为这首《摇篮曲》的伴奏，仿佛是母亲在轻拍着宝宝入睡。

原曲的歌词为“安睡安睡，乖乖在这里睡，小床满插玫瑰，香风吹入萝里，蚊蝇寂无声，宝宝睡得甜蜜，愿你舒舒服服睡到太阳升起。”那恬静、优美的旋律本身就是一首抒情诗。后人曾将这首歌曲改编为轻音乐，在世界上广为流传，就像一首民谣那样深入人心。
