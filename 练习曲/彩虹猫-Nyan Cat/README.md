_彩虹猫_ ，英语Nyan Cat，是由初音未来洗脑曲Nyanyanyanyanyanyanya!改编的萌系音乐视频。

桃音モモ版本的“Nyan
Cat”（Nyanyanyanyanya）是最流行的版本，最初是daniwell创作由初音演绎，后来もももも（即桃音モモ的画师）创作了桃音モモ版的“Nyan
Cat”，接着YouTube用户saraj00n将喵喵猫的动画与もももも的歌编辑在了一起，取名叫“Nyan
Cat”上传在YouTube上面，并造成流行，如今很多其他改造的“Nyan Cat”都是桃音モモ版。

彩虹猫 Nyan Cat与它的网页版Flash游戏一样，保持了Nyan
Cat欢快有趣的形象和动作，并且带着2D像素风格的画面，也得到了很好的保持。然后配合上那首从头到尾都是[Nyan]的曲子，绝对是洗脑度十足。另外玩的时候强烈建议带着耳机玩，可以加深洗脑度。
