金瓶似的小山_ 是一首创作于上世纪五十年代的歌曲，由朱丁作词，经中央乐团正式配器演唱，李双江，阎维文曾经演唱过这首歌曲。  
  
《金瓶似的小山》的词作者朱丁，二十世纪五十年代他在青海湖草原采访时，受到感染启发，写下了脍炙人口的歌词。歌词曾在《青海日报》上刊发，后在著名诗人郭沫若的一篇文章中引用，经中央乐团正式配器演唱，由中央人民广播电台播出后传唱全国城乡。
