

《我爱北京天安门》（I Love Beijing
Tiananmen）是一首颂赞毛泽东的歌曲。创作于1970年，于1971年正式在中央人民广播电台首播。因为歌曲旋律清新、节奏活泼，只有9度音域，演唱的适应面广，富于儿童特点，不同于当时占据乐坛的"语录歌"、"忠字歌"千篇一律的进行曲、战歌那样的高亢、声嘶力竭，在当时迅速脱颖而出，并成为了70年代文革后期的儿童和青年人传唱的经典红歌。改革开放后的21世纪出版了同名的电影、电视剧和公益广告。
