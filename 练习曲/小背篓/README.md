

《小背篓》是一首脍炙人口的民族歌曲，由中国著名女高音歌唱家，国家一级演员宋祖英演唱后曾红遍大江南北。《小背篓》还是著名电视连续剧《婉君》的插曲。《婉君》是台湾著名女作家琼瑶的系列电视剧《六个梦》中的第一部作品，1990年在台湾首映，后大陆引进本剧，播出之后迅速红透了华夏大地。
