

**G小调第一叙事曲** 作于1831-1835年之间，编号23。 _G小调第一叙事曲_
是肖邦四首叙事曲中创作最早的一首，是第一首采用这种体裁名称的钢琴作品。席林斯基说这个叙事曲“不仅没有词，而且也没有任何具体化的内容，但却与曲名十分一致和贴切”。这部作品无论在音乐史上还是在肖邦的全部创作生涯中都具有不可替代的历史意义。  
  
此外，不知道大家有没有观看过这部比较有名的电影《钢琴家》。电影中讲述了钢琴家斯皮尔曼正在经历黎明前最黑暗的时候，他最后的住所被炮火摧毁，衣衫褴褛地在废墟中等着胜利的到来。他巧妙地躲在德军指挥所的阁楼中，却被德国军官，威廉发现。不过万幸的是，威廉并没有处决这个犹太人，当得知了斯皮尔曼的职业后，令他演奏了一首钢琴曲，并让他继续藏匿在阁楼中，还不时地送来食物和衣物。斯皮尔曼为威廉所演奏的，正是《
_G小调第一叙事曲_ 》。
