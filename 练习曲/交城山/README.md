

**交城山**
是山西中部传统民歌。是旧社会一个嫁到交城县山区去的农村妇女对贫穷生活的叙述：最好的食物是“莜面饸饹饹”（即莜面卷）和“山药蛋”（即土豆）；出门就爬山，只能步行或骑毛驴，连姑娘出嫁也没法坐上好车马。

_交城山_
此曲音乐十分纯朴憨厚，旋律很美，富有鲜明的晋中地方特色和乡土生活的气息。乐段由两乐句构成，旋法多下行。现已成为晋中秧歌中常用的曲调之一。歌剧《刘胡兰》、新民歌《妇女自由歌》等音乐作品也完整地运用它。

歌词下方是 _交城山钢琴谱_ ，希望大家喜欢。
