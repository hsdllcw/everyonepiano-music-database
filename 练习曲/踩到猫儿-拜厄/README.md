_踩到猫儿_
是德国钢琴家拜厄创作的曲子。他一生都致力于钢琴教育研究，其代表作为《钢琴基本教程》。国内外很多学习钢琴的孩子，都是从这一本书的练习开始的，这是一本非常适合儿童的钢琴演奏简易入门教科书。

这是一首非常欢快的曲子，深受大家的欢迎。这首曲子被改编成了很多版本，比如日版，采用了钢琴曲的同名标题，看起来“乱糟糟”的歌词非常有意思，被踩到脚的猫咪吓得上串下跳，踢到了金鱼缸、扰乱了鸡窝、跳上了衣橱……整个情景看上去非常糟糕，但却将活泼跳跃的曲调诠释得恰到好处，上串下跳、不停动来动去的小猫，像极了黑白琴键上跳跃的手指，也像极了五线谱上上下跳动的音符。这首儿歌真是可爱极了！

歌词下方是 _踩到猫儿钢琴谱_ ，大家可以免费下载学习。
