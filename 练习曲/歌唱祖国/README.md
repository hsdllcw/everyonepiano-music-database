

《歌唱祖国》由王莘作词、曲，是著名的爱国歌曲。创作于1950年9月，适逢新中国成立一周年，看着天安门广场五星红旗随风飘扬，鲜花如海的热闹景象，王莘脑海里反复酝酿，《歌唱祖国》是在回津的列车上一气呵成。这首歌曲发表后因其明快雄壮的韵律而广为传唱，现在已经成为中国各种重大活动的礼仪曲、开场曲或结束曲，号称"第二国歌"。
