_进行曲 9_
是作曲家巴赫创作的曲子。巴赫，巴洛克时期的德国作曲家，杰出的管风琴、小提琴、大键琴演奏家，被普遍认为是音乐史上最重要的作曲家之一，并被尊称为“西方近代音乐之父”，也是西方文化史上最重要的人物之一。

进行曲主要是军队中用来统一行进步伐一致的要求，以偶数拍作周期性反复，常用2/4 ，4/4 的拍子。 进行曲最初出现在古希腊的悲剧中。
进行曲原是舞曲的一种，多用于群众出场，退场的时候。十七世纪起，渐渐传入音乐艺术的领域。 现代进行曲则指十七世纪以后，用铜管乐队（此处有异）吹奏的乐曲。.
