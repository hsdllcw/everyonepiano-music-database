

**湖上天鹅** 是约翰汤姆森的钢琴练习曲之一，节奏，音节都非常简单，是初学钢琴者的最佳之选。  
  
约翰·汤普森（John Thompson），男，美国人，是一名钢琴教育家。出版有约翰·汤普森钢琴教程系列。  
  
其中最著名的是约翰·汤普森钢琴简易教程[1] ，是世界公认的初学钢琴的权威性教材。 _湖上天鹅_
就出自该教程，该教程由美国威利斯（WILLIS）音乐出版公司出版，现在在全世界已有7种文字的版本。汤普森钢琴教程以对音乐名作的简易改编为特点，从“学钢琴首先是学音乐”的战略眼光来看有着十分积极的意义。对琴童进一步学习复杂节奏、聆听内声部及低音旋律、训练手指的触键和跑动、熟悉和理解各种风格均提供了丰富的训练内容。
