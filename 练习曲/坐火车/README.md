

《坐火车》节奏明快，语言押韵，能呈现出鲜明的音乐性和节奏感，为培养孩子的乐感打下基础！在听儿歌，唱儿歌的过程中，孩子在慢慢的理解儿歌中讲述的故事，这样有利与培养孩子的理解能力、记忆力和模仿能力。在这熟悉的旋律中，带您的宝宝一起来体验儿时的经典儿歌吧。

### 《坐火车》歌词欣赏：

小板凳呀摆一排，  
小朋友们坐上来，  
坐上来呀，坐上来。  
我的火车跑得快呀，  
大家坐稳就开车，  
轰隆隆隆，轰隆隆隆，  
轰隆隆隆，呜~~~

抱小娃娃的前面坐，  
牵小狗的往后拉，  
往后拉呀，往后拉。  
大站小站我都停呀，  
大家坐稳就开车，  
轰隆隆隆，轰隆隆隆，  
轰隆隆隆，呜~~~
