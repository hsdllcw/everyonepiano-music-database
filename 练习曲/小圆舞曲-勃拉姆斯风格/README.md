

勃拉姆斯是德国音乐史上最后一个有重大影响的古典作曲家，被视为19世纪浪漫主义音乐时期的“复古”者。他是创作与演奏并重的作曲家。他继承贝多芬交响乐的传统，以深刻的人道主义和热烈的爱国主义精神出发，着力表现时代精神风貌和斗争生活，作品成为贝多芬之后西欧交响音乐的杰出典范。

他一生作有九十余首奥地利民歌的改编曲。所作形式繁多的重奏曲，提高了室内乐的地位。此外，他还作有二百余首歌曲、四部交响曲、一批钢琴小品与主题变奏曲、协奏曲，其中包括著名的《D大调小提琴协奏曲》。他的《匈牙利舞曲》第一集和第二集，可以说是他的最为人们熟知的代表作品，乐曲运用匈牙利舞曲的节奏特点、旋转的风格和浓郁的乡土气息，表达了匈牙利人民的精神，而且旋律色彩丰富，富有生气，很受人们欢迎，最著名的就是其中的《匈牙利舞曲第5号》。

然而他所作的圆舞曲却并不为人们熟知。1863年，勃拉姆斯来到维也纳，次年夏天他结识了约翰.斯特劳斯，并与他有着亲密的来往。1865年，勃拉姆斯出版了这套《圆舞曲集》。该曲集共有16首乐曲，原来是四手联弹钢琴曲，后来作者又把它们改编成钢琴独奏曲。后人又改编成小提琴曲、管弦乐曲。其中最有名的是第15首降A大调圆舞曲，全曲表现了与浪漫而华丽的维也纳圆舞曲相异其趣的一种质朴而清朗的美，代表了勃拉姆斯的风格。此外他的第11曲充满吉普赛音乐风格；第12曲忧郁，伤感，像勃拉姆斯的自述，也是十分杰出的作品。勃拉姆斯的圆舞曲延续了他的其他许多作品的风格，乐曲短小精悍，清新温柔，又有着作者一贯的忧郁，淡雅，孤傲和落寞的气质，与约翰.斯特劳斯的圆舞曲风格有着完全不同的表现方式，虽然与众不同，却仍不失为圆舞曲中的精品。

欣赏勃拉姆斯的圆舞曲，请不要把它和华尔兹舞蹈联系起来，这样的乐曲，适合在晚霞余辉渐渐淡去的傍晚，静下心来慢慢品味，让音乐轻轻洗去白天繁杂纷乱的工作带给自己的混乱心情，让音乐温柔地洗涤自己被世事污秽的心灵，让自己的心回归它本来的纯净、平和、温柔而充满爱。
