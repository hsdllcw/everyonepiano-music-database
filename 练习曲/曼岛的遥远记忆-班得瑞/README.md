

**曼岛的遥远记忆** ，是首轻音乐，英文名为Beyond the Memory of Man，是班得瑞《雾色山脉》专辑的第2首曲子。

  
《雾色山脉》（Misty Land）是班得瑞第11张专辑山林音乐，走出瑞士山林，寻找地球上最纯净人间净土收录《魔戒三部曲》May It Be, Into
the
West主题曲演奏版。本辑中，班得瑞除了以孕育其音乐灵感的瑞士山林为出发，更出走国境，让南美、爱尔兰、与东方的自然风光也融入创作中，歌颂著无国界的大地恩赐与鬼斧神工的自然魅力。在编曲上除了运用吉他、排笛、键盘等在乐坛奠定鲜明旗帜的配器外，更加重了弦乐组的表现，让他们的音乐在空灵飘渺之中，更增添一份来自田园，沁入心灵的温暖情感。
