

**念故乡** (Going
Home)。是从德沃夏克著名的第九交响乐《自新大陆》的第二章的慢板改编的的歌。动人的旋律和优美的音乐，加把游子的思乡之情，表达得淋漓尽致。另外在上世纪三、四十年代李抱枕先生为这段乐曲填上了贴切歌词并广为国人传唱。

此曲现也常作为美国军人在海外殉难后运回国内时的哀乐(背景音乐)。

歌词下方是 _念故乡钢琴谱_ ，希望大家喜欢。
