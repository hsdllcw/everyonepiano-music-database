

**望星空**
，此歌由石祥作词，铁源作曲，军旅歌唱家董文华演唱。董文华（1962年6月29-）生于沈阳，中国民歌坛殿堂级实力派天后，亦是20世纪80年代至今最富盛名的中国歌坛巨星之一。中国人民解放军总政治部歌舞团独唱演员，国家一级演员，享受国务院“政府特殊津贴”。

  
歌词下方是 _望星空钢琴谱_ ，希望大家喜欢。

### 望星空歌词：

夜蒙蒙望星空我在寻找一颗星  
它是那么明亮它是那么深情  
那是我早已熟悉的眼睛  
我望见了你呀你可望见了我  
天遥地远息息相通  
即使你顾不上看我一眼看上我一眼  
我也理解你呀此刻的心情  
夜深沉难入梦我在凝望那颗星  
它是那么灿烂它是那么晶莹  
那是我敬慕的一颗心灵  
我思念着你呀你可思念着我  
海誓山盟彼此忠诚  
即使你化作流星毅然离去  
你也永远闪耀在我的心中在我的心中
