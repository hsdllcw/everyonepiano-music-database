

**美人鱼之歌**
是德国作曲家卡尔•马利亚•冯•韦伯创作的曲子。韦伯（1786一1826）出生于一个戏剧之家，对德国民间音乐、风俗也有很深的体验。这对他以后的音乐创作具有重要的意义。韦伯十岁学钢琴，以后又学作曲，十二岁开始写作歌剧音乐。1813年以后，他先后任布拉格剧院和德累斯顿交响乐团常任指挥，对这两个演出团的成名起了重要的作用。代表作是歌剧《自由射手》即《魔弹射手》。它那浓郁的德国风格和浪漫气息，被认为是德国第一部浪漫主义歌剧。此外，他还写了歌剧《奥伯龙》，以及少序曲、协奏曲，奏鸣曲。其中，钢琴曲《邀舞》最为脍炙人口。
