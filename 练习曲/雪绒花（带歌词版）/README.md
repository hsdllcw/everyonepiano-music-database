随着电影的播放，其舒缓、轻柔的风格深深抓住了听众的心，现已成为世界各国人民喜爱的歌曲之一。  
  
_雪绒花_
此曲为带再现的单二部曲式，旋律的节奏基本以四分音符、二分音符、附点二分音符这三种音符构成，三四拍子和中等速度使歌曲没有较大的起伏。歌曲通过对雪绒花的赞美，抒发了对大自然的热爱，寄托了主人公对亲人、家乡深深的怀念和祝福。  
  
全分解和弦伴奏：将和弦分别弹奏的一种伴奏形式，这种伴奏适合于抒情曲或欢快曲，由于旋律不同，因此可以分为两种：以原位主和弦为中心的分解和弦伴奏和以主和弦第二转位为中心的分解和弦伴奏。该曲属于三拍子的全分解和弦伴奏，常用于旋律中带有低音的5音。
