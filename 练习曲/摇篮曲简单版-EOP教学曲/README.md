

这首谱子为EOP教学曲，简谱和五线谱是对应的。

摇篮曲，原是母亲抚慰小儿入睡的歌曲，通常都很简短。旋律轻柔甜美，伴奏的节奏型则带摇篮的动荡感。许多大作曲家如莫扎特、舒伯特、勃拉姆斯都写有这种歌曲。由于音乐平易、动人，常被改编为器乐独奏曲。此外也有专为器乐写的摇篮曲。
