

《孩子们的拜厄》分上下两本书，由上海音乐出版社出版（第一版2008年11月1日），是时下中国最流行的钢琴练习教程之一。

说明：这本练习曲并没有100%按照原著来做，是钢琴教师按照教学需求并根据多年的教学经验与英文原版改编而成的，希望对钢琴初学者有一定的帮助。

费迪南德·拜厄，（Ferdinand
Beyer，1803～1863）德国钢琴家、钢琴教师。1803年生于德国奎尔福特，一生中长期受雇于乐谱出版商——德国迈恩斯的索特公司（Schott
co.），1863年5月14日逝世于该地。作品有大量简易改编曲、集锦曲、游嬉曲与幻想等钢琴乐曲。著名教材——《钢琴基本教程》为学习钢琴演奏艺术的简易入门教科书，以儿童（包括学龄前幼儿）为对象。内容有乐理知识、大小调音阶、半音音阶联系以及双音、三连音。倚音、单手、三手、四手连弹等各种联系条目与旋律优美、节奏鲜明的简短小曲109首。是初学钢琴者必修的教本。

拜厄的钢琴教程经过一百多年的考验，仍被教师和学生所喜爱。拜厄钢琴基本教程是钢琴学习过程重要的基础教材，科学、规范地练习它，对学琴者钢琴演奏水平的发展和提高，有着非常重要的作用。
